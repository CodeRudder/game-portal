## 一、总览

本计划收录 **100 款经典小游戏**，按复杂度和类型分批迭代交付。目标是构建一个覆盖益智、街机、策略、跑酷、休闲、对战、卡牌七大类型的完整游戏库。

## 五、技术架构（当前状态）

### 5.1 引擎基类

```
GameEngine (Abstract Base)
├── 生命周期: init → start → pause → resume → reset → destroy
├── 游戏循环: gameLoop → update(dt) + render(ctx)
├── 事件系统: on/off/emit (statusChange/scoreChange/levelChange)
├── 子类必须实现:
│   ├── onInit() / onStart() / update() / onRender()
│   ├── handleKeyDown() / handleKeyUp() / getState()
│   └── 可选: onPause() / onResume() / onReset() / onDestroy() / onGameOver()
```

### 5.2 Canvas 规格

- 标准尺寸: 480 × 640（3:4 比例）
- 渲染: `requestAnimationFrame` 60fps
- 响应式: CSS `width: 100%; max-width: 480px; aspect-ratio: 3/4`
