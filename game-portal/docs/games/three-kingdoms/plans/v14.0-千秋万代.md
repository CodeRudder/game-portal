## 二、游戏引擎需求

### 新增子系统
| 子系统 | 职责 | 依赖 |
|--------|------|------|
| PrestigeSystem | 声望核心：等级管理/阈值计算/升级判定/加成注册 | — |
| PrestigeShopSystem | 声望商店：点数兑换/等级解锁/限购管理 | CurrencySystem(v8.0) |
| RebirthSystem | 转生核心：条件检查/倍率计算/数据重置/加速机制 | 全系统(重置涉及所有系统) |
| RebirthSimulatorSystem | 转生模拟：收益预测/时机推荐/倍率对比 | PrestigeSystem, RebirthSystem |
| AchievementSystem | 成就核心：多维度成就/进度追踪/完成判定/奖励发放 | 全系统(监听各系统事件) |

### 改进现有子系统
| 子系统 | 改进内容 |
|--------|----------|
| QuestSystem(v7.0) | 新增声望专属日常任务+转生专属任务 |
| OfflineRewardSystem(v9.0) | 新增声望离线累积(领土产出×50%系数) |
| ResourcePointSystem(v1.0) | 接入声望加成(领土产出+N%) |
| CampaignBattleSystem(v3.0) | 新增通关声望奖励(50~500/关) |

## 三、数据流设计

```
声望系统
  │
  ├─ 声望获取
  │   ├─ 通关新关卡: 50~500声望(按难度)
  │   ├─ 三星通关: 基础×1.5
  │   ├─ 每日声望任务: 100~300声望/日(3个专属日常)
  │   ├─ 领土产出: 5~30声望/小时
  │   ├─ NPC好感度升级: 200~800/级
  │   ├─ 成就完成: 100~2000/个
  │   ├─ 远征完成: 50~200/次
  │   ├─ 活动排名: 500~5000/次
  │   └─ 离线累积: 领土产出×50%(受离线效率系数)
  │
  ├─ 声望等级
  │   ├─ 阈值公式: 1000 × N^1.8
  │   ├─ 达到阈值 → 自动升级 → 金色横幅通知
  │   ├─ 升级动画: 进度条满格闪烁→金色粒子爆发→等级跳变
  │   └─ 接近升级(≥90%) → 进度条脉冲闪烁提醒
  │
  ├─ 声望加成
  │   ├─ Lv.2/6/12/19: 领土产出 +5%/+10%/+15%/+20%
  │   ├─ Lv.7/13: 远征收益 +10%/+15%
  │   ├─ Lv.9/16: 武将经验 +10%/+15%
  │   ├─ Lv.11/18: 建筑升级速度 +5%/+10%
  │   └─ Lv.14: 科技研究速度 +5%
  │
  └─ 声望商店
      ├─ 货币: 声望点数(与声望值独立计算)
      ├─ Lv.1: 扫荡令/经验书
      ├─ Lv.5: 招贤榜
      ├─ Lv.10: 求贤令/稀有装备图纸
      └─ Lv.20: 神话武将碎片

转生系统
  │
  ├─ 转生条件检查
  │   ├─ ✅ 主城等级 ≥ 30
  │   ├─ ✅ 通关"赤壁之战"全部
  │   ├─ ✅ 武将数量 ≥ 15
  │   ├─ ✅ 总战力 ≥ 50,000
  │   ├─ ✅ 完成"初露锋芒"成就链(5子成就)
  │   └─ ✅ 距上次转生 ≥ 72小时
  │
  ├─ 转生倍率
  │   ├─ 公式: 1.5 × (1 + 声望等级×0.05) × (1 + (次数-1)×0.1)
  │   ├─ 1次Lv.5: 1.88× | 1次Lv.10: 2.25×
  │   ├─ 3次Lv.15: 3.19× | 5次Lv.20: 4.50×
  │   └─ 10次Lv.30: 7.20× | 20次Lv.50: 12.00×
  │
  ├─ 倍率作用范围
  │   ├─ ✅ 全资源产出 / 武将经验 / 建筑速度 / 科技速度
  │   ├─ ✅ 关卡扫荡收益 / 离线收益
  │   └─ ❌ 商店价格 / 招募消耗 / PvP属性
  │
  ├─ 保留/重置
  │   ├─ ✅保留: 武将拥有权/升星/好感度/装备/建筑蓝图
  │   ├─ ✅保留: 关卡记录/三星记录/社交/成就/图鉴/VIP
  │   ├─ ⚠️衰减: 武将等级50%/技能70%/科技50%/天命30%
  │   └─ ❌重置: 主城/建筑等级/领土/铜钱/粮草/兵力
  │
  ├─ 转生后加速
  │   ├─ 赠送: 粮草×5000 + 铜钱×3000
  │   ├─ 建筑1~10级: 升级时间÷倍率(几乎瞬间)
  │   ├─ 已通关关卡: 可直接扫荡
  │   └─ 一键重建: 自动按上次优先级升级建筑
  │
  └─ 收益模拟器
      ├─ 预测未来X小时声望增长
      ├─ 计算声望等级提升带来的倍率增量
      ├─ 标注"推荐时机"(边际收益递减拐点)
      └─ 对比: 立即转生 vs 再等2h/6h/12h/24h

成就系统
  │
  ├─ 成就维度
  │   ├─ 养成成就: 武将收集/等级/升星/装备
  │   ├─ 战斗成就: 关卡通关/三星/远征/PvP
  │   ├─ 收集成就: 武将图鉴/装备图鉴/NPC好感度
  │   ├─ 社交成就: 好友/联盟/借将/聊天
  │   └─ 经济成就: 资源积累/商店消费/贸易利润
  │
  ├─ 成就奖励
  │   ├─ 声望值: 100~2000(按难度)
  │   ├─ 元宝: 少量(里程碑成就)
  │   ├─ 限定称号: 特殊成就专属
  │   └─ 成就点数: 累积解锁成就宝箱
  │
  └─ 转生成就链"初露锋芒"
      ├─ 崭露头角: 主城达到20级
      ├─ 猛将如云: 拥有10位武将
      ├─ 兵强马壮: 单武将战力达到5000
      ├─ 谋定后动: 完成5项科技研究
      └─ 威震四方: 占领20块领土
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 转生数据重置出错 | 玩家进度丢失导致严重不满 | 转生前完整备份+分步重置+回滚机制 |
| 转生门槛过高 | 玩家无法达到转生条件 | 5项条件覆盖多系统，正常游玩可自然达成 |
| 转生后体验断裂 | 重置后感觉"白玩了" | 武将/装备/成就全部保留+倍率补偿+一键重建 |
| 倍率失衡 | 高转生次数倍率过高 | 20次上限+倍率公式递减+商店价格不受影响 |
| 成就系统涉及面广 | 需要监听所有系统事件 | 使用EventBus统一监听，各系统只需抛出事件 |
| 声望获取过慢 | 玩家感觉声望等级提升无感 | 9种途径覆盖全玩法+每日声望任务保底 |
