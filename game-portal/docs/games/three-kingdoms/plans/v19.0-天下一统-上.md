## 三、数据流设计

```
设置系统
  │
  ├─ 基础设置
  │   ├─ 语言: 跟随系统→可手动切换→切换需重启(确认弹窗)
  │   ├─ 时区: 跟随设备→可手动选择→即时生效
  │   └─ 通知: 总开关(关闭屏蔽所有)+5项独立开关+免打扰时段
  │
  ├─ 音效系统
  │   ├─ 音量层级
  │   │   ├─ 主音量(0~100%) → 总控
  │   │   ├─ BGM音量(60%) → 实际=60%×主音量
  │   │   ├─ 音效音量(70%) → 实际=70%×主音量
  │   │   └─ 语音音量(80%) → 实际=80%×主音量
  │   ├─ 开关控制
  │   │   ├─ 音效总开关 → 等同主音量=0%
  │   │   ├─ BGM开关 → 停止播放
  │   │   ├─ 语音开关 → 仅保留字幕
  │   │   └─ 战斗音效开关 → 战斗场景静音
  │   └─ 特殊场景
  │       ├─ 后台运行 → BGM渐弱至静音(1s过渡)
  │       ├─ 来电/闹钟 → 立即静音，恢复后渐入(0.5s)
  │       ├─ 首次启动 → BGM延迟3秒播放
  │       └─ 低电量(<20%) → BGM音量自动降低50%
  │
  ├─ 画质系统
  │   ├─ 自动检测(首次启动)
  │   │   ├─ CPU≥8核+内存≥8GB → 高画质
  │   │   ├─ CPU≥4核+内存≥4GB → 中画质(推荐)
  │   │   └─ 其他 → 低画质
  │   ├─ 4档预设
  │   │   ├─ 低: 粒子关+阴影关+水墨关+30fps
  │   │   ├─ 中: 粒子开+阴影关+水墨开+60fps
  │   │   ├─ 高: 粒子开+阴影开+水墨开+60fps
  │   │   └─ 自动: 系统检测自动切换
  │   └─ 高级选项(独立控制5项)
  │
  ├─ 账号与存档
  │   ├─ 账号绑定
  │   │   ├─ 手机号/邮箱/第三方(微信/QQ/Apple)
  │   │   ├─ 首次绑定奖励: 元宝×50
  │   │   └─ 游客账号30天自动清除
  │   ├─ 云存档
  │   │   ├─ 自动同步: 退出时/每小时/仅手动(可选)
  │   │   ├─ WiFi仅同步: 可选开关
  │   │   ├─ 端到端加密
  │   │   └─ 冲突策略: 始终最新/始终云端/始终询问
  │   ├─ 存档管理
  │   │   ├─ 3免费+1付费(200元宝)槽位
  │   │   ├─ 自动存档: 每15分钟
  │   │   └─ 存档内容: 全量数据(2-5MB)
  │   └─ 账号删除
  │       ├─ 输入「确认删除」→ 二次确认 → 7天冷静期
  │       └─ 冷静期内可撤销
  │
  └─ 设置持久化
      ├─ 修改后立即保存localStorage
      ├─ 随云存档同步到云端
      ├─ 多设备冲突: 以最新修改时间为准
      └─ 恢复默认: 重置当前分类所有设置项

动画规范
  │
  ├─ 过渡动画(场景/面板切换)
  │   ├─ 面板展开: 300ms, ease-out, 卷轴展开感
  │   ├─ 面板关闭: 200ms, ease-in, 快速收起
  │   ├─ Tab切换: 200ms, ease-in-out, 平滑切换
  │   ├─ 页面过渡: 500ms, ease, 水墨晕染
  │   ├─ 弹窗弹出: 250ms, spring(1,0.8), 弹性感
  │   └─ 场景切换: 800ms, ease, 四角墨色遮罩→中央扩散
  │
  ├─ 状态动画(元素状态变化)
  │   ├─ 按钮悬停: 150ms, 亮度+10%, 阴影扩大
  │   ├─ 按钮按下: 80ms, scale(0.96), 暗化
  │   ├─ 按钮释放: 120ms, 弹回scale(1.0)
  │   ├─ 开关切换: 200ms, 虎符旋转180°
  │   └─ 卡片选中: 200ms, 边框加粗+高亮
  │
  └─ 反馈动画(操作结果可视化)
      ├─ 资源飘字: +N绿色上飘 / -N红色下飘, 800ms
      ├─ 升级光效: 金色粒子爆发, 1000ms
      ├─ Toast滑入: 从顶部滑入, 停留2s, 滑出
      └─ 战斗结算: 星星逐颗亮起, 每颗间隔300ms
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 云存档同步冲突 | 多设备数据丢失 | 三种冲突策略可选+始终询问为默认+同步前本地备份 |
| 音频资源体积 | 音频文件过大影响加载 | 使用压缩格式+按需加载+低画质不加载高品质音频 |
| 画质切换闪烁 | 切换画质时画面短暂异常 | 水墨晕染过渡遮罩(0.6s)+切换完成后才显示 |
| 账号删除误操作 | 玩家误删账号无法恢复 | 输入确认文字+二次确认+7天冷静期+冷静期可撤销 |
| 动画时长不统一 | 不同模块动画节奏不一致 | AnimationController统一管理+配置化时长/缓动 |
| 多语言文本遗漏 | 部分文本未翻译 | 缺失翻译回退简体中文+翻译覆盖率检测工具 |
