**PC端规格**：
- **路线地图区** (1280×496px)：C区上方主体
  - 8座城市节点分布在地图上，位置参考三国地理
  - 城市节点 (80×60px)：城市名 + 繁荣度星级 + 特产图标
  - 商路连线：已开通为实线（繁荣度高时金色粗线），未开通为灰色虚线
  - 商队动画：🐫骆驼图标沿商路移动，到达时有弹跳效果
  - 点击城市 → 弹出城市详情卡（特产/需求/已开通商路）
  - 点击商路连线 → 弹出商路详情（利润/繁荣度/商队状态）
- **贸易概况栏** (1280×80px)：
  - 左侧：核心数据（活跃商路/商队数/今日利润）
  - 右侧：快捷操作按钮组
- **商队状态栏** (1280×60px)：
  - 每支商队一行，显示：路线 + 状态 + 进度条/操作按钮
  - 状态：运输中(进度条) / 等待派遣(选择商品) / 已到达(收税按钮)


**派遣弹窗规格**：
- **尺寸**：480×350px，居中弹窗
- **路线信息**：起止城市 + 繁荣度进度条
- **商品选择区** (480×140px)：
  - 上半部分：购入商品（出发城市特产），单选
  - 下半部分：预计售出价格（目的城市需求），显示利润率
  - 利润率颜色：>20%绿色 / 0~20%黄色 / <0%红色
- **商队配置区** (480×100px)：
  - 商队选择下拉（显示空闲商队）
  - 载量：当前装载/最大载量
  - 运输时间：基于路线距离和商队等级
  - 预计利润：基于当前行情计算
- **自动往返**：勾选后商队到达目的地自动返程（不自动选商品）
- **按钮**：取消 + 确认派遣（骆驼图标按钮）

  - 城市节点缩小至40×30px，字号10px
  - 商队状态改为纵向列表（每行56px），可折叠
