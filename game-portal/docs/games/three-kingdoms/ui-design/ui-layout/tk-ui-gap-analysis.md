## 二、核心问题清单

### 问题1：缺少3个完整模块的布局设计
- **M9 NPC系统**：NPC名册、对话界面、NPC信息弹窗、好感度系统
- **M10 事件系统**：急报横幅、急报堆弹窗、历史剧情事件、随机遭遇
- **M25 贸易路线**：商路地图、商队编组、商品行情、贸易结算

### 问题2：缺少全局布局规范文档
- 弹窗尺寸规范（已在05中部分定义，需独立成文）
- 配色方案统一
- 字号规范
- 间距规范
- 组件库（按钮/输入框/进度条/Tab等）

### 问题3：部分模块缺少"整体位置"说明
用户要求每个功能都有"整个游戏窗口中具体功能的布局位置"，即：
- 当打开某功能时，游戏窗口的完整状态（资源栏+导航栏+场景区+弹窗的精确位置）
- 目前大部分模块只展示了弹窗/面板本身的布局，缺少在完整窗口中的定位

### 问题4：部分模块弹窗不完整
- 邮件详情面板
- 设置手机端
- PVP结算面板
- 好友详情/赠送/借用面板
- 公会战布局
