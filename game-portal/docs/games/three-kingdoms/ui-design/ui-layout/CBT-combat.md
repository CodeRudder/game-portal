
**战斗模式系统**:

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 全自动 | 所有行动由AI自动释放 | 日常刷图、碾压局 |
| 半自动 | 普攻自动，大招手动（含时停机制） | 中等难度 |
| 全手动 | 所有行动需玩家手动指定 | 高难度BOSS、竞技场 |
