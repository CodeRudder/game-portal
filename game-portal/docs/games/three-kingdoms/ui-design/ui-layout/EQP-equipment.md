## [EQP-4] 穿戴操作 {#eqp-4}

穿戴装备采用直接穿戴 + Toast 方式，无确认弹窗。

```
┌──────────────────────────────────┐
│  ✅ 已装备「寒铁枪」给赵云       │
│  攻击 +120  暴击 +8%             │
└──────────────────────────────────┘
  Toast 宽度 auto（最大320px），高度 48px
  背景: rgba(44,24,16,0.95) + 绿色边框 1px
  弹入: 300ms ease-out 从顶部滑入
  停留: 2秒
  消失: 300ms ease-in 向上滑出
```

**穿戴流程**:
1. 点击装备格/推荐区域 → 武将选择列表弹窗
2. 选择武将 → 直接穿戴（无二次确认）
3. 装备从背包飞向武将装备槽（300ms ease-out 弧线）
4. Toast 弹出显示装备名 + 属性变化
5. 背包中该装备显示「已:武将名」标签

**卸下流程**:
1. 点击已装备 → 操作菜单 → [卸下装备]
2. 直接卸下（无二次确认）
3. 装备从槽位飞回背包（300ms ease-out）
4. Toast「已卸下「寒铁枪」」
