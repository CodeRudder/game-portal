## 四、弹窗关闭/遮挡规范

### 4.1 弹窗分类与关闭规则

| 类型 | 关闭方式 | 遮挡范围 | 示例 |
|------|---------|---------|------|
| Toast | 3秒自动消失 | 不遮挡 | 升级成功/装备穿戴 |
| D区右侧面板 | [×]关闭 / [—]折叠 / ESC | 遮挡场景区右侧 | 武将详情/建筑详情 |
| E区居中弹窗 | [×]关闭 / 点击遮罩 / ESC | 半遮挡(遮罩50%透明) | 装备详情/确认操作 |
| F区全屏 | [←返回] / ESC | 全屏遮挡 | 战斗/布阵/对话 |
| 小操作菜单 | 点击其他区域关闭 | 局部遮挡 | 装备操作菜单 |

### 4.2 D区面板关闭/折叠组件

```
┌─ 面板标题 ────────────────── [—] [×] ┐
│                                        │
│  内容区域                              │
│                                        │
└────────────────────────────────────────┘

[×] 关闭: 右上角 20×20px，rgba(255,255,255,0.4)
         悬停: rgba(255,80,80,0.8) + 放大1.1倍
         点击: 面板滑出右侧消失

[—] 折叠: 右上角 20×20px
         点击: 面板折叠为底部小条
         折叠后: 显示 60×40px 小图标条
         再次点击: 面板重新展开
```

### 4.3 升级提示修复（不再使用悬浮框）

**原问题**：升级提示悬浮框无法关闭，遮挡地图
**修复方案**：

| 场景 | 原方案 | 新方案 |
|------|--------|--------|
| 建筑升级成功 | 悬浮框提示 | Toast"农田升级至Lv.6" + 建筑卡片闪烁 |
| 建筑升级中 | 无 | 建筑卡片上显示进度条(小型) |
| 武将升级成功 | 确认弹窗 | Toast"关羽升级至Lv.26" |
| 装备穿戴 | 确认弹窗 | Toast"已装备青龙偃月刀" |
| 资源获得 | 悬浮框 | Toast"获得木材×50" |

### 4.4 所有弹窗必须有[×]关闭按钮

**强制规则**：
1. 所有E区弹窗右上角必须有[×]关闭按钮（16×16px）
2. 所有D区面板右上角必须有[×]关闭和[—]折叠按钮
3. 所有F区全屏左上角必须有[←返回]按钮
4. Toast不需要关闭按钮（3秒自动消失）
5. 小操作菜单不需要关闭按钮（点击外部关闭）
