## 动效规范

| 动效 | 时长 | 缓动 | 触发条件 |
|------|------|------|---------|
| 弹窗弹入 | 300ms | `ease-out` | 打开活动列表/详情 |
| 弹窗关闭 | 250ms | `ease-in` | 关闭弹窗 |
| Tab切换 | 200ms | `ease-in-out` | 切换分类 |
| 里程碑达成 | 2800ms | 序列动画 | 里程碑积分达标 |
| 里程碑达成·鼎震动 | 300ms | 左右5° | 达成开始 |
| 里程碑达成·金光爆发 | 500ms | 粒子向上 | 鼎震后 |
| 里程碑达成·奖励飞出 | 500ms | 弹性缓动 | 金光后 |
| 兑换成功 | 1250ms | 序列动画 | 确认兑换 |
| 即将结束脉冲 | 2000ms循环 | `ease-in-out` | 剩余<24h |
| 活动横幅轮播 | 600ms | `ease-out` 水墨晕染 | 5s自动/手动 |
| 商店阶层解锁 | 1500ms | 序列动画 | 满足解锁条件 |
| 手机端页面滑入 | 300ms | `ease-out` | 进入详情 |

## 音效设计

| 事件 | 音效 |
|------|------|
| 打开活动面板 | 卷轴展开声 |
| 里程碑达成 | 铜钟长鸣 + 烟雾嘶嘶声 |
| 兑换成功 | 铜钱碰撞声 + 竹简展开声 |
| 活动即将结束 | 低沉鼓声（单次） |
| 商店阶层解锁 | 锁链断裂 + 卷轴展开 |
| 排名上升 | 号角短鸣 |
