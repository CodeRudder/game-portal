```
┌─ 翻倍按钮区 440×48 ─────────────────────────────────────┐
│                                                           │
│  ┌──────────────────┐  ┌──────────────────────────────┐  │
│  │  🎬 观看广告      │  │  💎 消耗 50 元宝             │  │
│  │  收益翻倍 ×2     │  │  收益翻倍 ×2                 │  │
│  └──────────────────┘  └──────────────────────────────┘  │
│                                                           │
│   200×44px                    224×44px                    │
│   左侧按钮                    右侧按钮                    │
│   间距：16px                                              │
│                                                           │
└───────────────────────────────────────────────────────────┘
```

**翻倍按钮规格**：

| 属性 | 观看广告按钮 | 消耗元宝按钮 |
|------|------------|------------|
| 尺寸 | 200×44px | 224×44px |
| 底色 | 墨绿色 `#2D5016` | 古铜金 `#C8A45C` |
| 文字 | 白色 14px | 墨色 14px |
| 图标 | 🎬 播放图标 20×20px | 💎 元宝图标 20×20px |
| 圆角 | 6px | 6px |
| Hover | 底色加亮10% | 底色加亮10% |
| 按下 | scale(0.97) | scale(0.97) |

**翻倍交互流程**：

| 方式 | 流程 | 说明 |
|------|------|------|
| 观看广告 | 点击 → 广告播放(15-30s) → 翻倍生效 → 数值重新滚动 | 每日限3次 |
| 消耗元宝 | 点击 → 二次确认 → 扣除元宝 → 翻倍生效 → 数值重新滚动 | 无限制 |

**翻倍后数值动画**：
- 当前数值先缩小至50%（200ms）
- 然后从50%位置重新滚动至200%（600ms ease-out）
- 翻倍标签 `×2` 从右侧弹入（300ms spring）

**已翻倍状态**：

```
┌──────────────────────────────────────────────────────┐
│  ✅ 已翻倍 ×2  (观看广告)                            │
│  按钮变为禁用态，灰色底+勾选图标                      │
└──────────────────────────────────────────────────────┘
```

## 全局动效规范

| 场景 | 动效 | 时长 | 缓动 |
|------|------|------|------|
| PC弹窗弹入 | 缩放弹入 | 300ms | ease-out-back |
| PC弹窗关闭 | 卷起淡出 | 400ms | ease-in |
| 手机Bottom Sheet滑入 | 从底部滑出 | 300ms | ease-out |
| 手机Bottom Sheet滑出 | 向下滑入 | 200ms | ease-in |
| 遮罩淡入 | 半透明黑纱渐显 | 200ms | ease-out |
| 遮罩淡出 | 黑纱渐隐 | 150ms | ease-in |
| 数值滚动 | 从0滚动至最终值 | 800ms | ease-out |
| 翻倍数值重滚 | 从50%滚动至200% | 600ms | ease-out |
| 翻倍标签弹入 | `×2` 从右侧弹入 | 300ms | spring(1,0.8) |
| 资源飞入粒子 | 贝塞尔曲线飞向资源栏 | 500ms | ease-in |
| 领取按钮脉动 | scale(1.0→1.02) | 2000ms | ease-in-out（循环） |
| Tooltip淡入 | 淡入+微上移 | 150ms | ease-out |
| Tooltip淡出 | 淡出 | 100ms | ease-in |
| 溢出警告闪烁 | 红色文字闪烁 | 500ms | ease-in-out（循环2次） |
