## [TUT-1-1] 高亮区域 — 目标元素周围高亮框+脉冲动画 {#tut-1-1}

```
                    ┌─ 镂空外扩 8px ─┐
                    │                 │
              ┌─────┼─────────────────┼─────┐
              │     │                 │     │
              │  ┌──┼─────────────────┼──┐  │
              │  │  │  目标元素       │  │  │
              │  │  │  (原尺寸)       │  │  │
              │  │  │                 │  │  │
              │  └──┼─────────────────┼──┘  │
              │     │                 │     │
              └─────┼─────────────────┼─────┘
                    │                 │
                    └─────────────────┘

  边框动画：古铜金流光 #C8A45C → #D4AF5E → #C8A45C (1.5s循环)
  box-shadow: 0 0 8px #C8A45C, 0 0 16px rgba(200,164,92,0.3)
  border-radius: 8px (PC) / 12px (手机)
```

**高亮区域规格**：

| 属性 | PC端 | 手机端 |
|------|------|--------|
| 镂空外扩 | 8px | 12px（手指更粗，扩大触摸区域） |
| 镂空圆角 | 8px | 12px |
| 描边颜色 | 古铜金 `#C8A45C` | 古铜金 `#C8A45C` |
| 流光周期 | 1.5s infinite | 1.5s infinite |
| box-shadow | `0 0 8px #C8A45C` → `0 0 12px #D4AF5E` | 同左 |
| 脉冲动画 | 无额外脉冲 | 无额外脉冲（流光已足够） |

**镂空实现方式**：
- 使用 CSS `clip-path` 或 Canvas `globalCompositeOperation: 'destination-out'` 在遮罩层中抠出目标区域
- 目标元素通过提升 z-index（>80）浮于遮罩之上，保持可交互
