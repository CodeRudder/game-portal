## [PVP-1-4] 竞技商店入口 {#pvp-1-4}

> 位于对手列表下方，竞技场主场景C区内，点击打开竞技商店弹窗。

### 竞技商店入口 (C区内 120×40px)

- 位于对手列表下方，居中排列
- 按钮样式：金色渐变背景 + 🏪 图标 + "竞技商店"文字
- 悬停效果：按钮微上浮 + 光晕边框

### 竞技商店弹窗 (E区 480×500px)

点击入口按钮后弹出居中弹窗，展示可用竞技币兑换的商品列表。

```
┌─ 竞技商店 ──────────── [×] ─┐
│ 💰竞技币: 2,450              │
│ ┌────┐ ┌────┐ ┌────┐       │
│ │武将│ │装备│ │资源│ Tab    │
│ └────┘ └────┘ └────┘       │
│ 商品卡片 160×180px × 6      │
│ [购买] 按钮在每个卡片底部    │
└──────────────────────────────┘
```

**竞技商店弹窗规格**：
- **尺寸**：480×500px，E区居中弹窗
- **顶部余额栏** (480×40px)：
  - 左侧：竞技币图标 + 当前余额数字（金色高亮）
  - 右侧：[+] 充值按钮（跳转充值页）
- **分类Tab** (480×36px)：
  - 三个Tab：武将碎片 / 装备 / 资源
  - 选中态：金色下划线 + 文字高亮
  - 未选中态：次文字灰色
- **商品网格** (480×380px)：
  - 每行3个商品卡片(160×180px)，共2行，纵向可滚动
  - 每个商品卡片内容：
    - 商品图标(64×64px) + 品质边框
    - 商品名称(14px) + 品质色文字
    - 兑换价格：竞技币图标 + 数量
    - 限购标签：每日限购/每周限购/不限
    - [购买] 按钮：金色渐变，余额不足时灰色禁用
  - 已售罄商品：整体opacity 0.5 + "已售罄"遮罩
- **刷新机制**：
  - 每日0点自动刷新商品列表
  - 手动刷新：消耗50竞技币，每日3次免费
- **购买流程**：
  1. 点击[购买] → 余额充足直接扣除 + Toast提示
  2. 余额不足 → 弹出提示"竞技币不足" + [前往获取]按钮
  3. 限购商品达到上限 → 按钮变为"已购买"(灰色禁用)
