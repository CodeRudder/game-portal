**布局规格**:
- 三条科技路线（军事🔴/经济🟡/文化🟣）并排纵向展开，从左到右排列
- 每条路线 7 个主节点 + 互斥分支，共约 21 节点
- 融合科技位于三条路线交汇区域
- 支持滚轮缩放（50%~150%）和拖拽平移
