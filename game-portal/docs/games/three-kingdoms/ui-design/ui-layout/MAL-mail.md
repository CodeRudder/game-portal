### [MAL-1-1] 邮件列表

**邮件分类视觉区分**:

| 分类 | 图标 | 左边框色 | 典型内容 |
|------|------|----------|---------|
| 📜 系统 | 卷轴 | `#C6A456` 古铜金 | 维护公告、版本更新 |
| ⚔️ 战斗 | 刀剑 | `#B8372D` 朱砂红 | 攻城战报、竞技场结算 |
| 🕊️ 社交 | 信鸽 | `#4A8C6F` 松石绿 | 好友赠礼、联盟邮件 |
| 🎁 奖励 | 宝箱 | `#D4A843` 明金色 | 补偿奖励、活动奖励 |

**邮件条目规格**:

```
┌──────────────────────────────────────────┐
│  🎁 [奖励] 七日登录奖励 Day 3         ●  │  ← 分类图标+标签+标题14px加粗+未读红点8px
│     2024-01-15 06:00      📎 待领取      │  ← 时间戳12px #A09080 + 附件标记
└──────────────────────────────────────────┘
  388×72px
  背景: rgba(30,25,20,0.85) + 分类色左边框3px
  悬停: 背景加深 + 左边框加宽至4px + 轻微上移1px
```

**邮件状态**:

| 状态 | 视觉 |
|------|------|
| 未读 | 标题加粗 + 右侧红色圆点● + 左边框高亮 |
| 已读未领 | 标题正常 + 附件区「待领取」金色标签 |
| 已读已领 | 整体透明度60% + 「已领取」灰色标签 |
| 已过期 | 整体透明度30% + 灰色叠加 + 「已过期」戳记 |

### [MAL-1-3] 批量操作栏

位于分类Tab下方，固定不随列表滚动。

```
┌─ 批量操作 ──────────────────────────────────┐
│  ☑ 全选  [一键领取全部] [全部已读] [删除已读] │
└──────────────────────────────────────────────┘
  高度 36px
  一键领取: 古铜金按钮，点击后所有附件飞向资源栏
  全部已读: 文字按钮，点击后所有邮件标记已读
  删除已读: 暗红文字按钮，仅删除已读已领邮件
```

**邮箱容量条**:

| 容量状态 | 颜色 | 提示 |
|---------|------|------|
| 0~70% | `#4A8C6F` 绿色 | 正常 |
| 70~90% | `#D4A843` 黄色 | 提示清理 |
| 90~100% | `#B8372D` 红色脉动 | 强烈提示清理 |
| 100% | `#B8372D` 红色闪烁 | 新邮件暂存队列 |

## [MAL-2] 新邮件提示 (A区资源栏旁 24×24px) {#mal-2}

主界面资源栏旁的新邮件提示图标。

```
┌─ A区资源栏 ──────────────────────────────────────┐
│  💰12,500  🌾8,300  ⚔️3,200  📚1,500  │  🕊️●5  │
│                                         ↑ 新邮件  │
└───────────────────────────────────────────────────┘
  🕊️ 邮件图标 24×24px
  ● 红点 16×16px + 白色数字
```

| 属性 | 规格 |
|------|------|
| 图标 | 飞鸽造型 24×24px |
| 红点 | 右上角 16×16px 红色圆点 + 白色未读数 |
| 位置 | A区资源栏最右侧 |
| 有新邮件 | 鸽子上下浮动动画（2s周期）+ 红点脉动1.5s |
| 无未读 | 图标静态，无红点 |
| 点击 | 打开 [MAL-1] 邮件面板 |

**新邮件到达动画**：
- 飞鸽从屏幕右侧飞入 → 落在邮件图标位置（800ms ease-out）
- 红点数字 +1 弹跳（300ms bounce）
- 3秒后鸽子归位静止

## 动效规范

| 动效 | 时长 | 缓动 | 触发条件 |
|------|------|------|---------|
| 面板滑入 | 300ms | `ease-out` | 打开邮件面板 |
| 面板滑出 | 250ms | `ease-in` | 关闭邮件面板 |
| 新邮件飞鸽飞入 | 800ms | `ease-out` | 收到新邮件 |
| 邮件详情展开 | 250ms | `ease-out` | 点击邮件条目 |
| 附件领取飞出 | 600ms | `cubic-bezier(0.4,0,0.2,1)` | 领取附件 |
| 一键领取全部 | 1200ms | `ease-out` 逐个 | 批量领取 |
| 邮件删除滑出 | 200ms | `ease-in` | 删除邮件 |
| 未读红点脉动 | 1500ms循环 | `ease-in-out` | 有未读邮件 |
| 过期邮件闪烁 | 5000ms循环 | `ease-in-out` | 过期当天 |
| Tab切换 | 200ms | `ease-in-out` | 切换分类 |
| 容量条变色 | 300ms | `ease-out` | 容量变化 |

## 音效设计

| 事件 | 音效 |
|------|------|
| 新邮件到达 | 飞鸽扑翅声 + 鸽哨音 |
| 打开邮件面板 | 卷轴展开声 |
| 打开邮件详情 | 竹简翻动声 |
| 领取附件 | 金币掉落声（单个）/ 连续金币声（批量） |
| 邮件删除 | 纸张撕碎声 |
| 过期提醒 | 低沉鼓声（单次） |
