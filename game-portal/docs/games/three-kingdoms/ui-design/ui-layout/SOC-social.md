**好友面板规格**：
- **尺寸**：420×696px，D区右侧面板
- **顶部Tab栏** (420×40px)：
  - 「好友列表」：显示在线数/总数
  - 「申请(N)」：红色角标显示待处理数量
  - 「搜索添加」：搜索框+结果列表
- **好友条目** (420×72px)：
  - 左侧：在线状态点(🟢在线/🔘离线) + 头像(48×48px)
  - 中部：昵称 + 等级 + 阵营标签 + 战力
  - 右侧：操作按钮组（私聊/赠送/借用）
- **底部状态栏** (420×36px)：好友数量 + 添加按钮
- **空状态**：无好友时显示水墨风「广交天下豪杰」插画

**好友申请Tab**：

## [SOC-2] 聊天面板 (D区 420×696px) {#soc-2}

右侧D区聊天面板，支持多频道切换与实时消息展示。

```
┌──────────────────────────────────┐
│  💬 聊天                         │
│  [世界] [联盟] [私聊] [系统]     │
│──────────────────────────────────│
│  ┌── 消息区域 (420×540) ──────┐ │
│  │                             │ │
│  │  [系统] 📢 全服公告:        │ │
│  │    新赛季即将开启...        │ │
│  │                             │ │
│  │  ┌─────────────────────┐   │ │
│  │  │ [头像] 赵子龙 Lv.35 │   │ │
│  │  │ 有人要组队打远征吗？ │   │ │
│  │  │          14:32      │   │ │
│  │  └─────────────────────┘   │ │
│  │                             │ │
│  │        ┌────────────────┐   │ │
│  │        │ 我来！带赵云    │   │ │
│  │        │      14:33     │   │ │
│  │        └────────────────┘   │ │
│  │                             │ │
│  │  ┌─────────────────────┐   │ │
│  │  │ [头像] 曹孟德 Lv.38 │   │ │
│  │  │ 魏国联盟收人！       │   │ │
│  │  │          14:35      │   │ │
│  │  └─────────────────────┘   │ │
│  │                             │ │
│  │  ··· 向上滚动加载更多 ···   │ │
│  │                             │ │
│  └─────────────────────────────┘ │
│──────────────────────────────────│
│  ┌── 输入区 (420×80) ────────┐ │
│  │ [😀] [输入消息...] [📎] [➤]│ │
│  └────────────────────────────┘ │
│  在线: 1,234人                   │
└──────────────────────────────────┘
```

**聊天面板规格**：
- **尺寸**：420×696px，D区右侧面板
- **频道Tab栏** (420×40px)：
  - 「世界」：全服聊天，有新消息时Tab发光
  - 「联盟」：仅联盟成员可见
  - 「私聊」：显示最近联系人列表，未读红点角标
  - 「系统」：系统公告+奖励通知
- **消息区域** (420×540px)：
  - 自己的消息靠右，蓝色气泡
  - 他人消息靠左，灰色气泡 + 头像 + 昵称 + 等级
  - 系统消息居中，金色背景条
  - 支持表情、装备/武将分享卡片
  - 向上滚动加载历史消息
- **输入区** (420×80px)：
  - 表情按钮 | 文本输入框 | 附件按钮(分享) | 发送按钮
  - 输入框最大3行，超出滚动
- **聊天限制**：世界频道发言间隔10秒，VIP缩短至3秒

**私聊子面板**：
```
┌──────────────────────────────┐
│  ← 返回    赵子龙 Lv.35  🟢 │
│──────────────────────────────│
│  (消息区域，同主聊天格式)     │
│                              │
│  ┌────────────────────┐     │
│  │ 分享了一张武将卡片: │     │
│  │ ┌──────────────┐   │     │
│  │ │ [赵云 ★★★]   │   │     │
│  │ │ Lv.35 9100战力│   │     │
│  │ └──────────────┘   │     │
│  └────────────────────┘     │
│                              │
│  [输入消息...]        [➤]   │
└──────────────────────────────┘
```

## [SOC-3] 排行榜面板 (D区 480×696px) {#soc-3}

排行榜面板，支持多维度Tab切换，展示全服排名数据。

```
┌──────────────────────────────────────┐
│  🏆 排行榜                           │
│  [战力榜] [等级榜] [声望榜] [联盟榜]  │
│──────────────────────────────────────│
│  🥇 1. 吕奉先      群  战力 45,200  │
│  ─────────────────────────────────── │
│  🥈 2. 诸葛孔明    蜀  战力 42,800  │
│  ─────────────────────────────────── │
│  🥉 3. 曹孟德      魏  战力 40,500  │
│  ─────────────────────────────────── │
│    4. 周公瑾        吴  战力 38,100  │
│  ─────────────────────────────────── │
│    5. 赵子龙        蜀  战力 35,600  │
│  ─────────────────────────────────── │
│    6. 司马仲达      魏  战力 34,200  │
│  ─────────────────────────────────── │
│    7. 孙仲谋        吴  战力 33,800  │
│  ─────────────────────────────────── │
│    8. 黄汉升        蜀  战力 31,500  │
│  ─────────────────────────────────── │
│    ··· 向下滚动 ···                  │
│  ─────────────────────────────────── │
│  ┌── 我的排名 ─────────────────────┐│
│  │  #42  我  蜀  战力 15,200  ↑3  ││
│  └─────────────────────────────────┘│
│                                      │
│  [查看奖励]          [分享排名]      │
└──────────────────────────────────────┘
```

**排行榜面板规格**：
- **尺寸**：480×696px，D区右侧面板（比好友/聊天稍宽）
- **Tab栏** (480×40px)：
  - 「战力榜」：按总战力排名
  - 「等级榜」：按主城等级排名
  - 「声望榜」：按声望值排名
  - 「联盟榜」：按联盟总战力排名
- **排名条目** (480×56px)：
  - 前三名：金银铜虎符图标 + 特殊金色/银色/铜色背景
  - 普通排名：序号 + 头像(32×32) + 昵称 + 阵营标签 + 数值
  - 点击条目 → 弹出玩家简要信息卡
- **我的排名** (480×48px)：
  - 固定在底部，金色边框高亮
  - 显示排名变化箭头（↑上升/↓下降/─不变）
- **奖励按钮**：点击显示当前段位对应奖励预览
- **刷新机制**：每小时刷新一次，下拉手动刷新

**手机端规格**：
- **通用**：全屏面板，从右侧滑入，顶部固定Tab栏
- **好友面板**：
  - 条目高度64px，操作按钮改为图标（节省空间）
  - 点击好友 → 弹出详情卡片（非新页面）
  - 长按好友 → 快捷操作菜单（私聊/赠送/删除）
- **聊天面板**：
  - 消息区域自适应，输入框固定底部（避开安全区）
  - 表情面板从底部弹出，高度200px
  - 私聊列表：显示最近5个联系人缩略
- **排行榜**：
  - 前三名条目高度64px（含头像），其余48px
  - 横向滑动切换Tab（非点击）
  - 我的排名固定底部，金色边框
- **适配说明**：
  - iPhone SE (375×667)：Tab栏使用图标+文字
  - iPhone 14 Pro (393×852)：排行榜显示更多条目
  - 横屏模式：好友/聊天变为分栏布局
