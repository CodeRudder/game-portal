## 一、系统概述

### 1.1 转生定位

声望转生是本游戏的核心放置循环引擎。玩家在积累足够资源后选择"转生"，重置大部分进度换取**永久倍率加成**，使下一轮游戏效率更高、走得更远。

### 1.2 核心循环

```
开局 → 建城 → 养将 → 推图 → 积累资源 → 达到瓶颈
  ↑                                              ↓
  └── 永久加成加速 ← 转生（重置+获得倍率）← 决定转生
```

## 三、声望倍率公式与收益计算 ★核心新增★

### 3.1 声望获取途径

| 途径 | 声望值 | 频率 | 说明 |
|------|--------|------|------|
| 通关新关卡 | 50-500/关 | 一次性 | 关卡越后声望越多 |
| 三星通关奖励 | 额外×1.5 | 一次性 | 首次三星通关 |
| 每日声望任务 | 100-300/日 | 每日 | 3个声望专属任务 |
| 占领领土产出 | 5-30/小时 | 持续 | 领土越多产出越高 |
| NPC好感度升级 | 200-800/级 | 一次性 | 好感度每升一级 |
| 成就完成 | 100-2000/个 | 一次性 | 成就越难声望越多 |
| 远征完成 | 50-200/次 | 每次远征 | 路线越长声望越多 |
| 活动排名奖励 | 500-5000 | 每次活动 | 排名越高声望越多 |
| 离线累积 | 正常产出的50% | 离线时 | 每小时结算一次 |

### 3.2 声望等级与倍率

```
声望等级 = floor(总声望 / 等级阈值)

等级阈值公式:
  等级N所需累计声望 = 1000 × N^1.8

示例:
  Lv.1: 1,000
  Lv.2: 3,475
  Lv.3: 7,136
  Lv.5: 18,946
  Lv.10: 63,096
  Lv.20: 275,423
  Lv.50: 1,000,000
```

### 3.3 转生倍率公式 ★核心参数★

```
转生倍率 = 基础倍率 × (1 + 声望等级加成) × (1 + 转生次数加成)

其中:
  基础倍率 = 1.5（固定）

  声望等级加成 = 声望等级 × 0.05
    例: 声望Lv.10 → 加成 = 10 × 0.05 = 0.5 → 总倍率 = 1.5 × 1.5 = 2.25×

  转生次数加成 = (转生次数 - 1) × 0.1
    例: 第3次转生 → 加成 = 2 × 0.1 = 0.2

完整公式:
  转生倍率 = 1.5 × (1 + 声望等级×0.05) × (1 + (转生次数-1)×0.1)
```

### 3.4 倍率作用范围

| 系统 | 是否受倍率影响 | 说明 |
|------|:-------------:|------|
| 全资源产出 | ✅ | 粮草/银两/兵符等所有资源产出 |
| 武将经验获取 | ✅ | 战斗经验和经验书效果 |
| 建筑升级速度 | ✅ | 升级时间缩短（除以倍率） |
| 科技研究速度 | ✅ | 研究时间缩短（除以倍率） |
| 关卡扫荡收益 | ✅ | 扫荡获得的资源翻倍 |
| 离线收益 | ✅ | 离线累积速度翻倍 |
| 商店价格 | ❌ | 不受倍率影响 |
| 招募消耗 | ❌ | 不受倍率影响 |
| PVP属性 | ❌ | 竞技场/切磋属性不受倍率影响 |
