# 设置与存档 UI 设计 — 水墨江山·铜纹霸业

> **风格基调**：宣纸底纹 + 青铜边框 + 虎符控件 + 墨迹动效
> **适配分辨率**：PC 1280×800 / 手机 375×667

---

## 一、入口与唤起

| 场景 | PC 端入口 | 手机端入口 |
|------|-----------|------------|
| 主界面 | 右上角青铜齿轮图标，hover 发出铜绿锈光 | 底部导航栏最右侧「设置」Tab（齿轮+竹简图标） |
| 战斗中 | 暂停菜单 → 设置按钮 | 顶部暂停按钮 → 设置按钮 |
| 登录界面 | 右下角小型齿轮，半透明 | 右下角齿轮，触控热区 ≥ 44×44pt |

- **唤起动效**：卷轴从屏幕中线向两侧徐徐展开（0.4s ease-out），背景叠加半透明墨纱遮罩（rgba(0,0,0,0.55)）
- **关闭动效**：卷轴向中心收卷，墨纱渐隐（0.3s ease-in）

## 四、画质设置

### 4.1 画质档位

| 档位 | 虎符单选样式 | 说明 |
|------|-------------|------|
| 低画质 | 灰铜虎符（未选中） | 省电模式，关闭粒子/阴影 |
| 中画质 | 青铜虎符（选中） | 默认推荐，平衡画质与性能 |
| 高画质 | 鎏金虎符（选中） | 全特效，水墨粒子满载 |
| 自动 | 铜绿虎符（选中） | 根据设备性能自动调节 |

- **设备推荐标签**：首次进入时检测设备，在推荐档位旁显示「★ 推荐」朱砂印章
- **切换动效**：全屏水墨晕染过渡（0.6s），画质即时生效

### 4.2 高级画质选项（可折叠）

| 选项 | 类型 | 默认 |

| 实时阴影 | 虎符开关 | 开启 |
| 水墨晕染 | 虎符开关 | 开启 |
| 帧率限制 | 下拉：30/60/不限 | 60 |
| 抗锯齿 | 虎符开关 | 开启 |

```
┌──────────────────────────────────────┐
│  ╔════════════════════════════════╗   │
│  ║   [青铜锁图标]                 ║   │
│  ║   数据授权申请                 ║   │
│  ╠════════════════════════════════╣   │
│  ║                               ║   │

│  ║                               ║   │

│  ║  「通知权限」                  ║   │

│  ║                               ║   │

│  ║  用于附近玩家和本地排行        ║   │
│  ║  ○ 必需  [✓ 可选]             ║   │
│  ║                               ║   │

│  ╚════════════════════════════════╝   │
└──────────────────────────────────────┘
```

**授权流程规则**：
1. 首次启动时弹出数据授权面板
2. 「必需」权限不可取消，标注朱砂色「必需」印章
3. 「可选」权限可随时在隐私设置中修改
4. 拒绝必需权限 → 弹出说明弹窗，告知影响功能
5. 可在设置中随时重新授权/撤销
6. 每次权限变更记录到隐私日志（可在隐私设置底部查看）

### 7.4 冲突解决机制

当本地与云端存档冲突时，弹出冲突解决面板：

```
┌──────────────────────────────────────────────────┐
│  ╔══════════════════════════════════════════════╗ │
│  ║   ⚠ 检测到存档冲突                          ║ │
│  ╠══════════════════════════════════════════════╣ │
│  ║                                              ║ │
│  ║  ┌─── 本地存档 ───┐  ┌─── 云端存档 ───┐    ║ │
│  ║  │ 📍 PC 端        │  │ ☁ 云端         │    ║ │
│  ║  │ 时间: 14:32     │  │ 时间: 14:28    │    ║ │
│  ║  │ 主城: Lv.23     │  │ 主城: Lv.22    │    ║ │
│  ║  │ 时长: 45h 20m   │  │ 时长: 44h 55m  │    ║ │
│  ║  │ 武将: 28        │  │ 武将: 27       │    ║ │
│  ║  │ 存档大小: 4.1MB │  │ 存档大小: 4.0MB│    ║ │
│  ║  └─────────────────┘  └─────────────────┘    ║ │
│  ║                                              ║ │
│  ║  请选择保留方案：                             ║ │
│  ║                                              ║ │
│  ║  [ 保留本地（上传覆盖云端）]                  ║ │
│  ║  [ 保留云端（下载覆盖本地）]                  ║ │
│  ║  [ 保留两者（新建本地存档槽位）]              ║ │
│  ║                                              ║ │
│  ╚══════════════════════════════════════════════╝ │
└──────────────────────────────────────────────────┘
```

**冲突解决策略**：
| 策略 | 说明 |
|------|------|
| 保留本地 | 本地存档上传覆盖云端，触发上传动效 |
| 保留云端 | 云端存档下载覆盖本地，触发下载动效 |
| 保留两者 | 云端存档下载到空闲槽位，两者都保留 |
| 自动解决 | 设置中可配置：始终保留最新 / 始终保留云端 / 始终询问 |

**冲突检测时机**：
- 启动游戏时自动检测
- 手动点击「立即下载」时
- 切换设备后首次登录时

### 7.5 同步动效

| 操作 | 动效 |
|------|------|
| 上传中 | 青铜云纹图标旋转，墨迹粒子从底部向云端飘升，进度条水墨渐变填充 |

| 冲突检测 | 警告铜锣震动效果，面板边框闪烁朱砂色 |
