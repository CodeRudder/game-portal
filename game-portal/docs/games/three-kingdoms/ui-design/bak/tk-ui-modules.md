## 模块清单

---

### M1: 主界面布局

**文档**：01-main-layout.md

- **页面**：
  - 主场景区域（根据Tab切换显示不同模块内容）
  - 顶部资源栏（卷轴横幅造型，4种资源+产出速率）
  - 导航Tab栏（PC 7个Tab / 手机 5个Tab）
  - 军师建议悬浮面板（竹简面板240×320px）
  - 小地图（PC端专属，180×140px，地图场景时显示）

- **弹窗/面板**：
  - 离线收益弹窗（离线>5分钟触发，不可跳过，必须领取）
  - 回归欢迎面板（离线>1小时，综合变化摘要）
  - 版本更新公告
  - 限时活动提醒
  - 军师建议更新

- **回归欢迎流程（弹窗优先级队列）**：
  1. 离线收益弹窗（不可跳过）
  2. 版本更新公告（可跳过）
  3. 限时活动提醒（可跳过）
  4. 回归欢迎面板（可跳过）
  5. 军师建议更新（不可跳过）

- **导航入口**：→ 所有Tab页、→ 更多菜单所有子项

- **PC/手机差异**：
  - PC端：7个主Tab + 更多下拉菜单16项 + 右下角小地图 + 军师建议面板
  - 手机端：5个主Tab + 内政/社交子菜单 + 快捷图标条 + 悬浮按钮组（离线收益+军师建议）
  - 手机端资源栏：进度条改为环形进度（28px直径，≥2mm物理尺寸）
  - 手机端无小地图，以双指缩放+单指拖拽替代

### M2: 地图系统

**文档**：02-map-system.md

- **页面**：
  - 世界大地图（20×15六边形瓦片，2400×1800px）
  - 城池内地图（建筑交互视图）
  - 筛选工具栏（阵营/地形/收益/等级/热力图）

- **弹窗**：
  - 领土详情弹窗（PC端右侧滑入半屏卷轴 / 手机端Bottom Sheet）
  - 兵力部署面板（征服时弹出）
  - 离线变化总览面板（有未查看变化时显示）
  - 筛选面板（手机端Bottom Sheet）

- **交互**：
  - 点击领土→详情弹窗、点击资源点→采集
  - 拖拽平移地图、滚轮/双指缩放（50%~200%）
  - 边缘滚动（PC端30px触发区）
  - 筛选过滤（阵营/地形/收益三维筛选）
  - 热力图模式切换

- **导航入口**：「天下」Tab

- **PC/手机差异**：
  - PC端：小地图(180×140px)、筛选栏可折叠、产出气泡常驻显示
  - 手机端：无小地图、筛选为Bottom Sheet、产出气泡长按触发、领土详情为Bottom Sheet
  - 手机端手势：单指点击/长按/拖拽、双指缩放、双指点击复位

### M3: 战斗系统

**文档**：03-combat-system.md

- **页面**：
  - 战役地图（6个历史阶段，30个关卡节点）
  - 战斗场景（全屏，我方/敌方武将区+战斗特效区）
  - 战后结算面板

- **弹窗/面板**：
  - 战斗准备面板（敌方信息+我方阵容+战力对比）
  - 智能编队推荐面板（1-3套推荐方案）
  - 扫荡弹窗（三星通关后可扫荡）
  - 战斗失败面板（战力差距提示+快速强化）
  - 离线战报面板

- **交互**：
  - 三种战斗模式：全自动/半自动/全手动
  - 大招时停机制（≥2个大招就绪时触发慢动作0.3×）
  - 一键布阵、智能推荐编队
  - 扫荡（单次/批量，最多10次）
  - 离线推图（每小时自动尝试，最多推进3关）

- **导航入口**：「出征」Tab、地图→攻击城池

- **PC/手机差异**：
  - PC端：战役横向长卷滚动、战斗日志可折叠面板、大招集中释放栏底部
  - 手机端：战役竖向时间轴、战斗日志隐藏（仅Toast）、大招栏横向滑动
  - 手机端战斗布局：上下对称（敌方上方、我方下方）、武将2×3网格(48×48px)
