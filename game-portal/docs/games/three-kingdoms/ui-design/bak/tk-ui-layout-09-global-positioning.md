## 四、弹窗/面板层级关系

```
┌─ z-index层级规范 ──────────────────┐
│                                      │
│  z-100:  资源栏(A区) + 导航栏(B区)  │
│  z-200:  场景区(C区)                │
│  z-300:  右侧面板(D区)              │
│  z-400:  全屏面板(E区)              │
│  z-500:  居中弹窗遮罩               │
│  z-600:  居中弹窗内容               │
│  z-700:  全屏覆盖(F区)              │
│  z-800:  新手引导遮罩               │
│  z-900:  急报横幅                   │
│  z-1000: Toast通知                  │
│  z-1100: 悬浮组件(G区)              │
│                                      │
│  规则:                               │
│  - 高层级完全覆盖低层级              │
│  - 同层级后打开的覆盖先打开的        │
│  - 最多同时2层弹窗叠加              │
│  - 新手引导遮罩在所有弹窗之上        │
│  - Toast始终在最顶层                 │
└──────────────────────────────────────┘
```
