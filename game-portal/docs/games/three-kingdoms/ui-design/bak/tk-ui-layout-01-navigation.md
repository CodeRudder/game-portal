## 四、回归欢迎流程

### 4.1 弹窗优先级队列（Mermaid时序图）

```mermaid
sequenceDiagram
    participant P as 玩家打开游戏
    participant S as 系统
    participant D1 as 离线收益弹窗
    participant D2 as 版本更新公告
    participant D3 as 限时活动提醒
    participant D4 as 回归欢迎面板
    participant D5 as 军师建议更新

    P->>S: 加载游戏
    S->>S: 检查离线时长
    
    alt 离线>5分钟
        S->>D1: 弹出离线收益弹窗(不可跳过)
        D1-->>P: 领取收益
    end
    
    alt 有新版本
        S->>D2: 弹出版本公告(可跳过)
        D2-->>P: 查看/跳过
    end
    
    alt 有进行中活动
        S->>D3: 弹出活动提醒(可跳过)
        D3-->>P: 查看/跳过
    end
    
    alt 离线>1小时
        S->>D4: 弹出回归欢迎面板(可跳过)
        D4-->>P: 查看/跳过
    end
    
    alt 有新建议
        S->>D5: 更新军师建议(不可跳过)
        D5-->>P: 确认
    end
    
    S->>P: 进入主界面
```

## 六、全局弹窗/面板规范

### 弹窗类型与定位

| 弹窗类型 | PC端位置 | 手机端位置 | 触发方式 |
|---------|---------|-----------|---------|
| 居中弹窗 | 屏幕正中，有遮罩 | 全屏居中 | 按钮点击 |
| 右侧滑入面板 | 右侧滑入，宽度320-480px | 全屏面板 | Tab/按钮 |
| Bottom Sheet | — | 底部推入，高度50-85% | 点击/手势 |
| 悬浮面板 | 固定位置悬浮 | 悬浮按钮→Bottom Sheet | 自动/点击 |
| Toast提示 | 顶部中央，3秒消失 | 顶部中央，2秒消失 | 系统触发 |
| 急报横幅 | 顶部降下，3秒收起 | 顶部降下，3秒收起 | 系统触发 |

### 弹窗通用规格

| 属性 | PC端 | 手机端 |
|------|------|--------|
| 遮罩 | rgba(0,0,0,0.6) | rgba(0,0,0,0.7) |
| 关闭按钮 | 右上角 × 24px | 右上角 × 32px |
| 圆角 | 8px | 12px |
| 动效 | scale 0.8→1.0, 300ms bounce | translateY 100%→0, 300ms ease-out |
| 背景 | rgba(13,17,23,0.95) + 古铜金描边 | rgba(13,17,23,0.98) + 古铜金描边 |
| 按钮高度 | 44px | 48px（≥44px触控标准） |
