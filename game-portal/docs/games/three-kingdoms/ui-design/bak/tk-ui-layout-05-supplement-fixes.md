### 5.6 Tab切换动画规范（修复P1-1）

```
  Tab切换:
  - 过渡: fade + slide 200ms ease-out
  - 新Tab从右滑入(前进)或左滑入(后退)
  - 旧Tab向相反方向滑出
  - 背景无变化（仅场景区内容切换）

  弹窗打开:
  - 居中弹窗: scale(0.9→1.0) + fade-in 150ms ease-out
  - 右侧面板: translateX(100%→0) 250ms ease-out
  - Bottom Sheet: translateY(100%→0) 250ms ease-out

  弹窗关闭:
  - 与打开动画相反，时长减半(125ms)
  - 点击遮罩关闭: 先fade-out遮罩100ms，再关闭弹窗

  面板滑入/滑出:
  - 右侧面板: translateX 250ms ease-out
  - 底部操作栏: translateY 200ms ease-out
```
