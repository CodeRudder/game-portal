# 远征系统 UI 设计（迭代版 v2.1）

> **迭代重点**：远征自动化设计（放置核心特性）、离线进度与收益、手机端完整方案  
> **v2.1 变更**：离线效率公式统一引用全局规范（§22），离线时间上限12h→72h，消除跨模块冲突
> **风格基调**：水墨江山·铜纹霸业

---

## 一、远征系统概述

### 1.1 远征在放置循环中的定位

```
放置核心循环中的远征角色：

  挂机产出 ──→ 资源积累 ──→ 消费升级 ──→ 解锁新内容
      ↑                                        │
      │         ┌──────────────────┐            │
      └─────────│ 远征系统         │←───────────┘
                │ · 消耗武将战力   │
                │ · 获取稀有资源   │
                │ · 解锁新地图区域 │
                │ · 离线自动推进   │
                │ · 扫荡重复收益   │
                └──────────────────┘
```

远征是放置游戏中"离线也能推进"的核心载体之一：
- **离线远征**：玩家下线后远征队伍继续行军和战斗
- **扫荡远征**：已通关路线可自动重复获取收益
- **远征队列**：多支队伍并行远征，最大化离线收益

### 1.2 解锁条件

- 主城5级以上解锁远征营
- 远征队列槽位随主城等级逐步解锁：
  - 主城5级：1支远征队
  - 主城10级：2支远征队
  - 主城15级：3支远征队
  - 主城20级：4支远征队（最大）

### 1.3 入口

- 城池地图「远征营」建筑（主入口）
- "更多"菜单「远征」入口（快捷入口）
- 主界面侧边栏远征状态图标（显示行军进度）

## 四、🔴 远征自动化设计（新增 — 放置核心特性）

### 4.1 扫荡系统

**解锁条件**：路线三星通关后解锁扫荡

| 扫荡类型 | 消耗 | 收益 | 次数限制 |
|---------|------|------|:-------:|
| 普通扫荡 | 扫荡令×1 | 基础奖励×100% | 每日5次/路线 |
| 高级扫荡 | 扫荡令×3 | 基础奖励×150% + 保底稀有掉落 | 每日3次/路线 |
| 免费扫荡 | 无 | 基础奖励×50% | 每日1次/路线 |

**扫荡UI**：
- PC端：路线节点旁「扫荡」按钮（虎符造型，带次数角标）
- 手机端：节点卡片内嵌「扫荡」按钮
- 扫荡结果：简化的结算面板（跳过战斗，直接显示奖励）

### 4.2 自动远征（挂机远征）

**核心机制**：设置后远征自动重复执行，无需手动操作

```
自动远征流程：
  选择路线 → 设置自动重复 → 队伍自动出发 → 
  自动战斗 → 自动结算 → 自动重新出发 → 循环

暂停条件：
  · 武将体力耗尽
  · 背包已满
  · 战斗失败（连续失败2次自动暂停）
  · 玩家手动暂停
```

**自动远征设置面板**：

| 设置项 | 选项 | 默认值 |
|-------|------|:-----:|
| 重复次数 | 1/3/5/10/无限 | 无限 |
| 失败处理 | 暂停/跳过继续 | 暂停 |
| 背包满时 | 暂停/自动出售白装 | 暂停 |
| 自动使用药品 | 开/关 | 开 |
| 体力不足时 | 暂停/使用体力道具 | 暂停 |

### 4.3 远征队列系统

多支队伍可同时远征，最大化离线收益：

| 队伍 | 状态 | 路线 | 剩余时间 | 操作 |
|:----:|:----:|------|:-------:|------|
| 队伍1 | 🔄 行军中 | 汜水关·捷径 | 12:30 | [查看][召回] |
| 队伍2 | ⏸️ 待命 | — | — | [编队][出发] |
| 队伍3 | 🔒 需主城15级 | — | — | [解锁条件] |
| 队伍4 | 🔒 需主城20级 | — | — | [解锁条件] |

**队列管理面板**：
- PC端：远征地图右上角「队列」按钮 → 右侧滑出面板
- 手机端：远征地图底部「队列」Tab → 底部Sheet展开
- 每支队伍独立显示状态、进度、操作按钮

### 4.4 一键远征

- 入口：远征地图底部「一键远征」按钮
- 功能：自动为所有空闲队伍分配最优路线并出发
- 分配逻辑：
  1. 按队伍战力从高到低排序
  2. 按路线推荐战力从高到低排序
  3. 匹配：最高战力队伍 → 最高推荐战力路线
  4. 已扫荡路线优先（稳定收益）

## 五、🔴 远征离线进度与收益（新增）

### 5.1 离线远征规则

| 离线场景 | 处理方式 | 说明 |
|---------|---------|------|
| **行军中** | 继续行军 | 离线时间正常计入行军时间 |
| **遭遇战斗** | 自动战斗 | 按队伍战力自动计算胜负（无操作损耗×0.8） |
| **路线完成** | 自动结算+重新出发 | 自动领取奖励，若有自动远征设置则重新出发 |
| **扫荡** | 离线继续扫荡 | 消耗扫荡令，自动执行（受每日上限限制） |
| **队列全部完成** | 停止等待 | 所有队伍任务完成后停止，等待玩家上线 |

### 5.2 离线收益计算

```
离线远征收益 = 正常收益 × 离线效率系数 × 0.85（战斗惩罚系数）

离线效率系数 = 引用全局离线规范（22-offline-reward-spec.md §2）
  · 0~2h: 100% | 2~8h: 80% | 8~24h: 60%
  · 24~48h: 40% | 48~72h: 25% | >72h: 15%
  · 可叠加科技/VIP/声望加成（上限+100%）
  · 最终效率上限100%

离线战斗胜率 = 正常胜率 × 0.85（离线战斗效率降低15%）
离线时间上限 = 72小时（与全局离线规范一致，超出部分不计入）

离线扫荡上限 = 每日剩余扫荡次数（不额外增加）
```

## 八、动效规范

| 动效名称 | 触发场景 | 动画描述 | 时长 | 缓动 |
|---------|---------|---------|:----:|:----:|
| 武将跃入 | 编队确认 | 武将头像弹跳进入槽位 | 300ms | bounce |
| 战旗飘扬 | 出发远征 | 战旗图标展开飘扬 | 500ms | ease-out |
| 行军脚印 | 队伍行军中 | 队伍图标沿路线移动 + 淡墨脚印 | 持续 | linear |
| 水墨涟漪 | 遭遇触发 | 路线节点水墨涟漪扩散 | 500ms | ease-out |
| 金字天降 | 大捷结算 | "大捷"金色大字从天而降 | 600ms | ease-out |
| 烟花绽放 | 大捷结算 | 金色粒子烟花爆发 | 1000ms | ease-out |
| 进度流动 | 行军进度条 | 金色虚线沿路线流动 | 持续 | linear |
| 扫荡闪过 | 扫荡结算 | 路线快速闪过 + 奖励弹出 | 400ms | ease-in-out |
| 离线脉动 | 回归摘要中远征进行 | 队伍图标金色脉动 | 1500ms循环 | ease-in-out |

### 性能降级策略

- 低端设备：关闭烟花粒子、关闭行军脚印、关闭水墨涟漪
- 中端设备：简化烟花（5个粒子→3个）、保留基础动效
- 高端设备：全部动效开启
