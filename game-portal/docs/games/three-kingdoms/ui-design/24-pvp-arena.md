# 对抗系统 UI 设计（v1.0）

> **风格基调**：水墨江山·铜纹霸业
> **分辨率**：PC 端 1280×800 + 手机端 375×667

---

## 一、入口设计

### 主入口
- 城池地图「演武场」建筑（PC端点击/手机端点击）
- "更多"菜单「竞技」入口
- 主界面右侧快捷入口：演武场图标（剑盾交叉），有挑战次数时脉动发光

### 解锁条件
- 主城等级 ≥ 10
- 拥有 ≥ 3 名武将
- 完成新手引导第5步

## 六、战斗记录

### 6.1 战斗记录列表
- 每条记录：胜/负标识 + 对手名 + 排名变化 + 时间
- 点击展开战斗回放
- 最多保留 50 条记录

### 6.2 战斗回放
- 复用战斗引擎回放功能
- 可调速：0.5x / 1x / 2x / 4x
- 关键时刻标注（暴击/大招/击杀）
- 支持分享回放链接给好友

## 七、防守编队

### 7.1 防守阵容编辑
- 独立于进攻阵容的防守编队
- 5个阵位 + 阵型选择（鱼鳞/锋矢/雁行/长蛇/方圆）
- AI 防守策略选择：
  - 「均衡」：攻防兼顾
  - 「猛攻」：优先攻击
  - 「坚守」：优先防御
  - 「智谋」：优先释放技能

### 7.2 防守日志
- 被挑战记录列表
- 防守胜率统计
- 被攻破时推荐调整阵容

## 九、多人对战（联盟战）

### 9.1 入口
- 公会界面→「公会战」按钮
- 每周固定时间开放（周六20:00-22:00）

### 9.2 公会战地图
- 三国势力地图，3个阵营据点
- 公会选择加入阵营（蜀/魏/吴）
- 据点争夺：攻击/防守
- 据点产出：占领后全公会获得资源加成

### 9.3 公会战战斗
- 3v3 或 5v5 对战
- 匹配规则：按公会总战力匹配
- 战斗模式：半自动（可指挥集火目标）
- 战场技能：公会会长释放「号令」增益

### 9.4 公会战结算
- 占领据点数排名
- 个人贡献排名（击杀/助攻/占领）
- 公会战专属奖励

| 挑战失败 | 「惜败」墨字淡出+鼓声 | 800ms |
| 公会战开始 | 战旗飘扬+号角声 | 1000ms |

## 十一、手机端专项适配

### 11.1 对手选择
- 横向滑动切换3个对手（一次显示1个）
- 左右箭头导航
- 下拉刷新对手

### 11.2 战斗界面
- 全屏战斗（与 PvE 共用）
- 底部大招按钮栏（3个按钮一行）
- 左上角暂停/加速按钮

### 11.3 排行榜
- 全屏列表
- 自己排名固定在底部
- 下拉加载更多

### 11.4 触控规范
- 所有可点击区域 ≥ 44×44px
- 挑战按钮 ≥ 48×48px
- 防守编队武将卡片支持长按拖拽
- 左滑删除战斗记录
