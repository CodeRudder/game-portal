## [SOC-1] 好友系统 {#soc-1}

### 功能描述

放置游戏玩家在线时间不固定，好友系统以异步互动为核心设计理念。支持添加/删除好友、借将出征、资源援助、切磋回放等互动方式，让玩家感受到其他玩家的存在但不强制同步。

### 好友上限

| 玩家等级 | 好友上限 |
|:--------:|:-------:|
| 1~10 | 20 |
| 11~20 | 30 |
| 21~30 | 40 |
| 31~40 | 50 |
| 41+ | 60 |

### 添加好友规则

| 规则 | 说明 |
|------|------|
| 搜索添加 | 输入玩家昵称或ID精确搜索 |
| 推荐好友 | 系统推荐相近战力/等级的玩家 |
| 申请限制 | 每日最多发送 20 次好友申请 |
| 待处理上限 | 最多同时存在 30 条待处理申请 |
| 申请有效期 | 7天未处理自动过期 |

### 好友互动类型与奖励

| 互动类型 | 每日次数 | 获得奖励 |
|---------|:-------:|---------|
| 赠送兵力 | 10次/天 | 友情点 ×5/次 |
| 拜访主城 | 5次/天 | 铜钱 ×100/次 + 随机小概率获得资源 |
| 切磋 | 3次/天 | 胜利友情点 ×20，失败友情点 ×5 |
| 借将 | 3次/天（不同好友） | 借用方获得100%经验，出借方获得友情点 ×10 |

### 借将系统

| 规则 | 说明 |
|------|------|
| 借用限制 | 每天可借用 3 次（不同好友/公会成员） |
| 武将互斥 | 同一武将每天只能被借用 1 次 |
| 战力折算 | 借用武将战力按 80% 计算 |
| 禁用场景 | 借将不可用于 PVP |
| 归还时机 | 一次出征（单关卡）结束后自动归还 |

### 友情点系统

| 参数 | 数值 | 说明 |
|------|------|------|
| 单次出借友情点 | 10点 | 每次武将被借用固定获得 |
| 每日友情点获取上限 | 200点 | 出借方每日通过借将获得的友情点上限 |
| 每日借出次数上限 | 20次 | 友情点满额后仍可借出，但不获得友情点 |
| 重置时间 | 每日 0:00 | 进度条重置 |

### 友情商店兑换

| 商品 | 价格（友情点） | 限购 |
|------|:----------:|:----:|
| 招募令 ×1 | 100 | 每周 3 次 |
| 绿色装备箱 ×1 | 50 | 每日 2 次 |
| 兵力道具 ×1 | 30 | 每日 3 次 |
| 铜钱 ×500 | 20 | 每日 5 次 |

### 好友删除规则

| 规则 | 说明 |
|------|------|
| 单向删除 | 删除后双方好友关系解除 |
| 冷却期 | 删除好友后 24 小时内不可再次添加对方 |
| 借将中保护 | 武将正被借用时不可删除出借方好友 |

## [SOC-2] 聊天系统 {#soc-2}

### 功能描述

聊天系统采用异步留言板模式，支持多频道通信。非实时消息机制适配放置游戏玩家碎片化在线的特点。

### 频道类型

| 频道 | 可见范围 | 消息保留 | 发言间隔 | 说明 |
|------|---------|---------|:-------:|------|
| 世界频道 | 全服玩家 | 最近100条 / 48小时 | 10秒 | 公开聊天，所有人可见 |
| 公会频道 | 公会成员 | 最近100条 / 48小时 | 5秒 | 留言板模式，支持置顶公告 |
| 私聊 | 一对一 | 最近50条 / 7天 | 3秒 | 双方均可见 |
| 系统 | 全服 | 30天 | — | 版本更新、维护通知（仅官方） |

### 消息规则

| 规则 | 说明 |
|------|------|
| 单条消息长度 | 最长 200 字符 |
| 预设快捷消息 | 「求援」「借将」「打BOSS」「日常问候」 |
| 特殊消息卡片 | 资源请求 / 借将请求以特殊卡片样式展示 |
| 公会公告 | 盟主/军师可置顶公告，最多 3 条 |

### 禁言机制

| 违规等级 | 处罚 | 时长 | 说明 |
|---------|------|------|------|
| 一级（轻度） | 禁言 | 1小时 | 发送广告、刷屏 |
| 二级（中度） | 禁言 | 24小时 | 辱骂、恶意言论 |
| 三级（重度） | 禁言 + 警告 | 7天 | 严重违规、多次累犯 |
| 永久封禁 | 永久禁言 | 永久 | 极端情况 |

### 举报系统

| 规则 | 说明 |
|------|------|
| 每日举报上限 | 3次/天 |
| 举报类型 | 作弊/外挂、数据异常、恶意利用BUG、其他 |
| 自动附带数据 | 回放ID、双方玩家ID、战斗时间戳、回合数据快照 |
| 恶意举报处罚 | 3次恶意举报后，7天内不可使用举报功能 |
| 处理反馈 | 处理结果通过邮件通知举报者 |

## [SOC-3] 排行榜 {#soc-3}

### 功能描述

排行榜提供多维度排名展示，激励玩家追求更高目标。排名数据定期刷新，赛季结束时根据最高排名发放奖励。

### 排名维度

| 排行榜类型 | 排序依据 | 刷新频率 | 展示条目 |
|-----------|---------|---------|---------|
| 战力榜 | 总战力降序 | 实时 | 前100名 + 自己 |
| 财富榜 | 铜钱持有量降序 | 每日0:00 | 前100名 + 自己 |
| 远征榜 | 远征通关数降序 | 实时 | 前100名 + 自己 |
| 公会榜 | 公会总战力降序 | 每日0:00 | 前50公会 + 自己公会 |
| 竞技榜 | PVP积分降序 | 实时 | 前100名 + 自己 |
| 赛季战绩榜 | 本赛季胜场数降序 | 实时 | 前100名 + 自己 |

### 排行榜条目信息

| 字段 | 说明 |
|------|------|
| 排名 | 1~100，前三名金银铜虎符标识 |
| 玩家昵称 | 显示名称 |
| 等级 | 当前主城等级 |
| 阵营 | 魏/蜀/吴 |
| 战力/积分 | 对应排行维度的核心数值 |
| 胜率 | 竞技榜额外显示 |
| 自己排名 | 底部固定高亮独立条目 |

### 排行榜奖励

| 排名 | 每日奖励（战力榜/财富榜） | 每日奖励（竞技榜） |
|:----:|------------------------|------------------|
| 第1名 | 元宝 ×50 + 铜钱 ×5,000 | 竞技币 ×200 + 元宝 ×30 |
| 第2~3名 | 元宝 ×30 + 铜钱 ×3,000 | 竞技币 ×150 + 元宝 ×20 |
| 第4~10名 | 元宝 ×15 + 铜钱 ×2,000 | 竞技币 ×100 + 元宝 ×10 |
| 第11~50名 | 元宝 ×5 + 铜钱 ×1,000 | 竞技币 ×50 + 元宝 ×5 |
| 第51~100名 | 铜钱 ×500 | 竞技币 ×20 |

### 邮件系统

| 邮件类型 | 图标 | 保留时间 | 说明 |
|---------|------|:-------:|------|
| 系统邮件 | 📜 | 30天 | 版本更新、维护通知 |
| 奖励邮件 | 🎁 | 7天 | 活动奖励、离线收益、排行奖励 |
| 社交邮件 | ✉️ | 7天 | 借将通知、援助通知、切磋结果 |
| 公会邮件 | 🏛️ | 7天 | 公会公告、BOSS击败通知 |

| 操作 | 说明 |
|------|------|
| 批量领取 | 一键领取所有奖励邮件附件 |
| 一键已读 | 标记所有邮件为已读 |
| 未读标识 | 未读邮件红点 + 加粗显示 |
