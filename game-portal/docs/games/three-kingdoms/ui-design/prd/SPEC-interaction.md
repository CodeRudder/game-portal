## [ITR-1] 通用交互 {#itr-1}

### 操作手势定义

#### 桌面端

| 操作 | 触发方式 | 应用场景 | 响应时间 |
|------|---------|---------|:--------:|
| 点击 | 鼠标左键按下+释放 | 选择、确认、打开面板 | ≤100ms |
| 右键 | 鼠标右键 | 快捷菜单（NPC交互、任务操作） | ≤100ms |
| 悬停 | 鼠标停留>300ms | Tooltip预览、效果预览 | ≤300ms |
| 拖拽 | 按住左键>150ms且移动>5px | 武将编队、装备穿戴、建筑放置 | ≤16ms |
| 滚轮 | 鼠标滚轮滚动 | 地图缩放、列表滚动 | ≤16ms |
| 长按 | 按住左键>500ms | 快速连续升级、批量选择 | ≤500ms |
| Shift+点击 | 组合键 | 批量选择武将/装备 | ≤100ms |

#### 移动端

| 操作 | 触发方式 | 应用场景 | 响应时间 |
|------|---------|---------|:--------:|
| 点击 | 单指触摸+释放 | 选择、确认 | ≤100ms |
| 长按 | 按住>500ms | 快捷菜单、连续升级 | ≤500ms |
| 拖拽 | 按住>150ms且滑动>10px | 武将编队、装备穿戴 | ≤16ms |
| 双指缩放 | 双指张合 | 地图缩放 | ≤16ms |
| 左滑 | 向左滑动>80px | 删除邮件、快速操作 | ≤300ms |
| 下拉 | 向下拉动>60px | 刷新列表 | ≤300ms |
| 双击 | 快速双击（间隔<300ms） | 快速升级建筑 | ≤300ms |

|------|------|--------|

| 类别 | 触发时机 | 示例 |

## [ITR-4] 弹窗交互 {#itr-4}

### 弹窗类型

| 类型 | 用途 | 关闭方式 |
|------|------|---------|
| 信息弹窗 | 仅展示信息 | 点击确认/点击遮罩/ESC |
| 确认弹窗 | 需要用户确认操作 | 点击确认/取消/ESC=取消 |
| 输入弹窗 | 需要用户输入内容 | 点击确认/取消/ESC=取消 |
| 危险操作弹窗 | 不可逆操作确认 | 仅点击确认（3秒倒计时）/取消 |

### 弹窗通用规则

| 属性 | 规则 |
|------|------|
| 居中显示 | PC 420px宽 / 手机 min(340px, 90vw)宽 |
| 遮罩 | 60%黑色半透明 |
| 弹出动画 | 250ms spring(1, 0.8) |
| 关闭动画 | 200ms ease-in |
| ESC行为 | 默认等同于「取消」 |
| 焦点锁定 | 弹窗打开时Tab键仅在弹窗内循环 |
| 多弹窗 | 禁止同时显示多个弹窗，新弹窗排队等待 |

### 确认弹窗按钮规则

| 按钮位置 | 样式 | 键盘快捷键 |
|---------|------|-----------|
| 左侧「取消」 | 灰色边框按钮 | ESC / N |
| 右侧「确认」 | 古铜金实心按钮 | Enter / Y |

### 危险操作弹窗

| 属性 | 规则 |
|------|------|
| 确认按钮倒计时 | 3秒倒计时才能点击：「请确认(3)...」→「确认」 |
| 确认按钮颜色 | 红色 #C0392B |
| 警告文字 | 红色标注「此操作不可撤销！」 |
| 适用场景 | 转生/重置/删除存档/退出联盟/删除好友 |

## [ITR-5] 确认弹窗策略 {#itr-5}

### 核心原则

确认弹窗仅用于**不可逆操作**和**高代价操作**，约11%的操作场景使用确认弹窗。日常操作不应被打断。

### 需要确认弹窗的场景（白名单）

| 场景 | 类型 | 原因 |
|------|------|------|
| 转生确认 | 危险操作 | 大量数据重置，不可逆 |
| 删除存档 | 危险操作 | 数据永久丢失 |
| 删除账号 | 危险操作 | 账号永久删除 |
| 退出联盟 | 危险操作 | 24小时内不可重新加入 |
| 删除好友 | 危险操作 | 需重新申请 |
| 消耗元宝（>100） | 高代价操作 | 付费货币消耗 |
| 消耗天命（>200） | 高代价操作 | 高级货币消耗 |
| 科技分支选择 | 不可逆选择 | 互斥分支不可更改 |
| 切换语言 | 需重启 | 需要重启游戏生效 |
| 跳过剧情 | 体验确认 | 提醒不可逆体验损失 |

### 不需要确认弹窗的场景（黑名单）

| 场景 | 替代方案 | 原因 |
|------|---------|------|
| 建筑升级 | 直接执行，支持降级 | 可逆操作 |
| 武将升级 | 直接执行 | 可逆操作 |
| 装备穿戴 | 直接执行，旧装备自动卸下 | 可逆操作 |
| 出征战斗 | 直接进入 | 核心玩法不应打断 |
| 资源收取 | 直接执行 | 日常高频操作 |
| NPC对话选择 | 直接执行 | 剧情体验不应打断 |
| 招募武将 | 直接执行，无撤销需求 | 消耗低 |
| 购买商店物品（铜钱） | 直接执行 | 基础货币，可重新获取 |

### Toast替代确认弹窗的规则

| 操作类型 | 反馈方式 | 示例 |
|---------|---------|------|
| 可逆操作 | 操作后Toast确认 | 「建筑已升级至Lv.5」 |
| 资源消耗 | 资源变化飘字 | -500 💰 飘字动画 |
| 状态变化 | 状态图标变化+Toast | 「装备已穿戴」 |
| 批量操作 | 进度Toast | 「已出售10件装备，获得💰2,500」 |

### 撤销机制（替代确认弹窗）

| 操作 | 撤销方式 | 撤销时限 |
|------|---------|---------|
| 出售装备 | Toast右侧「撤销」按钮 | 5秒内 |
| 分解装备 | Toast右侧「撤销」按钮 | 5秒内 |
| 武将下阵 | 直接重新上阵 | 无时限 |
| 装备卸下 | 直接重新穿戴 | 无时限 |
