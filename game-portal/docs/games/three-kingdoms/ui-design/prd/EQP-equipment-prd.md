## [EQP-1] 装备类型 {#eqp-1}

### 功能描述

装备系统为武将提供属性加成，是战力提升的核心养成线之一。装备分为四大类型，对应武将的四个装备槽位，每件装备拥有基础属性、附加属性和潜在的套装效果。

> 🎨 → [UI: 装备背包](../ui-layout/EQP-equipment.md#eqp-1)

### 装备部位定义 {#eqp-1-1}

| 装备部位 | 图标 | 主属性 | 副属性池 | 特殊词条池 | 说明 |
|---------|------|--------|---------|-----------|------|
| 武器 | ⚔️ | 攻击力 | 暴击率、暴击伤害、命中率 | 吸血、破甲、连击 | 提升输出能力 |
| 防具 | 🛡️ | 防御力 | 闪避率、抗暴率、生命值 | 反伤、格挡、韧性 | 提升生存能力 |
| 饰品 | 💍 | 智力/统帅 | 技能伤害、冷却缩减、怒气恢复 | 技能连发、范围增大、控制增强 | 提升技能效果 |
| 坐骑 | 🐴 | 速度/生命 | 移动力、先攻值、士气 | 追击、撤退、冲锋加成 | 提升机动性 |

### 装备来源 {#eqp-1-2}

| 来源 | 品质上限 | 说明 |
|------|---------|------|
| 普通关卡掉落 | 蓝色 | 基础装备获取途径 |
| 精英关卡掉落 | 紫色 | 中高级装备获取 |
| Boss 关卡掉落 | 金色 | 高级装备主要来源 |
| 装备炼制 | 金色 | 同品质装备合成升品 |
| 商店购买 | 紫色 | 集市/军需处/活动商店 |
| 活动奖励 | 金色 | 限时活动/成就奖励 |
| 装备箱开启 | 金色 | 各品质装备箱随机产出 |

## [EQP-2] 装备品质 {#eqp-2}

### 功能描述

装备品质决定装备的属性上限、副属性条数和特殊效果。品质从低到高分为五级，对应不同的强化上限和视觉表现。

> 🎨 → [UI: 品质展示](../ui-layout/EQP-equipment.md#eqp-2)

### 品质等级定义 {#eqp-2-1}

| 品质 | 中文名 | 色值 | 星级上限 | 副属性条数 | 特殊词条 |
|------|--------|------|:--------:|:---------:|---------|
| 白色 | 凡品 | `#B0B0B0` | 最高 +5 | 0~1 条 | 无 |
| 绿色 | 良品 | `#5CB85C` | 最高 +8 | 1 条 | 无 |
| 蓝色 | 上品 | `#4A90D9` | 最高 +10 | 1~2 条 | 5% 概率附带 1 个特殊词条 |
| 紫色 | 精品 | `#9B59B6` | 最高 +12 | 2 条 | 20% 概率附带 1 个特殊词条 |
| 金色 | 传说 | `#D4A843` | 最高 +15 | 2~3 条 | 100% 附带 1 个特殊词条 + 套装归属 |

| 白色 | 1.0x | 0.5x |
| 绿色 | 1.3x | 0.8x |
| 蓝色 | 1.6x | 1.0x |
| 紫色 | 2.0x | 1.2x |
| 金色 | 2.5x | 1.5x |

## [EQP-3] 装备属性 {#eqp-3}

### 功能描述

每件装备由基础属性、附加属性和特殊词条组成。高品质装备拥有更多属性条数和更强的属性倍率。集齐同套装备可激活套装效果。

> 🎨 → [UI: 装备详情](../ui-layout/EQP-equipment.md#eqp-3)

### 属性构成

| 属性类型 | 说明 | 数值规则 |
|---------|------|---------|
| 基础属性（主属性） | 由装备部位决定（武器=攻击、防具=防御等） | 受品质倍率 × 强化等级加成 |
| 附加属性（副属性） | 从对应部位副属性池中随机抽取 | 条数由品质决定，数值受品质倍率影响 |
| 特殊词条 | 从对应部位特殊词条池中随机抽取 | 品质越高出现概率越大，金色 100% |

### 属性数值示例（武器 · 1 级未强化）

| 品质 | 攻击力（主属性） | 暴击率（副属性） | 特殊词条 |
|------|:---------------:|:---------------:|---------|
| 白色 | 48 | — | 无 |
| 绿色 | 62 | 4% | 无 |
| 蓝色 | 77 | 5% ~ 6% | 5% 概率：破甲 |
| 紫色 | 96 | 6% ~ 8% | 20% 概率：吸血/破甲/连击 |
| 金色 | 120 | 8% ~ 12% | 100%：吸血/破甲/连击（随机 1 个） |

### 套装效果

| 套装名称 | 品质要求 | 2 件套效果 | 4 件套效果 | 风格主题 |
|---------|:-------:|----------|----------|---------|
| 青铜套装 | 绿色+ | 攻击 +5% | 普攻 10% 概率触发额外一次攻击 | 远古青铜 |
| 寒铁套装 | 蓝色+ | 防御 +8% | 受击时 15% 概率减免 30% 伤害 | 北方寒铁 |
| 玄铁套装 | 蓝色+ | 暴击率 +10% | 暴击时额外造成 20% 真实伤害 | 精炼玄铁 |
| 赤焰套装 | 紫色+ | 技能伤害 +12% | 释放技能后 20% 概率重置冷却 | 火焰锻造 |
| 龙鳞套装 | 紫色+ | 全属性 +8% | 生命值低于 30% 时攻击力提升 25% | 龙之鳞甲 |
| 霸王套装 | 金色 | 攻击 +15% + 防御 +10% | 每回合获得「霸王」：攻击+5%，最多叠 5 层 | 西楚霸王 |
| 天命套装 | 金色 | 全属性 +12% | 战斗开始全体友军获得「天命护盾」（吸收 20% 最大生命伤害） | 真龙天命 |

### 套装规则

| 规则 | 说明 |
|------|------|
| 套装归属 | 仅金色品质装备固定归属套装；紫色+装备有概率属于某套装 |
| 套装部位 | 每套包含武器/防具/饰品/坐骑 4 个部位 |
| 套装激活 | 同一武将穿戴同套 2 件激活 2 件套效果，4 件激活 4 件套效果 |
| 套装不叠加 | 同一套装效果不可叠加，多件同套装仅计算一次 |
| 跨武将不共享 | 套装效果仅在穿戴同一武将时生效 |

### 强化保护符

| 道具 | 适用等级 | 效果 | 获取途径 |
|------|---------|------|---------|

### 强化属性成长

### 自动强化规则

| 功能 | 说明 |
|------|------|
| 自动强化至目标等级 | 设置目标等级，系统自动连续强化直至达到或材料耗尽 |
| 自动强化停止条件 | 材料不足 / 铜钱不足 / 达到目标等级 / 玩家手动停止 |
| 保护符策略 | 可设置「自动使用保护符」开关，开启时每次强化自动消耗保护符 |

## [EQP-5] 穿戴规则 {#eqp-5}

### 功能描述

武将通过穿戴装备获得属性加成。每个武将拥有 4 个装备槽位，支持一键穿戴推荐装备和快速更换。

> 🎨 → [UI: 武将装备槽位](../ui-layout/EQP-equipment.md#eqp-5)

### 装备槽位

| 槽位 | 对应类型 | 说明 |
|------|---------|------|
| 槽位 1 | ⚔️ 武器 | 攻击类装备 |
| 槽位 2 | 🛡️ 防具 | 防御类装备 |
| 槽位 3 | 💍 饰品 | 辅助类装备 |
| 槽位 4 | 🐴 坐骑 | 机动类装备 |

### 穿戴规则

| 规则 | 说明 |
|------|------|
| 部位匹配 | 武器只能穿戴到武器槽位，不可跨部位穿戴 |
| 唯一性 | 同一件装备同一时间只能被一个武将穿戴 |
| 换装 | 穿戴新装备时自动卸下同槽位旧装备，旧装备返回背包 |
| 等级限制 | 装备无等级限制，获得即可穿戴 |
| 武将限制 | 装备无武将限制，任何武将可穿戴任何装备 |

### 一键穿戴

| 功能 | 说明 |
|------|------|
| 一键穿戴推荐 | 军师推荐系统自动为当前武将挑选最优装备并穿戴 |
| 推荐逻辑 | 基于武将四维属性短板自动推荐补强装备，按品质优先 > 属性增量优先 > 等级匹配优先排序 |
| 推荐展示 | 每个装备槽下方展示最多 2 件推荐装备缩略图 |
| 确认机制 | 一键穿戴弹窗展示更换前后属性对比，确认后执行 |

### 装备卸下

| 操作 | 说明 |
|------|------|
| 单件卸下 | 点击已穿戴装备 → 选择「卸下」→ 装备返回背包 |
| 全部卸下 | 武将详情页「卸下全部」按钮 → 所有装备返回背包 |
| 背包满处理 | 背包满时不可卸下，提示「背包已满，请先清理」 |

### 装备分解

| 规则 | 说明 |
|------|------|
| 可分解装备 | 未穿戴的装备 |
| 不可分解 | 正在穿戴的装备（需先卸下） |
| 分解产出 | 铜钱（装备品质 × 等级对应数量）+ 强化石（概率产出） |
| 品质越高产出越多 | 白色：铜钱 ×100；绿色：铜钱 ×300 + 强化石 ×1；蓝色：铜钱 ×800 + 强化石 ×2；紫色：铜钱 ×2000 + 强化石 ×3；金色：铜钱 ×5000 + 强化石 ×5 |
| 批量分解 | 支持多选后批量分解，选中品质可设置最低品质过滤（如：仅分解白色和绿色） |

### 背包管理

| 规则 | 说明 |
|------|------|
| 默认容量 | 50 格 |
| 扩容方式 | 消耗元宝扩容，每次 +10 格，最多扩至 100 格 |
| 排序方式 | 按品质（金→紫→蓝→绿→白）、按部位、按套装分组 |
| 筛选功能 | 全部 / 武器 / 防具 / 饰品 / 坐骑 / 只看推荐 / 只看未穿戴 / 只看套装 |
