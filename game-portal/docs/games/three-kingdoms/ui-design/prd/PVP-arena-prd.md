## [PVP-1] 匹配规则 {#pvp-1}

### 功能描述

PVP竞技场为玩家提供异步对战平台。系统根据玩家积分和战力自动匹配对手，玩家从3名候选对手中选择挑战目标。主城10级以上解锁。

### 解锁条件

| 条件 | 数值 |
|------|------|
| 主城等级 | ≥ 10 |
| 武将数量 | ≥ 3 名 |
| 新手引导 | 完成第5步 |
| 入口 | 城池地图「演武场」建筑 / "更多"菜单「竞技」 |

### 对手选择规则

| 参数 | 说明 |
|------|------|
| 候选对手数 | 同时展示 3 名对手 |
| 战力范围 | 自身战力 ×0.7 ~ ×1.3 |
| 排名范围 | 自身排名 ±5 ~ ±20（根据当前段位调整） |
| 阵营分布 | 尽量覆盖不同阵营 |

### 刷新机制

| 参数 | 说明 |
|------|------|
| 免费刷新 | 每 30 分钟自动刷新1次候选对手 |
| 手动刷新 | 消耗铜钱 ×500 立即刷新 |
| 刷新上限 | 每日手动刷新 10 次 |
| 刷新后冷却 | 手动刷新后 5 秒内不可再次刷新 |

### 挑战次数

| 参数 | 数值 |
|------|------|
| 每日挑战次数 | 5 次 |
| 购买额外次数 | 元宝 ×50/次，每日最多购买 5 次 |
| 重置时间 | 每日 0:00 |

## [PVP-2] 战斗规则 {#pvp-2}

### 功能描述

PVP战斗支持全自动和半自动两种模式。进攻方和防守方使用不同阵容，战斗结果影响双方积分和排名。

### 战斗模式

| 模式 | 说明 |
|------|------|
| 全自动 | AI控制所有行动，玩家仅观战 |
| 半自动 | 武将自动普攻，大招手动释放（时停机制） |
| 全手动 | 不可选（PVP不开放全手动，防止操作延迟影响异步对战公平性） |

### 战斗参数

| 参数 | 说明 |
|------|------|
| 回合上限 | 10回合 |
| 超时判定 | 10回合未分胜负 → 防守方获胜 |
| 战斗加速 | 1x / 2x / 3x |
| 阵营特效 | 蜀红 / 魏蓝 / 吴绿 |

### 攻防区别

| 维度 | 进攻方 | 防守方 |
|------|-------|-------|
| 阵容 | 进攻编队（玩家手动选择） | 防守编队（玩家预设） |
| 控制 | 可选半自动/全自动 | AI自动控制 |
| 战力加成 | 无 | 防守方城墙加成：全属性 +5% |
| 超时判定 | 失败 | 获胜 |

### 战力分析维度

| 维度 | 说明 |
|------|------|
| 攻击力 | 物理输出能力 |
| 防御力 | 物理防御能力 |
| 速度 | 行动顺序和频率 |
| 谋略 | 技能伤害和效果 |

### 战斗结果

| 结果 | 条件 | 积分变化 |
|------|------|---------|
| 进攻胜利 | 防守方全灭 | 进攻方 +30~60（根据战力差） |
| 进攻失败 | 进攻方全灭或超时 | 进攻方 -15~30 |
| 防守胜利 | 进攻方全灭或超时 | 防守方 +20~40 |
| 防守失败 | 防守方全灭 | 防守方 -10~25 |

### 战斗回放

| 参数 | 说明 |
|------|------|
| 保存数量 | 最多 50 条记录 |
| 保留时长 | 7天 |
| 播放速度 | 0.5x / 1x / 2x / 4x |
| 关键时刻标注 | 暴击 / 大招 / 击杀 |
| 分享 | 支持分享回放链接给好友 |

## [PVP-4] 防守阵容 {#pvp-4}

### 功能描述

防守阵容独立于进攻阵容，由玩家预设后由AI自动控制进行防守。合理设置防守阵容是保住排名的关键。

### 阵容设置规则

| 规则 | 说明 |
|------|------|
| 阵位数 | 5个阵位 |
| 阵型选择 | 鱼鳞 / 锋矢 / 雁行 / 长蛇 / 方圆 |
| 独立编队 | 防守阵容与进攻阵容独立，可使用相同武将 |
| 修改限制 | 无限制，随时可修改 |
| 生效时间 | 修改后立即生效（正在进行的防守战斗使用旧阵容） |

### AI防守策略

| 策略 | 说明 |
|------|------|
| 均衡 | 攻防兼顾，优先攻击血量最低目标 |
| 猛攻 | 优先攻击，集中火力击杀单一目标 |
| 坚守 | 优先防御，使用防御技能和回复技能 |
| 智谋 | 优先释放控制和AOE技能，最大化技能效果 |

### 防守日志

| 参数 | 说明 |
|------|------|
| 记录内容 | 被挑战记录列表（胜/负标识 + 对手名 + 排名变化 + 时间） |
| 保存数量 | 最多 50 条 |
| 点击展开 | 查看战斗回放 |
| 统计数据 | 防守胜率统计 |
| 智能提示 | 被攻破时推荐调整阵容（"建议更换防守策略为坚守"） |

### 防守奖励

| 事件 | 奖励 |
|------|------|
| 防守成功 | 积分 +20~40 + 铜钱 ×200 |
| 防守失败 | 积分 -10~25（无其他惩罚） |
| 连续防守成功 | 每连续5次额外奖励竞技币 ×20 |
