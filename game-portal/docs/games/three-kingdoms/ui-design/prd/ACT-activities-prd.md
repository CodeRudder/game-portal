## [ACT-1] 活动类型 {#act-1}

### 功能描述

活动系统是放置游戏长线运营的核心模块，通过限时/常驻/节日三类活动为玩家提供周期性的内容更新和奖励追求，驱动活跃度和付费转化。

> 🎨 → [UI: 活动布局](../ui-layout/ACT-activities.md#act-1)

### 活动类型矩阵

| 活动类型 | 持续时间 | 频率 | 代币类型 | 离线效率 | 核心玩法 |
|---------|:-------:|:----:|---------|:-------:|---------|
| **赛季活动** | 14-28天 | 每月1次 | 赛季令牌 | 50%自动 | 里程碑+排行+商店 |
| **限时活动** | 3-7天 | 每周1-2次 | 活动铜钱 | 30%自动 | 限定任务+兑换 |
| **日常活动** | 24h重置 | 每天 | 日常代币 | 100%自动 | 每日任务+签到 |
| **节日活动** | 7-14天 | 节日期间 | 节庆灯笼 | 50%自动 | 特殊玩法+限定奖励 |
| **联盟活动** | 3-5天 | 每两周1次 | 联盟勋章 | 仅个人部分 | 协作目标+排行 |

### 活动在放置循环中的角色

```
放置核心循环：挂机产出 → 资源积累 → 消费升级 → 解锁新内容 → 挂机产出（加速）

活动系统在循环中的角色：
  挂机产出 ←── 活动期间产出倍率加成（丰收祭/双倍周/节日加成）
  资源积累 ←── 活动奖励提供大量资源（里程碑/排行/兑换商店）
  消费升级 ←── 活动代币兑换稀有道具/武将碎片/加速道具
  解锁内容 ←── 限定活动解锁专属武将/皮肤/建筑外观
  挂机加速 ←── 活动自动推进，离线持续累积活动积分与代币
```

### 限时活动（Limited-Time Event）

- **定义**：固定周期内开放的短期活动，提供限定奖励和特殊玩法
- **持续时间**：3-7 天
- **频率**：每周 1-2 次
- **核心玩法**：限定任务 + 代币兑换商店
- **离线效率**：30% 自动累积积分和代币
- **典型示例**：商队节、丰收祭典、武将试炼、寻宝奇兵

### 常驻活动（Permanent Activity）

- **定义**：每日/每周循环的常规活动，提供稳定的资源获取渠道
- **子类型**：
  - **日常活动**：24h 重置，包含每日签到、每日试炼、活跃度任务
  - **周常活动**：每周重置，包含周目标、周排行
  - **赛季活动**：14-28 天长周期，包含里程碑、排行榜、七阶商店
- **离线效率**：日常 100% / 赛季 50% 自动累积

### 节日活动（Festival Event）

- **定义**：在现实节日或游戏内特殊日期开放的限定活动
- **持续时间**：7-14 天
- **频率**：跟随节日排期
- **核心玩法**：特殊小游戏 + 限定剧情 + 独占奖励
- **离线效率**：50% 自动累积
- **典型示例**：

| 节日 | 活动名称 | 特色玩法 | 限定奖励 |
|------|---------|---------|---------|
| 春节 | 新年庆典 | 放烟花/发红包/年兽讨伐 | 限定武将「赵云·新春」 |
| 端午 | 龙舟竞渡 | 龙舟赛/包粽子 | 限定皮肤「龙舟战船」 |
| 中秋 | 月满三国 | 赏月/猜灯谜/玉兔寻宝 | 限定建筑「月宫殿」 |
| 国庆 | 天下大庆 | 全服目标/国战排行 | 限定称号「开国功臣」 |
| 周年庆 | 霸业周年 | 回归奖励/全服福利/限定抽卡 | 限定武将「曹操·霸主」 |

### 联盟活动（Alliance Activity）

- **定义**：需要联盟成员协作完成的活动
- **持续时间**：3-5 天
- **频率**：每两周 1 次
- **核心玩法**：协作目标 + 联盟排行 + 个人贡献奖励
- **离线效率**：仅个人部分自动累积

#### 里程碑规则

- **积分获取**：完成任务、战斗胜利、离线累积均获得活动积分
- **线性推进**：里程碑按顺序解锁，不可跳过
- **手动领取**：里程碑达成后奖励需手动领取（防止误操作）
- **最终节点**：最高里程碑需手动确认领取，离线不自动完成

## [ACT-4] 活动奖励 {#act-4}

### 功能描述

定义活动的奖励体系，包括代币经济、奖励池、保底机制和结算规则，确保玩家在不同投入程度下都能获得合理回报。

> 🎨 → [UI: 活动布局](../ui-layout/ACT-activities.md#act-4)

### 代币经济体系

#### 代币生命周期

```
代币生命周期：获取 → 累积 → 消耗（兑换/加速） → 结算转化

  获取层（7种途径）：
  ├─ 战斗积分：每次战斗自动 +5~20 代币（按活动类型）
  ├─ 里程碑达成：一次性 +200~5000 代币
  ├─ 活动任务：每任务 +100~500 代币
  ├─ 排行结算：期末 +500~5000 代币
  ├─ 离线累积：+2~10 代币/小时（按活动效率系数）
  ├─ 每日活跃：达标 +150 代币/天
  └─ 元宝购买：1 代币/元宝（可选付费加速）

  消耗层（3种途径）：
  ├─ 兑换商店：购买道具/武将碎片/限定物品
  ├─ 里程碑加速：跳过当前任务，消耗 = 当前需求 ×50% 代币
  └─ 重置挑战：重置挑战次数，消耗因活动而异

  结算层：
  └─ 活动结束后未使用代币 → 按 10% 比例转化为普通铜钱
```

#### 各活动类型代币获取效率

| 代币获取途径 | 赛季活动 | 限时活动 | 日常活动 | 节日活动 |
|------------|:-------:|:-------:|:-------:|:-------:|
| 战斗积分 | +10~20/次 | +5~15/次 | +5~10/次 | +10~20/次 |
| 里程碑 | +200~5000 | +100~2000 | +50~500 | +200~3000 |
| 活动任务 | +100~500 | +50~300 | +30~150 | +100~500 |
| 排行结算 | +500~5000 | — | — | +300~3000 |
| 离线累积 | +5~10/h | +2~8/h | +5~10/h | +5~10/h |
| 每日活跃 | +150/天 | +100/天 | +100/天 | +150/天 |
| 元宝购买 | 1代币/元宝 | 1代币/元宝 | 2代币/元宝 | 1代币/元宝 |

### 离线积分计算公式

```
离线活动积分 = 基础每秒积分 × 离线效率系数 × min(有效离线时长, 上限时长)

离线效率系数 = 基础效率 × (1 + 声望等级 × 0.03)
  - 赛季活动基础效率: 50%
  - 限时活动基础效率: 30%
  - 日常活动基础效率: 100%
  - 节日活动基础效率: 50%

有效离线时长 = 实际离线时长（扣除超过上限部分）

声望等级加成示例：
  - 声望 Lv.0: 效率系数 = 基础 × 1.00
  - 声望 Lv.5: 效率系数 = 基础 × 1.15
  - 声望 Lv.10: 效率系数 = 基础 × 1.30
  - 声望 Lv.20: 效率系数 = 基础 × 1.60
```

### 自动完成引擎

```
自动完成触发规则：
  ✅ 自动完成（无需玩家操作）：
  ├─ 累积型任务：达到目标值自动标记完成
  ├─ 每日任务中的自动类（登录/收取资源）：离线期间正常计入
  ├─ 里程碑积分：达标自动解锁节点
  └─ 战斗积分：按离线效率自动累积

  ⚠️ 半自动（需玩家确认领取奖励）：
  ├─ 已完成的任务奖励：标记完成但奖励存入「待领取」队列
  ├─ 已解锁的里程碑：标记解锁但奖励需手动领取
  └─ 排行榜结算：自动计算排名，奖励需手动领取

  ❌ 不自动完成（必须在线操作）：
  ├─ 挑战任务（需手动操作如「击败精英 Boss」）
  ├─ 商店兑换（绝不自动消耗代币）
  ├─ 最终里程碑奖励（需手动确认，防止误操作）
  └─ 联盟协作目标（需多人在线协作）
```

### 活动结算规则

```
活动结束时结算流程：
  1. 关闭所有进行中的活动任务
  2. 计算最终排行榜排名
  3. 发放排行奖励至邮箱
  4. 未领取的里程碑奖励自动发放至邮箱
  5. 未使用的活动代币按 10% 比例转化为铜钱
  6. 生成活动结算报告（获得总积分/排名/总奖励清单）
  7. 结算报告在回归面板中展示
```
