## [MAP-2] 筛选逻辑 {#map-2}

### 筛选维度

| 筛选组 | 筛选项 | 交互方式 | 组合逻辑 |
|--------|--------|---------|---------|
| **阵营** | 全部 / 蜀 / 魏 / 吴 / 中立 | 单选 | AND |
| **地形** | 平原 / 山地 / 城池 / 森林 / 水域 / 关隘 | 多选 | OR |
| **收益类型** | 粮草 / 铜钱 / 兵力 | 多选 | OR |
| **等级范围** | 全部 / Lv5+ / Lv10+ / Lv15+ | 单选 | AND |
| **热力图** | 开/关 | 切换 | 独立模式 |

### 组合逻辑

- **阵营 ∩ 等级** = AND 关系
- **地形 ∪ 收益** = OR 关系
- 示例：蜀国 + 平原+城池 + 粮草 → 显示蜀国中（平原或城池）且产粮草的领土

### 过滤视觉效果

| 状态 | 效果 |
|------|------|
| 匹配领土 | 正常显示 + 古铜金高亮边框 |
| 不匹配领土 | 亮度降至 40% + 灰度滤镜 |

### 快捷筛选按钮

| 按钮 | 功能 |
|------|------|
| 🏠 我的领土 | 过滤为己方阵营 |
| 📊 高产领地 | 按总收益排序，高亮前 5 |
| ⚔️ 可征服 | 显示与己方相邻的非己方领土 |
| 🗺️ 热力图 | 切换收益热力图模式 |

### 热力图模式

开启后领土按收益高低显示颜色深浅：

| 收益区间 | 颜色 |
|---------|------|
| Top 20%（高收益） | 深金 #C9A84C |
| 中高收益 | 浅金 #E8D48B |
| 中等收益 | 翠绿 #7EC850 |
| 中低收益 | 浅绿 #A8D88A |
| Bottom 20%（低收益） | 灰绿 #6B8B6B |

### 手机端筛选差异

- 入口：右上角「漏斗」图标按钮（44×44px）
- 面板：底部弹出 Bottom Sheet
- 操作：分组纵向排列，底部「重置」+「应用筛选」双按钮

## [MAP-3] 领土系统 {#map-3}

### 领土产出计算

```
领土总产出 = 基础产出 × 地形加成 × 阵营加成 × 科技加成 × 声望加成 × 地标加成
```

| 加成来源 | 示例 |
|---------|------|
| 地形加成 | 平原粮草 +30%、水域铜钱 +25% |
| 阵营加成 | 己方领土全产出 +10% |
| 科技加成 | 「屯田」粮草 +15% |
| 声望加成 | 声望等级 × 2% |
| 地标加成 | 洛阳全资源 +50% |

### 产出气泡显示规则

| 条件 | 行为 |
|------|------|
| 己方领土 | 始终显示产出气泡 |
| 非己方领土 | 不显示（悬停 Tooltip 显示潜在收益） |
| 刚占领领土 | 气泡从 0 开始充能动画 |
| 产出为 0 | 灰色气泡，提示需开发 |
| 缩放 < 60% | 隐藏气泡避免拥挤 |

### 驻防机制

| 参数 | 规则 |
|------|------|
| 驻防兵力上限 | 领土等级 × 100 |
| 驻防效果 | 每点兵力增加防守方 0.1% 防御 |
| 驻防消耗 | 从总兵力池中分配，可随时调回 |
| 自动驻防 | 新占领领土自动分配 50% 兵力上限 |

### 征服规则

```
征服条件：
  - 目标领土与己方领土相邻
  - 可用兵力 ≥ 目标驻防兵力 × 1.5

征服流程：
  1. 点击领土 → 查看详情（驻防兵力、产出、地形）
  2. 点击「征服」→ 兵力部署滑块
  3. 系统预估胜率 = 我方战力 / (敌方战力 × 地形防御系数)
  4. 确认出征 → 进入战斗（或自动结算）

征服结果：
  - 胜利：领土归己方，驻防自动部署
  - 失败：损失 30% 出征兵力，领土不变
```

### 胜率预估公式

```
胜率 = min(95%, max(5%, (我方战力 / 敌方战力) × 50% + 地形修正))
```

| 胜率区间 | 显示颜色 | 建议 |
|---------|---------|------|
| >80% | 翠绿 | 稳操胜券 |
| 60%~80% | 金色 | 有一定风险 |
| 40%~60% | 琥珀橙 | 势均力敌 |
| <40% | 赤红 | 建议强化 |

### 领土等级

| 等级 | 解锁条件 | 产出倍率 | 驻防上限 |
|:----:|---------|:-------:|:-------:|
| 1 | 初始占领 | ×1.0 | 100 |
| 5 | 占领 24 小时 | ×1.2 | 500 |
| 10 | 占领 72 小时 + 投资 | ×1.5 | 1,000 |
| 15 | 占领 7 天 + 重金投资 | ×2.0 | 1,500 |

### 离线领土变化

| 变化类型 | 视觉标记 | 持续时间 |
|---------|---------|---------|
| 新占领领土 | 金色脉冲光环 + "新"标签 | 首次点击后消失 |
| 失去领土 | 红色脉冲光环 + "失"标签 | 首次点击后消失 |
| 收益变化 | 产出气泡闪烁 + ↑↓箭头 | 10 秒后恢复 |
| 防御变化 | 领土边缘闪烁 | 10 秒后恢复 |

## [MAP-4] 攻城战玩法 {#map-4}

### 功能描述

攻城战是领土征服中的高难度战斗模式，针对城池类领土。城池拥有独立的城防值和守军配置，需要玩家投入更多兵力和策略才能攻占。

> 🎨 → [UI: 攻城确认弹窗](../ui-layout/MAP-world.md#map-3)

### 攻城条件

| 条件 | 说明 |
|------|------|
| 领土相邻 | 目标城市必须与己方领土相邻 |
| 兵力消耗 | 兵力 ×100 |
| 粮草消耗 | 粮草 ×500 |
| 每日次数上限 | 3 次 |

### 城防计算

| 参数 | 公式/规则 |
|------|---------|
| 城防值 | 基础城防（500~5000）+ 城墙建筑等级加成 + 驻防兵力 × 0.5 |
| 城防恢复 | 每小时恢复城防上限的 5%，战斗中不恢复 |
| 城防效果 | 每点城防值降低攻方伤害 0.02%（上限 -50%） |

### 攻城消耗

| 消耗项 | 计算规则 |
|--------|---------|
| 出征兵力 | ≥ 城池驻防兵力 × 2.0（最低门槛） |
| 粮草消耗 | 出征距离 × 50 + 城防等级 × 100 |
| 攻城时间 | 基础 30 分钟 + 城防值 / 100（分钟） |

### 占领条件

| 条件 | 说明 |
|------|------|
| 城防值归零 | 攻方需将城防值降至 0 |
| 守军全灭 | 消灭所有驻防部队 |
| 战力优势 | 攻方战力 ≥ 守方战力 × 1.3 |
| 占领冷却 | 攻占后 24 小时内不可被反攻（新手保护期） |

### 占领规则

- 占领后获得该城市产出，初始 50%，24 小时后 100%

### 攻城奖励

| 奖励 | 说明 |
|------|------|
| 首次攻占 | 元宝 ×100 + 声望 +50 + 专属称号 |
| 重复攻占 | 铜钱 ×5000 + 资源产出 ×2（24h） |
| 攻城失败 | 返还 50% 出征兵力，粮草不返还 |

## [MAP-5] 地图事件 {#map-5}

### 功能描述

地图事件是在世界地图上随机触发的限时玩法事件，为玩家提供额外的策略选择和资源奖励，增加地图探索的趣味性。

> 🎨 → [UI: 地图事件标记](../ui-layout/MAP-world.md#map-4)

### 事件类型

| 类型 | 概率 | 效果 |
|------|------|------|
| 商队遇险 | 30% | 援助→获金币 / 忽略 |
| 流民 | 25% | 接纳→获兵力 / 遣散 |
| 宝箱 | 20% | 直接获资源 |
| 山贼 | 25% | 战斗→获装备 / 绕行 |

| 事件类型 | 图标 | 出现频率 | 持续时间 | 说明 |
|---------|:----:|:-------:|:-------:|------|
| 🏴‍☠️ 流寇入侵 | 刀剑 | 每日 2~3 次 | 2 小时 | 击败流寇获取资源 |
| 📦 商队经过 | 马车 | 每日 1~2 次 | 1.5 小时 | 护送/截获商队 |
| 🌪️ 天灾降临 | 乌云 | 每周 1 次 | 24 小时 | 领土产出降低/提升 |
| 🏛️ 遗迹发现 | 宝箱 | 每周 2 次 | 4 小时 | 探索获取稀有道具 |
| ⚔️ 阵营冲突 | 旗帜 | 每周 1 次 | 48 小时 | 阵营间争夺资源点 |

### 触发规则

| 规则 | 说明 |
|------|------|
| 触发概率 | 每小时 10% 概率触发 |
| 未处理上限 | 最多 3 个未处理事件 |
| 随机触发 | 基于玩家等级和地图探索进度随机生成 |
| 区域限制 | 事件仅在已探索区域触发 |
| 数量上限 | 同时存在的地图事件不超过 3 个 |
| 通知方式 | 事件图标在地图上脉冲闪烁 + 顶部事件条提示 |

### 选择分支

每个地图事件提供 2~3 个选择分支，不同选择影响奖励和后续事件：

| 选择类型 | 风险 | 收益 |
|---------|:----:|:----:|
| 强攻 | 高 | 高 |
| 谈判 | 低 | 中 |
| 忽略 | 无 | 无（可能触发后续负面事件） |

## [MAP-6] 地图统计 {#map-6}

### 功能描述

地图统计面板汇总展示玩家在地图系统中的各项数据，帮助玩家评估领土经营状况和战略方向。

> 🎨 → [UI: 地图统计面板](../ui-layout/MAP-world.md#map-5)

### 统计维度

- 领土数量 / 占比
- 势力分布
- 总产出汇总

| 维度 | 数据项 | 说明 |
|------|--------|------|
| 领土概览 | 总领土数 / 各阵营领土数 | 当前占领情况 |
| 资源产出 | 总产出 / 各领土产出明细 | 领土资源收益 |
| 战斗统计 | 攻城次数 / 胜率 / 伤亡 | 战斗表现 |
| 探索进度 | 已探索区域 / 未探索区域 | 地图覆盖率 |
| 事件参与 | 已完成事件 / 事件奖励总计 | 事件参与度 |

### 数据来源

| 数据 | 来源系统 | 更新频率 |
|------|---------|---------|
| 领土数据 | 领土系统 | 实时 |
| 战斗数据 | 战斗系统 | 战斗结束后 |
| 资源数据 | 资源系统 | 每秒更新 |
| 探索数据 | 地图系统 | 探索触发时 |

## [MAP-4] 攻城战玩法

### 攻城条件
- 目标城市必须与己方领土相邻
- 消耗：兵力*100 + 粮草*500
- 每日攻城次数上限：3次

### 城防计算
- 城防值 = 基础城防(1000) x 城市等级 x (1 + 城防科技加成)
- 城防值归零则城市占领

### 占领规则
- 占领后获得该城市产出，初始50%，24小时后100%
