## [TRD-1] 贸易路线 {#trd-1}

### 功能描述

贸易路线系统是放置游戏中后期核心收入来源。玩家开通城市间商路，派遣商队运输商品赚取差价利润。商品价格随行情波动，低买高卖是核心策略。离线期间商队继续自动运输。

### 解锁条件

| 条件 | 数值 |
|------|------|
| 主城等级 | ≥ 6 |
| 资源建筑 | ≥ 2 座 |
| 新手引导 | 完成第3步 |
| 入口 | 城池地图「集市」建筑 → 「商路管理」 / "更多"菜单「贸易」 |

### 城市节点

8座主要城市，每座城市有特产商品和需求商品：

| 城市 | 特产商品 | 需求商品 |
|------|---------|---------|
| 洛阳（帝都） | 书籍、杜康酒、兵器 | 蜀锦、西凉马、药材 |
| 长安 | 西凉马、兵器、和田玉（极低概率） | 书籍、蜀锦 |
| 成都 | 蜀锦、药材 | 兵器、西凉马 |
| 许昌 | 书籍 | 杜康酒、木材 |
| 邺城 | 兵器、铁矿 | 书籍、药材 |
| 建业 | 木材 | 蜀锦、杜康酒 |
| 南中 | 药材、粮草 | 兵器、书籍 |
| 会稽 | 木材、粮草 | 兵器、西凉马 |

### 商路开通条件

| 商路 | 费用（铜钱） | 前置条件 | 基础利润/趟 |
|------|:----------:|---------|:---------:|
| 洛阳→许昌 | 2,000 | 主城 Lv6 | 200 |
| 洛阳→长安 | 3,000 | 主城 Lv8 | 300 |
| 长安→成都 | 5,000 | 主城 Lv10 + 长安已开通 | 450 |
| 成都→南中 | 4,000 | 主城 Lv12 + 成都已开通 | 380 |
| 许昌→邺城 | 3,500 | 主城 Lv10 + 许昌已开通 | 320 |
| 许昌→建业 | 6,000 | 主城 Lv14 + 许昌已开通 | 550 |
| 建业→会稽 | 2,500 | 主城 Lv10 + 建业已开通 | 250 |
| 邺城→建业 | 8,000 | 主城 Lv16 + 两城均已开通 | 700 |

### 商品系统

| 商品 | 类型 | 基础价格 | 产出城市 | 价格波动范围 |
|------|------|:-------:|---------|:----------:|
| 🌾 粮草 | 基础 | 10 | 成都、南中 | ±20% |
| 🪵 木材 | 基础 | 15 | 会稽、南中 | ±15% |
| ⛏️ 铁矿 | 基础 | 25 | 邺城、长安 | ±20% |
| 📜 书籍 | 精品 | 50 | 洛阳、许昌 | ±30% |
| 🍶 杜康酒 | 精品 | 80 | 洛阳 | ±35% |
| 🗡️ 兵器 | 精品 | 100 | 邺城、长安 | ±25% |
| 🐎 西凉马 | 稀有 | 200 | 长安 | ±40% |
| 🌸 蜀锦 | 稀有 | 150 | 成都 | ±35% |
| 🌿 药材 | 稀有 | 120 | 南中 | ±30% |
| 💎 和田玉 | 传说 | 500 | 长安（极低概率） | ±50% |

### 价格波动规则

| 规则 | 说明 |
|------|------|
| 刷新频率 | 每 6 小时刷新一次行情 |
| 波动公式 | 实际价格 = 基础价格 × (1 ± 波动率) |
| 连续限制 | 连续涨/跌不超过 3 次 |
| 稀有商品 | 波动更大但利润更高 |
| 科技加成 | 「商业天赋」科技可查看未来 1 次行情趋势 |

### 贸易利润占比（按游戏阶段）

| 游戏阶段 | 主城等级 | 贸易利润占比 | 主要收入来源 |
|---------|:-------:|:----------:|------------|
| 早期 | 1~5 | 10% | 建筑产出 |
| 中期 | 6~15 | 30% | 建筑 + 贸易 |
| 后期 | 16~20 | 40% | 贸易 + 远征 |
| 终局 | 20+ | 50% | 贸易为主 |

## [TRD-2] 商队系统 {#trd-2}

### 功能描述

商队是贸易的执行单元。每支商队拥有载重、速度、护卫等属性，派遣后沿商路运输商品。商队数量随主城等级增加。

### 商队数量上限

| 主城等级 | 商队上限 |
|:-------:|:-------:|
| 6~9 | 2 |
| 10~14 | 4 |
| 15~19 | 6 |
| 20+ | 8 |

### 商队属性

| 属性 | 基础值 | 提升方式 | 说明 |
|------|:-----:|---------|------|
| 载重 | 50 | 科技研究 / 武将加成 | 决定单趟运输商品数量 |
| 速度 | 基础 | 道路等级 / 加速道具 | 影响运输时间 |
| 护卫 | 0 | 派遣武将护卫 | 降低遇盗概率 |
| 议价能力 | 基础 | 科技 / 武将加成 | 买入更低 / 卖出更高 |

### 商队状态

| 状态 | 说明 |
|------|------|
| 🐫 运输中 | 正在路上，显示预计到达时间 |
| ⏳ 等待 | 空闲，可派遣至任意已开通商路 |
| ✓ 已到达 | 已到达目的地，等待收税领取利润 |
| 🔒 已锁定 | 商队槽位未解锁 |

### 运输时间

| 商路距离 | 基础运输时间 |
|---------|:----------:|
| 短途（相邻城市） | 2~3 小时 |
| 中途（隔一个城市） | 4~6 小时 |
| 长途（跨区域） | 6~8 小时 |

### 护卫武将规则

| 规则 | 说明 |
|------|------|
| 护卫位数 | 每支商队最多派遣 1 名护卫武将 |
| 武将互斥 | 护卫武将不可同时编入远征队伍 |
| 护卫效果 | 降低遇盗事件概率 50% + 自动处理低风险事件 |
| 武将属性影响 | 武将战力越高，护卫效果越强 |

### 离线贸易规则

| 离线场景 | 处理方式 |
|---------|---------|
| 商队运输中 | 继续运输（引用全局离线效率系数） |
| 商队到达 | 自动完成交易，货物存入仓库，铜钱自动入账 |
| 贸易事件 | 自动处理（默认选择保守选项） |
| 高风险事件 | 等待上线处理 |

### 自动贸易

| 条件 | 说明 |
|------|------|
| 开启条件 | 科技「自动商路」研究完成 |
| 自动选择 | 自动选择最优路线和商品 |
| 事件处理 | 自动处理低风险事件，高风险事件等待上线 |
| 利润优化 | 优先选择当前行情利润最高的商路 |

## [TRD-3] 贸易事件 {#trd-3}

### 功能描述

商队运输途中会随机触发各类事件，带来风险或机遇。玩家可选择不同处理方式，护卫武将可自动处理部分事件。

### 随机事件表

| 事件 | 触发概率 | 效果 | 处理选项 |
|------|:-------:|------|---------|
| 山贼劫道 | 15% | 损失 30% 货物 | 武将战斗 / 支付赎金（损失 15% 货物） |
| 暴雨泥泞 | 20% | 延迟 1 小时 | 等待 / 使用加速道具 |
| 商路发现 | 10% | 额外获得稀有商品 | 自动获得 |
| 官府设卡 | 15% | 缴纳 10% 税金 | 缴纳 / 贿赂（50%概率免税，50%概率罚双倍） |
| 商队遇友 | 10% | 速度 +50% | 自动触发 |
| 路遇流民 | 10% | 选择施舍或无视 | 施舍：声望 +5 / 无视：无效果 |
| 天降横财 | 5% | 利润翻倍 | 自动触发 |
| 商路断绝 | 5% | 商路暂时关闭 2 小时 | 等待恢复 / 绕路（多 50% 时间） |

### 事件处理规则

| 规则 | 说明 |
|------|------|
| 触发时机 | 每趟运输触发 0~2 个随机事件 |
| 护卫自动处理 | 有护卫武将时，山贼劫道/暴雨泥泞自动处理（无损失） |
| 离线处理 | 默认选择保守选项（缴纳/等待/缴纳） |
| 高风险事件 | 贿赂/绕路等高风险选项离线时不自动选择 |

### NPC特殊商人

| 商人 | 出现条件 | 出售商品 | 特殊能力 |
|------|---------|---------|---------|
| 丝绸之路商人 | 每周日 | 稀有商品（半价） | 可购买传说品质商品 |
| 江东商贾 | 建业繁荣度 ≥80% | 吴国特产 | 以物易物（免铜钱） |
| 蜀中行商 | 成都繁荣度 ≥80% | 蜀锦、药材 | 批量折扣（满5件打8折） |
| 北方马贩 | 邺城繁荣度 ≥80% | 西凉马 | 可购买战马（装备类） |
| 黑市商人 | 随机出现（5%概率） | 禁品（高利润） | 高风险高回报，有概率被没收 |

### NPC商人规则

| 规则 | 说明 |
|------|------|
| 出现方式 | 城池地图上出现商人NPC（特殊造型） |
| 存在时长 | 4小时，倒计时显示 |
| 交易次数 | 每个NPC每次出现可交易 3 次 |
| 黑市风险 | 黑市商人交易有 20% 概率被没收货物（损失全部交易商品） |

### 商路繁荣度

| 繁荣度等级 | 繁荣度范围 | 效果 |
|:---------:|:---------:|------|
| 萧条 | 0%~30% | 基础利润 ×0.8 |
| 平淡 | 31%~60% | 基础利润 ×1.0 |
| 繁荣 | 61%~80% | 基础利润 ×1.2 + 解锁对应城市NPC商人 |
| 鼎盛 | 81%~100% | 基础利润 ×1.5 + 解锁NPC商人 + 事件触发率降低 |

### 繁荣度提升规则

| 操作 | 繁荣度变化 |
|------|:---------:|
| 完成一次贸易 | +2% |
| 触发商路发现事件 | +5% |
| 商路断绝事件 | -10% |
| 自然衰减 | 每日 -1%（最低0%） |
