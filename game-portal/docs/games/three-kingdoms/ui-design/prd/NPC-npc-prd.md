## [NPC-1] NPC类型 {#npc-1}

### 功能描述

游戏包含5类NPC，各具独特功能和交互方式。NPC以Q版立绘显示在主城地图上，头顶悬浮职业图标和名字铭牌。

> 🎨 → [UI: NPC地图展示与名册](../ui-layout/NPC-npc.md#npc-1)

### NPC类型定义

| 类型 | 图标 | 核心功能 | 交互选项 | 代表NPC |
|------|:----:|---------|---------|---------|
| 🌾 农民 | 稻穗 | 资源产出加成、农业知识 | 对话、赠送、任务 | 张老农、王佃户 |
| ⚔️ 士兵 | 刀盾 | 战斗辅助、城防巡逻 | 对话、切磋、任务 | 赵校尉、陈百夫 |
| 💰 商人 | 铜钱 | 商品交易、稀有道具 | 对话、交易、赠送 | 王掌柜、孙客商 |
| 📜 学者 | 卷轴 | 科技加速、情报分析 | 对话、情报、赠送 | 李学者、郑书生 |
| 🔭 斥候 | 望远镜 | 地图探索、隐藏区域线索 | 对话、情报、任务 | 陈斥候、刘探子 |

### NPC属性

| 属性 | 说明 |
|------|------|
| 职业 | 农民/士兵/商人/学者/斥候，决定功能类型 |
| 等级 | 随好感度和互动提升，影响功能效果 |
| 好感度 | 0~1000点，分5个等级，解锁更多交互选项 |
| 当前状态 | 闲逛/劳作/待命/巡逻/研究中，影响可交互性 |
| 日程 | NPC按时间表移动（巡逻/工作/休息），有行走动画 |

### 地图展示规则

| 平台 | 地图NPC上限 | 超出处理 |
|------|:---------:|---------|
| PC端 | 8个 | 显示前8 + 右侧"名册"入口徽标（带数量角标） |
| 手机端 | 5个 | 显示前5 + 底部"名册"浮动按钮 |

**展示优先级权重**：

```
优先级 = 有待领取任务(+100) + 好感度≥4(+50) + 同区域(+30) + 新对话(+80) + 特殊状态(+60) + 稀有NPC(+70) + 好感度×10
```

- 同区域NPC≥3个时自动聚合为"人群气泡"（显示职业图标群+数字角标）
- 点击气泡展开该区域NPC列表

> 🎨 → [UI: NPC地图展示与名册](../ui-layout/NPC-npc.md#npc-1)

## [NPC-2] NPC交互 {#npc-2}

### 功能描述

玩家通过与NPC对话、交易、切磋等方式提升好感度，解锁更多功能和专属奖励。

> 🎨 → [UI: NPC交互面板](../ui-layout/NPC-npc.md#npc-2)

### 交互选项解锁条件

| 交互选项 | 解锁条件 | 说明 |
|---------|---------|------|
| 💬 对话 | 好感度Lv.1（默认） | 基础对话，每日1次获得好感度 |
| 💰 交易 | 好感度Lv.1（默认） | NPC专属商品，好感度越高折扣越大 |
| 📜 情报 | 好感度Lv.2（100点） | 提供地图线索、隐藏区域信息 |
| 🎁 赠送 | 好感度Lv.3（300点） | 赠送礼物提升好感度，获得回礼 |
| ⚔️ 切磋 | 好感度Lv.4（600点） | 与NPC战斗，胜利获得奖励和好感度 |
| 🌟 羁绊技能 | 好感度Lv.5（1000点） | 解锁NPC专属羁绊加成 |

### 对话系统规则

- 对话以**竹简条目**形式展示选项，选中后高亮展开
- NPC表情随对话内容变化（立绘切换）
- 逐字显示文本 + 打字机音效
- 部分对话包含**分支选择**，影响好感度和后续任务
- 对话历史可通过「往昔」功能回看（按时间倒序排列）

### 好感度系统

#### 好感度等级与效果

| 等级 | 所需点数 | 解锁效果 | 视觉标识 |
|:----:|:-------:|---------|---------|
| Lv.1 | 0 | 基础对话 + 交易 | ⭐ 灰色 |
| Lv.2 | 100 | 解锁「情报」选项 | ⭐⭐ 铜色 |
| Lv.3 | 300 | 解锁「赠送」+ 商品折扣10% | ⭐⭐⭐ 银色 |
| Lv.4 | 600 | 解锁「切磋」+ 专属任务线 | ⭐⭐⭐⭐ 金色 |
| Lv.5 | 1000 | 解锁羁绊技能 + 专属称号 | ⭐⭐⭐⭐⭐ 金色脉动 |

#### 好感度获取途径

| 途径 | 每次获得 | 每日上限 | 说明 |
|------|:-------:|:-------:|------|
| 日常对话 | +5 | 1次/NPC | 每日首次对话 |
| 完成NPC任务 | +20~50 | 无上限 | 任务难度决定 |
| 赠送礼物（普通） | +10 | 3次/NPC | 普通品质礼物 |
| 赠送礼物（稀有） | +30 | 1次/NPC | 稀有品质礼物 |
| 切磋胜利 | +15 | 1次/NPC | 需好感度Lv.4 |
| 离线自然增长 | +0.1/小时 | +2/次 | 离线期间微量增长 |
| NPC好感度满级声望 | +200~800 | 一次性 | 每升一级奖励声望 |

#### 好感度规则

- 好感度**只增不减**（不会因任何操作降低）
- 转生时好感度**100%保留**
- 好感度满级后进度条替换为"羁绊已结"金色印章图标
- 每位NPC满级首次奖励：天命×150

### NPC专属羁绊技能（好感度Lv.5）

| NPC类型 | 羁绊技能名 | 效果 |
|---------|-----------|------|
| 🌾 农民 | 丰年之恩 | 农田产出永久+10% |
| ⚔️ 士兵 | 同袍之义 | 城防武力+15% |
| 💰 商人 | 知己之利 | 全商店永久折扣5%（与其他折扣叠加） |
| 📜 学者 | 智者之鉴 | 科技研究速度+8% |
| 🔭 斥候 | 鹰眼之能 | 远征发现隐藏宝箱概率+20% |

## [NPC-3] NPC任务 {#npc-3}

### 功能描述

每位NPC拥有专属任务链，随好感度等级逐步解锁。任务链提供丰厚奖励并深化NPC个人剧情。

> 🎨 → [UI: NPC任务面板](../ui-layout/NPC-npc.md#npc-3)

### 任务链结构

每个NPC类型拥有**1条主线任务链**（5~8个任务），好感度等级决定解锁进度：

| 好感度等级 | 解锁任务数 | 任务难度 | 奖励品质 |
|:---------:|:---------:|:-------:|:-------:|
| Lv.1 | 1~2个 | 简单 | 普通 |
| Lv.2 | +1个 | 简单~中等 | 普通~精良 |
| Lv.3 | +1~2个 | 中等 | 精良 |
| Lv.4 | +1~2个 | 中等~困难 | 精良~稀有 |
| Lv.5 | +1个（终章） | 困难 | 稀有~史诗 |

### 任务类型

| 任务类型 | 说明 | 示例 |
|---------|------|------|
| 收集类 | 收集指定数量资源或道具 | "收集粮草×5,000" |
| 战斗类 | 击败指定敌人或通关关卡 | "击败黄巾贼首×3" |
| 建造类 | 升级指定建筑至目标等级 | "将农田升至Lv.10" |
| 探索类 | 探索指定地图区域 | "探索襄阳周边" |
| 对话类 | 与其他NPC对话获取信息 | "向陈斥候打听消息" |
| 赠送类 | 向NPC赠送指定礼物 | "赠送美酒×1" |

### 任务奖励

| 奖励类型 | 说明 |
|---------|------|
| 资源奖励 | 粮草、铜钱、兵力等基础资源 |
| 道具奖励 | 扫荡令、经验书、征兵令等 |
| 好感度 | 完成任务获得20~50点好感度 |
| 声望值 | 任务完成奖励100~500声望 |
| 专属道具 | 任务链终章奖励NPC专属稀有道具 |
| 天命 | 高难度任务链终章奖励天命×50~100 |

### 任务链示例（商人·王掌柜）

| 任务序号 | 任务名 | 要求 | 奖励 | 解锁条件 |
|:-------:|--------|------|------|---------|
| 1 | 初次交易 | 出售任意物品×3 | 铜钱×500 + 好感度+20 | 好感度Lv.1 |
| 2 | 稀缺货源 | 收集丝绸×5 | 铜钱×1,200 + 好感度+25 | 好感度Lv.1 |
| 3 | 商路开拓 | 占领3座商业城池 | 精良装备×1 + 好感度+30 | 好感度Lv.2 |
| 4 | 贵客临门 | 赠送美酒×1给王掌柜 | 商店折扣券 + 好感度+35 | 好感度Lv.3 |
| 5 | 商战风云 | 击败敌对商人（战斗任务） | 稀有图纸×1 + 好感度+40 | 好感度Lv.4 |
| 6 | 富甲天下 | 累积获得铜钱100,000 | 史诗饰品×1 + 天命×80 + 好感度+50 | 好感度Lv.5 |

### 任务规则

- 任务**无时间限制**，接受后永久有效直至完成或放弃
- 同一NPC同时只能进行**1个任务**
- 任务可放弃，放弃后可重新接取（进度重置）
- 任务链必须**按顺序完成**，不可跳过
- 转生时任务进度**重置**，但已完成记录保留（可快速跳过已看剧情）

## [NPC-5] 高级交互 {#npc-5}

### 功能描述

高级交互是好感度达到较高等级后解锁的特殊玩法，包括赠送系统、切磋系统和结盟系统，为玩家与NPC建立深度羁绊。

> 🎨 → [UI: NPC高级交互面板](../ui-layout/NPC-npc.md#npc-2)

### 赠送系统

| 赠送类型 | 好感度要求 | 每日上限 | 好感度提升 |
|---------|:---------:|:-------:|:---------:|
| 普通礼物（粮草/铜钱） | Lv.3 | 3次/NPC | +10 |
| 稀有礼物（装备/道具） | Lv.3 | 1次/NPC | +30 |
| 专属礼物（NPC偏好物品） | Lv.4 | 1次/NPC | +50 |

### NPC偏好物品

| NPC类型 | 偏好物品 | 额外加成 |
|---------|---------|---------|
| 🌾 农民 | 美酒、种子 | 好感度 ×1.5 |
| ⚔️ 士兵 | 兵书、武器 | 好感度 ×1.5 |
| 💰 商人 | 珍珠、丝绸 | 好感度 ×1.5 |
| 📜 学者 | 书籍、古董 | 好感度 ×1.5 |
| 🔭 斥候 | 地图、望远镜 | 好感度 ×1.5 |

### 切磋系统

| 参数 | 规则 |
|------|------|
| 解锁条件 | 好感度 ≥ Lv.4 |
| 每日次数 | 1次/NPC |
| 战斗模式 | 自动战斗，3回合制 |
| 胜利奖励 | 好感度+15 + 随机道具 |
| 失败奖励 | 好感度+5（安慰奖） |
| NPC战力 | 随NPC等级和类型变化 |

### 结盟系统

| 参数 | 规则 |
|------|------|
| 解锁条件 | 好感度 = Lv.5（满级） |
| 结盟效果 | 永久羁绊加成（见NPC-2羁绊技能表） |
| 结盟仪式 | 专属剧情动画（30秒） |
| 结盟数量 | 每类NPC最多结盟1位 |

### NPC日程系统

NPC按24小时日程循环运作：

| 时段 | 农民 | 士兵 | 商人 | 学者 | 斥候 |
|------|------|------|------|------|------|
| 06:00~12:00 | 🌾 劳作 | ⚔️ 巡逻 | 💰 营业 | 📜 研究 | 🔭 侦察 |
| 12:00~14:00 | 🍚 休息 | 🍚 休息 | 💰 营业 | 🍚 休息 | 🍚 休息 |
| 14:00~20:00 | 🌾 劳作 | ⚔️ 巡逻 | 💰 营业 | 📜 研究 | 🔭 侦察 |
| 20:00~22:00 | 🍚 休息 | ⚔️ 守夜 | 🍚 休息 | 📜 夜读 | 🔭 夜巡 |
| 22:00~06:00 | 😴 睡眠 | ⚔️ 守夜 | 😴 睡眠 | 😴 睡眠 | 🔭 夜巡 |

- **劳作/营业/研究/侦察**状态：产出资源，可正常交互
- **休息**状态：可交互但无产出
- **睡眠**状态：不可交互（显示"Zzz"气泡）
- **巡逻/守夜/夜巡**状态：士兵和斥候特殊状态，降低被袭击概率

## [NPC-5] NPC高级交互

### 赠送系统
- 可赠送资源或装备给NPC提升好感度

### 切磋系统

- 好感度达到亲密后可结盟
- 结盟NPC提供永久被动加成，最多结盟3个

## [NPC-6] 手机端NPC适配

### 适配规则
- NPC列表用Bottom Sheet展示
- 对话和交易用全屏面板
