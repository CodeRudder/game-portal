## [HER-1] 武将属性 {#her-1}

### 四维属性

| 属性 | 缩写 | 影响范围 | 成长方向 |
|------|:----:|---------|---------|
| 武力 | ATK | 普攻伤害、物理技能伤害 | 猛将型武将高 |
| 智力 | INT | 技能伤害、策略效果、灼烧/中毒伤害 | 谋士型武将高 |
| 统率 | CMD | 带兵量、防御、部队生存 | 统帅型武将高 |
| 政治 | POL | 内政效果、资源产出、民心 | 内政型武将高 |

### 品质体系

| 品质 | 边框色 | 基础属性范围 | 获取难度 |
|------|--------|:----------:|---------|
| Uncommon | 灰绿 | 60~75 | 普通 |
| Rare | 蓝银 | 70~85 | 较易 |
| Epic | 紫金 | 80~95 | 中等 |
| Legendary | 赤金 | 90~105 | 较难 |
| Mythic | 天命紫+流光 | 100~120 | 极难 |

### 战力计算 {#her-1-3}

```
单将战力 = (ATK × 2.0 + INT × 2.0 + CMD × 1.5 + POL × 1.0) × 等级系数 × 星级系数 × 装备系数
```

| 系数 | 公式 |
|------|------|
| 等级系数 | 1 + 等级 × 0.05 |
| 星级系数 | 1 + (星级 - 1) × 0.10 |
| 装备系数 | 1 + Σ 装备战力 / 1000 |

### 属性展示

| 平台 | 展示方式 |
|------|---------|
| PC 端 | 雷达图（260×260px），4 轴辐射，可叠加对比 |
| 手机端 | 条形图（每条 28px 高），阵营色渐变填充 |

## [HER-2] 武将招募 {#her-2}

### 招募方式

| 方式 | 消耗 | 每日免费 | 保底 |
|------|------|:-------:|------|
| 普通招贤（招贤榜） | 招贤榜 ×1 | 1 次/日 | 无 |
| 高级招贤（求贤令） | 求贤令 ×100 | 无 | 100 次必出 Legendary+ |

### 招募概率

| 品质 | 普通招贤 | 高级招贤 |
|------|:-------:|:-------:|
| Uncommon | 60% | 20% |
| Rare | 30% | 40% |
| Epic | 8% | 25% |
| Legendary | 2% | 13% |
| Mythic | 0% | 2% |

### 保底机制

```
高级招贤保底：
  - 每次高级招贤计数 +1
  - 计数满 100 必出 Legendary+ 武将
  - 出 Legendary 后计数器重置
  - UP 武将概率：出 Legendary 时 50% 为本期 UP 武将
```

### 重复武将处理

| 品质 | 转化碎片数 |
|------|:---------:|
| Uncommon | 5 片 |
| Rare | 10 片 |
| Epic | 20 片 |
| Legendary | 40 片 |
| Mythic | 80 片 |

## [HER-3] 武将升级 {#her-3}

### 经验获取

| 来源 | 经验值 |
|------|:-----:|
| 战斗通关 | 关卡等级 × 10 |
| 经验书（小） | 100 |
| 经验书（中） | 500 |
| 经验书（大） | 2,000 |
| 离线挂机 | 关卡基础经验 × 离线效率 |

### 升级消耗

| 等级范围 | 经验需求（每级） | 铜钱消耗 |
|---------|:--------------:|:-------:|
| 1~10 | 等级 × 50 | 等级 × 20 |
| 11~20 | 等级 × 120 | 等级 × 50 |
| 21~30 | 等级 × 250 | 等级 × 100 |
| 31~40 | 等级 × 500 | 等级 × 200 |
| 41~50 | 等级 × 1000 | 等级 × 400 |

### 突破系统

| 突破节点 | 等级要求 | 消耗 | 效果 |
|:-------:|:-------:|------|------|
| 突破 1 | Lv.10 | 碎片 ×20 + 铜钱 ×2000 | 全属性 +5%，解锁被动技能强化 |
| 突破 2 | Lv.20 | 碎片 ×40 + 铜钱 ×5000 | 全属性 +8%，解锁新技能 |
| 突破 3 | Lv.30 | 碎片 ×80 + 铜钱 ×12000 | 全属性 +12%，技能等级上限 +1 |
| 突破 4 | Lv.40 | 碎片 ×150 + 铜钱 ×30000 | 全属性 +15%，解锁终极技能强化 |

### 一键强化

系统自动选择材料，将武将强化到当前资源允许的最高等级：

```
流程：
  1. 系统计算资源上限可达目标等级
  2. 弹出预览：当前等级→目标等级、消耗明细、属性变化、战力变化
  3. 玩家确认 → 执行强化
  4. 强化动画：卡片金光闪烁 + 等级数字滚动递增
```

### 一键强化全部

```
优先级排序：
  1. 出战中的武将（优先强化主力）
  2. 战力最高的武将
  3. 稀有度最高的武将

流程：
  1. 按优先级排序所有可强化武将
  2. 弹出批量预览（前 5 个武将方案）
  3. 显示总资源消耗和总战力提升
  4. 确认后依次执行（每个间隔 200ms 动画）
```

### 批量升级

```
流程：
  1. 进入多选模式（勾选武将）
  2. 快捷按钮：全选 / 全选出战 / 全选未出战
  3. 弹出批量预览：每位武将升级方案 + 总资源消耗
  4. 资源不足时提示"仅够升级前 N 位"
  5. 确认后依次执行（每个间隔 150ms）
```

## [HER-4] 武将技能 {#her-4}

### 技能类型

| 类型 | 说明 | 触发方式 |
|------|------|---------|
| 主动技能（大招） | 强力技能，有 CD | 手动/自动释放 |
| 被动技能 | 持续生效的增益 | 自动 |
| 阵营技能 | 同阵营武将触发羁绊 | 条件触发 |
| 觉醒技能 | 突破后解锁 | 条件触发 |

### 技能升级 {#her-4-2}

| 等级 | 消耗 | 效果提升 |
|:----:|------|---------|
| 1→2 | 铜钱 ×500 | 基础伤害/治疗 +10% |
| 2→3 | 铜钱 ×1500 + 技能书 ×1 | 基础伤害/治疗 +10%，CD -1 |
| 3→4 | 铜钱 ×4000 + 技能书 ×3 | 基础伤害/治疗 +15% |
| 4→5 | 铜钱 ×10000 + 技能书 ×8 | 基础伤害/治疗 +15%，额外效果 |

### 阵营羁绊

| 羁绊 | 条件 | 效果 |
|------|------|------|
| 同乡之谊 | 2 同阵营 | 攻击 +5% |
| 同仇敌忾 | 3 同阵营 | 攻击 +15% |
| 众志成城 | 6 同阵营 | 攻击 +25% + 防御 +15% |
| 混搭协作 | 3+3 不同阵营 | 攻击 +10% + 特效加成 |

### 技能搭配推荐 {#her-4-5}

智能编队推荐系统根据关卡特性推荐最优技能组合：

| 敌方特性 | 推荐策略 |
|---------|---------|
| 多灼烧敌人 | 高智力 + 治疗型武将 |
| 多物理敌人 | 高防 + 坦克型武将 |
| BOSS 关 | 高爆发 + 控制型武将 |

## [HER-6] 武将编队 {#her-6}

### 编队规则

| 参数 | 规则 |
|------|------|
| 队伍人数 | 6 人（前排 3 + 后排 3） |
| 编队数量 | 最多 3 支编队 |
| 武将互斥 | 同一武将只能在一支编队中 |
| 出战编队 | 当前激活的编队用于战斗 |

### 编队操作

| 操作 | PC 端 | 手机端 |
|------|-------|--------|
| 部署武将 | 拖拽到阵型格 | 点击武将→点击空格 |
| 移除武将 | 拖出阵型格 / 右键移除 | 长按阵型格→移除 |
| 交换位置 | 拖拽互换 | 点击两个格子互换 |
| 一键布阵 | 按钮自动填充 | 按钮自动填充 |
| 智能推荐 | 弹出 1~3 套方案 | 弹出 1~3 套方案 |

### 红点提示系统

| 位置 | 红点条件 | 角标内容 |
|------|---------|---------|
| 武将卡片 | 材料足够升级 | 红点 |
| 武将卡片 | 碎片足够升星 | 红点 + 金色边框 |
| 武将卡片 | 新装备可穿戴 | 蓝点 |
| 武将 Tab | 有武将可升级 | 数字角标 |
| 武将 Tab | 有武将可升星 | 金色角标 |

### 今日待办聚合

武将列表顶部"今日待办"横幅，聚合所有待处理事项：

| 待办项 | 触发条件 | 快捷操作 |
|--------|---------|---------|
| N 位武将可升级 | 材料充足 | [一键强化] |
| N 位武将可升星 | 碎片充足 | [去升星] |
| N 件新装备可穿戴 | 有更好装备 | [去穿戴] |
| 招贤馆免费次数 | 每日未使用 | [去招募] |
