## [SET-1] 基础设置 {#set-1}

### 功能描述

基础设置提供语言、时区、通知三项全局配置，影响游戏内所有文本显示、时间计算和推送行为。

> 🎨 → [UI: 基础设置区](../ui-layout/SET-settings.md#set-1)

### 语言设置 {#set-1-1}

| 属性 | 规则 |
|------|------|
| 可选语言 | 简体中文 / 繁體中文 / English / 日本語 |
| 默认语言 | 跟随系统语言，无法识别时默认简体中文 |
| 切换生效 | 需重启游戏，切换前弹出确认：「切换语言需重启游戏，是否继续？」 |
| 切换冷却 | 无冷却，可随时切换 |
| 文本覆盖 | 所有UI文本、剧情对话、系统提示均跟随语言切换 |
| 未翻译处理 | 缺失翻译的文本回退至简体中文 |

### 时区设置 {#set-1-2}

| 属性 | 规则 |
|------|------|
| 默认时区 | 跟随设备时区 |
| 可选范围 | UTC-12 ~ UTC+14 全部时区 |
| 影响范围 | 日常任务重置时间（每日0点）、活动开始/结束时间、排行榜结算时间 |
| 切换生效 | 即时生效，无需重启 |
| 日常重置规则 | 日常任务在玩家所选时区的0点重置，非服务器时间 |

### 通知设置 {#set-1-3}

| 属性 | 规则 |
|------|------|
| 通知总开关 | 关闭后屏蔽所有推送通知 |
| 建筑完成通知 | 默认开启，建筑升级完成时推送 |
| 远征归来通知 | 默认开启，远征队返回时推送 |
| 活动提醒通知 | 默认开启，限时活动开始/即将结束时推送 |
| 好友消息通知 | 默认开启，好友私聊/借将请求时推送 |
| 联盟通知 | 默认开启，联盟战/联盟BOSS时推送 |
| 免打扰时段 | 可设置开始/结束时间，该时段内不推送（默认关闭） |
| 推送频率上限 | 同类型通知最多1条/5分钟，防止刷屏 |

## [SET-2] 音效设置 {#set-2}

### 功能描述

音效设置提供精细化的音频控制，包括主音量、分类音量、音效总开关等，所有设置即时生效。

> 🎨 → [UI: 音效设置区](../ui-layout/SET-settings.md#set-2)

### 音量控制

| 控制项 | 范围 | 默认值 | 步进 | 说明 |
|--------|------|--------|------|------|
| 主音量 | 0% ~ 100% | 80% | 5% | 总音量，影响所有音频输出 |
| BGM音量 | 0% ~ 100% | 60% | 5% | 背景音乐音量，受主音量约束 |
| 音效音量 | 0% ~ 100% | 70% | 5% | UI音效、战斗音效等，受主音量约束 |
| 语音音量 | 0% ~ 100% | 80% | 5% | 武将语音、剧情配音，受主音量约束 |

### 音量计算规则

```
实际输出音量 = 分类音量 × 主音量
示例：BGM实际音量 = 60% × 80% = 48%
```

### 开关控制

| 开关 | 默认 | 说明 |
|------|------|------|
| 音效总开关 | 开启 | 关闭后所有音频静音，等同于主音量=0% |
| BGM开关 | 开启 | 关闭后停止播放背景音乐 |
| 语音开关 | 开启 | 关闭后武将语音/剧情配音静音，仅保留字幕 |
| 战斗音效开关 | 开启 | 关闭后战斗场景音效静音 |

### 特殊规则

| 场景 | 规则 |
|------|------|
| 后台运行 | 游戏切至后台时BGM渐弱至静音（1s过渡），音效暂停 |
| 来电/闹钟 | 系统中断音频时立即静音，恢复后渐入（0.5s过渡） |
| 首次启动 | 首次进入游戏BGM延迟3秒播放，先展示画面 |
| 低电量模式 | 手机低电量（<20%）时自动降低BGM音量50%，音效不变 |

## [SET-3] 画面设置 {#set-3}

### 功能描述

画面设置提供多档画质预设和高级选项，支持自动检测设备性能并推荐合适的画质档位。

> 🎨 → [UI: 画质设置区](../ui-layout/SET-settings.md#set-3)

### 画质档位 {#set-1-4}

| 档位 | 粒子特效 | 实时阴影 | 水墨晕染 | 帧率 | 适用设备 |
|------|---------|---------|---------|------|---------|
| 低画质 | 关闭 | 关闭 | 关闭 | 30fps | 低端设备、省电模式 |
| 中画质 | 开启 | 关闭 | 开启 | 60fps | 中端设备（推荐默认） |
| 高画质 | 开启 | 开启 | 开启 | 60fps | 高端设备 |
| 自动 | 系统检测自动切换上述档位 | — | — | — | 所有设备 |

### 自动画质检测规则

```
检测逻辑（首次启动时执行）：
  CPU核心 ≥ 8 且 内存 ≥ 8GB → 高画质
  CPU核心 ≥ 4 且 内存 ≥ 4GB → 中画质
  其他 → 低画质
  
  检测结果标注「★ 推荐」朱砂印章
  玩家可手动覆盖自动推荐
```

### 高级画质选项

| 选项 | 类型 | 默认 | 说明 |
|------|------|------|------|
| 粒子特效 | 开关 | 开启 | 战斗/建筑产出粒子动画 |
| 实时阴影 | 开关 | 开启 | 建筑和角色的动态阴影 |
| 水墨晕染 | 开关 | 开启 | 面板过渡/转场水墨效果 |
| 帧率限制 | 选择：30/60/不限 | 60 | 最大渲染帧率 |
| 抗锯齿 | 开关 | 开启 | 边缘平滑处理 |

### 画质切换规则

| 属性 | 规则 |
|------|------|
| 切换生效 | 即时生效，全屏水墨晕染过渡（0.6s） |
| 切换冷却 | 无冷却 |
| 低端设备限制 | 低端设备自动隐藏高级选项，显示「当前设备不支持高级画质」 |
| 手机省电模式 | 开启后强制低画质+30fps，关闭后恢复原设置 |

### 手机端专属设置

| 设置项 | 类型 | 默认 | 说明 |
|--------|------|------|------|
| 省电模式 | 开关 | 关闭 | 降低帧率至30fps，关闭粒子特效 |
| 触控反馈 | 开关 | 开启 | 操作时震动反馈 |
| 左手模式 | 开关 | 关闭 | 镜像翻转UI布局 |
| 屏幕常亮 | 开关 | 关闭 | 游戏中保持屏幕不熄灭 |
| 字体大小 | 三档：小/中/大 | 中 | 调整游戏内文字大小 |

## [SET-4] 账号设置 {#set-4}

### 功能描述

账号设置管理玩家账号的绑定、云存档、设备管理和账号删除等核心数据操作。

> 🎨 → [UI: 账号设置区](../ui-layout/SET-settings.md#set-4)

### 账号绑定 {#set-1-5}

| 属性 | 规则 |
|------|------|
| 绑定方式 | 手机号 / 邮箱 / 第三方（微信/QQ/Apple ID） |
| 绑定数量 | 每个账号最多绑定1个手机号 + 1个邮箱 + 1个第三方 |
| 绑定奖励 | 首次绑定任意方式奖励元宝×50 |
| 换绑冷却 | 解绑后7天内不可重新绑定同类型 |
| 游客账号 | 未绑定的游客账号30天后自动清除 |

### 云存档设置

| 设置项 | 默认 | 说明 |
|--------|------|------|
| 云存档开关 | 开启 | 登录账号后自动开启 |
| 自动同步 | 开启 | 每次退出游戏自动上传 |
| 同步频率 | 退出时 | 可选：退出时/每小时/仅手动 |
| WiFi仅同步 | 关闭 | 仅在WiFi下自动同步 |
| 存档加密 | 开启 | 端到端加密存档数据 |

### 多设备管理

| 属性 | 规则 |
|------|------|
| 最大绑定设备 | 5台 |
| 主力设备 | 标记1台为主力设备，优先同步 |
| 解绑冷却 | 解绑设备后24小时内不可再次绑定新设备 |
| 冲突解决策略 | 可选：始终保留最新 / 始终保留云端 / 始终询问（默认） |

### 存档管理

| 属性 | 规则 |
|------|------|
| 存档槽位 | 3个免费 + 1个付费解锁（200元宝） |
| 自动存档 | 默认开启，每15分钟自动保存 |
| 存档内容 | 主城等级、资源、武将、科技、建筑等全量数据 |
| 存档大小 | 约2-5MB/槽位 |

### 账号删除

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 输入确认文字 | 输入「确认删除」四个字 |
| 2 | 二次确认弹窗 | 显示数据删除范围和不可逆警告 |
| 3 | 7天冷静期 | 确认后账号进入7天冷静期，期间可随时撤销 |
| 4 | 永久删除 | 冷静期结束后永久删除所有数据，不可恢复 |

### 退出登录

| 属性 | 规则 |
|------|------|
| 退出前保存 | 自动触发云存档上传 |
| 本地数据 | 退出后本地缓存保留，下次登录可快速恢复 |
| 退出冷却 | 无冷却 |

## 全局规则 {#global-rules}

### 设置持久化

| 规则 | 说明 |
|------|------|
| 保存时机 | 每次修改设置项后立即保存至本地存储 |
| 云端同步 | 设置项随云存档一起同步到云端 |
| 冲突处理 | 多设备设置冲突时，以最新修改时间为准 |
| 恢复默认 | 「恢复默认」按钮重置当前分类下所有设置项 |

### 首次启动检测

| 检测项 | 时机 | 说明 |
|--------|------|------|
| 语言检测 | 首次启动 | 自动识别系统语言并设置 |
| 画质检测 | 首次启动 | 检测设备性能推荐画质档位 |
| 权限申请 | 首次启动 | 弹出数据授权面板（存储/网络/通知/定位） |
| 新手引导 | 首次启动 | 自动触发6步核心引导流程 |
