## 5. 算法改进记录

### 5.1 BFS路径枚举算法增强

**改进**: 增加 `allowBackEdges` 参数支持回退边枚举

```typescript
// R2改进前
function enumeratePaths(nodes, edges, entryPoints, maxDepth, maxPaths)

// R2改进后
function enumeratePaths(nodes, edges, entryPoints, maxDepth, maxPaths, allowBackEdges = false)
```

**效果**:
- StateDAG路径从30条增加到46条（+53%）
- 覆盖了15条状态回退边
- 不影响其他4类DAG的枚举结果

### 5.2 覆盖率匹配算法

**已修复（R1之前）**: 冒号分割支持
- 旧算法: `split(/[-_]/)` — 不支持StateDAG的 `entity:state` 格式
- 新算法: `split(/[-_:]/)` — 支持冒号分割
- 效果: StateDAG节点覆盖率从47.7%提升至100%
