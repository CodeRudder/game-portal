# ASCII 地图数据格式规范

> **版本**: v1.0 | **日期**: 2026-05-02

## 设计原则

- **数据层**: ASCII文本，人类可读可编辑
- **渲染层**: 像素风格，8×8色块渲染
- **通用性**: 同一渲染器支持多种地图(天下/城池/副本)

## 符号定义

### 地形符号

| 符号 | 地形 | 像素渲染 | 色值 |
|:----:|------|---------|------|
| `.` | 平原 | 绿色渐变+草地纹理 | #7ec850 |
| `^` | 山地 | 棕灰色三角+阴影 | #8b7355 |
| `~` | 水域 | 蓝色渐变+波浪 | #2d6b8a |
| `#` | 森林 | 深绿树冠+树干 | #2e5a2e |
| `=` | 关隘 | 灰墙+狭窄通道 | #8b6914 |
| `-` | 道路(横) | 土黄色3px路径 | #b8a07a |
| `\|` | 道路(竖) | 土黄色3px路径 | #b8a07a |
| `/` | 道路(斜) | 土黄色3px路径 | #b8a07a |
| `+` | 道路(交叉) | 土黄色交叉路口 | #b8a07a |
| `:` | 小路 | 土黄色2px窄路 | #9a8a6a |
| `*` | 沙漠 | 黄褐色+沙粒 | #d4b896 |
| `,` | 草地 | 浅绿色稀疏纹理 | #a8d88a |
| `_` | 泥地 | 深褐色 | #6b5b3e |

### 城市符号

| 符号 | 含义 | 像素渲染 |
|:----:|------|---------|
| `A`~`Z` | 城市(字母=城市ID映射) | 16×16城池图标+阵营旗 |
| `a`~`z` | 资源点/小型据点 | 8×8资源图标 |
| `0`~`9` | 关卡/哨站 | 12×12关隘图标 |

### 装饰符号

| 符号 | 含义 | 像素渲染 |
|:----:|------|---------|
| ` ` (空格) | 空地/未探索 | 黑色/雾效 |
| `@` | 玩家位置 | 闪烁标记 |
| `!` | 事件点 | 脉冲图标 |
| `?` | 未知区域 | 迷雾 |
| `%` | 废墟 | 残垣断壁 |
| `&` | 宝箱 | 发光宝箱 |
| `$` | 商队 | 驮马精灵 |

### 轮廓/边框符号

| 符号 | 含义 | 像素渲染 |
|:----:|------|---------|
| `┌` `┐` `└` `┘` | 城墙角 | 灰色城墙转角 |
| `─` | 城墙(横) | 灰色城墙横段 |
| `│` | 城墙(竖) | 灰色城墙竖段 |
| `├` `┤` `┬` `┴` | 城墙T型 | 灰色城墙交叉 |
| `┼` | 城墙十字 | 灰色城墙十字 |

## 地图文件格式

```
# 地图名称
MAP: 天下大地图
SIZE: 80x60
TILE: 8x8

# 城市映射表
CITY: L=洛阳, C=长安, X=许昌, Y=邺城, ...
CITY: G=成都, H=汉中, A=永安, N=南中, ...
CITY: J=建业, K=会稽, S=柴桑, B=庐江, ...

# 地图数据(每行=地图的一行)
# 注释行以#开头
........................................
.........^^^^....................^^^^...
........^^^^^^..................^^^^^^..
.......^^Y^^...................^^B^^...
......^^^---^^................^^^---^^..
.....^^---X--^^..............^^---S--^^.

....^^---------^^............^^---------^^

........^^^^....................^^^^.....
.......^^^G^^...................^^^......

.....^^^--H--^^^.........................
......^^^---^^^..........................
.......^^^A^^............................
........^^^..............................
.........N...............................

```

## 多地图支持

每张地图是一个独立的ASCII文件:

```
src/games/three-kingdoms/core/map/maps/
  world-map.txt        # 天下大地图(80×60)
  city-xuchang.txt     # 许昌城内(20×15)
  city-chengdu.txt     # 成都城内(20×15)
  outskirts-wei.txt    # 魏国郊外(40×30)
  dungeon-ruins.txt    # 遗迹副本(20×20)
```

## 渲染器工作流

```
ASCII文本文件 → 解析器 → 地图数据矩阵 → 像素渲染器 → Canvas输出
                  ↓
            符号→地形映射
            城市ID→城市数据
            道路符号→路径计算
```
