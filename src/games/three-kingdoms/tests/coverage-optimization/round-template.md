## 7. 自检清单

- [ ] 每个新增测试文件的 happy path 已覆盖

- [ ] 测试命名清晰，能从名称推断测试意图
- [ ] 无跳过的测试 (skip/todo)
- [ ] 无硬编码的魔法数字
- [ ] Mock/Stub 使用合理，不过度 mock
- [ ] 集成测试覆盖了完整的业务流程
- [ ] 断言充分，验证了所有关键输出和副作用
