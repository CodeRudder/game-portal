### 🟡 设计问题 (建议改进)

| ID | 严重度 | 描述 | 位置 |
|----|--------|------|------|
| DES-1 | 低 | **OfflineRewardSystem.applyDouble 不跟踪 ad source 次数** — 仅 VIP source 会递增计数器，ad source 无内部计数。需依赖 OfflineSnapshotSystem.recordAdDouble() 外部管理 | `OfflineRewardSystem.ts:120-130` |
| DES-2 | 低 | **OfflineSnapshotSystem 未从 index.ts 导出** — 外部需直接 import 文件路径 | `index.ts` |
| DES-3 | 低 | **OFFLINE_POPUP_THRESHOLD 未从 index.ts 导出** — 需从 offline-config 单独导入 | `index.ts` |
| DES-4 | 低 | **Play文档§1.2说6档但实现只有5档** — 文档说">72h→15%(封底值)"但代码中无第6档(>72h=0%)。72h后收益直接封顶不新增，与文档描述不完全一致 | `offline-config.ts` |
