# 五、关键经验教训

## 经验 1: P0模式库是最具成本效益的投入

**场景**: Phase 2 的 building/quest/formation 三个模块均未发现P0缺陷，而Phase 0的3个模块共发现18个P0。

**数据支撑**:
- Phase 0: 3模块 × 平均3.7轮 = 11轮，发现18个P0
- Phase 2: 3模块 × 平均2.7轮 = 8轮，发现0个P0
