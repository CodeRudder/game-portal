## 一、招贤馆系统 [BLD-F23]

> **来源**: BLD-6 招贤馆设计
> **核心功能**: 武将招募+概率加成，连接英雄子系统

### BLD-F23-01 武将招募流程

**流程步骤**:
1. [玩家操作] 点击招贤馆→进入招募面板
2. [面板展示] 显示三种招募类型:
   - 普通招募: 消耗铜钱5,000
   - 高级招募: 消耗招贤令×1 + 铜钱50,000
   - 友情招募: 消耗友情点×100
3. [概率计算] 招募概率 = 基础概率 × (1 + 招贤馆等级加成) × (1 + 科技加成) × (1 + 武将加成)
4. [招募执行] 随机抽取武将→品质判定(白/绿/蓝/紫/橙)
5. [结果展示] 展示招募结果→武将信息→确认收下
6. [数据记录] 武将进入英雄系统→招募记录+1

**概率加成公式**:
```
实际概率 = 基础概率 × (1 + 招贤馆等级 × 2%) × (1 + 科技「求贤若渴」5%) × (1 + 武将智力%)
```

### BLD-F23-02 招募保底机制

**流程步骤**:
1. [计数器] 每次高级招募计数器+1
2. [保底检查] 计数器≥50时触发保底
3. [保底执行] 必出紫色及以上品质武将
4. [计数器重置] 保底触发后计数器归零
5. [UI提示] 显示"距保底还剩X次"

### BLD-F23-03 招贤馆升级流程

**流程步骤**:
1. [玩家操作] 点击升级按钮
2. [校验] 检查资源/等级/队列
3. [扣费] 扣除粮草+铜钱
4. [升级执行] 等级+1→概率加成更新
5. [解锁检查] 检查新功能解锁(高级招募/友情招募/保底)
6. [UI反馈] Toast提示+外观变化

## 二、工坊系统 [BLD-F24]

> **来源**: BLD-6 工坊设计
> **核心功能**: 装备炼制+效率加成，连接装备子系统

### BLD-F24-01 装备炼制流程

**流程步骤**:
1. [玩家操作] 点击工坊→进入炼制面板
2. [材料检查] 检查铁匠铺产出的材料是否充足
3. [炼制选择] 选择炼制类型:
   - 快速炼制: 消耗材料×10 → 绿色装备×1
   - 精炼: 消耗材料×30 → 蓝色装备×1
   - 神炼: 消耗材料×100 → 紫色装备×1 (需Lv10+)
4. [效率计算] 炼制效率 = 基础效率 × (1 + 工坊等级加成) × (1 + 科技加成)
5. [炼制执行] 消耗材料→产出装备→品质判定
6. [结果展示] 展示装备属性→确认收下

**效率加成公式**:
```
实际效率 = 基础效率 × (1 + 工坊等级 × 3%) × (1 + 科技「冶炼工艺」10%) × (1 + 武将武力%)
```

### BLD-F24-02 批量炼制流程

**解锁条件**: 工坊Lv10

**流程步骤**:
1. [玩家操作] 点击批量炼制按钮
2. [批量选择] 选择炼制数量(10/50/100)
3. [材料检查] 检查总材料是否充足
4. [批量执行] 一次性消耗材料→批量产出装备
5. [结果汇总] 展示总产出→品质分布→确认

### BLD-F24-03 装备分解流程

**流程步骤**:
1. [玩家操作] 点击装备分解按钮
2. [装备选择] 选择要分解的装备(可多选)
3. [回收率计算] 回收率 = 基础60% + 工坊等级 × 1.5%
4. [分解执行] 消耗装备→回收材料
5. [结果展示] 展示回收材料数量→确认

### BLD-F24-04 工坊升级流程

**流程步骤**:
1. [玩家操作] 点击升级按钮
2. [校验] 检查资源/等级/队列
3. [扣费] 扣除粮草+铜钱
4. [升级执行] 等级+1→效率加成更新
5. [解锁检查] 检查新功能解锁(批量炼制/高品质/分解回收率)
6. [UI反馈] Toast提示+外观变化

## 三、市舶司系统 [BLD-F25]

> **来源**: BLD-6 市舶司设计
> **核心功能**: 贸易管理+商队+繁荣度，连接贸易子系统

### BLD-F25-01 商队派遣流程

**流程步骤**:
1. [玩家操作] 点击市舶司→进入商队面板
2. [商队状态] 显示当前商队(最多3支):
   - 商队1: 空闲/派遣中/返回中
   - 商队2: 🔒 Lv5解锁
   - 商队3: 🔒 Lv10解锁
3. [派遣选择] 选择目的地(不同目的地收益不同):
   - 近郊: 5分钟，铜钱×1,000
   - 城镇: 15分钟，铜钱×3,000 + 繁荣度×50
   - 都城: 30分钟，铜钱×8,000 + 繁荣度×150 + 随机道具
4. [派遣执行] 消耗粮草(派遣费用)→商队出发
5. [定时收益] 商队返回后自动结算→铜钱+繁荣度+道具

### BLD-F25-02 繁荣度事件流程

**触发条件**: 繁荣度达到阈值(2,000/5,000/10,000)

**流程步骤**:
1. [触发检查] 繁荣度≥阈值时触发
2. [事件生成] 从事件池随机选择贸易事件
3. [事件展示] 显示事件描述+双向收益选择:
   - 选项A: 即时收益(铜钱/道具)
   - 选项B: 持续收益(繁荣度加成/折扣提升)
4. [玩家选择] 选择A或B
5. [结果执行] 根据选择发放奖励
6. [繁荣度重置] 繁荣度归零，重新积累

### BLD-F25-03 贸易折扣流程

**折扣计算公式**:
```
实际折扣 = 基础折扣(市舶司等级) + 科技「航海术」3% + 武将政治%
```

**折扣应用场景**:
- 商店购买: 价格 × (1 - 折扣%)
- 资源交易: 交易价格 × (1 - 折扣%)
- 招募费用: 招募费用 × (1 - 折扣% / 2)

### BLD-F25-04 跨服贸易流程

**解锁条件**: 市舶司Lv15

**流程步骤**:
1. [玩家操作] 点击跨服贸易按钮
2. [服务器选择] 显示可贸易服务器列表
3. [商品选择] 选择要出售/购买的商品
4. [价格设定] 设定出售价格(参考市场均价)
5. [挂单执行] 商品上架→等待其他玩家购买
6. [交易完成] 商品售出→获得跨服货币→兑换稀有道具

### BLD-F25-05 市舶司升级流程

**流程步骤**:
1. [玩家操作] 点击升级按钮
2. [校验] 检查资源/等级/队列
3. [扣费] 扣除粮草+铜钱
4. [升级执行] 等级+1→折扣+繁荣度更新
5. [解锁检查] 检查新功能解锁(高级商队/跨服贸易)
6. [UI反馈] Toast提示+外观变化

## 四、铁匠铺重设计 [BLD-F26]

> **来源**: BLD-8 资源链设计
> **核心变更**: 从"产出材料无消费端"变为"材料加工→工坊消费"的装备链

### BLD-F26-01 材料加工流程(重设计)

**旧设计**: 铁匠铺产出材料→无消费端→材料堆积

**新设计**: 铁匠铺产出材料→工坊消费→装备炼制

**流程步骤**:
1. [定时产出] 铁匠铺每小时产出材料(受等级影响)
2. [材料存储] 材料存入材料仓库(容量=产出速率×24小时)
3. [工坊消费] 工坊炼制时消耗材料
4. [联动加成] 工坊等级影响铁匠铺材料消耗折扣(+1.5%/级)

### BLD-F26-02 强化折扣流程(新增)

**流程步骤**:
1. [玩家操作] 在装备系统中选择强化
2. [折扣计算] 铁匠铺等级提供强化折扣(每级-1%材料消耗)
3. [强化执行] 消耗材料(已折扣)→装备属性提升
4. [联动效果] 铁匠铺等级越高，强化越省钱

## 五、兵营编队系统 [BLD-F27]

> **来源**: BLD-6 兵营扩展设计
> **核心功能**: 编队管理+训练模式，连接战斗子系统

### BLD-F27-01 编队管理流程

**流程步骤**:
1. [玩家操作] 点击兵营→进入编队面板
2. [编队列表] 显示最多3支编队:
   - 编队1: 默认解锁
   - 编队2: 🔒 兵营Lv10解锁
   - 编队3: 🔒 兵营Lv20解锁
3. [编队配置] 每支编队可配置:
   - 主将: 选择武将(影响编队属性)
   - 兵力: 分配兵力(受兵营产出限制)
   - 兵种: 步兵/骑兵/弓兵(有克制关系)
4. [编队保存] 保存配置→编队可用

### BLD-F27-02 训练模式流程

**流程步骤**:
1. [玩家操作] 点击训练按钮
2. [训练选择] 选择训练类型:
   - 普通训练: 消耗粮草，训练速度+0%
   - 加速训练: 消耗粮草×2，训练速度+50%
   - 精英训练: 消耗粮草×5 + 铜钱×1000，训练速度+100%
3. [训练执行] 兵力开始训练→进度条显示
4. [训练完成] 兵力+1→可分配到编队

**训练速度公式**:
```
实际训练速度 = 基础速度 × (1 + 兵营等级 × 2%) × (1 + 招贤馆等级 × 1%) × (1 + 武将统率%)
```

### BLD-F27-03 编队出征流程

**流程步骤**:
1. [玩家操作] 在地图系统中选择征服/攻城
2. [编队选择] 选择要出征的编队
3. [兵力检查] 检查编队兵力是否充足
4. [出征执行] 编队出发→战斗结算
5. [战后处理] 伤兵→医馆恢复；战利品→资源系统

## 六、资源链循环 [BLD-F28]

> **来源**: BLD-8 资源链设计
> **核心功能**: 打通建筑间资源流转，消除僵尸建筑

### BLD-F28-01 粮草→兵力→战斗链

**链路**: 农田(粮草) → 兵营(兵力) → 战斗(战利品)

**流程步骤**:
1. [产出] 农田每秒产出粮草
2. [消费] 兵营训练消耗粮草
3. [转化] 粮草转化为兵力
4. [消费] 战斗消耗兵力
5. [回流] 战斗获得战利品(资源/材料/道具)

### BLD-F28-02 铜钱→材料→装备链

**链路**: 市集(铜钱) → 铁匠铺(材料) → 工坊(装备) → 英雄

**流程步骤**:
1. [产出] 市集每秒产出铜钱
2. [消费] 铁匠铺升级/运转消耗铜钱
3. [产出] 铁匠铺每小时产出材料
4. [消费] 工坊炼制消耗材料
5. [产出] 工坊产出装备
6. [消费] 英雄装备→战力提升

### BLD-F28-03 铜钱→贸易→折扣链

**链路**: 市集(铜钱) → 市舶司(贸易) → 商店(折扣)

**流程步骤**:
1. [产出] 市集每秒产出铜钱
2. [消费] 市舶司商队派遣消耗粮草
3. [产出] 商队返回产出铜钱+繁荣度
4. [效果] 繁荣度→市集铜钱产出加成
5. [效果] 贸易折扣→所有购买价格降低

### BLD-F28-04 科技点→科技→全建筑加成链

**链路**: 书院(科技点) → 科技(研究) → 全建筑(加成回流)

**流程步骤**:
1. [产出] 书院每秒产出科技点
2. [消费] 科技研究消耗科技点
3. [效果] 科技完成→全建筑加成
4. [回流] 加成→所有建筑产出提升

### BLD-F28-05 粮草→招募→英雄链

**链路**: 农田(粮草) → 招贤馆(招募) → 英雄(战力)

**流程步骤**:
1. [产出] 农田每秒产出粮草
2. [消费] 招募消耗铜钱(市集产出)
3. [产出] 招募获得武将
4. [消费] 武将派驻建筑→产出加成
5. [效果] 武将出征→战斗胜利→战利品
