## 二、迭代计划

### Sprint 1: 核心资源流(P0)

**目标**: 4种资源产出→消费链路完整闭环

| 任务 | 流程 | 内容 | 验收标准 |
|------|------|------|---------|
| 1.1 | BLD-F01 | 资源产出公式适配4资源 | 农田/市集/矿场/伐木场均按公式产出 |
| 1.2 | BLD-F02 | 升级消耗适配4资源 | Lv1~5仅粮草+铜钱，Lv6+引入矿石/木材 |
| 1.3 | BLD-F26 | 矿场/伐木场持续产出 | tick产出→库存→溢出降速50% |
| 1.4 | BLD-F07 | 离线收益适配4资源 | 离线期间4种资源均正常产出 |
| 1.5 | BLD-F10 | 一键收取适配4资源 | 收取时4种资源飞入资源栏 |
| 1.6 | BLD-F15 | 资源上限适配4资源 | 4种资源各自独立上限 |

**测试用例**:
- [ ] 矿场Lv5产出=农田Lv5产出(基础速率一致)
- [ ] 建筑升级Lv6消耗包含矿石
- [ ] 离线8h后4种资源均正确累加
- [ ] 一键收取后4种资源归零
- [ ] 资源达上限后产出降速50%

### Sprint 2: 工坊装备系统(P1)

**目标**: 装备锻造→强化→穿戴→英雄属性完整链路

| 任务 | 流程 | 内容 | 验收标准 |
|------|------|------|---------|
| 2.1 | BLD-F24-01 | 装备锻造 | 矿石+木材+铜钱→装备(品质判定) |
| 2.2 | BLD-F24-01b | 装备强化 | 消耗矿石→属性提升(含折扣) |
| 2.3 | BLD-F24-02 | 批量锻造 | Lv10解锁→一次锻造多件 |
| 2.4 | BLD-F24-03 | 装备分解 | 装备→回收矿石(回收率受等级影响) |
| 2.5 | BLD-F24-05 | 装备穿戴 | 装备绑定武将→属性加成注入 |
| 2.6 | XI-013 | EQP→HER | 装备属性→武将属性→建筑加成/战斗属性 |
| 2.7 | XI-009 | BLD→EQP | 工坊等级→锻造效率/强化折扣 |

**测试用例**:
- [ ] 锻造消耗矿石×10+木材×10+铜钱×1000→产出绿色装备
- [ ] 强化消耗矿石(含工坊折扣)→装备属性+5%/级
- [ ] 批量锻造10件→材料消耗×10→产出10件装备
- [ ] 分解绿色装备→回收矿石×5(50%回收率)
- [ ] 穿戴装备→武将攻击+10%→派驻建筑产出+10%
- [ ] 工坊Lv10→强化折扣-10%→实际消耗减少

## 三、评测标准

### 3.1 功能完整性

| 等级 | 标准 |
|:----:|------|
| A | 30主流程+124子流程+16跨系统+46触发事件全部实现 |
| B | 核心流程(P0)全部实现，P1/P2完成80%+ |
| C | 核心流程(P0)全部实现，P1完成60%+ |
| D | 核心流程(P0)有缺失 |

### 3.2 数据一致性

| 检查项 | 标准 |
|--------|------|
| 资源产出 | 4种资源产出公式一致，加成叠加正确 |
| 升级消耗 | 消耗梯度正确(粮草+铜钱→+矿石→+木材) |
| 跨系统数据 | XI-001~016数据流转无丢失/错误 |
| 离线收益 | 离线期间产出与在线一致 |

### 3.3 性能指标

| 指标 | 标准 |
|------|------|
| tick帧率 | 60fps(每帧资源计算<1ms) |
| 离线计算 | 8h离线结算<100ms |
| 内存占用 | 建筑系统<50MB |
