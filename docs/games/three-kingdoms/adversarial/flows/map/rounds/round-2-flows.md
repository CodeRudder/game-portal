## R2 新增/补充流程

### MAP-F04-01 征服失败恢复机制 {#map-f04-01}

**玩家视角**: 征服失败 → 损失兵力 → "重整旗鼓" → 降低下次出征消耗

**流程步骤**:
1. [前置] 征服失败(MAP-F04 Step 12)
2. [UI反馈] 失败弹窗增加"重整旗鼓"按钮
3. [玩家操作] 点击"重整旗鼓"
4. [系统执行] 消耗铜钱×1000，获得"重整"BUFF(持续1h)
5. [BUFF效果] 下次出征该领土消耗降低20%
6. [UI反馈] 领土标记"重整中"图标

**数据变更**:
| 数据 | 变更前 | 变更后 | 来源 |
|------|--------|--------|------|
| resources.gold | X | X-1000 | 重整消耗 |
| buff.rally | 无 | {target, expire} | 系统添加 |
| conquestCost.reduction | 0% | 20% | BUFF生效 |

### MAP-F06-01 攻城中断处理 {#map-f06-01}

**玩家视角**: 攻城中途退出 → 攻城暂停 → 返回后继续

**流程步骤**:
1. [前置] 攻城进行中(MAP-F06 Step 6)
2. [玩家操作] 退出天下Tab/关闭游戏
3. [系统执行] 攻城状态暂停，记录当前城防值
4. [玩家操作] 重新进入天下Tab
5. [系统检查] 检查是否有进行中的攻城
6. [UI反馈] 弹出"攻城继续"提示，显示当前进度
7. [玩家操作] 点击"继续攻城"
8. [系统执行] 从暂停点继续攻城

**取消攻城**:
1. [玩家操作] 点击"取消攻城"
2. [系统执行] 攻城取消，不退还已消耗资源
3. [UI反馈] 显示"攻城已取消"提示

### MAP-F06-02 攻城策略选项 {#map-f06-02}

**玩家视角**: 选择攻城策略 → 不同消耗/时间/成功率 → 执行攻城

**策略选项**:
| 策略 | 消耗倍率 | 时间倍率 | 成功率修正 | 特殊效果 |
|------|---------|---------|-----------|---------|
| 强攻 | ×1.5 | ×0.5 | -10% | 城防损坏(占领后城防-50%) |
| 围困 | ×0.8 | ×2.0 | +10% | 民心下降(占领后产出-20%持续24h) |
| 夜袭 | ×1.2 | ×0.8 | +5% | 需要"夜袭令"道具 |
| 内应 | ×1.0 | ×1.0 | +20% | 需要"内应信"道具+好感度达标 |

**流程步骤**:
1. [前置] 攻城确认弹窗(MAP-F06 Step 4)
2. [UI反馈] 弹窗增加策略选择区域(4个策略卡片)
3. [玩家操作] 选择策略
4. [系统计算] 根据策略调整消耗/时间/成功率
5. [UI反馈] 更新预估信息
6. [玩家操作] 确认攻城
7. [系统执行] 按策略执行攻城

### MAP-F05-01 领土放弃机制 {#map-f05-01}

**玩家视角**: 选择放弃 → 确认 → 损失驻防 → 领土变中立

**流程步骤**:
1. [玩家操作] 点击己方领土，点击"放弃"按钮
2. [UI反馈] 确认弹窗: "放弃后将损失所有驻防兵力，领土变为中立。确认放弃？"
3. [玩家操作] 点击"确认放弃"
4. [系统执行] 损失驻防兵力，领土变为中立
5. [系统触发] 24h冷却期(不可立即重新占领)
6. [UI反馈] 领土变为中立色，标记"已放弃"

**数据变更**:
| 数据 | 变更前 | 变更后 | 来源 |
|------|--------|--------|------|
| territory.owner | 己方 | 中立 | 放弃操作 |
| territory.garrison | X | 0 | 损失驻防 |
| resources.troops(可用) | X | X+原驻防 | 兵力回收(损失50%) |

### MAP-F07-01 筛选器实现逻辑 {#map-f07-01}

**"高产领地"筛选**:
1. 计算每个领土总产出 = 基础×地形×阵营×科技×声望×地标
2. 按总产出降序排序
3. 高亮前5名，其他灰度显示

**"可征服"筛选**:
1. 遍历所有非己方领土
2. 检查是否与己方领土相邻
3. 检查可用兵力是否≥驻防×1.5(非城池)/×2.0(城池)
4. 显示满足条件的领土

**热力图分档算法**:
1. 计算所有领土总产出
2. 排序后按百分位分5档:
   - Top 20%: 深金 #C9A84C
   - 20-40%: 浅金 #E8D48B
   - 40-60%: 翠绿 #7EC850
   - 60-80%: 浅绿 #A8D88A
   - Bottom 20%: 灰绿 #6B8B6B

### MAP-F04-02 胜率公式完善 {#map-f04-02}

**完善后的胜率公式**:
```
胜率 = min(95%, max(5%, (我方战力/敌方战力)×50% + 地形修正))

地形修正:
  平原: 0%
  山地: -10%(防守方+10%)
  水域: 0%
  城池: -20%(城防加成)
  森林: +5%(伏兵加成)
  关隘: -30%(一夫当关)
```
