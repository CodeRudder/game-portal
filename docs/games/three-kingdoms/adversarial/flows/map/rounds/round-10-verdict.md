### R10新增方法论: "三方交叉校验"

R10在模拟产出流程中增加"三方交叉校验"步骤:

```
R10模拟产出验证流程(五步法):
  1. 行为描述: 定义玩家行为(登录频率/事件参与/获取来源)
  2. 数值推算: 从行为描述推算每日/每周净变化(含衰减/豁免计算)
  3. 模拟表: 运行30天/60天模拟
  4. 三方校验: 模拟表结果必须与数值推算一致(误差<10%)
  5. 如不一致: 检查推算假设(衰减计算/豁免判定/per-faction语义)
```

**方法论评价**: 这是R10最重要的方法论贡献，直接解决了R9"模拟数值不可信"的核心缺陷。R9的休闲玩家-3/天矛盾(P0)如果在产出前执行了三方校验，会被立即发现。R10的8类玩家逐类三方校验是该方法论的最佳实践——每类玩家的行为描述、数值推算、模拟表三方一致，使质疑者能独立验证任何一类玩家的数值。

**方法论的局限性**: "三方交叉校验"解决了"模拟数值可信度"问题，但R10的期望值表(CH-MAP-R10-01)暴露了一个缺口——三方校验仅应用于声望模拟，未应用于期望值分析。**R11建议**: 将三方校验扩展到所有数值产出(不仅是模拟表)，包括期望值表、样本量计算等。

### R10新增方法论: "沉没成本检查"

R10在决策分析流程中增加"沉没成本检查"步骤:

```
R10决策分析验证流程:
  1. 明确决策点: "是否使用重试令牌"
  2. 列出相关成本: 重试胜利损耗、重试失败损耗、令牌价值
  3. 沉没成本检查: 首次失败损耗是否已发生? -> 是，则排除
  4. 计算期望值: E(重试) = P(胜利) x (奖励 - 重试胜利损耗) + P(失败) x (-重试失败损耗) - 令牌价值
  5. 决策: 若E(重试) > E(不重试) = 0，则重试
```

**方法论评价**: 这是R10在经济学方法论上的重要贡献。R9的期望值公式纳入沉没成本(首次失败损耗)是经典经济学错误，R10通过"沉没成本检查"步骤从方法论层面防止了同类错误的再次发生。该步骤具有通用性——任何涉及"是否继续投入"的决策分析(重试令牌/资源追加/项目继续)都应执行沉没成本检查。

### R10新增方法论: "A/B测试方法论模板"

R10建立A/B测试方案模板(14要素):

```
R10 A/B测试方案模板:
  1. 测试假设(H0/H1)
  2. 测试组设计(对照组+实验组)
  3. 分流策略(方式/比例/时机)
  4. 样本量计算(公式/结果/验证)
  5. 测试持续时间(最短/建议/提前终止条件)
  6. 主要成功指标(指标/判定标准)
  7. 次要指标(辅助判断)
  8. 统计分析方法(检验/显著性/多重比较)
  9. 效应量评估(方法/阈值)
  10. 决策规则(场景/行动)
  11. 验证失败回退方案
```

**方法论评价**: 这是R10在验证方法论上的重要贡献。R9的A/B测试方案仅列出3种k值和4个指标，缺少样本量、持续时间、统计方法等关键要素——无法作为可执行的测试方案。R10的14要素模板从"方案骨架"升级为"可执行方案"，可复用于任何需要A/B测试验证的设计假设。

**方法论的局限性**: 模板的"主要成功指标"选择需要领域知识——R10选择了"战斗参与率"但逻辑方向存疑(CH-MAP-R10-05)。**R11建议**: 在模板中增加"指标方向验证"步骤——明确预期方向(哪个组应更高)并给出理论依据。
