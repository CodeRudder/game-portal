## R3 新增/补充流程

### MAP-F01-02 产出上限机制 {#map-f01-02}

**统一方案**: 与建筑系统产出上限统一

**流程步骤**:
1. [系统计算] 领土产出上限 = 领土等级 × 基础上限系数
2. [系统检查] 当前资源 ≥ 上限 → 产出停止累加
3. [UI反馈] 产出气泡显示"已满"标识
4. [玩家操作] 消耗资源后产出恢复累加

**上限公式**:
```
领土产出上限 = 基础上限(10000) × 领土等级 × (1 + 科技加成)
```

### MAP-F06-05 攻城锁定机制 {#map-f06-05}

**规则**:
1. 攻城开始后，目标领土锁定(其他玩家不可攻)
2. 攻城暂停时，锁定持续2h
3. 超时自动取消攻城，锁定解除

### XI-MAP-09 产出上限→资源系统

**涉及系统**: MAP → RES
**衔接点**: MAP-F01-02(产出上限) → RES-F01(资源累积)
