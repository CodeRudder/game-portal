## 四、公式验证检查

### 4.1 产出公式

```
领土产出 = 基础产出 × 地形 × 阵营 × 科技 × 声望 × 等级倍率
```

- [ ] 6种地形加成正确(平原1.0/山地0.8/水域0.9/城池1.2/森林1.1/关隘0.7)
- [ ] 3种阵营加成正确(己方1.0/中立0.5/敌方0.0)
- [ ] 7项科技加成正确(每级+1.5%/+5%/+3%/+2%)
- [ ] 声望等级×2%加成正确
- [ ] 4档等级倍率正确(×1.0/×1.2/×1.5/×2.0)

### 4.2 胜率公式

```
胜率 = min(95%, max(5%, (我方/敌方)×50% + 地形修正))
```

- [ ] 6种地形修正正确
- [ ] 截断至[5%, 95%]

### 4.3 山贼战斗公式(R6/R7/R8)

```
山贼战力 = max(100, 玩家等级×100) × 难度系数
胜率 = min(95%, max(5%, (兵力/战力)×50% + 修正))
胜利损耗 = 出征兵力 × min(35%, max(5%, 20%×(战力/兵力)))
失败损耗 = 出征兵力 × 20%
```

- [ ] 独立战力(不依赖玩家兵力)
- [ ] 4档难度系数正确(×0.3/×0.5/×0.7/×0.9)
- [ ] 兵力正向参与胜率
- [ ] 胜利损耗上限35%(k=1.75推导)
- [ ] 胜/败损耗比≤1.75x

### 4.4 遗迹探索三档判定(R6/R7)

```
失败阈值 = max(5%, 30% - 科技 - 地形)
部分成功阈值 = max(失败+5%, 50% - 科技×0.5 - 地形×0.5)
```

- [ ] 三档随机判定(非成功率判定)
- [ ] 科技加成=攻城术×2%
- [ ] Lv10数据表: 失败10%/部分30%/成功60%(R7修正)

### 4.5 情报值期望值(R10/R11)

```
R_critical = L_win + (P(L)/P(W)) × L_lose + V_T/P(W)
```

- [ ] 移除沉没成本(R10)
- [ ] 精确分数求解(非近似小数)(R11)
- [ ] 三方交叉校验通过(公式/推算/表格)
