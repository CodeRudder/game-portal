## 评测标准

### 功能完整性

| 等级 | 标准 |
|:----:|------|
| A | 6工具+5图层+20图块+实体管理+导入导出+鸟瞰图+快捷键全部实现 |
| B | 核心功能(图层+画笔+导出+导入+实体)全部实现 |
| C | 基本绘制+导出可用 |

### 渲染质量

| 检查项 | 标准 |
|--------|------|
| 地形过渡 | 相邻不同地形之间有渐变 |
| 地形纹理 | 山地/水域/森林/草地纹理丰富 |
| 实体渲染 | 纯色填充+名称居中，无多余边框 |
| 缩放 | 25%~400%平滑缩放 |
| 大面积地形 | 批量合并渲染，无接缝 |

### 交互体验

| 检查项 | 标准 |
|--------|------|
| 默认工具 | 打开即为查看模式，防误操作 |
| 视窗平移 | 触摸板双指平滑移动，停止后对齐 |
| 缩放 | 按钮+触摸板捏合两种方式 |
| 边界 | 地图不可完全移出视窗 |
| 鸟瞰图 | 小地图可点击跳转 |
