## 一、战斗准备 (Battle Preparation)

### BATTLE-CAMP-001: 查看敌方信息面板
- 前置: 无
- 操作: 进入关卡前点击"敌方信息"按钮，查看敌方阵容详情
- 预期: 正确展示敌方武将列表、等级、兵种、战力数值及阵型排列；未解锁关卡不展示详情
- 优先级: P0
- 关联系统: 情报系统 / 关卡系统
- 状态: covered

### BATTLE-CAMP-002: 编队确认界面
- 前置: BATTLE-CAMP-001
- 操作: 进入编队界面，查看当前默认阵容，确认武将站位与兵种配置
- 预期: 展示6个站位槽位，已上阵武将显示头像/等级/战力，空槽位显示"+"号；战力总和正确计算
- 优先级: P0
- 关联系统: 编队系统 / 武将系统
- 状态: covered

### BATTLE-CAMP-003: 武将上阵与下阵
- 前置: BATTLE-CAMP-002
- 操作: 点击空槽位选择武将上阵；点击已上阵武将选择"下阵"
- 预期: 上阵后武将出现在对应站位，总战力实时更新；下阵后槽位清空，战力扣除；同一武将不可重复上阵
- 优先级: P0
- 关联系统: 编队系统
- 状态: covered

### BATTLE-CAMP-004: 武将站位调整
- 前置: BATTLE-CAMP-002
- 操作: 长按已上阵武将拖拽至另一槽位进行站位交换
- 预期: 两个武将位置互换，前排/后排属性加成随之变化；拖拽过程有视觉反馈
- 优先级: P1
- 关联系统: 编队系统 / 属性系统
- 状态: covered

### BATTLE-CAMP-005: 装备检查与快速更换
- 前置: BATTLE-CAMP-002
- 操作: 在编队界面点击武将头像查看装备详情，使用"一键强化"或"推荐装备"功能
- 预期: 显示6个装备槽位及当前装备属性；推荐装备按战力提升排序；更换后属性实时刷新
- 优先级: P1
- 关联系统: 装备系统 / 背包系统
- 状态: partial

### BATTLE-CAMP-006: 战力评估与推荐
- 前置: BATTLE-CAMP-002
- 操作: 编队完成后点击"战力评估"，查看与推荐战力的对比
- 预期: 显示我方总战力、关卡推荐战力、战力差额；战力不足时给出红色警告及提升建议（升级/装备/羁绊）
- 优先级: P1
- 关联系统: 战力系统 / 推荐系统
- 状态: covered

### BATTLE-CAMP-007: 羁绊效果预览
- 前置: BATTLE-CAMP-003
- 操作: 编队中存在可激活羁绊的武将组合时，查看羁绊效果
- 预期: 已激活羁绊高亮显示并标注属性加成数值；未激活羁绊灰色显示并提示缺失武将
- 优先级: P1
- 关联系统: 羁绊系统
- 状态: partial

### BATTLE-CAMP-008: 出战确认与进入战斗
- 前置: BATTLE-CAMP-002
- 操作: 点击"出战"按钮，确认消耗体力并进入战斗加载界面
- 预期: 体力不足时弹出提示并引导补充；体力充足时扣除体力，播放加载动画后进入战斗场景
- 优先级: P0
- 关联系统: 体力系统 / 关卡系统
- 状态: covered

## 二、战斗执行 (Battle Execution)

### BATTLE-CAMP-009: 回合开始与行动顺序
- 前置: BATTLE-CAMP-008
- 操作: 战斗开始，观察第一回合的行动顺序排列
- 预期: 按速度属性从高到低排列行动顺序；速度相同时武将优先于敌方；行动条正确展示当前回合数
- 优先级: P0
- 关联系统: 战斗引擎 / 属性系统
- 状态: covered

### BATTLE-CAMP-010: 普通攻击执行
- 前置: BATTLE-CAMP-009
- 操作: 武将轮到行动时执行普通攻击（自动战斗模式或手动选择目标）
- 预期: 播放攻击动画，目标血条扣减；伤害数值飘字显示；攻击后怒气值增加
- 优先级: P0
- 关联系统: 战斗引擎
- 状态: covered

### BATTLE-CAMP-011: 技能释放（主动技能）
- 前置: BATTLE-CAMP-009
- 操作: 怒气值满时手动/自动释放主动技能
- 预期: 播放技能特效动画，正确计算技能伤害倍率；范围技能命中所有目标；怒气值清零
- 优先级: P0
- 关联系统: 技能系统 / 战斗引擎
- 状态: covered

### BATTLE-CAMP-012: 伤害计算验证
- 前置: BATTLE-CAMP-010
- 操作: 记录普通攻击的伤害数值，与公式计算结果对比
- 预期: 伤害 = 攻击力 × 技能倍率 × (1 - 防御减免) × 兵种克制系数；数值误差不超过±5%
- 优先级: P0
- 关联系统: 战斗引擎 / 数值系统
- 状态: covered

### BATTLE-CAMP-013: 暴击触发与伤害加成
- 前置: BATTLE-CAMP-012
- 操作: 战斗中触发暴击事件，观察暴击伤害数值
- 预期: 暴击时显示特殊飘字（红色大字"暴击"）；暴击伤害 = 基础伤害 × (1 + 暴击伤害加成)；暴击率由暴击属性决定
- 优先级: P0
- 关联系统: 战斗引擎 / 属性系统
- 状态: covered

### BATTLE-CAMP-014: 闪避触发
- 前置: BATTLE-CAMP-010
- 操作: 战斗中触发闪避事件
- 预期: 显示"闪避"飘字，受击方播放闪避动画；本次攻击伤害为0；闪避率由闪避属性决定
- 优先级: P0
- 关联系统: 战斗引擎 / 属性系统
- 状态: covered

### BATTLE-CAMP-015: 护盾吸收机制
- 前置: BATTLE-CAMP-011
- 操作: 释放护盾类技能后，观察受击时的伤害吸收
- 预期: 护盾值在血条上方以独立条显示；受击时优先扣除护盾值再扣血；护盾值归零后护盾消失
- 优先级: P1
- 关联系统: 战斗引擎 / Buff系统
- 状态: covered

### BATTLE-CAMP-016: Buff/Debuff施加与生效
- 前置: BATTLE-CAMP-011
- 操作: 释放带Buff/Debuff效果的技能，观察状态图标及属性变化
- 预期: Buff图标显示在武将头顶；属性面板实时更新加成/减益数值；持续回合数每回合递减，归零后自动移除
- 优先级: P1
- 关联系统: Buff系统 / 战斗引擎
- 状态: covered

### BATTLE-CAMP-017: 单位死亡与退场
- 前置: BATTLE-CAMP-012
- 操作: 武将/敌方单位血量降为0时触发死亡
- 预期: 播放死亡动画后单位退场；该单位不再参与后续回合行动；若触发复活技能则复活并恢复一定血量
- 优先级: P0
- 关联系统: 战斗引擎
- 状态: covered

### BATTLE-CAMP-018: 兵种克制验证
- 前置: BATTLE-CAMP-010
- 操作: 使用克制兵种攻击被克制兵种，对比同战力无克制伤害
- 预期: 步兵克弓兵、弓兵克骑兵、骑兵克步兵形成三角克制；克制方伤害+30%，被克制方伤害-30%
- 优先级: P1
- 关联系统: 战斗引擎 / 兵种系统
- 状态: covered

### BATTLE-CAMP-019: 回合结束与下一回合
- 前置: BATTLE-CAMP-009
- 操作: 所有单位行动完毕后回合结束，进入下一回合
- 预期: 回合计数+1；持续型Buff回合数-1；怒气自然回复；行动条重新排序
- 优先级: P0
- 关联系统: 战斗引擎
- 状态: covered

### BATTLE-CAMP-020: 战斗加速与跳过
- 前置: BATTLE-CAMP-009
- 操作: 点击加速按钮切换1x/2x/3x速度；点击跳过按钮直接结算
- 预期: 加速时动画播放速率对应提升，伤害数值仍正确显示；跳过时直接进入战斗结算界面
- 优先级: P2
- 关联系统: 战斗引擎 / UI系统
- 状态: covered

### BATTLE-CAMP-021: 战斗中断与重连
- 前置: BATTLE-CAMP-009
- 操作: 战斗过程中强制关闭应用后重新登录
- 预期: 重连后恢复战斗现场，当前回合状态完整保留；若超时则按失败处理，不扣除体力
- 优先级: P1
- 关联系统: 战斗引擎 / 网络系统
- 状态: missing

## 三、战斗结算 (Battle Settlement)

### BATTLE-CAMP-022: 胜负判定
- 前置: BATTLE-CAMP-017
- 操作: 一方全灭后触发战斗结束判定
- 预期: 敌方全灭=胜利，我方全灭=失败；20回合内未分胜负=超时失败；胜利/失败播放对应动画和音效
- 优先级: P0
- 关联系统: 战斗引擎 / 结算系统
- 状态: covered

### BATTLE-CAMP-023: 奖励发放（胜利）
- 前置: BATTLE-CAMP-022
- 操作: 战斗胜利后查看奖励面板
- 预期: 显示金币、经验、道具掉落列表；首通奖励额外标注"首通"标签；奖励自动存入背包
- 优先级: P0
- 关联系统: 结算系统 / 奖励系统 / 背包系统
- 状态: covered

### BATTLE-CAMP-024: 经验分配与升级
- 前置: BATTLE-CAMP-023
- 操作: 查看参战武将获得的经验值及是否升级
- 预期: 经验按参战武将平分；升级时播放升级特效；属性面板更新为新等级数值；若突破等级上限则提示突破
- 优先级: P0
- 关联系统: 经验系统 / 武将系统
- 状态: covered

### BATTLE-CAMP-025: 三星评价判定
- 前置: BATTLE-CAMP-022
- 操作: 战斗胜利后查看星级评价
- 预期: 0阵亡=3星；1-2阵亡=2星；3+阵亡=1星；超过N回合=降1星；星级影响奖励倍率
- 优先级: P1
- 关联系统: 结算系统 / 评价系统
- 状态: covered

### BATTLE-CAMP-026: 掉落物品展示与领取
- 前置: BATTLE-CAMP-023
- 操作: 查看战斗掉落物品列表，点击领取
- 预期: 掉落物品按稀有度排列（金>紫>蓝>绿>白）；装备/材料/碎片分类展示；背包满时提示扩容
- 优先级: P1
- 关联系统: 掉落系统 / 背包系统
- 状态: covered

### BATTLE-CAMP-027: 失败结算与重试
- 前置: BATTLE-CAMP-022
- 操作: 战斗失败后查看结算面板
- 预期: 显示失败原因（全灭/超时）；不发放通关奖励但保留已获得经验；提供"重试"和"调整阵容"两个入口
- 优先级: P0
- 关联系统: 结算系统
- 状态: covered

## 四、攻城推进 (Campaign Progression)

### BATTLE-CAMP-028: 关卡选择与地图展示
- 前置: 无
- 操作: 进入攻城模式，查看章节地图和关卡节点
- 预期: 已通关关卡显示星级；当前关卡高亮闪烁；未解锁关卡灰色锁定；章节间有路线连接
- 优先级: P0
- 关联系统: 关卡系统 / 地图系统
- 状态: covered

### BATTLE-CAMP-029: 关卡挑战消耗
- 前置: BATTLE-CAMP-028
- 操作: 点击可挑战关卡，查看消耗信息并确认
- 预期: 显示体力消耗量；精英关卡额外显示每日挑战次数限制；体力不足时弹出购买/等待提示
- 优先级: P0
- 关联系统: 体力系统 / 关卡系统
- 状态: covered

### BATTLE-CAMP-030: 关卡战斗执行
- 前置: BATTLE-CAMP-029
- 操作: 确认挑战后进入战斗（复用战斗执行流程 BATTLE-CAMP-009 ~ BATTLE-CAMP-021）
- 预期: 战斗流程与标准战斗一致；关卡特殊机制（如地形效果、天气系统）正确生效
- 优先级: P0
- 关联系统: 关卡系统 / 战斗引擎
- 状态: covered

### BATTLE-CAMP-031: 关卡解锁条件验证
- 前置: BATTLE-CAMP-030
- 操作: 通关当前关卡后检查下一关解锁状态
- 预期: 前置关卡通关后自动解锁下一关；部分关卡需额外条件（等级/道具/前置章节全通）；解锁动画播放
- 优先级: P0
- 关联系统: 关卡系统 / 解锁系统
- 状态: covered

### BATTLE-CAMP-032: 星级累计与宝箱奖励
- 前置: BATTLE-CAMP-025
- 操作: 收集章节内关卡的星级，达到阈值领取宝箱
- 预期: 章节星数实时累计；达到6/12/18星分别解锁铜/银/金宝箱；宝箱内容预览可查看；奖励一次性领取
- 优先级: P1
- 关联系统: 评价系统 / 奖励系统
- 状态: covered

### BATTLE-CAMP-033: 扫荡功能解锁与执行
- 前置: BATTLE-CAMP-025
- 操作: 关卡三星通关后使用扫荡功能快速获取奖励
- 预期: 三星通关后解锁扫荡按钮；扫荡直接跳过战斗进入结算；可批量扫荡最多10次；VIP等级影响扫荡次数上限
- 优先级: P1
- 关联系统: 关卡系统 / 扫荡系统 / VIP系统
- 状态: covered

### BATTLE-CAMP-034: 章节切换与进度同步
- 前置: BATTLE-CAMP-031
- 操作: 当前章节全部通关后切换至下一章节
- 预期: 播放章节完成动画；下一章节地图展开；章节进度在主界面同步更新
- 优先级: P2
- 关联系统: 关卡系统 / 地图系统
- 状态: partial

## 五、远征系统 (Expedition System)

### BATTLE-CAMP-035: 远征入口与规则展示
- 前置: 无
- 操作: 从主界面进入远征系统，查看远征规则说明
- 预期: 显示远征开放等级条件；说明远征为连续战斗不可回退；展示远征奖励预览；每日重置时间倒计时显示
- 优先级: P1
- 关联系统: 远征系统 / UI系统
- 状态: covered

### BATTLE-CAMP-036: 远征阵容选择
- 前置: BATTLE-CAMP-035
- 操作: 选择远征出征阵容（与普通编队独立）
- 预期: 远征有独立阵容槽位；武将在远征中阵亡则后续关卡不可再使用；显示可用武将池及已阵亡武将
- 优先级: P0
- 关联系统: 远征系统 / 编队系统
- 状态: covered

### BATTLE-CAMP-037: 远征连续战斗
- 前置: BATTLE-CAMP-036
- 操作: 依次挑战远征中的多个关卡节点
- 预期: 每关战斗后血量/怒气/阵亡状态继承至下一关；每关难度递增；关卡间可选择"继续"或"撤退"
- 优先级: P0
- 关联系统: 远征系统 / 战斗引擎
- 状态: covered

### BATTLE-CAMP-038: 远征中途撤退
- 前置: BATTLE-CAMP-037
- 操作: 在远征中途选择"撤退"
- 预期: 弹出确认提示"撤退将只获得已通关关卡的奖励"；确认后结算已通过关卡的奖励；不扣除远征次数
- 优先级: P1
- 关联系统: 远征系统 / 结算系统
- 状态: covered

### BATTLE-CAMP-039: 远征奖励累计与结算
- 前置: BATTLE-CAMP-037
- 操作: 通关远征全部关卡或中途完成所有可通关节点后结算
- 预期: 展示所有关卡奖励累计总和；远征币单独计数；奖励一次性发放至背包
- 优先级: P0
- 关联系统: 远征系统 / 奖励系统
- 状态: covered

### BATTLE-CAMP-040: 远征重置机制
- 前置: BATTLE-CAMP-039
- 操作: 远征完成后或每日重置时间到达后重置远征状态
- 预期: 每日0点自动重置，所有关卡状态恢复初始；武将阵亡状态清除；重置后可重新挑战；VIP可额外购买重置次数
- 优先级: P1
- 关联系统: 远征系统 / VIP系统
- 状态: covered
