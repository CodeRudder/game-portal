## 阶段一：新游戏初始化 (ONBOARD-001 ~ ONBOARD-005)

---

### ONBOARD-001: 启动游戏进入主界面
- 前置: 无
- 操作: 点击游戏图标启动应用，等待加载完成
- 预期: 进入游戏主界面，加载时间 ≤ 3秒；主界面显示"开始游戏""继续游戏""设置"三个按钮；背景为三国风格插画
- 优先级: P0
- 关联系统: 启动器 / 资源加载
- 状态: covered

### ONBOARD-002: 点击开始游戏
- 前置: ONBOARD-001
- 操作: 点击"开始游戏"按钮
- 预期: 弹出难度选择面板（简单/普通/困难），默认选中"普通"；面板动画时长 ≤ 0.5秒
- 优先级: P0
- 关联系统: 主菜单 / 游戏初始化
- 状态: covered

### ONBOARD-003: 选择难度并确认
- 前置: ONBOARD-002
- 操作: 选择"简单"难度，点击"确认"按钮
- 预期: 进入势力选择界面，显示魏、蜀、吴三个势力卡片；每个势力显示基础属性（攻击/防御/智力各60~85区间差异）；蜀国默认高亮
- 优先级: P0
- 关联系统: 势力系统 / 游戏初始化
- 状态: covered

### ONBOARD-004: 选择势力（蜀国）
- 前置: ONBOARD-003
- 操作: 点击"蜀"势力卡片，点击"确认"按钮
- 预期: 播放势力介绍动画（≤ 5秒），显示刘备势力简介文本；随后进入君主命名界面，默认名称"刘备"
- 优先级: P0
- 关联系统: 势力系统 / 君主系统
- 状态: covered

### ONBOARD-005: 确认君主名称进入游戏世界
- 前置: ONBOARD-004
- 操作: 保持默认名称"刘备"，点击"开始霸业"按钮
- 预期: 加载游戏世界地图，进入新手引导状态；屏幕中央显示主城（Lv1），四周为迷雾；触发新手引导NPC对话："主公，乱世已至，请先建设城池！"
- 优先级: P0
- 关联系统: 君主系统 / 引导系统 / 地图系统
- 状态: covered

- 预期: 城池等级 Lv1；城池名称"成都"；拥有空地 5 块（可建造位）；城防值 100/100；民心 80
- 优先级: P0
- 关联系统: 城池系统 / 建筑系统

## 阶段三：教程步骤 — 首栋建筑农田Lv1 (ONBOARD-009 ~ ONBOARD-012)

---

### ONBOARD-009: 引导点击空地建造农田
- 前置: ONBOARD-006, ONBOARD-007
- 操作: 跟随引导手指动画，点击第一块空地
- 预期: 弹出建筑选择面板，显示可建造建筑列表；农田图标高亮并有脉冲动画提示；建筑列表包含：农田（消耗粮草100）、市集（消耗铜钱200）、兵营（消耗铜钱300，灰色不可用）
- 优先级: P0
- 关联系统: 引导系统 / 建筑系统
- 状态: covered

### ONBOARD-010: 选择农田并确认建造
- 前置: ONBOARD-009
- 操作: 点击"农田"图标，点击"建造"按钮
- 预期: 弹出建造确认面板：建筑名"农田"、等级"Lv1"、消耗"粮草 ×100"、产出"粮草 +10/分钟"；确认后扣除粮草 100，资源变为 粮草 900；空地位置出现建造进度条
- 优先级: P0
- 关联系统: 建造系统 / 资源系统
- 状态: covered

### ONBOARD-011: 等待农田建造完成
- 前置: ONBOARD-010
- 操作: 等待建造倒计时结束（教程模式加速：10秒）
- 预期: 建造进度条从 0% 走到 100%（约10秒）；完成后播放建造完成音效；农田建筑出现在地图上，显示"Lv1"标签；触发NPC对话："农田已建成，粮草将源源不断！"
- 优先级: P0
- 关联系统: 建造系统 / 计时器系统
- 状态: covered

### ONBOARD-012: 验证农田产出生效
- 前置: ONBOARD-011
- 操作: 等待60秒后查看粮草资源数值
- 预期: 粮草每分钟增加 10；60秒后粮草从 900 增长至 910±2（允许计时器误差）；点击农田可查看详情面板：等级 Lv1、产出 +10粮草/分钟、升级消耗 粮草×200
- 优先级: P0
- 关联系统: 资源产出系统 / 建筑系统
- 状态: covered

## 阶段六：首次招募与获武将 (ONBOARD-019 ~ ONBOARD-023)

---

### ONBOARD-019: 点击招募按钮进入招募界面
- 前置: ONBOARD-017
- 操作: 跟随引导点击"招募"按钮
- 预期: 进入招募界面，显示招募方式列表：普通招募（铜钱×300）、高级招募（元宝×30）；普通招募高亮并有脉冲动画提示
- 优先级: P0
- 关联系统: 招募系统 / 引导系统
- 状态: covered

### ONBOARD-020: 执行普通招募
- 前置: ONBOARD-019
- 操作: 点击"普通招募"按钮，确认消耗铜钱 300
- 预期: 扣除铜钱 300；播放招募动画（卡牌翻转特效，≤ 3秒）；动画结束后显示获得的武将卡牌
- 优先级: P0
- 关联系统: 招募系统 / 资源系统 / 动画系统
- 状态: covered

### ONBOARD-021: 验证获得武将
- 前置: ONBOARD-020
- 操作: 查看招募结果界面
- 预期: 获得一名 1~3星 武将（教程保底为2星武将"关平"）；武将属性：攻击 45~55、防御 35~45、智力 30~40、兵力 500；显示武将立绘、名称、星级、阵营"蜀"
- 优先级: P0
- 关联系统: 招募系统 / 武将系统
- 状态: covered

### ONBOARD-022: 确认获得武将返回主界面
- 前置: ONBOARD-021
- 操作: 点击"确认"按钮关闭招募结果界面
- 预期: 返回主界面；NPC对话："恭喜主公获得良将！请查看武将详情。"
- 优先级: P1
- 关联系统: 招募系统 / 引导系统
- 状态: covered

### ONBOARD-023: 查看武将详情

- 关联系统: 武将系统 / UI系统
- 状态: covered

## 阶段八：首次战斗 (ONBOARD-027 ~ ONBOARD-030)

---

### ONBOARD-027: 进入世界地图选择战斗目标
- 前置: ONBOARD-026
- 操作: 跟随引导点击"出征"按钮，进入世界地图
- 预期: 世界地图显示主城周边区域；附近显示一个新手野地（Lv1黄巾贼），有红色脉冲标记；点击野地弹出信息面板：敌人"黄巾贼"、战力 300、奖励"粮草×50、铜钱×30"
- 优先级: P0
- 关联系统: 地图系统 / 战斗系统 / 引导系统
- 状态: covered

### ONBOARD-028: 发起战斗进入战斗场景
- 前置: ONBOARD-027
- 操作: 点击"攻击"按钮
- 预期: 弹出编队选择面板，默认选中"第一队"（关平）；点击确认后加载战斗场景（加载 ≤ 2秒）；进入回合制战斗界面，显示我方（关平，兵力500）vs 敌方（黄巾贼×3，每个兵力100）
- 优先级: P0
- 关联系统: 战斗系统 / 编队系统
- 状态: covered

### ONBOARD-029: 执行战斗操作
- 前置: ONBOARD-028
- 操作: 回合1选择"横扫"技能攻击敌方全体；后续回合选择普通攻击逐一消灭残血敌人
- 预期: "横扫"技能造成 60点伤害（攻击50×120%）到每个敌人；敌方每回合对我方造成 15~25点伤害；3~4回合内战斗结束；我方剩余兵力 ≥ 350；战斗过程无卡顿，动画流畅
- 优先级: P0
- 关联系统: 战斗系统 / 技能系统 / 动画系统
- 状态: covered

### ONBOARD-030: 战斗胜利与奖励结算
- 前置: ONBOARD-029
- 操作: 战斗结束后查看结算面板
- 预期: 显示"胜利！"横幅；奖励面板：粮草 +50、铜钱 +30、经验 +80；关平经验从 0 升至 80/100（未升级）；战斗评级显示"A"级（基于剩余兵力比例 ≥ 70%）；点击"确认"返回世界地图
- 优先级: P0
- 关联系统: 战斗系统 / 奖励系统 / 经验系统
- 状态: covered
