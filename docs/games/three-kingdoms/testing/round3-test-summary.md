2. **settings/01无限循环** — levelUpTo辅助函数动态计算+安全上限
3. **Play文档初始资源值不一致** — 粮草500/铜钱300对齐源码
4. **Play文档离线衰减系数不一致** — 48-72h: 20%对齐源码
5. **招贤榜资源校验缺失** — GameEventSimulator.recruitHero检查返回值
6. **Play文档概率表不一致** — 普通/高级池概率对齐PRD
7. **星级奖励倍率不一致** — ★★1.5/★★★2.0对齐PRD
8. **碎片掉率不一致** — 30%→10%对齐PRD
9. **首通碎片必掉未实现** — isFirstClear参数
10. **胜率预估公式不一致** — 幂函数→PRD线性比率
11. **24h占领冷却未实现** — captureTimestamps+冷却检查
