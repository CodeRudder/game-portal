## 每版本迭代子流程（7步循环）
```
① 根据game-flow-integration-test-methodology.md修正集成测试脚本 → 执行集成测试发现问题
② 修复集成测试脚本 + 修复发现的代码问题（PRD逻辑漏洞/缺失流程/实现问题）
③ 重新运行集成测试确认全通过
④ 游戏评测师评审（play流程完整性/PRD一致性/合理性）+ 修复
⑤ 架构评审（ISubsystem/类型安全/模块边界/代码质量）+ 修复
⑥ 封版提交
```
