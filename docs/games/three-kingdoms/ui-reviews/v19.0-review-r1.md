| # | 警告项 | 详情 | 分析 |

| 1 | 场景区未找到 | `canvas,.tk-scene,.building-panel` 未命中 | 可能使用 DOM 渲染而非 Canvas，选择器需更新 |
| 2 | 设置按钮未找到 | `[data-feature="settings"]` 未命中 | 设置入口可能在"更多"Tab内，需先切换Tab |

### 警告根因分析

1. **场景区选择器**: v19 使用 DOM 节点渲染城池场景而非 Canvas，建议补充 `.tk-city-scene` 等新选择器
2. **设置面板入口**: `[data-feature="settings"]` 选择器在主界面不可见，需先点击"更多"Tab后才能定位到设置按钮
