## 四、问题清单

### P1 (重要 — 不阻塞但需修复)

| # | 问题 | 文件 | 说明 |
|---|------|------|------|
| P1-1 | **Props `engine: any` 缺乏类型约束** | ShopPanel.tsx:13, TradePanel.tsx:16 | 两个面板的 `engine` prop 均为 `any` 类型，无法在编译期捕获调用错误。建议定义 `EngineFacade` 接口 |
| P1-2 | **零 ARIA 无障碍属性** | ShopPanel.tsx, TradePanel.tsx | 两个组件共 0 处 `aria-*` 或 `role` 属性，Tab切换、弹窗、按钮均无屏幕阅读器支持 |

### P2 (建议)

| # | 问题 | 文件 | 说明 |
|---|------|------|------|
| P2-1 | **大量内联样式** | ShopPanel.tsx (24处), TradePanel.tsx (45处) | 应抽取为 CSS Modules 或 Tailwind 类 |
| P2-2 | **硬编码颜色值** | ShopPanel (8种), TradePanel (12种) | `#d4a574`, `#1a1a2e`, `#4CAF50` 等应使用 CSS 变量 |
| P2-3 | **TODO 残留** | TradePanel.tsx:123 | `const castleLevel = 1; // TODO: 从引擎获取实际主城等级` — 功能未对接 |
| P2-4 | **`as any` 类型断言** | ShopPanel.tsx (9处), TradePanel.tsx (9处) | `borderRadius: 'var(--tk-radius-md)' as any` — CSSProperties 类型不识别 CSS 变量 |
| P2-5 | **TradePanel 超过 500 行** | TradePanel.tsx (582行) | 建议拆分为 RoutesTab/CaravansTab/PricesTab 子组件 |
