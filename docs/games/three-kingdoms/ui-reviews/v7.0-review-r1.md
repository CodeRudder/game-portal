## P0修复建议

### P0-1: 任务面板FeatureMenu入口缺失 🔴
**问题**: QuestPanel已在FeaturePanelOverlay注册（id='quest'），但FeatureMenu的FEATURE_ITEMS数组中缺少quest菜单项，导致用户无法通过UI打开任务面板。
**影响**: 任务系统功能完全不可达，用户无法查看日常/主线/支线任务。
**修复**: 在TabBar.tsx的FEATURE_ITEMS中添加quest菜单项：
```typescript
{ id: 'quest', icon: '📋', label: '任务', description: '日常/主线/支线任务', available: true },
```

### P0-2: NPC对话弹窗响应延迟 🟡
**问题**: 点击NPC对话按钮后，dialog overlay未在预期时间内渲染。
**影响**: 对话交互体验不稳定。
**建议**: 检查NPCDialogModal的打开逻辑，确认好感度前置条件是否过于严格。
