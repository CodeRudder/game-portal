## 警告项分析

### TAB-02: 武将列表网格初始未找到
- **现象**: 切换到武将Tab时，`.tk-hero-grid` 未找到
- **原因**: 初始状态无武将，HeroTab显示空状态(`.tk-hero-empty`)而非网格
- **影响**: 低 — 招募武将后网格正常显示
- **建议**: 测试脚本可优化为先判断空状态，再检查网格
