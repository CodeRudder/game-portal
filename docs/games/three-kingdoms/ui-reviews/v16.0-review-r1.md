## 四、控制台警告

发现2条React style property警告：
```
Warning: Removing borderColor border — shorthand and non-shorthand conflict
Source: EquipmentTab.tsx:43
```
**严重级别**: P4 (不影响功能，建议修复)
**原因**: EquipmentTab组件中同时设置了border简写和borderColor属性，React在重渲染时会移除冲突属性。
**建议**: 将border简写拆分为borderWidth/borderStyle/borderColor单独属性。
