## P1 重要问题（已修复）

### P1-1: AllianceBossSystem 和 AllianceShopSystem 未注册到引擎 ✅
- **文件**: `engine/ThreeKingdomsEngine.ts`, `engine-getters.ts`, `engine-getters-types.ts`
- **问题**: AllianceBossSystem 和 AllianceShopSystem 已实例化但未注册到 SubsystemRegistry，无 getter
- **影响**: UI 层无法通过引擎门面访问联盟Boss和联盟商店功能
- **修复**:
  - ThreeKingdomsEngine.ts: 添加字段声明、构造函数实例化、registry注册
  - engine-getters.ts: 添加 `getAllianceBossSystem()` 和 `getAllianceShopSystem()` getter
  - engine-getters-types.ts: 添加对应类型声明和导入

### P1-2: AllianceSystem.ts 超过500行限制（570行） ✅
- **文件**: `engine/alliance/AllianceSystem.ts`
- **问题**: 570行超过500行限制
- **修复**: 将权限检查、工具方法、存档序列化拆分到 `AllianceHelper.ts`
  - AllianceSystem.ts: 570行 → 338行
  - 新增 AllianceHelper.ts: 103行
  - AllianceSystem 通过 require() 委托调用 AllianceHelper
