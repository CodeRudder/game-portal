## 三、遭遇系统架构评估

### 遭遇模板结构
```
EncounterTemplate (接口)
├── id: string                    — 模板唯一标识
├── subType: EncounterSubType     — combat/diplomatic/exploration/disaster
├── name: string                  — 遭遇名称
├── descriptionTemplate: string   — 描述模板（支持{player}占位符）
├── difficulty: EncounterDifficulty — easy/normal/hard/epic
├── environments: EncounterEnvironment[] — 适用环境
├── baseWeight: number            — 基础触发权重
├── options: EncounterTemplateOption[] — 2-3个分支选项
└── cooldownTurns: number         — 冷却回合数
```
