## P1 重要问题

### P1-1: ChatSystem 和 social/LeaderboardSystem 未注册到引擎
- **文件**: `engine/ThreeKingdomsEngine.ts`
- **问题**: ChatSystem 和 social/LeaderboardSystem 未在引擎中实例化、注册和导出
- **影响**: UI 层无法通过引擎门面访问聊天和社交排行榜功能
- **修复**: 在 ThreeKingdomsEngine 中创建实例、注册、添加 getters

### P1-2: FriendSystem.ts 超过500行限制（527行）
- **文件**: `engine/social/FriendSystem.ts`
- **问题**: 527行超过500行限制
- **建议**: 将借将系统(BorrowHero)和每日重置(DailyReset)拆分到独立文件
