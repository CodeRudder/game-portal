# 装备系统 UI 设计（v3.0）

> **v3.0 迭代说明**：新增装备品质系统详细定义、强化费用与成功率表、炼制配方与产出概率、套装效果设计、手机端完整适配方案。保留 v2.1 军师推荐系统。评审反馈 8.6/10 → 目标 9.0+。

---

## 一、入口

| 入口位置 | 触发方式 | 说明 |
|---------|---------|------|
| "更多"菜单「背包」入口→「装备」Tab | 点击 | 主入口 |
| 武将详情页装备槽位 | 点击槽位 | 快捷穿戴入口 |
| 武将详情页推荐装备快捷入口 | 点击推荐区域 | v2.1 新增 |
| 战斗结算→获得新装备 | 自动弹出 | 新装备展示 |

## 四、装备炼制配方与产出概率

### 4.1 炼制配方表

| 配方 | 投入材料 | 产出品质分布 | 额外消耗 |
|------|---------|------------|---------|
| 基础炼制 | 3 件同品质装备 | 见下方概率表 | 铜钱 ×2000 |
| 高级炼制 | 5 件同品质装备 | 基础概率 +10% 高品质 | 铜钱 ×5000 |
| 定向炼制 | 3 件同品质 + 定向石 | 指定部位产出 | 铜钱 ×8000 + 定向石 ×1 |
| 保底炼制 | 3 件同品质 + 保底符 | 必定不低于紫色 | 铜钱 ×10000 + 保底符 ×1 |

### 4.3 保底机制（保留 v2.1）

- **紫色保底**：连续炼制 10 次未获得紫色及以上 → 第 11 次必出紫色
- **金色保底**：连续炼制 30 次未获得金色 → 第 31 次必出金色
- 保底计数器跨会话保持（localStorage 持久化）
- 炼制界面底部显示保底进度条
