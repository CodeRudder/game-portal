### 公会BOSS系统

> **设计理念**：异步协作BOSS，不要求成员同时在线，每人每天可挑战一次，累计伤害解锁奖励。

#### BOSS列表界面

```
+--[公会BOSS]--------本周BOSS: 董卓(魔化)---+
|  BOSS立绘(水墨风)    HP: ████████░░ 72%    |
|  等级: Lv.30        剩余时间: 3天12小时    |
|                                            |
|  累计参与: 18人     最高伤害: 关羽 12,340   |
|                                            |
|  [⚔️挑战(今日剩余1次)]  [查看排行]          |
+--------------------------------------------+
```

- **BOSS轮换**：每周更换一个BOSS，难度递增
- **挑战规则**：每人每天1次挑战机会（VIP可购买额外次数）
- **战斗方式**：使用当前编队进行1vBOSS战斗，限时5回合
- **伤害累计**：所有成员伤害累加，BOSS HP归零则击败

#### BOSS奖励分配

| 奖励类型 | 条件 | 内容 |
|---------|------|------|
| 参与奖 | 造成任意伤害 | 铜钱×500 + 公会贡献×10 |
| 伤害排名奖 | 伤害排名前3 | 稀有装备箱 + 公会贡献×50 |
| 击败奖 | BOSS被击败后全员 | 天命×20 + 公会贡献×30 |
| 最后一击奖 | 给予致命一击 | 传说装备箱 + 公会贡献×100 |

#### BOSS详情面板

- BOSS立绘+属性预览（HP/攻击/防御/技能）
- 当前伤害排行榜（前10名）
- 我的累计伤害+排名
- 推荐战力提示
- 历史BOSS战绩回顾

## 异步互动系统

> **设计理念**：放置游戏玩家在线时间不固定，社交系统必须支持异步互动。核心原则是"让玩家感受到其他玩家的存在，但不强制同步"。

### 1. 借将系统

#### 概述
- 玩家可向好友/公会成员「借用」一名武将，编入自己的出征队伍
- 被借武将的等级、装备、技能完整保留
- 每次借用持续一次出征（单关卡），用完归还

#### 借将界面

```
+--[借将]--------选择借用的武将---+
|  好友武将:                       |
|  [关羽Lv.30] [张飞Lv.28]        |
|  [赵云Lv.35] [马超Lv.25]        |
|                                  |
|  公会武将:                       |
|  [吕布Lv.40] [曹操Lv.38]        |
|  [周瑜Lv.32]                    |
|                                  |
|  已选: [赵云 Lv.35 ⚔️3200]      |
|  [确认借用]                      |
+----------------------------------+
```

#### 规则
- 每天可借用3次（不同好友/成员）
- 同一武将每天只能被借用1次（避免抢夺）
- 借用武将战力按80%计算（防止完全依赖借将）
- 借用方获得100%经验，出借方获得友情点×10
- 借将不可用于PVP

#### 借将友情点每日上限（v2.1 新增）

| 参数 | 数值 | 说明 |
|------|------|------|
| 单次出借友情点 | 10点 | 每次武将被借用固定获得 |
| 每日友情点获取上限 | **200点** | 出借方每日通过借将获得的友情点上限 |
| 每日借出次数上限 | 20次 | 即使友情点已达上限，仍可借出（但不获得友情点） |
| 友情点用途 | 兑换商店 | 可在友情商店兑换招募令、绿色装备箱等 |

**友情点上限 UI 展示**：
- 借将面板顶部显示友情点进度条：「今日友情点：120/200」
- 达到上限后进度条变金色+提示「今日友情点已满，继续借出将不获得友情点」
- 每日0点重置

#### 通知
- 武将被借用时，出借方收到邮件通知："你的赵云被[玩家名]借用出征，获得友情点×10"
- 武将归还时自动回到出借方阵容

### 2. 资源援助

#### 概述
- 玩家可向公会成员请求/赠送资源
- 每日援助上限限制，防止小号养大号

#### 请求援助界面

```
+--[请求援助]-----+
|  需要的资源:     |
|  🌾粮草 [500]  |
|  💰铜钱 [300]  |
|                  |
|  [发送请求]      |
|  已有2人援助     |
+------------------+
```

#### 规则
- 每天可请求1次援助
- 每天可赠送3次援助给不同成员
- 单次赠送上限：粮草≤500，铜钱≤300
- 赠送方获得公会贡献×5
- 请求在公会频道显示24小时，过期失效

#### 援助量公式（v2.1 具体化）

**单次援助上限公式**：

```
单次援助上限 = min(500, 公会等级 × 50, 援助方等级 × 10)
```

| 公会等级 | 公会等级×50 | 援助方等级 | 援助方等级×10 | 实际上限 |
|---------|------------|-----------|-------------|---------|
| 3 | 150 | 10 | 100 | **100** |
| 5 | 250 | 20 | 200 | **200** |
| 8 | 400 | 30 | 300 | **300** |
| 10 | 500 | 50 | 500 | **500** |
| 10 | 500 | 20 | 200 | **200** |

> 取三者最小值，确保低等级玩家和高等级玩家之间的援助量受到合理限制。其中"援助方等级"指发起赠送操作的玩家等级，非请求援助方的等级。

**援助量 UI 展示**：
- 援助输入框上方显示：「单次上限：粮草 200 / 铜钱 200」（根据公式动态计算）
- 输入超过上限时自动截断+红色提示「已达单次援助上限」

#### 同IP/设备防刷机制（v2.1 新增）

**规则**：
- 系统检测援助双方是否来自**同一IP地址**或**同一设备指纹**
- 如判定为同IP/设备，援助收益**减半**（50%）

```
实际到账量 = 援助量 × 关系系数
关系系数 = {
  1.0,  正常（不同IP + 不同设备）
  0.5,  同IP 或 同设备
  0.3,  同IP 且 同设备
}
```

**检测范围**：
- IP地址比对（IPv4/IPv6）
- 设备指纹：User-Agent + 屏幕分辨率 + Canvas指纹 + WebGL指纹的哈希值
- 检测在援助发送时实时执行，结果缓存24小时

**UI 提示**：
- 收到减半援助时，邮件中标注：「⚠️ 本次援助因安全策略，实际到账量为申请量的50%」
- 不直接暴露「同IP」等具体原因，避免引导规避
- 赠送方不受影响（仍获得全额公会贡献×5）

#### 新公会成员冷却期（v2.1 新增）

**规则**：
- 新加入公会的成员，**24小时内**不可进行以下操作：
  - ❌ 发送援助请求
  - ❌ 赠送资源给公会成员
  - ❌ 借用公会成员武将
  - ✅ 可正常参与公会BOSS挑战（不影响他人）
  - ✅ 可查看公会成员列表和聊天

**冷却期 UI 展示**：
- 新成员加入时弹出提示：「欢迎加入公会！部分社交功能将在24小时后解锁」
- 援助/借将按钮在冷却期内显示为灰色+倒计时：「🔒 剩余 18:32:15」
- 公会成员列表中，新成员头像旁显示「新」标签（24小时后自动消失）

#### 援助通知
- 收到援助时：邮件通知 + 飞鸽动画
- 公会频道滚动："[玩家A]向公会请求粮草援助"

### 3. 切磋回放系统

#### 概述
- 玩家可与好友/公会成员进行异步切磋（PVP）
- 战斗结果生成回放录像，双方均可观看

#### 切磋流程

```
发起方:                    防守方:
选择对手 → 编队 → 出战  →  收到挑战邮件
                            查看对方阵容 → 编队防守 → 战斗结算
                            ↓
                        双方收到回放邮件
```

#### 回放界面
- 与正常战斗界面相同，但：
  - 右上角标注「回放」水印
  - 底部增加进度条（可拖拽快进/回退）
  - 显示双方玩家昵称+战力
  - 可切换视角（我方/敌方/全局）
- 回放保存7天，最多保存10条

#### 切磋规则
- 每天可发起3次切磋
- 切磋不消耗资源/兵力
- 切磋胜利获得友情点×20，失败获得×5
- 切磋结果不影响排行榜

#### 「举报异常」按钮（v2.1 新增）

**位置与样式**：
- 回放界面右上角「回放」水印旁，增加「⚠️ 举报异常」按钮
- 按钮样式：灰色小按钮，悬停/点击时变为红色
- 手机端：回放进度条上方，右对齐

**举报弹窗**：

```
+--[举报异常]----------------------------------+
|                                               |
|  举报类型（单选）：                            |
|  ○ 作弊/外挂（异常伤害/无敌等）               |
|  ○ 数据异常（属性超出正常范围）               |
|  ○ 恶意利用BUG                               |
|  ○ 其他                                       |
|                                               |
|  补充说明（选填）：                            |
|  ┌──────────────────────────────────┐        |
|  │                                  │        |
|  └──────────────────────────────────┘        |
|                                               |
|  ⚠️ 恶意举报将受到处罚                         |
|                                               |
|    [取消]              [提交举报]              |
+----------------------------------------------+
```

**举报规则**：
- 每名玩家每天最多举报3次（防止滥用）
- 提交时自动附带：回放ID、双方玩家ID、战斗时间戳、回合数据快照
- 举报记录存储于 localStorage，同步至服务端（如有）
- 举报后回放保留期延长至30天（便于核查）
- 恶意举报处罚：3次恶意举报后，7天内不可使用举报功能

**举报反馈**：
- 提交后弹出：「举报已提交，感谢维护游戏公平！」
- 处理结果通过邮件通知举报者：「您举报的异常行为已核实/未发现异常」

### 4. 公会聊天（异步留言板）

#### 概念
- 非实时聊天，采用留言板模式
- 消息保留48小时，最多显示最近100条

#### 界面
- 竹简风格留言板
- 每条消息：头像+昵称+时间+内容
- 预设快捷消息：「求援」「借将」「打BOSS」「日常问候」
- 可发送资源请求/借将请求（特殊样式卡片）
- 盟主/军师可置顶公告

## 动效
- 好友上线：社交图标闪绿光+提示
- 排名上升：条目上滑+古铜金边框闪烁
- 赠送成功：飞鸽动画
- 借将出征：被借武将头像飞入编队（300ms）
- 援助到达：资源图标从屏幕边缘飞入仓库
- 切磋回放就绪：邮件图标震动+剑光特效
- 公会BOSS击败：全屏烟花+公会名横幅展示
- **v2.1 新增**：
  - 友情点满额：进度条金色脉冲+「已满」徽章弹入
  - 冷却期倒计时：锁链图标缓慢旋转动画
  - 援助减半提示：资源图标分裂为两半，半数淡出（500ms）
  - 举报提交：卷轴封印动画（信件折叠+火漆封口，400ms）
