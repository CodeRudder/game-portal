> **创建背景**：Round 2 评测发现 P1-15 问题——"天命""求贤令""招贤榜"等货币名称在不同文档中混用，命名不统一。本文档统一所有货币定义，消除歧义。
> **引用关系**：所有模块中涉及货币的内容均以本文档为准。

## 二、各货币详细定义

### 2.1 💰 铜钱（基础资源货币）

| 属性 | 定义 |
|------|------|
| **定位** | 游戏内最基础的流通货币，用于绝大多数日常消耗 |
| **上限** | 无上限 |
| **获取途径** | ① 建筑产出（钱庄，主要来源）<br>② 领土收益（占领领土持续产出）<br>③ 关卡通关奖励<br>④ 任务奖励（日常/周常/成就）<br>⑤ 远征战利品<br>⑥ 出售装备/道具<br>⑦ 离线收益 |
| **消耗途径** | ① 建筑升级（主要消耗）<br>② 科技研究<br>③ 装备强化/炼制<br>④ 武将升级（经验书购买）<br>⑤ NPC交易<br>⑥ 建筑维护费 |
| **防通胀** | ① 建筑升级费用指数增长（每级×1.8）<br>② 科技研究费用递增<br>③ 装备强化费用随等级飙升<br>④ 建筑维护费随建筑等级增加 |

### 2.2 👑 天命（高级货币）

| 属性 | 定义 |
|------|------|
| **定位** | 转生后解锁的高级货币，代表"天命所归"，用于核心高级消耗 |
| **上限** | 无上限 |
| **解锁条件** | 第1次声望转生后解锁 |
| **获取途径** | ① 转生奖励（200~1000/次，随转生次数递增）<br>② 三星通关奖励（+50/阶段首次全三星）<br>③ 每日活跃度满100（+10）<br>④ 公会BOSS击败奖（+20/次）<br>⑤ 成就里程碑（50~500，一次性）<br>⑥ 天命阁建筑产出（转生后解锁） |
| **消耗途径** | ① 高级招贤（求贤令×100/次）<br>② 转生加速道具（×50，缩短冷却12h）<br>③ 稀有装备兑换（×200~1000）<br>④ 神话武将碎片兑换（×150/片）<br>⑤ 跨服远征门票（×100/次） |
| **是否付费** | **非付费**。不可通过充值直接获取 |
| **转生保留** | 每次转生仅保留30% |
| **防通胀** | ① 转生保留率仅30%（大量消耗存量）<br>② 高级招贤保底机制大量消耗<br>③ 每日获取量有软上限 |

### 2.3 📜 招贤榜（普通抽卡代币）

| 属性 | 定义 |
|------|------|
| **定位** | 普通武将招募的消耗代币 |
| **上限** | 99（堆叠上限） |
| **获取途径** | ① 每日免费招贤1次（不消耗代币）<br>② 日常任务奖励（+1~2/天）<br>③ 活动奖励<br>④ 关卡通关奖励<br>⑤ 商店购买（💰500/张） |
| **消耗途径** | ① 普通招贤（消耗×1/次） |
| **产出武将** | Uncommon(60%) / Rare(30%) / Epic(9%) / Legendary(1%) |
| **防通胀** | ① 堆叠上限99<br>② 每日获取量有限（约3-5张）<br>③ 普通池武将价值有限，自然消耗 |

### 2.4 🏷️ 求贤令（高级抽卡代币）

| 属性 | 定义 |
|------|------|
| **定位** | 高级武将招募的消耗代币，可视为天命的等价物 |
| **上限** | 9,999（堆叠上限） |
| **获取途径** | ① 天命兑换（100天命 = 100求贤令）<br>② 元宝购买（💎100 = 100求贤令）<br>③ 活动奖励（限时活动大量产出）<br>④ 充值礼包（主要付费获取途径） |
| **消耗途径** | ① 高级招贤（消耗×100/次） |
| **产出武将** | Rare(40%) / Epic(35%) / Legendary(20%) / Mythic(5%) |
| **保底机制** | 每100次高级招贤必出Legendary+武将 |
| **是否付费** | **可通过充值获取**（元宝购买） |
| **防通胀** | ① 保底机制确保大量消耗<br>② 100/次的高单价 |

### 2.7 🌟 声望值（声望货币）

| 属性 | 定义 |
|------|------|
| **定位** | 声望/转生系统的核心数值，非流通货币 |
| **上限** | 无上限（累积制） |
| **获取途径** | ① 通关新关卡（50~500/关）<br>② 三星通关奖励（×1.5倍）<br>③ 成就奖励<br>④ 活动奖励 |
| **消耗途径** | 不直接消耗，用于：① 达到阈值解锁转生<br>② 声望等级决定加成系数 |
| **转生处理** | 转生后声望值重置为0，重新累积 |
| **防通胀** | ① 转生重置机制<br>② 阈值递增（每次转生需要更多声望） |

### 2.8 💎 元宝（付费货币）

| 属性 | 定义 |
|------|------|
| **定位** | 唯一付费货币，用于购买高级道具和加速服务 |
| **上限** | 999,999 |
| **获取途径** | ① 充值（主要途径）<br>② 成就奖励（少量）<br>③ 每日签到（少量，+5/天）<br>④ 新手礼包（一次性，+50） |
| **消耗途径** | ① 购买求贤令（💎100 = 100求贤令）<br>② 建筑加速（按剩余时间计费）<br>③ 科技加速<br>④ 购买资源礼包<br>⑤ 购买体力/扫荡令 |
| **汇率** | 1元人民币 ≈ 10元宝（基础汇率） |
| **防通胀** | ① 付费货币天然防通胀<br>② 消耗项定价确保付费价值 |

## 三、货币汇率关系

### 3.1 官方汇率表

| 兑换方向 | 汇率 | 说明 |
|---------|------|------|
| 💎 → 🏷️ | 100💎 = 100求贤令 | 固定汇率 |
| 👑 → 🏷️ | 100天命 = 100求贤令 | 固定汇率 |
| 💰 → 📜 | 500铜钱 = 1招贤榜 | 商店购买价 |
| 💎 → 资源 | 按礼包定价 | 浮动，礼包更优惠 |
| ⚔️ → 物品 | 按商店定价 | 远征商店专属 |
| 🏛️ → 物品 | 按商店定价 | 公会商店专属 |

### 3.2 等价换算参考

```
以"高级招贤1次"为基准单位（= 100求贤令）：

100求贤令 = 100💎 ≈ 10元人民币
100求贤令 = 100天命（天命兑换）
100求贤令 = 50,000铜钱（间接，通过购买招贤榜×100）

即：1天命 ≈ 500铜钱（间接等价）
```

## 四、防通胀机制总览

### 4.1 三级防通胀体系

| 层级 | 机制 | 适用货币 |
|------|------|---------|
| **L1 消耗膨胀** | 升级/强化费用指数增长，自动消耗过剩货币 | 铜钱、天命 |
| **L2 获取限制** | 每日获取量有软上限，堆叠有硬上限 | 招贤榜、远征币、公会币、活动代币 |
| **L3 重置机制** | 转生时部分货币按比例重置 | 天命（保留30%）、声望值（归零） |

### 4.2 各货币防通胀详情

| 货币 | 主要防通胀手段 | 预期效果 |
|------|-------------|---------|
| 💰 铜钱 | 升级费用指数增长（×1.8/级） | 中后期铜钱持续紧缺 |
| 👑 天命 | 转生保留30% + 高价消耗项 | 天命始终稀缺 |
| 📜 招贤榜 | 堆叠上限99 + 每日获取有限 | 自然消耗，不囤积 |
| 🏷️ 求贤令 | 100/次高单价 + 保底机制 | 大量消耗，付费驱动 |
| ⚔️ 远征币 | 每周获取上限 + 商店定期刷新 | 周期内消耗 |
| 🏛️ 公会币 | 每日获取上限 + 退出不获取 | 持续参与驱动 |
| 🌟 声望值 | 转生重置 + 阈值递增 | 每次转生重新累积 |
| 💎 元宝 | 付费货币天然限制 | 付费控制 |
