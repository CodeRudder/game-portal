### 子Tab设计规范

**Tab栏样式** (仅复杂建筑显示):

```
┌─ Tab栏 ───────────────────────────────────┐
│  [概览]    [功能]                            │
│   活跃态: 底部2px金色线条 + 文字#c8a96e      │
│   非活跃: 文字rgba(200,169,110,0.5)          │
│   悬停: 文字#c8a96e + 背景微亮               │

  背景: rgba(30,35,48,0.95)
  分隔线: 底部1px solid rgba(200,169,110,0.2)
```

**Tab切换交互**:
- 点击Tab标签切换内容区
- 切换动画: 内容区淡入淡出(150ms)
- 默认显示: 概览Tab
- 记忆: 记住用户最后选择的Tab(会话内)
- **按钮联动**: 底部按钮随Tab切换而变化（概览Tab→升级按钮，功能Tab→功能按钮）

|------|-----------|---------|-----------|
| 🌾 农田 | 资源系统、离线系统 | `resource` / `offline` | 产出速率、库存上限、离线效率 |
| 💰 市集 | 货币系统、商店系统 | `currency` / `shop` | 铜钱产出、商品价格、购买折扣 |
| ⚔️ 兵营 | 远征系统、英雄系统 | `expedition` / `hero` | 编队管理、兵力训练、出征状态 |
| 🔨 铁匠铺 | 装备系统(材料) | `equipment` | 材料产出、工坊消费联动、强化折扣 |
| 📚 书院 | 科技系统 | `tech` | 科技点产出、研究队列、科技树 |
| 🏥 医馆 | 战斗系统(伤兵) | `battle` | 伤兵数量、恢复速度、预计恢复时间 |
| 🏯 城墙 | 地图系统(攻城) | `map` | 城防值、守城Buff、陷阱库存 |
| 🎯 招贤馆 | 英雄系统 | `hero` | 招募概率、卡池武将、保底计数 |
| ⚒️ 工坊 | 装备系统(炼制) | `equipment` | 装备品质、属性词条、分解回收 |
| 🚢 市舶司 | 贸易系统 | `trade` | 商路、商品行情、商队、NPC商人 |

**设计原则**:
1. 建筑UI展示的数据必须来自子系统，不能是独立的假数据
2. 建筑操作的结果必须写入子系统状态
3. 建筑间的联动通过子系统数据流实现，而非建筑内部逻辑

### 简单建筑互动设计补充

> 以下简单建筑虽无Tab，但需展示子系统关联数据和资源流向

**市集（连接货币系统/商店系统）**:

| 互动元素 | 说明 | 数据来源 |
|---------|------|---------|
| 铜钱产出 | 当前产出速率+加成明细 | `currency` 系统 |
| 资源流向 | 铜钱→招募/商队/强化/购买 | 资源链 BLD-F28-02/03 |
| 贸易折扣 | 市舶司提供的购买折扣 | `shop` 系统 |
| 繁荣度加成 | 市舶司繁荣度→铜钱产出加成 | `trade` 系统 |

**铁匠铺（连接装备系统-材料）**:

| 互动元素 | 说明 | 数据来源 |
|---------|------|---------|
| 材料产出 | 每小时产出材料，库存容量=产出×24h | `equipment` 材料 |
| 工坊消费 | 材料→工坊炼制装备，显示消费速率 | 资源链 BLD-F28-02 |
| 强化折扣 | 铁匠铺等级→装备强化材料消耗减少 | `equipment` 强化 |
| 材料库存 | 当前库存/最大容量，进度条显示 | 材料仓库 |

**书院（连接科技系统）**:

| 互动元素 | 说明 | 数据来源 |
|---------|------|---------|
| 科技点产出 | 当前产出速率+加成明细 | `tech` 系统 |
| 研究队列 | 当前研究项目+进度+剩余时间 | `tech` 研究 |
| 科技树预览 | 可研究科技列表+前置条件 | `tech` 树 |
| 全建筑加成 | 已研究科技对各建筑的加成效果 | 科技效果回流 |

**城墙（连接地图系统-攻城）**:

| 互动元素 | 说明 | 数据来源 |
|---------|------|---------|
| 城防值 | 当前城防值+防御加成百分比 | `map` 攻城系统 |
| 守城Buff | 城墙等级→守城时属性加成 | 攻城防御计算 |
| 陷阱库存 | 可用陷阱数量+类型 | 城墙防御设施 |
| 最近攻防 | 最近一次攻城记录(如有) | 攻城战斗记录 |
