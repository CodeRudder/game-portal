## 一、全局层级定义

| 层级 | 含义 | 界面类型 | 尺寸范围 | 示例 |
|------|------|---------|---------|------|
| L0 | 游戏窗口 | 完整窗口 | PC:1280×800 / 手机:375×667 | — |
| L1 | 主Tab | B区Tab切换C区场景 | C区: PC 1280×696 / 手机 375×507 | 天下/出征/武将/科技/建筑/声望 |
| L1.5 | 更多菜单 | B区下拉→C区/D区/E区 | 同L1或D/E区 | 背包/商店/任务/好友/设置等 |
| L2 | Tab内子Tab | C区场景内子导航 | C区全宽×(696-36-48)=612px高 | 武将→[已招募/名册/招募] |
| L3 | 详情面板/全屏 | D区面板/F区全屏 | D:360~480×696 / F:1280×696 | 武将详情/战前布阵/NPC对话 |
| L4 | 操作弹窗 | E区居中弹窗 | 320~600×200~500px | 装备选择/技能升级/确认操作 |
| L5 | Toast/飘字 | 顶部居中悬浮 | PC:400×40 / 手机:300×36, 3秒消失 | 升级成功/装备穿戴/资源获得 |
