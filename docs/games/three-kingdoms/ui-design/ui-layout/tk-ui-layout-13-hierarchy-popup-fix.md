# 三国霸业 UI布局设计 — 功能层级重构+弹窗精简策略

> **版本**: v2.0 | **日期**: 2026-04-18
> **修复**: R5-问题2（减少弹窗）+ R5-问题4（弹窗关闭/遮挡）+ R5-问题5（功能层级正确性）
> **核心原则**: 功能不混杂、层级清晰、弹窗最少化、所有弹窗可关闭

---

## 一、功能层级定义

### 1.1 层级结构总览

```
L0  游戏主窗口（资源栏A + 导航栏B + 场景区C）
│
├─ L1  主Tab（B区Tab切换，C区全屏场景）
│   ├─ 天下 → 世界地图
│   ├─ 出征 → 战役长卷
│   ├─ 武将 → 武将管理（含子Tab）
│   ├─ 科技 → 科技树
│   ├─ 建筑 → 城池建设
│   └─ 声望 → 声望系统
│
├─ L1.5 更多菜单功能（按功能类型分区域打开）
│   ├─ C区全屏: 背包、商店、公会、远征、竞技场、贸易路线
│   ├─ D区面板: 任务、好友、排行榜、NPC名册、聊天、邮件
│   └─ E区弹窗: 活动、设置
│
├─ L2  Tab内子Tab（C区内子导航）
│   ├─ 武将Tab: [已招募] [武将名册] [招募武将]
│   ├─ 声望Tab: [每日任务] [成就] [转生]
│   ├─ 公会: [信息] [成员] [公会战] [商店]
│   └─ 远征: [选择路线] [进行中] [历史]
│
├─ L3  详情面板/弹窗（信息展示为主）
│   ├─ D区右侧面板: 武将详情、建筑详情
│   ├─ E区居中弹窗: 关卡详情、科技详情、装备详情
│   └─ F区全屏: 战前布阵、战斗场景、NPC对话
│
├─ L4  操作弹窗（在L3内触发的子操作）
│   ├─ 武将选择列表、装备选择列表
│   ├─ 技能升级、突破、升星界面
│   ├─ 强化界面、交易界面
│   └─ 确认弹窗（仅重大决策）
│
└─ Toast  即时反馈（3秒消失，不遮挡）
    └─ 升级成功、装备穿戴、资源获得、操作完成
```

### 2.1 核心原则

**规则1**: 能用Toast解决的，不用弹窗
**规则2**: 能在面板内展开的，不弹窗
**规则3**: 只有不可逆操作才需要确认弹窗
**规则4**: 所有弹窗必须有[×]关闭按钮

## 三、弹窗关闭/折叠统一规范

### 3.1 各区域关闭规范

#### D区右侧面板（360~480×696px）
```
┌─ 面板标题 ──────────────────── [—] [×] ┐
│                                          │
│  内容区域                                │
│                                          │
└──────────────────────────────────────────┘

关闭方式（3种）：
1. [×] 按钮 - 右上角 20×20px，面板滑出右侧消失
2. [—] 折叠 - 面板折叠为底部小条(60×40px)，点击恢复
3. ESC键 - PC端快捷关闭

样式规范：
- [×]: rgba(255,255,255,0.4) → 悬停 rgba(255,80,80,0.8)
- [—]: rgba(255,255,255,0.4) → 悬停 rgba(255,255,255,0.8)
- 折叠后小条: 半透明背景+面板图标，点击展开
```

#### E区居中弹窗（320~600×200~500px）
```
╔════════════════════════════════════╗
║  弹窗标题                  [×]    ║
║                                    ║
║  内容区域                          ║
║                                    ║
╚════════════════════════════════════╝

关闭方式（3种）：
1. [×] 按钮 - 右上角 16×16px
2. 点击遮罩层 - rgba(0,0,0,0.5)半透明遮罩
3. ESC键 - PC端快捷关闭

遮罩规范：
- 遮罩层 rgba(0,0,0,0.5)
- 点击遮罩 = 关闭弹窗
- 设置弹窗除外（防止误关闭丢失修改）
```

#### F区全屏覆盖（1280×696px）
```
┌─ [←返回] 标题 ──────────────────────┐
│                                       │
│  全屏内容                             │
│                                       │
└───────────────────────────────────────┘

关闭方式（2种）：
1. [←返回] 按钮 - 左上角，返回上一级
2. ESC键 - PC端快捷返回

注意：F区无遮罩层，本身就是全屏内容
```

### 3.2 特殊弹窗关闭规则

| 弹窗类型 | 关闭规则 | 原因 |
|---------|---------|------|
| 确认弹窗 | [×]关闭=取消操作 | 防止误操作 |
| 战斗结算 | [×]关闭=返回出征 | 结算已自动发放 |
| 新手引导 | 不可关闭 | 必须完成引导 |
| NPC对话 | [←返回]关闭 | 返回NPC名册 |
| 设置 | [×]关闭需确认 | 防止丢失修改 |

### 3.3 升级提示修复方案

| 原问题 | 修复方案 |
|--------|---------|
| 升级悬浮框无法关闭遮挡地图 | 改为Toast通知(3秒消失) |
| 升级进度无反馈 | 建筑卡片上显示小型进度条 |
| 升级完成无感知 | 建筑卡片闪烁+Toast提示 |
| 资源不足无提示 | 升级按钮灰色+面板内红色文字提示 |

### 5.2 弹窗比例分析

- **确认弹窗**: 11个（仅占11%）— 只用于不可逆操作
- **操作弹窗**: 37个（37%）— 功能性弹窗（详情/选择/操作）
- **面板**: 10个（10%）— 信息展示面板（可折叠）
- **Toast**: 42个（42%）— 即时反馈（不遮挡）

**对比原设计**：原设计约60%操作需要确认弹窗，新设计仅11%需要确认，其余89%为直接执行+Toast或功能性弹窗。
