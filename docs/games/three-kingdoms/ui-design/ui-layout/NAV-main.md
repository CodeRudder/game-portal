### [NAV-1-1] 资源栏 (A区 1280×56px)
- 固定顶部，z-index:100
- 内容: [三国霸业] [🌾1000/2000 ▲+30/s] [💰500 ▲+20/s] [⚔️200/500 ▲+5/s] [👑50] [📊]
- 点击资源→Tooltip显示详情(2秒消失)
> 📖 → [PRD: 资源系统](../prd/RES-resources-prd.md#res-1)

### [NAV-1-2] 导航Tab栏 (B区 1280×48px)
- 固定顶部，z-index:90
- Tab: [天下] [出征] [武将] [科技] [建筑] [声望] [更多▼]
- 点击Tab→切换C区场景内容
> 📖 → [PRD: 主界面导航](../prd/NAV-main-prd.md#nav-2)

### [NAV-1-3] 场景区 (C区 1280×696px)
- Tab内容区，z-index:10，根据当前Tab显示不同场景

### [NAV-1-4] 右侧面板区 (D区 360~480×696px)
- 从右侧滑入，z-index:50
- [×]关闭 + [—]折叠 + ESC关闭
- 用于: 武将详情/建筑详情/任务/好友/排行榜/NPC名册/聊天/邮件
> 📖 → [PRD: 交互规范](../prd/SPEC-interaction.md#itr-3)

### [NAV-1-5] 居中弹窗区 (E区 320~600×200~500px)
- 居中显示，遮罩rgba(0,0,0,0.5)，z-index:60
- [×]关闭 + 点击遮罩关闭 + ESC关闭
> 📖 → [PRD: 交互规范](../prd/SPEC-interaction.md#itr-4)

### [NAV-1-6] 全屏覆盖区 (F区 1280×696px)
- 完全覆盖场景区，z-index:70
- [←返回]左上角 + ESC返回
- 用于: 战斗场景/战前布阵/NPC对话/剧情/引导

### [NAV-1-7] 悬浮组件区 (G区)
- z-index:40(弹窗之下)
- 右下角: 军师建议面板240×320px [×]关闭 [—]最小化
- 右下角: 离线收益按钮48×48px
- 左下角: 任务追踪280×auto(最多3条)

## [NAV-4] 全局导航路径 {#nav-4}

> 完整路径树见 [NAV-4 全局导航路径](NAV-main.md#nav-4)（本节即为完整路径树）

**关键路径规则**:
- 战前布阵→调整阵容: 在布阵内弹武将选择列表，**不跳转武将系统**
- 武将名册: 武将Tab→[武将名册]子Tab，**不在地图右侧**
- 建筑升级: 直接执行+Toast，**无确认弹窗**
- 装备穿戴: 直接穿戴+Toast，**无确认弹窗**
