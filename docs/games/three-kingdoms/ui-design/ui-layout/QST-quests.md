## [QST-3] 任务完成Toast {#qst-3}

任务完成时自动弹出，3秒后消失。位置：屏幕顶部中央。

```
┌──────────────────────────────────┐
│  ✅ 任务完成！  击败黄巾军首领    │
│  +100🌾  +5000💰  +2🧪           │
└──────────────────────────────────┘
  宽度 auto（最大360px），高度 56px
  背景: rgba(44,24,16,0.95) + 金色边框 1px
  弹入: 300ms ease-out 从顶部滑入
  消失: 300ms ease-in 向上滑出
```

**动画序列**（总时长 2.5s）：
1. 0ms → 任务卡片边框闪烁古铜金（200ms × 3次）
2. 600ms → 青铜印章从上方落下盖在卡片上（300ms bounce）
3. 900ms → 墨迹从印章中心飞溅（8颗墨滴）
4. 1500ms → Toast从顶部弹入（400ms ease-out）
5. 2500ms → Toast自动消失

## 动效规范

| 动效 | 时长 | 缓动 | 触发条件 |
|------|------|------|---------|
| 面板滑入 | 300ms | `ease-out` | 打开任务面板 |
| 面板滑出 | 250ms | `ease-in` | 关闭任务面板 |
| Tab切换 | 200ms | `ease-in-out` | 切换分类 |
| 新任务插入 | 400ms | `ease-out` | 新任务解锁 |
| 进度条流光 | 600ms | `linear` | 进度更新 |
| 印章盖下 | 300ms | `ease-out bounce` | 任务完成 |
| 奖励飞出 | 600ms | `cubic-bezier(0.4,0,0.2,1)` | 一键领取 |
| 追踪条更新 | 200ms | `ease-out` | 进度变化 |
| 高亮脉冲 | 3000ms | `ease-in-out` 循环 | 跳转后引导 |
| 手机端Bottom Sheet弹入 | 300ms | `ease-out` | 打开面板 |
