## [EXP-1-1] 路线节点 (60×60px) {#exp-1-1}

每个路线节点的详细规格与状态样式。

```
  已完成节点          当前节点            锁定节点
 ┌─────────┐      ┌─────────┐      ┌─────────┐
 │  ╭───╮  │      │  ╭───╮  │      │  ╭───╮  │
 │  │ ✓ │  │      │  │ ● │  │      │  │🔒 │  │
 │  ╰───╯  │      │  ╰┳┳╯  │      │  ╰───╯  │
 │  黄巾寨 │      │  波才营 │      │  广宗城 │
 │  ★★    │      │  ★★★  │      │  ★★★★ │
 └─────────┘      └─────────┘      └─────────┘
   灰色边框          金色脉动           灰色+锁
   绿色✓标记         发光效果           主城Lv12
```

**节点规格**：
| 属性 | 已完成 | 当前 | 锁定 |
|------|--------|------|------|
| 尺寸 | 60×60px | 60×60px | 60×60px |
| 边框 | 2px solid #4CAF50 | 2px solid #FFD700 + glow | 2px solid #666 |
| 图标 | ✓ 绿色对勾 | ● 金色圆点+脉动 | 🔒 灰色锁 |
| 名称 | 节点名（下方12px字） | 同左 | 同左+解锁条件Tooltip |
| 难度 | ★~★★★★ | 同左 | 同左 |
| 交互 | 点击→查看战报 | 点击→进入战斗/查看详情 | 点击→Tooltip显示解锁条件 |

**节点类型**：
- 🏰 **城池战**：常规战斗节点
- ⚔️ **遭遇战**：随机事件+战斗
- 💰 **资源点**：无战斗，获取资源
- 👑 **BOSS战**：章节最终节点，难度最高
