### [MAP-1-1] 工具栏 (36px)
- 筛选: 阵营/地形/收益下拉
- 热力图: 开/关切换
- 行军模式: 点击进入行军选择模式
- 撤退: 取消当前行军
- 缩放: 显示当前缩放比例
- 时间: 游戏内时间(影响昼夜)

### [MAP-1-2] 像素地图 Canvas
- **渲染方式**: 逐色块(8×8px)绘制，非网格布局
- **地形**: 有机分布，过渡自然
- **城市**: 独特像素图标 + 阵营旗帜
- **道路**: 3px土黄色路径连接城市
- **行军**: 精灵队列沿道路移动
- **交互**:
  - 点击己方城市 → 领土详情面板
  - 点击敌方城市 → 攻城确认弹窗
  - 点击空白 → 取消选择
  - 拖拽 → 平移地图
  - 滚轮 → 缩放

### [MAP-1-3] 小地图 (180×140px)
- 全局缩略图(1px=2色块)
- 当前视口白色方框
- 城市标记(阵营色圆点)
- 行军路线(黄色线)
- 点击跳转

### [MAP-5-1] 天下地图攻城特效叠加层
- 攻城动画直接渲染在PixelWorldMap Canvas上，不切换场景
- 行军精灵在攻城期间保留不消失，切换为攻城动作(攻击姿态+武器动画)
- 目标城池周围显示策略对应特效:
  - 强攻: 撞车撞击城门 + 箭雨(叠加在城池像素图标上方)
  - 围困: 军队围城环形排列 + 城内炊烟减少
  - 夜袭: 地图局部变暗 + 火把光圈 + 翻墙精灵
  - 内应: 城门打开动画 + 军队涌入
- 守军在城墙上射箭(城墙像素图标上方箭矢精灵)
- 攻方军队在城下推进(行军精灵切换为战斗帧)
- 特效范围限定在城池周围约64×64px区域，不影响地图其他部分

### [MAP-5-2] 攻城战况条(地图内悬浮HUD)
- 浮动在目标城池下方，不遮挡攻城动画
- 城防值实时显示(递减进度条)
- 攻守双方信息(将领/兵力/战力)
- 当前策略及预计剩余时间
- 背景半透明(alpha=0.7)，不遮挡地图
