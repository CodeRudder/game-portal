### [MAP-1-1] 工具栏 (36px)
- 筛选: 阵营/地形/收益下拉
- 热力图: 开/关切换
- 行军模式: 点击进入行军选择模式
- 撤退: 取消当前行军
- 缩放: 显示当前缩放比例
- 时间: 游戏内时间(影响昼夜)

### [MAP-1-2] 像素地图 Canvas
- **渲染方式**: 逐色块(8×8px)绘制，非网格布局
- **地形**: 有机分布，过渡自然
- **城市**: 独特像素图标 + 阵营旗帜
- **道路**: 3px土黄色路径连接城市
- **行军**: 精灵队列沿道路移动
- **交互**:
  - 点击己方城市 → 领土详情面板
  - 点击敌方城市 → 攻城确认弹窗
  - 点击空白 → 取消选择
  - 拖拽 → 平移地图
  - 滚轮 → 缩放

### [MAP-1-3] 小地图 (180×140px)
- 全局缩略图(1px=2色块)
- 当前视口白色方框
- 城市标记(阵营色圆点)
- 行军路线(黄色线)
- 点击跳转
