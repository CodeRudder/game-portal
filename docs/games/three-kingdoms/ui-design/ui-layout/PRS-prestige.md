## [PRS-3-S] 声望商店面板 (D区 420×696px) {#prs-3-s}

> 📖 **玩法设计**: → [PRD: 声望商店兑换](../prd/PRS-prestige-prd.md#prs-4)

### 功能描述

声望商店使用声望货币（👑）兑换稀有道具，通过声望等级解锁更多商品。声望商店从声望分栏场景左栏底部的「商店」按钮进入，以侧滑面板形式展开。

```
┌─ 声望商店 ──────────────── [—] [×] ─┐
│  👑声望: 12,800                      │
│                                       │
│  ┌──────┐ ┌──────┐ ┌──────┐         │
│  │武将  │ │装备  │ │资源  │ Tab      │
│  └──────┘ └──────┘ └──────┘         │
│                                       │
│  ┌─────────────────────────────┐     │
│  │ 🎭 传说武将碎片(随机)       │     │
│  │ 👑 5000 | 已购 0/1          │     │
│  │ [购买]                       │     │
│  └─────────────────────────────┘     │
│  ┌─────────────────────────────┐     │
│  │ ⚔️ 橙色装备宝箱             │     │
│  │ 👑 3000 | 已购 0/2          │     │
│  │ [购买]                       │     │
│  └─────────────────────────────┘     │
│  ┌─────────────────────────────┐     │
│  │ 🌾 粮草×5000                │     │
│  │ 👑 500 | 已购 3/10          │     │
│  │ [购买]                       │     │
│  └─────────────────────────────┘     │
└───────────────────────────────────────┘
  侧滑面板 420×696px
  从左栏底部「🛒 商店」按钮触发
  关闭方式: [×] / [—]最小化 / ESC
```
