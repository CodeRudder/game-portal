### [MAP-1-1] 筛选工具栏 (36px)
- 可折叠，背景rgba(13,17,23,0.85)
- 筛选: 全部/魏/蜀/吴 + 地形/收益/等级下拉 + 热力图开关
> 📖 → [PRD: 地图筛选](../prd/MAP-world-prd.md#map-2)

### [MAP-1-2] 六边形瓦片地图 (2400×1800px)
- 20×15六边形瓦片，可拖拽+缩放
- 点击己方领土→[MAP-2]领土详情
- 点击资源点→直接采集+Toast
- 点击敌对城市→攻城确认弹窗(E区 480×350px)
> 📖 → [PRD: 地图交互](../prd/MAP-world-prd.md#map-3)

### [MAP-1-3] 小地图 (180×140px)
- 右下角悬浮，显示全局缩略图+当前位置标记
- 点击小地图→跳转到对应位置
