### [MAP-2-1] 领土产出详情展开 (E区 360×320px) {#map-2-1}

**触发**: 点击 [MAP-2] 领土详情面板中的"产出详情"区域

```
┌─ 许昌 产出详情 ────────── [×] ─┐
│  基础产出: 🌾+1.5/s 💰+1.0/s   │
│  科技加成: 🌾+0.4/s (农学I)     │
│  建筑加成: 🌾+0.2/s (农田Lv.3)  │
│  ──────────────────────         │
│  总计: 🌾+2.1/s 💰+1.0/s       │
│                                  │
│  占领时间: 3天12小时             │
│  下次产出提升: 农学II(需TEC-3)  │
└──────────────────────────────────┘
```

- **基础产出**: 领土等级对应的原始产出
- **科技加成**: 已研发科技带来的产出提升，标注科技名称
- **建筑加成**: 已建造建筑带来的产出提升，标注建筑名称和等级
- **总计**: 三项加总后的实际每秒产出
- **占领时间**: 当前领土已被占领的持续时间
- **下次产出提升**: 提示下一个可提升产出的途径（科技/建筑/升级）
> 📖 → [PRD: 领土产出](../prd/MAP-world-prd.md#map-4)

## [MAP-3] 攻城确认弹窗 (E区 480×380px) {#map-3}

**触发**: 点击敌对城市或 [MAP-2] 面板中的 [进攻] 按钮

```
┌─ ⚔️ 进攻许昌 ──────────── [×] ─┐
│  目标: 许昌 (Lv.5 魏🔴)        │
│  守军: 800/800                  │
│  城防: ████████░░ 80%           │
│                                  │
│  我方战力: 3,200                 │
│  预估胜率: 72%                   │
│                                  │
│  出战武将: 关羽 张飞 赵云        │
│  [调整阵容]                      │
│                                  │
│  [⚔️ 进攻]    [取消]            │
└──────────────────────────────────┘
```

- **目标信息**: 城市名称、等级、所属阵营
- **守军**: 当前驻军数/最大驻军数
- **城防**: 城防值百分比进度条，城防越低进攻越容易
- **我方战力**: 当前出战阵容总战力
- **预估胜率**: 基于战力对比自动计算，颜色编码(绿>60% / 黄40-60% / 红<40%)
- **出战武将**: 当前已选武将列表，点击 [调整阵容] 跳转战前布阵(CBT-2)
- **进攻按钮**: 确认后进入战前布阵→战斗流程
> 📖 → [PRD: 攻城战](../prd/MAP-world-prd.md#map-5)

## [MAP-4] 地图事件标记 {#map-4}

**触发**: 地图上随机刷新，以脉冲图标显示在对应瓦片上

### [MAP-4-1] 事件类型

| 事件类型 | 图标 | 刷新频率 | 奖励类型 |
|---------|------|---------|---------|
| 商队遇险 | 🐴 | 每2小时 | 💰金币 + 道具 |
| 流民 | 👥 | 每3小时 | 👷人口 + 🌾粮草 |
| 宝箱 | 🎁 | 每4小时 | 随机装备/材料 |
| 山贼 | ⚔️ | 每1.5小时 | 💰金币 + 经验 |

### [MAP-4-2] 事件交互弹窗 (E区 420×300px)

```
┌─ 🎁 宝箱事件 ──────────── [×] ─┐
│                                  │
│  你发现了一个古老的宝箱！        │
│  看起来里面藏着珍贵的宝物。      │
│                                  │
│  ── 选项 ──                     │
│                                  │
│  [🗡️ 强行打开]                  │
│   消耗: 战力500                  │
│   奖励: 蓝色装备×1 + 💰200      │
│                                  │
│  [🔑 精巧开锁]                  │
│   消耗: 开锁工具×1               │
│   奖励: 紫色装备×1 + 💰500      │
│                                  │
│  [🚶 离开]                      │
└──────────────────────────────────┘
```

- **事件描述**: 简短叙事文本，增强沉浸感
- **选项机制**: 每个事件2~3个选项，不同消耗对应不同奖励品质
- **脉冲动画**: 地图上事件图标以1.5s周期脉冲闪烁，吸引注意
- **自动消失**: 事件存在30分钟后自动消失，未领取的奖励丢失
> 📖 → [PRD: 地图事件](../prd/MAP-world-prd.md#map-6)

## [MAP-5] 地图统计面板 (D区 420×696px) {#map-5}

**触发**: 点击地图右上角统计图标📊

```
┌─ 地图统计 ────────────── [—] [×] ┐
│  领土: 12/50 (24%)               │
│  势力: 魏🔴 35% 蜀🟢 24% 吴🔵 28%│
│  ── 我方领土 ──                   │
│  许昌 Lv.5 🌾+2.1/s             │
│  成都 Lv.3 🌾+1.5/s             │
│  汉中 Lv.2 🌾+1.0/s             │
│  ── 总产出 ──                     │
│  🌾+4.6/s 💰+2.8/s ⚔️+0.5/s    │
└──────────────────────────────────┘
```

- **领土概览**: 当前占领数/总领土数 + 占比百分比
- **势力分布**: 三方势力占比，颜色对应阵营
- **我方领土列表**: 每个领土的名称、等级、主要产出，按产出排序
- **总产出汇总**: 所有领土产出加总，实时更新
- **面板操作**: [—] 折叠为标题栏 / [×] 关闭面板
