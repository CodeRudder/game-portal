## 二、科技研究时间体系 ★新增★

### 2.1 研究时间总览

> **设计原则**：科技研究时间与玩家在线时长挂钩，确保活跃玩家 2~3 周可完成单条路线，放置玩家依靠离线研究稳步推进。

| 节点等级 | 基础研究时间 | 对应游戏阶段 | 累计时间（单路线前 4 节点） |
|:-------:|:----------:|:----------:|:------------------------:|
| Lv1（节点 1） | 5 分钟 | 新手期（第 1 天） | 5 分钟 |
| Lv2（节点 2） | 15 分钟 | 新手期（第 1 天） | 20 分钟 |
| Lv3（节点 3） | 45 分钟 | 发展期（第 2~3 天） | 1 小时 5 分钟 |
| Lv4（节点 4） | 2 小时 | 发展期（第 3~5 天） | 3 小时 5 分钟 |
| Lv5（节点 5A/B） | 5 小时 | 中期（第 5~7 天） | 8 小时 5 分钟 |
| Lv6（节点 6A/B） | 10 小时 | 中后期（第 7~14 天） | 18 小时 5 分钟 |
| Lv7（节点 7A/B） | 20 小时 | 后期（第 14~21 天） | 38 小时 5 分钟 |
| 融合科技 | 15~30 小时 | 终局（第 21~28 天） | — |

### 2.2 科技点系统

> 科技研究依赖「科技点」产出，由书院建筑持续生成。科技点不足时无法启动研究。

| 参数 | 规则 |
|------|------|
| **科技点产出** | 书院等级决定科技点/秒（Lv1: 0.3/s → Lv20: 8.0/s） |
| **研究消耗** | 研究启动时消耗科技点 = 基础费用 × (1 - 书院研究速度加成) |
| **科技点存储** | 无上限，持续累积（离线也累积，受离线效率系数影响） |
| **研究启动条件** | ① 科技点足够 ② 前置科技已研究 ③ 无其他正在研究的科技 |

### 2.3 加速机制

| 加速方式 | 效果 | 消耗 | 限制 |
|---------|------|------|------|
| **铜钱加速** | 立即完成剩余时间的 50% | 铜钱 × 剩余时间（秒）× 0.5 | 无限制 |
| **天命加速** | 立即完成剩余时间的 80% | 天命 × Math.ceil(剩余时间/3600) | 每次研究最多用 1 次 |
| **元宝秒完成** | 立即完成全部剩余时间 | 元宝 × Math.ceil(剩余时间/1800) | 每日限 3 次 |
| **书院等级加速** | 研究速度+2%~+38%（被动） | 无（建筑升级） | 永久 |
| **声望等级加速** | 研究速度 × (1 + 声望等级×0.02) | 无（声望系统） | 永久 |
| **联盟加速** | 研究速度+10% | 联盟成员协助 | 需联盟功能 |

**加速叠加公式**：
```
实际研究时间 = 基础研究时间 / (1 + 书院加成 + 声望加成 + 联盟加成)
加速道具效果 = 实际研究时间 × (1 - 道具减免比例)
```

**加速示例**：
```
玩家：书院 Lv10（+18%）+ 声望 Lv5（+10%）
研究「铁骑冲锋」：基础 5h
实际时间 = 5h / (1 + 0.18 + 0.10) = 5 / 1.28 ≈ 3h 54min
使用天命加速（-80%）：3h 54min × 20% ≈ 47min
```

### 2.4 研究队列

| 参数 | 规则 |
|------|------|
| **默认队列** | 1 个研究槽位（同时只能研究 1 项科技） |
| **扩展槽位** | VIP3 解锁第 2 研究槽位 |
| **队列规则** | 当前研究完成后，队列中下一项自动开始 |
| **取消研究** | 返还 80% 已消耗资源，研究进度清零 |
| **切换研究** | 消耗 100 铜钱可暂停当前研究，切换到另一项（进度保留） |

## 三、科技离线研究规则 ★新增★

### 3.1 离线研究效率

> **引用全局规范**：`22-offline-reward-spec.md` §2~§3。科技系统离线行为与全局离线效率系数完全一致。

| 属性 | 规则 |
|------|------|
| **计算公式** | 研究剩余时间 - (离线秒数 × 离线效率系数) |
| **效率系数** | 引用全局公式：基础衰减 × (1 + Σ加成系数) |
| **封顶时长** | 72 小时（全局统一） |
| **完成处理** | 离线期间完成的科技研究正常结算，效果立即生效 |
| **科技点累积** | 离线期间书院科技点持续产出（受离线效率系数影响） |
| **特殊** | 科技离线效率 = 资源效率 × 1.0（无额外加成，与建筑升级 ×1.2 区分） |

### 3.2 离线研究计算示例

```
玩家状态：
  - 正在研究「锐兵器铸」（剩余 30 分钟）
  - 书院 Lv8（科技点 2.5/s）
  - 科技加成 +15%，声望 Lv3（+5%），无 VIP

离线 10 小时（36,000 秒）：
  1. 效率系数计算：
     - 0~2h: 100% → 7,200s × 100% = 7,200s
     - 2~8h: 80%  → 21,600s × 80% = 17,280s
     - 8~10h: 60% → 7,200s × 60% = 4,320s
     - 总有效时间 = 28,800s（8 小时）
     - 加成系数 = 15% + 5% = 20%
     - 最终有效时间 = 28,800 × 1.20 = 34,560s（9.6 小时）

  2. 研究进度：
     - 30 分钟 = 1,800s < 34,560s → 研究已完成 ✅
     - 剩余有效时间 32,760s 累积到科技点

  3. 科技点累积：
     - 书院产出 2.5/s × 34,560s = 86,400 科技点
```

## 四、科技与建筑/武将/资源联动关系 ★新增★

### 4.1 科技 ↔ 建筑联动

| 联动方向 | 具体行为 | 涉及科技 | 涉及建筑 |
|---------|---------|---------|---------|
| 建筑 → 科技速度 | 书院等级提供研究速度加成（+2%~+38%） | 所有科技 | 📚 书院 |
| 建筑 → 科技点 | 书院产出科技点/秒 | 所有科技 | 📚 书院 |
| 建筑 → 解锁条件 | 部分科技需对应建筑达到指定等级 | 屯田术/通商路/固城术 等 | 各对应建筑 |
| 科技 → 建筑产出 | 经济路线科技提升建筑产出效率 | 农耕改良/商贸繁荣/垦荒令 | 农田/市集 |
| 科技 → 建筑费用 | 治国安邦降低建筑升级费用-10% | 治国安邦 | 所有建筑 |
| 科技 → 建筑解锁 | 高级科技解锁建筑特殊功能 | 锻造术/活字印刷 | 铁匠铺/书院 |

### 9.2 研究完成动效

```

  1. 节点爆发金光粒子（0.5s，20 个粒子向外扩散）
  2. 节点边框从脉冲变为实色金色（300ms）
  3. 效果文字从节点飞向屏幕上方对应属性区（800ms，弧线轨迹）
  4. 解锁的后续节点亮起（500ms，从暗到亮渐入）
总时长：2.1s
音效：铜钟长鸣 + 金石交击声
```

### 9.3 互斥分支选择动效

```
触发：确认选择互斥分支

  1. 选中路线金光粒子从节点向外扩散（1s）
  2. 未选路线渐隐：节点缩小+透明度降至 30%（800ms ease-out）

  4. 虎符印章落下确认（300ms bounce）

  1. 两条/三条路线连接线同时亮起金色流光（1s）
  2. 流光汇聚到融合科技节点
  3. 节点从灰色爆炸式变为彩色（双色/三色渐变）
  4. 「新科技解锁」横幅从顶部滑入（Toast，3s 自动消失）
音效：编钟合奏 + 风声
```

## 十一、交互边缘场景

### 11.1 资源不足时研究

| 场景 | 处理方式 | UI 反馈 |
|------|---------|--------|
| 铜钱不足 | 无法启动研究 | 研究按钮灰化 + 「铜钱不足」提示 + [前往获取→] 链接 |
| 科技点不足 | 无法启动研究 | 研究按钮灰化 + 「科技点不足」提示 + [升级书院→] 链接 |
| 前置未完成 | 无法启动研究 | 节点锁定 + 显示前置依赖树 |
| 研究中再点其他 | 提示加入队列 | 「当前研究进行中，是否加入队列？」确认弹窗 |
| 队列已满 | 无法加入 | 「研究队列已满（VIP3 解锁第 2 槽位）」 |

### 11.2 网络异常

| 场景 | 处理方式 |
|------|---------|
| 研究启动断网 | 本地缓存请求，重连后自动提交，显示「同步中」 |
| 加速购买断网 | 加速效果本地立即生效，重连后同步服务器 |
| 科技树数据加载失败 | 使用缓存数据 + 黄条提示「数据可能不是最新」 |

### 11.3 转生后首次进入科技树

```
触发：转生后首次打开科技 Tab
表现：
  1. 短暂水墨晕染过渡（500ms）
  2. 保留的 6 项科技节点亮起（依次亮起，每项间隔 200ms）
  3. 重置的科技节点为灰色初始状态
  4. 互斥分支选择入口重新可用（无已选标记）
  5. 弹出提示：「轮回新生，前路可期。保留科技已生效，请重新规划路线。」
```
