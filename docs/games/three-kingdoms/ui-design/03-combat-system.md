### 3.3 半自动模式"大招时停"机制 ★核心新增★

**问题**：3-4个大招在2-3秒内集中就绪时，玩家逐个点击释放，但战斗未暂停，导致手忙脚乱。

**解决方案：大招时停（Ultimate Time Dilation）**

```
时停触发条件：
- 半自动模式下，同时≥2个大招就绪时触发
- 或单次大招就绪时已有未释放大招

时停效果：
- 战斗速度降至0.3×（慢动作，非完全暂停）
- 屏幕边缘出现水墨涟漪特效（提示时停状态）
- 大招集中释放栏高亮放大（scale 1.1）
- 时停持续最多5秒，超时后恢复正常速度

时停结束条件（任一）：
- 所有大招已释放完毕
- 5秒超时
- 玩家点击"跳过"按钮
- 未释放大招自动排队释放
```

**时停视觉设计**：
- 屏幕四边出现水墨涟漪扩散效果（暗色半透明）
- 顶部显示"时停"标识（太极图标+倒计时5→0）
- 大招释放栏从底部升起至更醒目位置
- 背景战斗动画以0.3×速度播放（武将动作明显变慢）

**时停交互流程**：
```
1. 检测到≥2个大招同时就绪
2. 触发时停（0.3×慢动作）
3. 大招集中释放栏显示所有就绪大招
4. 玩家依次点击释放（每次释放播放该武将大招动画）
5. 全部释放完毕 → 时停结束 → 恢复原速
6. 或5秒超时 → 未释放大招自动排队 → 恢复原速
```

### 3.4 战斗模式系统

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| **全自动** | 所有行动（普攻+大招）由AI自动释放 | 日常刷图、碾压局 |
| **半自动** | 普攻自动，大招手动释放（含时停机制） | 中等难度、策略关卡 |
| **全手动** | 所有行动需玩家手动指定 | 高难度BOSS、竞技场 |

**模式切换UI**：
- PC端：右上角三选一Toggle + 速度按钮组
- 手机端：底部一行紧凑按钮组
- 手动→自动切换增加二次确认（2秒内二次点击确认）
- 默认模式根据关卡难度自动选择：
  - 推荐战力 ≤ 我方战力×0.7 → 全自动
  - 推荐战力 ≤ 我方战力×1.0 → 半自动
  - 推荐战力 > 我方战力×1.0 → 全手动

## 五、一键布阵 / 智能编队推荐 ★核心新增★

### 5.1 一键布阵

**功能说明**：系统自动将战力最高的6个空闲武将填入当前阵型空位。

**规则**：
- 优先填充空位，不替换已有武将
- 按战力从高到低依次填入
- 已在其他编队的武将标记"已出战"，默认不选（可手动覆盖）
- 填入后显示总战力变化（如"战力+320 ↑"）

**交互**：
- PC端：战斗准备面板中的"一键布阵"按钮，点击即填充
- 手机端：底部操作栏的"一键布阵"按钮
- 填充动画：武将头像依次飞入阵型格（每个间隔100ms）

### 5.2 智能编队推荐

**推荐算法维度**：
1. **战力优先**：纯战力最高的6人组合
2. **阵营羁绊**：优先激活阵营加成（如3蜀+3蜀羁绊）
3. **职业搭配**：坦克+输出+治疗+控制的均衡阵容
4. **关卡克制**：根据关卡敌人特性推荐（如多灼烧→推荐高智/治疗）

**展示方式**：
- 提供1-3套推荐方案，每套显示总战力+匹配度百分比
- 匹配度 = 综合评分（战力×40% + 羁绊×25% + 克制×20% + 均衡×15%）
- 选择方案后自动填入，玩家可手动微调

## 八、离线战斗机制 ★核心新增★

### 8.1 离线推图规则

```
离线推图条件：
- 已设置"离线推图"开关（默认开启）
- 当前关卡为已三星通关的下一关
- 我方战力 ≥ 推荐战力×1.2（确保碾压）

离线推图规则：
- 每小时自动尝试1次推图
- 成功则推进到下一关（获得1星评价+基础奖励）
- 失败则停留在当前关卡
- 离线推图不消耗扫荡令
- 离线最多推进3关（防止过度推进错过三星）

离线收益：
- 已三星通关关卡的挂机收益正常累积
- 每小时产出 = 关卡基础产出 × (1 + 科技加成 + 声望加成)
- 离线收益上限 = 12小时产出（防止无限累积）
```

### 8.2 离线收益展示

- 回归时弹出"离线战报"面板：
  - 离线时长
  - 推进关卡数（如有）
  - 累积资源收益（各资源图标+数值）
  - "一键领取"按钮
- 面板风格：竹简卷轴展开动画
