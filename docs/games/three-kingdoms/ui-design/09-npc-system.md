## 一、地图NPC展示与拥挤管理

### 1.1 基础展示规则

- NPC以Q版立绘显示在主城地图上
- 头顶悬浮：职业图标（农民🌾 / 士兵⚔️ / 商人💰 / 学者📜 / 斥候🔭）+ 名字铭牌
- 空闲时头顶「！」（有任务）或「💬」（有对话）
- NPC按日程移动（巡逻/工作/休息），有行走动画

### 1.2 🔴 地图拥挤解决方案（P0-17）

**核心策略：地图仅展示"高关联"NPC，其余收纳至名册面板**

#### 展示优先级算法

```
地图NPC显示上限：8个（PC端）/ 5个（手机端）
优先级权重：
  - 有待领取任务：+100
  - 好感度≥4：+50
  - 距离玩家位置近（同一区域）：+30
  - 有新对话/剧情：+80
  - 特殊状态（受伤/求助）：+60
  - 稀有NPC（限时出现）：+70
  - 基础权重（按好感度等级）：好感度×10
```

#### 地图NPC折叠机制

| NPC数量 | PC端表现 | 手机端表现 |
|:-------:|---------|-----------|
| ≤5 | 全部显示 | 全部显示 |
| 6-8 | 全部显示，间距自动调整 | 显示前5 + "更多N"气泡 |
| 9-15 | 显示前8 + 右侧"名册"入口徽标（带数量角标） | 显示前5 + 底部"名册"浮动按钮 |
| >15 | 仅显示前8，其余完全隐藏，名册角标脉动提醒 | 仅显示前3高优先级 + "名册"按钮 |

#### NPC聚合气泡

当同区域NPC≥3个时，自动聚合为"人群气泡"：
- 气泡显示：职业图标群（最多3个）+ 数字角标
- 点击展开该区域NPC列表（半透明浮层）
- 浮层中可点击单个NPC进入交互

## 四、🔴 NPC离线行为设计（新增 P0-15）

### 4.1 离线行为规则

NPC在玩家离线期间持续按自身日程运作，产生以下行为：

| 行为类型 | 触发条件 | 离线产出 | 上限 |
|---------|---------|---------|------|
| **劳作产出** | 农民/商人自动工作 | 基础资源（粮草/铜钱） | 累积8小时 |
| **情报收集** | 斥候自动巡逻 | 地图情报碎片（解锁隐藏区域线索） | 每次离线最多3条 |
| **学术研究** | 学者自动研究 | 科技点数加速（×0.5倍率） | 累积4小时 |
| **巡逻警戒** | 士兵自动巡逻 | 降低被袭击概率（事件系统联动） | 全程生效 |
| **好感度自然增长** | 所有NPC | 好感度微量+（每小时+0.1） | 每次离线最多+2 |
| **NPC特殊事件** | 随机触发（概率5%/小时） | NPC个人剧情触发，回归后可查看 | 每次离线最多1个 |

### 4.3 NPC离线特殊事件

离线期间NPC可能触发个人剧情事件，回归后以"急报"形式推送：
- 触发条件：离线≥4小时 + 好感度≥3 + 随机概率
- 事件类型：NPC求助（需要资源）、NPC送礼（赠送稀有道具）、NPC引荐（解锁新NPC）
- 处理方式：回归后弹出NPC对话界面，玩家选择应对

## 五、好感度系统深化

### 5.1 好感度等级与效果

| 等级 | 所需点数 | 解锁效果 | 视觉标识 |
|:----:|:-------:|---------|---------|
| 1 | 0 | 基础对话 + 交易 | ⭐ 灰色 |
| 2 | 100 | 解锁「情报」选项 | ⭐⭐ 铜色 |
| 3 | 300 | 解锁「赠送」选项 + 商品折扣10% | ⭐⭐⭐ 银色 |
| 4 | 600 | 解锁「切磋」选项 + 专属任务线 | ⭐⭐⭐⭐ 金色 |
| 5 | 1000 | 解锁NPC羁绊技能 + 专属称号 | ⭐⭐⭐⭐⭐ 金色脉动 |

### 5.2 好感度获取途径

| 途径 | 每次获得 | 每日上限 |
|------|:-------:|:-------:|
| 日常对话 | +5 | 1次/NPC |
| 完成NPC任务 | +20-50 | 无上限 |
| 赠送礼物（普通） | +10 | 3次/NPC |
| 赠送礼物（稀有） | +30 | 1次/NPC |
| 切磋胜利 | +15 | 1次/NPC |
| 离线自然增长 | +0.1/小时 | +2/次 |

### 5.3 好感度进度可视化

- 进度条：渐变色（灰→铜→银→金），对应当前等级色
- 满级时：进度条替换为"羁绊已结"金色印章图标 + 脉动光效
- hover进度条显示：当前点数 / 下一级所需点数

## 七、动效规范

| 动效名称 | 触发场景 | 动画描述 | 时长 | 缓动 |
|---------|---------|---------|:----:|:----:|
| 弹窗展开 | NPC信息弹窗 | 卷轴从两端向中间展开 | 300ms | ease-out |
| 水墨涟漪 | NPC被选中 | 脚下水墨涟漪光圈扩散 | 500ms | ease-out |
| 好感飘心 | 好感度提升 | 头顶飘出❤️粒子（3-5个） | 800ms | ease-out |
| 拱手作揖 | 切磋胜利 | NPC拱手作揖动画 | 1000ms | ease-in-out |
| 名册滑入 | 打开名册面板 | 右侧面板滑入 + 地图微缩 | 300ms | ease-out |
| 聚合展开 | 点击人群气泡 | 气泡爆散为单个NPC图标 | 200ms | ease-out |
| 离线脉动 | 离线收益中有NPC事件 | NPC图标金色脉动 | 1500ms循环 | ease-in-out |
| 好感升级 | 好感度等级提升 | 星星逐个点亮 + 金色粒子爆发 | 600ms | ease-out |

### 性能降级策略

- 低端设备（FPS<30）：关闭粒子效果，仅保留基础位移动画
- 中端设备：减少粒子数量（3→2），关闭背景水墨涟漪
- 高端设备：全部动效开启
