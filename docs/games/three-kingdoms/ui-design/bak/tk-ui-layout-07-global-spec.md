## 八、全局引用规范（M21-M23/M27）

### 8.1 响应式断点（M21）
- Desktop-L ≥1920: 画布放大1.5×
- Desktop 1280-1919: 基准尺寸
- Tablet-L 1024-1279: 画布缩小，部分面板改为Bottom Sheet
- Mobile ≤767: 完全手机端布局

### 8.2 离线收益效率（M22）
- 0-2h: 100% / 2-8h: 80% / 8-24h: 60% / 24-48h: 40% / 48-72h: 25% / >72h: 15%

### 8.3 货币体系（M23）
- 基础: 💰铜钱 / 高级: 👑天命 / 抽卡: 📜招贤榜 / 付费: 💎元宝
- 活动代币: 🪙赛季令牌/日常代币/节庆灯笼/联盟勋章

### 8.4 精灵动画（M27）
- 建筑外观4级: 茅屋→瓦房→楼阁→宫殿
- 武将动画: 待机/攻击/受击/技能/死亡
- 战斗特效: 水墨风粒子系统
- 同屏粒子上限: PC 200 / 手机 100
