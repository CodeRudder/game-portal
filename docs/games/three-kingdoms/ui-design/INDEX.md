## 布局规范

### PC端（1280×800）
- **顶部**：游戏标题 + 资源栏（粮草/铜钱/兵力/天命，含容量进度条）
- **Header下**：水平Tab导航栏（天下/出征/武将/科技/建筑/声望/更多）
- **中央**：地图/主场景（占满剩余空间，无左右常驻面板）
- **右下角**：半透明小地图缩略图（仅地图场景显示）
- **右下角悬浮**：「军师建议」按钮，展开后显示1-3条动态推荐
- **底部**：快捷键提示栏

### 手机端（375×667）
- **顶部**：游戏标题 + 精简资源栏（含容量微型进度条）
- **资源栏下方**：可左右滑动的快捷图标条（任务/活动/背包/远征）
- **中央**：地图/主场景
- **右下角悬浮**：「军师建议」按钮
- **底部**：5个Tab导航栏（天下/出征/武将/内政/社交）
- 面板从底部弹出（Bottom Sheet）
