## 二、领土收益可视化系统

### 己方领土产出气泡（常驻显示）

每个己方领土上方常驻显示产出气泡：

```
      ┌──────────────┐
      │ 🌾+4.2/s     │  ← 主产出行（资源图标+数值）
      │ 💰+1.8/s     │  ← 副产出行（如有）
      │ 📊Lv.12      │  ← 领土等级
      └──────┬───────┘
             │          ← 连接线（2px古铜金）
      ┌──────┴───────┐
      │   领土区域    │
      │  (六边形)     │
      └──────────────┘
```

**气泡规格（PC端）**：
- **尺寸**：自适应宽度（最小70px，最大120px）× 高度自适应（单行24px/双行40px）
- **背景**：rgba(13,17,23,0.80) 夜墨色半透明 + 1px古铜金描边
- **圆角**：8px
- **连接线**：2px古铜金细线，从气泡底部中心到领土中心
- **字体**：13px 宣纸白，数值部分金色(#C9A84C)加粗

**显示逻辑**：
- **己方领土**：始终显示产出气泡
- **非己方领土**：不显示气泡（悬停时Tooltip显示潜在收益）
- **刚占领领土**：气泡从0开始充能动画（scale 0→1, 300ms bounce）
- **产出为0**：灰色气泡 `🌾+0/s`，提示需开发
- **缩放<60%时**：隐藏气泡避免拥挤，改为悬停显示

**性能优化**：
- 气泡使用离屏Canvas预渲染模板，仅数值文字动态绘制
- 视口外领土气泡不渲染（视口裁剪）
- 气泡数值每5秒批量更新
- 缩放级别变化时批量显示/隐藏

### 城池产出标识（特殊地标增强）

城池类领土（洛阳、长安、建业等）显示增强型产出标识：

- **尺寸**：160×80px（比普通气泡更大）

- **星标**：1~3颗星表示地标等级

## 三、地图筛选/过滤功能


**筛选栏规格**：
- **高度**：36px（展开时），0px（收起时）
- **背景**：rgba(26,35,50,0.90) 半透明墨玉青底
- **位置**：地图场景区顶部
- **收起/展开**：左侧「筛选」按钮切换，默认收起

**三组筛选维度 + 等级筛选 + 热力图**：

| 筛选组 | 筛选项 | 交互方式 | 逻辑 |
|--------|--------|---------|------|
| **阵营** | 全部 / 蜀(红) / 魏(蓝) / 吴(绿) / 中立(灰) | 单选 | AND |
| **地形** | 平原 / 山地 / 城池 / 森林 / 水域 / 关隘 | 多选 | OR |
| **收益类型** | 粮草🌾 / 铜钱💰 / 兵力⚔️ | 多选 | OR |
| **等级范围** | 全部 / Lv5+ / Lv10+ / Lv15+ | 单选 | AND |
| **热力图** | 开/关 | 切换 | 独立模式 |

**组合逻辑**：
- 阵营与等级为AND关系
- 地形和收益为OR关系
- 例：蜀国 + 平原+城池 + 粮草 → 显示蜀国中（平原或城池）且产粮草的领土

**过滤视觉效果**：
- **匹配领土**：正常显示 + 古铜金高亮边框1px
- **不匹配领土**：降低亮度至40% + 灰度滤镜 `grayscale(0.6) brightness(0.4)`
- **筛选栏收起时**：所有领土恢复正常

**快捷筛选按钮（筛选栏右侧）**：

| 按钮 | 功能 | 图标 |
|------|------|------|
| 我的领土 | 过滤为己方阵营 | 🏠 |
| 高产领地 | 按总收益排序，高亮前5 | 📊 |
| 可征服 | 显示与己方相邻的非己方领土 | ⚔️ |
| 热力图 | 切换收益热力图模式 | 🗺️ |

## 四、边缘滚动与Tooltip冲突修复（v3.0 完整方案）

### 问题分析

原方案仅"Tooltip显示时暂停滚动"一个规则，存在以下边界问题：
1. 快速划过边缘领土时Tooltip闪烁
2. Tooltip消失后滚动立即恢复导致抖动
3. 多个领土紧密排列时Tooltip与滚动频繁切换
4. 手机端无此问题（触屏无hover），但需定义长按与拖拽的冲突

### 修复方案：状态机模型

定义3种交互状态，严格互斥：

```
┌──────────┐  鼠标进入领土   ┌──────────────┐
│  空闲态   │ ──────────────→ │  Tooltip态   │
│ (可滚动)  │ ←────────────── │  (滚动暂停)  │
└──────────┘  鼠标离开领土   └──────────────┘
     │                              │
     │ 鼠标进入边缘30px             │ 鼠标移出边缘
     ▼                              ▼
┌──────────┐                 ┌──────────────┐
│  滚动态   │                 │  恢复过渡态   │
│ (持续滚动)│                 │  (200ms延迟) │
└──────────┘                 └──────────────┘
```

### 状态转换规则

| 当前状态 | 触发条件 | 目标状态 | 行为 |
|---------|---------|---------|------|
| 空闲态 | 鼠标进入边缘30px | 滚动态 | 开始边缘滚动，显示金色光带 |
| 空闲态 | 鼠标进入领土且停留>200ms | Tooltip态 | 显示Tooltip，暂停滚动 |
| 滚动态 | 鼠标进入领土且停留>200ms | Tooltip态 | 暂停滚动，显示Tooltip |
| Tooltip态 | 鼠标离开领土 | 恢复过渡态 | 隐藏Tooltip，启动200ms计时器 |
| 恢复过渡态 | 200ms计时器到期 | 空闲态/滚动态 | 根据鼠标位置判断是否在边缘 |
| 恢复过渡态 | 200ms内鼠标进入新领土 | Tooltip态 | 取消计时器，显示新Tooltip |
| 任意态 | 鼠标拖拽地图 | 拖拽态 | 暂停滚动+隐藏Tooltip |
| 拖拽态 | 释放鼠标 | 空闲态 | 根据鼠标位置判断后续状态 |

### 关键时间参数

| 参数 | 值 | 说明 |
|------|------|------|
| Tooltip显示延迟 | 200ms | 鼠标进入领土后延迟显示，避免快速划过闪烁 |
| 边缘滚动恢复延迟 | 200ms | Tooltip消失后延迟恢复滚动，避免抖动 |
| 拖拽判定阈值 | 5px | 鼠标移动>5px判定为拖拽而非点击 |
| 边缘滚动触发区 | 30px | 距地图边缘30px范围 |
| 滚动速度上限 | 8px/帧 | 防止滚动过快 |

### 实现伪代码

```javascript
const MapInteraction = {
  state: 'idle', // idle | scrolling | tooltip | recovering | dragging
  tooltipTimer: null,
  recoveryTimer: null,
  
  onMouseMove(x, y) {
    const inEdge = isEdgeZone(x, y, 30);
    const territory = getTerritoryAt(x, y);
    
    switch(this.state) {
      case 'idle':
        if (inEdge) this.transitionTo('scrolling');
        if (territory) this.startTooltipTimer(territory);
        break;
        
      case 'scrolling':
        if (territory) this.startTooltipTimer(territory);
        if (!inEdge && !territory) this.transitionTo('idle');
        break;
        
      case 'tooltip':
        // Tooltip显示中，不响应边缘滚动
        break;
        
      case 'recovering':
        if (territory) {
          clearTimeout(this.recoveryTimer);
          this.showTooltip(territory);
        }
        break;
    }
  },
  
  startTooltipTimer(territory) {
    clearTimeout(this.tooltipTimer);
    this.tooltipTimer = setTimeout(() => {
      this.showTooltip(territory);
    }, 200); // 200ms延迟
  },
  
  showTooltip(territory) {
    this.state = 'tooltip';
    tooltipManager.show(territory);
    edgeScroll.pause(); // 暂停边缘滚动
    hideEdgeGlow();     // 隐藏边缘光带
  },
  
  hideTooltip() {
    tooltipManager.hide();
    this.state = 'recovering';
    this.recoveryTimer = setTimeout(() => {
      this.state = isEdgeZone(mouseX, mouseY, 30) ? 'scrolling' : 'idle';
      if (this.state === 'scrolling') {
        edgeScroll.resume();
        showEdgeGlow();
      }
    }, 200); // 200ms恢复延迟
  }
};
```

### 手机端交互冲突处理

手机端无hover，不存在边缘滚动与Tooltip冲突。但需处理**长按查看信息 vs 拖拽移动地图**的冲突：

| 手势 | 判定条件 | 行为 |
|------|---------|------|
| 单指点击 | 按下→抬起，移动<10px，时长<300ms | 选中领土，弹出Bottom Sheet |
| 单指长按 | 按下不动，移动<10px，时长>500ms | 显示领土产出气泡（全部领土） |
| 单指拖拽 | 按下→移动>10px | 移动地图 |
| 双指捏合 | 双指距离变化 | 缩放地图 |
| 双指点击 | 双指同时按下 | 复位到默认缩放和位置 |

**防冲突规则**：
- 拖拽判定优先于长按（移动>10px立即判定为拖拽，取消长按计时器）
- 长按触发后手指可小范围移动（±10px容差）不取消气泡显示
- 气泡显示期间单指抬起→气泡延迟1秒淡出

## 七、小地图系统（PC端专属）

### 布局规格

```
中央场景区右下角：
+-------------------------------+
|          主地图区域             |
|                     +---------+
|                     | 小地图   |  ← 180×140px
|                     | □ □ □   |    半透明背景
|                     | □ ★ □   |    ★=当前视口
|                     | □ □ □   |
|                     +---------+
+-------------------------------+
```

### 视觉设计

- **尺寸**：180×140px
- **背景**：rgba(13,17,23,0.75) 半透明夜墨色
- **边框**：1px古铜金描边 + 四角青铜纹饰
- **领土缩略**：简化色块（阵营色填充），已占领实色，未占领半透明
- **特殊地标**：小圆点标注（直径3px）
- **视口指示框**：白色半透明矩形（rgba(240,230,211,0.4)）

### 小地图上的产出指示

- 己方领土中心显示**微型产出指示点**（直径4px彩色圆点）
- 颜色对应主要产出：粮草=绿、铜钱=金、兵力=红
- 多产出领土显示双色半圆
- 仅缩放>80%时显示

### 小地图上的离线变化标记

- 新占领：金色小点（3px）脉冲
- 失去：红色小点（3px）脉冲
- 脉冲频率1Hz，持续至主地图查看

### 交互行为

| 操作 | 行为 |
|------|------|
| 点击小地图某位置 | 主地图平滑滚动到该位置（lerp 0.2） |
| 拖拽小地图视口框 | 主地图实时跟随 |
| 鼠标悬停领土 | 显示领土名称Tooltip |
| 主地图滚动 | 小地图视口框同步更新 |

### 显隐控制

- 地图场景时显示，非地图场景淡出隐藏（200ms）
- 右上角收起按钮「×」，收起后变为小图标「🗺️」
- 快捷键 `Tab` 切换显隐
- 战斗中隐藏

### 手机端说明

> **设计决策**：手机端不实现小地图。原因：(1) 屏幕空间有限；(2) 双指缩放+单指拖拽导航效率足够；(3) 默认缩放80%已提供良好全局视野。替代方案：长按空白处→弹出全屏缩略图（未来扩展）。
