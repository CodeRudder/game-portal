## [ANI-1] 动画类型 {#ani-1}

### 四类动画体系

| 类型 | 定义 | 时长范围 | 示例 |
|------|------|:--------:|------|
| **过渡动画** | 场景/面板/页面切换 | 200-800ms | 面板展开、页面切换、水墨晕染转场 |
| **状态动画** | 元素状态变化反馈 | 50-300ms | 按钮按下、开关切换、选中高亮 |
| **反馈动画** | 操作结果可视化 | 200-1500ms | 资源飘字、升级光效、Toast滑入 |
| **装饰动画** | 氛围营造 | 持续循环 | 建筑产出粒子、旗帜飘扬、水墨纹理流动 |

## [ANI-2] 时间规范 {#ani-2}

### 时长标准

| 动画类别 | 时长范围 | 缓动函数 | 说明 |
|---------|:--------:|---------|------|
| 微交互 | 80-150ms | ease-out | 即时反馈，不拖泥带水 |
| 小过渡 | 150-250ms | ease-out | 轻盈感 |
| 中过渡 | 250-400ms | ease-out | 自然展开 |
| 大过渡 | 400-800ms | ease-in-out | 沉浸感 |
| 特殊动画 | 800-2000ms | 自定义 | 戏剧性 |
| 持续动画 | 800-2000ms/周期 | linear/ease-in-out | 循环播放 |

### 缓动函数库

```javascript
const Easing = {
  linear:      t => t,
  easeIn:      t => t * t,
  easeOut:     t => t * (2 - t),
  easeInOut:   t => t < 0.5 ? 2*t*t : -1+(4-2*t)*t,

  // 弹性缓动 — 用于弹窗/卡片弹入
  spring: (stiffness, damping) => { /* 弹性物理模拟 */ },

  // 水墨晕染专用 — 快速展开→缓慢渗透
  waterInk: t => {
    return t < 0.3 ? 2 * t : 0.6 + 0.4 * (t - 0.3) / 0.7;
  }
};
```

### 推荐缓动对照

| 场景 | 推荐缓动 | 原因 |
|------|---------|------|
| 面板展开 | ease-out | 快速展开，结尾减速，自然 |
| 面板关闭 | ease-in | 快速启动，加速离开 |
| 弹窗弹出 | spring(1, 0.8) | 弹性过冲，生动感 |
| 页面过渡 | ease | 通用平滑 |
| 水墨转场 | waterInk | 匹配水墨扩散物理特性 |
| 按钮按下 | ease-in | 即时响应，快速收缩 |
| 拖拽弹回 | spring(0.5, 0.7) | 过冲弹回，物理感 |
| 数字滚动 | ease-out | 快速开始，精确停在目标值 |

### 粒子池管理

| 规则 | 说明 |
|------|------|

## [ANI-4] 禁止项 {#ani-4}

### 严格禁止的视觉效果

| 禁止项 | 原因 | 替代方案 |
|--------|------|---------|
| **Glow/外发光** | 性能消耗大，廉价感 | 古铜金光晕（box-shadow脉动） |
| **Neon/霓虹** | 与水墨风格严重冲突 | 水墨晕染扩散效果 |
| **闪烁/频闪** | 引发光敏性癫痫风险 | 缓慢脉动（周期≥800ms） |
| **彩虹渐变** | 与三国主题不符 | 单色渐变（古铜金/朱砂红） |
| **3D旋转** | 2D游戏风格不一致 | 2D翻转（Y轴180°） |
| **弹跳文字** | 过于卡通化 | 毛笔书法风格数字上飘 |
| **粒子爆炸（大量）** | 性能杀手 | ≤80粒子的定向飞溅 |
| **全屏闪烁** | 癫痫风险+体验差 | 半屏水墨晕染过渡 |
| **抖动/摇晃（持续）** | 引发眩晕 | 单次震动（≤200ms） |
| **自动播放声音** | 惊吓用户 | 用户操作触发音效 |

### 动画降级策略

```
性能检测（每5秒检测帧率）：
  fps ≥ 55  → 全品质动画
  fps 40-54 → Level 1：关闭粒子效果，简化阴影
  fps 30-39 → Level 2：关闭所有特效，仅保留基础过渡
  fps < 30  → Level 3：关闭所有动画，纯静态UI

首次降级时弹出一次性Toast：
  「检测到设备性能不足，已自动降低动画品质。
   可在设置→显示中手动调整。」

恢复：帧率连续10秒≥55fps → 自动恢复上一品质等级
```

### 减少动画模式

| 触发方式 | 设置→显示→减少动画 / 系统偏好 prefers-reduced-motion: reduce |
|---------|------|
| 关闭粒子效果 | ✅ |
| 过渡动画时长 | 缩短为原来的1/3 |
| 关闭脉动/闪烁 | ✅ |
| 保留必要状态变化 | 按钮按下缩放保留，光效去除 |
| 战斗特效 | 简化为静态关键帧 |
| 水墨晕染过渡 | 改为简单淡入淡出 |
