### 建筑外观演进

建筑外观随等级递进变化：

| 等级范围 | 外观风格 |
|---------|---------|
| Lv1~5 | 茅屋/木栅（简朴） |
| Lv6~12 | 瓦房/石墙（规整） |
| Lv13~20 | 楼阁/砖墙（精美） |
| Lv21~30 | 宫殿/铜门（辉煌） |

### 建筑依赖关系

```
                    ┌─────────┐
                    │  主城Lv1 │ ← 初始解锁
                    └────┬────┘
              ┌──────────┼──────────┐
              ▼          ▼          ▼
          农田Lv1    市集Lv2     兵营Lv2   ← 主城Lv2 解锁
                                    │
              ┌──────────┬──────────┤
              ▼          ▼          ▼
          书院Lv3    铁匠Lv3    医馆Lv4   ← 主城Lv3/4 解锁
                    │
              ┌─────┴─────┐
              ▼            ▼
          城墙Lv5     招贤馆Lv5        ← 主城Lv5 解锁
```

**核心规则**：每座建筑等级 ≤ 主城等级（主城 Lv10 时，其他建筑最高 Lv10）。

## [BLD-3] 资源产出 {#bld-3}

### 产出公式

```
建筑实际产出 = 基础产出 × 主城加成 × 科技加成 × 武将加成 × 声望加成
```

| 加成来源 | 说明 |
|---------|------|
| 主城加成 | 主城等级对应的"全资源加成"百分比 |
| 科技加成 | 对应科技路线的产出加成 |

> 关联: [TEC-2 科技研究](../prd/TEC-tech-prd.md#tec-2)

| 武将加成 | 派驻武将的统率/政治属性加成 |
| 声望加成 | 声望等级提供的全局加成 |

### 各建筑产出汇总

| 建筑 | 主要产出 | 产出单位 | Lv1 | Lv10 | Lv20 | Lv25/30 |
|------|---------|:-------:|:---:|:----:|:----:|:-------:|
| 农田 | 粮草 | /秒 | 0.8 | ~5.5 | ~22 | ~45 |
| 市集 | 铜钱 | /秒 | 0.6 | ~4.8 | ~18 | ~35 |
| 兵营 | 兵力 | /秒 | 0.4 | ~3.2 | ~14 | ~28 |
| 书院 | 科技点 | /秒 | 0.2 | ~1.8 | ~6.0 | ~8.0 |
| 铁匠铺 | 材料 | /小时 | 0.5 | ~8 | ~30 | ~40 |
| 医馆 | 恢复速率 | 百分比 | +3% | +20% | +45% | +60% |
| 城墙 | 城防值 | 固定值 | 300 | ~3,000 | ~8,000 | ~15,000 |

### ROI 计算（投资回报）

```
ROI = 产出提升量 / 升级总消耗
```

**优先升级建议**：

| 阶段 | 推荐优先升级 | 原因 |
|------|-------------|------|
| 新手期（1~3 天） | 主城 → 农田 → 市集 | 解锁建筑 + 基础产出 |
| 发展期（3~7 天） | 主城 → 兵营 → 书院 | 解锁军事/科技 |
| 中期（7~14 天） | 农田 → 市集 → 兵营 | 产出最大化 |
| 后期（14 天+） | 主城 → 均衡升级 | 全资源加成 |

## [BLD-4] 建筑联动 {#bld-4}

### 前置关系汇总

| 建筑 | 解锁条件 | 等级上限限制 |
|------|---------|-------------|
| 农田 | 初始 | ≤ 主城等级 |
| 市集 | 主城 Lv2 | ≤ 主城等级 |
| 兵营 | 主城 Lv2 | ≤ 主城等级 |
| 铁匠铺 | 主城 Lv3 | ≤ 主城等级 |
| 书院 | 主城 Lv3 | ≤ 主城等级 |
| 医馆 | 主城 Lv4 | ≤ 主城等级 |
| 城墙 | 主城 Lv5 | ≤ 主城等级 |

### 联动加成

| 来源 | 目标建筑 | 加成效果 |
|------|---------|---------|
| 科技「农耕改良」 | 农田 | 粮草产出 +10% |
| 科技「屯田制」 | 农田 | 离线粮草 +30%，容量 +50% |
| 科技「商贸繁荣」 | 市集 | 铜钱产出 +15% |
| 科技「治国安邦」 | 所有建筑 | 升级费用 -10% |
| 武将（统率型） | 兵营 | 兵力产出 + 统率% |
| 武将（政治型） | 农田/市集 | 产出 + 政治% |
| 主城等级 | 所有建筑 | 全资源加成 +2%/级 |

### 建筑队列管理

| 主城等级 | 可用队列槽位 | 说明 |
|:-------:|:----------:|------|
| Lv1~5 | 1 个 | 同时只能升级 1 座建筑 |
| Lv6~10 | 2 个 | 第 2 槽位解锁 |
| Lv11~20 | 3 个 | 第 3 槽位解锁 |
| Lv21~30 | 4 个 | 第 4 槽位解锁 |

**队列规则**：

| 规则 | 说明 |
|------|------|
| 队列顺序 | 当前升级完成后，队列中下一项自动开始 |
| 取消升级 | 返还 80% 已消耗资源 |
| 自动升级 | 可开启开关，系统自动将可升级建筑加入队列 |
| 加速选项 | 铜钱加速 / 天命加速 / 元宝秒完成 |

### 离线建造进度

```
离线建造规则：
  - 离线期间升级进度正常推进（受离线效率系数影响）
  - 离线效率 = 资源效率 × 1.2（建造额外 20% 效率）
  - 完成的升级正常结算，效果立即生效
  - 队列中等待的升级自动开始
```

## [BLD-5] 升级路线 {#bld-5}

### 推荐升级顺序

#### 新手期（第 1~3 天）

| 步骤 | 操作 | 目的 |
|:----:|------|------|
| 1 | 主城 → Lv2 | 解锁市集、兵营 |
| 2 | 农田 → Lv3 | 基础粮草产出 |
| 3 | 市集 → Lv2 | 基础铜钱产出 |
| 4 | 主城 → Lv3 | 解锁铁匠铺、书院 |
| 5 | 兵营 → Lv3 | 基础兵力产出 |
| 6 | 主城 → Lv4 | 解锁医馆 |

#### 发展期（第 3~7 天）

| 步骤 | 操作 | 目的 |
|:----:|------|------|
| 1 | 主城 → Lv5 | 解锁城墙，全资源 +8% |
| 2 | 农田 → Lv8 | 粮草产出加速 |
| 3 | 市集 → Lv7 | 铜钱产出加速 |
| 4 | 书院 → Lv5 | 科技点产出 + 研究速度 |
| 5 | 兵营 → Lv7 | 兵力跟得上出征需求 |

#### 中后期（第 7 天+）

| 策略 | 说明 |
|------|------|
| 主城优先 | 主城每级 +2% 全资源加成，ROI 最高 |
| 产出均衡 | 农田/市集/兵营保持同步等级 |
| 科技联动 | 书院等级影响科技研究速度，尽早升高 |
| 防御不急 | 城墙/医馆在 PvP 内容开放前可延后 |

### 资源死锁避免

**常见死锁场景**：

```
死锁：主城升级需要粮草 → 粮草不够需要升级农田 → 农田升级需要主城等级更高
```

**避免策略**：

| 策略 | 说明 |
|------|------|
| 主城等级预判 | 升主城前确保农田/市集等级已跟上 |
| 资源储备 | 升级前储备至少 1.5 倍所需资源 |
| 队列规划 | 利用队列系统提前安排升级顺序 |
| 离线收益 | 离线期间资源持续产出，可缓解短期缺口 |
| 科技辅助 | 点出「治国安邦」降低 10% 升级费用 |

### 一键收取

- 右下角悬浮按钮，一键收取所有建筑累积产出
- 收取动画：资源数字从各建筑飞向顶部资源栏
- 自动收取：VIP3+ 解锁离线自动收取（无需手动点击）
