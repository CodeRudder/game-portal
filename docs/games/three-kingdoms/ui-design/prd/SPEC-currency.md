```

            ┌───────────┼───────────┐
            ↓           ↓           ↓
        🏷️ 求贤令    加速道具   资源礼包
            │
            ↓
        👑 天命 ←── 转生奖励/成就/活跃度
```

## [CUR-4] 汇率规则 {#cur-4}

### 官方汇率表

| 兑换方向 | 汇率 | 方向 | 说明 |
|---------|------|------|------|
| 💎 → 🏷️ | 100💎 = 100求贤令 | 单向 | 元宝购买求贤令 |
| 👑 → 🏷️ | 100天命 = 100求贤令 | 单向 | 天命兑换求贤令 |
| 💰 → 📜 | 500💰 = 1招贤榜 | 单向 | 铜钱购买招贤榜 |
| 🏷️ → 💎 | **不可逆** | — | 求贤令不可换回元宝 |
| 📜 → 💰 | **不可逆** | — | 招贤榜不可换回铜钱 |
| 💎 → 💰 | 按礼包定价 | 单向 | 仅通过购买资源礼包间接兑换 |

### 等价换算参考

```
以"高级招贤1次"（= 100求贤令）为基准：

100求贤令 = 100💎 ≈ 10元人民币
100求贤令 = 100天命
100求贤令 ≈ 50,000铜钱（间接，通过购买招贤榜×100）

即：1天命 ≈ 500铜钱（间接等价）
    1💎 ≈ 500铜钱（间接等价）
```

### 防通胀体系

| 层级 | 机制 | 适用货币 |
|------|------|---------|
| **L1 消耗膨胀** | 升级/强化费用指数增长（×1.8/级），自动消耗过剩货币 | 铜钱、天命 |
| **L2 获取限制** | 每日获取量有软上限，堆叠有硬上限 | 招贤榜(99)、远征币、公会币、活动代币 |
| **L3 重置机制** | 转生时部分货币按比例重置 | 天命（保留30%）、声望值（归零） |
