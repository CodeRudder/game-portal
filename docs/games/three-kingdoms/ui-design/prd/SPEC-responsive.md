## [RSP-1] 断点定义 {#rsp-1}

### 7级断点体系

| 断点名称 | 宽度范围 | 典型分辨率 | 目标设备 | 布局策略 |
|---------|---------|-----------|---------|---------|
| Desktop-L | ≥1920px | 2560×1440 / 3840×2160 | 大屏/4K | 画布放大+三栏布局 |
| Desktop | 1280–1919px | 1366×768 / 1920×1080 | 标准PC | 1:1像素映射（设计基准） |
| Tablet-L | 1024–1279px | 1024×768 | 横屏平板 | 等比缩小 |
| Tablet | 768–1023px | 768×1024 | 竖屏平板 | 等比缩小+底部Tab |
| Mobile-L | 480–767px | 540×960 | 大屏手机 | 流式重排+Bottom Sheet |
| Mobile | 375–479px | 375×667 / 390×844 | 标准手机 | 流式重排+全屏面板 |
| Mobile-S | <375px | 320×568 | 小屏/旧手机 | 极简模式 |

### 画布缩放算法

```
核心公式（PC/平板端）：
  scale = min(viewW / 1280, viewH / 800)
  scale = min(scale, 2.0)  // 4K上限
  offsetX = (viewW - 1280 × scale) / 2
  offsetY = (viewH - 800 × scale) / 2

移动端（<768px）：
  scale = 1  // 不使用画布缩放，采用流式布局
  canvasW = viewW
  canvasH = viewH
```

### 留白区域处理

| 缩放情况 | 留白位置 | 处理策略 |
|---------|---------|---------|
| 宽屏（16:9） | 上下留白 | 深色水墨纹理背景 `#0a0a14`，远山剪影SVG |
| 窄屏（4:3） | 两侧留白 | 同色水墨纹理，竹叶/云纹装饰 |
| 超宽屏（21:9） | 两侧大面积留白 | 展开辅助面板（战报/武将信息） |
| 4K（≥3840） | 整体放大 | scale限幅2.0，剩余空间装饰背景 |

### 断点检测实现

```javascript
function detectBreakpoint(viewW) {
  if (viewW >= 1920) return 'desktop-l';
  if (viewW >= 1280) return 'desktop';
  if (viewW >= 1024) return 'tablet-l';
  if (viewW >= 768)  return 'tablet';
  if (viewW >= 480)  return 'mobile-l';
  if (viewW >= 375)  return 'mobile';
  return 'mobile-s';
}
```

## [RSP-3] 交互适配 {#rsp-3}

### PC鼠标 vs 手机触摸差异处理

| 交互场景 | PC端（鼠标） | 手机端（触摸） |
|---------|-------------|---------------|
| 悬停预览 | hover显示Tooltip（>300ms） | 长按显示Tooltip（>500ms） |
| 右键菜单 | 右键弹出快捷菜单 | 长按弹出快捷菜单 |
| 滚动列表 | 鼠标滚轮 | 单指上下滑动 |
| 地图缩放 | 鼠标滚轮 | 双指张合 |
| 拖拽武将 | 按住左键拖拽 | 按住滑动（手指上方30px显示） |
| 批量选择 | Shift+点击 | 长按进入多选模式 |
| 关闭面板 | ESC / 点击× | 下拉手势 / 点击× / 系统返回键 |
| 确认操作 | Enter | 无快捷键，必须点击按钮 |

### 切换过渡动画

```
竖屏 → 横屏（400ms）：
  0ms    现有界面淡出（opacity 1→0，200ms ease-out）
  200ms  应用新布局（瞬间切换，无动画）
  220ms  新界面淡入（opacity 0→1，200ms ease-in）

横屏 → 竖屏（450ms）：
  0ms    侧边面板先滑出（150ms ease-in）
  150ms  界面淡出（150ms ease-out）
  300ms  应用新布局
  320ms  新界面淡入（150ms ease-in）
```

### 切换时状态保持

| 状态类型 | 处理方式 |
|---------|---------|
| 当前Tab | 保持不变 |
| 场景内容 | 保持地图位置/战斗进度 |
| 弹出面板 | 全部关闭（避免布局错乱） |
| 战斗中 | **禁止切换**，锁定当前方向 |

### 性能降级策略

| 设备等级 | 判定条件 | 粒子上限 | 动画帧率 | 资源品质 |
|---------|---------|:--------:|:--------:|---------|
| 高性能 | CPU≥8核 且 内存≥8GB | 200 | 60fps | @3x WebP |
| 中性能 | CPU≥4核 且 内存≥4GB | 80 | 30fps | @2x WebP |
| 低性能 | 其他 | 30 | 24fps | @1x WebP |

### 自动降级检测（每5秒）

```
fps ≥ 55  → 全品质动画
fps 40-54 → Level 1：关闭粒子效果，简化阴影
fps 30-39 → Level 2：关闭所有特效，仅保留基础过渡
fps < 30  → Level 3：关闭所有动画，纯静态UI

恢复：帧率连续10秒≥55fps → 自动恢复上一品质等级
```
