## [SHP-1] 商品分类 {#shp-1}

### 功能描述

商店系统是放置循环中资源消费的核心出口，玩家通过消耗货币购买资源、道具、装备和限定商品，形成「挂机产出 → 资源积累 → 商店消费 → 战力提升 → 解锁新内容」的核心闭环。

> 🎨 → [UI: 商店主界面](../ui-layout/SHP-shop.md#shp-1)

### 商店类型矩阵

| 商店类型 | 入口 | 主货币 | 刷新规则 | 离线补货 | 核心商品 |
|---------|------|--------|---------|:-------:|---------|
| 集市（常驻） | 城池「集市」建筑 / 右侧快捷图标 | 铜钱·粮草·兵力 | 每 8h 部分刷新 | ✅ | 基础资源·材料·加速道具 |
| 黑市（限时） | NPC 随机触发 | 铜钱 | 随机出现，持续 2h | ❌ 错过消失 | 稀有道具·折扣品 |
| 军需处 | 城池「兵营」建筑 | 兵力 + 铜钱 | 每日 0 点刷新 | ✅ | 军事道具·装备·兵书 |
| 限时特惠 | 主界面弹窗 / 卷轴卡片 | 元宝 | 限时 24h | ❌ | 付费礼包 |
| 活动商店 | 活动面板入口 | 活动代币 | 跟随活动周期 | ❌ | 活动限定道具 |
| NPC 交易 | NPC 对话「交易」选项 | 以物易物 + 铜钱补差 | 好感度驱动 | ⚠️ 部分 | 特色道具·好感度限定 |

### 集市商品分类

| Tab 分类 | 商品类型 | 说明 |
|---------|---------|------|
| 资源兑换 | 粮草箱、铜钱袋、兵力令 | 基础资源兑换，常驻供应 |
| 道具购买 | 加速道具、强化石、医疗包、精铁锭、战马 | 消耗品与材料，部分随机刷新 |
| 限时特惠 | 礼包组合、折扣商品 | 限时限量，高性价比 |
| 我的收藏 | 玩家收藏的商品 | 按状态分组：降价中 / 可购买 / 即将刷新 / 已下架 |

### 商品卡片信息

| 信息项 | 说明 |
|--------|------|

## [SHP-2] 定价规则 {#shp-2}

### 价格体系

| 商品类别 | 价格范围（铜钱） | 说明 |
|---------|-----------------|------|
| 基础资源 | 100 ~ 800 | 粮草、铜钱袋、兵力令等 |
| 消耗道具 | 500 ~ 2,000 | 加速道具、医疗包、强化石 |
| 稀有材料 | 2,000 ~ 5,000 | 精铁锭、战马、高级材料 |
| 军事道具 | 300 ~ 1,000（兵力） | 兵书、兵符、军备 |
| 限时礼包 | 100 ~ 500（元宝） | 组合礼包，原价打折 |
| NPC 特产 | 好感度折扣价 | 受 NPC 好感度影响，-5% ~ -20% |

### 折扣机制

| 折扣类型 | 折扣幅度 | 出现条件 | 持续时间 |
|---------|---------|---------|---------|
| 常规折扣 | -10% ~ -20% | 商店刷新时 20% 概率出现 | 至下次刷新（8h） |
| 限时特惠折扣 | -30% ~ -60% | 每日特惠活动 | 24h |
| 黑市折扣 | -40% ~ -70% | NPC 随机触发黑市 | 2h |
| NPC 好感度折扣 | -5% ~ -20% | 好感度等级提升 | 永久（好感度不降） |
| 活动折扣 | -20% ~ -50% | 节日/运营活动 | 跟随活动周期 |

### NPC 好感度价格修正

| 好感度等级 | 价格折扣 | 商品解锁 | 特殊效果 |
|:---------:|:-------:|---------|---------|
| ♡（陌生） | 无折扣 | 基础商品 | — |
| ❤️♡（熟悉） | -5% | +1 稀有商品 | 偶尔赠送小礼物 |
| ❤️❤️♡（信赖） | -10% | +2 稀有商品 | 优先通知黑市出现 |
| ❤️❤️❤️♡（亲密） | -15% | +3 稀有商品 | 每日免费赠送 1 件基础商品 |
| ❤️❤️❤️❤️❤️（挚友） | -20% | 全部解锁 | 专属限定商品 |

## [SHP-3] 购买逻辑 {#shp-3}

### 库存规则

| 商品类型 | 库存机制 | 刷新规则 |
|---------|---------|---------|
| 常驻基础商品 | 无限库存（显示 ×99） | 不刷新 |
| 随机商品 | 有限库存（5 ~ 30 件） | 每 8h 刷新，补充至满库存 |
| 折扣商品 | 极少库存（1 ~ 5 件） | 20% 概率在刷新时出现 |
| 限时特惠 | 限量（1 ~ 3 件/人） | 24h 更换 |
| 黑市商品 | 限量（3 ~ 8 件） | 随机出现，错过消失 |

### 限购规则

| 限购类型 | 重置规则 | 显示方式 |
|---------|---------|---------|
| 每日限购 | 每日 0 点重置购买次数 | 「已购 2/3」 |
| 终身限购 | 活动/账号生命周期内限量 | 「限量 1 件」 |
| 好感度限购 | NPC 好感度达到后解锁 | 未解锁时显示🔒 |

### 购买确认策略（五级确认）

| 级别 | 触发条件 | 确认方式 | 可否跳过 |
|:----:|---------|---------|:-------:|
| L0 免确认 | 铜钱 ≤ 200 | 直接购买，Toast 提示 | — |
| L1 轻确认 | 铜钱 201 ~ 2,000 | 内联确认条（卡片下方展开） | ✅ 勾选「不再提示」 |
| L2 标准确认 | 铜钱 > 2,000 或兵力/粮草消耗 | 标准确认弹窗 | ❌ |
| L3 严格确认 | 元宝消耗（付费货币） | 弹窗 + 3 秒倒计时 | ❌ |
| L4 高危确认 | 元宝 ≥ 500 或限定商品 | 弹窗 + 滑动确认 | ❌ |

### 误操作防护

| # | 防护场景 | 规则 |
|:-:|---------|------|
| 1 | 快速连续点击 | 同一商品 500ms 内仅响应首次点击 |
| 2 | 购买中切换 Tab | 弹出「购买进行中，确认离开？」 |
| 3 | 货币不足 | 价格数字抖动变红 + Toast「铜钱不足」 |
| 4 | 商品已售罄 | 购买按钮灰化 + Toast「已售罄」 |
| 5 | 网络延迟 | 按钮 loading 旋转 + 「处理中…」禁止重复点击 |

### 网络异常处理

| 场景 | 处理方式 |
|------|---------|
| 购买请求超时 | 本地缓存请求，重连后自动提交 |
| 购买失败（服务端拒绝） | 回滚本地状态 + Toast 显示失败原因 |
| 刷新失败 | 使用上次缓存数据 + 黄条提示 |
| 重复提交 | 服务端幂等处理 + 本地 500ms 节流 |

### 自动补货引擎

| 触发条件 | 补货范围 |
|---------|---------|
| 定时补货（每 8h） | 常驻商品补满库存 + 随机生成 1~3 件新商品 + 20% 概率出现折扣 |
| 离线补货（每 8h 离线） | 最多累积 2 次（16h 上限）+ 10% 概率出现离线限定稀有商品 |
| 手动刷新 | 消耗 💰500 立即刷新随机商品 |

### 收藏系统

| 功能 | 说明 |
|------|------|
| 收藏上限 | 30 件 |
| 降价提醒 | 收藏商品出现折扣时触发多渠道提醒（红点脉动 + Toast + 降价标签） |
| 上架提醒 | 收藏商品被刷新下架后重新上架时通知 |
| 分组展示 | 按状态分组：降价中 / 可购买 / 即将刷新 / 已下架 |

## [SHP-4] 货币体系 {#shp-4}

### 功能描述

商店系统支持多种货币，不同商店类型使用不同货币，形成分层消费体系。

> 🎨 → [UI: 货币显示](../ui-layout/SHP-shop.md#shp-4)

### 货币定义

| 货币 | 图标 | 获取途径 | 主要用途 | 稀有度 |
|------|------|---------|---------|:------:|
| 铜钱 | 💰 | 挂机产出、任务奖励、出售道具 | 集市基础消费、强化消耗 | ★☆☆☆☆ |
| 粮草 | 🌾 | 挂机产出、农场建筑 | 资源兑换、士兵消耗 | ★☆☆☆☆ |
| 兵力 | ⚔️ | 兵营产出、征兵 | 军需处消费、战斗消耗 | ★★☆☆☆ |
| 元宝 | 💎 | 充值、成就奖励、活动奖励 | 限时特惠、高级道具 | ★★★★☆ |
| 活动代币 | 🎫 | 活动任务、活动关卡 | 活动商店限定商品 | ★★★☆☆ |

### 货币消耗优先级

| 场景 | 消耗规则 |
|------|---------|
| 集市购买 | 优先消耗铜钱，不足时提示 |
| 军需处购买 | 兵力 + 铜钱组合消耗，任一不足不可购买 |
| 限时特惠 | 仅消耗元宝 |
| NPC 交易 | 以物易物 + 铜钱补差价 |
| 活动商店 | 仅消耗对应活动代币 |

### 货币不足提示

| 状态 | 反馈 |
|------|------|
| 铜钱不足 | 价格数字变红 + 抖动 + 鼓声音效 + Toast「铜钱不足」 |
| 元宝不足 | 价格数字变红 + 弹窗引导获取途径 |
| 兵力不足 | 灰色兵力图标 + Toast「兵力不足，请先征兵」 |
| 活动代币不足 | 显示当前代币数 / 所需代币数 + 引导完成任务 |
