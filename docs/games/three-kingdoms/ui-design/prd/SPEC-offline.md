## [OFR-1] 离线计算 {#ofr-1}

### 核心公式 {#ofr-1-1}

```
离线收益 = 基础产出速率 × 离线秒数 × 离线效率系数

其中：
  离线效率系数 = 基础衰减系数 × (1 + Σ加成系数)
  最终效率系数上限 = 100%（即使加成后不超过在线效率）
```

### 基础衰减系数表 {#ofr-1-2}

| 离线时长 | 效率系数 | 覆盖场景 | 设计意图 |
|:--------:|:--------:|---------|---------|
| 0 ~ 2小时 | 100% | 午休、短暂离开 | 短期离线无惩罚 |
| 2 ~ 8小时 | 80% | 夜间睡眠 | 睡眠期间仍获较高收益 |
| 8 ~ 24小时 | 60% | 工作日白天 | 鼓励午休或下班后上线 |
| 24 ~ 48小时 | 40% | 周末/出差 | 仍有可观收益 |
| 48 ~ 72小时 | 25% | 短期旅行 | 收益大幅衰减，温和提醒回归 |
| > 72小时 | 15%（封底） | 长期未登录 | 保底收益，欢迎回来 |

### 加成系数 {#ofr-1-3}

| 加成来源 | 加成值 | 累计上限 |
|---------|:------:|:-------:|
| 科技「离线收益I/II/III」 | +10%/+15%/+20% | +30% |
| VIP等级 1/3/5 | +5%/+10%/+20% | +20% |
| 声望等级 1/3/5/10 | +3%/+5%/+10%/+25% | +25% |
| **总计上限** | — | **+100%** |

### 加成计算示例

```
玩家A：科技II(+15%) + VIP3(+10%) + 声望Lv5(+10%)
加成系数 = 15% + 10% + 10% = 35%

离线8小时：效率 = 60% × (1 + 0.35) = 81%
离线24小时：效率 = 40% × (1 + 0.35) = 54%
```

### 各系统离线效率修正

| 系统 | 效率系数修正 | 说明 |
|------|:-----------:|------|
| 资源产出 | 全局系数 × 1.0 | 基准效率 |
| 建筑升级 | 全局系数 × 1.2 | 建筑升级有额外加成 |
| 科技研究 | 全局系数 × 1.0 | 无额外加成 |
| 远征战斗 | 全局系数 × 0.85 | 战斗惩罚系数 |
| 自动推图 | 全局系数 × 0.80 | 战斗效率略低于在线 |

### 快照机制

| 属性 | 规则 |
|------|------|
| 快照时机 | 玩家下线时立即记录 |
| 快照内容 | 各系统当前状态：资源净产出速率、建筑队列、科技队列、远征队列 |
| 快照存储 | 本地存储 + 云端同步 |
| 快照有效期 | 72小时（超过封顶时长后不再使用） |

## [OFR-2] 收益上限 {#ofr-2}

### 封顶时长

| 属性 | 规则 |
|------|------|
| 全局封顶 | **72小时** |
| 超过72小时 | 不再产出任何收益 |
| 封顶提示 | 离线>72小时回归时，弹窗顶部显示警告条：「离线超过72小时，超出部分无收益」 |

### 各系统离线行为

| 系统 | 封顶时长 | 离线行为 | 溢出处理 |
|------|:-------:|---------|---------|
| 资源产出 | 72h | 按效率系数自动产出 | 有上限资源溢出截断，无上限资源全额发放 |
| 建筑升级 | 72h | 自动推进，完成后自动接续队列 | — |
| 科技研究 | 72h | 自动推进，完成后正常结算 | — |
| 远征战斗 | 72h | 自动推进，胜率=在线×0.85 | 失败获30%保底奖励 |
| 自动推图 | 72h | 自动推进已三星通关关卡 | 仅一星评价，奖励≈在线60% |
| 事件系统 | 72h | 不自动处理，最多堆积5个 | 超时以保守方案处理 |
| NPC系统 | — | NPC正常刷新，好感度不衰减 | 限时NPC正常计时 |

### 资源溢出规则

| 资源类型 | 上限 | 溢出处理 |
|---------|------|---------|
| 铜钱 | 无上限 | 全额发放 |
| 粮草 | 粮仓容量 | 溢出部分截断，不丢失（提示升级粮仓） |
| 兵力 | 兵营容量 | 溢出部分截断（提示升级兵营） |
| 天命 | 无上限 | 全额发放（需天命阁建筑） |

### 离线收益预估面板

| 属性 | 规则 |
|------|------|
| 入口 | 资源总览面板内 |
| 时长选项 | 2h / 8h / 24h / 48h / 72h（滑块选择） |
| 显示内容 | 各资源预估收益 + 效率系数 + 是否溢出 |
| 实时计算 | 基于当前产出速率和加成系数 |

## [OFR-3] 翻倍机制 {#ofr-3}

### 广告翻倍

| 属性 | 规则 |
|------|------|
| 触发时机 | 离线收益弹窗领取时 |
| 翻倍方式 | 观看激励视频广告（30秒） |
| 翻倍效果 | 离线收益×2 |
| 每日次数 | 3次/天 |
| 广告失败 | 广告加载失败时自动降级为1.5倍（补偿机制） |
| 离线时长要求 | 离线≥30分钟才可使用广告翻倍 |

### 元宝翻倍

| 属性 | 规则 |
|------|------|
| 触发时机 | 离线收益弹窗领取时 |
| 翻倍方式 | 消耗元宝 |
| 消耗公式 | 元宝消耗 = ceil(离线收益总价值 / 500) × 10 |
| 最低消耗 | 10元宝 |
| 最高消耗 | 200元宝 |
| 翻倍效果 | 离线收益×2 |
| 每日次数 | 无限制（只要元宝足够） |
| 叠加规则 | 广告翻倍和元宝翻倍不可叠加，只能二选一 |

### 翻倍计算示例

```
场景1：离线8小时，获得铜钱×8,640 + 粮草×2,160
  - 广告翻倍：铜钱×17,280 + 粮草×4,320（免费）
  - 元宝翻倍：消耗30元宝，获得翻倍收益

场景2：离线48小时，获得铜钱×25,000 + 粮草×6,000
  - 广告翻倍：铜钱×50,000 + 粮草×12,000（免费）
  - 元宝翻倍：消耗100元宝，获得翻倍收益
```

### 回归欢迎流程

```
[玩家上线]
     │
     ├── 离线 ≤ 5分钟 → [静默添加收益] → [正常游戏]
     │
     └── 离线 > 5分钟 → [Step 1: 离线收益弹窗]
                              │
                              │ （展示收益 + 翻倍选项）
                              │ （必须领取后关闭）
                              ↓
                        [Step 2: 回归综合面板]
                              │
                              │ （建筑/科技/远征/事件摘要）
                              │ （可跳过）
                              ↓
                        [正常游戏]
```

### Step 1 离线收益弹窗内容

| 展示项 | 说明 |
|--------|------|
| 离线时长 | 「离线8时32分」 |
| 效率系数 | 「效率系数 60%」 |
| 资源收益 | 各资源 +数值（数字滚动动画800ms） |
| 封顶提示 | >72h时显示警告条 |
| 溢出警告 | 资源将溢出时标注⚠️ |
| 翻倍选项 | [观看广告×2] / [消耗XX元宝×2] |
| 领取按钮 | 「一键领取」 |

### Step 2 回归综合面板内容

| 信息 | 展示条件 |
|------|---------|
| 建筑升级完成 | 有建筑在离线期间完成 |
| 科技研究完成 | 有科技在离线期间完成 |
| 远征队归来 | 有远征队在离线期间返回 |
| 自动推图进度 | 开启了自动推图 |
| 待处理事件 | 有堆积事件 |
| 限时NPC | 有即将消失的NPC |
| 版本更新 | 离线期间有更新 |

### 回归流程时间预算

| 阶段 | 时长 |
|------|:----:|
| Step 1 弹窗弹入 + 数字滚动 | 1.1s |
| Step 1 玩家阅读 + 操作 | ~2s |
| Step 1 领取动画 + 关闭 | 0.9s |
| Step 2 面板淡入 + 玩家操作 | ~1.2s |
| **总计** | **~5.2s** |
