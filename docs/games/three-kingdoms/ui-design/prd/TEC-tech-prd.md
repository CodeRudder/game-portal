## [TEC-1] 科技系统概述 {#tec-1}
> 🎨 → [UI: 科技树场景](../ui-layout/TEC-tech.md#tec-1)

### 玩法规则

科技系统通过研究科技节点为全系统提供永久加成。玩家选择三条路线之一或混合发展，形成个性化的战略方向。

#### 1.1 三条科技路线

| 路线 | 图标 | 核心定位 | 节点数 |
|------|:----:|---------|:------:|
| 🔴 军事路线 | 刀剑 | 攻击、暴击、攻城、防御 | 10（含互斥分支） |
| 🟡 经济路线 | 铜钱 | 粮草产出、铜钱产出、离线收益 | 10（含互斥分支） |
| 🟣 文化路线 | 书卷 | 民心、武将忠诚、属性加成 | 10（含互斥分支） |

#### 1.2 科技树结构

- 每条路线包含 **7 个层级节点**，其中第 5~7 层为互斥分支（A/B 二选一）
- 三条路线之间有 **4 个融合科技**（跨路线强力科技）
- 总计：21 个基础节点 + 4 个融合科技 = **25 个科技节点**
- 画布支持缩放（50%~150%）和拖拽平移

#### 1.3 互斥分支机制

每条路线在第 4 节点后出现互斥分支，形成**六选三**的战略抉择：

| 路线 | 分支 A | 核心定位 | 分支 B | 核心定位 |
|:----:|:------:|:-------:|:------:|:-------:|
| 🔴 军事 | 进攻路线 | 攻击+暴击+攻城 | 防御路线 | 防御+减伤+生命 |
| 🟡 经济 | 重农路线 | 粮草产出+容量+离线 | 重商路线 | 铜钱产出+交易+折扣 |
| 🟣 文化 | 仁政路线 | 民心+忠诚+属性 | 霸道路线 | 士气+征服+掠夺 |

### 数值设计

- 单条路线前 4 节点累计研究时间：3 小时 5 分钟
- 单条路线全 7 节点累计研究时间：38 小时 5 分钟
- 全部融合科技累计研究时间：80 小时

### 解锁条件

- 科技系统在主城 Lv.3 且书院建成后解锁
- 各科技节点的前置条件见 [TEC-2]

### 与其他系统的关联

| 关联系统 | 关系 |
|---------|------|
| 建筑系统 | 书院产出科技点；科技提升建筑产出效率和降低升级费用 |
| 武将系统 | 科技提升武将经验、忠诚、属性；内政型武将加速科技研究 |
| 声望系统 | 声望等级提供研究速度加成；转生时科技保留 50% |
| 资源系统 | 经济路线科技直接提升粮草/铜钱产出和离线收益 |

## [TEC-2] 科技研究 {#tec-2}
> 🎨 → [UI: 科技详情面板](../ui-layout/TEC-tech.md#tec-2)

### 玩法规则

#### 2.1 研究流程

1. 玩家选择可研究的科技节点
2. 消耗铜钱（高级科技额外消耗天命）启动研究
3. 等待研究时间完成（离线也计入，受离线效率系数影响）
4. 研究完成后效果立即生效

#### 2.2 研究启动条件

| 条件 | 说明 |
|------|------|
| 科技点足够 | 研究消耗科技点 = 基础费用 × (1 - 书院研究速度加成) |
| 前置科技已研究 | 当前研究节点的前置节点必须已完成 |
| 无其他正在研究的科技 | 默认 1 个研究槽位，VIP3 解锁第 2 槽位 |

#### 2.3 研究队列规则

| 参数 | 规则 |
|------|------|
| 默认队列 | 1 个研究槽位 |
| 扩展槽位 | VIP3 解锁第 2 研究槽位 |
| 队列规则 | 当前研究完成后，队列中下一项自动开始 |
| 取消研究 | 返还 80% 已消耗资源，研究进度清零 |
| 切换研究 | 消耗 100 铜钱可暂停当前研究，切换到另一项（进度保留） |

#### 2.4 离线研究规则

| 属性 | 规则 |
|------|------|
| 计算公式 | 研究剩余时间 - (离线秒数 × 离线效率系数) |
| 封顶时长 | 72 小时（全局统一） |
| 完成处理 | 离线期间完成的科技研究正常结算，效果立即生效 |
| 科技点累积 | 离线期间书院科技点持续产出（受离线效率系数影响） |
| 离线效率 | 科技离线效率 = 资源效率 × 1.0（与建筑升级 ×1.2 区分） |

### 数值设计

#### 研究时间总览

| 节点等级 | 基础研究时间 | 对应游戏阶段 | 累计时间（单路线前 4 节点） |
|:-------:|:----------:|:----------:|:------------------------:|
| Lv1（节点 1） | 5 分钟 | 新手期（第 1 天） | 5 分钟 |
| Lv2（节点 2） | 15 分钟 | 新手期（第 1 天） | 20 分钟 |
| Lv3（节点 3） | 45 分钟 | 发展期（第 2~3 天） | 1 小时 5 分钟 |
| Lv4（节点 4） | 2 小时 | 发展期（第 3~5 天） | 3 小时 5 分钟 |
| Lv5（节点 5A/B） | 5 小时 | 中期（第 5~7 天） | 8 小时 5 分钟 |
| Lv6（节点 6A/B） | 10 小时 | 中后期（第 7~14 天） | 18 小时 5 分钟 |
| Lv7（节点 7A/B） | 20 小时 | 后期（第 14~21 天） | 38 小时 5 分钟 |
| 融合科技 | 15~30 小时 | 终局（第 21~28 天） | — |

#### 科技点系统

| 参数 | 规则 |
|------|------|
| 科技点产出 | 书院等级决定（Lv.1: 0.3/s → Lv.20: 8.0/s） |
| 研究消耗 | 基础费用 × (1 - 书院研究速度加成) |
| 科技点存储 | 无上限，持续累积（离线也累积） |

#### 加速机制

| 加速方式 | 效果 | 消耗 | 限制 |
|---------|------|------|------|
| 铜钱加速 | 立即完成剩余时间的 50% | 铜钱 × 剩余时间(秒) × 0.5 | 无限制 |
| 天命加速 | 立即完成剩余时间的 80% | 天命 × ⌈剩余时间/3600⌉ | 每次研究最多 1 次 |
| 元宝秒完成 | 立即完成全部 | 元宝 × ⌈剩余时间/1800⌉ | 每日限 3 次 |
| 书院等级加速 | 研究速度 +2%~+38%（被动） | 无 | 永久 |
| 声望等级加速 | 研究速度 × (1 + 声望等级×0.02) | 无 | 永久 |
| 联盟加速 | 研究速度 +10% | 联盟成员协助 | 需联盟功能 |

#### 加速叠加公式

```
实际研究时间 = 基础研究时间 / (1 + 书院加成 + 声望加成 + 联盟加成)
加速道具效果 = 实际研究时间 × (1 - 道具减免比例)
```

#### 加速示例

```
玩家：书院 Lv.10（+18%）+ 声望 Lv.5（+10%）
研究「铁骑冲锋」：基础 5h
实际时间 = 5h / (1 + 0.18 + 0.10) = 5 / 1.28 ≈ 3h 54min
使用天命加速（-80%）：3h 54min × 20% ≈ 47min
```

### 解锁条件

- 科技系统解锁：主城 Lv.3 + 书院建成
- 各节点前置条件见 [TEC-3] 各科技数据表

### 与其他系统的关联

- 书院建筑等级 → 研究速度和科技点产出
- 声望等级 → 研究速度永久加成
- 内政型武将 → 派遣至书院增加研究速度 +10%~30%

## [TEC-4] 融合科技 {#tec-4}
> 🎨 → [UI: 科技树场景](../ui-layout/TEC-tech.md#tec-1)

### 玩法规则

融合科技是跨路线的强力科技，需要同时完成两条或三条路线的特定节点后才能解锁。

#### 4.1 融合科技一览

| 融合科技 | 图标 | 效果 | 费用（铜钱） | 研究时间 | 解锁条件 |
|---------|:----:|------|:----------:|:-------:|---------|
| **军经合一** | ⚔️💰 | 出征消耗粮草 -20%，战斗收益 +15% | 20,000 | 15 小时 | 军事 Lv.3 + 经济 Lv.3 |
| **军文并举** | ⚔️📜 | 武将大招伤害 +20%，冷却 -1 回合 | 20,000 | 15 小时 | 军事 Lv.3 + 文化 Lv.3 |
| **经世济民** | 💰📜 | 建筑产出 +25%，民心 +20 | 20,000 | 15 小时 | 经济 Lv.3 + 文化 Lv.3 |
| **霸王之道** | 👑 | 全属性 +8%，全产出 +8%（三项全能） | 50,000 | 30 小时 | 军事 Lv.5 + 经济 Lv.5 + 文化 Lv.5 |

#### 4.2 融合科技解锁规则

- 解锁条件中的"Lv.N"指对应路线的第 N 个节点已完成（含分支）
- 融合科技不占用基础路线的研究槽位
- 融合科技可以与基础路线科技同时研究（需第 2 研究槽位）
- 转生时融合科技保留 50%（与基础科技一致）

#### 4.3 转生重置规则

| 项目 | 转生后状态 | 说明 |
|------|:---------:|------|
| 已完成科技 | 保留 50% | 已完成的科技保留一半（随机保留或按路线优先级） |
| 未完成科技 | 清零 | 研究进度清零 |
| 路线选择记录 | 保留 100% | 记住上次选择的互斥分支，可重新选择 |
| 融合科技 | 保留 50% | 与基础科技一致 |

### 数值设计

- 融合科技费用为基础科技的 2~4 倍
- 霸王之道为终极科技，费用 50,000 铜钱 + 30 小时研究时间
- 融合科技效果为全局加成，与基础科技乘法叠加

### 解锁条件

见 4.1 融合科技一览表中的"解锁条件"列。

### 与其他系统的关联

| 融合科技 | 关联系统 | 效果 |
|---------|---------|------|
| 军经合一 | 战斗+资源 | 降低出征粮草消耗，提升战斗资源收益 |
| 军文并举 | 武将+战斗 | 提升武将技能伤害和冷却效率 |
| 经世济民 | 建筑+民心 | 提升建筑产出和民心值 |
| 霸王之道 | 全系统 | 全属性和全产出加成（终极目标） |
