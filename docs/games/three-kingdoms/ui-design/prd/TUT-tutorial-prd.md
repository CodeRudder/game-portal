## [TUT-1] 引导流程 {#tut-1}

### 功能描述

引导系统通过6步核心引导 + 6步扩展引导 + 8段剧情事件，帮助新玩家从零掌握游戏核心玩法。核心引导不可跳过但可加速，扩展引导在对应功能解锁时自动触发。

> 🎨 → [UI: 引导遮罩与气泡](../ui-layout/TUT-tutorial.md#tut-1)

### 引导状态机

```
[未开始] ──首次进入──→ [核心引导中] ──步骤6完成──→ [自由探索过渡] ──过渡完成──→ [自由游戏]
                              │                                                │
                              └──加速跳过──→ [自由探索过渡]                       └──条件触发──→ [Mini-tutorial]
```

| 步骤 | 名称 | 教学目标 | 子步骤数 | 完成奖励 |

## [TUT-2] 军师建议 {#tut-2}

### 功能描述

军师建议系统在玩家自由游戏阶段，根据当前游戏状态主动推送策略建议，帮助玩家做出更优决策。

> 🎨 → [UI: 军师建议面板](../ui-layout/TUT-tutorial.md#tut-2)

### 建议触发规则

| 触发条件 | 建议类型 | 优先级 | 冷却 |
|---------|---------|:------:|------|
| 资源接近上限（>80%） | 「粮仓即将满载，建议升级粮仓」 | 高 | 30分钟 |
| 资源严重不足（<10%） | 「粮草告急，建议建造更多农田」 | 高 | 30分钟 |
| 有空闲建筑队列 | 「建造队列空闲，建议升级建筑」 | 中 | 15分钟 |
| 武将可升级但未升级 | 「武将{名}可升级，建议消耗经验书」 | 中 | 20分钟 |
| 科技队列空闲 | 「科技研究空闲，建议开启新研究」 | 中 | 15分钟 |
| 兵力满值未使用 | 「兵力已满，建议出征消耗」 | 低 | 10分钟 |
| 限时NPC即将消失（<2h） | 「{NPC名}即将在{X}小时后离开」 | 高 | 5分钟 |
| 新功能解锁 | 「{功能名}已解锁，点击了解」 | 高 | 一次性 |
| 离线收益预估 > 50%溢出 | 「离线后部分资源将溢出，建议升级仓库」 | 中 | 60分钟 |

### 建议内容结构

| 字段 | 说明 |
|------|------|
| 标题 | 一句话概括建议（≤20字） |
| 描述 | 建议原因和预期收益（≤50字） |
| 行动按钮 | 一键跳转到对应功能界面 |
| 置信度 | 高/中/低，影响显示样式 |
| 过期时间 | 建议的有效时长，过期后自动移除 |

### 建议展示规则

| 属性 | 规则 |
|------|------|
| 最大同时显示 | 3条建议 |
| 排序规则 | 按优先级排序，同优先级按触发时间排序 |
| 自动消失 | 玩家执行建议操作后自动移除该条 |
| 手动关闭 | 可左滑/点击×关闭单条建议 |
| 关闭冷却 | 关闭同类型建议后30分钟内不再推送 |
| 每日上限 | 每天最多推送15条建议（防止打扰） |

## [TUT-3] 引导跳过 {#tut-3}

### 功能描述

核心引导（步骤1-6）不可跳过但可加速，扩展引导和剧情事件可跳过。提供多种加速手段减少等待感。

> 🎨 → [UI: 跳过确认弹窗](../ui-layout/TUT-tutorial.md#tut-3)

### 加速机制

| 加速方式 | 触发条件 | 效果 | 冷却 |
|---------|---------|------|------|
| 对话加速 | 点击任意位置 | 跳过当前打字机效果，立即显示完整文字 | 无 |
| 剧情快进 | 点击「跳过▶▶」按钮 | 跳过当前剧情事件，直接进入下一步 | 每段剧情仅可跳过一次 |
| 动画加速 | 连续点击3次 | 当前动画速度×3（建造/战斗/研究） | 无 |
| 一键完成 | 引导步骤停留超过60秒 | 出现「快速完成」按钮，一键跳过当前子步骤 | 每步限用1次 |

### 不可跳过内容

以下内容**绝对不可跳过**：

| 内容 | 原因 |
|------|------|
| 步骤1-1（主城概览总览） | 首次必须看到完整界面 |
| 步骤2-3（确认建造） | 必须完成一次建造操作 |
| 步骤4-4（首次战斗） | 必须观看至少一场战斗 |
| 所有「引导点击」子步骤 | 必须实际点击目标元素 |

### 剧情跳过规则

| 属性 | 规则 |
|------|------|
| 跳过确认 | 弹出确认：「跳过剧情将无法获得沉浸体验，确认跳过？」 |
| 跳过后过渡 | 水墨晕染过渡（300ms）→ 下一步引导 |
| 跳过奖励 | 跳过剧情不影响阶段奖励获取 |
| 重播 | 跳过的剧情可在「设置→引导回顾」中重播 |

### 引导重玩

| 属性 | 规则 |
|------|------|
| 入口 | 设置 → 引导回顾 / 图鉴 → 剧情回顾 |
| 模式 | 观看模式：不消耗资源、不影响游戏进度 |
| 遮罩 | 简化遮罩（50%透明），不锁定操作 |
| 奖励 | 每次重播奖励铜钱×100（每日限3次） |
| 可重玩内容 | 核心引导6步 + 扩展引导6步 + 剧情事件8段 |

### 自由探索过渡

| 属性 | 规则 |
|------|------|
| 触发 | 步骤6完成后自动触发 |
| 内容 | 阶段奖励领取 + 3个推荐下一步行动 + 已解锁功能列表 |
| 推荐行动 | ① 继续出征 ② 升级建筑 ③ 招募更多武将 |
| 首次提示 | 进入自由游戏后显示3条轻量提示（非强制，5秒自动消失） |
