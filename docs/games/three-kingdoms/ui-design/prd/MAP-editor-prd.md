# [MAP-EDITOR] 地图编辑器 PRD

> **版本**: v2.0 | **日期**: 2026-05-03
> **目的**: 可视化编辑ASCII地图数据，支持多图层、实体管理、导入导出
> **渲染**: 使用游戏内 PixelMapRenderer，表现效果一致

---

## 1. 功能概述

地图编辑器是一个独立运行的工具，用于创建和编辑游戏地图数据。

**核心能力**:
- 可视化编辑ASCII地图(默认100×60色块 = 800×480px)
- 多图层叠加(地形/道路/建筑/装饰/事件)
- 多种工具(查看/画笔/擦除/填充/拾取)
- 实体管理(建筑/NPC/事件点的放置/选择/拖动/删除/重命名)
- 鸟瞰小地图(缩略图+视口指示+点击跳转)
- 导入/导出(ASCII地图文件 + JSON编辑数据)
- 平滑视窗控制(触摸板双指平移/捏合缩放/按钮缩放)
- 使用游戏内同样的渲染器，所见即所得

## 2. 工具系统

### 2.1 工具栏

| 工具 | 图标 | 快捷键 | 说明 |
|------|:----:|:------:|------|
| 查看 | 👁️ | V | 默认工具，只查看不绘制 |
| 画笔 | ✏️ | B | 在活动图层上绘制当前符号 |
| 擦除 | 🧹 | E | 将活动图层的符号替换为空格 |
| 填充 | 🪣 | F | 油漆桶，填充相连区域 |
| 拾取 | 💉 | I | 吸取画布上的符号作为当前符号 |

### 2.2 画笔大小

支持1~5级画笔大小，快捷键1~5：
- 1: 单格 | 2: 2×2 | 3: 3×3 | 4: 4×4 | 5: 5×5

---

## 3. 图层系统

### 3.1 默认图层

| 图层 | 类型 | 说明 | 渲染顺序 |
|:----:|------|------|:--------:|
| 地形层 | terrain | 平原/山地/水域/森林/草地 | 1(最底) |
| 道路层 | road | 道路/小路 | 2 |
| 建筑层 | building | 城墙/城池/关卡 | 3 |
| 装饰层 | decoration | 事件点/宝箱/商队 | 4 |
| 事件层 | event | 事件触发区域 | 5(最顶) |

### 3.2 图层操作

| 操作 | 说明 |
|------|------|
| 选择 | 点击图层选为活动图层 |
| 可见性 | 切换👁️显示/隐藏 |
| 锁定 | 切换🔒锁定/解锁(不可编辑) |
| 上移/下移 | 调整图层渲染顺序 |
| 添加 | 新建自定义图层 |
| 删除 | 删除图层(至少保留1个) |

### 3.3 合并规则

导出时，从底到顶合并所有可见图层：
- 空格(` `)不覆盖下层
- 非空格字符覆盖下层
- 最终输出为单层ASCII地图

## 4. 实体管理

### 4.1 实体类型

| 类型 | 预设 | 符号 | 说明 |
|------|------|:----:|------|
| 建筑 | 城池 | C | 3×3色块 |
| 建筑 | 关隘 | G | 3×3色块 |
| 建筑 | 村落 | V | 3×3色块 |
| NPC | 商人 | $ | 1×1色块 |
| NPC | 将领 | @ | 1×1色块 |
| 事件 | 事件 | ! | 1×1色块 |
| 事件 | 宝箱 | & | 1×1色块 |
| 资源 | 矿脉 | m | 1×1色块 |
| 资源 | 农田 | f | 1×1色块 |

### 4.2 实体操作

| 操作 | 说明 |
|------|------|
| 放置 | 左侧面板选择实体预设 → 点击画布放置 |
| 选择 | 点击实体选中，显示红色虚线高亮 |
| 拖动 | 鼠标拖拽移动实体位置 |
| 键盘移动 | 方向键移动1格，Ctrl+方向键移动5格 |
| 重命名 | 右侧面板点击✏️按钮 |
| 删除 | Delete键或右侧面板点击🗑️按钮 |

### 4.3 实体渲染

- 实体按阵营颜色填充(魏蓝/蜀红/吴绿/中立灰)
- 无边框轮廓，纯色块填充
- 实体名称居中显示
- 选中实体显示红色虚线边框

## 5. 视窗控制

### 5.1 缩放

| 操作 | 说明 |
|------|------|
| 缩放按钮 −/+ | 工具栏缩放按钮，25%步进 |
| 百分比点击 | 点击百分比重置到100% |
| 适配按钮 ⊞ | 自动缩放使地图完整显示在画布中 |
| 触摸板捏合 | 双指张开/合拢缩放 |

缩放范围: 25% ~ 400%

### 5.2 平移

| 操作 | 说明 |
|------|------|
| 触摸板双指 | 双指拖动平移视窗(0.5阻尼系数) |
| 鼠标右键拖拽 | 右键拖动平移视窗 |
| 鼠标中键拖拽 | 中键拖动平移视窗 |
| 鸟瞰小地图 | 点击/拖拽小地图跳转视窗 |

### 5.3 平滑与对齐

- 平移过程为像素级平滑移动(浮点偏移)
- 停止移动后150ms自动对齐到色块边界
- 鼠标拖拽松开时立即对齐

### 5.4 边界限制

- 地图始终部分可见，不允许完全移出视窗
- 允许边缘最多露出30%黑色区域
- 地图比视口小时自动居中

---

## 6. 鸟瞰小地图

**位置**: 画布区域左下角
**尺寸**: 180×120px
**内容**:
- 全局缩略图(按地形颜色渲染)
- 实体标记(红色点)
- 当前视口范围(白色方框)
- 点击/拖拽跳转主视窗

## 7. 图块符号

### 7.1 地形符号

| 符号 | 名称 | 像素渲染 |
|:----:|------|---------|
| `.` | 平原 | 绿色渐变+稀疏噪点 |
| `^` | 山地 | 三角形山峰+阴影+雪顶(部分) |
| `~` | 水域 | 多层波浪动画+反光点 |
| `#` | 森林 | 多种树型(大树/锥树/灌木)+树干 |
| `,` | 草地 | 草丛+偶见小花 |
| `*` | 沙漠 | 沙丘纹理+沙粒 |
| `_` | 泥地 | 深褐 |
| ` ` | 空地 | 黑色 |

### 7.2 道路符号

| 符号 | 名称 | 像素渲染 |
|:----:|------|---------|
| `░` | 道路 | 土黄色(渲染器自动处理轮廓) |

### 7.3 建筑符号

| 符号 | 名称 | 说明 |
|:----:|------|------|
| `▒` | 填充 | 建筑内部色块 |

**建筑绘制规则**:
- 使用实体管理功能放置建筑
- 建筑在画布上按阵营颜色纯色填充，无边框

### 7.4 装饰符号

| 符号 | 名称 | 说明 |
|:----:|------|------|
| `!` | 事件点 | 可交互事件位置 |
| `?` | 未知 | 未探索区域 |
| `%` | 废墟 | 残垣断壁 |
| `&` | 宝箱 | 可拾取宝箱 |
| `$` | 商队 | NPC商队 |
| `@` | 玩家 | 玩家出生点 |

## 8. 导入/导出

### 8.1 导出格式

**ASCII地图文件(.txt)**:
```
# 地图名称
# 描述
MAP:地图名称
SIZE:100x60
TILE:8

CITY: 城市ID=城市名,...

(地形数据行)
```

**JSON编辑数据(.json)**: 完整编辑器状态(图层+实体+元信息)

### 8.2 导入格式

| 格式 | 按钮 | 说明 |
|------|------|------|
| JSON编辑数据 | 📂 导入JSON | 恢复完整编辑器状态(含图层/实体/元信息) |
| ASCII地图文件 | 🗺️ 导入地图 | 解析ASCII文件，显示预览后确认导入 |

## 10. 技术架构

```
┌─────────────────────────────────────────────────┐
│ MapEditorStandalone (React UI)                  │
│  ├── 工具栏(文件/名称/工具/大小/撤销/缩放)       │
│  ├── 符号面板(左侧: 图块+实体放置)               │
│  ├── Canvas画布(中间) + 鸟瞰小地图(左下)         │
│  ├── 图层+实体面板(右侧)                         │
│  └── 状态栏                                      │
├─────────────────────────────────────────────────┤
│ MapEditor (引擎)                                 │
│  ├── 图层管理(增删改查/可见/锁定/排序)            │
│  ├── 画笔工具(查看/绘制/擦除/填充/拾取)           │
│  ├── 实体管理(CRUD/选择/拖动/重命名)              │
│  ├── 历史记录(整体快照撤销/重做)                  │
│  ├── 视口控制(缩放/平移/边界限制)                 │
│  ├── 元信息管理(城市/NPC/事件)                   │
│  └── 导入/导出(ASCII + JSON)                     │
├─────────────────────────────────────────────────┤
│ ASCIIMapParser (解析器)                          │
│  └── 解析ASCII文本 → 结构化地图数据              │
├─────────────────────────────────────────────────┤
│ PixelMapRenderer (渲染器)                        │
│  ├── 地形渲染(批量合并+过渡色+纹理)              │
│  ├── 道路渲染                                    │
│  ├── 建筑渲染                                    │
│  └── 网格线(调试)                                │
└─────────────────────────────────────────────────┘
```
