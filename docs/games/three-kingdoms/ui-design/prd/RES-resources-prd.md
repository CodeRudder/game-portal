## [RES-2] 资源产出 {#res-2}

### 功能描述

资源通过建筑、领土、科技、武将等多来源产出，各来源独立计算后汇总。

### 基础产出公式

```
资源净产出 = Σ(各来源产出) - Σ(各消耗项)
资源实际获得/秒 = 净产出 × (1 + 科技加成) × (1 + 主城加成) × (1 + 武将加成) × 转生倍率
```

### 各来源产出明细

#### 🌾 粮草产出

| 来源 | 基础产出 | 影响因素 | 说明 |
|------|:-------:|---------|------|
| 农田建筑 | 1.0 + 0.5×等级/s | 建筑等级 | 主力产出源 |
| 领土收益 | 0.2×占领城池数/s | 城池数量和品质 | 第二产出源 |
| 科技「屯田术」 | +15%/+25%/+40% | 科技等级 | 百分比加成 |
| 主城等级加成 | Lv10 +18% / Lv20 +35% | 主城等级 | 全局加成 |
| 武将派遣 | 视武将而定 | 武将属性和技能 | 诸葛亮+2.0/s |

#### 💰 铜钱产出

| 来源 | 基础产出 | 影响因素 | 说明 |
|------|:-------:|---------|------|
| 市集建筑 | 0.8 + 0.4×等级/s | 建筑等级 | 主力产出源 |
| 战斗奖励 | 50~500/次 | 关卡难度 | 一次性奖励 |
| 出售装备 | 视装备品质 | 装备等级 | 一次性 |
| 科技「贸易术」 | +15%/+25%/+40% | 科技等级 | 百分比加成 |

#### ⚔️ 兵力产出

| 来源 | 基础产出 | 影响因素 | 说明 |
|------|:-------:|---------|------|
| 兵营建筑 | 0.5 + 0.3×等级/s | 建筑等级 | 主力产出源 |
| 征兵令道具 | 固定量/次 | 道具品质 | 一次性补充 |
| 科技「征兵术」 | +15%/+25%/+40% | 科技等级 | 百分比加成 |

#### 👑 天命获取

| 来源 | 单次获取量 | 频率 | 说明 |
|------|:---------:|:----:|------|
| 声望转生奖励 | 100 + 转生次数×50 | 每次转生 | 转生次数越多越高 |
| 首次通关章节 | 100 | 每章一次 | 首次通关奖励 |
| 成就里程碑 | 50~200 | 一次性 | 特定成就 |
| 日常任务全套完成 | 20 | 每日 | 完成所有日常 |
| 周常任务全套完成 | 80 | 每周 | 完成所有周常 |
| 限时活动排名 | 50~300 | 活动期间 | 按排名梯度 |
| 离线产出（天命阁） | 约5/天 | 持续 | 声望≥3且天命阁已解锁 |
| NPC好感度满级 | 150/位 | 一次性 | 每位NPC首次满级 |
| 远征特殊节点 | 30~80 | 通关时 | 远征天命宝箱节点 |

### 加成叠加规则

```
最终产出 = 基础产出 × (1 + 科技加成) × (1 + 主城加成) × (1 + 武将加成) × 转生倍率 × (1 + VIP加成)
```

- 各类加成为**乘法叠加**（非加法），防止线性膨胀
- 同类加成取最高级生效（如声望等级加成不叠加，取最高）
- VIP加成：VIP1 +5%、VIP3 +10%、VIP5 +20%

## [RES-4] 资源存储 {#res-4}

部分资源有存储上限，超出后产出停止。存储上限通过建筑升级和科技研究提升。

|------|:-------:|---------|------------|

| 💰 铜钱 | ∞ | 无上限 | — |
| ⚔️ 兵力 | 500 | 兵营建筑 | 升级兵营（每级+100~500） |
| 👑 天命 | ∞ | 无上限 | — |

### 溢出规则

| 规则 | 说明 |
|------|------|
| 粮草溢出 | 产出停止，不丢失已有粮草，速率显示"已满！" |
| 兵力满营 | 恢复停止，不丢失已有兵力 |
| 铜钱/天命 | 无上限，永不溢出 |
| 离线收益溢出 | 溢出部分截断，显示"溢出！实际获得 {截断值}" |
| 最低粮草保留 | 军队消耗导致粮草减少时，始终保留10（防止完全归零卡死） |

### 容量警告规则

| 容量百分比 | 状态 | 系统行为 |
|:---------:|:----:|---------|
| 0%~70% | 安全 | 正常显示 |
| 70%~90% | 注意 | 进度条变色（古铜金） |
| 90%~95% | 警告 | 进度条变红 + 脉冲动画 + 军师建议弹出 |
| 95%~100% | 紧急 | 进度条深红 + 快速脉冲 + 强烈建议 |
| 100% | 已满 | 产出停止 + "已满！"标识 + 军师强制建议 |

### 资源保护机制

| 保护规则 | 触发条件 | 行为 |
|---------|---------|------|
| 天命消耗保护 | 单次消耗>100 | 弹出二次确认弹窗 |
| 新手保护期 | 游戏前7天 | 资源不可被其他玩家掠夺 |
| 最低粮草保留 | 军队消耗导致 | 始终保留10粮草 |
| 离线收益封顶 | 离线>72小时 | 收益按72小时计算 |

### 资源转化规则

| 转化方向 | 比例 | 解锁条件 | 手续费 |
|---------|:----:|---------|:-----:|
| 粮草 → 铜钱 | 10:1 | 市集Lv.5+ | 5% |
| 铜钱 → 粮草 | 1:8 | 市集Lv.5+ | 5% |
| 粮草 → 兵力 | 20:1 | 兵营Lv.3+ | 无 |
| 铜钱 → 科技点 | 100:1 | 书院Lv.5+ | 无 |

### 离线效率系数（全局统一）

> 引用全局规范 `SPEC-offline.md` §二，全游戏唯一标准。

| 离线时长 | 效率系数 | 说明 |
|:--------:|:-------:|------|
| 0~2小时 | 100% | 全额收益 |
| 2~8小时 | 80% | 轻度衰减 |
| 8~24小时 | 60% | 中度衰减 |
| 24~48小时 | 40% | 明显衰减 |
| 48~72小时 | 25% | 大幅衰减 |
| >72小时 | 15%（封底） | 封底值，不再继续衰减 |

### 离线收益计算公式

```
离线收益 = Σ(各时段产出) × (1 + 科技加成) × (1 + VIP加成)

各时段产出分段计算:
  时段1 (0~2h):    基础速率 × min(离线秒数, 7200) × 100%
  时段2 (2~8h):    基础速率 × min(离线秒数-7200, 21600) × 80%
  时段3 (8~24h):   基础速率 × min(离线秒数-28800, 57600) × 60%
  时段4 (24~48h):  基础速率 × min(离线秒数-86400, 86400) × 40%
  时段5 (48~72h):  基础速率 × min(离线秒数-172800, 86400) × 25%
  时段6 (>72h):    不再产出额外收益

科技加成: 离线收益I +10% / 离线收益II +15% / 离线收益III +20%
VIP加成: VIP1 +5% / VIP3 +10% / VIP5 +20%
```

### 离线收益触发规则

| 离线时长 | 行为 |
|---------|------|
| ≤5分钟 | 不弹窗，收益静默添加 |
| >5分钟 | 弹出完整离线收益弹窗 |
| >72小时 | 弹窗顶部显示封顶提示，收益按72小时计算 |
