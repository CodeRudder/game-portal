## [MAP-1] 地图规则 {#map-1}

### 地图基础参数

| 参数 | PC 端 | 手机端 |

| 地图尺寸 | 20×15 瓦片（2400×1800px） | 同尺寸（可缩放浏览） |

| 领土数量 | 15 块 | 15 块 |
| 默认缩放 | 100% | 80% |
| 缩放范围 | 50%~200% | 50%~200% |

### 三大区域划分

| 区域 | 位置 | 包含领土 | 阵营色 |
|------|------|---------|--------|
| 魏国 | 左上 | 邺城、许昌、濮阳、北海 | 靛蓝 #2E5090 |
| 蜀国 | 左下 | 成都、汉中、永安、南中 | 赤红 #8B2500 |
| 吴国 | 右侧 | 建业、会稽、柴桑、庐江 | 翠绿 #2E6B3E |
| 中立 | 中心/散布 | 洛阳、长安、襄阳 | 灰褐 #6B5B3E |

### 地形类型（6 种）

| 地形 | 资源倾向 | 战斗效果 |
|------|---------|---------|
| 平原 | 粮草 +30% | — |
| 山地 | 粮草 -20% | 防守方防御 +20% |
| 水域 | 铜钱 +25%（漕运） | — |
| 城池 | 全资源 +10% | 防守方城防加成 |
| 森林 | 兵力 +15%（伏兵） | 伏兵突袭加成 |
| 关隘 | — | 防守方防御 +40%，战略要地 |

### 特殊地标

| 地标 | 位置 | 特殊效果 |
|------|------|---------|
| 洛阳 | 中心 | 全资源产出 +50% |
| 长安 | 左侧 | 科技点产出 +30% |
| 建业 | 右下 | 铜钱产出 +30% |

## [MAP-2] 筛选逻辑 {#map-2}

### 筛选维度

| 筛选组 | 筛选项 | 交互方式 | 组合逻辑 |
|--------|--------|---------|---------|
| **阵营** | 全部 / 蜀 / 魏 / 吴 / 中立 | 单选 | AND |
| **地形** | 平原 / 山地 / 城池 / 森林 / 水域 / 关隘 | 多选 | OR |
| **收益类型** | 粮草 / 铜钱 / 兵力 | 多选 | OR |
| **等级范围** | 全部 / Lv5+ / Lv10+ / Lv15+ | 单选 | AND |
| **热力图** | 开/关 | 切换 | 独立模式 |

### 组合逻辑

- **阵营 ∩ 等级** = AND 关系
- **地形 ∪ 收益** = OR 关系
- 示例：蜀国 + 平原+城池 + 粮草 → 显示蜀国中（平原或城池）且产粮草的领土

### 过滤视觉效果

| 状态 | 效果 |
|------|------|
| 匹配领土 | 正常显示 + 古铜金高亮边框 |
| 不匹配领土 | 亮度降至 40% + 灰度滤镜 |

### 快捷筛选按钮

| 按钮 | 功能 |
|------|------|
| 🏠 我的领土 | 过滤为己方阵营 |
| 📊 高产领地 | 按总收益排序，高亮前 5 |
| ⚔️ 可征服 | 显示与己方相邻的非己方领土 |
| 🗺️ 热力图 | 切换收益热力图模式 |

### 热力图模式

开启后领土按收益高低显示颜色深浅：

| 收益区间 | 颜色 |
|---------|------|
| Top 20%（高收益） | 深金 #C9A84C |
| 中高收益 | 浅金 #E8D48B |
| 中等收益 | 翠绿 #7EC850 |
| 中低收益 | 浅绿 #A8D88A |
| Bottom 20%（低收益） | 灰绿 #6B8B6B |

### 手机端筛选差异

- 入口：右上角「漏斗」图标按钮（44×44px）
- 面板：底部弹出 Bottom Sheet
- 操作：分组纵向排列，底部「重置」+「应用筛选」双按钮

## [MAP-3] 领土系统 {#map-3}

### 领土产出计算

```
领土总产出 = 基础产出 × 地形加成 × 阵营加成 × 科技加成 × 声望加成 × 地标加成
```

| 加成来源 | 示例 |
|---------|------|
| 地形加成 | 平原粮草 +30%、水域铜钱 +25% |
| 阵营加成 | 己方领土全产出 +10% |
| 科技加成 | 「屯田」粮草 +15% |
| 声望加成 | 声望等级 × 2% |
| 地标加成 | 洛阳全资源 +50% |

### 产出气泡显示规则

| 条件 | 行为 |
|------|------|
| 己方领土 | 始终显示产出气泡 |
| 非己方领土 | 不显示（悬停 Tooltip 显示潜在收益） |
| 刚占领领土 | 气泡从 0 开始充能动画 |
| 产出为 0 | 灰色气泡，提示需开发 |
| 缩放 < 60% | 隐藏气泡避免拥挤 |

### 驻防机制

| 参数 | 规则 |
|------|------|
| 驻防兵力上限 | 领土等级 × 100 |
| 驻防效果 | 每点兵力增加防守方 0.1% 防御 |
| 驻防消耗 | 从总兵力池中分配，可随时调回 |
| 自动驻防 | 新占领领土自动分配 50% 兵力上限 |

### 征服规则

```
征服条件：
  - 目标领土与己方领土相邻
  - 可用兵力 ≥ 目标驻防兵力 × 1.5

征服流程：
  1. 点击领土 → 查看详情（驻防兵力、产出、地形）
  2. 点击「征服」→ 兵力部署滑块
  3. 系统预估胜率 = 我方战力 / (敌方战力 × 地形防御系数)
  4. 确认出征 → 进入战斗（或自动结算）

征服结果：
  - 胜利：领土归己方，驻防自动部署
  - 失败：损失 30% 出征兵力，领土不变
```

### 胜率预估公式

```
胜率 = min(95%, max(5%, (我方战力 / 敌方战力) × 50% + 地形修正))
```

| 胜率区间 | 显示颜色 | 建议 |
|---------|---------|------|
| >80% | 翠绿 | 稳操胜券 |
| 60%~80% | 金色 | 有一定风险 |
| 40%~60% | 琥珀橙 | 势均力敌 |
| <40% | 赤红 | 建议强化 |

### 领土等级

| 等级 | 解锁条件 | 产出倍率 | 驻防上限 |
|:----:|---------|:-------:|:-------:|
| 1 | 初始占领 | ×1.0 | 100 |
| 5 | 占领 24 小时 | ×1.2 | 500 |
| 10 | 占领 72 小时 + 投资 | ×1.5 | 1,000 |
| 15 | 占领 7 天 + 重金投资 | ×2.0 | 1,500 |

### 离线领土变化

| 变化类型 | 视觉标记 | 持续时间 |
|---------|---------|---------|
| 新占领领土 | 金色脉冲光环 + "新"标签 | 首次点击后消失 |
| 失去领土 | 红色脉冲光环 + "失"标签 | 首次点击后消失 |
| 收益变化 | 产出气泡闪烁 + ↑↓箭头 | 10 秒后恢复 |
| 防御变化 | 领土边缘闪烁 | 10 秒后恢复 |
