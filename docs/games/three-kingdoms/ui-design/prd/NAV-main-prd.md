## [NAV-1] 主界面功能定位 {#nav-1}

### 功能描述

主界面是玩家进入游戏后的**核心枢纽**，承载导航、资源展示、智能推荐三大职责。放置游戏的核心闭环为：

```
回来 → 看到增长 → 知道做什么 → 行动 → 离开（期待下次回归）
```

主界面通过 7 个一级 Tab 覆盖全部核心玩法：

| Tab | 场景内容 | 对应 PRD |
|-----|---------|---------|
| 天下 | 全屏地图 + 领土标注 + 小地图 | [MAP](./MAP-world-prd.md) |
| 出征 | 战役长卷 + 关卡节点 | [CBT](./CBT-combat-prd.md) |
| 武将 | 武将网格列表 + 筛选 | [HER](./HER-heroes-prd.md) |
| 科技 | 科技树全景（可缩放画布） | [TEC](./TEC-tech-prd.md) |
| 建筑 | 城池俯瞰 + 建筑交互 | [BLD](./BLD-buildings-prd.md) |
| 声望 | 转生对比面板 | → RSP（待创建） |
| 更多 | 下拉菜单（16 项子功能入口） | 见 NAV-3 |

### 资源体系

顶部资源栏常驻显示 4 种核心资源 + 1 种高级货币：

| 资源 | 图标 | 容量 | 产出方式 | 溢出规则 |
|------|:----:|:----:|---------|---------|
| 粮草 🌾 | 🌾 | 有上限 | 农田产出 + 领地产出 | >95% 军师建议推送 |
| 铜钱 💰 | 💰 | 无上限 | 市集产出 + 领地产出 | — |
| 兵力 ⚔️ | ⚔️ | 有上限 | 兵营产出 | >95% 军师建议推送 |
| 科技点 📚 | 📚 | 无上限 | 书院产出 | — |
| 天命 👑 | 👑 | 无上限 | 充值 + 活动 + 首通奖励 | — |

## [NAV-2] 导航逻辑 {#nav-2}

### Tab 切换规则

| 规则 | 说明 |
|------|------|
| 单选互斥 | 同一时刻只有 1 个 Tab 处于激活态 |
| 场景替换 | 切换 Tab 时中央场景区整体替换（无嵌套） |
| 状态保留 | 离开某 Tab 时保留其内部状态（滚动位置、筛选条件等），返回时恢复 |
| 快捷键 | PC 端：T/M/H/B/R 分别对应前 6 个 Tab |

### 手机端导航差异

手机端将 7 个 Tab 压缩为 **5 个底部 Tab**，通过子菜单聚合：

| 底部 Tab | 包含功能 |
|---------|---------|
| 🏠 天下 | 地图 |
| ⚔️ 出征 | 战役 |
| 👤 武将 | 武将列表 |
| 🏛️ 内政▼ | 子菜单：建筑 / 科技 / 装备 / 商店 |
| 👥 社交▼ | 子菜单：好友 / 公会 / 排行 / 邮件 |

**子菜单交互**：点击「内政/社交」展开二级菜单条（40px 高），再次点击收起。

### 手机端快捷图标条

顶部横向滑动图标条，**仅放时效性功能**（不与底部 Tab 交叉）：

| 图标 | 显示条件 | 角标内容 |
|------|---------|---------|
| 📋 任务 | 始终显示 | 可领取数量 |
| 🎪 活动 | 有进行中活动 | 活动数量 |
| 📬 邮件 | 有未读邮件 | 未读数量 |
| ⚔️ 远征 | 远征有空位/归来 | "!" |
| 🎁 签到 | 每日未签到 | — |

## [NAV-4] 导航路径规则 {#nav-4}

### 关键路径

| 用户意图 | 导航路径 | 跳转方式 |
|---------|---------|---------|
| 查看离线收益 | 回归弹窗 → 领取 → 自动跳转军师建议 | 自动 |
| 升级武将 | 军师建议 → 武将详情 → 一键强化 | 点击跳转 |
| 出征新关卡 | 军师建议 → 战役地图 → 战前布阵 → 出征 | 点击跳转 |
| 升级建筑 | 军师建议 → 建筑详情 → 升级 | 点击跳转 |
| 领取任务奖励 | 快捷图标 → 任务面板 → 一键领取 | 点击跳转 |

### 军师建议推荐算法

```
优先级评分 = 基础权重 × 紧迫系数 × 收益系数
```

| 推荐类型 | 基础权重 | 紧迫/收益系数 |
|---------|:-------:|-------------|
| 资源将溢出（>90%容量） | 100 | 紧迫系数 = 容量百分比 |
| 公会 BOSS/活动限时 | 90 | 紧迫系数 = 剩余时间反比 |
| 武将可升级且材料充足 | 80 | 收益系数 = 战力提升百分比 |
| 新关卡解锁且战力足够 | 70 | 收益系数 = 经验/装备收益 |
| 建筑可升级且资源充足 | 65 | 收益系数 = 产出提升百分比 |
| 科技研究完成可开启新项目 | 60 | 收益系数 = 科技效果 |
| 日常任务接近完成（>80%） | 55 | 紧迫系数 = 完成度百分比 |
| 远征队列有空位 | 45 | 收益系数 = 预计奖励 |

**显示规则**：取评分最高的 **3 条** + 离线收益入口（固定），每条可点击跳转对应功能。

## [NAV-5] 离线收益规则 {#nav-5}

### 回归欢迎流程

玩家打开游戏后的**弹窗优先级队列**：

| 优先级 | 弹窗类型 | 触发条件 | 可否跳过 |
|:------:|---------|---------|:-------:|
| 1 | 离线收益弹窗 | 离线 > 5 分钟 | ❌ 不可 |
| 2 | 版本更新公告 | 有新版本 | ✅ 可跳 |
| 3 | 限时活动提醒 | 有进行中活动 | ✅ 可跳 |
| 4 | 回归欢迎面板 | 离线 > 1 小时 | ✅ 可跳 |
| 5 | 军师建议更新 | 有新建议 | ❌ 不可 |

### 离线收益计算公式

```
有效离线时间 = Σ (各时间段 × 对应效率系数)

效率系数阶梯：
  0~2 小时：100%
  2~8 小时：80%
  8~24 小时：60%
  24~48 小时：40%
  48~72 小时：20%
  >72 小时：15%（封顶 72 小时）

离线收益 = 各资源产出速率 × 有效离线时间 × (1 + Σ 加成系数)
```

### 离线收益弹窗内容

| 信息项 | 说明 |
|--------|------|
| 离线时长 | 显示实际离线时间 |
| 离线效率 | 显示百分比（如 80%） |
| 各资源收益 | 图标 + 数值 + 每小时速率 |
| 加速选项 | 观看广告/消耗加速令 → 收益 +50% |

### 回归欢迎面板内容（离线 > 1 小时）

| 分区 | 内容 |
|------|------|
| 领土变动 | 新占领/失去的领土列表 |
| 建筑升级 | 离线期间完成的建筑升级 |
| 武将动态 | 等级变化、远征归来 |
| 未处理事件 | 急报 ×N、日常任务 ×N、活动倒计时 |
| 离线总收益 | 各资源汇总 |

### 通知排队规则

- 同时只显示 **1 个弹窗/横幅**，其余按优先级排队等待
- 弹窗关闭后检查队列，显示下一个
- 免打扰模式：设置中可开启，关闭所有非关键通知（仅保留离线收益弹窗）
- 横幅自动消失：5 秒后淡出
- Toast 堆叠：最多 3 条，新消息从底部推入

### 红点系统规则

| 位置 | 红点条件 | 角标内容 |
|------|---------|---------|
| 武将 Tab | 有可升级武将 | 可升级数量 |
| 建筑子菜单 | 有可升级建筑 | 可升级数量 |
| 声望 Tab | 满足转生条件 | "!" |
| 更多按钮 | 有未读邮件/任务 | 总未读数 |
| 任务图标 | 有可领取奖励 | 可领取数量 |
| 邮件图标 | 有未读邮件 | 未读数量 |
