## [CBT-1] 战役长卷 {#cbt-1}

### 章节结构

战役以水墨山水长卷形式横向展开，共 **6 个历史阶段**，每阶段 **5 个关卡**（共 30 关卡）：

| 序号 | 阶段名 | 关卡数 | 推荐战力范围 | 解锁条件 |
|:----:|--------|:-----:|:----------:|---------|
| 1 | 黄巾之乱 | 5 | 100~500 | 初始解锁 |
| 2 | 群雄割据 | 5 | 500~1,200 | 通关阶段 1 |
| 3 | 官渡之战 | 5 | 1,200~2,500 | 通关阶段 2 |
| 4 | 赤壁之战 | 5 | 2,500~5,000 | 通关阶段 3 |
| 5 | 三国鼎立 | 5 | 5,000~10,000 | 通关阶段 4 |
| 6 | 一统天下 | 5 | 10,000~20,000 | 通关阶段 5 |

### 关卡设计

每个关卡包含：

| 要素 | 说明 |
|------|------|
| 敌方首领 | 1 名武将（有名字、阵营、稀有度） |
| 敌方阵容 | 3 波敌人，每波 3~6 个单位 |
| 推荐战力 | 系统评估的通关所需最低战力 |
| 首通奖励 | 天命、武将碎片、装备等 |
| 重复奖励 | 粮草、铜钱、经验书、兵符 |

### 关卡状态

| 状态 | 视觉标记 | 可操作 |
|------|---------|--------|
| 已三星通关 | 古铜金填充 + ★★★ | 出征 / 扫荡 |
| 已通关（未满星） | 古铜金填充 + 对应星级 | 出征（追求满星） |
| 当前关卡 | 脉冲发光 + 水墨涟漪 | 出征 |
| 未解锁 | 灰色半透明 + 锁图标 | — |

### 星级评定

| 星级 | 条件 | 奖励倍率 |
|:----:|------|:-------:|
| ★☆☆ | 通关（任意 HP > 0） | ×1.0 |
| ★★☆ | 通关 + 我方存活 ≥ 4 人 | ×1.5 |
| ★★★ | 通关 + 我方存活 ≥ 4 人 + 回合数 ≤ 6 | ×2.0 |

## [CBT-2] 战前布阵 {#cbt-2}

### 阵型结构

```
前排（坦克/近战）:  [格1] [格2] [格3]
后排（远程/辅助）:  [格4] [格5] [格6]
```

- 每场战斗最多上阵 **6 名武将**
- 前排承受更多伤害，后排相对安全
- 部分武将技能有位置要求（如"前排攻击+15%"）

### 一键布阵规则

| 规则 | 说明 |
|------|------|
| 优先填充空位 | 不替换已有武将 |
| 战力优先 | 按战力从高到低依次填入 |
| 已出战排除 | 已在其他编队的武将默认不选（可手动覆盖） |
| 填入反馈 | 显示总战力变化（如"战力 +320 ↑"） |

### 智能推荐算法

```
匹配度 = 战力匹配(40%) + 阵营羁绊(25%) + 职业均衡(20%) + 关卡克制(15%)
```

| 维度 | 权重 | 说明 |
|------|:----:|------|
| 战力匹配 | 40% | 总战力尽可能接近关卡推荐战力 |
| 阵营羁绊 | 25% | 3 同阵营：攻击 +15%；6 同阵营：攻击 +25% + 防御 +15% |
| 职业均衡 | 20% | 坦克 + 输出 + 治疗 + 控制合理搭配 |
| 关卡克制 | 15% | 多灼烧敌人→高智/治疗型；多物理→高防/坦克型 |

输出 **1~3 套推荐方案**，每套包含：6 名武将及站位、总战力、匹配度评分、推荐理由。

### 战力预估

```
战力对比条：
  我方战力 ≥ 推荐战力 → 绿色"战力充足 ✓"
  我方战力 < 推荐战力 → 红色"战力不足 ✗"+ 差距百分比
```

## [CBT-3] 战斗机制 {#cbt-3}

### 回合制规则

| 参数 | 规则 |
|------|------|
| 回合上限 | 8 回合 |
| 行动顺序 | 按武将速度属性排序 |
| 普攻 | 自动执行（半自动/全自动模式） |
| 大招 | 需手动释放（半自动模式）或自动释放（全自动模式） |

### 伤害计算公式

```
基础伤害 = 攻击方攻击力 × (1 + 攻击加成) - 防守方防御力 × (1 + 防御加成)
最终伤害 = max(1, 基础伤害) × 暴击系数 × 技能倍率 × 克制系数 × 随机波动(0.9~1.1)
```

| 系数 | 触发条件 | 效果 |
|------|---------|------|
| 暴击系数 | 暴击率随机触发 | ×1.5 |
| 技能倍率 | 使用技能时 | ×技能倍率（1.5~3.0） |
| 克制系数 | 属性克制 | ×1.5（克制）/ ×0.7（被克制） |

### 技能释放规则

| 要素 | 说明 |
|------|------|
| 大招冷却 | 每个大招有独立 CD（通常 3~5 回合） |
| 怒气系统 | 普攻/受击积攒怒气，满怒气可释放大招 |
| 被动技能 | 自动触发，无消耗 |
| 状态效果 | 灼烧/冰冻/中毒/眩晕/流血，持续 2~3 回合 |

### 状态效果

| 状态 | 效果 | 持续回合 |
|------|------|:-------:|
| 灼烧 | 每回合损失最大 HP 的 5% | 2~3 |
| 冰冻 | 无法行动 | 1 |
| 中毒 | 每回合损失最大 HP 的 3% | 3 |
| 眩晕 | 无法行动 | 1 |
| 流血 | 每回合损失攻击力 10% 的 HP | 2~3 |

### 战斗模式

| 模式 | 说明 | 自动选择条件 |
|------|------|-------------|
| 全自动 | 普攻 + 大招全部 AI 释放 | 推荐战力 ≤ 我方 ×0.7 |
| 半自动 | 普攻自动，大招手动（含时停机制） | 推荐战力 ≤ 我方 ×1.0 |
| 全手动 | 所有行动需手动指定 | 推荐战力 > 我方 ×1.0 |

### 大招时停机制（半自动模式）

```
触发条件：同时 ≥ 2 个大招就绪
时停效果：战斗速度降至 0.3×（慢动作）
最大持续：5 秒
结束条件：所有大招释放完毕 / 5 秒超时 / 玩家点击跳过
超时处理：未释放大招自动排队释放
```

#### 兵种克制关系

| 攻击方 | 克制 | 被克 |
|--------|------|------|
| 骑兵 | 步兵(×1.5) | 枪兵(×0.7) |
| 步兵 | 枪兵(×1.5) | 骑兵(×0.7) |
| 枪兵 | 骑兵(×1.5) | 步兵(×0.7) |
| 弓兵 | 无特殊克制 | 无特殊被克 |
| 谋士 | 无特殊克制 | 无特殊被克 |

- 克制系数: ×1.5 (伤害提升50%)
- 被克系数: ×0.7 (伤害降低30%)
- 克制关系影响最终伤害计算: `最终伤害 = 基础伤害 × 克制系数 × 防御减免`

## [CBT-4] 战斗结算 {#cbt-4}

### 奖励计算

```
基础奖励 = 关卡奖励表 × 星级倍率 × 首通倍率
```

| 倍率来源 | 条件 | 倍率 |
|---------|------|:----:|
| 星级倍率 | ★ / ★★ / ★★★ | ×1.0 / ×1.5 / ×2.0 |
| 首通倍率 | 首次通关 | ×3.0（仅一次） |

### 掉落表（示例）

| 掉落物 | 概率 | 说明 |
|--------|:----:|------|
| 粮草 | 100% | 固定掉落，数值随关卡 |
| 铜钱 | 100% | 固定掉落 |
| 经验书 | 80% | 随机品质（小/中/大） |
| 兵符 | 30% | 武将升级材料 |
| 武将碎片 | 10% | 关卡关联武将的碎片 |
| 装备 | 5% | 随机品质装备 |

### 操作评分（半自动/全手动模式）

| 评分 | 条件 | 说明 |
|:----:|------|------|
| S | 零阵亡 + 回合数 ≤ 4 | 完美操作 |
| A | 阵亡 ≤ 1 + 回合数 ≤ 6 | 优秀 |
| B | 通关（其他情况） | 普通 |

## [CBT-5] 扫荡系统 {#cbt-5}

### 解锁条件

- 关卡已**三星通关**
- 拥有足够扫荡令（或消耗元宝替代）

### 扫荡令获取

| 来源 | 数量 |
|------|------|
| 每日登录 | 3 枚 |
| 任务奖励 | 不定 |
| 商店购买 | 10 元宝/枚 |
| VIP5+ 每日免费 | 3 次（不消耗扫荡令） |

### 扫荡规则

| 类型 | 消耗 | 说明 |
|------|------|------|
| 单次扫荡 | 扫荡令 ×1 | 直接获得 1 次通关奖励 |
| 批量扫荡 | 扫荡令 ×N | 直接获得 N 次通关奖励（最多 10 次） |
| 免费扫荡 | 无（VIP5+） | 每日 3 次 |

### 扫荡产出

- 奖励按 ★★★ 倍率（×2.0）计算
- 批量扫荡结果合并显示（如"粮草 ×7500"）
- 稀有掉落额外高亮 + 金色光效

## [CBT-6] 战斗加速 {#cbt-6}

### 加速倍率

| 倍率 | 解锁条件 | 说明 |
|:----:|---------|------|
| 1× | 默认 | 正常速度 |
| 2× | 默认 | 双倍速 |
| 3× | VIP3+ | 三倍速 |
| ⏩ 极速 | VIP5+ | 跳过战斗动画，直接结算 |

### 高倍速信息简化

| 倍率 | 显示策略 |
|:----:|---------|
| 1× | 显示所有伤害数字 |
| 2× | 隐藏普攻数字，仅显示暴击和技能伤害 |
| 3× | 仅显示暴击数字 |
| ⏩ | 完全隐藏伤害数字，仅显示 KO 提示 |

### 自动战斗

```
自动战斗条件：
  - 已设置"自动战斗"开关
  - 当前关卡为已通关的下一关
  - 我方战力 ≥ 推荐战力 × 1.2（确保碾压）

规则：
  - 自动选择全自动模式
  - 战斗结束后自动进入下一场
  - 战败则停止自动战斗
  - 可随时手动中断
```

## [CBT-7] 离线战斗 {#cbt-7}

> 🎨 → [UI: 战役长卷](../ui-layout/CBT-combat.md#cbt-1)

### 离线推图规则

```
触发条件：
  - 已设置"离线推图"开关（默认开启）
  - 当前关卡为已三星通关的下一关
  - 我方战力 ≥ 推荐战力 × 1.2

规则：
  - 每小时自动尝试 1 次推图
  - 成功：推进到下一关（获得 1 星评价 + 基础奖励）
  - 失败：停留在当前关卡
  - 离线最多推进 3 关（防止过度推进错过三星）
  - 离线推图不消耗扫荡令
```

### 离线挂机收益

```
已三星通关关卡的挂机收益持续累积：
  每小时产出 = 关卡基础产出 × (1 + 科技加成 + 声望加成)
  离线收益上限 = 12 小时产出
```
