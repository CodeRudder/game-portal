## [EXP-1] 远征路线 {#exp-1}

### 功能描述

远征路线是放置循环中"离线也能推进"的核心载体。玩家组建远征队伍沿树状分支路线行军，途经各类遭遇节点，最终抵达目标城池。已通关路线可扫荡重复获取收益，多支队伍可并行远征。

### 解锁条件

| 条件 | 数值 |
|------|------|
| 主城等级 | ≥ 5 |
| 入口 | 城池地图「远征营」建筑 / "更多"菜单「远征」 |

### 队列槽位解锁

| 主城等级 | 远征队伍上限 |
|:--------:|:----------:|
| 5 | 1 |
| 10 | 2 |
| 15 | 3 |
| 20 | 4（最大） |

### 路线结构

起点（主城）→ 终点（目标城池），树状分支结构。每条路线包含多个节点，节点类型如下：

| 节点类型 | 说明 | 出现频率 |
|---------|------|---------|
| 山贼 | 普通战斗，低难度 | 高 |
| 天险 | 地形挑战，中等难度 | 中 |
| BOSS战 | 精英战斗，高难度 | 低（路线末端） |
| 宝箱 | 直接获取奖励 | 低 |
| 休息点 | 恢复部分兵力 | 中 |

### 路线难度与时间

| 路线类型 | 难度星级 | 行军时长 | 推荐战力倍率 |
|---------|:-------:|:-------:|:----------:|
| 简单路线 | ⭐~⭐⭐ | 30min~45min | ×1.0 |
| 普通路线 | ⭐⭐⭐ | 45min~1.5h | ×1.5 |
| 困难路线 | ⭐⭐⭐⭐ | 1h~2h | ×2.0 |
| 奇袭路线 | ⭐⭐⭐⭐⭐ | 2h~3h | ×3.0（含稀有掉落） |

### 节点状态

| 状态 | 图标 | 说明 |
|------|------|------|
| 已通关 | ✅ 铜色印章 | 可扫荡 |
| 行军中 | 🔄 金色脉动 | 显示进度条 |
| 未解锁 | 🔒 灰色锁链 | 显示解锁条件（需通关前置节点） |

### 路线解锁规则

- 路线按区域逐步解锁：虎牢关 → 汜水关 → 洛阳 → …
- 后续区域需通关前置区域的任意一条路线
- 奇袭路线需通关同区域的困难路线后解锁

## [EXP-2] 远征队伍 {#exp-2}

### 功能描述

玩家从拥有的武将中挑选最多5名组成远征队伍。队伍总战力决定能否通过路线中的遭遇战斗。同一武将不可同时编入多支队伍。

### 武将选择规则

| 规则 | 说明 |
|------|------|
| 队伍人数 | 最多5名武将 |
| 阵型系统 | 鱼鳞 / 鹤翼 / 锋矢 / 雁行 / 长蛇 / 方圆 |
| 阵营筛选 | 全部 / 魏 / 蜀 / 吴 / 群 |
| 互斥规则 | 同一武将不可编入多支队伍，已出战武将灰显 |
| 羁绊加成 | 同阵营武将 ≥3 名时触发阵营羁绊，全属性 +10% |

### 兵力消耗

| 操作 | 兵力消耗 |
|------|:-------:|
| 每次远征出发 | 每名武将消耗 20 兵力 |
| 扫荡（单次） | 每名武将消耗 10 兵力 |
| 兵力恢复 | 自然恢复 1点/5分钟，兵力道具补充 |

### 阵型效果

| 阵型 | 效果 |
|------|------|
| 鱼鳞 | 防御 +15%，攻击 -5% |
| 鹤翼 | 攻击 +15%，防御 -5% |
| 锋矢 | 速度 +20%，防御 -10% |
| 雁行 | 谋略 +15%，速度 +5% |
| 长蛇 | 攻击 +10%，速度 +10%，防御 -15% |
| 方圆 | 防御 +20%，速度 -10% |

### 智能编队逻辑

1. **智能编队**：基于战力 + 阵营羁绊自动填充，优先激活羁绊
2. **一键最强**：直接选择战力最高的5名可用武将

## [EXP-3] 远征战斗 {#exp-3}

### 功能描述

远征战斗为自动战斗模式，玩家无需手动操作。战斗结果基于双方战力、阵型克制、属性对比自动计算。离线期间战斗同样自动进行。

### 战斗规则

| 参数 | 说明 |
|------|------|
| 战斗模式 | 全自动（不可手动） |
| 回合数 | 最多10回合，10回合内未分胜负判为进攻方失败 |
| 战力判定 | 总战力差距 ±20% 以内为"苦战"，超过 ±20% 为"轻松"或"困难" |
| 阵型克制 | 鱼鳞>锋矢>雁行>鹤翼>鱼鳞（循环克制），克制方全属性 +10% |

### 战斗结果评定

| 结果 | 条件 | 星级 |
|------|------|:----:|
| 大捷 | 剩余血量 > 50% 且无武将阵亡 | ⭐⭐⭐ |
| 小胜 | 剩余血量 10%~50% 或有武将阵亡 | ⭐⭐ |
| 惨胜 | 剩余血量 < 10% | ⭐ |
| 惜败 | 战斗失败 | 无 |

### 自动远征设置

| 设置项 | 选项 | 默认值 |
|-------|------|:-----:|
| 重复次数 | 1 / 3 / 5 / 10 / 无限 | 无限 |
| 失败处理 | 暂停 / 跳过继续 | 暂停 |
| 背包满时 | 暂停 / 自动出售白装 | 暂停 |
| 自动使用药品 | 开 / 关 | 开 |
| 兵力不足时 | 暂停 / 使用兵力道具 | 暂停 |

### 自动远征暂停条件

- 武将兵力耗尽
- 背包已满
- 战斗连续失败 2 次
- 玩家手动暂停

### 离线战斗规则

| 离线场景 | 处理方式 |
|---------|---------|
| 行军中 | 继续行军，离线时间正常计入 |
| 遭遇战斗 | 自动战斗，战斗效率 ×0.8 |
| 路线完成 | 自动结算 + 按自动远征设置决定是否重新出发 |
| 扫荡 | 离线继续扫荡（受每日上限限制） |
| 队列全部完成 | 停止等待玩家上线 |

### 离线收益计算

```
离线远征收益 = 正常收益 × 离线效率系数 × 0.85（战斗惩罚系数）

离线效率系数（引用全局离线规范）：
  · 0~2h: 100%  |  2~8h: 80%  |  8~24h: 60%
  · 24~48h: 40% |  48~72h: 25% | >72h: 15%
  · 可叠加科技/VIP/声望加成（上限 +100%）
  · 最终效率上限 100%

离线战斗胜率 = 正常胜率 × 0.85
离线时间上限 = 72小时（超出部分不计入）
```

## [EXP-4] 远征奖励 {#exp-4}

### 功能描述

远征奖励分为战斗掉落、路线结算、首通奖励和里程碑奖励四大类。扫荡系统允许已通关路线快速获取收益。

### 基础奖励表（按路线难度）

| 路线难度 | 粮草 | 铜钱 | 铁矿 | 装备碎片 | 经验值 |
|---------|:----:|:----:|:----:|:-------:|:-----:|
| 简单 | +200 | +400 | +1 | ×1 | +500 |
| 普通 | +400 | +800 | +2 | ×2 | +1,200 |
| 困难 | +600 | +1,200 | +4 | ×4 | +2,500 |
| 奇袭 | +1,000 | +2,000 | +6 | ×6 | +5,000 |

### 掉落表

| 掉落物 | 普通节点掉率 | BOSS节点掉率 | 奇袭路线BOSS掉率 |
|-------|:----------:|:----------:|:--------------:|
| 装备碎片 | 30% | 60% | 80% |
| 武将碎片（随机） | 5% | 15% | 25% |
| 技能书 | 2% | 10% | 20% |
| 稀有材料 | — | 8% | 15% |
| 传说装备（整件） | — | 1% | 3% |

### 首通奖励

| 奖励 | 说明 |
|------|------|
| 稀有武将碎片 ×1 | 路线关联武将 |
| 元宝 ×50 | 固定奖励 |
| 声望 +20 | 固定奖励 |
| 路线三星成就 | 解锁扫荡功能 |

### 扫荡系统

**解锁条件**：路线三星通关（大捷评级 ×3 或累计通关 ×3）

| 扫荡类型 | 消耗 | 收益 | 每日次数限制 |
|---------|------|------|:----------:|
| 普通扫荡 | 扫荡令 ×1 | 基础奖励 ×100% | 5次/路线 |
| 高级扫荡 | 扫荡令 ×3 | 基础奖励 ×150% + 保底稀有掉落 | 3次/路线 |
| 免费扫荡 | 无 | 基础奖励 ×50% | 1次/路线 |

### 里程碑奖励（区域进度）

| 里程碑 | 条件 | 奖励 |
|-------|------|------|
| 初出茅庐 | 通关第1条路线 | 铜钱 ×1,000 + 扫荡令 ×5 |
| 百战之师 | 通关10条路线 | 元宝 ×100 + 稀有装备箱 ×1 |
| 远征名将 | 通关30条路线 | 传说武将碎片 ×5 + 元宝 ×200 |
| 天下布武 | 通关全部路线 | 专属称号「征伐天下」+ 传说装备箱 ×3 |

### 一键远征分配逻辑

1. 按队伍战力从高到低排序
2. 按路线推荐战力从高到低排序
3. 最高战力队伍 → 最高推荐战力路线
4. 已扫荡路线优先（稳定收益）

### 远征日志

| 属性 | 说明 |
|------|------|
| 内容 | 队伍编号 + 路线名称 + 开始/结束时间 + 遭遇列表 + 战斗结果 + 奖励 |
| 失败记录 | 标红 + 失败原因（战力不足/阵型克制等） |
| 保留时长 | 7天（VIP延长至14天） |
