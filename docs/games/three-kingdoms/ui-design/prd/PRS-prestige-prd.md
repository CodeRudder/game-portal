## [PRS-1] 声望等级 {#prs-1}

### 功能描述

声望等级是玩家全局进度的核心度量。声望值通过游戏行为累积，达到阈值自动升级。声望等级直接影响转生倍率和特权解锁。

### 等级阈值公式

```
等级N所需累计声望 = 1000 × N^1.8
```

### 升级规则

- 声望值达到阈值后**自动升级**，无需手动操作
- 升级时弹出金色横幅通知："声望等级提升至 Lv.X！解锁：{特权名}"
- 升级动画：声望进度条满格闪烁 → 金色粒子爆发 → 等级数字跳变
- 接近升级（≥90%进度）时进度条脉冲闪烁提醒

## [PRS-2] 声望获取 {#prs-2}

### 功能描述

声望值通过多种游戏行为获取，鼓励玩家参与所有系统。

### 获取途径与数值

| 途径 | 声望值 | 频率 | 说明 |
|------|:------:|:----:|------|
| 通关新关卡 | 50~500/关 | 一次性 | 关卡难度越高声望越多 |
| 三星通关奖励 | 基础×1.5 | 一次性 | 首次三星通关额外乘算 |
| 每日声望任务 | 100~300/日 | 每日 | 3个声望专属日常任务 |
| 占领领土产出 | 5~30/小时 | 持续 | 领土数量和品质决定产出 |
| NPC好感度升级 | 200~800/级 | 一次性 | 每位NPC好感度每升一级 |
| 成就完成 | 100~2,000/个 | 一次性 | 成就难度越高声望越多 |
| 远征完成 | 50~200/次 | 每次远征 | 路线长度决定声望量 |
| 活动排名奖励 | 500~5,000 | 每次活动 | 排名梯度发放 |
| 离线累积 | 正常产出的50% | 离线时 | 受统一离线效率系数影响 |

### 声望值获取规则

- **每日获取上限**：无硬性上限，鼓励持续游玩
- **离线声望**：领土产出部分按50%系数累积，受离线效率阶梯影响（引用 `SPEC-offline.md`）
- **重复关卡**：已通关关卡扫荡不获得声望（仅首次通关奖励）
- **声望值不可消耗**：声望值只增不减，转生时保留累计值

## [PRS-4] 转生系统 {#prs-4}

### 功能描述

转生是核心放置循环引擎。玩家重置大部分进度，换取**永久倍率加成**，使下一轮效率更高、走得更远。

### 转生解锁条件

玩家必须**同时满足**以下所有条件：

| 条件类型 | 具体要求 | 说明 |
|---------|---------|------|
| 主城等级 | ≥ 30级 | 确保已体验完整建筑养成线 |
| 通关进度 | ≥ 第4阶段"赤壁之战"全部通关 | 确保已体验核心剧情 |
| 武将数量 | ≥ 15位 | 确保武将系统深度参与 |
| 总战力 | ≥ 50,000 | 确保基础养成完成度 |
| 特定成就 | 完成"初露锋芒"成就链（5个子成就） | 引导体验多系统 |
| 转生冷却 | 距上次转生 ≥ 72小时（首次无限制） | 防止频繁转生 |

### 成就链"初露锋芒"

| 子成就 | 要求 | 引导目的 |
|--------|------|---------|
| 崭露头角 | 主城达到20级 | 建筑养成 |
| 猛将如云 | 拥有10位武将 | 武将收集 |
| 兵强马壮 | 单个武将战力达到5,000 | 武将强化 |
| 谋定后动 | 完成5项科技研究 | 科技树 |
| 威震四方 | 占领20块领土 | 地图扩张 |

### 转生次数与上限

- **转生次数上限**：20次（可随版本更新扩展）
- 每次转生解锁新内容：

| 转生次数 | 解锁内容 |
|:-------:|---------|
| 第1次 | 解锁"天命"资源 + 第5阶段"三国鼎立" |
| 第2次 | 解锁"转生专属科技树" |
| 第3次 | 解锁"神话武将"招募池 |
| 第5次 | 解锁"跨服远征"玩法 |
| 第10次 | 解锁"帝王模式"（全系统加成翻倍） |

### 转生倍率公式

```
转生倍率 = 1.5 × (1 + 声望等级 × 0.05) × (1 + (转生次数 - 1) × 0.1)
```

**倍率速查表**：

| 转生次数 | 声望等级 | 最终倍率 | 意义 |
|:-------:|:-------:|:-------:|------|
| 1 | 5 | 1.88× | 首次转生，效率接近翻倍 |
| 1 | 10 | 2.25× | 首次转生高声望 |
| 2 | 10 | 2.48× | 二次转生 |
| 3 | 15 | 3.19× | 效率超3倍 |
| 5 | 20 | 4.50× | 五次转生 |
| 10 | 30 | 7.20× | 效率超7倍 |
| 20 | 50 | 12.00× | 满转生 |

### 倍率作用范围

| 系统 | 受倍率影响 | 说明 |
|------|:---------:|------|
| 全资源产出 | ✅ | 粮草/铜钱/兵符等所有资源产出 |
| 武将经验获取 | ✅ | 战斗经验和经验书效果 |
| 建筑升级速度 | ✅ | 升级时间缩短（除以倍率） |
| 科技研究速度 | ✅ | 研究时间缩短（除以倍率） |
| 关卡扫荡收益 | ✅ | 扫荡获得的资源翻倍 |
| 离线收益 | ✅ | 离线累积速度翻倍 |
| 商店价格 | ❌ | 不受倍率影响 |
| 招募消耗 | ❌ | 不受倍率影响 |
| PVP属性 | ❌ | 竞技场/切磋属性不受影响 |

### 保留/重置规则

| 系统/数据 | 转生后状态 | 保留比例 | 说明 |
|----------|:---------:|:-------:|------|
| **武将拥有权** | ✅ 保留 | 100% | 所有已获得武将保留 |

| **建筑解锁状态** | ✅ 保留 | 100% | 已解锁的建筑蓝图保留 |
| **粮草/铜钱/兵符** | ❌ 清零 | 0% | 全部清零 |

| **公会/好友/NPC好感度** | ✅ 保留 | 100% | 社交关系保留 |
| **成就/任务/图鉴** | ✅ 保留 | 100% | 永久保留 |
| **VIP等级** | ✅ 保留 | 100% | 充值相关永久保留 |

### 转生后加速机制

| 机制 | 说明 |
|------|------|
| 初始资源赠送 | 粮草×5,000 + 金币×3,000 |
| 建筑1~10级 | 升级时间÷倍率（几乎瞬间完成） |
| 已通关关卡 | 可直接扫荡（三星记录保留） |
| 一键重建 | 自动按上次优先级升级建筑至资源允许的最高等级 |
| 转生专属任务 | 3个简单任务，奖励额外资源包 |

### 收益模拟器规则

转生面板内置收益模拟器，帮助玩家决策"转还是等"：

```
计算逻辑:
  - 根据当前声望获取速度，预测未来X小时的声望增长
  - 计算对应声望等级提升带来的倍率增量
  - 标注"推荐时机"（边际收益递减拐点）
  - 展示对比: 立即转生 vs 再等2h/6h/12h/24h 的倍率差异
```
