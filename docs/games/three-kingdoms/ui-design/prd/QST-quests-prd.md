## [QST-1] 任务类型 {#qst-1}

### 功能描述

任务系统是玩家行为引导的核心模块，通过主线/支线/日常/成就四类任务为玩家提供短期目标和长期追求，驱动放置核心循环的持续运转。

> 🎨 → [UI: 任务布局](../ui-layout/QST-quests.md#qst-1)

### 任务分类总览

| 任务类型 | 总数量 | 刷新规则 | 重置规则 | 核心作用 |
|---------|:------:|---------|---------|---------|
| **主线任务** | 80-120 个 | 不刷新，永久存续 | 不重置 | 引导解锁新玩法 |
| **支线任务** | 30-50 个 | 不刷新，一次性 | 不重置 | 提供额外稀有奖励 |
| **日常任务** | 每日 6 个 | 每日 05:00 重置 | 进度清零 | 每日行为引导 |
| **周常任务** | 每周 4 个 | 每周一 05:00 重置 | 进度清零 | 每周行为引导 |
| **成就任务** | 60-100 个 | 不刷新，永久追踪 | 不重置 | 长期累计目标 |

### 主线任务（Main Quest）

| 属性 | 规格 |
|------|------|
| 总数量 | 约 80~120 个（随版本章节逐步解锁） |
| 解锁规则 | 前置任务完成后自动解锁下一个 |
| 任务链结构 | 每章 8~12 个任务，完成后解锁下一章 |

**主线章节结构：**

```
第一章「黄巾之乱」：1-1 → 1-2 → ... → 1-10（通关后解锁第二章）
第二章「讨伐董卓」：2-1 → 2-2 → ... → 1-12
第三章「群雄割据」：3-1 → ...
第四章「官渡之战」：4-1 → ...
第五章「赤壁之战」：5-1 → ...
...
```

**主线任务类型标签：**

| 标签 | 类型 | 示例目标 |
|------|------|---------|
| 🏯 攻城类 | 战斗推进 | 击败 X 个敌军 / 攻占 Y 座城池 |
| ⚔️ 养成类 | 角色培养 | 将武将升级到 N 级 / 强化装备到 +M |
| 📜 剧情类 | 剧情推进 | 阅读剧情对话 / 触发特定事件 |
| 🏗️ 建设类 | 城池发展 | 建造/升级 N 座建筑 |

### 支线任务（Side Quest）

| 属性 | 规格 |
|------|------|
| 总数量 | 每章 3~5 个，全章节约 30~50 个 |
| 解锁规则 | 对应主线章节推进到指定节点后解锁 |
| 特点 | 奖励比主线更丰厚（含稀有道具），但难度更高 |

### 日常任务（Daily Quest）

| 属性 | 规格 |
|------|------|
| 总数量 | 每日 6 个（固定池 20 个中随机抽取） |
| 刷新规则 | 每日 05:00 重置，未完成进度清零 |

### 周常任务（Weekly Quest）

| 属性 | 规格 |
|------|------|
| 总数量 | 每周 4 个（固定池 12 个中随机抽取） |
| 刷新规则 | 每周一 05:00 重置 |
| 奖励等级 | 比日常高 3~5 倍，含周限稀有道具 |

### 成就任务（Achievement）

| 属性 | 规格 |
|------|------|
| 总数量 | 约 60~100 个（永久存续） |
| 解锁规则 | 部分初始可见，部分隐藏（达成条件后揭示） |
| 分类标签 | 「战斗」「养成」「收集」「社交」「探索」五大类 |
| 奖励 | 成就点数 + 专属称号 + 稀有道具 |

**成就分级：**

| 等级 | 图标 | 定位 | 示例 |
|------|------|------|------|
| 🥉 铜级 | 铜牌 | 基础目标 | "首次招募武将" |
| 🥈 银级 | 银牌 | 进阶目标 | "招募 10 名武将" |
| 🥇 金级 | 金牌 | 高级目标 | "集齐全阵营武将" |

## [QST-2] 任务结构 {#qst-2}

### 功能描述

定义单个任务的数据结构、目标类型、进度计算规则和奖励分配机制。

> 🎨 → [UI: 任务布局](../ui-layout/QST-quests.md#qst-2)

### 任务数据结构

```
任务 = {
  ID: string,                    // 唯一标识，如 "MAIN_1_8"
  名称: string,                  // 任务名称
  描述: string,                  // 任务描述文本
  类型: 主线 | 支线 | 日常 | 周常 | 成就,
  子标签: 攻城 | 养成 | 剧情 | 建设 | ...,
  章节: string,                  // 所属章节（主线/支线）
  
  目标: {
    类型: 击败 | 升级 | 收集 | 建造 | 对话 | 登录 | ...,
    目标ID: string,              // 关联的游戏实体ID
    目标数量: int,
    当前进度: int,
  },
  
  奖励: [
    { 资源类型: 铜钱 | 粮草 | 元宝 | 经验 | 道具 | 装备箱 | ..., 数量: int },
    ...
  ],
  
  前置条件: [
    { 类型: 任务完成, 目标任务ID: string },
    { 类型: 等级要求, 最小等级: int },
    { 类型: 关卡进度, 最小关卡: string },
  ],
  
  后续任务ID: string | null,     // 完成后自动解锁的任务
  状态: 未解锁 | 进行中 | 待领取 | 已完成,
}
```

### 目标类型与进度计算

| 目标类型 | 进度计算方式 | 离线是否计入 | 说明 |
|---------|------------|:----------:|------|
| 击败敌军 | 击杀数累加 | ✅ 是 | 离线挂机战斗自动推进，上限 2 小时进度 |
| 攻占城池 | 占领数累加 | ✅ 是 | 离线挂机战斗计入 |
| 升级武将 | 等级达到阈值 | ❌ 否 | 需在线手动操作 |
| 强化装备 | 强化次数/等级 | ❌ 否 | 需在线手动操作 |
| 建造建筑 | 建造/升级次数 | ❌ 否 | 需在线手动操作 |
| 收集资源 | 资源量累计 | ✅ 是 | 离线挂机产出计入 |
| 阅读剧情 | 完成对话标记 | ❌ 否 | 需在线操作 |
| 招募武将 | 招募次数 | ❌ 否 | 需在线操作 |
| 登录游戏 | 登录标记 | ✅ 自动 | 自动完成 |
| 消耗货币 | 消耗量累计 | ❌ 否 | 需在线操作 |
| 炼制合成 | 炼制次数 | ❌ 否 | 需在线手动操作 |

### 奖励规则

#### 奖励稀有度分级

| 任务类型 | 奖励等级 | 典型奖励 |
|---------|---------|---------|
| 主线任务 | 基础~进阶 | 铜钱×2000~8000 + 经验×1000~5000 + 蓝色装备箱 |
| 支线任务 | 进阶~稀有 | 铜钱×5000~15000 + 元宝×30~100 + 紫色装备箱 + 稀有道具 |
| 日常任务 | 基础 | 铜钱×1000~3000 + 强化石×1~2 |
| 周常任务 | 进阶 | 铜钱×5000~15000 + 元宝×50~150 + 紫色装备箱 |
| 成就任务（铜级） | 基础 | 成就点数+10 + 铜色称号 + 铜钱×1000 |
| 成就任务（银级） | 进阶 | 成就点数+30 + 银色称号 + 元宝×50 |
| 成就任务（金级） | 稀有 | 成就点数+100 + 金色称号 + 金色装备碎片×5 |

#### 奖励缩放公式

```
主线任务奖励 = 基础奖励 × (1 + 章节序号 × 0.15)
  - 第一章：1.15x
  - 第五章：1.75x
  - 第十章：2.50x

日常任务奖励 = 基础奖励 × (1 + 玩家等级 × 0.05)
  - 等级 10：1.50x
  - 等级 20：2.00x
  - 等级 30：2.50x
```

### 任务跳转映射

| 任务关键词 | 跳转目标 | 跳转后高亮 |
|-----------|---------|-----------|
| 击败/战斗/攻占 | 战斗关卡选择界面 | 对应关卡按钮脉冲高亮 |
| 强化/升级装备 | 背包→装备详情→强化 | 强化按钮高亮 |
| 炼制/合成 | 背包→炼制界面 | 炼炉中心高亮 |
| 招募/获得武将 | 招募界面 | 招募按钮高亮 |
| 建造/升级建筑 | 城池建设界面 | 对应建筑高亮 |
| 登录/签到 | 自动完成，不跳转 | — |
| 消耗铜钱/元宝 | 商店界面 | 对应商品区域高亮 |
| 阅读剧情 | 剧情回放界面 | 对应章节高亮 |

### 跳转失败处理

| 情况 | 处理方式 |
|------|---------|
| 目标功能未解锁 | Toast 提示「需完成主线 X-X 解锁」，「前往」按钮置灰 |
| 目标功能需要更高等级 | Toast 提示「主公等级需达到 N 级」 |
| 跳转目标在子菜单深处 | 自动展开菜单层级，逐级高亮引导 |

### 离线任务进度效率衰减

```
离线时间      任务进度上限
0~1小时      100% 正常计算
1~4小时      80% 效率（线性衰减）
4~8小时      50% 效率
8~12小时     30% 效率
12小时以上    20% 效率（最低值）
```

### 任务追踪规则

| 属性 | 规格 |
|------|------|
| 最大追踪数 | 3 个（超出需手动替换） |
| 自动追踪 | 主线任务默认追踪，日常/周常手动添加 |
| 追踪面板位置 | 主界面常驻显示（PC左上角 / 手机顶部窄条） |

## [QST-3] 日常任务 {#qst-3}

### 功能描述

日常任务为玩家提供每日行为目标，通过活跃度系统引导玩家参与核心玩法循环，是每日留存的核心驱动力。

> 🎨 → [UI: 任务布局](../ui-layout/QST-quests.md#qst-3)

### 刷新规则

| 参数 | 规则 |
|------|------|
| 重置时间 | 每日 05:00 |
| 重置内容 | 任务列表重新抽取、进度清零、活跃度清零 |
| 未领取奖励 | 保留至下次重置前（不会丢失当日已完成的奖励） |
| 抽取规则 | 从固定池 20 个中随机抽取 6 个，保证类型多样性 |

### 抽取保证规则

```
6 个日常任务抽取保证：
  - 至少 1 个战斗类（D02/D03/D17）
  - 至少 1 个养成类（D04/D05/D06/D08）
  - 至少 1 个自动完成类（D01/D13/D15）
  - 最多 2 个同类任务
  - D01（登录）必定出现
```

### 活跃度系统

**活跃度获取：** 每完成 1 个日常任务获得对应活跃度（10-20点）

**活跃度宝箱奖励：**

| 活跃度阈值 | 奖励内容 |
|:---------:|---------|
| 40 | 铜钱 ×5000 + 强化石 ×2 |
| 60 | 元宝 ×50 + 招募令 ×1 |
| 80 | 元宝 ×100 + 紫色装备箱 ×1 |
| 100 | 元宝 ×200 + 金色装备碎片 ×3 |

**活跃度宝箱规则：**
- 达到阈值后自动解锁，需手动领取
- 每日重置时未领取的宝箱奖励**不保留**
- 活跃度上限为 100（超出部分不累计）

### 成就解锁条件

| 类型 | 规则 |
|------|------|
| 初始可见 | 基础成就（铜级）默认可见，玩家可预览目标 |
| 条件揭示 | 进阶/高级成就在前置成就完成或特定条件满足后揭示 |
| 隐藏成就 | 特殊成就不显示条件，达成后弹出惊喜通知 |

```
隐藏揭示规则：
  - 🥉 铜级成就：全部初始可见
  - 🥈 银级成就：对应铜级成就完成后揭示
  - 🥇 金级成就：对应银级成就完成后揭示
  - 特殊隐藏：满足隐藏条件时才揭示（如"以弱胜强"）
```

### 成就奖励规则

| 奖励类型 | 说明 |
|---------|------|
| **成就点数** | 累计成就点数可解锁成就商店中的专属物品 |

| **剧情解锁** | 少数特殊成就解锁隐藏剧情线 |
