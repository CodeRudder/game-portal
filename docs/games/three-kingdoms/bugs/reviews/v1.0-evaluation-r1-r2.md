## R1→R2 修复记录

### P1-1: 资源栏未固定顶部
- **根因**：游戏容器被IdleGamePage包裹，未撑满视口
- **修复**：IdleGamePage三国游戏直接全屏 + flex布局
- **Commit**：6b4ffdb + 0d5284c

### P1-2: Tab栏不在底部
- **根因**：同上，容器布局问题
- **修复**：flex column布局，Tab栏flex-shrink:0在底部

### P1-3: 字体过小
- **根因**：tk-res-cap/tk-res-rate设10px
- **修复**：PC端12px，移动端11px
