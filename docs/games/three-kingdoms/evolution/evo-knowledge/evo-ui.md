# UI 实践知识 (EVO-UI)

> **来源**: R1~R20 评测迭代中发现的UI级经验。
> **不归档**: 持续有效，新增时追加。

---

### EVO-UI-001: 弹窗独立
- 所有弹窗组件必须独立为单独文件，使用SharedPanel统一容器
- **来源**: R1 | 原EVO-002

### EVO-UI-002: 选择器先探后测
- 测试脚本先做DOM探测确认class名，再编写断言
- **来源**: R2 | 原EVO-007

### EVO-UI-003: CSS拆分最小成本原则
- CSS文件超限时优先提取独立功能块，而非大规模重构
- **来源**: R2 | 原EVO-021

### EVO-UI-004: data-testid 覆盖率要求
- 所有UI组件根元素必须有data-testid，kebab-case命名
- 动态列表项使用模板，优先级: P0主界面→P1交互→P2辅助
- **来源**: R7 | 原EVO-056
