# v3.0 攻城略地 — 详细开发计划

> **核心目标**: 关卡推进、自动战斗、战利品奖励

---

## 功能清单

### 3.1 关卡地图（关卡Tab）
- 6章15关线性地图：黄巾之乱→群雄割据→官渡之战→赤壁之战→三国鼎立→一统天下
- 节点状态：已完成(金色+星级) / 当前(脉冲发光) / 未解锁(灰色+锁图标)
- 点击关卡弹出详情：描述+推荐战力+敌方阵容+奖励预览

### 3.2 战斗准备
- 武将编队：从已招募武将中选择最多6位
- 战力对比：我方总战力 vs 敌方总战力
- 推荐武将提示
- 开始战斗按钮

### 3.3 自动战斗
- 基于属性的自动计算（攻击vs防御，回合制模拟）
- 战斗过程动画（简化版：双方血条+伤害数字）
- 战斗报告：伤害/治疗/回合数/关键事件
- 1-3星评定（基于剩余血量百分比）

### 3.4 战利品
- 通关奖励：资源+经验+特殊道具
- 奖励领取弹窗
- 扫荡功能：已通关关卡可快速扫荡获得奖励

### 3.5 关卡解锁
- 前置关卡通关 → 下一关解锁
- 战力门槛提示（低于推荐战力时警告但仍可挑战）

## 数据流
```
开始战斗: engine.startBattle(stageId, generalIds)
  → CampaignBattleSystem.simulate(attackers, defenders)
  → BattleResult { victory, stars, rewards, report }
  → 胜利: 标记关卡完成, 发放奖励, 解锁下一关
  → 失败: 显示报告, 可重试
```
