# v2.0 招贤纳士 — 详细开发计划

> **核心目标**: 武将招募、查看、升级、派遣到建筑增加产出

---

## 功能清单

### 2.1 武将招募
- 招募面板：显示可招募武将列表（铜钱招募）
- 招募费用：普通(500铜钱) / 精良(2000) / 史诗(5000) / 传奇(10000)
- 招募动画：武将卡片翻转出现
- 首次免费：许褚自动加入

### 2.2 武将列表（武将Tab）
- 卡片网格：头像+名字+等级+品质+阵营色条
- 阵营筛选按钮：全部/蜀/魏/吴/群
- 品质筛选：全部/传奇/史诗/精良/普通
- 排序：等级/攻击/防御/综合战力

### 2.3 武将详情面板
- 点击卡片展开详情
- 属性面板：攻击/防御/智谋/统率/魅力（1-100数值条）
- 技能列表：技能名+图标+描述+效果数值
- 历史传记：2-3句简介
- 品质评级+阵营归属

### 2.4 武将升级
- 消耗资源（铜钱+粮草）提升等级
- 属性随等级线性成长
- 升级动画+属性变化飘字

### 2.5 武将派遣
- 将武将分配到建筑（每个建筑最多1位）
- 派遣加成：统率→兵营+兵力产出，智谋→书院+科技点等
- 建筑面板显示已派遣武将头像
- 武将列表显示派遣状态

## 数据流
```
招募: engine.recruitGeneral(id) → 扣铜钱 → generals[id] = {level:1, exp:0}
升级: engine.levelUpGeneral(id) → 扣资源 → generals[id].level++
派遣: engine.assignGeneral(generalId, buildingId) → 建筑产出 += 武将加成
```
