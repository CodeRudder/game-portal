## 三、数据流设计

```
世界地图深化
  │
  ├─ 驻防机制
  │   ├─ 点击己方领土 → 选择「驻防」
  │   ├─ GarrisonPanelComponent: 显示可用武将列表
  │   ├─ 选择武将 → GarrisonSystem: 记录驻防关系
  │   ├─ 驻防效果: 领土产出+N% / 防御+N%
  │   └─ 驻防武将不可同时出战（编队互斥）
  │
  ├─ 攻城战深化
  │   ├─ 选择敌方领土 → SiegeSystem: 计算胜率预估
  │   │   ├─ 攻方: 出战武将战力 + 兵种克制
  │   │   └─ 守方: 驻防武将战力 + 城防值 + 地形加成
  │   ├─ 城池类领土 → 攻城战流程(MAP-4): 城防计算+消耗+占领
  │   ├─ 确认攻城 → 进入战斗
  │   └─ 胜利 → 领土归属变更 + 攻城奖励
  │
  ├─ 离线领土
  │   ├─ 离线期间: AI可能攻占我方无驻防领土
  │   ├─ OfflineCalculator: 计算离线领土变化
  │   └─ 回归面板: 显示领土得失摘要
  │
  ├─ 地图辅助
  │   ├─ MapFilterSystem: 阵营/地形/收益/等级/热力图筛选
  │   ├─ MapStatsPanelComponent: 领土/资源/战斗/探索/事件统计
  │   └─ 领土产出计算: 基础×地形×阵营×科技×声望×地标×驻防加成
  │
NPC系统
  │
  ├─ 地图NPC展示
  │   ├─ NPCMapSystem: 根据NPC位置在地图上渲染图标
  │   ├─ 拥挤检测: 同区域多个NPC → 聚合气泡(显示数量)
  │   └─ 点击NPC图标/气泡 → NPC信息弹窗
  │
  ├─ NPC交互
  │   ├─ NPC信息弹窗: 名字/职业/好感度/交互选项
  │   ├─ 交互选项: 对话/赠送/任务/交易（根据好感度解锁）
  │   └─ 对话: NPCDialogPanel → 选择选项 → 好感度变化
  │
  ├─ 好感度系统
  │   ├─ 每次对话: 好感度+N
  │   ├─ 赠送偏好物品: 好感度+N×倍率
  │   ├─ 好感度等级提升 → 解锁新交互选项
  │   └─ Lv.5: 解锁NPC专属羁绊技能
  │
  ├─ NPC高级交互
  │   ├─ 赠送系统: 普通礼物+10/稀有+30/专属+50(偏好×1.5)
  │   ├─ 切磋系统: 自动战斗3回合制，胜利+15/失败+5好感度
  │   ├─ 结盟系统: Lv.5满级结盟，每类NPC限1位
  │   └─ NPC任务链: 按好感度解锁5~8任务，含终章稀有奖励
  │
  ├─ NPC刷新与离线
  │   ├─ NPC出现: 按主城等级/建筑解锁，稀有NPC限时出现
  │   ├─ NPC日程: 24h循环(劳作/休息/睡眠/巡逻)
  │   └─ NPC离线行为: 农民产粮草/商人产铜钱/学者加速科技/斥候收集情报
  │
事件系统
  │
  ├─ 事件触发
  │   ├─ EventTriggerSystem: 检查触发条件（时间/概率/条件）
  │   ├─ 触发 → EventUINotification: 显示急报横幅
  │   └─ 玩家点击横幅 → 打开事件详情
  │
  ├─ 随机遭遇
  │   ├─ 弹窗显示事件描述 + 2~3个选项
  │   ├─ 每个选项显示后果预览（资源+/资源-/好感度变化）
  │   └─ 玩家选择 → 执行后果 → Toast反馈
  │
  ├─ 连锁事件
  │   ├─ ChainEventSystem: 选择特定分支→10~30min后触发下一环
  │   ├─ 链结构: 2~4环，最多同时1条(剧情链不受限)
  │   ├─ 链超时: 单环>24h未响应→自动终止+30%补偿
  │   └─ 剧情链: 无环间延迟，连续触发，选择影响后续章节
  │
  └─ 离线事件
      ├─ 离线期间: 事件按规则自动处理（选择最优选项）
      ├─ 回归面板: 显示离线事件摘要
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| NPC地图拥挤渲染性能 | 大量NPC图标导致地图卡顿 | 聚合气泡机制减少渲染数量，视口外NPC不渲染 |
| 对话树数据量大 | 每个NPC多条对话分支 | 使用JSON配置驱动，NPCDialogueSystem.ts已有基础 |
| 好感度平衡性 | 好感度获取过快/过慢 | 初期设定保守值，根据测试调整倍率 |
| 离线领土变化体验差 | 玩家回归发现领土丢失 | 离线领土损失设上限（最多丢失20%），提供夺回提示 |
| 事件触发频率 | 过频打扰/过少无感 | 设置事件冷却时间，优先级排序避免同时触发多个 |
| 时代推进节奏 | 推进过快导致内容消耗/过慢导致无聊 | 根据测试调整时代目标难度，预留动态调整接口 |
| 连锁事件中断体验 | 玩家忘记/离线导致链中断 | 提供30%补偿+链状态提醒，24h超时窗口足够长 |
| 攻城战数值平衡 | 城防过高无法攻占/过低无挑战 | 城防效果上限-50%，攻方有最低胜率5%保底 |
| NPC离线行为计算量 | 大量NPC离线产出计算 | 离线计算合并为公式化产出，不逐帧模拟 |
