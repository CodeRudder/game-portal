# v5.0 天下大势 — 详细开发计划

> **核心目标**: 天下地图展示势力范围，NPC交互

## 功能清单
1. **天下地图**: 三国势力范围，城池标注，地形渲染
2. **NPC系统**: NPC巡逻+交互(对话/任务/交易)+日程系统
3. **领土系统**: 占领城池获得产出，势力范围扩张
4. **资源点**: 地图上可采集的资源点

## 验收标准
- [ ] 天下地图显示势力范围和城池
- [ ] NPC在地图上移动，可点击交互
- [ ] 占领城池增加资源产出
- [ ] pnpm run build 成功
