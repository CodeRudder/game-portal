# v7.0 群雄逐鹿 — 详细开发计划

> **核心目标**: PvP竞技场、排行榜、联盟、远征

## 功能清单
1. **PvP竞技场**: 挑战AI模拟的其他玩家阵容
2. **排行榜**: 战力/等级/关卡进度排名
3. **联盟系统**: 创建/加入联盟，联盟加成
4. **远征系统**: 派遣武将远征获取资源

## 验收标准
- [ ] 竞技场可挑战对手，战斗结果合理
- [ ] 排行榜显示正确排名
- [ ] 联盟功能可创建和加入
- [ ] 远征派遣后自动获取资源
- [ ] pnpm run build 成功
