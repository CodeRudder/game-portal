# v4.0 百家争鸣 — 详细开发计划

> **核心目标**: 3条科技路线可视化，研究科技获得全局加成

## 功能清单
1. **科技树可视化**: 3路线(军事/经济/文化)各7节点，节点间连线，互斥分支
2. **科技研究**: 消耗资源+时间，进度条实时显示，完成通知
3. **科技效果**: 全局加成立即生效，面板显示加成值
4. **融合科技**: 跨路线融合节点
5. **科技加速**: 消耗天命加速研究

## 验收标准
- [ ] 科技Tab显示3条路线，节点连线清晰
- [ ] 点击节点消耗资源开始研究
- [ ] 进度条实时更新
- [ ] 完成后加成生效
- [ ] 互斥分支正确
- [ ] pnpm run build 成功
