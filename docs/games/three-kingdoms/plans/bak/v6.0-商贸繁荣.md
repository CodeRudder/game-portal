# v6.0 商贸繁荣 — 详细开发计划

> **核心目标**: 完善经济循环，商店/贸易/离线奖励

## 功能清单
1. **商店系统**: 购买/出售物品，每日刷新，折扣商品
2. **贸易商路**: 派遣商队自动往返获利
3. **离线奖励**: 回归时领取离线收益弹窗
4. **货币体系**: 铜钱/天命/招贤榜/求贤令完整流通
5. **邮件系统**: 奖励发放+系统通知

## 验收标准
- [ ] 商店可购买物品，铜钱扣除正确
- [ ] 贸易商路派遣后自动获利
- [ ] 离线回来弹出收益领取
- [ ] pnpm run build 成功
