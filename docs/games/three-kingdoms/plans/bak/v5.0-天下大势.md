# v5.0 天下大势 — 详细开发计划

> **版本代号**: 天下大势  
> **核心目标**: 天下地图渲染15块领土、NPC巡逻交互、领土征服系统，让玩家从"城池经营"扩展到"天下争霸"  
> **核心循环**: 征服领土 → 获得领土产出 → 产出加速发展 → 征服更多领土  
> **玩家能做什么**: 查看三国势力地图、点击领土查看详情、派遣兵力征服领土、与地图NPC交互获取任务和情报、通过领土产出获得额外资源  
> **预估迭代轮数**: 5-8轮  
> **依赖版本**: v4.0 百家争鸣（科技树加成影响征服战力）

---

## 一、版本目标

### 1.1 一句话描述
天下地图将游戏从"单城经营"升级为"三国争霸"，玩家可以征服领土、与NPC交互、获得地图收益加成。

### 1.2 核心循环
```
查看天下地图 → 选择目标领土 → 派遣兵力征服 → 获得领土产出
     ↑                                          │
     └── 产出加速发展 → 征服更强领土 ←──────────┘
```

### 1.3 玩家能做什么
- 在天下地图上查看15块领土的归属、地形、产出信息
- 点击领土查看详情（驻防、产出、征服条件）
- 派遣兵力征服中立/敌方领土
- 与地图上巡逻的NPC交互（对话、任务、交易、情报）
- 通过筛选栏过滤领土（阵营/地形/收益/等级）
- 查看领土收益热力图
- 离线后回归查看领土变化

## 三、数据流设计

### 3.1 引擎→UI

```
引擎数据流:
  TerritorySystem.getTerritories()
    → [{id, name, terrain, level, owner, garrison, production, ...}]
    → 地图渲染：六边形着色+产出气泡+状态标识

  NPCSystem.getVisibleNPCs(playerPos)
    → [{id, name, role, level, position, quest, favorability, ...}]
    → NPC立绘渲染+头顶标识+交互按钮

  TerritorySystem.getProduction()
    → {food: +X/s, gold: +Y/s, troops: +Z/s}
    → 资源栏更新：领土产出合并到总产出

  NPCSystem.getQuests(npcId)
    → [{id, name, description, rewards, status}]
    → NPC任务面板渲染
```

### 3.2 UI→引擎

```
玩家操作流:
  点击领土 → engine.getTerritoryDetail(id) → 显示详情面板
  点击征服 → engine.deployTroops(territoryId, troopCount) → 胜率计算
  确认出征 → engine.conquerTerritory(territoryId, troopCount)
    → CampaignBattleSystem.simulate(attackers, defenders)
    → BattleResult → 结算面板 → 领土归属变更

  点击NPC → engine.getNPCDetail(npcId) → NPC信息弹窗
  点击对话 → engine.startNPCDialog(npcId) → 对话界面
  接受任务 → engine.acceptNPCQuest(npcId, questId) → 任务追踪
  完成任务 → engine.completeNPCQuest(questId) → 奖励发放+好感度提升

  地图筛选 → UI本地过滤 → 地图渲染刷新（不经过引擎）
  地图缩放/拖拽 → UI本地变换（不经过引擎）
```

### 3.3 关键接口

```typescript
// 领土系统
interface TerritorySystem {
  getTerritories(): Territory[];
  getTerritoryDetail(id: string): TerritoryDetail;
  canConquer(id: string): boolean; // 是否相邻且满足条件
  getConquerEstimate(id: string, troops: number): { winRate: number; estimatedLoss: number };
  conquerTerritory(id: string, troops: number): BattleResult;
  getProduction(): Record<string, number>; // 领土总产出
  upgradeTerritory(id: string): boolean;
  adjustGarrison(id: string, troops: number): boolean;
}

// NPC系统扩展
interface NPCSystem {
  getVisibleNPCs(limit: number): NPC[];
  getNPCDetail(id: string): NPCDetail;
  startDialog(npcId: string): DialogTree;
  getQuests(npcId: string): Quest[];
  acceptQuest(npcId: string, questId: string): boolean;
  completeQuest(questId: string): Reward;
  getFavorability(npcId: string): { level: number; points: number };
  addFavorability(npcId: string, points: number): void;
  getNPCShop(npcId: string): ShopItem[];
}
```

### 6.2 征服战力公式
```
进攻战力 = 派遣兵力 × (1 + 科技加成) × (1 + 武将统率加成)
防御战力 = 驻防兵力 × (1 + 地形防御加成) × (1 + 城池等级加成)
胜率 = 进攻战力 / (进攻战力 + 防御战力)
损失率 = (1 - 胜率) × 0.3
```
