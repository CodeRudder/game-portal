## 三、数据流设计

```
联盟系统
  │
  ├─ 联盟管理
  │   ├─ 创建联盟: 消耗元宝×200 + 设定名称+宣言
  │   ├─ 搜索加入: 按名称搜索 / 系统推荐活跃联盟
  │   ├─ 权限体系: 盟主(全部) / 军师(审批+公告) / 成员(基础)
  │   └─ 成员上限: 联盟等级决定(初始20人,每级+5,上限50人)
  │
  ├─ 联盟等级
  │   ├─ 经验来源: 成员日常行为+联盟任务+Boss击杀
  │   ├─ 等级效果: 成员上限↑ / 联盟加成↑ / 商店解锁↑
  │   ├─ 成员上限: 初始20人, Lv2→25人, 每级+5, 上限50人
  │   └─ 加成效果: 全成员资源产出+N% / 远征收益+N%
  │
  ├─ 联盟Boss
  │   ├─ 每日刷新Boss(难度随联盟等级递增)
  │   ├─ 全员参与讨伐(异步,每人每日3次挑战机会)
  │   ├─ 伤害统计 → 排行 → 按伤害比例分配奖励
  │   ├─ 击杀奖励: 全员获得贡献×150 + 联盟经验×200
  │   └─ 未击杀: 按个人伤害发放参与奖(贡献+联盟经验按比例)
  │
  ├─ 联盟任务
  │   ├─ 每日3个联盟任务(全员共享进度)
  │   ├─ 个人贡献: 参与即获得贡献值
  │   └─ 完成奖励: 联盟经验 + 个人公会币
  │
  └─ 联盟商店
      ├─ 货币: 贡献值(每日签到+Boss+任务+捐献获取)
      ├─ 商品: 招募令/装备箱/加速道具/武将碎片
      └─ 等级解锁: 联盟等级越高,商品越稀有

PvP赛季深化
  │
  ├─ 赛季周期: 28天(4周)
  ├─ 赛季主题: 每赛季独特视觉风格+专属称号
  ├─ 赛季专属: 头像框(按段位)+称号(王者专属)
  ├─ 结算流程:
  │   ├─ 赛季最后一天: 冻结排名
  │   ├─ 结算动画: 排名变化+段位变化+奖励展示
  │   └─ 奖励发放: 按最高段位→邮箱
  └─ 新赛季: 积分重置为当前段位最低值

活动系统
  │
  ├─ 活动管理
  │   ├─ 活动列表弹窗: 进行中/即将开始/已结束
  │   ├─ 并行上限: 最多5个活动同时进行
  │   │   ├─ 最多1个赛季活动
  │   │   ├─ 最多2个限时活动
  │   │   ├─ 1个日常活动(常驻)
  │   │   └─ 最多1个节日/联盟活动
  │   └─ 活动到期: 未使用代币→10%转铜钱
  │
  ├─ 活动任务
  │   ├─ 每日任务: 5个/天, 0点重置
  │   ├─ 挑战任务: 3个/期, 需手动完成
  │   ├─ 累积任务: 持续累积, 达成即完成
  │   └─ 奖励: 代币+积分+资源
  │
  ├─ 里程碑
  │   ├─ 积分累积 → 按顺序解锁节点
  │   ├─ 手动领取奖励(防止误操作)
  │   └─ 最终节点需手动确认(离线不自动完成)
  │
  ├─ 每日签到
  │   ├─ 7天循环: Day1铜钱→Day7求贤令(大奖)
  │   ├─ 连续加成: 3天+20%/7天+50%代币获取
  │   └─ 补签: 元宝×50/次, 每周最多2次
  │
  └─ 活动离线
      ├─ 赛季活动: 50%效率自动累积积分+代币
      ├─ 限时活动: 30%效率自动累积
      ├─ 日常活动: 100%效率自动累积
      └─ 节日活动: 50%效率自动累积

跨系统串联 [v3新增]
  │
  ├─ 联盟→声望双向
  │   ├─ Boss击杀→声望+50/次, 对战胜利→声望+100/场, 捐献→声望+5/次
  │   ├─ 赛季Top3→声望+500(一次性)
  │   └─ 声望等级→联盟离线效率(×0.03)+商店额外商品(Lv5)+科技加成+5%(Lv10)
  │
  ├─ 联盟对战→MAP领土 [v4补充详细流程]
  │   ├─ 对战胜利→领土扩张令×1(全员邮箱,≤5min到账)
  │   ├─ 扩张令→联盟征服(盟主/军师操作,消耗扩张令+联盟兵力)
  │   ├─ 征服条件: 目标领土与联盟领土相邻 + 兵力≥驻防×1.5(参考MAP PRD MAP-3)
  │   ├─ 征服结果: 胜利→领土归属联盟+地图显示联盟旗帜 | 失败→扩张令不消耗
  │   ├─ 联盟领土产出→每日0:00结算→按成员贡献比例分配
  │   ├─ 联盟领土上限15块,每周最多获得1个扩张令
  │   └─ 赛季Top3→领土产出额外+10%
  │
  ├─ 积分独立性
  │   ├─ 联盟对战积分→仅联盟排行+联盟赛季结算
  │   └─ PVP积分→仅个人段位+PVP赛季结算(完全独立)
  │
  └─ 联盟解散→归档→重组 [v4补充详细流程]
      ├─ 解散→24h倒计时→归档(名称/等级/科技/成员/贡献/战绩)
      ├─ 归档保留90天→≥5名原成员→客服申请→重组恢复
      ├─ 恢复规则: 等级-2(最低Lv1) + 科技Lv0 + 贡献按归档恢复 + 免创建费
      └─ 90天后归档永久删除
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 联盟数据同步 | 多人异步操作导致数据冲突 | 使用乐观锁+冲突自动合并+定期同步 |
| 联盟Boss平衡 | Boss过强/过弱 | 按联盟等级动态调整Boss难度 |
| 活动并行复杂度 | 5个活动同时运行增加理解和操作负担 | 活动列表清晰分类+横幅轮播优先级+代币独立 |
| 赛季结算公平性 | 赛季末冲分导致体验恶化 | 最后一天冻结排名+按最高段位结算 |
| 活动代币通胀 | 活动代币囤积导致价值稀释 | 活动到期未使用→10%转铜钱+代币独立不可跨活动 |
| 联盟领土争夺失衡 | 大联盟垄断地图领土 | 联盟领土上限15块+扩张令每周最多1个+领土衰减机制 |
| 代币元宝购买经济平衡 | 付费玩家代币获取远超免费玩家 | 每日限额(赛季500/限时300)+免费保底≈690代币/天 |
| 联盟科技递增消耗过快 | 新联盟科技升级困难 | 低级科技性价比高(500贡献)+高级需集体积累 |
| 联盟重组滥用 | 频繁解散重组刷等级 | 重组后等级-2+科技重置+90天归档期限制+重组后24h不可解散 |
| 复赛匹配不足 | 周日复赛仍无匹配联盟 | 保底奖励(贡献×25)+顺延至下周+系统自动放宽匹配范围(±20%) |
