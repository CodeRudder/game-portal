## 二、游戏引擎需求

### 新增子系统
| 子系统 | 职责 | 依赖 |
|--------|------|------|
| PrestigeSystem | 声望核心：等级管理/阈值计算/升级判定/加成注册 | — |
| PrestigeShopSystem | 声望商店：点数兑换/等级解锁/限购管理 | CurrencySystem(v8.0), PrestigeSystem |

## 三、数据流设计

```
声望系统
  │
  ├─ 声望获取
  │   ├─ 通关新关卡: 50~500声望(按难度)
  │   ├─ 三星通关: 基础×1.5
  │   ├─ 每日声望任务: 100~300声望/日(3个专属日常)
  │   ├─ 领土产出: 5~30声望/小时
  │   ├─ NPC好感度升级: 200~800/级
  │   ├─ 成就完成: 100~2000/个
  │   ├─ 远征完成: 50~200/次
  │   ├─ 活动排名: 500~5000/次
  │   └─ 离线累积: 领土产出×50%(受离线效率系数)
  │
  ├─ 声望等级
  │   ├─ 阈值公式: 1000 × N^1.8
  │   ├─ 达到阈值 → 自动升级 → 金色横幅通知
  │   ├─ 升级动画: 进度条满格闪烁→金色粒子爆发→等级跳变
  │   └─ 接近升级(≥90%) → 进度条脉冲闪烁提醒
  │
  ├─ 声望加成
  │   ├─ Lv.2/6/12/19: 领土产出 +5%/+10%/+15%/+20%
  │   ├─ Lv.7/13: 远征收益 +10%/+15%
  │   ├─ Lv.9/16: 武将经验 +10%/+15%
  │   ├─ Lv.11/18: 建筑升级速度 +5%/+10%
  │   └─ Lv.14: 科技研究速度 +5%
  │
  └─ 声望商店
      ├─ 货币: 声望点数(与声望值独立计算)
      ├─ 声望点数获取: 声望等级提升→等级×100点 / 声望任务→50点/个
      ├─ Lv.1: 扫荡令/经验书
      ├─ Lv.5: 招贤榜
      ├─ Lv.10: 求贤令/稀有装备图纸
      └─ Lv.20: 神话武将碎片

加成叠加规则
  │
  ├─ 同类加成(声望/称号/里程碑) → 加法叠加
  ├─ 转生倍率 → 最外层乘算
  └─ 最终公式: 基础值 × (1 + 声望% + 称号% + 里程碑%) × 转生倍率

离线收益计算链路
  │
  ├─ 步骤1: 基础产出(领土/远征等)
  ├─ 步骤2: × 声望加成(1 + 领土产出% + 称号资源% + 里程碑%)
  ├─ 步骤3: × 转生倍率
  ├─ 步骤4: × 离线效率阶梯(SPEC 6级: 100%→80%→60%→40%→25%→15%)
  └─ 步骤5: × (1 + Σ离线加成系数) [声望阶梯+科技+VIP，总上限+100%]
      └─ 声望阶梯: Lv.1→+3%/Lv.3→+5%/Lv.5→+10%/Lv.10→+25%

转生系统
  │
  ├─ 转生条件检查
  │   ├─ ✅ 主城等级 ≥ 30
  │   ├─ ✅ 通关"赤壁之战"全部
  │   ├─ ✅ 武将数量 ≥ 15
  │   ├─ ✅ 总战力 ≥ 50,000
  │   ├─ ✅ 完成"初露锋芒"成就链(5子成就)
  │   └─ ✅ 距上次转生 ≥ 72小时
  │
  ├─ 转生倍率
  │   ├─ 公式: 1.5 × (1 + 声望等级×0.05) × (1 + (次数-1)×0.1)
  │   ├─ 1次Lv.5: 1.88× | 1次Lv.10: 2.25×
  │   ├─ 3次Lv.15: 3.19× | 5次Lv.20: 4.50×
  │   └─ 10次Lv.30: 7.20× | 20次Lv.50: 12.00×
  │
  ├─ 倍率作用范围
  │   ├─ ✅ 全资源产出 / 武将经验 / 建筑速度 / 科技速度
  │   ├─ ✅ 关卡扫荡收益 / 离线收益
  │   └─ ❌ 商店价格 / 招募消耗 / PvP属性
  │
  ├─ 保留/重置
  │   ├─ ✅保留: 武将拥有权/升星/好感度/装备/建筑蓝图
  │   ├─ ✅保留: 关卡记录/三星记录/社交/成就/图鉴/VIP
  │   ├─ ⚠️衰减: 武将等级50%/技能70%/科技50%/天命30%
  │   └─ ❌重置: 主城/建筑等级/领土/铜钱/粮草/兵力
  │
  ├─ 转生后加速
  │   ├─ 赠送: 粮草×5000 + 铜钱×3000
  │   ├─ 建筑1~10级: 升级时间÷倍率(几乎瞬间)
  │   ├─ 已通关关卡: 可直接扫荡
  │   └─ 一键重建: 自动按上次优先级升级建筑
  │
  └─ 收益模拟器
      ├─ 预测未来X小时声望增长
      ├─ 计算声望等级提升带来的倍率增量
      ├─ 标注"推荐时机"(边际收益递减拐点)
      └─ 对比: 立即转生 vs 再等2h/6h/12h/24h

成就系统
  │
  ├─ 成就维度
  │   ├─ 养成成就: 武将收集/等级/升星/装备
  │   ├─ 战斗成就: 关卡通关/三星/远征/PvP
  │   ├─ 收集成就: 武将图鉴/装备图鉴/NPC好感度
  │   ├─ 社交成就: 好友/联盟/借将/聊天
  │   └─ 经济成就: 资源积累/商店消费/贸易利润
  │
  ├─ 成就奖励
  │   ├─ 声望值: 100~2000(按难度)
  │   ├─ 元宝: 少量(里程碑成就)
  │   ├─ 限定称号: 特殊成就专属
  │   └─ 成就点数: 累积解锁成就宝箱
  │
  └─ 转生成就链"初露锋芒"
      ├─ 崭露头角: 主城达到20级
      ├─ 猛将如云: 拥有10位武将
      ├─ 兵强马壮: 单武将战力达到5000
      ├─ 谋定后动: 完成5项科技研究
      └─ 威震四方: 占领20块领土
```
