# v8.0 商贸繁荣

> **版本主题**: 商店系统（含基础购买，承接v7.0设计） + 贸易商路 + 货币体系，构建完整经济循环
> **前置版本**: v7.0 草木皆兵（v7.0已设计商店交互框架，本版本首次完整实现）
> **预估迭代**: 4~5轮
> **功能点数**: 32个（原22个 + R1补充8个 + R2补充2个）
> **修订记录**: R1 — 补充多商店类型、货币兑换/防通胀、科技联动、转生影响、离线贸易、仓库系统、自动贸易、新手引导功能点
> **修订记录**: R2 — 修正新手引导步骤一致性、补充仓库系统功能点、修正自动贸易科技解锁条件、补充NPC好感度等级统一映射、补充离线贸易效率交叉验证、补充声望/MAP跨系统联动、补充多商店并发边界测试
> **修订记录**: R3 — 统一声望转生规则(PRS-PRD一致性)、补充定价体系验证、补充仓库容量扩展流程、补充活动代币过期结算完整流程、补充MAP领土→商贸联动操作流程

## 三、数据流设计

```
商店系统
  │
  ├─ 集市商店
  │   ├─ ShopSystem: 加载商品列表 → 渲染商品网格
  │   ├─ 点击商品 → 选择数量 → 购买确认弹窗
  │   ├─ CurrencySystem: 检查货币余额 → 按优先级扣除
  │   ├─ 购买成功 → 库存更新 → 物品入背包 → Toast反馈
  │   └─ 购买失败 → 价格变红+抖动 → 提示获取途径
  │
  ├─ 多商店类型 【R1补充】
  │   ├─ 军需处: 兵营入口 → 兵力+铜钱双货币 → 每日0点刷新
  │   ├─ 黑市: NPC随机触发 → 限时2h → 折扣-40%~-70% → 错过消失
  │   ├─ 活动商店: 活动代币消费 → 限时闭环 → 过期10%转铜钱
  │   └─ NPC交易: 以物易物+铜钱补差 → 好感度折扣 → 好感度限购🔒
  │
  ├─ 补货引擎
  │   ├─ 定时补货(每8h): 常驻补满 + 随机1~3件 + 20%折扣概率
  │   ├─ 离线补货(每8h离线): 最多累积2次 + 10%离线限定稀有
  │   └─ 手动刷新: 消耗500铜钱立即刷新随机商品
  │
贸易路线
  │
  ├─ 商路开通
  │   ├─ 检查前置条件(主城等级+前置商路) → 扣除开通费
  │   ├─ 开通后显示商路连线 → 解锁对应城市商品
  │   └─ 繁荣度初始30% → 完成贸易提升 → 产出倍率变化
  │
  ├─ 商队派遣
  │   ├─ 选择已开通商路 → 选择运输商品(按载重)
  │   ├─ 可选派遣护卫武将(降低遇盗概率50%)
  │   ├─ 商队出发 → 沿路线移动(动画) → 运输时间倒计时
  │   ├─ 到达 → 自动交易 → 货物存入仓库 → 铜钱入账
  │   ├─ 自动贸易 【R2修正】
  │   ├─ 解锁条件: 科技「富甲天下」(经济7B)研究完成 → 附加解锁自动贸易
  │   ├─ 自动选择当前行情利润最高商路
  │   ├─ 自动处理低风险事件
  │   └─ 高风险事件等待上线处理
  │
  └─ 仓库管理 【R2补充】
      ├─ 商队到达 → 货物自动存入仓库
      ├─ 仓库面板: 商品按4级分类(基础/精品/稀有/传说)
      ├─ 手动取出 → 背包获得对应物品
      ├─ 自动出售 → 按当前行情价计算
      ├─ 容量溢出 → 自动按行情价出售溢出部分 + 通知玩家
      └─ 容量扩展: 基础100 → 科技/建筑升级扩展
  │
  ├─ 商队属性提升 【R1补充】
  │   ├─ 科技「市舶司」(经济5B): 交易利润+25%, 铜钱产出+20%
  │   ├─ 科技「通宝令」(经济6B): 商店折扣+20%, 出售价格+30%
  │   ├─ 科技「富甲天下」(经济7B): 全铜钱产出+50%, 交易利润+40%
  │   ├─ 武将战力 → 护卫效果增强(降低遇盗概率+自动处理)
  │   └─ 道路等级/加速道具 → 运输时间缩短
  │
  ├─ 价格波动
  │   ├─ 每6h刷新: 实际价格 = 基础价格 × (1 ± 波动率)
  │   ├─ 连续涨跌不超过3次 → 防止极端价格
  │   └─ 科技「商业天赋」可查看下次行情趋势
  │
  └─ 贸易事件
      ├─ 每趟运输触发0~2个随机事件
      ├─ 有护卫: 山贼/暴雨自动处理(无损失)
      ├─ 无护卫: 玩家选择处理方式(缴纳/战斗/贿赂)
      └─ 离线默认保守选项(缴纳/等待)

货币体系
  │
  ├─ CurrencySystem: 统一管理8种货币余额
  ├─ 消耗优先级: 按商店类型区分
  │   ├─ 集市: 优先铜钱
  │   ├─ 军需处: 兵力+铜钱组合
  │   ├─ 限时特惠: 仅元宝
  │   └─ NPC交易: 以物易物+铜钱补差
  ├─ 货币兑换 【R1补充】
  │   ├─ 500💰铜钱 → 1📜招贤榜 (单向不可逆)
  │   ├─ 100👑天命 → 100🏷️求贤令 (单向不可逆)
  │   ├─ 100💎元宝 → 100🏷️求贤令 (单向不可逆)
  │   └─ CurrencySystem记录所有兑换流水
  ├─ 防通胀三层机制 【R1补充】
  │   ├─ L1 消耗膨胀: 升级费用×1.8/级
  │   ├─ L2 获取限制: 招贤榜堆叠上限99/活动代币限时
  │   └─ L3 重置机制: 转生时天命保留30%/声望等级归零(累计值保留)
  └─ 不足提示: 价格数字变红+抖动+获取途径引导

跨系统串联 【R1补充】
  │
  ├─ 科技→贸易联动
  │   ├─ 经济路线5B「市舶司」→ 交易利润+25%
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 商店定价平衡性 | 定价过高/过低影响经济循环 | 参考PRD定价表，初期保守设定，根据数据调整 |
| 贸易利润占比失控 | 贸易收入过强导致建筑产出无意义 | 分阶段控制：早期10%→中期30%→后期40% |
| 价格波动极端化 | 连续涨跌导致暴利或暴亏 | 限制连续同方向不超过3次，设置波动上下限 |
| 货币体系复杂度 | 8种货币管理容易出错 | CurrencySystem统一管理，所有操作通过接口 |
| 商队离线计算 | 离线期间多次运输的数据量大 | 限制离线补货次数(最多2次/16h上限) |
| 【R1新增】多商店类型状态同步 | 6种商店类型同时运行，库存/刷新/限时状态管理复杂 | ShopSystem统一调度，按商店类型独立管理状态机 |
| 【R1新增】繁荣度衰减与NPC联动 | 繁荣度衰减导致NPC消失，玩家体验突兀 | NPC消失前30分钟倒计时提醒+红名闪烁预警 |
| 【R1新增】护卫武将互斥边界 | 武将同时被护卫/远征/其他系统引用时互斥冲突 | HeroSystem统一管理武将状态锁，所有派遣走同一接口 |
| 【R1新增】货币兑换套利风险 | 玩家利用汇率差进行套利 | 所有兑换单向不可逆，兑换流水记录+异常检测 |
| 【R2新增】"商业天赋"科技未定义 | TRD-PRD引用「商业天赋」可预览行情趋势，但TEC-PRD无对应节点 | 标记为待确认，建议归属经济3「治国安邦」附加效果或融合科技「经世济民」附加效果 |
| 【R2新增】离线贸易效率双重衰减 | 商队离线受SPEC-offline效率系数衰减 + 繁荣度自然衰减-1%/天，双重衰减可能导致离线收益过低 | 设置离线贸易最低保底收益(不低于在线的15%)，双重衰减叠加后不低于10% |
| 【R2新增】仓库溢出自动出售价格波动 | 仓库溢出时自动按行情价出售，若此时行情极低则玩家损失大 | 自动出售前检查当前行情，低于基础价70%时暂停自动出售+通知玩家 |
| 【R2新增】多商店状态同步延迟 | 6种商店类型+仓库+贸易系统同时运行，状态同步可能有延迟 | 采用事件驱动架构，关键操作(货币扣除)走服务端实时校验 |
