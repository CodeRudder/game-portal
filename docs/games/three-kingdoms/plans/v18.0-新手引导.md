## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 引导步骤过多导致流失 | 新玩家在引导中流失 | 核心引导6步控制在5分钟内+加速跳过+一键完成 |
| 剧情事件打断节奏 | 玩家想直接玩不想看剧情 | 跳过按钮+二次确认+不影响奖励 |
| 引导遮罩遮挡重要信息 | 玩家看不到全局状态 | 遮罩仅聚焦当前步骤+气泡说明清晰 |
| 扩展引导触发时机不当 | 玩家还未理解基础就弹出高级引导 | 扩展引导有明确前置条件+可关闭 |
| 手机端引导操作不便 | 触控操作在引导中不直觉 | 手机端引导使用点击(非拖拽)+触控区域扩大 |
| 引导状态存储丢失 | 清除缓存后引导重新开始 | 云端同步+引导重玩入口+跳过已完成的步骤 |
| E2/E3同时触发造成混乱 | 步骤3完成后两段剧情同时触发可能困惑玩家 | 排队机制+E2先播E3后播+跳过按钮可用 |
| 步骤6书院未解锁 | 主城未达Lv3时书院不可用 | 引导期间临时解锁+引导后未满足条件则锁定+已启动研究不受影响 |
| 步骤3招贤榜不足 | 玩家无招贤榜无法完成招募 | 系统预赠招贤榜×1（从步骤6礼包提前发放） |
| 新手保护期断线计时 | 离线时间计入保护期导致保护不足 | 保护期仅计算在线时间 |
| 引导完成统计面板数据不准确 | 统计数据与实际引导过程不一致 | 引导埋点数据实时记录+统计面板从埋点数据读取 |
| 保护期军师建议切换突兀 | 保护期结束瞬间建议风格突变 | 结束Toast后首次正常建议附带过渡提示 |
| 引导完成统计面板埋点数据丢失 | 面板展示空数据影响成就感 | 兜底显示"数据收集中"+全部丢失仅展示称号装备动画 |
| 招贤榜预赠与礼包数值混淆 | 步骤3预赠1张+步骤6礼包含1张=玩家只获得1张 | 步骤6礼包实际发放招贤榜×2（1张补偿预赠+1张新增），已在play中明确 |
