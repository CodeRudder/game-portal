## 三、数据流设计

```
响应式布局系统
  │
  ├─ 断点检测
  │   ├─ 监听viewport尺寸变化（resize事件+orientationchange）
  │   ├─ 计算当前断点：Desktop-L/Desktop/Tablet-L/Tablet/Mobile-L/Mobile/Mobile-S
  │   └─ 断点变化 → 触发布局策略切换
  │
  ├─ PC/平板端策略
  │   ├─ 画布缩放: scale = min(viewW/1280, viewH/800), 上限2.0
  │   ├─ 居中偏移: offsetX = (viewW - 1280×scale) / 2
  │   ├─ 留白处理: 侧边装饰纹理+背景填充+信息面板
  │   └─ 交互方式: 鼠标+键盘为主
  │
  ├─ 手机端策略
  │   ├─ 流式布局: scale=1, canvasW=viewW, canvasH=viewH
  │   ├─ 资源栏: 375×48px固定顶部（紧凑模式）
  │   ├─ 快捷图标条: 375×36px固定顶部
  │   ├─ 场景区: 375×507px（667-48-36-76）
  │   ├─ 底部Tab: 375×76px固定底部（含安全区域）
  │   └─ 安全区域: iPhone底部34px + 状态栏44px
  │
  └─ 面板适配
      ├─ PC端: 居中弹窗(480×400px)
      ├─ 平板端: 居中弹窗(等比缩小)
      └─ 手机端: 全屏面板(375×667px)+左滑返回

触控交互系统
  │
  ├─ 手势识别
  │   ├─ 点击: touchstart+touchend < 300ms, 移动 < 10px
  │   ├─ 长按: touchstart > 500ms, 移动 < 10px → 震动反馈
  │   ├─ 拖拽: touchstart > 150ms, 移动 > 10px → 实时跟随
  │   ├─ 双指缩放: pinch手势 → 地图缩放
  │   ├─ 左滑: 水平滑动 > 80px → 删除/快速操作
  │   ├─ 下拉: 垂直下拉 > 60px → 刷新列表
  │   └─ 双击: 两次点击间隔 < 300ms → 快速升级
  │
  ├─ 触控优化
  │   ├─ 触控区域扩大: 所有可点击元素 ≥ 44×44px
  │   ├─ 防误触: 快速连续点击只响应第一次
  │   ├─ 震动反馈: 关键操作(建造/升级/招募)轻震动
  │   └─ 视觉反馈: 按下scale(0.96)+暗化+释放弹回
  │
  └─ 编队触控
      ├─ PC端: 拖拽武将到阵型格
      ├─ 手机端: 点击武将→点击空格部署
      ├─ 长按阵型格→移除武将
      └─ 点击两个格子→互换位置

手机端专属设置
  │
  ├─ 省电模式
  │   ├─ 手动开关 / 自动检测(电量<20%)
  │   ├─ 帧率降至30fps + 关闭粒子特效
  │   └─ 充电/电量恢复后自动关闭
  │
  ├─ 左手模式
  │   ├─ 镜像翻转UI布局
  │   ├─ 右侧按钮移至左侧
  │   └─ 操作区域重新分配
  │
  ├─ 屏幕常亮
  │   ├─ 游戏中保持屏幕不熄灭
  │   └─ 切至后台时恢复正常
  │
  └─ 字体大小
      ├─ 三档: 小(12px)/中(14px)/大(16px)
      ├─ 影响所有UI文字
      └─ 即时生效无需重启

手机端二级功能适配 [R3]
  │
  ├─ 商店
  │   ├─ 全屏面板+分类Tab(推荐/资源/装备/武将/限时)
  │   ├─ 商品详情Bottom Sheet 80%
  │   ├─ 铜钱购买直接执行+Toast确认
  │   └─ 元宝购买>100触发确认弹窗+3秒倒计时
  │
  ├─ 装备
  │   ├─ 全屏面板+筛选(品质/类型/装备状态)
  │   ├─ 装备详情Bottom Sheet 80%
  │   ├─ 左滑快速操作(分解/出售)
  │   ├─ 批量操作: 长按进入多选+底栏操作
  │   └─ 撤销机制: 出售/分解Toast右侧撤销按钮5秒
  │
  ├─ 背包
  │   ├─ 全屏面板+分类Tab(全部/消耗品/材料/碎片/宝箱)
  │   ├─ 物品详情Bottom Sheet 80%
  │   ├─ 消耗品使用+宝箱开启动画
  │   └─ 背包已满警告(>90%红点)+溢出至邮件
  │
  └─ 社交/任务/活动/远征
      ├─ 好友: 列表+详情Bottom Sheet+私聊(键盘上推)+赠送/借将
      ├─ 公会: 全屏面板+BOSS挑战Bottom Sheet
      ├─ 邮件: 列表+左滑删除+附件批量领取
      ├─ 任务: 分类Tab+详情Bottom Sheet+一键领取
      ├─ 活动: 列表+倒计时+详情Bottom Sheet
      └─ 远征: 队列管理+目标选择Bottom Sheet+结果领取

红点系统联动 [R4]
  │
  ├─ 红点数据源
  │   ├─ 武将Tab: 可升级武将数量
  │   ├─ 建筑子菜单: 可升级建筑数量
  │   ├─ 任务图标: 可领取任务数量
  │   ├─ 邮件图标: 未读邮件数量
  │   ├─ 远征图标: 空位/归来"!"
  │   └─ 活动图标: 进行中活动数量
  │
  ├─ 角标规则
  │   ├─ 数字角标: 具体数量(1~99)
  │   ├─ 感叹号角标: 仅提示"!"(远征归来/声望可转生)
  │   ├─ 纯红点: 仅提示有更新(好友申请/公会BOSS)
  │   └─ 超过99: 显示"99+"
  │
  └─ 联动机制
      ├─ 操作后即时更新(升级武将→角标-1)
      ├─ 资源>90%容量→军师建议推送
      ├─ 活动即将结束(<1h)→图标脉冲动画
      └─ 签到完成后图标从图标条移除

云存档多设备同步 [R4]
  │
  ├─ 同步时机
  │   ├─ 自动: 每15分钟+退出游戏时
  │   ├─ 手动: 存档槽位手动保存
  │   └─ 冲突检测: 登录时比对时间戳
  │
  ├─ 冲突解决策略
  │   ├─ 始终保留最新(默认): 以最新修改时间为准
  │   ├─ 始终保留云端: 优先云端数据
  │   └─ 始终询问: 弹窗让用户选择
  │
  └─ 数据范围
      ├─ 游戏数据: 武将/资源/建筑/科技/装备全量
      ├─ 设置数据: 画质/字体/省电/左手模式等
      └─ 加密: 端到端加密存档数据
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 手机端性能不足 | 低端手机卡顿影响体验 | 省电模式+低画质+30fps降帧+自动检测 |
| 触控精度问题 | 小屏幕上误触率高 | 触控区域扩大≥44px+防误触+操作确认 |
| 布局适配工作量大 | 25个模块全部需要手机端适配 | 优先适配核心模块(主界面/武将/战斗/建筑) |
| 安全区域兼容性 | 不同手机安全区域不同 | 使用CSS env()变量+动态检测 |
| 左手模式测试覆盖 | 左手模式可能遗漏某些UI元素 | 自动化测试+手动走查所有面板 |
| 横竖屏切换闪烁 | 切换时布局短暂错乱 | 切换时显示过渡遮罩+延迟渲染 |
| 手机端战斗性能 | 低端设备战斗粒子/帧率不达标 | 自动降级检测(每5秒)+三级降级策略 |
| 手机端弹窗键盘遮挡 | 输入弹窗键盘弹出遮挡输入框 | 面板自动上推+可视区域调整 |
| 手机端新手引导中断 | 来电/切后台导致引导中断 | 断点续传机制+引导状态持久化 |
| 手机端二级功能适配遗漏 | 商店/装备/背包等二级功能适配不完整 | R3补充6条play流程覆盖所有二级功能+实现阶段逐个验证 |
| 手机端批量操作性能 | 大量装备批量分解/出售时卡顿 | 虚拟列表+分批处理(每批50件)+进度Toast |
| 手机端键盘弹出布局错乱 | 不同输入法键盘高度不同导致面板上推量不一致 | 动态检测键盘高度(visualViewport.resize)+最小上推量保证输入框可见 |
| 手机端红点系统性能 | 大量系统同时触发红点计算导致主线程卡顿 | 红点计算异步执行+合并更新(最多100ms批量)+缓存机制 |
