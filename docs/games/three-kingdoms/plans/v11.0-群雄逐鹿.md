## 三、数据流设计

```
PvP竞技场
  │
  ├─ 匹配流程
  │   ├─ 进入竞技场 → ArenaSystem: 生成3名候选对手
  │   │   ├─ 战力范围: 自身×0.7~×1.3
  │   │   ├─ 排名范围: 自身±5~±20
  │   │   └─ 阵营分布: 尽量覆盖不同阵营
  │   ├─ 刷新对手: 免费30min / 手动500铜钱(每日10次)
  │   └─ 选择对手 → 进入战斗
  │
  ├─ 战斗流程
  │   ├─ 进攻方: 玩家手动选择编队(复用远征编队)
  │   ├─ 防守方: 对手预设防守阵容+AI策略
  │   ├─ 战斗模式: 全自动(AI控制) / 半自动(大招手动)
  │   ├─ 防守方加成: 全属性+5%(城墙加成)
  │   ├─ 10回合超时: 防守方获胜
  │   └─ 战斗结束 → 积分变化 → 段位检查 → 战报邮件
  │
  ├─ 段位与赛季
  │   ├─ 积分累积 → 自动升段(青铜V→...→王者)
  │   ├─ 每日奖励: 按当前段位发放(铜钱+竞技币+元宝)
  │   ├─ 赛季28天: 结束时按最高段位结算
  │   └─ 积分重置: 赛季结束→重置为当前段位最低值
  │
  ├─ 防守阵容
  │   ├─ 设置5阵位+选择阵型(鱼鳞/锋矢/雁行/长蛇/方圆)
  │   ├─ 选择AI策略(均衡/猛攻/坚守/智谋)
  │   ├─ 被挑战 → AI自动控制防守 → 记录日志
  │   └─ 防守日志: 胜率统计+智能建议("建议切换坚守策略")
  │
好友系统
  │
  ├─ 好友管理
  │   ├─ 搜索添加(昵称/ID) / 系统推荐(相近战力)
  │   ├─ 每日申请上限20次 / 待处理上限30条
  │   └─ 删除好友后24h冷却期
  │
  ├─ 好友互动
  │   ├─ 赠送兵力: 10次/天 → 友情点×5/次
  │   ├─ 拜访主城: 5次/天 → 铜钱×100/次
  │   ├─ 切磋: 3次/天 → 胜友情点×20/败×5
  │   └─ 借将: 3次/天 → 借用方100%经验/出借方友情点×10
  │
  └─ 友情点系统
      ├─ 每日获取上限200点(出借方)
      ├─ 友情商店: 招募令/装备箱/兵力道具/铜钱
      └─ 每日0:00重置

聊天系统
  │
  ├─ 世界频道: 全服可见/100条48h/发言10s间隔
  ├─ 公会频道: 公会成员/100条48h/发言5s间隔
  ├─ 私聊: 一对一/50条7天/发言3s间隔
  └─ 系统频道: 官方通知/30天/仅官方
```
