## 三、数据流设计

```
武将羁绊系统
  │
  ├─ 羁绊检测
  │   ├─ 编队变化 → 检测阵营分布
  │   ├─ 2同阵营 → 同乡之谊：攻击+5%
  │   ├─ 3同阵营 → 同仇敌忾：攻击+15%
  │   ├─ 6同阵营 → 众志成城：攻击+25%+防御+15%
  │   └─ 3+3不同 → 混搭协作：攻击+10%+特效加成
  │
  ├─ 羁绊预览
  │   ├─ 编队界面实时显示当前激活羁绊
  │   ├─ 属性加成预览（攻击/防御/技能伤害变化）
  │   └─ 推荐羁绊组合提示
  │
  └─ 武将故事事件
      ├─ 好感度达到阈值 → 触发专属剧情
      ├─ 历史典故演绎（桃园结义/三顾茅庐等）
      └─ 完成奖励：好感度+碎片+声望

装备强化系统
  │
  ├─ 强化流程
  │   ├─ 选择装备 → 显示成功率+费用+材料
  │   ├─ 选择保护符策略（自动/手动/不使用）
  │   ├─ 确认强化 → 概率判定 → 成功(等级+1) / 失败(等级-1)
  │   ├─ 保护符生效 → 失败不降级
  │   └─ 自动强化：循环至目标等级或材料耗尽
  │
  ├─ 装备炼制
  │   ├─ 基础炼制：3件同品质 → 随机高一品质
  │   ├─ 高级炼制：5件同品质 → 概率+10%
  │   ├─ 定向炼制：3件+定向石 → 指定部位
  │   ├─ 保底炼制：3件+保底符 → 必定≥紫色
  │   └─ 保底计数器：10次未紫→第11次必紫 / 30次未金→第31次必金
  │
  └─ 套装系统
      ├─ 穿戴装备 → 检测套装归属
      ├─ 同武将2件同套 → 激活2件套效果
      ├─ 同武将4件同套 → 激活4件套效果
      └─ 套装效果不叠加，跨武将不共享

军师推荐系统
  │
  ├─ 触发检测（每分钟tick）
  │   ├─ 资源>80%上限 → "粮仓满载，建议升级"
  │   ├─ 资源<10% → "粮草告急，建议建造农田"
  │   ├─ 建筑队列空闲 → "建造队列空闲，建议升级"
  │   ├─ 武将可升级 → "武将{名}可升级"
  │   ├─ 科技队列空闲 → "科技研究空闲"
  │   ├─ 兵力满值 → "兵力已满，建议出征"
  │   ├─ 限时NPC<2h → "{NPC名}即将离开"
  │   ├─ 新功能解锁 → "点击了解{功能名}"
  │   └─ 离线溢出>50% → "建议升级仓库"
  │
  ├─ 建议管理
  │   ├─ 按优先级排序，最多展示3条
  │   ├─ 执行建议操作 → 自动移除
  │   ├─ 手动关闭 → 同类型30min冷却
  │   └─ 每日上限15条
  │
  └─ 装备推荐
      ├─ 分析武将属性短板（攻击/防御/智力/统率）
      ├─ 筛选背包可用装备
      ├─ 排序：品质优先 > 属性增量 > 等级匹配
      └─ 每槽位推荐最多2件

传承系统深化
  │
  ├─ 转生后加速
  │   ├─ 赠送：粮草×5000 + 铜钱×3000
  │   ├─ 建筑1~10级：升级时间÷倍率（几乎瞬间）
  │   ├─ 已通关关卡：可直接扫荡
  │   └─ 一键重建：按上次优先级自动升级建筑
  │
  ├─ 次数解锁
  │   ├─ 1次转生 → 天命系统解锁
  │   ├─ 2次转生 → 专属科技路线
  │   ├─ 3次转生 → 神话武将招募池
  │   └─ 5次转生 → 跨服竞技场
  │
  └─ 收益模拟器
      ├─ 输入：当前声望等级、转生次数、资源产出速率
      ├─ 计算：立即转生倍率 vs 等待X小时后转生倍率
      ├─ 标注：边际收益递减拐点 = 推荐转生时机
      └─ 展示：倍率对比表 + 声望增长预测曲线
```

## 五、风险点

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 强化失败降级体验差 | 玩家因强化失败产生挫败感 | +5以下不降级+保护符可用+金色+12以上不降级 |
| 套装收集周期过长 | 玩家难以集齐4件套 | 2件套即有实用效果+多途径获取(关卡/商店/活动) |
| 军师建议过于频繁 | 打扰玩家正常游戏节奏 | 每日15条上限+关闭30min冷却+最多展示3条 |
| 装备背包管理复杂 | 玩家不知道该保留哪些装备 | 一键分解低品质+套装标记+推荐标记 |
| 羁绊效果数值失衡 | 6同阵营羁绊过强导致阵容单一 | 混搭协作也有加成+不同关卡需要不同策略 |
| 转生模拟器预测不准 | 玩家按推荐转生但体验不如预期 | 模拟器标注置信度+提供多个等待时间选项 |

> **数值勘误（R2修订）**:
> - 背包扩容上限：PRD(EQP-5)原文"最多扩至100格"，Play文档写"200格"。**以PRD为准：默认50格，每次+10格，最多扩至100格**。Play文档需同步修正。
> - 存档槽位：Plan原文"VIP3解锁第2/VIP7解锁第3"，PRD(SET-4)原文"3个免费+1个付费(200元宝)"。**以PRD为准：3个免费槽位+1个付费槽位(200元宝)**。Plan功能点#21描述需同步修正。
> - 初始资源赠送：Plan用"铜钱×3000"，PRD用"金币×3,000"。**统一为"铜钱×3,000"**（铜钱为本游戏标准货币名称）。
>
> **数值精确化（R4修订）**:
> - 强化成功率表：Plan原文"+1~+2 100%"不够精确，PRD(EQP-4)为+1→+2 100%/+2→+3 100%/+3→+4 95%(不降级)/+4→+5 90%(不降级)。**以PRD为准逐级精确描述**。
> - 装备分解产出：Play原文"铜钱+强化石(品质越高产出越多)"不够精确，PRD(EQP-5)为白×100/绿×300+石×1/蓝×800+石×2/紫×2000+石×3/金×5000+石×5。**以PRD为准精确数值**。
> - 声望等级阈值：PRD(PRS-1)公式为1000×N^1.8，Play文档未引用。**R4补充引用**。
> - 三星通关声望：PRD(PRS-2)为"基础声望×1.5倍仅首次三星"，Play文档未提及。**R4补充**。
> - 转生保留清单：PRD(PRS-4)含"科技路线记录100%保留"和"当前关卡进度重置至第1章"，Play文档遗漏。**R4补充**。
