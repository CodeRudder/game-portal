## 模块A: NPC巡逻与高级交互

### 1. NPC巡逻路径（Plan#1）

#### 1.1 NPC地图巡逻
- 打开主城地图 → 观察NPC沿固定路径移动（NPCPathFollower驱动）
- 操作: 等待NPC从起点移动到终点 → 到达终点后自动折返 → 循环往复
- 验证: NPC图标在地图上实时更新位置，移动动画流畅（NPCPatrolSystem调度）
- 预期: NPC按预定义路径点序列移动，到达终点后折返，无卡顿无穿墙

#### 1.2 NPC日程状态切换
- 观察NPC在不同时段的状态切换（06:00~12:00劳作/营业/研究/侦察 → 12:00~14:00休息 → 14:00~20:00继续 → 20:00~22:00守夜/夜读/夜巡 → 22:00~06:00睡眠）
- 操作: 在不同时段打开地图 → 观察NPC状态标识（劳作/休息/睡眠Zzz气泡）
- 验证: 农民劳作/士兵巡逻/商人营业/学者研究/斥候侦察，各时段状态与日程表一致
- 预期: NPC按24小时日程循环运作，不同状态影响可交互性和产出

#### 1.3 NPC地图聚合与优先级
- 主城NPC数量超过平台上限（PC端8个/手机端5个）→ 低优先级NPC自动聚合为"人群气泡"
- 操作: 提升主城等级增加NPC → 观察聚合规则（有待领取任务+100 > 新对话+80 > 稀有NPC+70 > 好感度≥4(+50) > 特殊状态(+60) > 好感度×10 > 同区域+30）
- 验证: 同区域≥3个NPC自动聚合为气泡，气泡显示职业图标群+数字角标
- 预期: PC端显示前8+名册入口徽标，手机端显示前5+底部浮动按钮

### 2. NPC刷新规则（Plan#2）

#### 2.1 固定NPC解锁
- 主城升级 → 解锁新NPC类型和数量
- 操作: 主城Lv.2（2农民）→ Lv.5（+1士兵+1商人，共5个）→ Lv.10（共10个）→ Lv.30（共23个）
- 验证: 各NPC类型出现条件正确（农民:主城≥2 / 士兵:主城≥3+兵营 / 商人:主城≥5+市集 / 学者:主城≥8+书院 / 斥候:主城≥10+领土≥5）
- 预期: NPC数量与主城等级对应表一致，建筑前置条件满足才出现

#### 2.2 稀有NPC刷新
- 满足特殊条件 → 稀有NPC限时出现
- 操作: 触发旅行商人（每日5%概率，停留4h）/ 隐世高人（好感度总和≥2000，停留2h，72h冷却）/ 朝廷使者（声望≥10，停留6h，每周1次）/ 异域商队（领土≥15，停留8h，每2周1次）/ 流浪工匠（铁匠铺≥10，停留3h，每周10%）
- 验证: 稀有NPC按概率和条件刷新，停留时间倒计时正确，冷却期不重复出现
- 预期: 稀有NPC出现时有特殊通知，停留期间可交互，超时自动消失

### 3. NPC赠送系统（Plan#3）

#### 3.1 赠送操作流程
- 与NPC对话 → 好感度≥Lv.3（300点）→ 选择「赠送」→ GiftSelectComponent打开背包 → 偏好物品标记⭐ → 选择物品 → 确认赠送
- 操作: 选择普通礼物（粮草/铜钱，+10好感，3次/NPC/日）→ 选择稀有礼物（装备/道具，+30好感，1次/NPC/日）→ 选择专属礼物（偏好物品，+50好感，需Lv.4，1次/NPC/日）
- 验证: 赠送后好感度数值正确提升，偏好物品倍率×1.5生效，每日次数上限正确
- 预期: NPCGiftSystem计算好感度（偏好×倍率），好感度提升后Toast反馈，解锁检查触发
- 异常处理: 好感度未达Lv.3时「赠送」按钮置灰并提示"需好感度达到Lv.3(300点)解锁"；物品不足时确认按钮置灰；达到每日上限时提示"今日赠送次数已用完"

#### 3.2 NPC偏好物品匹配
- 查看各NPC类型的偏好：农民(美酒/种子) / 士兵(兵书/武器) / 商人(珍珠/丝绸) / 学者(书籍/古董) / 斥候(地图/望远镜)
- 操作: 赠送偏好物品 → 对比赠送普通物品的好感度增量
- 验证: 偏好物品好感度×1.5倍率正确，非偏好物品无额外加成
- 预期: 赠送界面偏好物品有⭐标记，赠送后数值反馈显示倍率加成

### 4. NPC偏好物品（Plan#4）

#### 4.1 偏好物品识别与展示
- 打开赠送界面 → 查看背包物品列表 → 偏好物品自动标记⭐图标
- 操作: 切换不同NPC → 观察偏好物品标记变化（农民偏好≠商人偏好）
- 验证: NPC类型与偏好物品对应关系正确，标记显示清晰
- 预期: 每种NPC类型有2种偏好物品，偏好物品在赠送列表中高亮显示

### 5. NPC切磋系统（Plan#5）

#### 5.1 切磋战斗流程
- 与武将NPC对话 → 好感度≥Lv.4（600点）→ 选择「切磋」→ NPCTrainingSystem进入简化战斗（3回合制，自动战斗，**无消耗**）
- 操作: 进入切磋 → 观察战斗过程（3回合）→ 胜利（好感度+15 + 随机道具）/ 失败（好感度+5安慰奖，无惩罚）
- 验证: 切磋**不消耗兵力/资源**，每日**1次/NPC**上限正确，NPC战力随等级和类型变化
- 预期: 简化战斗快速完成，胜利奖励即时到账，失败有提升方向提示
- ⚠️ **数值权威来源**: 以PRD NPC-5正文切磋系统表为准（1次/NPC/日，无消耗）。NPC PRD末尾简化摘要"消耗兵力/3次上限"为旧版废弃内容，已与MAP PRD v1.1统一声明同步废弃
- 🔧 **BattleEngine(v3.0)接口**: NPCTrainingSystem调用BattleEngine简化模式——`battleEngine.simplifiedBattle(playerTeam, npcTeam, { maxRounds: 3, autoPlay: true, noCost: true })`。伤害计算复用CBT战斗公式（攻击力×技能倍率-防御×减免系数），但无兵力消耗、无粮草消耗、无死亡惩罚。NPC战力 = NPC等级×类型系数（士兵1.5/斥候1.3/商人0.8/学者0.7/农民0.5）× 好感度加成(1+好感度/2000)

#### 5.2 切磋战斗平衡
- 不同NPC类型战力差异验证
- 操作: 与士兵NPC切磋（高战力）→ 与农民NPC切磋（低战力）→ 对比难度
- 验证: NPC战力与类型/等级挂钩，战斗结果合理
- 预期: 士兵/斥候战力较高，农民/学者战力较低，战斗难度有梯度

### 6. NPC结盟系统（Plan#6）

#### 6.1 结盟流程
- NPC好感度达到Lv.5（满级1000点）→ 解锁「结盟」选项 → 触发专属剧情动画（30秒）→ 结盟成功
- 操作: 将NPC好感度提升至满级 → 选择结盟 → 观看动画 → 获得永久羁绊加成
- 验证: 结盟后获得对应NPC类型的羁绊技能（农民:农田+10% / 士兵:城防+15% / 商人:商店折扣5% / 学者:科技速度+8% / 斥候:隐藏宝箱+20%）
- 预期: 每类NPC最多结盟1位（5类最多5个结盟位），结盟后好感度进度条替换为"羁绊已结"金色印章
- ⚠️ **数值权威来源**: 以PRD NPC-5正文结盟系统表为准（好感度Lv.5=1000点，每类1位）。NPC PRD末尾简化摘要"亲密/3个上限"为旧版废弃内容

#### 6.2 结盟数量限制
- 操作: 尝试与同类型第2个NPC结盟 → 系统提示"该类型已结盟"
- 验证: 每类NPC最多结盟1位（5类最多5个结盟位），限制正确
- 预期: 超出限制时有明确提示，不可重复结盟

### 7. NPC离线行为（Plan#7）

#### 7.1 离线NPC自动运作
- 玩家离线 → NPC按类型自动执行行为
- 操作: 离线4小时后回归 → 检查各NPC离线产出
  - 农民: 粮草（基础产出×50%，累积上限8h）
  - 商人: 铜钱（基础产出×50%，累积上限8h）
  - 学者: 科技点加速（×0.5倍率，累积上限4h）
  - 斥候: 地图情报碎片（3条/次）
  - 士兵: 降低被袭击概率（全程）
  - 全NPC: 好感度自然增长（+0.1/小时，上限+2/次）
- 验证: 离线产出数值与公式一致，累积上限正确，好感度微量增长
- 预期: 离线收益合理，不超过在线效率的50%，有累积上限防溢出
- 异常处理: 离线超过72小时按SPEC-offline.md封顶规则处理，超出部分无产出

#### 7.2 离线特殊事件
- 离线≥4h + 好感度≥3 + 随机5%/h → NPC特殊事件触发
- 操作: 离线足够时间 → 回归检查三种离线事件
  - NPC求助（离线≥4h+好感≥3+随机）: 回归后弹出对话，需提供资源，奖励稀有道具
  - NPC送礼（离线≥6h+好感≥4+随机）: NPC赠送稀有道具
  - NPC引荐（离线≥8h+好感≥5+随机）: 解锁新NPC
- 验证: 离线事件触发条件正确，回归后弹窗展示，奖励合理
- 预期: 离线事件增加回归惊喜感，不强制要求响应

### 8. NPC离线行为摘要面板（Plan#8）

#### 8.1 回归摘要展示
- 玩家回归 → 自动弹出NPC离线行为摘要面板

  - 各NPC离线产出汇总（粮草/铜钱/科技点/情报）
  - NPC特殊事件列表（求助/送礼/引荐）
  - 好感度变化汇总
- 验证: 摘要数据与实际离线计算一致，面板可关闭可回看
- 预期: 一目了然的离线活动汇总，重要事件高亮提示

### 9.5 NPC交易系统（Plan#24，v1.2新增）

#### 9.5.1 NPC交易交互流程
- 与NPC对话 → 选择「交易」→ 打开NPC专属商品面板 → 浏览商品 → 购买
- 操作: 与商人王掌柜对话 → 选择「交易」→ 查看专属商品列表（NPC独有商品）
- 验证: 交易选项在好感度Lv.1即默认解锁（与NPC-2交互选项解锁条件表一致）
- 预期: 每个NPC有专属商品，好感度越高折扣越大（Lv.1无折扣→Lv.3折扣10%→Lv.5折扣叠加结盟折扣5%）
- 异常处理: 铜钱/元宝不足时购买按钮置灰并提示"资源不足"；商品售罄/限时商品过期时显示"已售罄"

#### 9.5.2 NPC交易折扣验证
- 操作: 对比不同好感度等级的NPC商品价格
  - Lv.1 NPC: 商品原价
  - Lv.3 NPC: 商品价格×90%（解锁赠送时同步获得10%折扣）
  - Lv.5结盟NPC（商人类型）: 商品价格×90%×95%（好感度折扣+结盟折扣叠加）
- 验证: 折扣计算与NPC-2好感度等级效果表一致，多个折扣叠加计算正确
- 预期: NPC交易折扣驱动玩家提升好感度，高好感度NPC交易更划算

## 模块B: 事件系统深化

### 10. 连锁事件（Plan#10）

#### 10.1 连锁事件触发与推进
- 完成普通事件 → 选择了带有后续事件ID的选项 → 10-30分钟后触发下一环事件
- 操作: 触发"路遇豪杰" → 选择"结伴同行"（标记后续事件）→ 等待10-30分钟 → 触发"桃园设宴" → 选择"桃园结义" → 触发"义薄云天" → 选择"肝胆相照" → 链完成
- 验证: 连锁链最多3-4层，环间延迟10-30分钟随机，选择无后续标记的选项则链终止
- 预期: 连锁事件选择不同分支触发不同后续，形成差异化体验

#### 10.2 连锁事件规则验证
- 操作: 测试连锁事件边界情况
  - 链超时: 单环等待>24h未响应 → 链自动终止，按已完成环节数给予30%阶段奖励
  - 链并发: 已有1条链进行中 → 新链不可触发
  - 链冷却: 同一链完成后7天冷却
  - 离线暂停: 玩家离线 → 事件链暂停，上线后继续计时
- 验证: 各规则正确执行，无无限递归，中断补偿合理
- 预期: 连锁事件系统稳定，边界情况处理完善

#### 10.3 剧情事件链（特殊连锁）
- 主线章节推进 → 触发剧情事件链（如第一章"黄巾之乱": 1-1黄巾初现→1-2颍川之战→1-3广宗之战→1-4张角决战）
- 操作: 完成前置关卡 → 触发剧情链 → 连续体验（无环间延迟）→ 选择影响后续章节NPC关系和可招募武将
- 验证: 剧情链不受链并发限制（可与普通链并行），必须全部完成才能推进到下一章
- 预期: 剧情链与主线深度绑定，选择有长期影响

### 11. 历史剧情事件-PC端（Plan#11）

#### 11.1 全屏沉浸式剧情体验

- 预期: 视觉效果沉浸感强，选择分支有明确反馈，完成后记录到事件日志

### 13. 事件日志面板（Plan#13）

#### 13.1 历史事件记录查看

- 预期: 回归后急报堆清晰展示离线期间的重要事件，不遗漏关键信息

## 模块C: 任务系统

### 15. 主线任务（Plan#15）

#### 15.1 主线任务推进
- 打开任务面板 → 「主线」Tab → 查看当前主线任务 → 按顺序完成 → 解锁下一个
- 操作: 接受主线1-1 → 完成目标（击败敌军/升级武将/建造建筑等）→ 领取奖励 → 自动解锁1-2 → 继续
- 验证: 前置任务完成后自动解锁下一个，每章8-12个任务，完成后解锁下一章
- 预期: 主线任务引导解锁新玩法，奖励随章节缩放（基础×(1+章节×0.15)）

#### 15.2 主线任务类型覆盖
