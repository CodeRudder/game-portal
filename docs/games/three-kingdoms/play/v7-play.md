# v7.0 草木皆兵 — 玩游戏流程

## 1. 兵种系统

### 1.1 兵种查看与编组
- 打开武将面板 → 选择武将 → 进入「兵种」Tab → 查看当前兵种(步兵/骑兵/弓兵/谋士)
- 操作: 点击「更换兵种」→ 从已解锁兵种中选择 → 确认更换 → 返回战斗编组界面
- 验证: 兵种图标/属性面板正确显示，更换后属性数值即时更新
- 预期: 四大兵种(步/骑/弓/谋)各有独立属性模板，切换兵种后攻防速属性随之变化

### 1.2 兵种特性
- 选择步兵武将 → 查看特性「坚守」(防御+15%) → 进入战斗 → 观察减伤效果
- 操作: 分别选择骑兵(冲锋/速度+)、弓兵(远程/先手)、谋士(策略/控制)进入战斗
- 验证: 各兵种特性在战斗日志中体现，属性加成数值与面板一致
- 预期: 步兵高防、骑兵高攻高速、弓兵先手远程、谋士策略控制，特性差异明显

## 2. 兵种克制

### 2.1 克制关系验证
- 步兵 vs 骑兵 → 步兵克制骑兵(伤害+20%) → 骑兵被克制(伤害-20%)
- 操作: 骑兵 vs 弓兵(骑克弓)、弓兵 vs 谋士(弓克谋)、谋士 vs 步兵(谋克步) 依次验证
- 验证: 战斗结算界面显示「克制」/「被克制」标签，伤害修正值正确
- 预期: 克制循环 步→骑→弓→谋→步 闭环，克制方伤害+20%被克制方伤害-20%

### 2.2 克制提示与反馈
- 编组界面 → 上阵被克制兵种 → 系统弹出警告「对方XX克制我方XX」
- 操作: 战斗中触发克制 → 播放克制特效 → 战斗日志标注「克制攻击」
- 验证: 编组警告正确提示，战斗中克制特效播放，日志记录克制关系
- 预期: 编组阶段有克制预警，战斗中有视觉反馈，结算有克制统计

## 3. 兵种升级

### 3.1 升级操作
- 打开兵种面板 → 选择已解锁兵种 → 点击「升级」→ 消耗兵种经验书+银两 → 升级成功
- 操作: 步兵 Lv.1 → Lv.2 → 查看属性变化(攻击+5, 防御+8, 生命+50)
- 验证: 升级消耗正确扣除，属性增量与升级表一致，等级显示更新
- 预期: 每级提升固定属性，升级材料不足时按钮置灰并提示缺口

### 3.2 升级解锁与上限
- 兵种达到 Lv.5 → 解锁「高级兵种」(如: 步兵→重甲兵) → 属性大幅提升+新特性
- 操作: 各兵种升级至 Lv.10(当前上限) → 验证满级属性 → 查看满级奖励
- 验证: Lv.5 进阶解锁正确，Lv.10 满级不再消耗材料，进阶后特性面板更新
- 预期: Lv.5 解锁进阶兵种，进阶后外观变化，满级后显示「已满级」标识

## 4. 阵型系统

### 4.1 阵型选择与部署
- 进入战斗编组界面 → 点击「阵型」按钮 → 展开阵型列表(锋矢/鹤翼/鱼鳞/长蛇/方圆)
- 操作: 选择「锋矢阵」→ 武将自动按阵型站位 → 前排步兵/中排骑兵/后排弓兵谋士
- 验证: 阵型站位正确分配，阵型加成属性(锋矢:攻击+10%)在面板显示
- 预期: 5种阵型各有站位模板和属性加成，切换阵型后武将位置自动调整

### 4.2 阵型克制与效果
- 锋矢阵(攻击型) vs 鱼鳞阵(防御型) → 锋矢阵突破加成 → 鱼鳞阵减伤削弱
- 操作: 依次测试5种阵型两两对战 → 记录阵型克制效果 → 查看战斗加成日志
- 验证: 阵型克制关系生效，战斗日志显示阵型加成/削弱数值
- 预期: 阵型间存在攻防速克制关系，合理选择阵型可弥补兵种劣势

## 5. 交叉验证

### 5.1 兵种+克制+阵型联动
- 编组: 前排步兵( Lv.8 重甲兵) + 中排骑兵( Lv.6) + 后排弓兵( Lv.7) + 谋士( Lv.5)
- 操作: 选择鹤翼阵 → 对战敌方锋矢阵 → 敌方骑兵克制我方弓兵 → 鹤翼阵侧翼保护触发
- 验证: 兵种等级属性、克制修正、阵型加成三者叠加计算正确
- 预期: 最终伤害 = 基础属性(含等级) × 克制系数 × 阵型加成，计算结果与日志一致

### 5.2 全流程闭环
- 兵种升级 → 获得新兵种 → 编组上阵 → 选择阵型 → 进入战斗 → 克制生效 → 结算
- 操作: 完成一次完整的「升级→编组→战斗」流程 → 检查每个环节的数据传递
- 验证: 升级属性正确传递到战斗，阵型站位与编组一致，克制关系在结算中体现
- 预期: 兵种系统、克制系统、升级系统、阵型系统四大模块数据流通无断点
