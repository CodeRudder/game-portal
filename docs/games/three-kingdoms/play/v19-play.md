# v19.0 天下一统(上) — 玩游戏流程

## 1. 统一战争(设置系统)

### 1.1 基础设置
- 操作: 打开设置面板 → 切换语言(简中/繁中/EN/日本語) → 确认重启弹窗 → 重启后验证语言生效 → 设置时区(UTC+8) → 验证日常重置时间跟随变更 → 通知总开关关闭 → 验证5项子通知全部静默
- 验证: 语言切换需重启确认，时区即时生效影响日常重置/活动时间，通知总开关=0时屏蔽所有子项
- 预期: 4种语言完整切换无遗漏文本，时区变更后活动倒计时正确，通知总开关与子项联动一致

### 1.2 音效控制
- 操作: 主音量设80% → BGM设60% → 验证实际输出=48% → 音效总开关关闭 → 验证等同于主音量=0% → 切至后台 → 验证BGM渐弱1s至静音 → 模拟来电 → 验证立即静音 → 恢复后渐入0.5s → 低电量(<20%) → 验证BGM自动降50%
- 验证: 实际输出=分类音量×主音量公式正确；4通道(BGM/音效/语音/战斗)独立控制；特殊场景(后台/来电/低电量)自动处理
- 预期: 音量计算精确，开关状态与实际播放一致，特殊场景无缝切换无爆音

### 1.3 画质管理
- 操作: 首次启动 → 验证自动检测(CPU≥8核+内存≥8GB→高) → 手动切低画质 → 验证粒子关+阴影关+水墨关+30fps → 切高画质 → 验证全部开启+60fps → 展开高级选项 → 独立关闭水墨晕染 → 验证即时生效+水墨过渡0.6s
- 验证: 4档预设(低/中/高/自动)参数正确；5项高级选项独立可控；切换时水墨晕染过渡无闪烁
- 预期: 自动检测推荐合理，手动覆盖即时生效，低端设备隐藏高级选项，过渡遮罩消除闪烁

## 2. 天下系统(账号与存档)

### 2.1 账号绑定
- 操作: 游客登录 → 绑定手机号 → 验证短信验证码流程 → 确认绑定 → 检查元宝×50奖励到账 → 继续绑定邮箱 → 验证多方式绑定共存 → 绑定微信 → 验证第三方授权跳转正常
- 验证: 手机号/邮箱/第三方(微信/QQ/Apple)三种方式均可绑定；首次绑定奖励元宝×50仅发放一次；游客账号30天自动清除倒计时
- 预期: 绑定流程顺畅安全，奖励即时到账，多方式绑定互不冲突

### 2.2 云存档同步
- 操作: 设置同步频率"退出时" → 退出游戏 → 验证自动上传 → 另一台设备登录 → 验证云端下载 → 设置冲突策略"始终询问" → 两台设备分别存档 → 触发冲突弹窗 → 选择保留云端版本 → 验证数据正确
- 验证: 自动同步按频率执行(退出时/每小时/仅手动)；端到端加密存储；冲突策略(最新/云端/询问)三选一正确执行
- 预期: 同步过程有进度指示器，冲突解决不丢数据，WiFi仅同步开关在移动网络下正确阻止

### 2.3 存档槽位管理
- 操作: 查看存档列表 → 验证3免费+1付费(灰锁)槽位 → 手动保存到槽位1 → 等待15分钟 → 验证自动存档更新 → 读取槽位1 → 验证全量数据恢复(资源/建筑/武将/科技) → 删除槽位1存档 → 确认二次弹窗
- 验证: 3免费槽位可用，第4槽位需200元宝解锁；自动存档每15分钟覆盖；存档大小2-5MB含全量数据
- 预期: 存档读写完整无数据丢失，槽位状态清晰(空/已用/锁定)，自动存档不中断游戏

## 3. 势力终结(动画规范)

- 操作: 打开建筑面板 → 验证卷轴展开300ms ease-out → 关闭面板 → 验证收起200ms ease-in → 切换Tab → 验证平滑200ms ease-in-out → 进入战斗 → 验证页面过渡500ms水墨晕染 → 触发弹窗 → 验证spring弹性250ms
- 验证: 面板展开300ms/关闭200ms/Tab切换200ms/页面过渡500ms/弹窗250ms/场景切换800ms四角墨色遮罩
- 预期: 所有过渡动画时长与缓动曲线符合规范，节奏统一不突兀，水墨风格贯穿始终

### 3.2 状态与反馈动画
- 操作: 悬停按钮 → 验证150ms亮度+10%阴影扩大 → 按下 → 验证80ms scale(0.96)暗化 → 释放 → 验证120ms弹回 → 领取资源 → 验证+N绿色上飘800ms → 升级建筑 → 验证金色粒子爆发1000ms → 战斗结算 → 验证星星逐颗亮起间隔300ms
- 验证: 按钮(悬停150ms/按下80ms/释放120ms)、开关(虎符旋转180° 200ms)、卡片选中(边框高亮200ms)全部正确；飘字/光效/Toast/结算反馈动画到位
- 预期: 状态变化即时可感知，反馈动画增强操作确认感，全系统统一使用AnimationController无遗漏

## 4. 统一进度(全局设置规则)

### 4.1 设置持久化
- 操作: 修改BGM音量至70% → 验证localStorage立即保存 → 退出游戏 → 重新进入 → 验证BGM音量仍为70% → 触发云同步 → 另一设备读取 → 验证设置同步一致 → 修改时区 → 验证冲突取最新修改时间
- 验证: 修改后立即保存本地+随云存档同步+多设备冲突以最新时间为准
- 预期: 设置不丢失，多设备间设置一致，冲突解决自动且透明

### 4.2 恢复默认与账号删除
- 操作: 音效设置区点击"恢复默认" → 验证仅重置音效分类(主音量100%/BGM60%/音效70%/语音80%) → 基础设置不受影响 → 账号设置点击"删除账号" → 输入"确认删除" → 二次确认弹窗 → 验证7天冷静期倒计时显示 → 冷静期内点击"撤销删除" → 验证恢复正常
- 验证: 恢复默认仅重置当前分类；账号删除需输入确认文字+二次确认+7天冷静期；冷静期内可撤销
- 预期: 恢复默认范围精确不误伤，删除流程多重防护防误操作，冷静期撤销即时生效
