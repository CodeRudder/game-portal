# v19.0 天下一统(上) — 玩游戏流程

> 核心循环: 统一系统驱动全局打磨进程→设置系统完善(基础/音效/画面/账号)提供统一基石→动画规范统一全系统节奏→存档管理保障统一进度→品质感提升为天下一统铺路

---

## 1. 统一系统

### 1.1 基础设置统一
- 操作: 打开设置面板 → 切换语言(简中/繁中/EN/日本語) → 确认重启弹窗→重启验证语言生效 → 设置时区(UTC+8) → 验证日常重置/活动倒计时即时变更 → 通知总开关关闭 → 验证建筑完成/远征归来/活动提醒/好友消息/联盟通知5项全部静默
- 验证: 4种语言切换无遗漏文本 | 时区变更即时影响日常重置时间 | 通知总开关=OFF时屏蔽全部子项
- 预期: 基础设置为统一进程提供语言/时区/通知基础设施，全平台体验一致

### 1.2 音效系统统一
- 操作: 主音量设80% → BGM设60% → 验证实际输出=48%(分类×主音量) → 音效总开关关闭→验证等同于主音量=0% → 切后台→BGM渐弱1s至静音 → 模拟来电→立即静音→恢复渐入0.5s → 低电量(<20%)→BGM自动降50% → 首次启动→BGM延迟3s播放
- 验证: 实际输出=分类音量×主音量公式精确 | 4通道(BGM/音效/语音/战斗)独立可控 | 特殊场景(后台/来电/低电量/首次启动)自动处理
- 预期: 音效系统为统一进程营造沉浸式音频体验，4通道精细控制满足不同玩家偏好

### 1.3 画质管理统一
- 操作: 首次启动→自动检测(CPU≥8核+内存≥8GB→高) → 手动切低画质→验证粒子关+阴影关+水墨关+30fps → 切高画质→全部开启+60fps → 展开高级选项→独立关闭水墨晕染→验证即时生效+水墨过渡0.6s → 低端设备验证高级选项隐藏
- 验证: 4档预设(低/中/高/自动)参数正确 | 5项高级选项独立可控 | 切换时水墨晕染过渡无闪烁
- 预期: 画质系统为统一进程提供视觉品质保障，自动检测推荐合理，手动覆盖即时生效

## 2. 统一条件

### 2.1 账号绑定条件
- 操作: 游客登录→绑定手机号→短信验证码流程→确认绑定→检查元宝×50到账 → 继续绑定邮箱→验证多方式共存 → 绑定微信→验证第三方授权跳转 → 验证游客账号30天自动清除倒计时显示
- 验证: 手机号/邮箱/第三方(微信/QQ/Apple)三种方式均可绑定 | 首次绑定奖励元宝×50仅发一次 | 游客未绑定30天清除提示可见
- 预期: 账号绑定是统一条件第一步，确保玩家身份持久化，奖励激励完成绑定

### 2.2 云存档同步条件
- 操作: 设置同步频率"退出时" → 退出游戏→验证自动上传(进度指示器) → 另一台设备登录→云端下载 → 设置冲突策略"始终询问" → 两台设备分别存档→触发冲突弹窗→选择保留云端版本→验证数据正确 → 开启WiFi仅同步→移动网络下验证同步阻止
- 验证: 自动同步按频率执行(退出时/每小时/仅手动) | 端到端加密存储 | 冲突策略(最新/云端/询问)三选一正确 | WiFi仅同步在移动网络阻止
- 预期: 云存档确保统一进度跨设备不丢失，冲突解决策略保障数据完整性

### 2.3 存档槽位条件
- 操作: 查看存档列表→验证3免费+1付费(灰锁)槽位 → 手动保存到槽位1 → 等待15分钟→验证自动存档覆盖 → 读取槽位1→验证全量数据恢复(资源/建筑/武将/科技) → 删除槽位1→确认二次弹窗 → 第4槽位支付200元宝解锁
- 验证: 3免费槽位可用，第4槽位需200元宝 | 自动存档每15分钟覆盖 | 存档2-5MB含全量数据
- 预期: 存档槽位为统一进程提供多节点存档能力，玩家可安全回溯任意进度

## 3. 终极挑战

### 3.1 动画规范统一落地
- 操作: 打开建筑面板→验证卷轴展开300ms ease-out → 关闭面板→收起200ms ease-in → 切Tab→平滑200ms ease-in-out → 页面过渡→500ms水墨晕染 → 弹窗→spring弹性250ms → 场景切换→800ms四角墨色遮罩中央扩散
- 验证: 面板展开300ms/关闭200ms/Tab200ms/页面500ms/弹窗250ms/场景800ms全部符合规范 | 水墨风格贯穿始终
- 预期: 过渡动画统一全系统节奏，玩家感受不到模块边界，体验如一卷水墨长卷

### 3.2 状态与反馈动画挑战
- 操作: 悬停按钮→150ms亮度+10%阴影扩大 → 按下→80ms scale(0.96)暗化 → 释放→120ms弹回 → 切换开关→虎符旋转180° 200ms → 卡片选中→边框高亮200ms → 领取资源→+N绿色上飘800ms → 升级建筑→金色粒子爆发1000ms → 战斗结算→星星逐颗亮起间隔300ms
- 验证: 按钮(悬停150ms/按下80ms/释放120ms)/开关(虎符旋转)/卡片(边框高亮)全部正确 | 飘字/光效/Toast/结算反馈到位
- 预期: 全系统统一使用AnimationController，状态变化即时可感知，反馈增强操作确认感

## 4. 天下大势结局

### 4.1 设置持久化与全局统一
- 操作: 修改BGM音量至70%→验证localStorage立即保存 → 退出游戏→重新进入→BGM仍为70% → 触发云同步→另一设备读取→设置同步一致 → 修改时区→冲突取最新修改时间 → 音效区点击"恢复默认"→验证仅重置音效分类(主音量100%/BGM60%/音效70%/语音80%)→基础设置不受影响
- 验证: 修改后立即保存本地+随云存档同步+多设备冲突以最新时间为准 | 恢复默认仅重置当前分类
- 预期: 设置持久化确保统一成果不丢失，恢复默认精确不误伤，全局设置统一管理

### 4.2 账号删除与冷静期结局
- 操作: 账号设置点击"删除账号" → 输入"确认删除" → 二次确认弹窗 → 验证7天冷静期倒计时显示 → 冷静期内点击"撤销删除"→恢复正常 → 冷静期结束→验证永久删除不可恢复
- 验证: 删除需输入确认文字+二次确认+7天冷静期 | 冷静期内可撤销 | 冷静期后永久删除
- 预期: 删除流程多重防护防误操作，冷静期给予玩家反悔机会，体现对玩家数据尊重

## 5. 交叉验证

### 5.1 音量→画质→存档联动
- 操作: 设置BGM 50%+主音量80% → 切换低画质→验证音量设置不受影响 → 退出触发云存档 → 另一设备读取→验证音量+画质均同步正确 → 恢复默认音效→验证画质不变
- 预期: 各设置分类独立持久化互不干扰，云同步覆盖全部设置项，恢复默认仅影响目标分类

### 5.2 存档→云同步→冲突解决→多设备闭环
- 操作: 设备A手动存档+语言改English → 设备B手动存档+语言改简中 → 设备A触发云同步→冲突弹窗→选择"始终最新" → 验证保留最后修改的语言 → 两设备存档槽位数据均完整
- 预期: 冲突策略正确执行，存档数据不丢失，语言等设置项冲突以时间为准解决

### 5.3 动画→设置→画质联动
- 操作: 切换低画质→打开面板→验证过渡动画仍300ms(时长不受画质影响) → 关闭粒子特效→验证反馈动画(飘字/Toast)仍正常 → 切高画质→验证粒子恢复+动画节奏不变
- 预期: 动画时长与画质档位解耦，低画质关闭粒子不影响UI动画，高画质恢复后节奏一致

### 5.4 账号→绑定→存档→删除全流程
- 操作: 游客登录→绑定手机号→领取元宝×50 → 手动存档→触发云同步 → 解绑设备→重新绑定→验证存档可恢复 → 发起账号删除→冷静期内撤销 → 验证存档+设置+绑定状态完整保留
- 预期: 绑定-存档-同步-解绑-重绑-删除-撤销全链路无数据丢失，每个环节状态一致可追溯
