# v17.0 竖屏适配 — 玩游戏流程

> 核心循环: 响应式断点检测自动切换布局策略→手机端流式布局适配竖屏→触控手势优化提升操作精度→底部Tab导航+全屏面板重构移动端UI→多设备交叉验证确保全平台体验一致

---

## 1. 竖屏适配

### 1.1 7级断点检测与布局切换
- 操作: 启动游戏 → ResponsiveLayoutManager监听viewport尺寸 → 计算当前断点(Desktop-L ≥1920 / Desktop ≥1280 / Tablet-L ≥1024 / Tablet ≥768 / Mobile-L ≥428 / Mobile ≥375 / Mobile-S <375) → 断点变化时触发策略切换 → PC/平板端: 画布等比缩放 scale=min(viewW/1280, viewH/800)，上限2.0，居中偏移 offsetX=(viewW-1280×scale)/2 → 手机端: 流式布局 scale=1，canvasW=viewW，canvasH=viewH → 留白区域: 居中+侧边装饰纹理+背景填充+信息面板
- 验证: 7级断点切换时布局无闪烁 | PC/平板端画布等比缩放正确，4K上限scale≤2.0 | 手机端流式布局无水平滚动条 | 留白区域美观无空白
- 预期: 全平台无缝适配，玩家在任何设备上启动即获得最佳布局，无需手动调整

### 1.2 横竖屏切换与安全区域
- 操作: 旋转设备 → orientationchange事件触发 → 重新计算断点与布局策略 → 切换过程显示过渡遮罩(300ms渐变) → 延迟渲染新布局避免闪烁 → iPhone安全区域: 底部34px + 状态栏44px → 使用CSS env(safe-area-inset-*)动态检测 → 不同机型安全区域自适应
- 验证: 横竖屏切换时布局过渡平滑无错乱 | 安全区域在不同机型(iPhone SE/14/刘海屏/药丸屏)正确处理 | 切换过程无元素重叠或闪烁
- 预期: 横竖屏自由切换体验流畅，安全区域兼容所有主流机型

## 2. 响应式布局

### 2.1 手机端画布分区
- 操作: 手机端竖屏模式 → 画布基准375×667px → 资源栏固定顶部48px(粮草/铜钱/元宝紧凑横排) → 快捷图标条固定顶部36px(建筑/武将/科技/背包) → 场景区375×507px(667-48-36-76)承载主界面/地图/战斗 → 底部Tab栏固定底部76px(56px Tab+20px安全区域) → Tab项: 主城/武将/战役/背包/更多 → 各区域独立渲染互不干扰
- 验证: 各区域尺寸在375×667基准下精确匹配 | 固定区域(资源栏/Tab栏)滚动时不位移 | 场景区内容完整展示无裁切 | 375px以下(Mobile-S)等比缩小无溢出
- 预期: 手机端竖屏空间利用率最大化，核心信息始终可见，操作区域单手可达

### 2.2 面板全屏适配与Bottom Sheet
- 操作: 手机端点击面板入口(如武将列表/建筑详情) → 面板以全屏模式展示(375×667px) → 顶部显示面板标题+关闭按钮(≥44px触控区域) → 内容区可垂直滚动 → 左滑(>80px)返回上一级 → 筛选/选择操作使用Bottom Sheet从底部弹出 → Sheet支持半屏/全屏两档 → 点击Sheet外部区域收起
- 验证: 全屏面板左滑返回响应灵敏 | Bottom Sheet弹出/收起动画流畅(300ms) | 面板内容滚动无卡顿 | Sheet半屏/全屏切换正确
- 预期: 手机端面板交互符合移动端习惯，全屏展示信息密度最大化，左滑返回操作直觉化

## 3. 触控优化

### 3.1 七种手势识别与反馈
- 操作: TouchInputSystem识别7种手势 → 点击(touchstart+touchend <300ms，移动<10px) → 长按(touchstart >500ms，移动<10px，触发震动反馈) → 拖拽(touchstart >150ms，移动>10px，实时跟随) → 双指缩放(pinch手势→地图缩放) → 左滑(水平>80px→删除/快速操作) → 下拉(垂直>60px→刷新列表) → 双击(两次点击间隔<300ms→快速升级) → 所有可点击元素触控区域≥44×44px → 防误触: 快速连续点击只响应第一次 → 视觉反馈: 按下scale(0.96)+暗化+释放弹回
- 验证: 7种手势识别准确率≥99% | 触控区域扩大后小按钮可轻松点击 | 防误触机制有效阻止连击 | 震动反馈仅在关键操作触发 | 视觉反馈动画流畅无延迟
- 预期: 触控操作精准流畅，手势识别自然直觉，反馈及时增强操作信心

### 3.2 武将编队触控与桌面端交互
- 操作: 手机端编队 → 点击武将卡片→选中态高亮→点击阵型空格→武将部署到格→长按阵型格→移除武将→点击两个已 occupied格→互换位置 → 桌面端编队 → 拖拽武将到阵型格 → 桌面端7种交互: 点击/右键菜单/悬停提示/拖拽/滚轮缩放/长按/Shift+点击批量 → 全局快捷键: T地图/H武将/K科技/C关卡/Space暂停
- 验证: 手机端点击部署流程顺畅无卡顿 | 长按移除需震动确认 | 互换位置逻辑正确 | 桌面端拖拽精确到格 | 快捷键响应无冲突
- 预期: 编队操作在手机端和桌面端各有最优交互方式，功能完整度一致

## 4. 手机端UI

### 4.1 底部Tab导航与快捷图标
- 操作: 底部Tab栏固定底部 → 5个Tab: 主城/武将/战役/背包/更多 → 选中态: 图标高亮+文字显示+底部指示条 → 未选中态: 图标灰度+文字隐藏 → 点击Tab切换对应全屏面板 → 切换动画: 淡入淡出(200ms) → 快捷图标条固定顶部 → 图标尺寸36×36px，间距8px → 点击图标快速跳转对应功能面板 → 面包屑导航在深层页面显示路径(主城>建筑>铁匠铺>升级)
- 验证: Tab切换面板内容正确无残留 | 导航直觉单手可操作 | 快捷图标跳转准确 | 面包屑路径正确且可点击回退 | 深层页面(3级以上)可达性良好
- 预期: 底部Tab导航符合移动端标准交互模式，单手拇指可达所有核心功能

### 4.2 移动端专属设置
- 操作: 设置面板(更多Tab→设置) → 省电模式: 手动开关/自动检测(电量<20%) → 开启后帧率降至30fps+关闭粒子特效 → 充电/电量恢复自动关闭 → 左手模式: 镜像翻转UI布局+右侧按钮移至左侧 → 屏幕常亮: 游戏中保持屏幕不熄灭，切后台恢复正常 → 字体大小: 小(12px)/中(14px)/大(16px)三档切换即时生效 → 所有设置即时保存至本地存储
- 验证: 省电模式帧率稳定30fps且功能不受影响 | 左手模式所有UI元素正确镜像无遗漏 | 屏幕常亮游戏中生效切后台恢复 | 字体切换即时生效无需重启 | 设置持久化重启后保留
- 预期: 移动端专属设置覆盖手机用户核心需求，省电模式保障低端设备可玩性

- 预期: 从Desktop到Mobile全断点链路布局正确切换，手机端完整游戏流程操作流畅，触控反馈贯穿始终
