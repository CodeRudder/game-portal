- 预期: 面板数据与实际游戏状态一致，势力占比总和100%

### 1.1 急报横幅
- 全局事件触发时 → 顶部弹出急报横幅 → 横幅显示事件摘要 → 点击横幅跳转详情
- 验证: 横幅3秒后自动收起、点击可展开、多条急报排队轮播
- 预期: 事件触发后1秒内出现横幅，不遮挡核心操作区
- **通知优先级**: 限时机遇(倒计时<10min) > 天灾人祸 > 剧情事件 > 随机遭遇 > NPC事件 > 里程碑事件
- **排队上限**: 最多缓存5条急报，超出按优先级丢弃低优先级事件

### 1.2 事件类型筛选

### 3.1 领土攻占（征服流程）
- 世界地图选择敌方领土 → 查看胜率预估 → 派遣武将攻城 → 战斗结算 → 领土归属变化
- 验证: 胜率预估与实际结果偏差≤10%、攻城奖励(资源+道具)正确发放、领土颜色变更
- 预期: 攻占后势力领土数+1，防守方领土数-1，大势面板数据同步更新
- **征服条件**: 目标领土必须与己方领土相邻；可用兵力 ≥ 目标驻防兵力 × 1.5
- **征服流程**: 点击领土→查看详情(驻防兵力/产出/地形)→点击「征服」→兵力部署滑块→确认出征→战斗/自动结算
- **失败惩罚**: 损失30%出征兵力，领土不变

### 3.1.1 胜率预估
- 选择敌方领土 → 系统显示胜率预估 → 根据胜率颜色判断风险等级
- **胜率公式**: `胜率 = min(95%, max(5%, (我方战力 / 敌方战力) × 50% + 地形修正))`
- **胜率颜色**: >80%翠绿(稳操胜券) / 60%~80%金色(有风险) / 40%~60%琥珀橙(势均力敌) / <40%赤红(建议强化)
- **攻方战力**: 出战武将战力 + 兵种克制
- **守方战力**: 驻防武将战力 + 城防值 + 地形加成
- 验证: 胜率预估值与实际战斗结果偏差≤10%

### 3.1.2 攻城战（城池类领土专用）
- 选择城池类敌方领土 → 满足攻城条件 → 进入攻城战
- **攻城条件**: 领土相邻 + 兵力×100 + 粮草×500 + 每日次数上限3次
- **城防计算**: 城防值 = 基础城防(1000) × 城市等级 × (1 + 城防科技加成)（⚠️以MAP PRD v1.1统一声明为准）
- **城防效果**: 每点城防值降低攻方伤害0.02%（上限-50%）
- **城防恢复**: 每小时恢复城防上限的5%，战斗中不恢复
- **出征兵力门槛**: ≥ 城池驻防兵力 × 2.0
- **粮草消耗**: 粮草×500（⚠️以MAP PRD v1.1统一声明为准，旧距离公式已废弃）
- **攻城时间**: 基础30分钟 + 城防值/100（分钟）
- **占领条件**: 城防值归零即占领（⚠️以MAP PRD v1.1统一声明为准，旧三条件已简化）
- **占领冷却**: 攻占后24小时内不可被反攻（新手保护期）
- **占领产出**: 初始50%，24小时后100%
- **攻城奖励**:
  - 首次攻占: 元宝×100 + 声望+50 + 专属称号
  - 重复攻占: 铜钱×5000 + 资源产出×2（24h）
  - 攻城失败: 损失30%出征兵力，粮草不返还（⚠️以MAP PRD v1.1统一声明为准）
- 验证: 城防值正确计算、攻城消耗正确扣除、占领冷却生效、奖励正确发放

### 3.2 驻防机制
- 选择己方领土 → 点击"驻防" → 从武将列表选择驻防武将 → 确认派遣 → 领土防御值提升
- 验证: 驻防武将不可同时出战、防御加成数值正确、产出加成生效
- 预期: 派驻后领土防御面板显示加成后数值，驻防武将头像出现在领土上
- **驻防兵力上限**: 领土等级 × 100
- **驻防效果**: 每点兵力增加防守方0.1%防御
- **驻防消耗**: 从总兵力池中分配，可随时调回
- **自动驻防**: 新占领领土自动分配50%兵力上限
- **互斥规则**: 驻防武将与出战编队互斥，不可同时使用
- 验证: 武将驻防后编队列表中该武将标记为"驻防中"、编队时不可选中

### 3.2.1 领土等级
- 领土占领后随时间自动升级，也可通过投资加速
- **等级体系**:
  - Lv.1: 初始占领，产出×1.0，驻防上限100
  - Lv.5: 占领24小时，产出×1.2，驻防上限500
  - Lv.10: 占领72小时+投资，产出×1.5，驻防上限1000
  - Lv.15: 占领7天+重金投资，产出×2.0，驻防上限1500
- 验证: 领土等级正确随时间/投资提升、产出倍率和驻防上限正确生效

### 3.2.2 领土产出计算
- 领土总产出 = 基础产出 × 地形加成 × 阵营加成 × 科技加成 × 声望加成 × 地标加成 × 驻防加成 × 时代加成（时代加成见7.11节，初始×1.0）
- **地形加成**: 平原粮草+30% / 山地粮草-20%防守+20% / 水域铜钱+25% / 城池全资源+10% / 森林兵力+15% / 关隘防守+40%
- **阵营加成**: 己方领土全产出+10%
- **特殊地标**: 洛阳全资源+50% / 长安科技点+30% / 建业铜钱+30%
- **产出气泡规则**: 己方领土始终显示 / 非己方不显示(悬停Tooltip显示潜在收益) / 缩放<60%隐藏气泡
- 验证: 各项加成正确叠加、产出气泡显示规则正确

### 3.3 离线领土变化
- 离线一段时间后重新上线 → 弹出离线变化报告 → 查看领土得失详情 → 确认关闭
- 验证: 离线期间领土被攻占记录完整、新获得的领土显示来源
- 预期: 离线变化可视化标注在地图上，红色=失、绿色=得
- **离线损失上限**: 最多丢失20%领土
- **视觉标记**: 新占领=金色脉冲光环+"新"标签 / 失去=红色脉冲光环+"失"标签 / 标签首次点击后消失
- **产出变化**: 产出气泡闪烁+↑↓箭头，10秒后恢复
- 验证: 离线领土损失不超过20%上限、视觉标记正确显示和消失

## 7. 事件系统

### 7.1 事件触发
- EventTriggerSystem检测触发条件(时间/概率/条件) → 满足条件 → 触发事件 → 显示急报横幅
- **触发规则**:
  - 随机遭遇: 30~60min间隔，每日上限24次，进入游戏10min后首次触发
  - NPC事件: 60~120min间隔，每日上限12次，进入游戏20min后首次触发
  - 天灾人祸: 120~200min间隔，每日上限8次，进入游戏30min后首次触发
  - 限时机遇: 固定时段(10:00/14:00/20:00)，每日3次，持续30~60min
- **触发概率公式**: `P(触发) = 0.03 × (1 + 玩家等级 × 0.02) × max(0.5, 1 - 距上次事件分钟数/120)`
- **冷却机制**: 同子类型60min冷却 / 同事件4h冷却 / 新手前30min仅正面事件 / 连续2次负面后下次必正面
- 验证: 触发间隔在范围内、日上限生效、冷却机制正确、新手保护生效

### 7.2 随机遭遇弹窗
- 事件触发 → 弹窗显示事件描述+2~3个选项 → 每个选项显示后果预览 → 玩家选择 → 执行后果 → Toast反馈
- **选项后果标注**: 🟢绿色=正面 / 🔴红色=负面风险 / 🟡黄色=中性
- **奖励缩放**: `资源奖励 = 基础值 × (1 + 主城等级 × 0.1)`
- **损失减免**: `资源损失 = 基础值 × max(0.3, 1 - 防御建筑总等级/(防御建筑总等级+50))`
- **典型事件示例**:
  - 商队来访: 热情款待(产出×1.5,2h) / 适度交易(铜钱+200~500) / 征收税金(铜钱+500~1000,声望-10)
  - 流寇来袭: 出兵剿灭(声望+15,战利品,兵力-50~200) / 花钱买路(铜钱-300~800) / 坚守不出(粮草-10%~20%)
  - 天降祥瑞: 祭天祈福(全产出×2,1h) / 广施恩惠(民心+20,声望+10)
  - 瘟疫蔓延: 施药救治(铜钱-500~1500,民心+15) / 封锁隔离(产出×0.5,2h) / 花费消除(铜钱-800~2000)
- 验证: 选项后果与描述一致、奖励缩放正确、损失减免正确

### 7.3 连锁事件
- 在事件中选择特定分支 → 10~30min后触发下一环 → 继续选择 → 完成全部环节获得最终奖励
- **链结构**: 2~4环(大部分3环) / 最多同时进行1条事件链 / 剧情链不受此限制
- **链超时**: 单环等待>24h未响应 → 链自动终止
- **链中断补偿**: 按已完成环节数给予30%阶段奖励
- **链完成奖励**: 完成全部环节后额外获得最终奖励
- **重复规则**: 同一事件链完成后7天冷却
- **剧情事件链特性**: 不受链并发限制 / 无环间延迟(连续触发) / 必须全部完成才能推进下一章 / 选择影响后续NPC关系和可招募武将
- **典型连锁事件**:
  - 桃园结义链(3环): 路遇豪杰→桃园设宴→义薄云天 → 最终奖励: 武将「关羽·忠义」+声望+50+铜钱×3000
  - 黄巾余党链(4环): 黄巾残党出没→发现宝藏→首领现身→招降成功 → 最终奖励: 武将「周仓」+兵力+1000
  - 商路开辟链(3环): 商人求助→商路受阻→商路畅通 → 最终奖励: 解锁常驻商路(每日铜钱×500+粮草×300)
- 验证: 连锁事件环间延迟正确、选择分支正确触发后续、中断补偿正确、最终奖励发放

### 7.4 离线事件处理
- 离线期间 → 事件按规则自动处理 → 上线后回归面板显示离线事件摘要
- **离线处理规则**:
  - 随机遭遇(正面): 自动选择"保守"选项，获得50%基础奖励
  - 随机遭遇(负面): 按当前防御值自动计算损失(最低损失)
  - 随机遭遇(中性): 堆积待处理，最多3条
  - 剧情事件: 暂停等待，不触发不推进
  - NPC事件: 堆积待处理，最多2条
  - 天灾人祸: 按防御建筑等级计算减免，剩余损失自动扣除
  - 限时机遇: 错过消失
  - 里程碑事件: 暂停等待
- **离线收益公式**:
  - 正面收益 = 基础奖励 × 50% × min(离线时长/1小时, 8)
  - 负面损失 = 基础损失 × max(0, 1 - 防御减免率)
  - 天灾损失上限 = 当前资源总量15%
- **堆积上限**: 最多5条(超出按优先级丢弃) / 保留72h(超时自动保守处理) / 离线<30min不触发处理
- 验证: 各类型事件离线处理规则正确、收益/损失计算正确、堆积上限生效

### 7.5 事件跨系统联动
- **随机遭遇→战斗**: 选择"出兵"选项 → 进入战斗界面
- **随机遭遇→商店**: 商队事件选择"交易" → NPC交易面板

  - 稀有NPC位置与普通NPC重叠: 稀有NPC优先级更高，普通NPC被聚合气泡吸收

## 7.9 NPC离线特殊事件完整流程（F6.40补充）
- 玩家离线满足条件 → 触发NPC离线特殊事件 → 上线后弹出事件 → 玩家响应 → 获得奖励
- **触发条件**:
  - NPC求助: 离线≥4h + 好感度≥Lv.3 + 随机5%/h
  - NPC送礼: 离线≥6h + 好感度≥Lv.4 + 随机
  - NPC引荐: 离线≥8h + 好感度≥Lv.5 + 随机
- **操作步骤（以NPC求助为例）**:
  1. 离线期间系统判定NPC求助事件触发
  2. 玩家上线 → 弹出回归面板(含NPC求助条目)
  3. 点击"查看详情" → NPC对话面板弹出
  4. NPC陈述需求(如"急需粮草×500救济灾民")
  5. 玩家选择: "提供资源"(消耗资源获得稀有道具) / "婉拒"(无消耗无奖励，好感度不变)
  6. 选择"提供资源" → 资源扣除 → NPC感谢 → 获得稀有道具 → 好感度+20
- **操作步骤（NPC送礼）**:
  1. 上线 → 回归面板显示"NPC送礼"条目
  2. 点击查看 → NPC赠送动画(礼物盒打开特效)
  3. 获得稀有道具 → 自动入背包
- **操作步骤（NPC引荐）**:
  1. 上线 → 回归面板显示"NPC引荐"条目
  2. 点击查看 → NPC介绍新NPC的对话动画
  3. 新NPC加入固定NPC池 → 地图上出现新NPC标记
- **预期结果**: 离线回归有丰富的NPC互动内容，特殊事件触发率约5%~15%/次
- **异常处理**:
  - 背包已满(NPC送礼): 道具暂存"临时仓库"(24h内领取)，超时道具消失并补偿铜钱×500
  - 资源不足(NPC求助): 提供"部分援助"选项(消耗50%资源获得50%奖励)
  - 新NPC池已满(NPC引荐): 替换为"NPC好感度+50"奖励

## 7.10 事件类型筛选异常处理（F6.30补充）
- 打开大势面板 → 切换事件类型标签 → 异常场景处理
- **异常处理**:
  - 筛选结果为空: 显示空状态插画+"暂无此类事件，探索世界可能触发新事件"+推荐操作按钮
  - 快速切换标签: 防抖200ms，避免频繁请求
  - 事件列表数据同步延迟: 显示最后更新时间戳+"下拉刷新"手势
  - 网络中断: 使用本地缓存数据，显示"离线模式"标签，恢复后自动同步

## 7.11 时代推进×资源产出联动流程
- 时代变迁 → 时代加成因子生效 → 领土产出公式新增时代乘数
- **触发条件**: 时代变迁动画播放完成
- **操作步骤**:
  1. 时代目标全部完成 → 触发时代变迁动画(5秒)
  2. 动画结束 → 弹出时代结算面板(奖励+解锁内容)
  3. 玩家领取奖励 → 系统后台更新时代加成因子
  4. 领土产出公式新增: `总产出 = 基础×地形×阵营×科技×声望×地标×驻防×时代加成`
  5. 时代加成值: 黄巾之乱×1.0 / 群雄割据×1.1 / 官渡之战×1.2 / 赤壁之战×1.3 / 三国鼎立×1.5
  6. 所有己方领土产出气泡数值刷新(带↑动画)
- **预期结果**: 时代变迁后产出即时提升，产出气泡显示新数值
- **异常处理**:
  - 时代变迁动画中断(切后台): 回到前台后跳过动画直接显示结算面板
  - 产出计算溢出: 上限为当前时代最高产出×2

## 7.12 NPC好感度×时代推进联动流程
- 时代变迁 → 新NPC类型解锁 → 好感度保留规则 → 新交互内容
- **触发条件**: 时代变迁完成
- **操作步骤**:
  1. 时代变迁完成 → 解锁新NPC类型(如群雄割据解锁谋士NPC)
  2. 已有NPC好感度100%保留，不因时代变迁而重置
  3. 新NPC出现在地图上(带"新"标签+引导箭头)
  4. 时代奖励可能包含"全NPC好感度+20"(限时加成)
  5. 新时代解锁新的NPC任务链(与时代剧情关联)
- **预期结果**: 时代推进为NPC系统注入新内容，形成"推进→解锁→互动"正循环
- **异常处理**:
  - 新NPC类型已全部解锁: 替换为"现有NPC好感度上限提升(1000→1500)"
  - NPC地图位置冲突: 新NPC分配到空闲位置，无空闲则进入名册等待列表

## 7.13 连锁事件×时代推进联动流程
- 剧情事件链完成 → 推进时代进度 → 解锁新时代内容
- **触发条件**: 剧情事件链最后一环完成
- **操作步骤**:
  1. 完成剧情链最后一环 → 显示链完成动画+最终奖励
  2. 系统计算时代进度推进值: 每完成1环剧情链→时代进度+5%
  3. 完成整条剧情链→额外时代进度+10%
  4. 时代进度条实时更新(带进度增长动画)
  5. 若进度达到100% → 自动触发时代变迁流程(见7.11)
  6. 新时代解锁新的剧情事件链(如"群雄割据"链)
- **预期结果**: 剧情链是时代推进的主要驱动力之一，完成链后有明显进度感
- **异常处理**:
  - 时代进度已满但剧情链未完成: 时代变迁等待剧情链完成(最多等24h，超时自动变迁)
  - 剧情链选择影响后续时代: 记录选择到存档，后续时代NPC关系和可招募武将据此调整

## 7.14 攻城×活动系统联动流程
- 攻城成功 → 获得攻城奖励 → 奖励包含活动代币 → 活动积分增加
- **触发条件**: 攻城战胜利+有进行中的活动
- **操作步骤**:
  1. 攻城战胜利 → 弹出攻城奖励面板
  2. 奖励中包含活动代币(如"攻城勋章×10")
  3. 代币自动入账 → 活动积分面板实时更新
  4. 攻城行为同时推进活动任务(如"击败敌军50次"计数+1)
  5. 活动任务完成 → 额外弹出活动奖励领取提示
- **预期结果**: 攻城行为与活动系统联动，增加攻城的额外动力
- **异常处理**:
  - 无进行中活动: 攻城奖励不包含活动代币，正常发放基础奖励
  - 活动即将结束(<1h): 攻城奖励代币×1.5倍(限时加成提示)
  - 活动代币溢出: 上限9999，溢出部分自动转换为铜钱(1代币=100铜钱)
