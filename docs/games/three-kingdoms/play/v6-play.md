# v6.0 天下大势 — 玩游戏流程

## 1. 天下大势面板
- 打开世界地图 → 点击右上角"天下大势"按钮 → 展开大势面板 → 查看当前时代/全局事件/势力分布
- 验证: 面板显示当前时代名称、活跃全局事件列表、各势力领土占比饼图
- 预期: 面板数据与实际游戏状态一致，势力占比总和100%

### 1.1 急报横幅
- 全局事件触发时 → 顶部弹出急报横幅 → 横幅显示事件摘要 → 点击横幅跳转详情
- 验证: 横幅3秒后自动收起、点击可展开、多条急报排队轮播
- 预期: 事件触发后1秒内出现横幅，不遮挡核心操作区

### 1.2 事件类型筛选
- 打开大势面板 → 切换事件类型标签(随机/固定/连锁) → 列表按类型过滤
- 验证: 筛选结果仅显示对应类型、空状态提示"暂无此类事件"
- 预期: 切换标签后列表即时刷新，无闪烁

## 2. 时代推进
- 进入天下大势面板 → 查看当前时代进度条 → 完成时代目标 → 进度满 → 触发时代变迁动画 → 进入下一时代
- 验证: 进度条百分比准确、时代变迁动画播放完整、新时代解锁内容生效
- 预期: 黄巾之乱→群雄割据→官渡之战→赤壁之战→三国鼎立，按历史顺序推进

### 2.1 时代目标
- 点击时代进度条旁"目标"按钮 → 弹出时代目标列表 → 查看已完成/未完成目标
- 验证: 已完成目标打勾、未完成目标显示当前进度、奖励预览
- 预期: 所有目标达成后自动触发时代变迁，不可手动跳过

### 2.2 时代奖励
- 时代变迁完成 → 弹出时代结算面板 → 展示获得奖励(资源/道具/解锁内容) → 点击领取
- 验证: 奖励正确入账、解锁内容(新NPC/新区域/新事件)可访问
- 预期: 奖励一次性发放，不可重复领取

## 3. 势力消长
- 打开天下大势面板 → 切换到"势力"Tab → 查看各势力领土/兵力/武将数量变化趋势图
- 验证: 趋势图数据随时间更新、鼠标悬停显示具体数值、势力颜色与地图一致
- 预期: 玩家自身势力数据实时更新，NPC势力按回合更新

### 3.1 领土攻占
- 世界地图选择敌方领土 → 查看胜率预估 → 派遣武将攻城 → 战斗结算 → 领土归属变化
- 验证: 胜率预估与实际结果偏差≤10%、攻城奖励(资源+道具)正确发放、领土颜色变更
- 预期: 攻占后势力领土数+1，防守方领土数-1，大势面板数据同步更新

### 3.2 驻防机制
- 选择己方领土 → 点击"驻防" → 从武将列表选择驻防武将 → 确认派遣 → 领土防御值提升
- 验证: 驻防武将不可同时出战、防御加成数值正确、产出加成生效
- 预期: 派驻后领土防御面板显示加成后数值，驻防武将头像出现在领土上

### 3.3 离线领土变化
- 离线一段时间后重新上线 → 弹出离线变化报告 → 查看领土得失详情 → 确认关闭
- 验证: 离线期间领土被攻占记录完整、新获得的领土显示来源
- 预期: 离线变化可视化标注在地图上，红色=失、绿色=得

## 4. 世界地图交互
- 打开世界地图 → 拖拽平移/滚轮缩放 → 点击领土查看详情 → 点击NPC触发交互
- 验证: 地图拖拽流畅无卡顿、缩放范围合理(0.5x~3x)、领土详情弹窗正确
- 预期: PC端鼠标+手机端触摸手势均正常响应

### 4.1 地图事件标记
- 地图上出现随机事件图标 → 点击图标 → 弹出事件描述+选项 → 选择后获得奖励/惩罚
- 验证: 事件图标在地图上正确定位、选项后果与描述一致、奖励即时到账
- 预期: 事件图标有脉冲动画提示，完成后图标消失

### 4.2 NPC地图交互
- 地图上发现NPC标记 → 点击NPC → 弹出NPC信息卡 → 选择"对话"进入对话面板
- 验证: NPC信息卡显示名字/职业/好感度、对话选项随好感度等级解锁
- 预期: 好感度Lv.5解锁羁绊技能，NPC聚合气泡在拥挤区域折叠显示

### 4.3 手机端地图
- 手机端打开世界地图 → 触摸拖拽+双指缩放 → 点击领土从底部弹出Bottom Sheet → 查看详情
- 验证: Bottom Sheet可上滑展开/下滑收起、不遮挡顶部操作区
- 预期: 底部面板展示领土核心信息，上滑展开完整详情

## 5. 交叉验证

### 5.1 时代推进×势力消长
- 攻占领土完成时代目标 → 时代变迁 → 新时代解锁新NPC → 新NPC出现在地图上
- 验证: 时代变迁后大势面板势力数据重算、新NPC类型与时代匹配
- 预期: 群雄割据时代解锁谋士NPC，赤壁之战时代解锁水战事件

### 5.2 全局事件×世界地图
- 全局事件"天灾"触发 → 地图上受灾区域标红 → 受灾领土产出降低 → 急报横幅通知
- 验证: 受灾效果正确应用、事件结束后效果消失、产出恢复正常
- 预期: 全局事件影响所有势力，NPC势力领土同样受影响

### 5.3 NPC好感度×事件系统
- 与NPC对话提升好感度 → 好感度Lv.3解锁事件线索 → 触发专属连锁事件 → 完成获稀有奖励
- 验证: 好感度门槛正确拦截/放行、连锁事件顺序不可颠倒、最终奖励品质高于普通事件
- 预期: NPC羁绊技能在战斗中生效，好感度进度条实时更新

### 5.4 离线处理×全局事件
- 离线期间全局事件触发 → 上线后离线报告中包含事件自动处理结果 → 查看事件详情
- 验证: 离线事件自动处理规则正确(默认选择最优项)、处理结果可追溯
- 预期: 离线事件处理不导致负面后果超过手动处理的50%
