# v14.0 千秋万代 — 玩游戏流程

> 核心循环: 游戏行为→声望累积→等级提升→加成增强→满足转生→重置+倍率→更快更强→成就解锁→传承永续

---

## 1. 传承系统（转生）

### 1.1 转生条件检查
- 操作: 点击主界面"转生"入口(声望分栏场景右栏) → 系统展示5项条件检查清单: 主城≥30级 / 通关"赤壁之战"全部 / 武将≥15 / 总战力≥50000 / 完成"初露锋芒"成就链(5子成就) → 全部✅后"执行转生"按钮亮起 → 距上次转生≥72h才可再次执行
- 验证: 条件项逐条实时判定(不满足显示❌及差距) | 5项全部满足+72h冷却后才可点击转生 | 冷却时间精确到秒 | 成就链进度可点击跳转
- 预期: 条件面板一目了然，差距提示引导玩家补齐短板，72h冷却防止频繁转生破坏体验

### 1.2 转生倍率与确认
- 操作: 满足条件后点击"执行转生" → 弹出转生确认面板: 倍率公式 1.5×(1+声望等级×0.05)×(1+(次数-1)×0.1) → 展示保留项(武将/装备/成就/图鉴/VIP)与重置项(主城/建筑/领土/铜钱/粮草/兵力) → 衰减项按比例: 武将等级50%/技能70%/科技50%/天命30% → 确认后执行转生动画(全屏金色光效+王朝更迭文字)
- 验证: 倍率公式计算与速查表一致(1次Lv5=1.88×/3次Lv15=3.19×/5次Lv20=4.50×) | 保留项数据完整不丢失 | 衰减比例精确到整数 | 重置项全部归零 | 转生前完整备份存档
- 预期: 确认面板信息透明消除玩家顾虑，倍率展示激励转生意愿，备份机制保障数据安全

### 1.3 转生后加速与重建
- 操作: 转生完成 → 自动获得初始资源(粮草×5000+铜钱×3000) → 建筑1~10级升级时间÷倍率(几乎瞬间) → 已通关关卡可直接扫荡 → 点击"一键重建"按钮 → 系统按上次优先级自动升级建筑 → 转生次数解锁: 1次天命/2次专属科技/3次神话武将/5次跨服
- 验证: 初始资源到账数量准确 | 低级建筑升级时间正确÷倍率 | 扫荡功能复用已有关卡记录 | 一键重建按历史优先级执行 | 次数解锁内容正确触发
- 预期: 加速机制让转生后快速回到正轨，一键重建消除重复操作疲劳，次数解锁内容提供长期追求

## 2. 声望系统

### 2.1 声望等级与升级
- 操作: 打开声望分栏场景(左栏等级+右栏商店/转生) → 展示当前声望等级/经验值/下一级阈值(1000×N^1.8) → 声望值达到阈值自动升级 → 弹出金色横幅通知+升级动画(进度条满格闪烁→金色粒子爆发→等级跳变) → 进度≥90%时进度条脉冲闪烁提醒
- 验证: 阈值公式精确(Lv2=3466/Lv5=18197/Lv10=63096/Lv20=276335) | 升级瞬间声望值溢出部分保留 | 动画播放不阻塞操作 | 加成特权在对应等级即时生效
- 预期: 声望等级提供清晰的长期成长线，升级动画给予强烈正反馈，接近升级提醒激发期待感

### 2.2 声望获取与加成
- 操作: 通过9种途径获取声望: 通关新关卡(50~500/关，三星×1.5)/每日声望任务(100~300/日)/领土产出(5~30/h)/NPC好感度升级(200~800/级)/成就完成(100~2000/个)/远征完成(50~200/次)/活动排名(500~5000/次)/离线累积(领土×50%) → 声望等级触发产出加成: Lv2领土+5%/Lv7远征+10%/Lv9武将经验+10%/Lv11建筑速度+5%/Lv14科技速度+5%
- 验证: 9种途径声望产出量与配置表一致 | 三星加成×1.5正确 | 离线累积受离线效率系数影响 | 各等级加成数值精确 | 多项加成叠加顺序正确
- 预期: 多途径声望获取覆盖全玩法不遗漏，加成特权让等级提升有实质收益感，离线累积保证放置体验

### 2.3 声望商店
- 操作: 声望分栏场景右栏切换"商店"Tab → 商品按声望等级解锁: Lv1扫荡令/经验书 → Lv5招贤榜 → Lv10求贤令/稀有装备图纸 → Lv20神话武将碎片 → 使用声望点数(与声望值独立)兑换 → 热门商品限购(周限2~5件) → 每周一0:00刷新限购
- 验证: 未达等级商品灰显不可购买 | 声望点数余额充足才可兑换 | 限购次数递减且周一重置 | 兑换物品正确入背包 | 商店价格不受转生倍率影响
- 预期: 声望商店是声望点数核心出口，等级解锁激励声望提升，限购控制资源膨胀节奏

## 3. 历史记录

### 3.1 转生收益模拟器
- 操作: 声望分栏场景右栏点击"收益模拟器" → 输入当前声望等级/转生次数/预计游玩时长 → 系统预测未来X小时声望增长曲线 → 计算声望等级提升带来的倍率增量 → 标注"推荐转生时机"(边际收益递减拐点) → 对比展示: 立即转生 vs 再等2h/6h/12h/24h的收益差异
- 验证: 模拟预测值与实际声望增长偏差<5% | 推荐时机标注准确 | 对比数据各项倍率计算一致 | 模拟器不消耗任何资源或改变游戏状态
- 预期: 收益模拟器帮助玩家做出明智转生决策，对比展示消除决策焦虑，推荐时机降低认知负担

### 3.2 转生历史与王朝记录
- 操作: 转生面板点击"历史记录" → 展示历次转生摘要: 转生次数/时间/当时声望等级/获得倍率/保留武将数 → 按时间轴排列(最新在上) → 点击单条展开详情(转生前关键数据快照: 领土数/总战力/成就数) → 王朝纪元命名(第N次转生=第N王朝)
- 验证: 历史记录数据与转生时备份一致 | 时间戳精确 | 快照数据完整反映转生前状态 | 王朝命名连续不跳号
- 预期: 转生历史赋予每次重置仪式感和纪念意义，王朝纪元命名增强沉浸感，数据快照便于回顾成长轨迹

## 4. 成就系统

### 4.1 成就面板与进度追踪
- 操作: 主界面点击"成就"入口 → 成就面板展示5个维度Tab: 养成(武将收集/等级/升星/装备)/战斗(关卡通关/三星/远征/PvP)/收集(武将图鉴/装备图鉴/NPC好感度)/社交(好友/联盟/借将/聊天)/经济(资源积累/商店消费/贸易利润) → 每项成就显示进度条+当前值/目标值 → 完成时弹出成就解锁动画+奖励领取
- 验证: 5维度分类覆盖全系统 | 进度条数值实时更新 | 成就完成判定精确(≥目标值即完成) | 奖励领取后状态锁定不可重复领取
- 预期: 成就面板一目了然展示全游戏进度，维度分类引导玩家体验多元玩法，进度条提供持续追求动力

### 4.2 成就奖励与点数
- 操作: 完成成就 → 领取奖励: 声望值(100~2000按难度)/元宝(里程碑成就)/限定称号(特殊成就专属)/成就点数 → 成就点数累积解锁成就宝箱(每100点一档) → 宝箱内容: 资源包/武将碎片/装备箱 → 限定称号在个人主页展示
- 验证: 声望值奖励正确计入声望系统 | 元宝数量与配置一致 | 限定称号唯一不重复 | 成就点数累积无上限 | 宝箱奖励按档位正确发放
- 预期: 多层奖励满足不同追求，声望值奖励串联声望系统形成闭环，限定称号提供社交展示价值

### 4.3 转生成就链"初露锋芒"
- 操作: 成就面板"特殊"区域展示成就链"初露锋芒"(5个子成就依次解锁) → 崭露头角: 主城达20级 → 猛将如云: 拥有10位武将 → 兵强马壮: 单武将战力达5000 → 谋定后动: 完成5项科技研究 → 威震四方: 占领20块领土 → 全部完成解锁转生资格+专属称号+声望×2000
- 验证: 子成就按顺序解锁(前置未完成后续不可触发) | 每项判定条件与对应系统数据一致 | 全部完成后转生条件中成就链项变为✅ | 专属称号正确发放
- 预期: 成就链引导玩家体验多系统(城建/招募/强化/科技/领土)避免偏科，同时作为转生门槛确保玩家具备综合实力

## 5. 交叉验证

### 5.1 声望获取→等级→加成→转生倍率全链路
- 操作: 玩家完成关卡(获取声望500)→完成声望日常(获取声望300)→占领领土(持续产出声望/h) → 声望值累积达到阈值自动升级 → 加成特权生效(领土产出+N%) → 领土产出增加→声望获取加速 → 达到转生条件执行转生 → 倍率=1.5×(1+声望等级×0.05)×(1+(次数-1)×0.1) → 倍率作用于全资源产出/经验/建筑/科技/扫荡/离线 → 商店价格/招募消耗/PvP属性不受影响
- 预期: 声望→等级→加成→更快获取声望形成正循环，转生倍率正确作用于指定范围且不越界，核心放置循环引擎运转流畅

### 5.2 转生→成就链→声望→商店闭环
- 操作: 玩家完成"初露锋芒"成就链5子成就 → 获得声望×2000+成就点数 → 声望等级提升 → 声望商店解锁新商品(招贤榜/求贤令) → 成就点数累积开宝箱获得武将碎片 → 武将碎片合成新武将 → 武将数≥15满足转生条件 → 执行转生 → 转生后加速重建 → 完成转生专属任务获得资源包 → 资源强化武将→更高战力→更多成就
- 预期: 成就→声望→商店→武将→转生→任务→资源→强化形成完整闭环，各系统数据流转一致无断点

### 5.3 收益模拟器→转生决策→历史记录闭环
- 操作: 玩家打开收益模拟器 → 输入当前数据(声望Lv12/转生2次) → 模拟器预测: 再等6h声望达Lv13倍率从2.63×→2.78× → 推荐"再等6h后转生" → 玩家按推荐等待→执行转生 → 转生历史记录本次数据快照(声望Lv13/倍率2.78×/武将18/领土25) → 下次查看历史对比成长轨迹 → 再次使用模拟器规划下一轮转生
- 预期: 模拟器预测与实际转生结果偏差<5%，历史记录完整保存每次转生快照，多轮转生数据可追溯形成完整王朝编年史
