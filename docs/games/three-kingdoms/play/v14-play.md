# v14.0 千秋万代 — 玩游戏流程

> 核心循环: 游戏行为→声望累积→等级提升→加成增强→满足转生→重置+倍率→更快更强→更高声望
> 副循环: 成就解锁→里程碑达成→称号获取→图鉴收集→声望累积

---

## 1. 成就系统

### 1.1 成就面板与分类
- 操作: 主界面点击"成就"入口 → 成就面板展示5个维度Tab: 养成(武将收集/等级/升星/装备)/战斗(关卡通关/三星/远征/PvP)/收集(武将图鉴/装备图鉴/NPC好感度)/社交(好友/联盟/借将/聊天)/经济(资源积累/商店消费/贸易利润) → 每项成就显示进度条+当前值/目标值 → 完成时弹出成就解锁动画+奖励领取弹窗
- 验证: 5维度分类覆盖全系统无遗漏 | 进度条数值实时更新(≤1s延迟) | 成就完成判定精确(≥目标值即触发) | 奖励领取后状态锁定不可重复领取 | 未解锁成就灰显但可见进度
- 预期: 成就面板一目了然展示全游戏进度，维度分类引导玩家体验多元玩法，进度条提供持续追求动力

### 1.2 成就奖励与点数
- 操作: 完成成就 → 领取奖励: 声望值(100~2000按难度)/元宝(里程碑成就)/限定称号(特殊成就专属)/成就点数 → 成就点数累积解锁成就宝箱(每100点一档) → 宝箱内容: 资源包/武将碎片/装备箱 → 限定称号在个人主页展示并可装备
- 验证: 声望值奖励正确计入声望系统触发升级判定 | 元宝数量与配置表一致 | 限定称号唯一不重复发放 | 成就点数累积无上限 | 宝箱奖励按档位正确发放且不越级
- 预期: 多层奖励满足不同追求，声望值奖励串联声望系统形成闭环，限定称号提供社交展示价值

### 1.3 转生成就链"初露锋芒"
- 操作: 成就面板"特殊"区域展示成就链"初露锋芒"(5个子成就依次解锁) → 崭露头角: 主城达20级 → 猛将如云: 拥有10位武将 → 兵强马壮: 单武将战力达5000 → 谋定后动: 完成5项科技研究 → 威震四方: 占领20块领土 → 全部完成解锁转生资格+专属称号"初露锋芒"+声望×2000
- 验证: 子成就按顺序解锁(前置未完成后续不可触发) | 每项判定条件与对应系统实时数据一致 | 全部完成后转生条件中成就链项变为✅ | 专属称号唯一发放且不可重复获取
- 预期: 成就链引导玩家体验多系统(城建/招募/强化/科技/领土)避免偏科，同时作为转生门槛确保玩家具备综合实力

## 2. 里程碑

### 2.1 里程碑定义与触发
- 操作: 里程碑为成就系统中的高价值节点，独立展示在成就面板顶部"里程碑"区域 → 里程碑类型: 战力里程碑(总战力达1万/5万/10万/50万)/收集里程碑(武将达10/20/50位)/领土里程碑(占领10/30/50块)/关卡里程碑(通关第3/5/8章)/转生里程碑(首次转生/第3次/第5次) → 达成时全屏庆典动画+专属横幅通知
- 验证: 里程碑阈值与配置表精确匹配 | 触发判定在对应系统数据变更时即时检查 | 庆典动画播放期间不阻塞后台逻辑 | 同一里程碑不可重复触发
- 预期: 里程碑提供阶段性大目标，全屏庆典给予强烈仪式感，帮助玩家感知长期成长节奏

### 2.2 里程碑奖励与加成
- 操作: 达成里程碑 → 领取丰厚奖励: 元宝×200~1000(按里程碑等级)/稀有武将碎片×5~20/限定头像框/永久属性加成(如"战力达5万"永久攻击+3%) → 属性加成在个人信息页展示来源标签 → 多个里程碑加成可叠加
- 验证: 元宝/碎片数量与配置一致 | 属性加成正确写入玩家属性并参与战斗计算 | 加成来源标签可追溯 | 多项加成叠加顺序正确(加法叠加后乘以基础)
- 预期: 里程碑奖励丰厚激励玩家冲刺关键节点，永久加成提供长线收益，来源标签增强成就感

### 2.3 里程碑进度预览
- 操作: 成就面板"里程碑"区域 → 未达成里程碑显示进度条+距离下一里程碑差距提示 → 点击里程碑展开详情: 已完成子目标/待完成子目标/预估达成时间(基于近期游戏行为推算) → 接近达成(≥80%)时主界面显示脉冲提示图标
- 预期: 进度预览降低目标模糊感，预估时间帮助玩家规划游戏节奏，接近达成提示激发冲刺欲望

## 3. 称号系统

### 3.1 称号获取与分类
- 操作: 个人主页"称号"页签 → 展示已获得/未获得称号列表 → 称号来源分类: 成就称号(完成特定成就)/里程碑称号(达成里程碑)/排名称号(赛季排行)/活动称号(限时活动)/转生称号(转生次数解锁) → 未获得称号显示获取条件灰显展示
- 验证: 称号来源标签与实际获取途径一致 | 已获得称号可随时装备/卸下 | 未获得称号仅展示条件不可装备 | 同一称号不可重复获取
- 预期: 称号系统提供社交展示核心，分类清晰便于收集规划，未获得称号灰显激发收集欲望

### 3.2 称号装备与属性加成
- 操作: 选择已获得称号点击"装备" → 称号显示在玩家头像下方(主界面/排行榜/联盟/聊天) → 部分称号附带属性加成(如"威震四方"攻击+5%/"富甲天下"资源产出+10%) → 同一时间仅可装备1个称号 → 属性加成立即生效，卸下后立即移除
- 验证: 装备称号在所有社交场景正确显示 | 属性加成精确写入战斗计算公式 | 卸下称号后属性即时扣除 | 排行榜/联盟/聊天中称号展示一致
- 预期: 称号装备提供个性化展示，属性加成增加策略选择(战力型/资源型)，社交场景展示强化荣誉感

### 3.3 称号稀有度与展示
- 操作: 称号按稀有度分4级: 普通(白)/稀有(蓝)/史诗(紫)/传说(金) → 稀有度决定称号文字颜色和光效强度 → 传说级称号附带专属动态光效(如"千秋万代"金色龙纹环绕) → 个人主页称号墙展示全部已获得称号按稀有度排列
- 验证: 稀有度颜色和光效与配置一致 | 传说级动态光效渲染流畅不卡顿 | 称号墙按稀有度降序排列 | 称号总数统计准确
- 预期: 稀有度分级创造收集层次感，传说级光效提供炫耀资本，称号墙成为个人成就展示橱窗

## 4. 图鉴与收藏

### 4.1 武将图鉴
- 操作: 主界面点击"图鉴"入口 → 切换"武将图鉴"Tab → 展示全部武将卡片网格(已获得亮色/未获得剪影) → 点击已获得武将查看详情: 基础属性/技能描述/传记故事/获得途径 → 图鉴收集进度条(已获得数/总数) → 每收集10位武将解锁图鉴宝箱(铜钱×5000+元宝×50)
- 验证: 已获得武将数据与武将系统实时同步 | 未获得武将仅显示剪影和获取提示 | 收集进度百分比精确 | 图鉴宝箱每10位触发一次且不重复
- 预期: 武将图鉴满足收集欲望，传记故事丰富世界观，图鉴宝箱激励持续招募

### 4.2 装备图鉴
- 操作: 图鉴面板切换"装备图鉴"Tab → 展示全部装备按品质分类(白/绿/蓝/紫/金) → 已获得装备展示属性和套装信息 → 未获得装备显示获取途径(关卡掉落/商店兑换/活动奖励) → 收集完整套装额外解锁套装图鉴奖励(套装专属称号或资源包)
- 验证: 装备数据与背包系统实时同步 | 品质颜色与装备系统一致 | 套装收集判定精确(所有部件曾拥有即算) | 套装图鉴奖励正确发放
- 预期: 装备图鉴引导玩家追求套装收集，获取途径提示降低信息壁垒，套装奖励提供额外追求

### 4.3 收藏系统
- 操作: 图鉴面板切换"收藏"Tab → 展示收藏品分类: 限定道具(活动/赛季独占)/纪念勋章(转生/里程碑纪念)/特殊记录(首次通关/最高连斩/最大伤害) → 收藏品不可交易/丢弃，永久保留(转生不重置) → 收藏完成度影响个人主页展示框架(铜/银/金/钻石边框)
- 验证: 收藏品数据跨转生保留完整 | 限定道具过期后不再获取但已拥有不消失 | 特殊记录实时更新且仅保留历史最佳 | 个人主页边框与收藏完成度匹配
- 预期: 收藏系统赋予游戏行为纪念意义，跨转生保留让玩家安心，完成度边框提供长期追求目标

### 5.1 声望分栏场景与等级展示

- 验证: 左右分栏布局正确渲染(PC端左固定右分栏/手机端顶部折叠卡片+全屏面板) | 等级阈值按公式1000×N^1.8精确计算(前20级与配置表核对) | 进度条百分比=当前声望/(下级阈值-当前级阈值) | 加成列表仅展示已解锁且当前生效的加成 | 获取明细与后端数据一致

### 5.2 声望等级升级流程
- 操作: 玩家获得声望值 → 声望值累加到当前总量 → 判断是否达到下一等级阈值(1000×N^1.8) → 达到阈值 → 自动升级 → 触发升级动画(进度条满格闪烁→金色粒子爆发→等级数字跳变) → 弹出金色横幅通知"声望等级提升至 Lv.X！解锁：{特权名}" → 检查是否有新解锁奖励/特权(Lv.3天命产出/Lv.5商店/Lv.8稀有NPC/Lv.15称号) → 新加成立即生效
- 验证: 升级判定在每次声望值变动时即时检查 | 升级动画播放期间不阻塞后台逻辑 | 横幅通知内容与解锁配置一致 | 新加成立即写入对应系统(如领土产出+N%即时生效) | 跨级升级(一次声望值跨越多个阈值)时逐级触发动画但仅展示最终等级横幅
- 预期: 自动升级减少操作负担，金色动画和横幅提供强烈仪式感，新加成立即生效让玩家感受到声望等级的实际价值

### 5.3 声望获取途径(9种)
- 操作: 声望面板点击"获取途径"按钮 → 弹出声望获取途径总览面板(9种途径分3组展示) → **战斗组**: 通关新关卡(50~500声望/关，按难度)/三星通关(基础×1.5)/远征完成(50~200/次) → **日常组**: 每日声望任务(100~300声望/日，3个专属日常)/领土产出(5~30声望/小时)/离线累积(领土产出×50%，受离线效率系数) → **成就组**: 成就完成(100~2000/个)/NPC好感度升级(200~800/级)/活动排名(500~5000/次) → 每种途径显示今日已获取量和图标状态(已达日上限/可继续获取)
- 验证: 9种途径全部覆盖无遗漏 | 各途径声望值与PRD配置一致 | 已通关关卡扫荡不获得声望(仅首次通关) | 声望值只增不减，转生时保留累计值 | 离线声望按领土产出×50%×离线效率阶梯计算 | 每日无硬性获取上限
- 预期: 获取途径总览帮助玩家规划声望获取策略，9种途径覆盖全玩法鼓励多元参与，日获取量展示提供进度感知

### 5.4 产出加成特权
- 操作: 声望等级提升 → 检查是否解锁新加成特权 → 加成类型与等级对应: Lv.2/6/12/19→领土产出+5%/+10%/+15%/+20%(取最高级生效不叠加) / Lv.7/13→远征收益+10%/+15% / Lv.9/16→武将经验+10%/+15% / Lv.11/18→建筑升级速度+5%/+10% / Lv.14→科技研究速度+5% → 加成立即写入对应系统 → 资源产出面板/建筑升级面板/科技研究面板显示"声望加成"标签和具体数值
- 验证: 同类加成取最高级生效不叠加(如Lv.19领土+20%而非5%+10%+15%+20%) | 加成数值精确写入产出计算公式 | 加成标签在各系统面板正确展示 | 声望等级降低(理论上不会)时加成同步移除
- 预期: 产出加成是声望等级的核心价值体现，取最高级规则避免复杂叠加计算，标签展示增强声望系统存在感

### 5.5 等级解锁奖励
- 操作: 声望等级提升至特定等级 → 触发解锁奖励弹窗 → Lv.1: 基础声望特权(声望商店基础商品购买资格) → Lv.3: 离线天命产出(天命阁建筑解锁，离线时微量产出天命约5/天) → Lv.4: 商店折扣5%(声望商店所有商品享受5%折扣) → Lv.5: 声望商店解锁(使用声望点数兑换稀有道具) → Lv.8: 稀有NPC交互(解锁与稀有NPC的对话和交易选项) → Lv.10: 高级商品(声望商店上架高级道具) → Lv.15: 专属称号"威震华夏"(纯装饰，显示在头像旁) → Lv.20: 终极商品(声望商店上架终极道具)
- 验证: 解锁等级与配置精确匹配 | 解锁内容立即生效(如Lv.5商店入口出现) | Lv.4折扣正确作用于声望商店所有商品价格 | Lv.15称号自动发放至称号系统 | 转生后声望等级保留，已解锁奖励不重复发放
- 预期: 等级解锁奖励提供阶段性惊喜，商店/NPC/称号等解锁让玩家持续有新内容可体验

### 5.6 声望商店

## 6. 转生系统
