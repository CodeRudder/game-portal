# v3.0 攻城略地(上) — 玩游戏流程

## 1. 战役流程
- 打开战役Tab → 查看关卡列表 → 选择可挑战关卡 → 布阵 → 战斗 → 结算 → 星级评定
- 验证: 关卡状态(已通关/当前/锁定)、星级显示、奖励预览
- 交叉验证: 战斗胜利→campaignProgress更新→新关卡解锁→资源奖励到账

## 2. 扫荡流程
- 选择已三星关卡 → 设置扫荡次数 → 执行扫荡 → 查看收益
- 验证: 扫荡按钮状态(需三星通关)、收益计算正确
- 限制: 扫荡次数受体力/道具限制

## 3. 世界地图
- 打开地图Tab → 查看领土网格 → 选择敌方领地 → 攻城确认 → 战斗 → 占领
- 验证: 领土状态变化、占领进度、产出加成
- 交叉验证: 占领领土→资源产出增加→资源栏刷新

## 4. 自动推图
- 开启自动推图开关 → 观察自动战斗流程 → 验证进度推进
- 验证: 自动推图进度显示、卡住时停止
