## 1. 武将羁绊深化（模块A）

### 1.1 阵营羁绊效果
- 操作: 武将编队界面→部署6名武将→系统实时检测阵营分布→自动激活对应羁绊: 2同阵营→「同乡之谊」攻击+5% / 3同阵营→「同仇敌忾」攻击+15% / 6同阵营→「众志成城」攻击+25%+防御+15% / 3+3不同阵营→「混搭协作」攻击+10%+特效加成 → 多羁绊可同时激活（如2蜀+3魏+1吴可同时触发「同乡之谊」和「同仇敌忾」）→ 羁绊效果在战斗中全程生效
- 验证: 4种羁绊条件检测准确无遗漏 | 羁绊属性加成正确计算并应用到战斗数值 | 多羁绊叠加时加成数值精确 | 羁绊效果在战斗结算中体现
- 预期: 羁绊系统增加编队策略深度，激励收集同阵营武将，不同关卡需要不同羁绊策略

### 1.2 羁绊可视化
- 操作: 编队界面→部署或调整武将时→界面右侧/底部实时显示「羁绊面板」: 当前激活羁绊列表(图标+名称+效果) / 属性加成预览(攻击/防御/技能伤害变化值) / 推荐羁绊组合提示(如"再上阵1名蜀国武将可激活同仇敌忾") → 点击羁绊图标展开详情: 羁绊条件/加成数值/适用武将列表 → 替换武将时羁绊面板实时刷新
- 验证: 编队变化时羁绊面板0延迟刷新 | 属性加成预览数值与实际战斗一致 | 推荐提示逻辑正确（指向可达成的羁绊）
- 预期: 实时可视化让玩家直观理解羁绊策略，推荐提示降低策略门槛

### 1.3 武将故事事件
- 操作: 武将详情页→好感度进度条达到阈值(10/20/30/40/50)→弹出专属剧情事件→历史典故演绎(桃园结义/三顾茅庐/草船借箭等)→完成剧情对话→获得奖励: 好感度+N / 武将碎片×M / 声望×K → 剧情可在「图鉴→剧情回顾」中重播
- 验证: 好感度阈值触发准确 | 剧情内容与武将阵营/历史匹配 | 奖励发放正确 | 重播功能正常
- 预期: 武将故事增加角色认同感，好感度系统提供额外养成维度，历史典故增强文化沉浸

## 2. 装备强化系统（模块B）

### 2.1 装备强化流程
- 操作: 武将详情页→点击装备槽位→选择「强化」→强化面板展示: 当前强化等级/成功率(如+5→+6为80%)/铜钱消耗/强化石消耗 → 选择保护符策略: 不使用/铜质保护符(+5~+8)/银质保护符(+8~+12)/金质保护符(全等级) → 点击「强化」→概率判定 → 成功: 等级+1，主属性提升15%~18%，副属性提升5%~8% / 失败: 等级-1(不低于+5)，保护符生效时不降级 → 强化动画: 成功金光闪烁/失败红色震动
- 验证: 强化成功率表正确(+1→+2 100% / +2→+3 100% / +3→+4 95%不降级 / +4→+5 90%不降级 / +5→+6 80% / ...递减... / +14→+15 10%) | 失败降级不低于+5 | +3→+4和+4→+5失败不降级(仅消耗材料) | 金色装备+12以上失败不降级仅消耗材料 | 保护符消耗正确且生效 | 品质强化上限正确(白+5/绿+8/蓝+10/紫+12/金+15) | 达到品质上限时强化按钮灰显+提示「已达当前品质强化上限，请提升品质后继续」
- 预期: 强化提供明确的装备成长路径，成功率机制增加紧张感，保护符降低挫败感

### 2.2 强化保护符
- 操作: 强化面板→保护符选择区→展示可用保护符数量 → 铜质保护符: 适用于+5~+8强化，失败不降级 → 银质保护符: 适用于+8~+12强化，失败不降级 → 金质保护符: 适用于全等级强化，失败不降级 → 选择后保护符图标高亮→强化时自动消耗 → 保护符不足时提示获取途径(商店购买/活动奖励/成就奖励)
- 验证: 三种保护符适用等级范围正确 | 保护符消耗与强化次数对应 | 保护符不足时正确拦截
- 预期: 三级保护符满足不同阶段需求，获取途径多样保证可获得性

### 2.3 自动强化
- 操作: 强化面板→点击「自动强化」→设置目标等级(不超过品质上限)→开启/关闭「自动使用保护符」开关→确认 → 系统自动连续强化: 循环执行强化→成功继续/失败判断保护符→直至达到目标等级或材料耗尽或铜钱不足 → 自动强化过程中显示实时进度: 当前等级/目标等级/已消耗材料/已消耗铜钱 → 可随时点击「停止」中断 → 完成后弹出结果摘要: 成功次数/失败次数/总消耗/最终等级
- 验证: 自动强化停止条件正确(材料不足/铜钱不足/达到目标/手动停止) | 保护符策略开关生效 | 中断后数据状态正确 | 结果摘要数据准确
- 预期: 自动强化减少重复操作，保护符策略让玩家控制风险，进度展示提供透明度

### 2.4 装备炼制
- 操作: 装备背包→点击「炼制」→炼制面板4个Tab: 基础炼制/高级炼制/定向炼制/保底炼制 → 基础炼制: 投入3件同品质装备+铜钱×2000→随机产出高一品质(概率: 85%高一阶/14%高二阶/1%高三阶) → 高级炼制: 投入5件同品质装备+铜钱×5000→概率+10%(如3白→75%绿/20%蓝/4%紫/1%金) → 定向炼制: 投入3件同品质+定向石×1+铜钱×8000→指定部位产出(武器30%/防具30%/饰品25%/坐骑15%) → 保底炼制: 投入3件同品质+保底符×1+铜钱×10000→必定≥紫色 → 炼制结果弹出→装备入背包
- 验证: 4种炼制模式概率表正确 | 保底计数器跨会话保持(10次未紫→第11次必紫/30次未金→第31次必金) | 定向炼制部位概率正确 | 炼制消耗扣除准确
- 预期: 4种炼制模式满足不同需求，保底机制保证长期可获得性，定向炼制减少随机性

## 3. 套装系统（模块C）

### 3.1 套装效果
- 操作: 装备详情页→展示套装归属标签(如"青龙套装·武器")→套装信息面板展示: 套装名称/品质要求/2件套效果/4件套效果/已收集部位 → 7套装备一览: 青铜(绿色+，攻击+5%/普攻10%概率额外攻击) / 寒铁(蓝色+，防御+8%/受击15%概率减免30%) / 玄铁(蓝色+，暴击率+10%/暴击额外20%真实伤害) / 赤焰(紫色+，技能伤害+12%/20%概率重置冷却) / 龙鳞(紫色+，全属性+8%/低血量攻击+25%) / 霸王(金色，攻击+15%防御+10%/每回合攻击+5%最多5层) / 天命(金色，全属性+12%/开场全体护盾吸收20%最大生命)
- 验证: 7套装备2件/4件效果描述与PRD一致 | 套装品质要求正确 | 套装效果在战斗中正确触发
- 预期: 7套装备提供明确的追求目标，不同套装适配不同战术风格，2件套即有实用效果降低收集门槛

### 3.2 套装激活规则
- 操作: 武将装备面板→穿戴装备→系统自动检测套装归属→同一武将穿戴同套2件→激活2件套效果(图标亮起+属性加成生效)→同一武将穿戴同套4件→额外激活4件套效果 → 规则: 同一套装效果不可叠加(多件同套仅计算一次) / 套装效果跨武将不共享(仅穿戴武将生效) / 卸下装备导致件数不足→套装效果自动失效
- 验证: 2件/4件激活阈值正确 | 不叠加规则生效 | 跨武将不共享规则正确 | 卸装后效果失效
- 预期: 套装激活规则清晰易懂，2/4件分层激励完整收集

### 3.3 套装归属
- 操作: 装备获取时→系统判定套装归属: 金色品质装备固定归属某套装 / 紫色品质装备概率归属某套装(有套装标签) / 蓝色及以下无套装归属 → 每套包含4个部位(武器/防具/饰品/坐骑) → 背包中套装装备显示套装名称+色标 → 套装收集进度面板: 展示每套已收集部位/缺失部位/获取途径提示
- 验证: 金色固定归属正确 / 紫色概率归属符合设计 / 蓝色及以下无套装 / 套装4部位完整
- 预期: 金色固定归属提供确定性追求，紫色概率归属增加惊喜感，收集进度面板引导目标

## 4. 穿戴与背包管理（模块D）

### 4.1 一键穿戴推荐
- 操作: 武将装备面板→点击「一键穿戴」→军师推荐系统分析武将属性短板(攻击/防御/智力/统率)→筛选背包可用装备→排序: 品质优先>属性增量优先>等级匹配优先 → 每槽位推荐最多2件→弹出穿戴预览: 更换前后属性对比(四维雷达图)+战力变化 → 确认后执行穿戴→旧装备自动返回背包 → 武将详情页每个装备槽下方常驻展示推荐装备缩略图(蓝点提示有更好装备)
- 验证: 推荐逻辑排序正确(品质>增量>等级) | 预览属性变化与实际一致 | 旧装备正确返回背包 | 蓝点提示触发条件正确
- 预期: 一键穿戴降低装备管理门槛，属性对比让玩家理解推荐理由，蓝点提示避免错过更好装备

### 4.2 装备分解
- 操作: 装备背包→选择未穿戴装备→点击「分解」→展示分解产出预览: 铜钱+强化石(白色:铜钱×100 / 绿色:铜钱×300+强化石×1 / 蓝色:铜钱×800+强化石×2 / 紫色:铜钱×2000+强化石×3 / 金色:铜钱×5000+强化石×5) → 单件分解: 确认后执行→材料入账 → 批量分解: 点击「批量分解」→设置品质过滤(白色/绿色/蓝色以下)→展示待分解列表和总产出→确认执行 → 已穿戴装备不可分解(需先卸下) → 套装装备分解时额外确认提示"该装备属于XX套装，确认分解？"
- 验证: 分解产出与品质对应正确(白×100/绿×300+1石/蓝×800+2石/紫×2000+3石/金×5000+5石) | 批量分解品质过滤正常 | 已穿戴装备无法分解 | 套装装备分解有额外确认 | 背包满时无法卸下装备→提示「背包已满，请先清理」
- 预期: 分解系统清理背包冗余装备，批量操作提升效率，套装保护防止误操作

### 4.3 背包管理
- 操作: 装备背包面板→展示所有装备(卡片式/列表式切换)→默认50格容量→元宝扩容(每次+10格，最多扩至100格) → 排序: 品质降序/部位分类/套装归类/获取时间 → 筛选: 品质(白/绿/蓝/紫/金)/部位(武器/防具/饰品/坐骑)/套装(7套+无套装)/状态(已穿戴/未穿戴) → 装备卡片显示: 品质色框/部位图标/强化等级/套装标签/主属性 → 背包容量告警: >80%显示黄色提示/>95%显示红色提示并推荐分解
- 验证: 50格默认容量正确 | 扩容消耗元宝正确 | 排序和筛选功能正常 | 容量告警阈值触发正确
- 预期: 背包管理系统化处理装备，排序筛选快速定位目标，容量告警主动引导清理

## 5. 军师推荐系统（模块E）

### 5.1 军师建议触发规则
- 操作: 系统每分钟tick检测9种触发条件→满足条件时生成建议→按优先级排序(高/中/低)→检查冷却时间和每日上限 → 9种触发: 资源>80%上限→"粮仓满载，建议升级" / 资源<10%→"粮草告急，建议建造农田" / 建筑队列空闲→"建造队列空闲，建议升级" / 武将可升级→"武将{名}可升级" / 科技队列空闲→"科技研究空闲" / 兵力满值→"兵力已满，建议出征" / 限时NPC<2h→"{NPC名}即将离开" / 新功能解锁→"点击了解{功能名}" / 离线溢出>50%→"建议升级仓库"
- 验证: 9种触发条件检测准确 | 优先级排序正确(高>中>低) | 冷却时间正确: 资源>80%→30min / 资源<10%→30min / 建筑队列空闲→15min / 武将可升级→20min / 科技队列空闲→15min / 兵力满值→10min / 限时NPC<2h→5min / 新功能解锁→一次性 / 离线溢出>50%→60min | 每日上限15条正确计数
- 预期: 军师建议帮助新手理解"下一步该做什么"，减少迷茫感，触发条件覆盖核心游戏行为

### 5.2 建议内容结构与展示
- 操作: 主界面右侧/底部「军师建议面板」→最多同时展示3条建议→每条包含: 标题(≤20字)+描述(≤50字)+行动按钮(一键跳转)+置信度(高/中/低影响显示样式)+过期时间 → 按优先级排序，同优先级按触发时间排序 → 点击行动按钮→跳转至对应功能界面→执行操作后该建议自动移除 → 左滑/点击×关闭单条建议→同类型建议30分钟冷却不再推送 → 建议过期后自动移除
- 验证: 最多展示3条正确 | 排序规则正确 | 执行操作后自动移除 | 关闭后30分钟冷却生效 | 过期自动清理
- 预期: 精简展示避免信息过载，一键跳转降低操作成本，冷却机制防止打扰

### 5.3 装备推荐逻辑
- 操作: 武将装备面板→军师推荐分析武将四维属性短板→识别最弱属性(如防御最低)→筛选背包中对应部位装备→排序: 品质优先>属性增量优先>等级匹配优先 → 每槽位推荐最多2件→推荐装备标记「推荐」徽章→点击推荐装备查看属性对比 → 一键穿戴整合推荐结果(见4.1)
- 验证: 属性短板分析正确 | 推荐排序逻辑正确 | 每槽位最多2件限制生效 | 推荐徽章显示正确
- 预期: 装备推荐解决"不知道穿什么"的问题，属性短板分析引导均衡发展

### 15.3 军师建议置信度视觉样式（补充TUT-2）
