# v16.0 传承有序 — 玩游戏流程

> 核心循环: 传承系统积累声望→转生重置获得倍率加成→加速重建推进更快→存档系统保障数据安全→设置系统个性化配置→跨周目传承无缝衔接

---

## 1. 传承系统

### 1.1 声望积累与传承点
- 操作: 主界面点击"传承"入口 → 传承面板展示当前声望等级/传承点/历史转生次数 → 声望获取途径: 通关关卡(基础声望)/完成成就(额外声望)/限时活动(活动声望)/NPC好感度里程碑(社交声望) → 声望等级每提升1级获得传承点×1 → 传承点用于解锁转生后永久加成(资源产出+5%/建筑速度+3%/战斗经验+8%等) → 传承面板可视化展示: 声望进度条+传承点分配树+已解锁加成列表
- 验证: 声望获取途径数值与各系统产出一致 | 声望等级提升时传承点精确到账 | 传承点分配后加成百分比正确生效 | 传承面板数据与后端存储一致
- 预期: 声望系统为长线养成提供持续追求，传承点让每次转生都有永久成长感

### 1.2 传承奖励与跨周目继承
- 操作: 转生确认后弹出传承奖励结算面板 → 展示本次周目获得: 声望等级/传承点/专属称号/历史战绩摘要 → 永久继承内容: 武将图鉴解锁状态/成就进度/皮肤/称号/传承点加成 → 重置内容: 关卡进度/资源/建筑等级/兵力 → 继承确认弹窗逐项列出保留项与重置项 → 确认后执行转生，播放传承CG动画 → 转生完成进入新周目初始界面
- 验证: 永久继承内容在新周目中完整保留 | 重置内容确实归零 | 结算面板数据与实际一致 | CG动画播放流畅不卡顿
- 预期: 传承奖励给予周目完成的成就感，永久继承让重玩不浪费，重置内容保持新周目挑战性

## 2. 转生机制

### 2.1 转生条件与倍率计算
- 操作: 传承面板点击"转生"按钮 → 系统检测转生条件: 声望等级≥30且通关当前周目最终关卡 → 条件不满足时灰显按钮并提示缺失条件 → 条件满足后弹出转生预览: 基础倍率(1.0x)+声望加成倍率(每10级声望+0.2x)+转生次数加成(每次+0.1x，上限5.0x) → 展示倍率对比表(当前倍率vs转生后倍率) → 收益模拟器: 输入预计游玩时长→预测声望增长曲线→标注边际收益递减拐点=推荐转生时机
- 验证: 转生条件检测准确无遗漏 | 倍率计算公式正确(基础+声望+次数) | 收益模拟器预测与实际误差≤10% | 倍率上限5.0x正确生效
- 预期: 转生条件确保玩家有足够积累，倍率系统激励多次转生，收益模拟器帮助玩家做出最优决策

### 2.2 转生次数解锁与加速重建
- 操作: 转生次数累计面板展示已转生次数及解锁内容 → 次数解锁: 1次→天命系统/2次→专属科技路线/3次→神话武将招募池/5次→跨服竞技场 → 转生后加速机制: 初始资源赠送(粮草×5000+铜钱×3000)/建筑1~10级升级时间÷倍率(几乎瞬间)/已通关关卡可直接扫荡 → 一键重建: 按上次周目建筑优先级自动升级→确认后批量执行→加速材料消耗明细展示
- 验证: 次数解锁内容在对应转生次数后正确开放 | 初始资源赠送数量准确 | 低级建筑加速时间与倍率一致 | 一键重建顺序与上次周目优先级匹配
- 预期: 次数解锁提供渐进式内容发现，加速机制消除重玩痛苦，一键重建让老玩家快速回到有趣阶段

## 3. 存档系统

### 3.1 多存档槽位与快照
- 操作: 主界面点击"存档"入口 → 存档管理面板展示3个槽位(默认1个，VIP3解锁第2槽，VIP7解锁第3槽) → 每槽显示: 存档名/创建时间/最后保存时间/当前关卡/战力/武将数 → 创建存档: 选择空槽位→输入名称(2~8字符)→确认创建 → 删除存档: 二次确认弹窗(输入"DELETE")→清除该槽数据 → 手动快照: 暂停菜单→"保存游戏"→选择快照槽(最多3个)→写入含时间戳快照 → 快照回滚: 选择快照→确认→恢复至快照节点(扣除元宝×50)
- 验证: 槽位解锁条件与VIP等级匹配 | 各槽位数据完全隔离 | 快照回滚后数据精确恢复 | 删除需二次确认且不可恢复
- 预期: 多存档支持多路线体验，快照提供关键节点保护，回滚机制给予纠错机会

### 3.2 云存档同步与冲突解决
- 操作: 设置→"云存档"标签→登录账号后解锁 → 上传: 选择本地槽位→压缩+加密+传输→完成显示云端时间戳 → 下载: 展示云端列表→选择目标→冲突检测(对比时间戳) → 冲突解决面板: 展示本地vs云端(时间/关卡/战力/资源/武将数)→选择保留一方→确认执行 → 跨设备恢复: 新设备登录→自动检测云端存档→确认下载→进度与原设备一致 → 版本兼容: 低版本可向下加载/高版本提示"请更新游戏"
- 验证: 上传后MD5校验一致 | 冲突检测准确识别差异 | 跨设备恢复数据完整 | 网络异常时传输中断不损坏存档
- 预期: 云存档实现跨设备传承，冲突解决避免数据丢失，版本兼容平滑过渡

## 4. 设置系统

### 4.1 账号管理与系统设置
- 操作: 主界面点击齿轮图标→设置面板 → 账号管理: 游客登录(设备自动生成ID)/手机号注册(验证码60s冷却)/第三方授权(微信/QQ/Apple ID)→绑定设备(最多3台)→切换账号(登出→确认云存档上传→登录新账号→拉取存档) → 系统设置: 音频控制(背景音乐/音效/语音各0~100滑块)/画质选择(低/中/高/极致四档影响帧率30/45/60/120fps)/语言切换(简中/繁中/英语/日语，重启生效)/通知开关(推送/限时事件/活动倒计时) → 所有设置即时保存至本地存储
- 验证: 多种登录方式流程正确 | 设备绑定不超过3台 | 音量滑块实时反馈 | 画质切换后帧率与配置一致 | 语言重启后文本全部更新
- 预期: 多登录方式降低门槛，设备绑定防滥用，系统设置提供个性化体验

### 4.2 数据导出导入与重置
- 操作: 设置→"数据管理"标签 → 导出: 选择范围(全量/仅存档/仅设置)→生成加密.sgd文件至本地目录→支持系统分享 → 导入: 选择.sgd文件→解析版本号+校验码→展示导入预览→确认覆盖槽位→重启生效 → 重置: 选择范围(仅清空资源/仅清空武将/恢复出厂)→二次确认(输入"RESET")→执行重置→保留账号绑定→自动重启
- 验证: 导出文件加密不可篡改 | 导入校验失败时明确提示原因 | 重置仅清除选定范围 | 恢复出厂仅保留账号信息
- 预期: 导出提供本地备份，导入恢复玩家数据，重置给予重新开始机会，多重确认防误操作

## 5. 交叉验证

### 5.1 传承系统→转生机制→加速重建全链路
- 操作: 玩家声望等级达到35→点击"转生"→收益模拟器推荐"立即转生"(倍率2.8x)→确认转生→传承奖励结算获得传承点×3→永久加成解锁(资源产出+15%)→转生后获得初始资源(粮草×5000+铜钱×3000)→一键重建执行→建筑快速升至10级→开始新周目→验证声望获取速度较首次周目提升约2.8倍
- 预期: 声望→转生→传承→加速→新周目形成完整循环，倍率加成切实提升新周目体验，传承点永久成长感明确

### 5.2 存档系统→云存档→设置系统跨设备闭环
- 操作: 设备A游玩至声望等级20→手动创建快照→上传云存档→调整系统设置(音量70/画质高/语言繁中)→设置随存档保存→设备B登录同一账号→下载云存档→验证游戏进度与设备A一致→系统设置同步(音量70/画质高/语言繁中)→设备B修改设置(音量30)→自动存档→上传云存档→设备A下载→设置同步为音量30
- 预期: 存档与设置跨设备同步一致，云存档上传/下载链路数据完整，双向同步最终一致

### 5.3 转生→存档快照→数据导出安全网
- 操作: 玩家槽位1声望等级40→手动创建快照→执行转生→新周目游玩至声望等级15→发现想尝试不同路线→导出当前存档为.sgd备份→使用快照回滚至转生前(声望40)→验证回滚后武将/装备/资源与快照一致→重新上传云存档覆盖→确认云端已更新→重新转生选择不同传承点分配方案
- 预期: 快照回滚作为转生决策的安全网，导出备份提供额外保障，回滚精确恢复节点数据，云存档最终与本地一致
