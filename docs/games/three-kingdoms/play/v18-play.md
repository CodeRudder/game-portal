# v18.0 新手引导 — 玩游戏流程

## 1. 首次启动检测
- 首次打开游戏 → 检测新用户 → 启动引导流程
- 验证: 检测准确、老用户跳过

## 2. 引导步骤
- 按步骤引导(建造→招募→战斗→升级) → 高亮目标 → 完成验证
- 验证: 步骤顺序正确、高亮位置准确、不可跳过核心步骤

## 3. 引导遮罩
- 遮罩覆盖 → 仅露出引导目标 → 点击引导区域
- 验证: 遮罩层级正确、穿透点击防止

## 4. 引导存储
- 中途退出 → 重新进入 → 从断点继续
- 验证: 进度保存、断点恢复

## 5. 剧情播放
- 引导中穿插剧情 → 播放动画 → 选择分支
- 验证: 剧情连贯、播放流畅
