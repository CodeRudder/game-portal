# v18.0 新手引导 — 玩游戏流程

> 核心循环: 引导状态机驱动6步核心+6步扩展引导流程→步骤高亮+遮罩聚焦引导玩家操作→8段剧情事件穿插增强沉浸感→教学关卡实战验证→提示系统辅助自由探索→跳过/重玩机制兼顾老玩家与回顾需求

---

## 1. 新手引导系统

### 1.1 引导状态机与6步核心引导
- 操作: 首次启动游戏 → 检测无引导记录 → 状态机进入「核心引导中」 → 步骤1-1主城概览(高亮主城建筑+气泡说明"这是您的主城") → 步骤1-2建造农田(高亮空地→点击→确认建造) → 步骤1-3招募武将(高亮酒馆→进入→点击招募) → 步骤1-4首次出征(高亮战役入口→选择关卡→完成战斗) → 步骤1-5查看资源(高亮资源栏→逐项说明粮草/铜钱/元宝) → 步骤1-6科技研究(高亮书院→选择科技→开始研究) → 步骤6完成发放「初出茅庐」礼包 → 状态机切换至「自由探索过渡」
- 验证: 首次用户自动触发引导 | 老用户(有引导记录)直接进入自由游戏 | 6步顺序严格线性不可跳步 | 每步完成条件明确(如农田建造完成才算步骤通过) | 礼包发放时机准确
- 预期: 新玩家在5分钟内完成核心引导，理解游戏基本循环(建造→招募→战斗→升级)

### 1.2 引导遮罩与高亮聚焦
- 操作: 引导激活时 → 全屏半透明遮罩(rgba(0,0,0,0.75))覆盖 → 仅引导目标元素可见(裁剪洞口) → 洞口边缘发光描边(#FFD700 2px) → 气泡提示框定位在目标旁(自动避开屏幕边缘) → 点击遮罩区域无响应(防止穿透) → 点击高亮目标完成当前步骤 → 遮罩平滑过渡到下一步骤目标(300ms渐变)
- 验证: 遮罩完全覆盖非引导区域无视觉缝隙 | 洞口精确对齐目标元素(误差≤2px) | 点击遮罩区域无任何响应 | 气泡文字清晰可读不遮挡目标 | 步骤间过渡动画流畅无闪烁
- 预期: 玩家视觉焦点精准聚焦到引导目标，操作路径唯一明确，不会被其他元素干扰

### 1.3 6步扩展引导与阶段奖励
- 操作: 自由探索阶段 → 触发扩展引导(状态机「Mini-tutorial」) → 步骤7军师建议 → 步骤8半自动战斗 → 步骤9借将系统 → 步骤10背包管理 → 步骤11科技分支 → 步骤12联盟系统 → 步骤12完成发放「新手毕业」称号 → 中间每步完成发放小额奖励(铜钱×200/经验丹×1)
- 验证: 扩展引导在自由探索阶段非强制触发 | 各步骤独立可跳过 | 阶段奖励发放准确无遗漏 | 称号正确装备到玩家头像框
- 预期: 扩展引导覆盖进阶玩法，玩家完成全部12步后具备独立游玩能力

## 2. 教学关卡

### 2.1 首次战斗教学
- 操作: 核心引导步骤4触发 → 进入教学关卡「黄巾之乱·初战」 → 敌方配置3个弱化黄巾兵(HP降至30%) → 强制半自动战斗模式 → 高亮攻击按钮→提示"点击攻击" → 高亮技能按钮→提示"释放武将技能" → 战斗胜利→播放胜利动画→发放首战奖励(铜钱×500+经验×100) → 引导继续下一步
- 验证: 教学关卡难度极低确保必胜 | 半自动模式玩家仅需点击关键操作 | 胜利后奖励正确到账 | 战斗时长控制在30-60秒
- 预期: 玩家通过一场必胜战斗理解战斗基本操作，建立游戏信心

### 2.2 8段剧情事件
- 操作: 引导流程中按条件触发剧情 → E1桃园结义(首次进入) → E2黄巾之乱(首次战斗后) → E3三顾茅庐(首次招募) → E4草船借箭(首次科技研究) → E5赤壁之战(主城Lv5) → E6单刀赴会(首次借将) → E7七擒孟获(联盟系统解锁) → E8三国归一(新手毕业) → 剧情播放: 水墨风立绘+打字机效果(30ms/字)+点击推进+5秒自动播放+跳过按钮
- 验证: 8段剧情触发条件准确 | 打字机速度均匀 | 跳过按钮可用 | 自动播放计时准确 | 剧情播放中不可操作其他功能
- 预期: 剧情事件增强三国沉浸感，玩家在关键节点获得情感共鸣

## 3. 提示系统

### 3.1 上下文提示与军师建议
- 操作: 自由探索阶段 → 玩家闲置>15秒 → 军师头像浮现+气泡"主公，不妨试试升级农田" → 首次打开新面板 → 显示面板功能概述提示(3秒后自动消失) → 资源不足时 → 高亮获取途径(如铜钱不足→高亮征收按钮) → 可升级建筑时 → 建筑上方显示绿色箭头提示 → 点击提示→引导跳转到对应功能
- 验证: 闲置提示时机准确不频繁(同类型提示间隔≥60秒) | 面板概述文字简洁(≤30字) | 资源不足提示指向正确获取途径 | 绿色箭头仅在确实可升级时显示
- 预期: 提示系统在玩家迷茫时提供方向，不干扰已有目标的玩家

### 3.2 引导进度存储与跨设备同步
- 操作: 每完成一个引导步骤 → 实时写入localStorage(tutorial_progress: {step, timestamp}) → 登录账号后 → 引导进度绑定账号 → 换设备登录 → 拉取云端进度 → 冲突解决: 取completed_steps并集(取最大进度) → 中途退出 → 重新进入 → 从断点步骤继续(不重复已完成步骤)
- 验证: 步骤完成即时保存(刷新页面不丢失) | 跨设备同步进度取最大值 | 断点恢复位置准确 | 已完成的步骤不重复触发
- 预期: 玩家在任何设备任何时候退出都能无缝继续引导

## 4. 引导跳过与重置

### 4.1 加速与跳过机制
- 操作: 引导对话中 → 点击加速(打字机立即显示全文) → 剧情播放中 → 点击跳过按钮 → 二次确认弹窗"跳过将无法获得剧情沉浸体验" → 确认后水墨晕染过渡(500ms)跳至剧情结尾 → 动画播放中 → 加速按钮(×3倍速) → 单段引导>60秒 → 显示"一键完成"按钮 → 不可跳过内容: 步骤1-1主城概览/步骤2-3确认建造/步骤4-4首次战斗/所有引导点击操作
- 验证: 对话加速立即生效 | 剧情跳过需二次确认 | 水墨过渡动画流畅 | 一键完成仅对>60s步骤显示 | 不可跳过步骤无跳过按钮
- 预期: 老玩家或重玩玩家可快速通过引导，核心教学步骤强制完成确保理解

### 4.2 引导重玩与自由探索过渡
- 操作: 设置面板 → 引导回顾入口 → 选择重玩步骤(观看模式) → 简化遮罩(不阻断操作) → 每日重玩奖励: 铜钱×100(限3次/天) → 核心引导步骤6完成 → 自由探索过渡: 显示3个推荐行动(升级建筑/探索战役/招募更多武将)+已解锁功能列表 → 点击推荐行动→跳转对应功能
- 验证: 重玩模式不影响当前游戏进度 | 简化遮罩可随时关闭 | 每日奖励计数准确(3次上限) | 自由探索过渡推荐行动跳转正确
- 预期: 玩家可随时回顾教学内容，过渡阶段自然引导进入自由游戏

## 5. 交叉验证

### 5.1 新手完整流程: 首次启动→核心引导→教学战斗→自由探索
- 操作: 全新账号首次启动 → E1桃园结义剧情播放 → 步骤1主城概览(遮罩+高亮) → 步骤2建造农田(点击空地→确认) → 步骤3招募武将(进入酒馆→点击招募) → E2黄巾之乱剧情 → 步骤4首次出征→教学关卡→半自动战斗→胜利 → 步骤5查看资源 → 步骤6科技研究 → 「初出茅庐」礼包 → 自由探索过渡(3个推荐行动) → 扩展引导逐步触发 → E8三国归一 → 「新手毕业」称号
- 预期: 新玩家从零到独立游玩全流程顺畅，每一步有明确引导和验证，15分钟内完成核心引导

### 5.2 老玩家跳过+重玩+跨设备同步闭环
- 操作: 老玩家登录(已完成引导) → 直接进入自由游戏无引导干扰 → 设置→引导回顾→重玩步骤4首次战斗(观看模式) → 完成获得铜钱×100(1/3) → 退出游戏 → 手机端登录同账号 → 引导进度同步(已完成) → 无引导弹出 → 设置中回顾进度与PC端一致
- 预期: 老玩家不受引导打扰，重玩功能可选可用，跨设备进度同步准确

### 5.3 遮罩高亮→剧情播放→提示系统→跳过重置全链路
- 操作: 新账号启动 → 核心引导步骤2(遮罩+高亮农田空地) → 点击建造 → E3三顾茅庐剧情播放 → 点击加速显示全文 → 点击跳过→二次确认→水墨过渡跳过 → 完成步骤6 → 自由探索 → 闲置15秒 → 军师提示浮现 → 点击提示跳转到推荐功能 → 设置→重置引导 → 重新从步骤1开始 → 中途退出 → 重新进入 → 断点恢复
- 预期: 引导系统四大模块(遮罩/剧情/提示/跳过)协同工作无冲突，断点恢复可靠
