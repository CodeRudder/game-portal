## 🆕 〇、数据完整性验证

### DATA-FLOW-1: 武将数据完整性验证
**目标**: 验证所有14个武将的四维属性/品质/技能/阵营数据非空且在合理范围内

**触发条件**: 游戏启动时自动执行，或手动触发 `window.__GM__.validateGenerals()`

| 步骤 | 操作 | 验证条件 | 预期结果 |
|:----:|------|----------|----------|
| 1 | 游戏启动，加载 GeneralData.ts | 武将数据加载完成 | 无JS错误 |
| 2 | 遍历全部14个武将 | `engine.getGenerals().length === 14` | 武将数量=14 |
| 3 | 验证每个武将四维属性非空 | ATK/INT/CMD/POL 均为正整数 | 每个属性 > 0 |
| 4 | 验证四维属性在合理范围 | ATK/INT/CMD/POL ∈ [40, 120] | 符合PRD HER-1品质属性范围 |
| 5 | 验证品质数据非空 | quality ∈ {Uncommon, Rare, Epic, Legendary, Mythic} | 品质枚举合法 |
| 6 | 验证阵营数据非空 | faction ∈ {蜀, 魏, 吴, 群} | 阵营枚举合法 |
| 7 | 验证技能数据非空 | 每个武将至少有1个主动技能+1个被动技能 | 技能列表非空 |
| 8 | 验证品质与属性范围匹配 | Uncommon: 60~75 / Rare: 70~85 / Epic: 80~95 / Legendary: 90~105 / Mythic: 100~120 | 属性范围符合品质定义 |
| 9 | 验证阵营覆盖 | 4个阵营均有至少1个武将 | 无空阵营 |
| 10 | 验证品质覆盖 | 5个品质均有至少1个武将 | 无空品质 |

**异常处理**:

| 异常场景 | 处理方式 | 错误级别 |
|---------|---------|---------|
| 武将数量 ≠ 14 | 控制台 ERROR + Toast "武将数据不完整(N/14)" | 🔴 阻断 |
| 四维属性为 null/undefined/0 | 控制台 ERROR "武将{ID}属性缺失" + 跳过该武将 | 🔴 阻断 |
| 属性超出合理范围 | 控制台 WARN "武将{ID}属性异常: ATK={val}" + 使用默认值 | 🟡 警告 |
| 品质/阵营枚举非法 | 控制台 ERROR "武将{ID}品质/阵营非法" + 跳过该武将 | 🔴 阻断 |
| 技能列表为空 | 控制台 WARN "武将{ID}无技能定义" + 赋予默认普攻 | 🟡 警告 |
| 品质/阵营覆盖缺失 | 控制台 WARN "品质/阵营{X}无武将" | 🟡 警告 |

**GM调试命令**:

| 命令 | 功能 | 示例 |
|------|------|------|
| `validateGenerals()` | 执行完整数据验证 | `window.__GM__.validateGenerals()` |
| `getGeneralReport()` | 输出14个武将属性报告 | `window.__GM__.getGeneralReport()` |

**目标**: 验证战力计算公式正确性

| 步骤 | 操作 | 验证条件 | 预期结果 |
|:----:|------|----------|----------|

| 2 | 记录等级和星级 | level, star | — |
| 3 | 手动计算战力 | (ATK×2.0 + INT×2.0 + CMD×1.5 + POL×1.0) × (1+等级×0.05) × 品质系数 × 星级系数 | 记录 `power_calc` |
| 4 | 对比游戏显示战力 | `power_display == power_calc` | 误差<0.01 |
| 5 | 升级武将1级 | 重新计算 | 战力增量正确 |

| 7 | 检查装备系数（如有装备） | 战力含装备加成 | 装备系数 = 1 + Σ装备战力/1000 |

**战力公式**（PRD HER-1-3）:
```
单将战力 = (ATK × 2.0 + INT × 2.0 + CMD × 1.5 + POL × 1.0) × 等级系数 × 品质系数 × 星级系数 × 装备系数

等级系数 = 1 + 等级 × 0.05
品质系数 = QUALITY_MULTIPLIERS[品质]  (普通1.0 / 精良1.15 / 稀有1.3 / 史诗1.5 / 传说1.8)
星级系数 = getStarMultiplier(星级)    (1★1.0 / 2★1.15 / 3★1.35 / 4★1.6 / 5★2.0 / 6★2.5)
装备系数 = 1 + Σ 装备战力 / 1000
```
