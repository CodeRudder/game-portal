# v15.0 事件风云 — 玩游戏流程

## 1. 随机事件
- 游戏中触发随机事件 → 选择选项 → 获得奖励/惩罚
- 验证: 事件触发条件、选项效果、结果展示

## 2. 连锁事件
- 完成前置事件 → 触发连锁事件 → 按顺序完成 → 获得链奖励
- 验证: 链式触发、顺序正确、最终奖励

## 3. 故事事件
- 触发剧情事件 → 观看剧情 → 做出选择 → 影响后续
- 验证: 剧情连贯、选择分支、影响追踪

## 4. 离线事件
- 离线期间积累事件 → 上线查看 → 批量处理
- 验证: 离线事件积累上限、批量处理效率

## 5. 事件通知
- 事件触发 → UI通知弹窗 → 点击查看详情
- 验证: 通知及时、不遗漏、可回看
