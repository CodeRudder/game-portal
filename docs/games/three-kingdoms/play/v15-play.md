# v15.0 事件风云 — 玩游戏流程

> 核心循环: 挂机游玩→事件触发→做出选择→获得奖励/应对危机→影响资源与声望→连锁后续→活动代币累积→商店兑换

---

## 1. 事件系统

### 1.1 事件触发引擎
- 操作: 游戏运行中触发引擎每分钟tick检测 → 三大触发机制并行判定: 时间触发(限时机遇10:00/14:00/20:00整点触发)、条件触发(NPC好感度达阈值/关卡通关/声望等级提升)、概率触发(随机遭遇P=0.03×(1+等级×0.02)×时间衰减系数) → 冷却检查: 同类型60min/同事件4h/新手保护30min/连续2次负面后下次必正面 → 通过冷却后生成事件实例 → 按优先级排队(限时>天灾>剧情>随机>NPC>里程碑) → 急报横幅弹出(最多缓存5条,3秒自动收起) → 玩家点击横幅打开事件详情弹窗
- 验证: 概率触发公式计算精确(基础概率×等级系数×时间衰减) | 冷却计时器从事件处理完成起算不提前重置 | 优先级排队严格按6级排序不插队 | 急报横幅缓存上限5条超出后丢弃最低优先级 | 新手保护30min内无天灾人祸事件 | 连续负面保护计数器跨类型累计
- 预期: 触发引擎构建动态游戏世界节奏，冷却机制避免事件疲劳，优先级确保紧急事件不被淹没

### 1.2 事件选项与后果
- 操作: 事件详情弹窗展示事件标题+描述文本+2~3个选项按钮 → 每个选项旁标注后果颜色(🟢正面/🔴负面/🟡中性) → 玩家点击选项 → 系统计算后果: 资源变动(铜钱/粮草/兵力/元宝)、声望值变化(±50~500)、NPC好感度变化(±10~100) → 部分选项含概率后果(如"有70%概率获得稀有道具,30%概率损失兵力") → 按概率随机选择结果 → 展示结果动画+具体获得/损失数值 → 资源/声望/好感度即时更新 → 事件写入日志(最近100条可回看)
- 验证: 后果标注颜色与实际结果一致(绿色选项不会产生净负面结果) | 资源变动数值精确写入各子系统 | 声望值变化触发声望等级升降判定 | NPC好感度变化触发好感度等级变化 | 概率后果按配置概率随机(100次抽样偏差≤5%) | 日志记录完整可追溯
- 预期: 选项系统赋予玩家决策权，后果标注降低盲目选择焦虑，概率后果增加惊喜感

### 1.3 事件通知与日志
- 操作: 事件触发时急报横幅从顶部滑入 → 横幅显示事件图标+标题+优先级颜色(限时金色/天灾红色/剧情紫色/随机蓝色/NPC绿色) → 3秒未点击自动收起至右侧事件队列图标 → 队列图标显示待处理事件数量红点 → 点击队列图标展开待处理列表 → 主界面"事件日志"入口查看历史事件(按时间倒序,展示事件名+选择+结果摘要) → 连锁事件在日志中显示链进度标签
- 验证: 横幅动画流畅不卡顿 | 优先级颜色与配置一致 | 红点数量与待处理事件数精确匹配 | 日志记录最近100条不丢失 | 连锁事件链标签正确关联链ID
- 预期: 通知系统确保玩家不遗漏重要事件，日志回看满足复盘需求

## 2. 随机事件

### 2.1 随机遭遇事件池
- 操作: 挂机/战斗/探索过程中触发随机遭遇 → 从事件池按权重抽取事件 → 4子类型轮换: 奇遇(发现宝物/遇到隐士/获得秘籍,纯正面奖励)、天灾(洪水/旱灾/蝗灾,资源损失考验)、人祸(山贼袭击/流民涌入/官员贪腐,需选择应对策略)、商队(商人来访/丝绸之路/黑市交易,资源交换机会) → 每个子类型5+事件模板共20+ → 事件弹窗展示对应场景插图+事件描述+选项
- 验证: 事件池20+模板全部可触发无死模板 | 4子类型按配置权重分布(偏差≤3%) | 奇遇事件纯正面无隐藏惩罚 | 天灾/人祸事件必有减少损失的防御选项 | 商队事件交换比率与配置一致 | 事件描述文本无错字无重复
- 预期: 20+事件模板×多选项分支提供60+种体验，4子类型覆盖正负中性保持新鲜感

### 2.2 天灾人祸与限时机遇
- 操作: 天灾人祸按概率P=0.015×(1+关卡难度×0.01)触发 → 弹窗提示灾害类型+预估损失 → 玩家选择: 损失防御(消耗兵力减少50%损失)/全力抵抗(消耗更多资源但损失仅20%)/听天由命(不消耗但承受100%损失) → 限时机遇在10:00/14:00/20:00整点触发 → 急报横幅金色闪烁+推送通知 → 30~60分钟窗口内点击参与 → 错过窗口事件消失不补偿 → 限时机遇奖励为普通事件2~3倍
- 验证: 天灾概率公式计算正确 | 三种应对策略损失比例精确(50%/20%/100%) | 限时机遇仅在配置时段触发 | 窗口倒计时精确到秒 | 窗口过期后事件彻底移除不可恢复 | 限时奖励倍率与配置匹配
- 预期: 天灾考验防御准备度增加策略深度，限时机遇创造紧迫感驱动定时回归

## 3. 链式事件

### 3.1 连锁事件引擎
- 操作: 完成前置事件并选择特定选项 → 系统检查后续事件ID+分支条件 → 满足条件则延迟触发后续事件(冷却时间后) → 连锁事件追踪面板可视化展示: 链名称+节点路径图(已完成节点亮色/当前节点脉冲/未到达节点灰暗)+历史选择记录 → 每个节点仍是标准事件(描述+2~3选项) → 不同选择导向不同后续分支 → 链完成(到达终节点)后发放最终奖励+链状态归档 → 链进度持久化存储(关闭游戏不丢失)
- 验证: 前序选择正确映射到后续分支节点 | 节点路径图与实际链数据结构一致 | 链状态持久化后重新加载无数据丢失 | 每条链最多5个节点不超限 | 分支条件判定精确(资源阈值/声望等级/好感度等) | 最终奖励仅在链完成时发放一次
- 预期: 连锁事件提供"蝴蝶效应"体验，选择产生长期影响，路径可视化增强叙事沉浸感

### 3.2 连锁事件数据与分支
- 操作: 连锁事件数据结构包含: 链ID/当前节点ID/历史选择列表/链状态(进行中/已完成/已放弃) → 每个节点数据: 事件模板+后续节点映射表(选项A→节点X, 选项B→节点Y)+分支条件(如"声望≥500才能选A") → 不满足条件的选项灰显不可点击并提示原因 → 链进行中可在追踪面板查看全链预览 → 多条链可并行进行互不干扰 → 链完成时追踪面板展示完整路径回顾+所有选择汇总
- 验证: 分支条件不满足时选项灰显且不可点击 | 灰显提示文本准确说明所需条件 | 多链并行时各链状态独立不互相覆盖 | 路径回顾与实际选择历史完全一致 | 已完成链不可重复触发
- 预期: 分支条件增加策略考量(平时积累声望/好感度解锁更好选项)，多链并行提供丰富叙事线

## 4. 历史事件

### 4.1 剧情事件与NPC事件
- 操作: 剧情事件在指定关卡通关后自动触发(如第3章通关触发"三顾茅庐"事件) → 一次性触发不可重复 → 展示剧情动画+对话文本+历史背景介绍 → 玩家做出剧情选择影响主线走向 → NPC事件在特定NPC好感度达阈值时触发(如诸葛亮好感度达"信赖"触发"隆中对"事件) → 按NPC日程触发(特定NPC在特定时段出现在特定地点) → NPC事件解锁新社交内容(隐藏对话/特殊任务/独家交易)
- 验证: 剧情事件仅在指定关卡通关后触发且仅触发一次 | 关卡回放不重复触发剧情事件 | NPC好感度阈值与配置精确匹配 | NPC日程触发时段判定正确 | 解锁的社交内容在对应系统中正确开放 | 剧情选择影响后续关卡对话或NPC态度
- 预期: 剧情事件推进主线增强代入感，NPC事件深化社交系统，历史背景丰富三国世界观

### 4.2 离线事件处理
- 操作: 玩家离线期间事件持续触发并堆积 → 上线后弹出"离线事件报告"弹窗 → 报告分类展示: 正面事件(系统自动选择保守选项,获得50%奖励)/负面事件(按防御值自动减免损失,减免比例=防御值/防御值+100)/限时机遇(错过即消失,不展示)/剧情事件(暂停等待上线处理) → 弹窗展示每条事件摘要+自动处理结果 → 玩家确认后资源/声望一次性结算 → 剧情事件保留在待处理队列等待手动处理
- 验证: 正面事件自动奖励精确为手动选择奖励的50% | 负面事件减免公式计算正确 | 限时机遇离线期间触发的全部标记为错过 | 剧情事件离线不触发保留至上限 | 离线事件堆积上限20条超出后丢弃最低优先级 | 一次性结算后各子系统数据一致
- 预期: 离线处理保证放置游戏体验连续性，保守策略避免玩家离线损失过大，剧情保留尊重叙事体验

## 5. 交叉验证

### 5.1 事件→资源→声望→连锁分支全链路
- 操作: 玩家挂机触发随机遭遇"商队来访" → 选择"以优惠价格购买"选项 → 消耗铜钱×2000获得粮草×5000+声望+100 → 声望值累积触发声望等级提升 → 解锁新NPC事件(商人好感度达"友善") → NPC事件触发后解锁特殊交易选项 → 该选择触发连锁事件"丝绸之路"链第1节点 → 选择"冒险深入"导向链第2节点(需声望≥500) → 声望满足条件 → 完成整条链获得最终奖励稀有武将碎片×10
- 预期: 随机事件→选项后果→声望变化→NPC解锁→连锁触发形成完整因果链，声望值在事件/NPC/连锁三系统中数值一致

### 5.2 天灾防御→资源损失→离线处理→限时机遇闭环
- 操作: 玩家主动防御天灾事件选择"全力抵抗" → 消耗兵力×500但损失仅20%(粮草×-300) → 下线后负面事件自动按防御值减免(防御值800,减免率=800/900≈89%) → 上线查看离线报告确认减免正确 → 14:00限时机遇触发但玩家未上线 → 上线后发现错过限时机遇 → 20:00限时机遇准时上线参与 → 获得3倍奖励补偿早前损失 → 资源恢复正增长
- 预期: 天灾防御策略影响资源健康度，离线减免公式与主动防御策略数值一致，限时机遇奖励足以补偿错过损失

### 5.3 剧情事件→NPC好感→连锁分支→图鉴收集闭环
- 操作: 玩家通关第5章触发剧情事件"赤壁之战" → 选择"联合孙权" → 获得周瑜好感度+50 → 周瑜好感度达"信赖"触发NPC事件"火攻之计" → 完成NPC事件解锁连锁事件链"三分天下" → 链中分支选项需要周瑜好感度≥"信赖"才可选择最优路线 → 选择最优路线完成链 → 获得限定武将"周瑜(赤壁)" → 武将图鉴更新收集进度 → 触发收集成就
- 预期: 剧情选择→NPC好感→连锁分支条件→图鉴收集形成多系统联动闭环，好感度在剧情/NPC/连锁三系统中数值一致无冲突
