# v15.0 事件风云 — 玩游戏流程

> 核心循环: 挂机产出→随机事件触发→做出选择→获得奖励/应对危机→连锁事件推进→限时活动冲刺→NPC奇遇解锁

---

## 1. 事件系统

### 1.1 随机遭遇事件池
- 操作: 挂机期间触发引擎每分钟tick → 按公式 P = 0.03 × (1 + 等级×0.02) × 时间衰减系数 计算触发概率 → 触发后从4子类型池(奇遇/天灾/人祸/商队)随机抽取事件模板 → 急报横幅弹出事件通知(6级优先级排队: 限时>天灾>剧情>随机>NPC>里程碑) → 最多缓存5条急报，3秒自动收起 → 玩家点击横幅打开事件详情弹窗(标题+描述+2~3选项+后果标注🟢/🔴/🟡) → 选择后计算资源变动+声望+好感度变化 → 概率后果按配置随机选择结果
- 验证: 20+事件模板覆盖4子类型且权重分配合理 | 概率公式计算结果与预期一致(±5%误差) | 急报横幅按优先级排序显示不乱序 | 冷却机制正确(同类型60min/同事件4h/新手保护30min) | 连续2次负面后下一次必定正面或中性 | 选择后资源/声望/好感度数值精确变化
- 预期: 随机事件为挂机体验注入不确定性和惊喜，冷却与保护机制避免负面体验过载，后果标注帮助玩家做出知情选择

### 1.2 剧情事件与天灾人祸
- 操作: 剧情事件: 通关指定关卡后自动触发→一次性不可重复→推进主线故事(展示剧情CG+对话)→完成后解锁新功能或区域 → 天灾人祸: 概率 P = 0.015 × (1 + 关卡难度×0.01) 触发→消耗资源/考验防御→玩家选择应对策略(加固/疏散/硬抗)→结果影响资源产出和民心 → 离线期间天灾人祸按防御值自动减免损失
- 验证: 剧情事件仅在指定关卡通关后触发且不可重复 | 天灾概率与关卡难度正相关 | 离线自动处理结果不超过手动最优选择的损失 | 剧情事件离线时暂停等待玩家上线
- 预期: 剧情事件推进叙事节奏，天灾人祸增加策略深度，离线自动处理保障放置体验不被惩罚

### 1.3 事件选项与后果
- 操作: 事件弹窗展示2~3个选项按钮 → 每个选项标注后果颜色(🟢正面/🔴负面/🟡中性) → 部分选项附带概率后果(如"70%获得资源/30%遭遇伏击") → 玩家选择后: 资源即时增减+声望值变化+NPC好感度变化 → 结果弹窗展示具体变动数值和获得物品 → 确认后关闭事件，变动写入玩家数据
- 验证: 后果标注颜色与实际结果一致 | 概率后果随机分布符合配置权重 | 资源变动精确(扣除不超过持有量) | 声望/好感度变化正确触发对应系统判定
- 预期: 选项系统赋予玩家决策权，后果标注降低选择焦虑，概率后果增加紧张感和重玩价值

## 2. 链式事件

### 2.1 连锁事件引擎
- 操作: 完成某个事件选项后触发后续事件(链ID关联) → 连锁事件追踪面板展示链路径可视化(当前节点+历史选择+分支预览) → 每个节点2~3个分支选项 → 分支条件判定确定后续节点(如选择"帮助"→善线/选择"拒绝"→恶线) → 链状态持久化存储(断线/离线不丢失进度) → 链完成时发放最终奖励+链状态归档 → 每条链最多5个节点，总量可控
- 验证: 链ID追踪准确，节点跳转无死链 | 分支条件判定与玩家选择一致 | 链状态持久化后重登可继续 | 链完成奖励按配置正确发放 | 节点数不超过5个
- 预期: 连锁事件提供"蝴蝶效应"体验，选择有长期影响，追踪面板帮助玩家理解因果，链状态持久化保障连续性

### 2.2 连锁事件数据结构与分支
- 操作: 事件数据结构包含: 事件ID/链ID/后续事件ID列表/分支条件/链状态字段 → 玩家选择后引擎读取后续事件ID → 匹配分支条件(资源阈值/声望等级/好感度/前置选择) → 确定唯一后续节点并触发 → 无匹配后续时链自然结束 → 链中途可暂停(玩家不立即响应)，下次登录继续
- 验证: 数据结构字段完整无遗漏 | 分支条件匹配逻辑精确 | 无匹配时链正常结束不报错 | 暂停后恢复时上下文完整
- 预期: 结构化数据支撑复杂分支叙事，条件匹配增加事件与游戏状态的关联性，暂停机制尊重玩家节奏

## 3. 限时事件

### 3.1 限时机遇事件
- 操作: 每日固定时段(10:00/14:00/20:00)触发限时机遇 → 触发后急报横幅最高优先级通知 → 事件详情弹窗显示倒计时(30~60min窗口) → 玩家在窗口内响应获得丰厚奖励(稀有资源/武将碎片/元宝) → 窗口关闭后事件消失不可补领 → 每日3次机会，错过仅损失单次不影响核心进度
- 验证: 触发时段精确(服务器时间±1min) | 倒计时准确且窗口关闭后事件确实消失 | 奖励丰厚程度显著高于普通事件 | 离线期间限时机遇错过不补偿
- 预期: 限时机遇创造紧迫感，鼓励玩家定时回归，丰厚奖励补偿时间投入，错过惩罚温和不挫败

### 3.2 限时活动与节日活动
- 操作: 限时活动: 3~7天周期→生成任务列表+里程碑+代币兑换商店(七阶体系按积分解锁阶层)→完成任务获代币+积分→代币兑换商品(每日刷新+限购)→排行榜每5min刷新(全服/阵营分组)→活动结束1h内结算排行奖励 → 节日活动: 7~14天+特殊玩法(春节放烟花/端午龙舟/中秋猜灯谜)+限定剧情事件+独占奖励(限定武将/皮肤) → 离线进度: 限时30%/赛季50%/日常100%/节日50%效率自动累积
- 验证: 活动周期起止时间精确 | 七阶商店阶层解锁与积分阈值匹配 | 排行榜5min刷新且结算奖励正确发放 | 离线进度按效率系数累积无溢出 | 节日活动特殊玩法可正常交互
- 预期: 限时活动提供周期性内容追求驱动日常活跃，七阶商店创造分层目标，排行榜激发竞争，节日活动丰富文化体验

### 3.3 签到系统
- 操作: 主界面点击"签到"入口 → 签到面板展示7天循环格子(铜钱→粮草→兵力→招贤榜→经验书→强化石→元宝) → 连续加成: 3天+10%/5天+20%/7天+50% → 补签: 元宝×50/次，每月最多3次 → 累计奖励里程碑: 14天/21天/28天 → 每日首次登录自动弹出签到提示
- 验证: 7天循环顺序与配置一致 | 连续加成百分比精确叠加到基础奖励 | 补签扣费正确且月次数不超限 | 累计里程碑奖励按档位发放 | 断签后连续天数重置但累计天数保留
- 预期: 签到系统提供每日回归动力，连续加成激励不断签，补签机制降低错过焦虑，累计奖励提供长线目标

## 4. NPC与奇遇

### 4.1 NPC事件系统
- 操作: NPC好感度达到阈值时触发NPC专属事件 → NPC日程系统匹配(特定NPC在特定时段出现) → 事件类型: 拜访/求助/邀请/密信 → 完成事件提升好感度+解锁社交内容(新对话/隐藏任务/特殊商品) → 好感度等级: 陌生→相识→友好→亲密→知己，每级解锁不同内容 → NPC事件不占用随机事件冷却槽
- 验证: 好感度阈值触发精确 | 日程匹配与服务器时间一致 | 好感度等级提升后内容正确解锁 | NPC事件与随机事件冷却独立互不干扰
- 预期: NPC事件丰富社交维度，好感度系统提供渐进式解锁体验，日程机制增加世界真实感

### 4.2 奇遇系统
- 操作: 在特定地图区域挂机时概率触发奇遇事件(低于随机事件频率) → 奇遇类型: 隐士传授(获得技能/属性加成)/宝物现世(获得稀有装备)/神兽现身(限时挑战获大量奖励)/古墓探秘(多步骤解谜获传说道具) → 奇遇事件品质: 普通/稀有/史诗/传说，高品质奇遇概率更低奖励更丰厚 → 奇遇事件可触发连锁事件链(如古墓探秘→机关破解→宝藏选择)
- 验证: 奇遇触发概率显著低于普通事件(约1/10) | 地图区域与奇遇类型关联正确 | 品质概率分布符合配置 | 奇遇连锁事件与主连锁系统共用引擎
- 预期: 奇遇系统提供"意外之喜"的高光时刻，低频高品质设计保持新鲜感，与连锁事件联动增加深度

## 5. 交叉验证

### 5.1 随机事件→连锁事件→限时活动全链路
- 操作: 玩家挂机触发随机事件"商队求助" → 选择"护送"触发连锁事件链(护送→遇袭→抉择→送达) → 链完成获得代币+积分 → 积分推进限时活动里程碑 → 里程碑解锁活动商店新阶层 → 代币兑换限定装备 → 装备提升战力 → 战力提升影响后续事件触发概率(等级系数增大)
- 预期: 事件→连锁→活动→战力→事件概率形成正反馈循环，数据流转一致无断点，战力提升确实影响触发公式

### 5.2 NPC好感度→奇遇→连锁事件闭环
- 操作: 玩家完成NPC事件提升好感度至"友好" → 解锁隐藏地图区域 → 在新区域挂机触发奇遇"隐士传授" → 奇遇触发连锁事件链"师门试炼"(3节点) → 链完成获得稀有技能书+NPC好感度额外提升 → 好感度达到"亲密"解锁NPC特殊商品 → 购买道具强化武将 → 武将战力提升影响天灾防御值 → 离线天灾减免比例提高
- 预期: NPC→奇遇→连锁→好感度→战力→离线防御形成完整闭环，好感度作为钥匙解锁内容，战力提升反哺事件应对能力

### 5.3 签到→限时机遇→活动商店→离线处理闭环
- 操作: 玩家连续签到7天获得元宝×50%加成 → 元宝用于补签或活动商店购买 → 每日10:00限时机遇响应获得代币 → 代币兑换活动商店商品 → 离线期间: 日常活动100%效率累积+限时活动30%累积 → 上线后领取离线积分+处理离线事件堆(正面保守50%/负面防御减免/限时消失) → 离线事件堆弹窗展示汇总报告 → 积分推进活动排行 → 排行结算获得最终奖励
- 预期: 签到→元宝→活动→离线→排行形成日常闭环，离线处理规则正确(正面保守/负面减免/限时消失)，积分累积效率系数精确
