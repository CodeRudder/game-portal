# v20.0 天下一统(下) — 玩游戏流程

> 核心循环: 终局系统解锁统一后新玩法 → 转生系统重置进度换取永久倍率 → 声望商城兑换稀有道具 → 永恒挑战无尽模式冲击排行榜 → 全系统联调端到端验证闭环

---

## 1. 终局系统

### 1.1 统一后新玩法解锁
- 操作: 完成第六章"一统天下"全部关卡 → 触发统一动画(水墨四角墨色遮罩800ms) → 解锁终局面板 → 查看解锁内容列表: 帝王称号、全武将招募池、跨服远征、专属建筑蓝图 → 验证主界面新增终局入口(金色龙纹图标) → 进入终局面板 → 查看推荐目标(声望Lv.20/转生×5/永恒挑战100层)
- 验证: 第六章通关后终局系统自动解锁 | 终局入口仅在统一后显示 | 解锁内容与转生次数联动(第1次转生解锁天命+第五章，第3次解锁神话武将池，第5次解锁跨服远征，第10次解锁帝王模式)
- 预期: 统一后玩家获得明确长线目标，终局内容逐步解锁驱动持续游玩

### 1.2 终局挑战
- 操作: 进入终局挑战面板 → 选择挑战类型(资源极限/战力巅峰/领土扩张) → 查看挑战目标与奖励 → 接受挑战 → 在限定条件内完成目标(如: 1小时内产出粮草100,000/单武将战力突破80,000/占领40块领土) → 领取终局奖励(声望×2000+稀有装备图纸+天命×500) → 挑战失败可重试(无惩罚)
- 验证: 挑战目标数值合理(需终局养成才能达成) | 奖励发放准确 | 失败后进度重置可重试 | 同类型挑战每日限完成3次
- 预期: 终局挑战为满级玩家提供短期追求目标，奖励驱动多系统深度养成

## 2. 转生系统

### 2.1 转生条件检查
- 操作: 打开声望转生面板 → 查看条件检查列表(6项) → 主城≥30级(✅/❌) + 通关赤壁之战全阶段(✅/❌) + 武将≥15位(✅/❌) + 总战力≥50,000(✅/❌) + 完成"初露锋芒"成就链5项(崭露头角/猛将如云/兵强马壮/谋定后动/威震四方)(✅/❌) + 转生冷却≥72小时(✅/❌) → 全部✅时"确认转生"按钮亮起(金色脉冲动画) → 点击查看收益模拟器(立即转生倍率 vs 再等2h/6h/12h/24h倍率对比+推荐时机)
- 验证: 6项条件全部满足才可转生 | 冷却时间首次无限制后续72小时 | 收益模拟器基于当前声望获取速度实时计算 | 推荐时机标注边际收益递减拐点
- 预期: 转生条件引导玩家深度参与全系统，收益模拟器帮助做出最优决策

### 2.2 转生执行与数据重置
- 操作: 确认转生 → 播放太极旋转+水墨晕染动画(1500ms) → 数据重置: 主城→Lv.1 | 建筑→Lv.1(蓝图保留) | 粮草/铜钱/兵符→0 | 领土→0块 | 天命保留30% | 武将等级衰减50%(最低Lv.1) | 技能等级衰减70% | 科技保留50%(未完成清零) → 保留项不变: 武将拥有权/升星/装备/好感度/成就/图鉴/好友/公会/NPC → 发放初始资源(粮草×5000+金币×3000) → 倍率生效(全产出×倍率/升级速度÷倍率/离线收益×倍率)
- 验证: 保留项100%不丢失 | 衰减项按比例精确计算(向下取整) | 重置项完全归零 | 初始资源即时到账 | 倍率公式: 1.5×(1+声望等级×0.05)×(1+(转生次数-1)×0.1)正确生效
- 预期: 转生后玩家以更高效率重新开始，保留项提供加速基础，倍率带来明显加速体感

### 2.3 转生后加速重建
- 操作: 转生完成 → 进入主城(Lv.1) → 建筑1~10级升级时间÷倍率(几乎瞬间完成) → 已通关关卡直接扫荡(三星记录保留) → 点击"一键重建"按钮 → 系统自动按上次优先级升级建筑至资源允许最高等级 → 完成转生专属任务3个(简单任务，奖励额外资源包) → 验证加速体感: 首次转生1.88×效率约2小时恢复到转生前30%进度
- 验证: 一键重建按历史优先级正确执行 | 扫荡功能保留三星记录 | 转生专属任务仅首次出现 | 倍率加速覆盖全资源产出/武将经验/建筑升级/科技研究/离线收益
- 预期: 转生后30分钟内恢复基础运转，2小时内明显超越上一轮同期进度

## 3. 声望商城

### 3.1 声望等级与兑换资格
- 操作: 打开声望面板 → 查看当前声望等级(进度条+百分比) → 查看等级阈值公式(1000×N^1.8) → 验证升级自动触发(金色横幅通知+粒子爆发动画) → 声望Lv.1解锁基础商品(扫荡令×5/200点、经验书中×1/300点) → Lv.5解锁声望商店(招贤榜×3/500点) → Lv.10解锁高级商品(求贤令×1/1000点、稀有装备图纸/2000~3000点) → Lv.20解锁终极商品(神话武将碎片/5000点)
- 验证: 声望等级阈值公式计算正确(Lv.1=1000/Lv.5=18946/Lv.10=63096/Lv.20=275423) | 等级不足时商品显示灰锁+解锁条件 | 升级通知不遗漏
- 预期: 声望等级形成清晰阶梯，每5级一个里程碑解锁新商品类别

### 3.2 声望兑换与特权生效
- 操作: 进入声望商店 → 选择商品"求贤令×1"(Lv.10解锁/1000声望点/每周限购2次) → 确认兑换 → 验证声望点扣除1000+求贤令×1到账 → 购买计数1/2 → 尝试第3次购买 → 提示"本周已达购买上限" → 查看特权加成: Lv.4商店折扣5%/Lv.17折扣10%(取最高) → 验证折扣后价格=原价×(1-折扣率)
- 验证: 声望点消耗准确 | 限购计数按周期重置(每日/每周/每月) | 折扣仅影响声望商店不影响其他商店 | 声望值不可消耗(只增不减)，消耗的是声望点数
- 预期: 兑换流程顺畅，限购机制防刷，折扣特权自动生效无需手动激活

## 4. 永恒挑战

### 4.1 无尽模式
- 操作: 进入永恒挑战入口(终局面板内) → 选择难度(普通/困难/地狱) → 开始挑战 → 每层生成随机敌人(战力逐层递增×1.15) → 击败敌人→获得永恒积分+声望+资源 → 每10层出现BOSS关(掉落稀有装备) → 失败记录最高层数 → 重新挑战从第1层开始 → 查看历史最高记录(按难度分别记录) → 验证层数奖励: 每10层额外奖励(声望×500+天命×100)
- 验证: 敌人战力按层线性增长(基础×1.15^层数) | 永恒积分累积不因失败重置 | BOSS掉落品质高于普通层 | 历史记录持久保存(转生不重置)
- 预期: 无尽模式提供无限挑战空间，永恒积分作为独立货币驱动终局养成

### 4.2 排行榜
- 操作: 打开永恒挑战排行榜 → 查看全服排名(按最高层数排序) → 查看自身排名+层数+战力 → 查看前10名玩家详情(阵容/武将/装备) → 切换难度排行榜(普通/困难/地狱独立排名) → 验证排名实时更新 → 赛季结算(每月1日) → 前100名发放赛季奖励(声望×5000+专属称号+天命×1000) → 赛季重置层数(保留永恒积分)
- 验证: 排名按最高层数降序排列 | 同层数按达成时间先后排序 | 赛季奖励按排名梯度发放 | 赛季重置仅清零层数不清零积分
- 预期: 排行榜驱动竞争动力，赛季制提供周期性追求，赛季奖励激励持续参与

## 5. 交叉验证

### 5.1 终局→转生→声望→永恒挑战联动链路
- 操作: 完成第六章统一 → 终局系统解锁 → 推进声望至Lv.10(声望点累积6000+) → 声望商店兑换求贤令×1 → 招募神话武将 → 战力提升至80,000+ → 满足转生6项条件 → 收益模拟器推荐"再等6小时" → 等待后转生(倍率2.48×) → 加速重建2小时恢复进度 → 永恒挑战困难模式推进至50层 → 排行榜查看排名上升 → 声望商城兑换稀有图纸 → 装备强化武将 → 再次推进永恒挑战
- 预期: 终局→声望→转生→永恒挑战形成完整闭环，每个系统产出驱动下一系统进度

### 5.2 转生循环验证: 转生→重置→倍率→重建→再推图→再转生
- 操作: 第1次转生(倍率1.88×) → 一键重建30分钟 → 扫荡已通关关卡 → 推进至第五章"三国鼎立" → 声望提升至Lv.15 → 第2次转生(倍率2.70×) → 加速重建更快(20分钟恢复) → 推进至第六章 → 永恒挑战推进至80层 → 第3次转生(倍率3.19×) → 验证每次转生后恢复速度递增(1.88×→2.70×→3.19×体感加速明显)
- 预期: 转生循环每次加速明显，3次转生后玩家可在1小时内恢复至上一轮50%进度

### 5.3 声望商城→转生倍率→永恒积分→全系统数据流
- 操作: 声望商城兑换经验书 → 武将升级 → 战力提升 → 永恒挑战推进更高层 → 永恒积分累积 → 声望等级提升(挑战层数贡献声望) → 转生倍率提高(声望等级×0.05加成) → 转生后产出加速 → 资源更多 → 声望商城兑换更多 → 验证数据流: 声望等级→转生倍率→产出倍率→资源获取→声望获取→声望等级 全链路正向循环
- 预期: 四大终局系统形成飞轮效应，任一系统推进均带动其他系统加速

### 5.4 离线全系统验证: 离线收益+转生+声望+永恒挑战
- 操作: 离线前记录: 声望Lv.12/领土28块/资源粮草125000/转生倍率2.25× → 离线12小时 → 回归 → 验证离线收益: 领土产出声望(5~30/h×12h×50%×2.25×) + 资源产出(基础×12h×效率系数×2.25×) + 离线事件处理(正面保守50%/负面防御减免) + 永恒挑战离线不推进(需在线操作) → 验证离线声望累积正确推动声望进度条
- 预期: 离线收益受转生倍率加成，声望离线累积正确，永恒挑战仅在线推进
