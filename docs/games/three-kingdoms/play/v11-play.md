# v11.0 群雄逐鹿 — 玩游戏流程 (Round 2)

> 核心循环: 竞技场面板 → 对手列表 → 挑战 → 战斗 → 排名变化 → 赛季奖励 → 防守编队 → 商店

---

## 1. 竞技场面板与对手列表

- 打开竞技场Tab → 显示当前排名/段位/积分 → 自动生成3名候选对手
- 对手卡片展示: 头像/战力/排名/阵营/防守阵容预览
- 验证: 对手战力在自身×0.7~×1.3范围 | 排名在自身±5~±20 | 阵营分布多样化
- 刷新对手: 免费30min自动刷新 / 手动500铜钱(每日10次上限)
- 交叉验证: MatchConfig.powerMinRatio=0.7, powerMaxRatio=1.3, candidateCount=3 ✅

## 2. 挑战与PvP战斗

- 选择对手 → 确认消耗1次挑战次数(每日5次+元宝购买5次) → 进入战斗
- 进攻方: 玩家选择编队(复用远征编队) | 防守方: 对手预设阵容+AI策略
- 战斗模式: 全自动(AI控制)/半自动(大招手动释放)
- 防守方加成: 全属性+5%(城墙加成) | 10回合超时: 防守方获胜
- 交叉验证: ChallengeConfig.dailyFree=5, maxPurchase=5 ✅

## 3. 战斗结果与排名变化

- 战斗结束 → 积分变化(进攻胜+30~60/败-15~30) → 段位自动检查
- 升段: 青铜V→青铜IV→...→王者(21级段位)
- 排名实时刷新 → 排行榜更新 → 战报邮件通知
- 交叉验证: RankTier枚举7大段位×3小级=21级, 积分范围匹配 ✅

## 4. 赛季结算与奖励

- 赛季28天周期 → 每日按段位发放奖励(铜钱+竞技币+元宝)
- 赛季结束: 按最高段位结算 → 积分重置为当前段位最低值
- 赛季奖励通过邮件发放 → 竞技币存入货币系统
- 交叉验证: ArenaSeasonSystem.seasonDurationDays=28 ✅

## 5. 防守编队与AI策略

- 打开防守编队面板 → 设置5阵位 → 选择阵型(鱼鳞/锋矢/雁行/长蛇/方圆)
- 选择AI策略: 均衡/猛攻/坚守/智谋 → 即时生效
- 被挑战记录: 防守日志(胜率统计+智能调整建议)
- 交叉验证: FormationType 5种, AIDefenseStrategy 4种 ✅

## 6. 竞技商店

- 打开竞技商店 → 竞技币兑换: 武将碎片/强化石/装备箱/头像框
- 周限购机制 → 每周一0:00重置限购次数
- 交叉验证: ArenaShopSystem.buyItem() 扣除竞技币+周限购检查 ✅
