## 1. 竞技场系统

### 1.1 竞技场入口与主界面
- 操作: 点击主城"演武场"建筑(主城≥10级且武将≥3名且完成新手引导第5步后解锁) → 进入竞技场Tab → 顶部显示当前排名/段位/积分 → 中部展示3名候选对手卡片(头像/战力/排名/阵营/防守阵容缩略) → 底部显示挑战次数与刷新按钮
- 验证: 排名信息实时准确 | 段位图标与积分范围匹配 | 3名对手卡片数据完整无空值 | 未满足解锁条件时建筑显示锁定状态并提示解锁条件
- 预期: 界面信息层次清晰，一眼掌握自身排名和可挑战目标，新手引导提示首次挑战流程

### 1.2 对手匹配规则
- 操作: 进入竞技场自动触发匹配 → 系统按战力×0.7~×1.3范围+排名±5~±20(根据当前段位动态调整范围)+阵营多样化筛选候选 → 查看对手详情面板(阵容/战力分布/历史胜率)
- 验证: 对手战力不低于自身×0.7且不超×1.3 | 排名差值在±5~±20区间 | 3名对手阵营尽量不重复 | 匹配结果排除自身
- 预期: 匹配结果公平可控，既不碾压也不被碾压，阵营多样性增加策略博弈乐趣

### 1.3 刷新与挑战次数
- 操作: 等待30min自动刷新对手列表 / 手动点击刷新消耗500铜钱(每日10次上限，刷新后5秒冷却) → 选择目标对手 → 确认消耗1次挑战次数(每日5次免费+元宝×50/次购买5次) → 进入战斗
- 验证: 自动刷新倒计时准确 | 手动刷新扣铜钱正确且计数不超10 | 刷新后5秒内不可再次刷新 | 挑战次数0:00重置 | 元宝购买次数独立计数且每次扣元宝×50
- 预期: 免费节奏适中不焦虑，付费刷新满足急切玩家，每日挑战次数控制肝度

## 2. 排名挑战

### 2.1 进攻编队与战斗模式
- 操作: 挑战前选择进攻编队(复用远征编队槽位) → 进入PvP战斗 → 选择战斗模式: 全自动模式(AI控制全部技能释放) / 半自动模式(手动释放大招+时停机制，普攻自动) → 最多10回合 → 支持战斗加速1x/2x/3x
- 验证: 编队武将属性与远征编队同步 | 全自动模式技能释放逻辑合理 | 半自动模式大招按钮可点击且有冷却提示 | 时停机制在大招可用时触发 | 加速切换流畅无战斗数据异常
- 预期: 全自动适合快速刷分，半自动提供操作深度，两种模式战斗结果公平一致
- 备注: PVP不开放全手动模式，防止操作延迟影响异步对战公平性

### 2.2 防守方加成与超时判定
- 操作: 战斗开始 → 防守方全属性+5%(城墙加成) → 回合计数显示 → 阵营特效展示(蜀红/魏蓝/吴绿) → 达到10回合未分胜负 → 判定防守方获胜 → 显示超时结算动画
- 验证: 防守方武将攻防血属性均+5% | 回合计数器准确 | 阵营特效颜色正确 | 超时判定结果为防守方胜 | 结算面板标注"超时获胜"
- 预期: 城墙加成体现防守优势，超时规则鼓励进攻方追求效率，避免消耗战僵局

### 2.3 战斗结果与积分变化
- 操作: 战斗结束 → 结算面板展示胜负/积分变化/段位变动 → 进攻胜+30~60分(按对手战力差浮动) / 进攻败-15~30分 → 防守方独立积分计算(防守胜+20~40/防守败-10~25) → 排名实时刷新 → 战报邮件通知双方
- 验证: 积分增减在配置范围内 | 高战力对手胜利获得更多积分 | 防守方积分独立不互相干扰 | 排行榜即时更新 | 双方均收到战报邮件
- 预期: 积分奖惩合理激励挑战强敌，防守积分独立保证被攻击不亏太多，战报邮件信息完整

### 2.4 战斗回放
- 操作: 打开战报邮件 → 点击"回放" → 进入回放界面 → 支持0.5x/1x/2x/4x变速播放 → 关键回合自动标注(暴击/大招/击杀) → 最多保存50条/保留7天 → 支持分享回放链接给好友
- 验证: 回放战斗过程与实际一致 | 变速播放流畅无卡顿 | 关键时刻标注准确 | 超过50条自动删除最旧记录 | 分享链接好友可正常打开观看
- 预期: 回放帮助分析阵容弱点，变速播放节省时间，关键标注快速定位转折点，分享功能增加社交互动

## 3. 赛季系统

### 3.1 段位等级与每日奖励
- 操作: 查看段位面板 → **5大段位**(青铜/白银/黄金/钻石/王者)×多级小段=**21级** → 积分范围对应段位(青铜V: 0~300 → 王者: 前100名) → 每日0:00按当前段位发放奖励(铜钱+竞技币+元宝，高段位额外元宝)
- 验证: 21级段位积分范围无重叠 | 段位图标颜色渐变正确 | 每日奖励数量与段位匹配 | 奖励通过邮件发放 | 王者段位按前100名动态判定
- 预期: 段位梯度清晰有追求感，每日奖励激励保持段位，低段位也有基础奖励不弃坑
- 备注: **段位设计以PRD [PVP-3]段位等级表为权威源**，共5大段位×多级小段=21级: 青铜(V/IV/III/II/I)→白银(V/IV/III/II/I)→黄金(V/IV/III/II/I)→钻石(V/IV/III/II/I)→王者(前100名)。早期草案中的铂金/大师段位已废弃。

### 3.2 赛季周期与结算
- 操作: 赛季28天周期(4周) → 倒计时显示在竞技场顶部 → 赛季结束自动结算 → 按赛季最高段位发放结算奖励(大量竞技币+限定头像框) → 积分重置为当前段位最低值 → 新赛季开始 → 前100名上榜+段位变化动画展示
- 验证: 赛季倒计时准确 | 结算奖励按最高段位而非当前段位 | 积分重置后段位可能降级 | 结算展示动画播放完整 | 赛季专属头像框按段位发放
- 预期: 28天赛季节奏适中，最高段位结算鼓励全程参与，积分重置让新赛季有新鲜感

### 3.3 竞技商店
- 操作: 打开竞技商店 → 消耗竞技币兑换: 武将碎片(随机)50币/每日5次、强化石30币/每日10次、铜钱×1000(20币/每日20次)、加速道具(1h)40币/每日5次、赛季专属头像框500币/赛季1次、传说装备箱800币/赛季2次 → 每日限购商品于每日0:00重置 | 赛季限购商品于新赛季开始时重置
- 验证: 竞技币余额正确显示 | 兑换扣币准确 | 日限购次数每日0:00重置 | 赛季限购新赛季重置 | 商品列表赛季更新 | 日限购和赛季限购独立计数
- 预期: 竞技币有稳定消耗出口，日限购+赛季限购控制通胀，限定商品刺激竞技参与度

### 3.4 竞技商店异常处理 [R4新增]
- 操作: 竞技商店兑换商品 → 竞技币不足 → 弹窗提示「竞技币不足」+显示当前余额和所需数量+提供「前往竞技场」按钮引导获取 → 背包满(装备箱/道具类商品) → 弹窗提示「背包已满，请清理后兑换」+提供「前往背包」按钮 → 每日限购次数为0 → 商品卡片置灰+显示「今日已售罄」 → 赛季限购次数为0 → 商品卡片置灰+显示「本赛季已售罄，下赛季重置」
- 验证: 竞技币不足时兑换被拦截且不扣币 | 背包满时兑换被拦截且不扣币 | 日限购售罄商品不可点击 | 赛季限购售罄商品显示下赛季重置提示 | 异常处理后界面状态正确不卡死
- 预期: 异常处理友好不扣费，引导玩家解决问题后再次尝试，赛季限购提示增强期待感

## 4. 防守阵容

### 4.1 防守阵容设置
- 操作: 进入防守编队界面 → 设置5个阵位武将 → 选择阵型(鱼鳞/锋矢/雁行/长蛇/方圆) → 防守阵容与进攻阵容独立，可使用相同武将 → 保存后立即生效(正在进行的防守战斗使用旧阵容快照)
- 验证: 5阵位均可放置武将 | 5种阵型切换正确 | 防守阵容不影响进攻编队 | 保存后立即生效 | 进行中战斗不受影响
- 预期: 防守阵容独立设置不干扰进攻，即时生效让调整立竿见影，快照机制保证战斗公平

### 4.2 AI防守策略
- 操作: 在防守编队界面选择AI策略 → 均衡(攻防兼顾，优先攻击血量最低目标) / 猛攻(集中火力击杀单一目标) / 坚守(优先防御和回复) / 智谋(优先释放控制和AOE技能) → 保存策略
- 验证: 4种策略可正常切换 | AI行为与策略描述一致 | 策略保存后立即应用到后续防守战斗
- 预期: 4种策略覆盖主流防守思路，玩家可根据阵容特点选择最优策略

### 4.3 防守日志与优化建议
- 操作: 打开防守日志 → 查看被挑战记录列表(胜/负标识+对手名+排名变化+时间) → 最多50条记录 → 点击展开查看战斗回放 → 查看防守胜率统计 → 被攻破时查看智能调整建议(如"建议更换防守策略为坚守")
- 验证: 日志记录完整准确 | 胜率统计计算正确 | 智能建议基于实际防守数据生成 | 回放可正常播放
- 预期: 日志帮助分析防守弱点，智能建议降低优化门槛，回放验证调整效果

### 4.4 防守奖励
- 操作: 被其他玩家挑战 → 防守成功获得积分+20~40+铜钱×200 → 防守失败积分-10~25(无其他惩罚) → 连续防守成功每5次额外奖励竞技币×20 → 奖励通过邮件发放
- 验证: 防守成功/失败积分变化在范围内 | 连续防守奖励计数正确 | 奖励邮件及时送达
- 预期: 防守奖励激励玩家重视防守编队，连续防守奖励提供额外追求

- 预期: 排名奖励激励头部玩家竞争，限定称号提供炫耀资本，历史记录留存荣誉感

## 6. 好友系统

### 6.1 好友面板与添加
- 操作: 点击好友入口 → 打开好友面板(好友列表+申请+搜索) → 查看在线状态 → 搜索添加(输入昵称/ID精确搜索) / 系统推荐(相近战力/等级) → 发送好友申请(每日上限20次，待处理上限30条，申请7天未处理自动过期) → 对方审批通过后建立好友关系
- 验证: 好友上限随等级增长(20→30→40→50→60) | 搜索结果精确匹配 | 推荐列表战力相近 | 申请计数正确 | 过期申请自动清理
- 预期: 搜索+推荐双通道满足不同场景，上限随等级扩展合理，过期清理避免堆积

### 6.2 好友互动
- 操作: 在好友面板选择好友 → 选择互动类型:
  - 赠送兵力: 10次/天 → 友情点×5/次
  - 拜访主城: 5次/天 → 铜钱×100/次+随机小概率获得资源
  - 切磋: 3次/天 → 胜利友情点×20/失败友情点×5
  - 借将: 3次/天(不同好友) → 借用方100%经验/出借方友情点×10
- 验证: 各互动次数独立计数且0:00重置 | 友情点奖励正确发放 | 拜访随机资源概率正常 | 切磋战斗结果影响友情点数量
- 预期: 多种互动方式增加社交粘性，友情点统一奖励让所有互动有价值
