# v11.0 群雄逐鹿 — 玩游戏流程

## 1. 远征队伍
- 打开远征Tab → 选择关卡 → 编排队伍 → 出征
- 验证: 队伍编成、武将体力消耗、关卡难度

## 2. 远征战斗
- 自动/手动战斗 → 关卡通关 → 奖励结算 → 下一关
- 验证: 战斗过程、通关条件、奖励计算

## 3. 远征奖励
- 关卡奖励 → 累计奖励 → 里程碑奖励
- 验证: 奖励递增、特殊掉落

## 4. 自动远征
- 开启自动远征 → 自动推图 → 资源汇总
- 验证: 自动战斗逻辑、失败停止条件

## 5. 远征队伍辅助
- 一键推荐队伍 → 武将推荐 → 阵容优化
- 验证: 推荐算法合理性
