# v11.0 群雄逐鹿 — 玩游戏流程

> 核心循环: 竞技场面板 → 对手匹配 → 挑战战斗 → 积分排名 → 赛季奖励 → 防守编队优化

---

## 1. 竞技场系统

### 1.1 竞技场入口与主界面
- 操作: 点击主城"竞技场"建筑 → 进入竞技场Tab → 顶部显示当前排名/段位/积分 → 中部展示3名候选对手卡片(头像/战力/排名/阵营/防守阵容缩略) → 底部显示挑战次数与刷新按钮
- 验证: 排名信息实时准确 | 段位图标与积分范围匹配 | 3名对手卡片数据完整无空值
- 预期: 界面信息层次清晰，一眼掌握自身排名和可挑战目标，新手引导提示首次挑战流程

### 1.2 对手匹配规则
- 操作: 进入竞技场自动触发匹配 → 系统按战力×0.7~×1.3范围+排名±5~±20+阵营多样化筛选候选 → 查看对手详情面板(阵容/战力分布/历史胜率)
- 验证: 对手战力不低于自身×0.7且不超×1.3 | 排名差值在±5~±20区间 | 3名对手阵营尽量不重复
- 预期: 匹配结果公平可控，既不碾压也不被碾压，阵营多样性增加策略博弈乐趣

### 1.3 刷新与挑战次数
- 操作: 等待30min自动刷新对手列表 / 手动点击刷新消耗500铜钱(每日10次上限) → 选择目标对手 → 确认消耗1次挑战次数(每日5次免费+元宝购买5次) → 进入战斗
- 验证: 自动刷新倒计时准确 | 手动刷新扣铜钱正确且计数不超10 | 挑战次数0:00重置 | 元宝购买次数独立计数
- 预期: 免费节奏适中不焦虑，付费刷新满足急切玩家，每日挑战次数控制肝度

## 2. 排名挑战

### 2.1 进攻编队与战斗模式
- 操作: 挑战前选择进攻编队(复用远征编队槽位) → 进入PvP战斗 → 全自动模式(AI控制全部技能释放) / 半自动模式(手动释放大招，普攻自动) → 最多10回合
- 验证: 编队武将属性与远征编队同步 | 全自动模式技能释放逻辑合理 | 半自动模式大招按钮可点击且有冷却提示
- 预期: 全自动适合快速刷分，半自动提供操作深度，两种模式战斗结果公平一致

### 2.2 防守方加成与超时判定
- 操作: 战斗开始 → 防守方全属性+5%(城墙加成) → 回合计数显示 → 达到10回合未分胜负 → 判定防守方获胜 → 显示超时结算动画
- 验证: 防守方武将攻防血属性均+5% | 回合计数器准确 | 超时判定结果为防守方胜 | 结算面板标注"超时获胜"
- 预期: 城墙加成体现防守优势，超时规则鼓励进攻方追求效率，避免消耗战僵局

### 2.3 战斗结果与积分变化
- 操作: 战斗结束 → 结算面板展示胜负/积分变化/段位变动 → 进攻胜+30~60分(按对手战力浮动) / 进攻败-15~30分 → 防守方独立积分计算 → 排名实时刷新 → 战报邮件通知双方
- 验证: 积分增减在配置范围内 | 高战力对手胜利获得更多积分 | 防守方积分独立不互相干扰 | 排行榜即时更新
- 预期: 积分奖惩合理激励挑战强敌，防守积分独立保证被攻击不亏太多，战报邮件信息完整

### 2.4 战斗回放
- 操作: 打开战报邮件 → 点击"回放" → 进入回放界面 → 支持1x/2x/4x变速播放 → 关键回合自动标注(大招释放/击杀) → 最多保存50条/保留7天
- 验证: 回放战斗过程与实际一致 | 变速播放流畅无卡顿 | 关键时刻标注准确 | 超过50条自动删除最旧记录
- 预期: 回放帮助分析阵容弱点，变速播放节省时间，关键标注快速定位转折点

## 3. 赛季系统

### 3.1 段位等级与每日奖励
- 操作: 查看段位面板 → 7大段位(青铜/白银/黄金/铂金/钻石/大师/王者)×3小级(V/IV/III)=21级 → 积分范围对应段位 → 每日0:00按当前段位发放奖励(铜钱+竞技币+元宝)
- 验证: 21级段位积分范围无重叠 | 段位图标颜色渐变正确 | 每日奖励数量与段位匹配 | 奖励通过邮件发放
- 预期: 段位梯度清晰有追求感，每日奖励激励保持段位，低段位也有基础奖励不弃坑

### 3.2 赛季周期与结算
- 操作: 赛季28天周期 → 倒计时显示在竞技场顶部 → 赛季结束自动结算 → 按赛季最高段位发放结算奖励(大量竞技币+限定头像框) → 积分重置为当前段位最低值 → 新赛季开始
- 验证: 赛季倒计时准确 | 结算奖励按最高段位而非当前段位 | 积分重置后段位可能降级 | 结算展示动画播放完整
- 预期: 28天赛季节奏适中，最高段位结算鼓励全程参与，积分重置让新赛季有新鲜感

### 3.3 竞技商店
- 操作: 打开竞技商店 → 消耗竞技币兑换: 武将碎片/强化石/装备箱/限定头像框 → 周限购机制(热门商品限购2~5件) → 每周一0:00重置限购次数
- 验证: 竞技币余额正确显示 | 兑换扣币准确 | 限购次数递减且周一重置 | 商品列表赛季更新
- 预期: 竞技币有稳定消耗出口，周限购控制通胀，限定商品刺激竞技参与度

## 4. 排行榜

### 4.1 多维度排名
- 操作: 打开排行榜面板 → 切换Tab查看: 战力榜/竞技榜/赛季榜/阵营榜 → 每榜显示Top100+自己的排名固定底部 → 点击玩家查看详情(阵容/段位/战绩)
- 验证: 各榜单数据独立排序正确 | 自己排名固定底部不随滚动消失 | 点击详情数据与实际一致 | 排名变化用箭头标识(↑↓→)
- 预期: 多维度排名满足不同追求，固定底部排名随时掌握位置，详情查看增加社交互动

### 4.2 排名奖励与展示
- 操作: 赛季结算时 → Top100额外发放排名奖励(元宝+限定称号) → 排行榜展示赛季冠军专属形象 → 历史赛季可查看往届冠军
- 验证: 排名奖励按名次梯度发放 | 称号属性加成正确 | 历史赛季数据完整可查
- 预期: 排名奖励激励头部玩家竞争，限定称号提供炫耀资本，历史记录留存荣誉感

## 5. 交叉验证

### 5.1 竞技场→战斗→积分→段位全链路
- 操作: 进入竞技场 → 匹配3名对手 → 选择战力相近对手 → 消耗1次挑战次数 → 进攻胜利 → 积分+45 → 段位从青铜IV升至青铜III → 每日奖励按青铜III发放 → 排行榜名次上升
- 预期: 匹配→挑战→积分→段位→奖励→排名全链路数据一致，每步状态转换正确，无数据断裂

### 5.2 防守编队→被挑战→日志→优化闭环
- 操作: 设置防守阵容(鱼鳞阵+坚守策略) → 被其他玩家挑战3次(2败1胜) → 查看防守日志(胜率33%+建议"尝试锋矢阵+猛攻策略") → 调整防守阵容 → 再次被挑战验证调整效果
- 预期: 防守阵容设置即时生效，日志记录完整准确，AI建议有参考价值，调整后可验证效果

### 5.3 赛季→商店→奖励资源闭环
- 操作: 完整参与28天赛季 → 每日领取段位奖励累计竞技币2800 → 赛季结算获得王者段位奖励(竞技币1500+限定头像框) → 竞技商店兑换金色武将碎片×10(消耗2000竞技币) → 剩余2300竞技币留待下赛季
- 预期: 赛季奖励+每日奖励竞技币总量可兑换核心商品，资源获取与消耗形成正向循环，无死锁
