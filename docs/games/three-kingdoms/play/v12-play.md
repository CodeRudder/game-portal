# v12.0 远征天下 — 玩游戏流程

> 核心循环: 远征地图选路线 → 编组远征队 → 节点事件推进 → 战斗/探索/拾取 → 结算奖励 → 扫荡/自动远征

---

## 1. 远征系统

### 1.1 远征地图与路线选择
- 操作: 点击主城"远征营"建筑 → 进入远征地图场景 → 顶部显示可用队列槽位(主城5/10/15/20级→1/2/3/4支) → 地图展示树状分支路线 → 节点标注状态(✅已通关/🔄行军中/🔒未解锁) → 选择路线查看难度(简单⭐/普通⭐⭐⭐/困难⭐⭐⭐⭐/奇袭⭐⭐⭐⭐⭐)与行军时长 → 确认进入
- 验证: 节点状态与实际进度同步 | 未解锁路线灰色不可点击 | 奇袭路线需通关对应困难路线才解锁 | 队列槽位数与主城等级匹配
- 预期: 地图一目了然展示进度与选择，难度梯度清晰，多路线分支提供策略取舍

### 1.2 队列槽位与多队并行
- 操作: 主城5级解锁第1个队列 → 10/15/20级分别解锁第2/3/4队列 → 多支队伍可同时出征不同路线 → 切换Tab查看各队伍状态(行军中/战斗中/等待中/已完成) → 点击队伍快速跳转至对应路线节点
- 验证: 队列数量与主城等级对应 | 多队伍并行时各自独立推进 | 队伍间武将互斥(同一武将不可编入多队) | 切换队伍状态实时更新
- 预期: 多队并行增加策略深度，合理分配武将资源成为核心决策点

### 1.3 自动远征与离线推进
- 操作: 三星通关路线后开启自动远征设置 → 配置重复次数/失败处理(停止或跳过)/背包满处理/兵力不足处理 → 系统自动循环推进 → 离线期间继续执行(效率×0.85，上限72h) → 上线后展示离线期间摘要(战斗次数/获得资源/触发事件)
- 验证: 暂停条件触发时正确停止(兵力耗尽/背包满/连续失败2次) | 离线效率×0.85计算准确 | 离线超过72h后停止计算 | 摘要数据与实际一致
- 预期: 自动远征解放双手，离线推进保证进度不浪费，摘要清晰展示离线收获

## 2. 远征队编组

### 2.1 武将选择与编队
- 操作: 进入编队界面 → 从武将列表选择最多5名武将 → 已在其他队伍的武将灰显且标注"出战中" → 选择阵型(鱼鳞/鹤翼/锋矢/雁行/长蛇/方圆) → 右侧显示战力预览与阵营羁绊提示 → 确认编队
- 验证: 武将互斥规则正确执行 | 已出战武将灰显不可选 | 5名武将上限不可突破 | 战力预览数值与实际战斗一致
- 预期: 编队界面直观展示可用武将，互斥提示避免误操作，战力预览辅助决策

### 2.2 阵型效果与阵营羁绊
- 操作: 切换6种阵型 → 查看属性修正说明(鱼鳞+防御/锋矢+攻击/鹤翼+暴击/雁行+速度/长蛇+闪避/方圆+均衡) → 查看阵型克制关系(鱼鳞>锋矢>雁行>鹤翼>鱼鳞) → 同阵营≥3名武将触发羁绊"全属性+10%" → 羁绊激活时武将卡片发光提示
- 验证: 阵型属性修正数值正确 | 克制关系循环无断链 | 羁绊≥3名同阵营才触发 | 克制方战斗中全属性+10%
- 预期: 6种阵型覆盖主流战术，克制关系增加博弈深度，阵营羁绊鼓励收集同势力武将

### 2.3 智能编队与兵力管理
- 操作: 点击"一键最强"按钮 → 系统基于战力+阵营羁绊自动填充最优组合 → 手动微调后确认 → 查看兵力消耗(出发每武将-20兵力，扫荡每武将-10兵力) → 兵力不足的武将标红提示 → 兵力自然恢复(每分钟+1)或使用征兵令立即补满
- 验证: 智能编队优先选择战力最高且触发羁绊的组合 | 兵力扣除准确 | 兵力不足时无法出发 | 恢复速率与配置一致
- 预期: 智能编队降低新手门槛，兵力管理增加资源规划维度，避免无限出征

## 3. 远征事件

### 3.1 节点遭遇与战斗
- 操作: 队伍沿路线行军(进度条推进) → 遭遇节点触发事件 → 山贼节点: 低难度普通战斗，快速清除 → 天险节点: 中等难度地形挑战，阵型效果被削弱 → Boss节点: 高难度精英战斗，推荐战力×2.0~×3.0 → 宝箱节点: 直接获取奖励(无需战斗) → 休息节点: 恢复队伍20%兵力
- 验证: 5种节点类型触发逻辑正确 | 节点难度与路线等级匹配 | 宝箱直接给奖无战斗 | 休息节点恢复兵力数值准确
- 预期: 5种节点交替出现保持新鲜感，难度递增制造紧张感，宝箱和休息点提供喘息空间

### 3.2 远征战斗规则与结果评定
- 操作: 进入战斗 → 全自动模式(AI控制技能释放) → 阵型克制生效(克制方全属性+10%) → 最多10回合 → 战斗结束显示4级评定: 大捷⭐⭐⭐(无损或微损)/小胜⭐⭐(损失≤30%)/惨胜⭐(损失>30%)/惜败(挑战失败) → 三星通关解锁扫荡
- 验证: 战斗全自动不可手动干预 | 10回合超时判定惜败 | 星级评定与战损比例匹配 | 三星通关标记正确
- 预期: 全自动战斗节奏紧凑，星级评定激励追求完美通关，三星解锁扫荡形成长期目标

### 3.3 扫荡系统
- 操作: 三星通关路线后解锁扫荡 → 选择扫荡类型: 普通扫荡(扫荡令×1，奖励×100%，5次/路线/天)/高级扫荡(扫荡令×3，奖励×150%+保底稀有，3次/路线/天)/免费扫荡(无消耗，奖励×50%，1次/路线/天) → 确认执行 → 即时结算展示奖励
- 验证: 未三星通关路线不可扫荡 | 每日次数独立计数且0:00重置 | 高级扫荡保底至少1件稀有掉落 | 免费扫荡奖励减半正确
- 预期: 三种扫荡满足不同付费意愿，次数限制控制收益膨胀，扫荡收益≤手动保证手动玩法价值

## 4. 远征奖励

### 4.1 基础奖励与掉落
- 操作: 通关节点/路线 → 结算面板展示奖励列表 → 基础奖励: 粮草+铜钱+铁矿+装备碎片+经验(按路线难度倍率) → 随机掉落: 装备碎片(30~80%)/武将碎片(5~25%)/技能书(2~20%) → 点击物品查看详情 → 确认领取入背包
- 验证: 基础奖励数量与难度等级匹配 | 掉落概率在配置范围内 | 奖励物品正确入背包 | 结算面板无遗漏项
- 预期: 奖励层次丰富，基础奖励保底不白跑，随机掉落提供惊喜感，难度越高回报越丰厚

### 4.2 首通奖励与里程碑
- 操作: 首次通关某路线 → 触发首通奖励弹窗(稀有武将碎片×1+元宝×50+声望+20) → 标注"首次通关"金色标识 → 累计通关路线数达成里程碑(初出茅庐1条→百战之师5条→远征名将15条→天下布武30条) → 领取里程碑奖励(称号+头像框+大量资源)
- 验证: 首通奖励仅首次触发不可重复 | 里程碑计数跨路线累计 | 称号/头像框正确装备 | 重复通关不再触发首通弹窗
- 预期: 首通奖励激励探索新路线，里程碑系统提供长期追求目标，称号展示远征成就

### 4.3 远征币与资源循环
- 操作: 远征奖励包含远征币 → 远征币可在远征商店兑换: 武将碎片/装备箱/扫荡令/征兵令 → 查看远征商店限购(热门商品周限2~5件) → 每周一0:00重置限购 → 远征产出占中后期资源获取30%+
- 验证: 远征币余额正确 | 兑换扣币准确 | 限购次数递减且周一重置 | 远征产出占比达到设计目标
- 预期: 远征币形成独立经济循环，商店兑换满足定向需求，远征成为核心资源渠道之一

## 5. 交叉验证

### 5.1 编队→远征→战斗→奖励全链路
- 操作: 编组远征队(5名蜀国武将+锋矢阵) → 选择困难路线"剑阁道" → 出发消耗100兵力 → 推进遭遇山贼(大捷⭐⭐⭐)→天险(小胜⭐⭐)→Boss(惨胜⭐) → 通关结算获得粮草+铜钱+赵云碎片×2+首通元宝×50 → 三星未全满不可扫荡 → 返回编队查看武将兵力剩余
- 预期: 编队→出征→战斗→结算全链路数据一致，兵力扣除正确，掉落符合概率配置，首通标识仅显示一次

### 5.2 多队并行→互斥→自动远征闭环
- 操作: 主城15级解锁3个队列 → 编组队伍A(关羽+张飞+赵云+马超+黄忠) → 编组队伍B时蜀国五虎将灰显不可选 → 队伍A设置自动远征(重复5次/失败停止) → 队伍B手动出征简单路线 → 队伍A自动完成5次后停止 → 两队奖励独立结算
- 预期: 武将互斥规则在多队场景下严格生效，自动远征按配置执行，两队进度互不干扰，资源独立入账

### 5.3 扫荡→商店→强化→再远征资源闭环
- 操作: 三星通关"赤壁水战"普通路线 → 每日免费扫荡1次+普通扫荡5次 → 获得6次基础奖励+远征币累计450 → 远征商店兑换扫荡令×5(消耗200远征币) → 剩余250远征币存留 → 使用掉落装备碎片强化武将 → 武将战力提升 → 挑战困难路线"赤壁烈焰"
- 预期: 扫荡产出稳定可预期，远征币兑换补充扫荡令形成循环，强化提升战力解锁更高难度，资源获取与消耗形成正循环
