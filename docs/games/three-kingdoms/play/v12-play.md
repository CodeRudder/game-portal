# v12.0 远征天下 — 玩游戏流程

> 核心循环: 远征地图选路线 → 编组远征队 → 节点事件推进 → 战斗/探索/拾取 → 结算奖励 → 扫荡/自动远征
> 版本功能: 远征系统(EXP) + 排行榜(SOC) + 远征币经济
> 文档版本: v1.4（Gap分析第四轮修订：PRD数值权威性补全、跨系统串联闭环验证、离线效率表述统一、赛季结算完整流程、远征→装备→再远征闭环）

## 0. 解锁与入口

### 0.1 远征系统解锁
- **触发条件**: 主城等级提升至≥5级
- **操作步骤**:
  1. 主城等级达到5级 → 系统弹出解锁提示「远征系统已开启！」
  2. 主城地图出现"远征营"建筑(带"新"标识，脉动动画3秒)
  3. 点击"远征营"建筑 → 进入远征地图场景
  4. 备选入口: 主界面"更多"菜单 → 点击「远征」→ 进入远征地图场景
- **预期结果**: 双入口设计方便不同操作习惯的玩家，解锁条件与主城进度绑定确保玩家具备基础实力
- **异常处理**:
  - 主城<5级时远征营建筑不可见且"更多"菜单中远征入口灰显(点击提示「主城达到5级后解锁」)
  - 主城≥5级时两个入口均可进入
  - 网络中断时重连后建筑状态正确恢复

## 1. 远征系统

### 1.1 远征地图与路线选择
- **触发条件**: 进入远征地图场景
- **操作步骤**:
  1. 点击主城"远征营"建筑或"更多"菜单「远征」→ 进入远征地图场景
  2. 顶部显示可用队列槽位(主城5/10/15/20级→1/2/3/4支)
  3. 地图展示树状分支路线，起点(主城)→终点(目标城池)
  4. 节点标注状态: ✅已通关(铜色印章)/🔄行军中(金色脉动)/🔒未解锁(灰色锁链)
  5. 选择路线查看详情: 难度(简单⭐~⭐⭐行军30~45min/普通⭐⭐⭐行军45min~1.5h/困难⭐⭐⭐⭐行军1~2h/奇袭⭐⭐⭐⭐⭐行军2~3h)与推荐战力倍率(×1.0/×1.5/×2.0/×3.0)
  6. 确认进入路线详情 → 选择编队出发
- **预期结果**: 地图一目了然展示进度与选择，难度梯度清晰，多路线分支提供策略取舍
- **异常处理**:
  - 未解锁路线灰色不可点击(点击显示解锁条件提示)
  - 奇袭路线需通关对应困难路线才解锁
  - 所有队列都在使用时提示「所有队伍已出征，请等待归来」
  - 路线按区域逐步解锁(虎牢关→汜水关→洛阳→…)，后续区域需通关前置区域的任意一条路线
- **验证**: 节点状态与实际进度同步 | 队列槽位数与主城等级匹配 | 路线难度和推荐战力倍率与PRD EXP-1一致

### 1.2 队列槽位与多队并行
- **触发条件**: 主城等级达到对应阈值
- **操作步骤**:
  1. 主城5级解锁第1个队列 → 10/15/20级分别解锁第2/3/4队列
  2. 多支队伍可同时出征不同路线
  3. 切换Tab查看各队伍状态(行军中/战斗中/等待中/已完成)
  4. 点击队伍快速跳转至对应路线节点
- **预期结果**: 多队并行增加策略深度，合理分配武将资源成为核心决策点
- **异常处理**:
  - 队列数量与主城等级严格对应
  - 多队伍并行时各自独立推进互不干扰
  - 队伍间武将互斥(同一武将不可编入多队)
- **验证**: 队列上限=主城等级对应值 | 切换队伍状态实时更新

### 1.3 自动远征设置与离线推进
- **触发条件**: 至少有一条路线三星通关
- **操作步骤**:
  1. 三星通关路线后开启自动远征设置面板
  2. 配置设置项:
     | 设置项 | 选项 | 默认值 |
     |-------|------|:-----:|
     | 重复次数 | 1/3/5/10/无限 | 无限 |
     | 失败处理 | 暂停/跳过继续 | 暂停 |
     | 背包满处理 | 暂停/自动出售白装 | 暂停 |
     | 自动使用药品 | 开/关 | 开 |
     | 兵力不足处理 | 暂停/使用兵力道具 | 暂停 |
  3. 系统自动循环推进已通关路线
  4. 离线期间继续执行
  5. 上线后展示离线期间摘要(战斗次数/获得资源/触发事件)
- **离线收益计算**（引用SPEC-offline全局离线规范，与plan数据流设计一致）:
  - 离线远征收益 = 正常收益 × 离线效率系数 × 0.85（战斗惩罚系数，引用SPEC-offline远征战斗效率修正）
  - 离线效率系数:
    | 时长 | 效率 |
    |------|:----:|
    | 0~2h | 100% |
    | 2~8h | 80% |
    | 8~24h | 60% |
    | 24~48h | 40% |
    | 48~72h | 25% |
    | >72h | 15% |
  - 可叠加科技/VIP/声望加成（上限+100%），最终效率上限100%
  - 离线战斗胜率 = 正常胜率 × 0.85
  - 离线时间上限 = 72小时（超出部分不计入）
- **预期结果**: 自动远征解放双手，离线推进保证进度不浪费，摘要清晰展示离线收获
- **异常处理**:
  - 暂停条件触发时正确停止: 兵力耗尽/背包满/连续失败2次/手动暂停
  - 离线效率分段衰减计算准确
  - 离线超过72h后停止计算
  - 离线行军继续/遭遇战斗自动进行(效率×0.85)/路线完成自动结算/扫荡继续(受每日上限)/队列全部完成后停止等待
  - 摘要数据与实际一致
- **验证**: 离线收益公式与SPEC-offline一致 | 离线效率分段与plan数据流设计一致 | 0.85战斗惩罚系数为最终统一值(非PRD EXP-3旧版×0.8)

### 1.4 一键远征分配
- **触发条件**: 至少有1个空闲队列和1条可出征路线
- **操作步骤**:
  1. 点击"一键远征"按钮
  2. 系统按队伍战力从高到低排序
  3. 按路线推荐战力从高到低排序
  4. 最高战力队伍分配到最高推荐战力路线
  5. 已扫荡路线优先分配（稳定收益）
  6. 确认后所有队伍同时出发
- **预期结果**: 一键远征简化多队操作，智能分配降低决策负担，老玩家快速完成日常
- **异常处理**:
  - 无可用队伍时提示「所有队伍已出征」
  - 无可出征路线时提示「暂无可出征路线」
  - 互斥武将冲突时自动跳过该队伍并提示
- **验证**: 分配逻辑按战力匹配 | 已通关可扫荡路线优先 | 互斥武将不冲突

### 1.5 远征日志
- **触发条件**: 进入远征界面
- **操作步骤**:
  1. 进入远征界面 → 点击"远征日志"按钮
  2. 查看历史远征记录列表(按时间倒序)
  3. 每条记录包含: 队伍编号+路线名称+开始/结束时间+遭遇列表+战斗结果+奖励明细
  4. 失败记录标红+显示失败原因(战力不足/阵型克制/兵力耗尽等)
  5. 点击单条记录展开详细遭遇列表
- **预期结果**: 远征日志提供完整回顾，失败原因帮助优化策略，VIP延长保留增加付费价值
- **异常处理**:
  - 普通玩家日志保留7天，VIP玩家日志保留14天
  - 超期日志自动清理(清理时日志条目淡出动画)
  - 无日志时显示空状态提示「暂无远征记录」
- **验证**: 日志内容完整无遗漏 | 失败记录标红醒目 | 保留时长与VIP等级对应

## 2. 远征队编组

### 2.1 武将选择与编队
- **触发条件**: 选择路线后进入编队界面，或从远征地图点击"编队"按钮
- **操作步骤**:
  1. 进入编队界面 → 从武将列表选择最多5名武将
  2. 支持阵营筛选: 全部/魏/蜀/吴/群
  3. 已在其他队伍的武将灰显且标注"出战中"(不可选)
  4. 选择阵型(鱼鳞/鹤翼/锋矢/雁行/长蛇/方圆)
  5. 右侧显示战力预览与阵营羁绊提示
  6. 确认编队 → 消耗兵力(每武将-20) → 出发
- **预期结果**: 编队界面直观展示可用武将，互斥提示避免误操作，战力预览辅助决策
- **异常处理**:
  - 武将互斥规则正确执行(同一武将不可编入多队)
  - 已出战武将灰显不可选
  - 5名武将上限不可突破
  - 战力预览数值与实际战斗一致
  - 兵力不足的武将标红提示，无法出发
- **验证**: 阵营筛选正确过滤 | 互斥规则严格 | 战力预览=实际战斗战力

### 2.2 阵型效果与阵营羁绊
- **触发条件**: 编队界面中选择/切换阵型
- **操作步骤**:
  1. 切换6种阵型 → 查看属性修正说明
  2. 查看阵型克制关系(鱼鳞>锋矢>雁行>鹤翼>鱼鳞，循环克制)
  3. 同阵营≥3名武将触发羁绊"全属性+10%"
  4. 羁绊激活时武将卡片发光提示
- **阵型精确效果**（引用PRD EXP-2，与plan验收标准一致）:
  | 阵型 | 效果 |
  |------|------|
  | 鱼鳞 | 防御+15%，攻击-5% |
  | 鹤翼 | 攻击+15%，防御-5% |
  | 锋矢 | 速度+20%，防御-10% |
  | 雁行 | 谋略+15%，速度+5% |
  | 长蛇 | 攻击+10%，速度+10%，防御-15% |
  | 方圆 | 防御+20%，速度-10% |
- **预期结果**: 6种阵型覆盖主流战术，克制关系增加博弈深度，阵营羁绊鼓励收集同势力武将
- **异常处理**:
  - 克制关系循环无断链(鱼鳞>锋矢>雁行>鹤翼>鱼鳞)
  - 羁绊≥3名同阵营才触发(2名不触发)
  - 克制方战斗中全属性+10%
- **验证**: 阵型属性修正数值与PRD EXP-2一致 | 克制关系与plan验收标准一致

### 2.3 智能编队与兵力管理
- **触发条件**: 编队界面中点击"智能编队"或"一键最强"按钮
- **操作步骤**:
  1. 点击"智能编队"按钮 → 系统基于战力+阵营羁绊自动填充，优先激活羁绊
  2. 点击"一键最强"按钮 → 直接选择战力最高的5名可用武将
  3. 手动微调后确认
  4. 查看兵力消耗: 出发每武将-20兵力，扫荡每武将-10兵力
  5. 兵力不足的武将标红提示
  6. 兵力自然恢复(1点/5分钟)或使用征兵令立即补满
- **预期结果**: 智能编队降低新手门槛，兵力管理增加资源规划维度，避免无限出征
- **异常处理**:
  - 兵力不足时无法出发(提示「兵力不足，请补充兵力后再出发」)
  - 无可用武将时智能编队提示「无可用武将」
  - 恢复速率1点/5分钟与PRD EXP-2配置一致
- **验证**: 智能编队优先选择战力最高且触发羁绊的组合 | 一键最强直接取Top5战力 | 兵力扣除准确

## 3. 远征事件

### 3.1 节点遭遇与战斗
- **触发条件**: 队伍沿路线行军到达节点
- **操作步骤**:
  1. 队伍沿路线行军(进度条推进)
  2. 遭遇节点触发事件:
     - **山贼节点**(出现频率高): 低难度普通战斗，推荐战力×1.0，快速清除
     - **天险节点**(出现频率中): 中等难度地形挑战，**阵型效果削弱50%**（所有阵型加成和减益数值减半），推荐战力×1.5
     - **Boss节点**(出现频率低，路线末端): 高难度精英战斗，推荐战力×2.0~×3.0
     - **宝箱节点**(出现频率低): 直接获取奖励(无需战斗)
     - **休息节点**(出现频率中): 恢复队伍20%最大兵力（按每名武将最大兵力的20%计算，向上取整，不超过上限）
- **预期结果**: 5种节点交替出现保持新鲜感，难度递增制造紧张感，宝箱和休息点提供喘息空间
- **异常处理**:
  - 天险节点阵型效果削弱50%(加成和减益均减半)，与plan验收标准一致
  - 休息节点恢复后不超过最大兵力上限
  - Boss节点位于路线末端
  - 宝箱直接给奖无战斗
- **验证**: 5种节点类型触发逻辑正确 | 节点难度与路线等级匹配 | 休息节点恢复数值准确(每武将恢复20%最大兵力，向上取整)

### 3.2 远征战斗规则与结果评定
- **触发条件**: 队伍遭遇战斗节点(山贼/天险/Boss)
- **操作步骤**:
  1. 进入战斗 → 全自动模式(AI控制技能释放，不可手动干预)
  2. 阵型克制生效(克制方全属性+10%)
  3. 战力判定: 差距±20%以内为"苦战"，超过±20%为"轻松"或"困难"
  4. 最多10回合(10回合内未分胜负判进攻方失败)
  5. 战斗结束显示4级评定:
     | 结果 | 条件 | 星级 |
     |------|------|:----:|
     | 大捷 | 剩余血量>50%且无武将阵亡 | ⭐⭐⭐ |
     | 小胜 | 剩余血量10%~50%或有武将阵亡 | ⭐⭐ |
     | 惨胜 | 剩余血量<10% | ⭐ |
     | 惜败 | 战斗失败 | 无 |
- **预期结果**: 全自动战斗节奏紧凑，星级评定激励追求完美通关，三星解锁扫荡形成长期目标
- **异常处理**:
  - 战斗全自动不可手动干预
  - 10回合超时判定惜败(进攻方失败)
  - 星级评定与血量/阵亡条件精确匹配
  - 三星通关标记正确(大捷×3或累计通关×3)
- **验证**: 战斗规则与PRD EXP-3一致 | 星级条件与PRD EXP-3一致

### 3.3 扫荡系统
- **触发条件**: 路线三星通关(大捷评级×3或累计通关×3)
- **操作步骤**:
  1. 路线三星通关后解锁扫荡按钮
  2. 选择扫荡类型:
     | 扫荡类型 | 消耗 | 收益 | 每日次数限制 |
     |---------|------|------|:----------:|
     | 普通扫荡 | 扫荡令×1 | 基础奖励×100% | 5次/路线 |
     | 高级扫荡 | 扫荡令×3 | 基础奖励×150%+保底稀有掉落 | 3次/路线 |
     | 免费扫荡 | 无 | 基础奖励×50% | 1次/路线 |
  3. 确认执行 → 即时结算展示奖励
- **预期结果**: 三种扫荡满足不同付费意愿，次数限制控制收益膨胀，扫荡收益≤手动保证手动玩法价值
- **异常处理**:
  - 未三星通关路线不可扫荡(按钮灰显)
  - 解锁条件满足"大捷×3或累计通关×3"之一即可
  - 每日次数独立计数且0:00重置
  - 高级扫荡保底至少1件稀有掉落
  - 免费扫荡奖励减半正确
  - 扫荡令不足时提示「扫荡令不足」
- **验证**: 扫荡类型/消耗/收益/次数与PRD EXP-4一致

## 4. 远征奖励

### 4.1 基础奖励与掉落
- **触发条件**: 通关节点/路线
- **操作步骤**:
  1. 通关节点/路线 → 结算面板展示奖励列表
  2. 基础奖励(按路线难度，引用PRD EXP-4):
     | 路线难度 | 粮草 | 铜钱 | 铁矿 | 装备碎片 | 经验值 |
     |---------|:----:|:----:|:----:|:-------:|:-----:|
     | 简单 | +200 | +400 | +1 | ×1 | +500 |
     | 普通 | +400 | +800 | +2 | ×2 | +1,200 |
     | 困难 | +600 | +1,200 | +4 | ×4 | +2,500 |
     | 奇袭 | +1,000 | +2,000 | +6 | ×6 | +5,000 |
  3. 掉落表(引用PRD EXP-4):
     | 掉落物 | 普通节点掉率 | BOSS节点掉率 | 奇袭路线BOSS掉率 |
     |-------|:----------:|:----------:|:--------------:|
     | 装备碎片 | 30% | 60% | 80% |
     | 武将碎片(随机) | 5% | 15% | 25% |
     | 技能书 | 2% | 10% | 20% |
     | 稀有材料 | — | 8% | 15% |
