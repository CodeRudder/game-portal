# v8.0 商贸繁荣 — Round 2 Play 文档

> **版本**: v8.0 商贸繁荣 | **轮次**: R2
> **引擎域**: engine/trade/(TradeSystem,CaravanSystem), engine/shop/(ShopSystem), engine/currency/(CurrencySystem)

---

## 1. 商贸路线开通与利润计算

1. 打开贸易地图 → 查看8座城市节点与商路连线
2. 选择未开通商路 → 检查前置条件(主城等级+前置商路) → 扣除开通费 → 开通成功
3. 查看商路利润公式: 利润 = (目标价格 - 进货价格) × 载重 - 路费
4. 验证: 商路开通条件校验正确、费用扣除、繁荣度初始30%

## 2. 商队派遣与运输

1. 选择已开通商路 → 选择运输商品(受载重限制) → 可选派遣护卫武将
2. 商队出发 → 沿路线移动 → 运输时间倒计时
3. 到达目标城市 → 自动交易 → 货物存入仓库 → 铜钱入账
4. 验证: 护卫武将降低遇盗概率50%、武将互斥规则、离线继续运输

## 3. 商店购买流程

1. 打开集市商店 → 分类Tab切换(全部/资源/加速/特殊)
2. 点击商品卡片 → 查看价格/库存/折扣角标 → 选择数量
3. 触发购买确认(五级确认策略) → CurrencySystem检查余额
4. 余额充足 → 扣款 → 库存更新 → 物品入背包 → Toast反馈
5. 余额不足 → 价格变红+抖动 → 提示获取途径
6. 验证: 限购规则(每日/终身)、折扣计算、自动补货(8h周期)

## 4. 货币兑换与消耗优先级

1. 查看底部货币栏 → 8种货币余额(铜钱/天命/招贤榜/求贤令/远征币/公会币/声望值/元宝)
2. 集市购买 → 优先消耗铜钱
3. 限时特惠 → 仅消耗元宝
4. NPC交易 → 以物易物+铜钱补差
5. 验证: 消耗优先级按商店类型区分、汇率转换正确、不足检测+引导

## 5. 价格波动与贸易事件

1. 等待6h → 商品价格刷新(实际价格 = 基础价格 × (1 ± 波动率))
2. 连续涨跌不超过3次 → 防止极端价格
3. 商队运输途中触发0~2个随机事件
4. 有护卫: 山贼/暴雨自动处理(无损失)
5. 无护卫: 选择处理方式(缴纳/战斗/贿赂)
6. 验证: 价格波动上下限、事件触发概率、繁荣度随贸易提升
