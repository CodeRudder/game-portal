# v8.0 商贸繁荣 — 玩游戏流程

## 1. 商贸系统

### 1.1 集市商店浏览与购买
- 操作: 进入集市商店 → 切换分类Tab(全部/装备/材料/消耗品) → 点击商品卡片 → 查看详情 → 点击购买 → 确认弹窗 → 扣款获得
- 验证: 商品网格正确展示图标/名称/价格/库存/折扣角标；分类Tab切换过滤正确；购买后库存-1、货币扣除
- 预期: 完整购买流程 选择→确认→扣款→获得→反馈，五级确认策略按金额档位触发

### 1.2 库存与限购
- 操作: 购买限购商品至上限 → 检查商品显示"已售罄" → 等待补货周期(8h) → 验证补货
- 验证: 每日限购/终身限购计数正确；售罄商品置灰不可点击；自动补货引擎按8h周期刷新
- 预期: 常驻商品不缺货，随机/折扣商品库存有限，离线补货最多累积2次(16h上限)

### 1.3 货币体系
- 操作: 查看底部货币栏 → 确认8种货币(铜钱/天命/招贤榜/求贤令/远征币/公会币/声望值/元宝)余额 → 购买不同商店商品
- 验证: 货币按商店类型消耗优先级扣除(集市优先铜钱、限时特惠仅元宝)；余额不足时价格变红+抖动+获取途径引导
- 预期: CurrencySystem统一管理，所有货币操作通过接口，不足提示清晰引导

## 2. 贸易路线

### 2.1 商路开通
- 操作: 进入贸易路线地图 → 查看城市节点(8座)与商路连线 → 选择未开通商路 → 检查前置条件(主城等级+前置商路) → 扣除开通费 → 开通
- 验证: 开通后商路连线点亮；对应城市商品解锁；繁荣度初始30%
- 预期: 8条商路逐级解锁，开通条件清晰，繁荣度影响产出倍率

### 2.2 商队派遣
- 操作: 选择已开通商路 → 选择运输商品(按载重上限) → 可选派遣护卫武将(每商队1名) → 商队出发 → 等待运输完成
- 验证: 商队沿路线移动动画；运输时间倒计时；护卫武将不可重复派遣(互斥规则)
- 预期: 到达后自动交易→货物存入仓库→铜钱入账；离线期间继续运输并自动完成

## 3. 价格波动

### 3.1 行情刷新
- 操作: 记录当前商品价格 → 等待6h刷新周期 → 对比新旧价格 → 确认波动范围
- 验证: 实际价格 = 基础价格 × (1 ± 波动率)；连续同方向涨跌不超过3次
- 预期: 价格在合理区间波动，防止极端暴利/暴亏，科技「商业天赋」可预览下次趋势

### 3.2 低买高卖策略
- 操作: 在低价城市买入商品 → 派商队运至高价城市卖出 → 计算利润差
- 验证: 利润 = 卖出价 × 数量 - 买入价 × 数量 - 运输成本；繁荣度越高产出倍率越大
- 预期: 贸易收入占比 早期10%→中期30%→后期40%，低买高卖提供策略成就感

## 4. 商店系统

### 4.1 黑市与NPC商人
- 操作: 繁荣度提升至解锁条件 → 触发NPC特殊商人出现(5种) → 以物易物+铜钱补差交易
- 验证: NPC商人出现条件正确；交易规则按NPC类型区分；商品为稀有/限定物品
- 预期: NPC商人增加惊喜感，繁荣度驱动长期贸易目标

### 4.2 限时特惠与收藏
- 操作: 查看限时特惠商品(仅元宝) → 收藏感兴趣商品 → 等待降价 → 收到降价提醒(红点脉动+Toast)
- 验证: 限时商品倒计时显示；收藏列表分组展示；降价/上架提醒及时触发
- 预期: 限时商品制造紧迫感，收藏系统减少信息遗漏，折扣机制含常规/NPC好感度折扣

### 4.3 自动补货与手动刷新
- 操作: 消耗完随机/折扣商品库存 → 手动刷新(如有刷新次数) → 或等待8h自动补货
- 验证: 手动刷新立即补充新商品；自动补货按周期执行；离线补货累积不超过2次
- 预期: 补货机制保证商店持续可玩，手动刷新提供主动控制感

## 5. 交叉验证

### 5.1 商店购买→货币→库存联动
- 操作: 在集市购买商品 → 验证货币扣除(按优先级) → 验证库存减少 → 验证背包获得物品
- 预期: 购买→扣款→库存→背包 全链路数据一致，网络异常时回滚不丢货币

### 5.2 贸易→商店→繁荣度闭环
- 操作: 派遣商队完成贸易 → 验证铜钱收入+繁荣度提升 → 繁荣度达标解锁NPC商人 → NPC商人出售稀有商品 → 购买后战力提升
- 预期: 核心循环 挂机产出→商店消费→战力提升→解锁新商路→更多利润→更强实力 贯通

### 5.3 离线回归验证
- 操作: 派遣商队后离线 → 等待超过运输时间 → 上线检查 → 商队已完成交易+铜钱入账+商店补货
- 验证: 离线期间贸易事件默认保守选项(缴纳/等待)；商店离线补货最多2次；贸易利润正确计算
- 预期: 离线体验流畅，无数据丢失，回归有收益反馈
