## 1. 商贸系统

### 1.1 集市商店浏览与购买
- **触发条件**: 主城≥Lv.1，集市建筑已建造
- **操作**: 进入集市商店 → 切换分类Tab(全部/资源兑换/道具购买/限时特惠/我的收藏) → 点击商品卡片 → 查看详情 → 点击购买 → 确认弹窗 → 扣款获得
- **验证**: 商品网格正确展示图标/名称/价格/库存/折扣角标/限购角标/收藏按钮/NEW角标(3秒淡出)；分类Tab切换过滤正确；购买后库存-1、货币扣除
- **预期**: 完整购买流程 选择→确认→扣款→获得→反馈，五级确认策略按金额档位触发
- **异常处理**: 商品已售罄→购买按钮灰化+Toast"已售罄"；网络超时→本地缓存请求，重连后自动提交；购买失败→回滚本地状态+Toast显示失败原因
- **对应**: F8.01, F8.05, F8.06

### 1.2 五级确认策略逐级验证
- **触发条件**: 拥有对应数量的货币
- **操作L0**: 购买铜钱≤200的基础商品（如粮草箱）→ 直接购买，Toast提示，无弹窗
- **操作L1**: 购买铜钱201~2000的商品 → 卡片下方展开内联确认条 → 可勾选"不再提示"跳过
- **操作L2**: 购买铜钱>2000或消耗兵力/粮草的商品 → 弹出标准确认弹窗，不可跳过
- **操作L3**: 购买消耗元宝的商品（如限时特惠）→ 弹窗+3秒倒计时确认
- **操作L4**: 购买元宝≥500或限定商品 → 弹窗+滑动确认
- **验证**: 每级确认策略正确触发，不可跳过级别无绕过途径
- **异常处理**: 倒计时期间关闭弹窗→取消购买，不扣款；滑动确认未完成→弹窗保持
- **对应**: F8.08

### 1.3 误操作防护验证
- **触发条件**: 任意购买场景
- **操作1**: 快速连续点击同一商品购买按钮 → 验证500ms内仅响应首次点击
- **操作2**: 购买弹窗打开时切换Tab → 弹出"购买进行中，确认离开？"
- **操作3**: 购买时模拟网络延迟 → 按钮loading旋转+"处理中…"禁止重复点击
- **操作4**: 购买请求超时 → 本地缓存请求，重连后自动提交
- **操作5**: 购买失败（服务端拒绝）→ 回滚本地状态+Toast显示失败原因
- **验证**: 5种误操作防护场景均生效，无货币丢失、无重复购买
- **异常处理**: 服务端幂等处理+本地500ms节流，双重防重复
- **对应**: F8.08

### 1.4 库存与限购
- **触发条件**: 商品有库存且玩家有限购次数
- **操作**: 购买限购商品至上限 → 检查商品显示"已售罄" → 等待补货周期(8h) → 验证补货
- **验证**: 每日限购(0点重置)/终身限购计数正确，显示"已购2/3"；售罄商品灰化不可点击；好感度限购未解锁显示🔒
- **预期**: 常驻商品无限库存(显示×99)，随机商品库存5~30件，折扣商品1~5件，离线补货最多累积2次(16h上限)
- **异常处理**: 购买时库存恰好被其他渠道消耗→提示"商品已售罄"+刷新商品列表
- **对应**: F8.09

### 1.5 折扣机制完整验证
- **触发条件**: 各类折扣触发条件满足
- **操作1**: 等待商店8h刷新 → 验证20%概率出现常规折扣(-10%~-20%)，持续至下次刷新
- **操作2**: 查看每日限时特惠 → 验证折扣幅度-30%~-60%，持续24h
- **操作3**: 触发黑市NPC → 验证黑市折扣-40%~-70%，持续2h
- **操作4**: 提升NPC好感度至各等级 → 验证折扣（见§1.5.1好感度等级统一映射）
- **操作5**: 参与节日/运营活动 → 验证活动折扣-20%~-50%
- **验证**: 各类折扣互斥或叠加规则正确，折扣角标"火焰红色"正确显示折扣百分比
- **异常处理**: 折扣过期前5分钟→角标闪烁提醒；折扣期间商品售罄→折扣保留至补货
- **对应**: F8.07

#### 1.5.1 NPC好感度等级与折扣统一映射 【R2新增】
> **Gap修复**: G-18 — 统一NPC-PRD(⭐等级制)与SHP-PRD(❤️等级制)的数值映射

NPC-PRD使用⭐等级制(好感度点数0/100/300/600/1000)，SHP-PRD使用❤️等级制(5级名称)。统一映射如下：

| NPC-PRD等级 | 好感度点数 | SHP-PRD等级 | 折扣 | 商品解锁 | 特殊效果 |
|:---------:|:---------:|:---------:|:----:|---------|---------|
| Lv.1 ⭐ | 0~99 | ♡（陌生） | 无折扣 | 基础商品 | — |
| Lv.2 ⭐⭐ | 100~299 | ❤️♡（熟悉） | -5% | +1稀有商品 | 偶尔赠送小礼物 |
| Lv.3 ⭐⭐⭐ | 300~599 | ❤️❤️♡（信赖） | -10% | +2稀有商品 | 优先通知黑市出现 |
| Lv.4 ⭐⭐⭐⭐ | 600~999 | ❤️❤️❤️♡（亲密） | -15% | +3稀有商品 | 每日免费赠送1件基础商品 |
| Lv.5 ⭐⭐⭐⭐⭐ | 1000+ | ❤️❤️❤️❤️❤️（挚友） | -20% | 全部解锁 | 专属限定商品+羁绊技能 |

- **验证**: NPC-PRD好感度点数达到阈值时，SHP-PRD折扣等级同步生效
- **跨系统一致性**: NPC-PRD好感度只增不减，转生100%保留；SHP-PRD折扣永久有效
- **对应**: F8.07, F8.15

### 1.6 货币体系
- **触发条件**: 进入任意商店界面
- **操作**: 查看底部货币栏 → 确认8种常驻货币(💰铜钱/👑天命/📜招贤榜/🏷️求贤令/⚔️远征币/🏛️公会币/🌟声望值/💎元宝)余额 → 购买不同商店商品
- **验证**: 货币按商店类型消耗优先级扣除(集市优先铜钱、限时特惠仅元宝、军需处兵力+铜钱组合、NPC交易以物易物+铜钱补差、活动商店仅活动代币)；余额不足时价格变红+抖动+获取途径引导
- **预期**: CurrencySystem统一管理，所有货币操作通过接口，不足提示清晰引导
- **异常处理**: 货币余额在操作期间被其他系统消耗→实时检查余额，不足时中断操作+提示
- **对应**: F8.04, F8.22, F8.23, F8.24

### 1.7 货币兑换与汇率
- **触发条件**: 拥有足够的源货币
- **操作1**: 在集市花费500💰铜钱购买1张📜招贤榜 → 验证扣除和获得
- **操作2**: 使用100👑天命兑换100🏷️求贤令 → 验证等价兑换
- **操作3**: 使用100💎元宝购买100🏷️求贤令 → 验证等价兑换
- **验证**: 汇率正确(500💰=1📜, 100👑=100🏷️, 100💎=100🏷️)，反向兑换不可行(求贤令不可换回元宝，招贤榜不可换回铜钱)
- **预期**: 所有兑换为单向不可逆，CurrencySystem记录兑换流水
- **异常处理**: 【R2新增】
  - 源货币不足→兑换按钮灰化+Toast"XX不足"
  - 目标货币达上限(招贤榜堆叠上限99)→Toast"招贤榜已达上限"+兑换按钮灰化
  - 兑换过程中网络中断→本地缓存请求，重连后自动提交+幂等校验
  - 兑换成功→记录兑换流水(时间/源货币/目标货币/数量/汇率)
  - 短时间大量兑换(1分钟内>10次)→弹出确认"确认批量兑换？"
- **对应**: F8.25

### 1.8 货币防通胀与转生影响
- **触发条件**: 游戏运行中/执行转生操作
- **操作1**: 验证建筑升级费用指数增长(×1.8/级) → 铜钱持续紧缺
- **操作2**: 验证招贤榜堆叠上限99 → 达上限后不可再获取
- **操作3**: 执行转生 → 验证天命保留30%/声望等级归零(累计值保留，参考§9.3统一声明)/其他货币全额保留
- **验证**: 防通胀三层机制(L1消耗膨胀/L2获取限制/L3重置机制)生效
- **预期**: 各货币经济循环健康，无无限通胀风险
- **异常处理**: 转生时货币处理失败→事务回滚，保持转生前状态+提示重试
- **对应**: F8.26

## 2. 多商店类型

### 2.1 军需处商店
- **触发条件**: 主城≥Lv.3，兵营已建造
- **操作**: 进入城池「兵营」建筑 → 点击军需处 → 浏览军事道具/装备/兵书 → 选择商品 → 确认购买(兵力+铜钱组合扣除)
- **验证**: 主货币为兵力+铜钱，任一不足不可购买；商品每日0点刷新；离线可补货
- **预期**: 军需处提供军事类商品专属消费出口，兵力与铜钱双货币消耗形成策略选择
- **异常处理**: 兵力不足→灰色兵力图标+Toast"兵力不足，请先征兵"；铜钱不足→价格变红+抖动
- **对应**: F8.12

### 2.2 黑市商店
- **触发条件**: NPC随机触发（每日5%概率，参考NPC-PRD稀有NPC刷新规则）
- **操作**: 等待NPC随机触发黑市 → 进入黑市(持续2h) → 浏览稀有道具/折扣品(3~8件限量) → 购买 → 黑市消失
- **验证**: 黑市折扣-40%~-70%；错过不补货(离线不补货)；限量商品售罄灰化
- **预期**: 黑市制造限时紧迫感，稀有商品高性价比驱动活跃度
- **异常处理**: 黑市到期时正在购买→弹出"黑市即将关闭，是否完成购买？"→30秒倒计时自动关闭；离线期间黑市出现→回归后显示"黑市已结束"+提示下次出现时间
- **对应**: F8.13

### 2.3 活动商店
- **触发条件**: 限时活动开启
- **操作**: 参与限时活动获取活动代币(🎫赛季令牌/🪙活动铜钱/🪙日常代币/🏮节庆灯笼/🏅联盟勋章) → 进入活动商店 → 使用对应代币购买限定道具
- **验证**: 不同活动使用不同代币；活动到期后代币10%转铜钱结算；日常代币0点过期
- **预期**: 活动商店与活动代币形成限时消费闭环，过期结算避免资源浪费
- **异常处理**: 活动代币不足→显示当前代币数/所需代币数+引导完成任务；活动到期时购买中→强制结算+未完成购买退回代币
- **对应**: F8.14

### 2.4 NPC交易商店
- **触发条件**: 主城≥Lv.5，市集已建，NPC商人类型在地图上存在
- **操作**: 在城池地图找到NPC商人 → 对话选择"交易" → 以物易物+铜钱补差 → 完成交易
- **验证**: NPC好感度影响价格折扣(-5%~-20%)和商品解锁数量（见§1.5.1统一映射）；好感度限购未解锁显示🔒
- **预期**: NPC好感度系统驱动长期社交互动，好感度不降形成永久激励
- **异常处理**: NPC处于休息/睡眠状态→交易按钮灰化+提示NPC当前状态；以物易物时物品不足→补差铜钱也不足→提示"物品和铜钱均不足"
- **对应**: F8.15

## 3. 贸易路线

### 3.1 商路开通
- **触发条件**: 主城≥Lv.6 + 资源建筑≥2 + 新手引导步骤3完成（注意：TUT-PRD步骤3为"招募武将"，贸易解锁为扩展引导，主城≥Lv.6时自动触发）
- **操作**: 进入贸易路线地图 → 查看城市节点(8座: 洛阳/长安/成都/许昌/邺城/建业/南中/会稽)与商路连线 → 选择未开通商路 → 检查前置条件(主城等级+前置商路) → 扣除开通费 → 开通
- **验证**: 开通后商路连线点亮；对应城市商品解锁；繁荣度初始30%(萧条等级)
- **预期**: 8条商路逐级解锁，开通条件清晰(洛阳→许昌2K/洛阳→长安3K/长安→成都5K/成都→南中4K/许昌→邺城3.5K/许昌→建业6K/建业→会稽2.5K/邺城→建业8K)，繁荣度影响产出倍率
- **异常处理**: 开通费不足→提示"铜钱不足"+引导获取途径；前置商路未开通→灰化+提示需先开通XX商路
- **对应**: F8.16, F8.19

### 3.2 商品系统完整验证
- **触发条件**: 至少1条商路已开通
- **操作**: 查看各城市特产和需求商品 → 验证10种商品(🌾粮草/🪵木材/⛏️铁矿/📜书籍/🍶杜康酒/🗡️兵器/🐎西凉马/🌸蜀锦/🌿药材/💎和田玉)基础价格和波动范围
- **验证**: 基础商品(粮草10/木材15/铁矿25)波动±15~20%；精品(书籍50/杜康酒80/兵器100)波动±25~35%；稀有(西凉马200/蜀锦150/药材120)波动±30~40%；传说(和田玉500)波动±50%，仅长安极低概率产出
- **预期**: 商品分层清晰，越稀有波动越大利润越高
- **异常处理**: 商品价格异常(超出波动范围)→系统自动修正+记录异常日志
- **对应**: F8.17

### 3.3 商队管理
- **触发条件**: 至少1条商路已开通
- **操作**: 查看商队列表 → 确认商队数量上限(主城6~9级2队/10~14级4队/15~19级6队/20+级8队) → 查看商队属性(载重50/速度/护卫/议价能力)
- **验证**: 商队状态正确(🐫运输中/⏳等待/✓已到达/🔒已锁定)；载重决定单趟运输数量；速度影响运输时间
- **预期**: 商队数量随主城等级增加，属性可通过科技研究/武将加成提升
- **异常处理**: 所有商队均在运输中→提示"无空闲商队"+显示最早到达时间
- **对应**: F8.18

### 3.4 商队属性提升
- **触发条件**: 对应科技已研究/武将已派遣
- **操作1**: 研究科技「市舶司」(经济5B) → 验证交易利润+25%，铜钱产出+20%
- **操作2**: 研究科技「通宝令」(经济6B) → 验证商店折扣+20%，出售价格+30%
- **操作3**: 研究科技「富甲天下」(经济7B) → 验证全铜钱产出+50%，交易利润+40%
- **操作4**: 派遣高战力武将护卫 → 验证护卫效果随武将战力增强
- **操作5**: 使用加速道具/提升道路等级 → 验证运输时间缩短
- **验证**: 科技/武将加成正确影响商队属性，属性面板实时更新
- **预期**: 商队属性成长线提供长期养成目标，经济路线B分支与贸易深度联动
- **异常处理**: 科技未研究但UI显示加成→检查科技状态同步；武将被释放但护卫效果未消失→检查武将状态锁
- **对应**: F8.21

### 3.5 商队派遣与运输
- **触发条件**: 至少1条商路已开通 + 至少1支空闲商队
- **操作**: 选择已开通商路 → 选择运输商品(按载重上限) → 可选派遣护卫武将(每商队1名) → 商队出发 → 等待运输完成
- **验证**: 商队沿路线移动动画；运输时间按距离(短途2~3h/中途4~6h/长途6~8h)；护卫武将不可重复派遣(互斥规则：护卫武将不可同时编入远征队伍)
- **预期**: 到达后自动交易→货物存入仓库→铜钱入账；离线期间继续运输并自动完成
- **异常处理**: 派遣时商路突然断绝(事件导致)→提示"商路暂时不可用"+建议选择其他商路；载重超出→自动截断至载重上限+提示
- **对应**: F8.18, F8.20

### 3.6 自动贸易
- **触发条件**: 【R2修正】科技「富甲天下」(经济7B)研究完成 → 自动解锁自动贸易功能
