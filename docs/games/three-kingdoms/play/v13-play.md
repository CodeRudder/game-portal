# v13.0 联盟争霸 — 玩游戏流程

> 核心循环: 创建/加入联盟 → 成员管理与贡献 → 联盟战争对抗 → 科技升级与商店兑换 → 赛季结算奖励

---

## 1. 联盟系统

### 1.1 联盟创建与加入
- 操作: 点击主界面"联盟"入口 → 无联盟时进入联盟大厅 → 顶部Tab切换"推荐联盟/搜索/创建" → 搜索支持联盟名称/ID精确查找 → 申请加入后等待盟主或军师审批(申请有效期72h) → 创建联盟需满足: 主城≥10级+元宝×200+个人战力≥5000 → 创建时设定联盟名称(2~8字)/公告/加入条件(开放/审批/战力门槛)
- 验证: 搜索结果实时匹配 | 已满联盟不可申请 | 创建条件不满足时按钮灰显 | 同一玩家仅可加入1个联盟 | 退出联盟后24h冷却期
- 预期: 联盟大厅信息密度适中，搜索精准高效，创建门槛筛选活跃玩家，冷却期防止频繁跳槽

### 1.2 联盟等级与福利
- 操作: 联盟面板显示当前等级/经验/成员数/战力总和 → 成员完成联盟任务、捐献、战争均可获得联盟经验 → 等级提升解锁: Lv2成员上限30→40/Lv3开启联盟科技/Lv5开启联盟商店/Lv8联盟战争额外加成 → 退出联盟时个人贡献值保留50%，联盟等级不变
- 验证: 经验获取量与配置表一致 | 等级阈值正确触发解锁 | 成员上限硬性限制不可突破 | 退出贡献保留50%数值准确
- 预期: 联盟等级提供长期成长目标，福利解锁激励全员参与，贡献保留机制降低退出成本

### 1.3 联盟频道与公告
- 操作: 联盟面板内嵌"联盟频道"Tab → 成员实时发送文字/表情消息 → 盟主/军师可发布置顶公告(最多3条) → 公告支持编辑/删除/排序 → 聊天记录保留最近7天 → @成员触发推送通知 → 新成员入盟自动欢迎消息
- 验证: 消息发送延迟<1s | 置顶公告不超过3条 | 7天前消息自动清理 | @通知正确送达目标玩家 | 被踢成员无法继续发言
- 预期: 联盟频道促进社交粘性，公告功能保证关键信息传达，聊天记录7天留存平衡存储与体验

## 2. 联盟管理

### 2.1 成员权限与角色
- 操作: 联盟面板"成员"Tab → 显示成员列表(头像/名称/职位/贡献/战力/最后在线) → 职位三级: 盟主(全部权限)/军师(审批/踢人/公告/科技/商店)/成员(基础权限) → 盟主可任命/撤销军师(上限3名) → 盟主7天未登录自动转让(按贡献排序)
- 验证: 权限边界严格(成员不可踢人/改公告) | 军师上限3名 | 自动转让逻辑正确 | 转让后原盟主降为军师
- 预期: 三级权限清晰分工，军师协助管理减轻盟主负担，自动转让防止联盟陷入无人管理

### 2.2 成员审批与踢出
- 操作: 盟主/军师进入"申请列表" → 查看申请人信息(等级/战力/上次联盟/活跃度) → 批准/拒绝/忽略 → 踢人操作: 选中成员→确认踢出→被踢者收到邮件通知 → 被踢成员12h内不可重新申请同一联盟 → 联盟人数达到上限时自动拒绝新申请
- 验证: 审批后成员立即加入并显示在列表 | 踢出后成员数即时更新 | 12h冷却期正确执行 | 满员自动拒绝逻辑生效
- 预期: 审批流程高效可控，踢人机制维护联盟质量，冷却期防止恶意骚扰

### 2.3 联盟捐献与贡献
- 操作: 成员每日可进行联盟捐献 → 铜钱捐献(免费1次+额外3次，每次+10贡献)/元宝捐献(每日5次，每次+50贡献)/高级捐献(元宝×200，每日1次，+200贡献) → 捐献同时获得个人贡献值和联盟经验 → 贡献值用于联盟商店消费和科技加成
- 验证: 每日捐献次数独立计数且0:00重置 | 贡献值增减实时同步 | 联盟经验按比例获取 | 免费捐献不消耗资源
- 预期: 多档捐献满足不同付费意愿，免费捐献保证零氪玩家参与，贡献值串联商店与科技系统

## 3. 联盟战争

### 3.1 联盟Boss讨伐
- 操作: 盟主/军师开启联盟Boss(每日1次，Lv3解锁) → 全员参与讨伐(6h限时) → 个人挑战次数: 普通3次/元宝额外2次 → Boss等级随联盟等级提升(Lv1~Lv10) → 伤害排行榜实时更新 → 击杀后全员获得奖励(击杀奖励+参与奖励) → 未击杀按伤害比例发放参与奖励
- 验证: 开启次数每日1次且0:00重置 | 个人挑战次数扣减准确 | Boss血量全联盟共享同步 | 伤害排行数据实时刷新 | 奖励按伤害比例正确分配
- 预期: Boss讨伐凝聚联盟向心力，伤害排行激发竞争，限时机制制造紧迫感，全员奖励保证参与感

### 3.2 联盟对战
- 操作: 每周六20:00~21:00开启联盟对战(系统匹配战力相近联盟) → 匹配规则: 按联盟总战力±15%范围匹配 → 对战模式: 1v1车轮战(双方各派5名代表) → 出战顺序由盟主/军师编排 → 战斗自动进行，阵型克制生效 → 胜方联盟获得联盟积分+全员元宝×30+贡献×100
- 验证: 匹配双方战力差≤15% | 出战顺序锁定后不可更改 | 战斗结果与战斗系统一致 | 积分与奖励准确发放
- 预期: 联盟对战提供周常竞技高潮，匹配公平保证体验，车轮战增加策略编排深度

### 3.3 联盟排行榜与赛季
- 操作: 联盟面板"排行"Tab → 联盟战力榜/联盟Boss伤害榜/联盟对战积分榜 → 每赛季(28天)结算 → 赛季奖励按排名梯度发放: Top3联盟全员专属称号+头像框+元宝×200/Top4~10元宝×100+贡献×500/其余参与联盟贡献×200 → 赛季结束时弹窗展示本季成就与奖励
- 验证: 排行数据每5min刷新 | 赛季结算时间精确 | 奖励按排名梯度正确发放 | 专属称号/头像框赛季结束后可继续佩戴
- 预期: 排行榜激发联盟荣誉感，赛季节奏保持长期活跃，梯度奖励保证头部与中腰部联盟均有收获

## 4. 联盟科技与商店

### 4.1 联盟科技树
- 操作: 联盟面板"科技"Tab(Lv3解锁) → 展示科技树(攻击/防御/资源/特殊4条分支) → 消耗联盟贡献值升级科技 → 攻击分支: 全员攻击+2%/4%/6%/8%/10%(每级需贡献×500) → 防御分支: 全员防御+2%/4%/6%/8%/10% → 资源分支: 粮草/铜钱产出+5%/10%/15% → 特殊分支: 远征收益+5%/Boss伤害+5%/商店折扣5% → 科技效果对全员生效，退出联盟后失效
- 验证: 科技等级与消耗贡献值匹配 | 效果数值对全员正确加成 | 贡献值扣除后余额不为负 | 退出联盟后科技加成立即移除
- 预期: 科技树提供联盟集体成长线，4条分支满足不同发展路线，全员受益激励共同投入

### 4.2 联盟商店
- 操作: 联盟面板"商店"Tab(Lv5解锁) → 商品列表: 武将碎片/装备箱/技能书/征兵令/扫荡令/联盟专属头像框 → 使用贡献值兑换 → 部分商品需联盟等级解锁(Lv5/Lv8/Lv10) → 热门商品限购(周限2~5件) → 每周一0:00刷新限购与部分商品
- 验证: 贡献值余额充足才可兑换 | 限购次数递减且周一重置 | 等级未达商品灰显不可购买 | 兑换物品正确入背包
- 预期: 联盟商店是贡献值核心出口，等级解锁商品激励联盟升级，限购控制资源膨胀

### 4.3 联盟任务与活跃度
- 操作: 联盟面板"任务"Tab → 每日联盟任务: 捐献1次/参与Boss/完成1场对战/聊天发言等(每项+20联盟经验+10贡献) → 每周联盟任务: 累计捐献7次/Boss击杀1次/对战胜利1场(每项+100联盟经验+50贡献) → 全员任务完成度达80%触发联盟宝箱(全员额外奖励)
- 验证: 任务进度实时更新 | 奖励发放准确 | 80%完成度阈值正确触发宝箱 | 宝箱奖励全员到账
- 预期: 日常任务引导活跃行为，80%宝箱激励全员参与，贡献+经验双奖励串联成长体系

## 5. 交叉验证

### 5.1 加入→捐献→科技→商店全链路
- 操作: 玩家创建联盟(Lv10+元宝×200) → 招募4名成员达到5人 → 每日铜钱捐献+元宝捐献累计贡献×300 → 联盟升至Lv3解锁科技 → 升级攻击科技Lv1(消耗贡献×500) → 全员战斗中攻击属性+2% → 联盟升至Lv5解锁商店 → 贡献值兑换武将碎片
- 预期: 创建→捐献→升级→科技→商店全链路数据一致，科技加成在战斗中正确体现，贡献值收支平衡

### 5.2 Boss讨伐→排行→奖励→商店资源闭环
- 操作: 盟主开启Lv5 Boss → 5名成员各挑战3次(共15次攻击) → 累计伤害击杀Boss → 伤害排行展示 → 全员领取击杀奖励(贡献×150+联盟经验×200) → 贡献值用于商店兑换扫荡令 → 扫荡令用于远征获取资源 → 资源强化武将 → 下次Boss伤害提升
- 预期: Boss讨伐→贡献→商店→强化→再讨伐形成正循环，奖励数据与配置一致，排行激发持续投入

### 5.3 联盟对战→赛季→科技→活跃度闭环
- 操作: 联盟周六参与对战(匹配战力相近对手) → 盟主编排5名代表出战 → 车轮战胜利获得积分+贡献 → 28天赛季累计积分排名Top3 → 全员领取赛季专属称号+元宝×200 → 赛季奖励元宝用于高级捐献 → 贡献升级特殊科技(商店折扣5%) → 商店兑换效率提升 → 成员活跃度提升
- 预期: 对战→积分→赛季→捐献→科技→商店形成完整闭环，赛季节奏保持联盟长期活跃，科技与商店联动提升联盟整体实力
