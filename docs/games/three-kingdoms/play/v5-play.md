# v5.0 百家争鸣 — 玩游戏流程

## 1. 科技系统深化

### 1.1 科技树浏览与详情
- 操作: 点击底部导航→科技Tab→查看三条路线(军事/经济/文化)
- 验证: 节点按层级排列，连线清晰，状态标识正确(locked灰/available亮/researching动画/completed勾)
- 预期: 每条路线4层×2节点=8节点，共24个基础节点，详情弹窗显示效果/前置/消耗/时间

### 1.2 科技研究
- 操作: 选择available节点→确认研究→扣除科技点→等待/加速完成
- 验证: 研究进度条倒计时，科技点扣除正确，队列容量随书院等级提升(L1=1槽→L20=5槽)
- 预期: 研究完成后效果自动注册，属性加成即时生效

### 1.3 互斥分支选择
- 操作: 浏览Tier1/Tier3互斥节点→选择并研究其中一个(如锐兵术)
- 验证: 选中节点进入研究，同组另一节点(铁壁术)标记为mutex-locked永久锁定
- 预期: 6组互斥选择不可逆(转生除外)，每次游戏体验因选择不同产生差异

### 1.4 融合科技
- 操作: 完成跨路线前置科技→解锁融合科技节点→研究融合科技
- 验证: FusionTechSystem检测跨路线组合条件，融合节点显示所需前置路线
- 预期: 融合科技提供组合加成效果，叠加于基础科技之上

### 1.5 科技联动效果
- 操作: 研究经济路线科技→查看建筑产出；研究军事路线科技→查看武将属性；研究文化路线科技→查看研究速度
- 验证: TechLinkSystem将科技效果传递给建筑/资源/武将系统
- 预期: 经济→建筑产出+资源增产，军事→攻击/防御/生命乘数，文化→经验加成+研究速度缩短

### 1.6 离线研究回归
- 操作: 有研究中科技时关闭游戏→离线一段时间→回归游戏
- 验证: TechOfflineSystem按效率衰减计算(0~2h:100% | 2~8h:70% | 8~24h:40% | >24h:20%，封顶72h)
- 预期: 回归弹窗显示离线时长、各科技进度、获得科技点，已完成科技自动标记completed

## 2. 世界地图基础

### 2.1 地图渲染与浏览
- 操作: 点击天下地图入口→进入世界地图视图(1280×696px)
- 验证: 格子系统渲染正确，视口范围可拖拽/缩放，三大区域(中原/江南/西蜀)颜色区分
- 预期: 6种地形底纹(平原/山地/水域/森林/沙漠/雪地)清晰可辨

### 2.2 特殊地标
- 操作: 浏览地图→查看城池/关卡/资源点标记
- 验证: 地标图标正确显示，点击地标弹出简要信息(名称/等级/归属)
- 预期: 城池、关卡、资源点三类地标各有独立图标和交互逻辑

### 2.3 地图筛选过滤
- 操作: 使用筛选栏→按区域/地形/占领状态筛选
- 验证: 筛选条件即时生效，不符合条件的区域/地标淡化或隐藏
- 预期: 快速定位目标区域，辅助攻城决策

## 3. 领土系统

### 3.1 领土占领
- 操作: 攻城胜利→领土归属变更→地图上领土颜色更新
- 验证: TerritorySystem记录占领/产出/等级/归属，领土边界着色正确
- 预期: 己方领土提供持续资源产出，占领越多产出越高

### 3.2 领土产出与升级
- 操作: 查看己方领土→查看产出气泡→选择领土升级
- 验证: 产出气泡常驻显示资源信息，升级消耗资源并提升产出
- 预期: 领土等级越高产出越多，产出气泡数值实时更新

### 3.3 收益热力图
- 操作: 切换热力图模式→查看地图着色显示产出密度
- 验证: 高产出区域颜色偏暖(红/橙)，低产出区域偏冷(蓝/绿)
- 预期: 直观展示全局收益分布，辅助资源规划

## 4. 攻城战基础

### 4.1 攻城条件检查
- 操作: 选择相邻敌方领土→点击攻城→系统检查兵力/粮草
- 验证: SiegeSystem校验兵力充足+粮草足够+领土相邻，条件不满足时提示原因
- 预期: 不满足条件无法发起攻城，提示信息清晰引导玩家补充资源

### 4.2 攻城战斗与占领
- 操作: 满足条件→发起攻城→进入战斗→胜利后领土归属变更
- 验证: 战斗由BattleEngine执行，胜利后TerritorySystem更新归属，地图即时刷新
- 预期: 新领土纳入己方版图，产出开始累加，敌方失去该领土收益

## 5. 交叉验证

- 科技研究→属性加成→战斗伤害提升→攻城效率提高→领土扩张加速
- 经济科技→建筑产出增加→资源积累加快→科技点产出提升→研究加速
- 文化科技→研究速度缩短→更快解锁融合科技→跨路线组合加成
- 离线研究→回归补算进度→不浪费研究时间→保持发展节奏
- 领土扩张→产出增加→资源充裕→支撑更多科技研究和攻城行动
- 互斥分支选择→不同发展路线→影响攻城策略和领土发展优先级
