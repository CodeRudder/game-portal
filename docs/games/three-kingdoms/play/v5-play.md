# v5.0 百家争鸣 — 玩游戏流程

## 1. 科技树浏览
- 打开科技Tab → 查看科技树 → 选择科技节点 → 查看详情(效果/前置/消耗)
- 验证: 科技节点状态(已研究/可研究/锁定)、前置依赖连线、效果预览

## 2. 科技研究
- 选择可研究科技 → 确认消耗(科技点+铜钱) → 开始研究 → 等待/加速 → 完成
- 验证: 资源扣除正确、研究进度显示、完成通知

## 3. 融合科技
- 达到前置条件 → 解锁融合科技 → 研究 → 获得强力效果
- 验证: 融合科技解锁条件、效果叠加

## 4. 科技效果应用
- 研究完成 → 效果自动生效 → 查看加成面板
- 交叉验证: 科技加成→建筑加速/战斗强化/资源增产
