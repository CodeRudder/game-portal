## 1. 科技系统深化

### 1.1 科技树浏览与详情
- 操作: 点击底部导航→科技Tab→查看三条路线(军事/经济/文化)
- 验证: 节点按层级排列，连线清晰，状态标识正确(locked灰/available亮/researching动画/completed勾)
- 预期: 每条路线7层节点(前4层+互斥分支3层)，共21个基础节点+4个融合科技=25个科技节点
- PRD对齐: TEC-1.2 科技树结构 — 7层级节点，第5~7层互斥分支(A/B二选一)，画布缩放50%~150%
- 验收标准: 详情弹窗显示效果/前置/消耗/时间，画布支持缩放和拖拽平移

### 1.2 科技研究启动
- 操作: 选择available节点→查看详情弹窗(效果/前置/消耗/时间)→确认研究→扣除铜钱→等待/加速完成
- 验证: 研究启动条件三重校验——①科技点足够(基础费用×(1-书院研究速度加成)) ②前置科技已研究 ③无其他正在研究的科技(默认1槽位，VIP3解锁第2槽位)
- 预期: 研究进度条倒计时，铜钱扣除正确，队列容量随书院等级/VIP提升
- PRD对齐: TEC-2.1 研究流程 — 消耗铜钱启动(高级科技额外消耗天命)，离线也计入
- 验收标准: 研究完成后效果立即生效，属性加成即时注册

### 1.3 研究队列管理
- 操作: 研究中→点击队列→查看/管理研究队列
- 验证: 默认1个研究槽位，VIP3解锁第2槽位；当前研究完成后队列中下一项自动开始
- 预期: 支持取消研究(返还80%已消耗资源，进度清零)和切换研究(消耗100铜钱暂停当前，进度保留)
- PRD对齐: TEC-2.3 研究队列规则 — 取消/切换/自动续接机制
- 验收标准: 队列逻辑正确，资源返还/扣除准确

### 1.4 科技加速机制
- 操作: 研究中→点击加速→选择加速方式(铜钱加速/天命加速/元宝秒完成)
- 验证:
  - 铜钱加速: 立即完成剩余时间50%，消耗=铜钱×剩余时间(秒)×0.5，无限制
  - 天命加速: 立即完成剩余时间80%，消耗=天命×⌈剩余时间/3600⌉，每次研究最多1次
  - 元宝秒完成: 立即完成全部，消耗=元宝×⌈剩余时间/1800⌉，每日限3次
  - 书院等级加速: 研究速度+2%~+38%（被动，永久）
  - 声望等级加速: 研究速度×(1+声望等级×0.02)（被动，永久）
  - 联盟加速: 研究速度+10%（需联盟功能，**前置条件：玩家需加入联盟，由联盟成员协助触发**）
- 预期: 加速叠加公式: 实际研究时间 = 基础研究时间 / (1 + 书院加成 + 声望加成 + 联盟加成)；加速道具效果 = 实际研究时间 × (1 - 道具减免比例)
- PRD对齐: TEC-2 加速机制 — 6种加速方式+叠加公式
- 验收标准: 加速效果正确计算，限制条件严格执行

### 1.5 互斥分支选择
- 操作: 浏览Tier5/Tier6/Tier7互斥节点→选择并研究其中一个(如铁骑冲锋5A)
- 验证: 选中节点进入研究，同组另一节点(固若金汤5B)标记为mutex-locked永久锁定
- 预期: 3条路线×1组互斥=3组互斥选择不可逆(转生除外)，每次游戏体验因选择不同产生差异
- PRD对齐: TEC-1.3 互斥分支机制 — 军事(进攻/防御)、经济(重农/重商)、文化(仁政/霸道)
- 验收标准: 互斥锁定不可逆，转生可重选

### 1.6 融合科技
- 操作: 完成跨路线前置科技→解锁融合科技节点→研究融合科技
- 验证: FusionTechSystem检测跨路线组合条件，融合节点显示所需前置路线
- 预期: 4个融合科技——军经合一(军事Lv3+经济Lv3)、军文并举(军事Lv3+文化Lv3)、经世济民(经济Lv3+文化Lv3)、霸王之道(军事Lv5+经济Lv5+文化Lv5)
- PRD对齐: TEC-4 融合科技 — 解锁条件中"Lv.N"指第N个节点已完成(含分支)；融合科技不占基础路线槽位，可与基础科技同时研究(需第2槽位)
- 融合槽位规则:
  - 融合科技独立于三条基础路线，不占用基础路线的研究槽位
  - 融合科技可与基础路线科技同时研究，但需要第2研究槽位（VIP3解锁）
  - 若仅有1个槽位，融合科技与基础科技共享该槽位，需排队
  - 转生时融合科技保留50%（与基础科技一致）
- 验收标准: 融合科技提供组合加成效果，与基础科技乘法叠加

### 1.7 科技联动效果
- 操作: 研究经济路线科技→查看建筑产出；研究军事路线科技→查看武将属性；研究文化路线科技→查看研究速度
- 验证: TechLinkSystem将科技效果传递给建筑/资源/武将系统
- 预期: 经济→建筑产出+资源增产(农耕改良+10%粮草/商贸繁荣+15%铜钱/治国安邦-10%建筑升级费)，军事→攻击/防御/生命乘数(兵法入门+5%攻击/阵法精通+10%阵型)，文化→经验加成+忠诚+属性(礼贤下士+10%经验/天下归心-50%忠诚衰减)
- PRD对齐: TEC-1 关联系统 — 书院产出科技点→科技提升建筑产出效率→科技提升武将经验/忠诚/属性→声望提供研究速度加成→经济路线提升资源产出
- 验收标准: 联动效果即时生效，数值计算正确

### 1.8 离线研究回归
- 操作: 有研究中科技时关闭游戏→离线一段时间→回归游戏
- 验证: TechOfflineSystem按效率衰减计算(0~2h:100% | 2~8h:70% | 8~24h:40% | >24h:20%，封顶72h)
- 预期: 回归弹窗显示离线时长、各科技进度、获得科技点，已完成科技自动标记completed
- PRD对齐: TEC-2.4 离线研究规则 — 计算公式: 研究剩余时间-(离线秒数×离线效率系数)；科技离线效率=资源效率×1.0；离线期间书院科技点持续产出
- 验收标准: 离线补算准确，回归面板数据完整

### 1.9 科技重置（转生时）规则
- 操作: 触发转生→查看科技重置规则
- 验证: 已完成科技保留50%(随机保留或按路线优先级)；未完成科技进度清零；路线选择记录保留100%(可重选互斥分支)；融合科技保留50%(与基础科技一致)
- PRD对齐: TEC-4.3 转生重置规则 — 四项重置规则明确
- 验收标准: 转生后科技状态符合规则，互斥分支可重新选择

### 1.10 内政武将派遣加速研究
- 操作: 选择内政型武将→派遣至书院→查看研究速度变化
- 验证: 内政型武将增加研究速度+10%~30%
- PRD对齐: TEC-2 与其他系统关联 — 内政型武将→派遣至书院增加研究速度
- 验收标准: 武将派遣后研究速度正确提升

### 1.11 科技点管理流程 ★新增★
- 操作: 查看科技点余额→管理科技点产出→规划科技点消耗
- 验证:
  - **科技点产出**: 书院等级决定科技点/秒（Lv1: 0.3/s → Lv20: 8.0/s），持续累积，离线也累积（受离线效率系数影响）
  - **科技点存储**: 无上限，持续累积
  - **科技点消耗**: 研究启动时消耗科技点 = 基础费用 × (1 - 书院研究速度加成)
  - **科技点不足处理**: 研究按钮灰化 + "科技点不足"提示 + [升级书院→]链接引导
  - **铜钱兑换科技点**: 比率100:1（需书院Lv5+），辅助补充科技点
- 预期: 科技点作为独立资源在导航栏显示📚图标+实时余额+产出速率；科技点管理界面显示产出速率/当前余额/历史消耗
- PRD对齐: TEC-2 科技点系统 — 产出/消耗/存储/兑换完整规则
- 验收标准: 科技点产出速率随书院等级正确变化，消耗计算准确，不足时UI反馈正确

  - **战力差距分析**: 计算我方战力与敌方战力比值，确定差距等级
