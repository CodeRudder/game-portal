# v9.0 离线收益 — 玩游戏流程

## 1. 装备背包
- 打开装备Tab → 查看装备列表 → 筛选/排序 → 查看装备详情
- 验证: 装备属性、品质颜色、套装标记

## 2. 装备锻造
- 选择锻造类型 → 消耗材料 → 锻造 → 品质揭晓
- 验证: 材料扣除、保底计数、锻造结果

## 3. 装备强化
- 选择装备 → 消耗资源 → 强化 → 属性提升
- 验证: 强化等级、属性增量、消耗递增

## 4. 装备分解
- 选择多余装备 → 分解 → 获得材料
- 验证: 分解产出、批量分解

## 5. 套装效果
- 查看套装信息 → 装备同套件 → 激活套装加成
- 验证: 套装件数统计、加成激活阈值
