# v9.0 离线收益 — 玩游戏流程

## 1. 离线收益系统

### 1.1 离线计算核心
- 操作: 玩家下线 → 系统立即记录快照(资源产出/建筑队列/科技队列/远征队列/商队状态) → 玩家上线 → 离线≤5min静默入账不弹窗 / 离线>5min弹出收益弹窗
- 验证: 快照记录完整覆盖所有产出源；基础收益 = 净产出速率 × 离线秒数；效率系数 = 衰减系数 × (1 + Σ加成)；最终收益 = 基础收益 × 效率系数
- 预期: 离线计算公式 `净产出 × 秒数 × 衰减 × (1+加成)` 正确执行，快照72h有效期

### 1.2 六档衰减系数
- 操作: 分别测试离线 1h / 5h / 15h / 36h / 60h / 80h → 对比各档衰减系数 → 验证封顶72h
- 验证: 0~2h→100% | 2~8h→80% | 8~24h→60% | 24~48h→40% | 48~72h→25% | >72h→15%(封顶不产出)
- 预期: 衰减分档平滑递减，72h后收益归零，温和引导玩家2天内回归

### 1.3 加成系数叠加
- 操作: 提升科技加成(+30%) → 激活VIP加成(+20%) → 提升声望加成(+25%) → 验证叠加上限
- 验证: 三项加成累加 +75%；继续叠加时上限封顶 +100%；最终效率系数 = 衰减 × (1 + min(加成总和, 100%))
- 预期: 加成来源清晰可查，上限+100%防止收益膨胀，付费与免费玩家差距可控

### 1.4 离线收益弹窗与翻倍
- 操作: 离线>5min后上线 → 弹窗展示离线时长/效率系数/各资源收益/来源占比饼图 → 数字滚动动画(800ms) → 选择翻倍方式
- 验证: 弹窗数据与后台计算一致；广告翻倍(3次/天)和元宝翻倍(无限制)二选一不可叠加；领取后资源入账
- 预期: 回归第一眼有获得感，翻倍选项满足免费/付费两类玩家，饼图帮助理解收益来源

## 2. 离线经验

### 2.1 经验产出快照
- 操作: 下线前记录当前经验产出速率(主城等级+科技加成+VIP加成) → 离线后上线 → 验证经验收益
- 验证: 离线经验 = 基础经验速率 × 离线秒数 × 衰减系数 × (1+经验加成)；受72h封顶约束
- 预期: 经验离线收益与在线产出比例合理，衰减后仍有可观获得感

### 2.2 经验溢出与等级提升
- 操作: 低等级玩家长时间离线 → 上线领取大量经验 → 验证是否触发升级 → 检查升级奖励邮件
- 预期: 离线经验可推动升级；升级奖励通过邮件系统自动发放；回归综合面板提示等级变化

## 3. 离线资源

### 3.1 各系统离线行为
- 操作: 下线前启动建筑升级+科技研究+远征+商队运输 → 离线超过完成时间 → 上线验证各系统状态
- 验证: 资源产出×1.0 | 建筑升级×1.2(加速完成) | 科技研究×1.0 | 远征×0.85(保守结算) | 商队自动完成交易
- 预期: 各系统按独立效率修正结算，建筑有加成鼓励下线前排队，远征保守防损失

### 3.2 资源溢出处理
- 操作: 仓库接近上限时离线 → 上线领取收益 → 验证有上限资源截断+提示升级仓库
- 验证: 有上限资源(铜钱/粮草等)收益截断至仓库上限，弹窗提示溢出量+引导升级；无上限资源(声望/经验)全额发放
- 预期: 溢出提示清晰，引导玩家扩容而非"白挂"，减少负面体验

### 3.3 离线收益预估
- 操作: 打开预估面板 → 拖动滑块选择预估时长(1h~72h) → 查看各资源预估收益+效率系数曲线
- 验证: 预估数据与实际计算公式一致；效率系数随时长递减可视化；72h后显示封顶提示
- 预期: 帮助玩家决策"现在下线能拿多少"，降低离线焦虑

## 4. 回归奖励

### 4.1 回归综合面板
- 操作: 领取离线收益后 → 进入Step2回归综合面板 → 查看建筑/科技/远征/事件四类摘要
- 验证: 建筑升级完成列表含名称/等级/完成时间；科技研究完成列表同格式；远征归来摘要含战利品；待处理事件含限时NPC提醒
- 预期: 面板可跳过但不遗漏信息，帮助玩家快速了解离线期间变化并决策下一步

### 4.2 邮件系统奖励领取
- 操作: 打开邮件面板 → 切换分类Tab(全部/系统/战斗/社交/奖励) → 查看未读邮件 → 单封领取附件或批量领取
- 验证: 邮件四态流转正确(未读→已读未领→已读已领→已过期)；附件领取后物品入背包/资源入账；过期邮件灰色+戳记
- 预期: 批量操作减少操作负担，奖励邮件不遗漏，过期机制自动清理

## 5. 交叉验证

### 5.1 离线收益→资源→仓库联动
- 操作: 离线前记录各资源余额+产出速率 → 离线10h → 上线领取 → 验证资源增量与公式计算一致 → 检查仓库上限截断
- 预期: 收益计算→资源入账→仓库截断→溢出提示 全链路数据一致，无重复发放或遗漏

### 5.2 建筑排队→离线完成→回归面板→邮件闭环
- 操作: 下线前排3个建筑升级(预计2h/4h/8h完成) → 离线12h → 上线 → 验证3个建筑均完成 → 回归面板展示完成列表 → 检查邮件是否收到升级完成通知
- 预期: 建筑×1.2加速正确，回归面板与实际完成状态一致，邮件通知同步触发

### 5.3 远征→离线结算→战利品→邮件附件全链路
- 操作: 派遣远征队(预计6h返回) → 离线10h → 上线 → 验证远征×0.85保守结算 → 战利品通过邮件附件发放 → 领取后入背包
- 预期: 远征保守结算降低离线风险，战利品走邮件系统可追溯，领取后物品数量正确

### 5.4 翻倍机制→货币消耗→广告次数联动
- 操作: 离线收益弹窗 → 使用广告翻倍 → 验证收益×2且广告次数-1 → 再次离线回归 → 使用元宝翻倍 → 验证元宝扣除+收益×2 → 尝试同一次收益双重翻倍
- 预期: 广告与元宝翻倍互斥不可叠加；广告每日限3次计数正确；元宝翻倍无限制但扣费正确
