# v4.0 攻城略地(下) — 玩游戏流程

## 1. 战斗系统
- 布阵(选择武将+阵型) → 出征 → 战斗场景渲染 → 速度控制(1x/2x/3x) → 跳过 → 结算弹窗
- 验证: 战斗动画播放、伤害数字显示、技能释放特效、胜负判定正确
- 交叉验证: 武将属性→战斗伤害计算→星级评定→奖励发放

## 2. 扫荡系统
- 选择已通关关卡 → 点击扫荡 → 设置次数 → 确认 → 查看收益汇总
- 验证: 扫荡按钮仅在三星通关时可用、收益计算正确
- 限制: 需消耗扫荡道具或体力

## 3. 自动推图
- 自动推图开关 → 进度显示 → 自动战斗 → 遇到卡点停止
- 验证: 自动推图不跳过已锁定关卡、失败后重试逻辑
