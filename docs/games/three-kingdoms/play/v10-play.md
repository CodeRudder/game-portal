# v10.0 兵强马壮 — 玩游戏流程

## 1. 竞技场挑战
- 打开竞技场Tab → 查看对手列表 → 选择对手 → 布阵 → 战斗 → 结算
- 验证: 对手排名、战斗结果、排名变化

## 2. 防守阵容
- 设置防守阵容 → 保存 → 被攻击时使用
- 验证: 防守阵容保存、被攻击记录

## 3. 赛季系统
- 查看赛季信息 → 赛季任务 → 赛季奖励
- 验证: 赛季倒计时、段位划分、奖励发放

## 4. 竞技商店
- 消耗竞技币 → 兑换物品
- 验证: 商品列表、币种正确

## 5. 排行榜
- 查看竞技排行 → 自身排名 → 历史最高
- 验证: 排名实时更新、奖励预览
