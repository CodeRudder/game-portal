## 1. 装备系统

### 1.1 装备获取
- 操作: 推图普通关卡→结算掉落装备(白60%/绿30%/蓝8%/紫2%) → 精英关卡(白30%/绿40%/蓝22%/紫7%/金1%) → Boss关卡(白10%/绿25%/蓝35%/紫22%/金8%) → 商店购买装备箱/活动奖励
- 验证: 掉落概率与配置表一致；装备部位(武器/防具/饰品/坐骑)随机生成；品质颜色(白/绿/蓝/紫/金)正确显示
- 预期: 关卡难度越高掉落品质越好，装备获取渠道多样，初期以白绿为主快速填满槽位

### 1.2 装备背包管理
- 操作: 打开装备背包→查看50格网格→按品质/部位/套装筛选→按战力/等级/品质排序→装备过多时批量分解低品质→扩容背包(消耗元宝，每次+10格，最多扩至100格)
- 验证: 背包满时新装备不可获取并提示清理；批量分解仅选中未穿戴装备；分解产出铜钱+强化石数量正确
- 预期: 背包管理流畅，筛选排序高效定位目标装备，批量分解减少重复操作

### 1.3 装备品质与炼制
- 操作: 选择3件同品质装备→基础炼制(消耗铜钱×2000)→概率获得高一品质(3白→绿85%/蓝14%/紫1%) → 使用5件高级炼制(消耗铜钱×5000)提升概率 → 使用定向石指定部位炼制(消耗铜钱×8000+定向石×1) → 使用保底符保底炼制(消耗铜钱×10000+保底符×1，必定不低于紫色) → 连续10次未出紫→第11次必紫 / 30次未金→第31次必金
- 验证: 炼制消耗3/5件装备正确扣除；铜钱消耗正确(基础×2000/高级×5000/定向×8000/保底×10000)；产出品质概率符合配置；保底计数器跨会话保持；定向炼制产出指定部位；**非定向炼制产出部位概率：武器30%/防具30%/饰品25%/坐骑15%**（PRD EQP-2）；保底炼制使用保底符后必定产出紫色及以上品质
- 预期: 炼制是品质提升核心途径，四种配方满足不同需求(基础/高级/定向/保底)，保底机制兜底防止极端倒霉，定向石降低随机挫败感

### 1.4 装备箱开启（补充 — Plan #2 / PRD EQP-1-2）
- 操作: 从商店购买/活动获得装备箱→打开背包→选择装备箱→点击开启→随机获得对应品质范围装备→金色装备箱(紫30%/金70%)→活动限定装备箱(蓝20%/紫50%/金30%)
- 验证: 不同品质装备箱产出概率符合PRD掉落权重表；装备箱不可直接穿戴需先开启；开启后装备进入背包（背包满时不可开启）
- 异常处理: 背包满(50/50)时点击装备箱→弹出提示"背包已满，请先清理"→提供[去分解]和[扩容]两个快捷按钮→选择后跳转对应功能面板→清理/扩容后返回装备箱开启
- 预期: 装备箱是高品质装备的重要获取渠道，金色装备箱保底紫色以上给玩家明确期待

### 1.5 商店购买装备相关道具（补充 — Plan ShopSystem改进 / PRD EQP-1-2 / SHP-2）
- 操作: 进入商店→装备专区/军需处→购买装备箱/强化石/保护符/定向石→消耗元宝/铜钱→道具进入背包
- 验证: 商店新增装备类商品正确展示；价格与PRD SHP-2定价规则一致；购买后道具正确入账
- 异常处理: 背包满时购买装备箱→提示"背包已满，无法购买装备箱"；非装备箱道具(强化石/保护符/定向石)不受背包限制，进入道具栏
- 预期: 商店提供装备养成资源的稳定购买渠道，与关卡掉落互补形成多元获取

### 1.6 装备分解详细流程（v3补充 — Plan #4 / PRD EQP-5分解产出表）
- 操作: 打开背包→选择未穿戴装备→点击分解→预览产出(铜钱+强化石)→确认分解→资源入账
- 验证: 分解产出与PRD EQP-5分解产出表一致：白色→铜钱×100；绿色→铜钱×300+强化石×1；蓝色→铜钱×800+强化石×2；紫色→铜钱×2000+强化石×3；金色→铜钱×5000+强化石×5；批量分解时设置最低品质过滤(如仅分解白色和绿色)
- 异常处理: 正在穿戴的装备不可分解(按钮灰置+提示"请先卸下装备")；批量分解仅选中未穿戴装备，已穿戴装备自动排除
- 预期: 分解是装备回收核心手段，产出资源支撑强化和炼制消耗，形成正向循环

## 2. 装备强化

### 2.1 强化流程
- 操作: 选择装备→进入铁匠铺强化面板→消耗铜钱+强化石→+1→+2(100%)/+2→+3(100%)/+3→+4(95%)/+4→+5(90%)/+5起概率递减→成功等级+1属性提升→失败时：+3/+4失败不降级；+5→+6失败降至+4，+6→+7失败降至+5（等级-1）→ 查看强化动画反馈
- 验证: 强化费用按品质和等级递增正确（参照PRD EQP-4强化费用表：如白色+1=100铜+1强化石，金色+1=1500铜+1强化石）；成功率表(+1~+2必成，+3→+4为95%，+4→+5为90%，+5→+6为80%递减)符合PRD；+3/+4失败不降级；+5→+6失败降至+4（非+5）；金色+12以上失败不降级仅消耗材料
- 异常处理: 铜钱不足→提示"铜钱不足"并显示差额+快捷跳转市集；强化石不足→提示"强化石不足"并显示差额+快捷跳转商店；达到品质强化上限(白+5/绿+8/蓝+10/紫+12/金+15)→强化按钮灰置+提示"已达品质上限"
- 预期: 强化前两级无风险体验良好，+3起有概率但+3/+4失败不降级，+5起有降级风险但有保护机制，金色装备高等级强化有专属保障

> ⚠️ **PRD内部矛盾标注**: PRD EQP-4强化成功率表中，+5→+6失败后果写"降至+4"；但同节下方文字说明写"失败时等级-1，但不低于+5"。两者矛盾。**以成功率表格为准**：+5→+6失败降至+4。已提PRD修订工单。

### 2.2 强化保护符
- 操作: 强化前勾选使用保护符(铜/银/金三级)→强化失败→保护符消耗但等级不降→无保护符时正常降级
- 验证: 保护符消耗数量正确；失败时等级保持不变；不同等级保护符适用范围正确（铜质+5~+8/银质+8~+12/金质全等级）
- 异常处理: 无对应等级保护符时勾选框灰置+提示"无可用保护符"；保护符等级不匹配(如+10用铜质)→提示"该保护符不适用于当前等级"
- 预期: 保护符降低高等级强化心理压力，稀缺资源引导玩家策略性使用

### 2.3 自动强化
- 操作: 设置目标强化等级(如+10)→开启自动强化→系统连续执行强化→材料不足/铜钱不足/达到目标时自动停止→可选开启自动使用保护符
- 验证: 自动强化过程连续无中断；停止条件触发准确；自动保护符开关生效；中途可手动停止
- 异常处理: 目标等级超过品质上限→提示"目标超过品质上限(白色+5)"并自动修正为品质上限；自动强化中材料耗尽→停止并弹出汇总(成功N次/失败M次/当前等级+X)；自动强化中触发降级→继续执行直到其他停止条件触发
- 预期: 减少重复点击操作，目标明确时一键到位，资源耗尽自动停止避免浪费

## 3. 装备属性与战力

### 3.1 装备属性构成（补充 — Plan #10 / PRD EQP-3）
- 操作: 获得一件装备→查看装备详情面板→确认三部分属性：基础属性(主属性) + 附加属性(副属性) + 特殊词条→验证属性数值符合品质倍率
- 验证: 主属性 = 基础值 × 品质倍率(白1.0x/绿1.3x/蓝1.6x/紫2.0x/金2.5x) × (1 + 强化等级 × 0.15~0.18)；副属性条数符合品质定义(白0~1/绿1/蓝1~2/紫2/金2~3)；特殊词条触发概率正确(白0%/蓝5%/紫20%/金100%)
- 预期: 装备属性清晰展示，品质差异直观体现，金色装备显著强于低品质

### 3.2 武将战力计算验证（补充 — Plan #19 / HER PRD HER-1-3）
- 操作: 武将穿戴装备前后对比战力→记录裸装战力→穿戴1件装备→验证战力增量→穿戴4件装备→验证总战力
- 验证: 战力公式 = (ATK×2.0 + INT×2.0 + CMD×1.5 + POL×1.0) × 等级系数 × 星级系数 × 装备系数；装备系数 = 1 + Σ装备战力/1000；每件装备对战力的贡献独立计算正确
- 预期: 装备对战力提升显著且可预测，玩家能通过装备养成明确感知战力成长

### 3.3 属性精炼（推迟至v16.0 传承有序）
- 操作: 选择紫色及以上装备→进入精炼面板→消耗精炼石→重新随机副属性词条和数值→预览新旧属性对比→确认替换或放弃
- 验证: 精炼仅影响副属性，主属性不变；副属性词条池和数值范围符合品质定义；对比面板清晰展示增减
- 预期: 精炼为高端装备提供属性优化空间，预览对比降低决策风险，紫色以上才可精炼控制资源sink

> ⚠️ **已决议**: "属性精炼"功能在Plan v10.0和PRD EQP中均未定义。路线图v16.0"传承有序"版本明确包含"强化+军师"深化内容。**本功能推迟至v16.0实现**，v10.0 Play中保留描述作为v16.0需求参考。

### 3.4 特殊词条洗练（推迟至v16.0 传承有序）
- 操作: 金色装备→洗练特殊词条→消耗洗练石→随机更换特殊词条→锁定已有满意词条(额外消耗)→确认最终词条组合
- 验证: 金色装备100%附带特殊词条；洗练后词条从对应池随机；锁定词条不参与随机但消耗翻倍
- 预期: 金色装备词条可定制化，锁定机制保护已有好词条，洗练深度拉长金色装备养成线

> ⚠️ **已决议**: 同3.3，"特殊词条洗练"在Plan v10.0和PRD EQP中均未定义。**推迟至v16.0实现**，v10.0 Play中保留描述作为v16.0需求参考。

## 4. 套装效果

### 4.1 套装激活
- 操作: 为同一武将穿戴同套装2件→激活2件套效果→穿戴4件→激活4件套效果→查看武将面板套装效果标识
- 验证: 7套套装(青铜/寒铁/玄铁/赤焰/龙鳞/霸王/天命)效果正确；2件套/4件套分别独立激活；同一套装效果不叠加
- 预期: 套装搭配提供阵容策略深度，2件套易激活给基础奖励，4件套追求完美有明确目标

### 4.2 套装归属与搭配
- 操作: 金色装备固定套装归属→紫色装备概率附带套装→查看背包套装分组→一键穿戴推荐优先匹配套装→不同武将穿戴不同套装组合
- 验证: 金色固定套装标识不可更改；紫色套装概率生成（**v3确认：紫色装备套装概率为25%**，PRD EQP-3补充值）；推荐系统考虑套装完整性；多武将套装互不干扰
- 预期: 金色装备套装确定性降低随机挫败，紫色套装随机增加收集乐趣，推荐系统兼顾套装和属性

### 4.3 套装跨武将不共享验证（v3补充 — PRD EQP-3套装规则）
- 操作: 武将A穿戴龙鳞套装2件→激活2件套效果(全属性+8%)→武将B穿戴龙鳞套装另外2件→武将B也激活2件套效果→验证武将A的套装效果不受武将B影响→卸下武将A的1件龙鳞装备→武将A套装效果消失，武将B套装效果保持
- 验证: 套装效果仅在穿戴同一武将时生效；不同武将穿戴同套装互不干扰；卸下装备后套装效果实时更新
- 预期: 套装效果独立计算，跨武将不共享，避免套装搭配混乱

### 4.4 装备无限制穿戴验证（v3补充 — PRD EQP-5穿戴规则）
- 操作: 1级武将穿戴金色+15装备→验证无等级限制→同一武将穿戴不同阵营装备→验证无阵营限制→更换武将穿戴同一装备→验证无武将限制
- 验证: 装备无等级限制(获得即可穿戴)；装备无武将限制(任何武将可穿戴任何装备)；装备无阵营限制
- 预期: 装备穿戴门槛低，玩家自由搭配不受限制

## 5. 铁匠铺建筑入口（补充 — Plan BLD建筑系统改进）

### 5.1 铁匠铺功能入口
- 操作: 进入城池地图→点击铁匠铺建筑→进入铁匠铺主界面→选择功能Tab(强化/炼制/分解)→进入对应功能面板
- 验证: 铁匠铺建筑在城池地图正确显示；建筑等级满足时解锁对应功能；三个功能Tab切换流畅；从铁匠铺可快速跳转到装备背包
- 预期: 铁匠铺作为装备养成的统一入口，降低功能发现成本，建筑升级解锁更多装备玩法

### 5.2 铁匠铺建筑等级与功能解锁（v3补充 — Plan #25 / PRD BLD-1）
- 操作: 主城Lv1→铁匠铺未解锁→主城升至Lv3→铁匠铺解锁→铁匠铺Lv1→基础强化+基础炼制+基础分解可用→铁匠铺升至Lv5→解锁高级炼制→铁匠铺升至Lv10→解锁定向炼制→铁匠铺升至Lv15→解锁自动强化
- 验证: 铁匠铺解锁条件=主城Lv3（PRD BLD-1建筑依赖关系）；铁匠铺等级≤主城等级；各功能解锁等级与PRD BLD-1铁匠铺等级数据对应；未解锁功能Tab灰置+显示解锁条件
- 异常处理: 主城等级不足→铁匠铺建筑显示锁定图标+提示"主城Lv3解锁"；铁匠铺等级不足→对应功能Tab灰置+提示"铁匠铺LvX解锁"
- 预期: 铁匠铺随建筑等级逐步解锁功能，引导玩家升级建筑，功能解锁节奏与玩家成长匹配

## 6. 交叉验证

### 6.1 关卡掉落→炼制→强化→穿戴全链路
- 操作: 推图获得3件白色武器→基础炼制获得绿色武器→强化至+5→穿戴到武将武器槽→验证战力提升→卸下装备→分解获得铜钱+强化石
- 预期: 获取→炼制→强化→穿戴→分解全链路数据一致，每步资源消耗和产出正确，战力计算实时更新

### 6.2 强化降级→保护符→自动强化联动
- 操作: 蓝色装备+9→不使用保护符强化→失败降级至+8→使用保护符再次强化→失败等级保持+8→开启自动强化目标+10→自动连续强化至完成或资源耗尽
