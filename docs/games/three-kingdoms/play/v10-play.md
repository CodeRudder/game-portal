# v10.0 兵强马壮 — 玩游戏流程

## 1. 装备系统

### 1.1 装备获取
- 操作: 推图普通关卡→结算掉落装备(白60%/绿30%/蓝8%/紫2%) → 精英关卡(白30%/绿40%/蓝22%/紫7%/金1%) → Boss关卡(白10%/绿25%/蓝35%/紫22%/金8%) → 商店购买装备箱/活动奖励
- 验证: 掉落概率与配置表一致；装备部位(武器/防具/饰品/坐骑)随机生成；品质颜色(白/绿/蓝/紫/金)正确显示
- 预期: 关卡难度越高掉落品质越好，装备获取渠道多样，初期以白绿为主快速填满槽位

### 1.2 装备背包管理
- 操作: 打开装备背包→查看50格网格→按品质/部位/套装筛选→按战力/等级/品质排序→装备过多时批量分解低品质→扩容背包
- 验证: 背包满时新装备不可获取并提示清理；批量分解仅选中未穿戴装备；分解产出铜钱+强化石数量正确
- 预期: 背包管理流畅，筛选排序高效定位目标装备，批量分解减少重复操作

### 1.3 装备品质与炼制
- 操作: 选择3件同品质装备→基础炼制→概率获得高一品质(3白→绿85%/蓝14%/紫1%) → 使用5件高级炼制提升概率 → 使用定向石指定部位炼制 → 连续10次未出紫→第11次必紫 / 30次未金→第31次必金
- 验证: 炼制消耗3/5件装备正确扣除；产出品质概率符合配置；保底计数器跨会话保持；定向炼制产出指定部位
- 预期: 炼制是品质提升核心途径，保底机制兜底防止极端倒霉，定向石降低随机挫败感

## 2. 装备强化

### 2.1 强化流程
- 操作: 选择装备→进入铁匠铺强化面板→消耗铜钱+强化石→+1~+3必成/+4~+5高概率/+6起概率递减→成功等级+1属性提升→失败等级-1(不低于+5) → 查看强化动画反馈
- 验证: 强化费用按品质和等级递增正确；成功率表(+1必成→+6起递减)符合PRD；失败降级不低于+5；金色+12以上失败不降级
- 预期: 强化前三级无风险体验良好，高等级有风险但有保护机制，金色装备高等级强化有专属保障

### 2.2 强化保护符
- 操作: 强化前勾选使用保护符(铜/银/金三级)→强化失败→保护符消耗但等级不降→无保护符时正常降级
- 验证: 保护符消耗数量正确；失败时等级保持不变；不同等级保护符适用范围正确
- 预期: 保护符降低高等级强化心理压力，稀缺资源引导玩家策略性使用

### 2.3 自动强化
- 操作: 设置目标强化等级(如+10)→开启自动强化→系统连续执行强化→材料不足/铜钱不足/达到目标时自动停止→可选开启自动使用保护符
- 验证: 自动强化过程连续无中断；停止条件触发准确；自动保护符开关生效；中途可手动停止
- 预期: 减少重复点击操作，目标明确时一键到位，资源耗尽自动停止避免浪费

## 3. 装备精炼

### 3.1 属性精炼
- 操作: 选择紫色及以上装备→进入精炼面板→消耗精炼石→重新随机副属性词条和数值→预览新旧属性对比→确认替换或放弃
- 验证: 精炼仅影响副属性，主属性不变；副属性词条池和数值范围符合品质定义；对比面板清晰展示增减
- 预期: 精炼为高端装备提供属性优化空间，预览对比降低决策风险，紫色以上才可精炼控制资源 sink

### 3.2 特殊词条洗练
- 操作: 金色装备→洗练特殊词条→消耗洗练石→随机更换特殊词条→锁定已有满意词条(额外消耗)→确认最终词条组合
- 验证: 金色装备100%附带特殊词条；洗练后词条从对应池随机；锁定词条不参与随机但消耗翻倍
- 预期: 金色装备词条可定制化，锁定机制保护已有好词条，洗练深度拉长金色装备养成线

## 4. 套装效果

### 4.1 套装激活
- 操作: 为同一武将穿戴同套装2件→激活2件套效果→穿戴4件→激活4件套效果→查看武将面板套装效果标识
- 验证: 7套套装(攻击/防御/技能/机动等各有侧重)效果正确；2件套/4件套分别独立激活；同一套装效果不叠加
- 预期: 套装搭配提供阵容策略深度，2件套易激活给基础奖励，4件套追求完美有明确目标

### 4.2 套装归属与搭配
- 操作: 金色装备固定套装归属→紫色装备概率附带套装→查看背包套装分组→一键穿戴推荐优先匹配套装→不同武将穿戴不同套装组合
- 验证: 金色固定套装标识不可更改；紫色套装概率20%生成；推荐系统考虑套装完整性；多武将套装互不干扰
- 预期: 金色装备套装确定性降低随机挫败，紫色套装随机增加收集乐趣，推荐系统兼顾套装和属性

## 5. 交叉验证

### 5.1 关卡掉落→炼制→强化→穿戴全链路
- 操作: 推图获得3件白色武器→基础炼制获得绿色武器→强化至+5→穿戴到武将武器槽→验证战力提升→卸下装备→分解获得铜钱+强化石
- 预期: 获取→炼制→强化→穿戴→分解全链路数据一致，每步资源消耗和产出正确，战力计算实时更新

### 5.2 强化降级→保护符→自动强化联动
- 操作: 蓝色装备+9→不使用保护符强化→失败降级至+8→使用保护符再次强化→失败等级保持+8→开启自动强化目标+10→自动连续强化至完成或资源耗尽
- 预期: 降级规则正确执行(+5保底)，保护符消耗生效，自动强化综合成功率与手动一致，资源消耗可追溯

### 5.3 套装→精炼→武将战力闭环
- 操作: 收集4件同套装金色装备→精炼副属性至最优→洗练特殊词条→全套装穿戴同一武将→验证4件套效果+精炼属性+特殊词条全部叠加到战力
- 预期: 套装效果与精炼属性独立计算正确叠加，最终战力=基础+强化+精炼+套装+词条，武将面板完整展示所有加成来源

### 5.4 背包→分解→炼制资源循环
- 操作: 背包满(50件)→批量分解30件白色装备→获得铜钱+强化石→用强化石强化主力装备→用3件绿色装备炼制→验证分解产出与炼制消耗资源闭环
- 预期: 分解产出资源可支撑强化和炼制消耗，背包管理→资源回收→装备提升形成正向循环，无资源死锁
