## 二、核心流程覆盖评估

### 2.1 事件系统核心循环
```
挂机产出 → 触发引擎tick → 概率公式计算 → 冷却检查 → 事件触发 → 急报横幅通知(6级优先级) → 玩家响应 → 选项处理(2-3分支) → 后果计算(数值缩放) → 资源/声望/好感度变动 → 连锁检测 → 触发后续/归档 → 循环继续
```
**覆盖状态: ✅ 完整无断点** (§1.1-1.5 + §8.1)

### 2.2 连锁事件完整流程
```
前序事件选择 → 后续ID检测 → 分支条件判定 → 10-30min延迟触发 → 下一环节 → 链追踪面板可视化 → 暂停/恢复(离线) → 超时24h自动终止(30%补偿) → 链完成最终奖励 → 7天冷却
```
**覆盖状态: ✅ 完整无断点** (§2.1-2.5)

### 2.3 活动系统完整流程
```
活动开始 → 生成任务+里程碑+商店 → 任务完成(代币+积分) → 积分推进里程碑(线性解锁) → 代币兑换商店(七阶体系) → 排行榜5min刷新 → 离线积分累积(效率系数) → 活动到期 → 结算(排行+未领+代币10%转化) → 结算报告
```
**覆盖状态: ✅ 完整无断点** (§3.2-3.3 + §3.7-3.8 + §8.2)

### 2.4 离线处理全链路
```
玩家离线 → 时长记录 → ≥30min触发 → 正面保守50% / 负面防御减免 / 中性堆积3条 / NPC堆积2条 / 限时错过 / 剧情暂停 → 活动积分按效率累积 → 上线回归面板 → 一键领取+手动处理 → 数据写入存档
```
**覆盖状态: ✅ 完整无断点** (§5.1-5.3 + §8.3)

### 2.5 签到→代币→活动→排行闭环
```
每日签到(7天循环+连续加成+累计奖励) → 日常代币获取 → 活跃度任务(8项+4档宝箱) → 活动任务+战斗积分 → 代币兑换商店 → 积分推进里程碑+排行 → 离线效率累积 → 排行结算奖励
```
**覆盖状态: ✅ 完整无断点** (§3.5-3.6 + §6.4-6.5)
