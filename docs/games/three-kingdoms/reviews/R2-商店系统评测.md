### 修复详情

#### P1-2: normal商店商品分类筛选

**问题**：normal商店包含所有非limited商品（15个permanent + 6个random + 1个discount = 22个），商品列表过长，不利于浏览。

**修复方案**：在normal商店Tab下新增分类筛选栏（`GOODS_CATEGORY_FILTERS`），支持按6种分类过滤：

| 筛选项 | 说明 |
|--------|------|
| 全部 | 显示所有商品（默认） |
| 资源 | 粮草、铜钱等基础资源 |
| 材料 | 玉石、强化石等材料 |
| 装备 | 戒指、武器等装备 |
| 消耗品 | 经验卷轴、招募令等 |
| 特殊 | 蓝图、VIP包等特殊道具 |

**实现方式**：
- 商品列表 `useMemo` 中新增 `categoryFilter` 条件过滤
- 筛选按钮仅在 `activeTab === 'normal'` 时显示
- `data-testid="shop-panel-category-filter"` 覆盖测试

#### P1-4: NPC折扣接入UI

**问题**：引擎层 `setNPCDiscountProvider()` 已预留NPC好感度折扣接口，但UI层未使用。

**修复方案**：
- ShopPanel 新增 `npcId` 状态，支持传入当前交互NPC
- 当 `shopSystem.npcDiscountProvider` 存在且有 `npcId` 时，显示"🤝 NPC好友折扣已生效"提示
- `data-testid="shop-panel-npc-discount"` 覆盖测试

**引擎层验证**（已有测试覆盖）：
- `setNPCDiscountProvider()` 设置后 `calculateFinalPrice()` 正确应用折扣 ✅
- NPC折扣与限时折扣、商品折扣取最低值 ✅
- 折扣叠加测试（ACC-10-20等）全部通过 ✅

### 封版结论

> **✅ 有条件封版（R2评分 8.6/10）**
