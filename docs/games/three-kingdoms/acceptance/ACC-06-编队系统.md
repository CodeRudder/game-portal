# ACC-06 编队系统 — 用户验收标准

## 模块概述

编队系统是三国霸业中连接「武将收集」与「战斗出征」的核心枢纽。玩家可在武将面板的「编队」子Tab中创建最多3支编队，每支编队由前排3人+后排3人共6个武将槽位组成。系统支持手动上下阵、一键自动编队、编队推荐方案、编队保存/加载以及羁绊预览。激活的编队将直接用于出征关卡的战前布阵弹窗中。

**玩家体验目标**：
- 编队的创建、编辑、切换操作简洁直观，3步内完成核心操作
- 自动编队和推荐方案能真正帮助新手快速组建合理阵容
- 编队羁绊预览让玩家在编辑时就能感知阵容搭配的收益
- 编队数据持久化可靠，切换Tab/关闭弹窗后数据不丢失

**核心组件**：
| 组件 | 路径 | 职责 |
|------|------|------|
| `FormationPanel` | `panels/hero/FormationPanel.tsx` | 编队管理主面板（创建/删除/重命名/激活/编辑） |
| `FormationGrid` | `panels/hero/FormationGrid.tsx` | 编队网格（5槽位展示+战力+羁绊） |
| `FormationRecommendPanel` | `panels/hero/FormationRecommendPanel.tsx` | 编队推荐面板（3套方案+应用） |
| `FormationSaveSlot` | `panels/hero/FormationSaveSlot.tsx` | 编队收藏/存档槽（保存/加载/删除） |
| `HeroDispatchPanel` | `panels/hero/HeroDispatchPanel.tsx` | 武将派遣面板（派遣/召回） |
| `HeroRecommendTag` | `panels/hero/HeroRecommendTag.tsx` | 推荐标签（显示推荐原因+优先级） |
| `BattleFormationModal` | `panels/campaign/BattleFormationModal.tsx` | 战前布阵弹窗（敌方预览+我方编队+一键布阵+出征） |

**引擎支撑**：
| 引擎类 | 路径 | 核心方法 |
|--------|------|---------|
| `HeroFormation` | `engine/hero/HeroFormation.ts` | `createFormation()`, `setFormation()`, `addToFormation()`, `removeFromFormation()`, `setActiveFormation()`, `autoFormation()`, `renameFormation()`, `deleteFormation()` |
| `FormationRecommendSystem` | `engine/hero/FormationRecommendSystem.ts` | `recommend()`, `analyzeStage()` |
| `autoFormation()` | `engine/battle/autoFormation.ts` | 按防御降序自动分配前排/后排 |
| `HeroDispatchSystem` | `engine/hero/HeroDispatchSystem.ts` | `dispatchHero()`, `undeployHero()`, `getDispatchBonus()` |

**业务规则**：
- 最大编队数：3支（`MAX_FORMATIONS = 3`）
- 每编队最大武将数：6人（`MAX_SLOTS_PER_FORMATION = 6`）
- 编队名称最大长度：10字符
- 同一武将不可同时出现在多个编队中
- 默认编队名称：第一队、第二队、第三队
- 自动编队策略：防御最高的3人放前排，其余放后排
- 战力对比等级：碾压(≥1.2) / 优势(≥1.0) / 势均力敌(≥0.8) / 危险(<0.8)
