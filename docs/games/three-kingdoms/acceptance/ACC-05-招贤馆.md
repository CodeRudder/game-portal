# ACC-05 招贤馆 — 用户验收标准

## 模块概述

招贤馆是三国霸业中玩家获取武将的核心入口。玩家通过消耗招贤令资源，以"普通招贤"或"高级招贤"两种模式进行抽卡招募，获得不同品质的三国武将。系统提供概率公示、保底机制（十连保底稀有+、高级池硬保底传说）、每日免费招募、UP武将、十连招募、招募历史等完整功能。玩家体验目标：招募流程直观流畅、消耗透明、结果反馈清晰（品质揭示动画）、保底进度可见，让玩家感受到公平和期待感。

### 关键数值配置

| 配置项 | 普通招贤 | 高级招贤 |
|--------|---------|---------|
| 单次消耗 | 招贤令 ×5 | 招贤令 ×100 |
| 十连消耗 | 招贤令 ×50 | 招贤令 ×1000 |
| 每日免费 | 1次 | 0次 |
| 十连保底 | 每10次必出稀有+ | 每10次必出稀有+ |
| 硬保底 | 无 | 每100次必出传说+ |
| UP武将 | 不生效 | 出传说时50%为UP武将 |

### 品质概率表

| 品质 | 普通 | 高级 |
|------|------|------|
| 普通(COMMON) | 60% | 20% |

| 稀有(RARE) | 8% | 25% |
| 史诗(EPIC) | 2% | 13% |
| 传说(LEGENDARY) | 0% | 2% |
