# ACC-11 引导系统 — 用户验收标准

## 模块概述

引导系统负责新玩家首次进入三国霸业时的教学体验，目标是让玩家在 5~10 分钟内理解核心玩法（建筑→武将→出征→资源→科技），顺利过渡到自由游戏阶段。

系统由三层组成：
- **GuideOverlay**：步骤式引导覆盖层，4步核心引导（招募→详情→强化→编队），对接引擎 TutorialStateMachine
- **TutorialOverlay**：通用引导遮罩组件，支持圆形/矩形高亮区域、提示框自动定位、步骤指示器
- **InteractiveTutorial**：交互式教程组件，通过 CSS 选择器定位目标元素，带完成动画

引擎侧通过 5 阶段状态机（not_started → core_guiding → free_explore → free_play → mini_tutorial）管理引导生命周期，包含 6 步核心引导 + 6 步扩展引导、8 段剧情事件、阶段奖励、新手保护（30 分钟）、引导重玩等机制。
