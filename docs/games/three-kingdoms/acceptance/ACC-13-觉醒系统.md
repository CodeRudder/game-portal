# ACC-13 觉醒系统 — 用户验收标准

## 模块概述

觉醒系统是武将培养的终极阶段。当武将达到 **Lv.100 + 6星 + 4阶突破 + 品质≥稀有** 后，可消耗大量资源（铜钱、突破石、技能书、觉醒石、碎片）进行觉醒。觉醒后武将全属性 +50%、等级上限提升至 120、解锁专属终极技能，并为全队提供被动光环加成。

觉醒系统贯穿升星弹窗（HeroStarUpModal）、突破面板（HeroBreakthroughPanel）、技能面板（SkillPanel）和武将详情（HeroDetailModal）等多个 UI 组件，是玩家长期追求的核心养成目标。

**涉及 UI 组件**：HeroStarUpModal、HeroStarUpPanel、HeroBreakthroughPanel、SkillPanel、HeroDetailBreakthrough、HeroDetailModal
**引擎支撑**：AwakeningSystem、HeroStarSystem、SkillUpgradeSystem、awakening-config
