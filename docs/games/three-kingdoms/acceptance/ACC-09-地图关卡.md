# ACC-09 地图关卡 — 用户验收标准

## 模块概述

地图关卡模块是三国霸业的核心玩法入口之一，包含两大子系统：

1. **出征关卡（CampaignTab）**：玩家通过章节→关卡路径推进剧情战斗，通关获取资源与星级评价，三星后解锁扫荡功能实现快速收益。
2. **天下地图（WorldMapTab）**：玩家在领土网格地图上查看全局势力分布、筛选领地、查看产出收益，并对敌方领地发起攻城以扩张版图。

**玩家体验目标**：关卡推进节奏清晰、地图信息一目了然、攻城决策有据可依、扫荡操作高效便捷。
