# ACC-08 科技系统 — 用户验收标准

## 模块概述

科技系统是三国霸业的核心成长线之一，玩家通过消耗科技点研究军事/经济/文化三条科技路线上的节点，获得全军属性加成、资源产出提升、研究速度提升等永久增益。系统包含科技树浏览、节点研究、研究队列管理、加速/取消操作、互斥分支选择、离线研究进度等核心体验。玩家目标是在三条路线中做出策略选择（互斥分支二选一），合理分配科技点资源，逐步解锁高阶科技以全面提升实力。

**涉及组件**：`TechTab` → `TechResearchPanel` → `TechNodeDetailModal` → `TechOfflinePanel`
**引擎支撑**：TechTreeSystem / TechResearchSystem / TechPointSystem / FusionTechSystem / TechLinkSystem

## 验收重点说明

### 关键体验路径

1. **科技研究完整流程**：进入科技Tab → 选择路线 → 浏览节点 → 点击节点查看详情 → 确认消耗和效果 → 开始研究 → 观察进度 → 研究完成 → 后续节点解锁
2. **互斥分支决策**：到达互斥层级 → 查看两个互斥节点详情 → 权衡效果差异 → 选择一个研究 → 另一个永久锁定
3. **研究队列管理**：查看队列 → 加速/取消操作 → 队列空出后新增研究
4. **科技点经济循环**：书院产出科技点 → 积累 → 消耗研究 → 获得加成效果

### 数据校验要点

- 科技点消耗/退还数值精确
- 研究倒计时与实际经过时间一致（±2秒）
- 队列上限与书院等级对应（Lv1=1, Lv5=2, Lv10=3, Lv15=4, Lv20=5）
- 互斥组锁定逻辑：同组只能选一个，选择后替代项永久不可研究
- 路线进度统计：已完成数 / 该路线总节点数

### 与其他系统的关联

| 关联系统 | 关联点 | 验收关注 |
|---------|--------|---------|
| 建筑系统（ACC-02） | 书院等级决定科技点产出速率和研究队列上限 | 升级书院后，产出速率和队列上限应同步更新 |
