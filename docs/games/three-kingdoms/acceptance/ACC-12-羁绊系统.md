# ACC-12 羁绊系统 — 用户验收标准

## 模块概述

羁绊系统是三国霸业中提升编队战力的核心养成维度之一。玩家通过合理搭配武将阵容，激活**阵营羁绊**（魏/蜀/吴/群雄，同阵营2/3/4/5人逐级增强）和**搭档羁绊**（桃园结义、五虎上将等14组特定武将组合），获得攻击、防御、生命、暴击、策略等属性百分比加成。羁绊效果在编队界面实时预览，在战斗中真实生效，鼓励玩家围绕羁绊体系构建阵容。

**玩家体验目标**：
- 在编队界面直观看到当前阵容的羁绊激活情况与属性加成
- 通过羁绊图鉴了解所有可激活的羁绊及其效果，形成收集目标
- 羁绊加成在战斗中真实生效，数值可验证
- 潜在羁绊提示引导玩家优化阵容

**涉及UI组件**：
| 组件 | 文件 | 职责 |
|------|------|------|
| BondPanel | `panels/hero/BondPanel.tsx` | 编队羁绊面板（阵营分布+羁绊列表） |

| BondActivateModal | `panels/hero/BondActivateModal.tsx` | 羁绊激活弹窗（效果+参与武将） |

| HeroDetailBonds | `panels/hero/HeroDetailSections.tsx` | 武将详情中的羁绊标签 |
| FormationPanel（羁绊预览） | `panels/hero/FormationPanel.tsx` | 编队面板中的羁绊预览区域 |

### 羁绊加成叠加规则

1. **阵营羁绊只显示最高等级**：同一阵营只激活匹配的最高tier，不叠加低等级
2. **搭档羁绊独立判定**：每个搭档羁绊独立检查 minCount 条件
3. **阵营+搭档可叠加**：阵营羁绊和搭档羁绊效果通过 `mergeEffects()` 逐项相加
4. **加成应用到属性**：`applyBondBonus()` 公式为 `最终属性 = 基础属性 × (1 + 加成百分比)`，结果四舍五入
