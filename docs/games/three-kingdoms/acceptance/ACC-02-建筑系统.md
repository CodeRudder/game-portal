# ACC-02 建筑系统 — 用户验收标准

## 模块概述

建筑系统是三国霸业的核心放置收益模块。玩家通过管理城池中的 **8 座建筑**（主城、农田、市集、兵营、铁匠铺、书院、医馆、城墙），持续产出粮草、铜钱、兵力、科技点等资源，并通过升级建筑提升产出效率和特殊属性。建筑系统是游戏默认首屏，是玩家最频繁交互的模块之一，其体验直接影响留存率。

**玩家体验目标**：
- 进入游戏即看到完整的城池地图，8 座建筑布局清晰
- 一目了然地了解每座建筑的等级、产出状态和可升级性
- 点击建筑后能快速查看升级预览（产出变化、消耗明细），一键完成升级
- 升级过程中有进度反馈，完成后资源产出实时更新
- 资源不足时有明确提示，不产生困惑

**涉及组件**：`BuildingPanel`、`BuildingUpgradeModal`、`BuildingIncomeModal`
**引擎支撑**：`BuildingSystem`、`BuildingBatchOps`、`BuildingRecommender`、`HeroDispatchSystem`
**8 座建筑**：主城(castle)、农田(farmland)、市集(market)、兵营(barracks)、铁匠铺(smithy)、书院(academy)、医馆(clinic)、城墙(wall)
