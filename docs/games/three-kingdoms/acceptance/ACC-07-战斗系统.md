# ACC-07 战斗系统 — 用户验收标准

## 模块概述

战斗系统是三国霸业的核心玩法之一，覆盖从关卡选择、战前布阵、战斗过程回放到战斗结算的完整体验链路。玩家通过出征Tab进入关卡地图，选择关卡后进入战前布阵弹窗查看敌方阵容并调整我方编队，点击出征后进入全屏战斗场景观看自动战斗动画（支持加速/跳过），战斗结束后弹出结算弹窗展示星级评定、奖励和统计数据。三星通关的关卡可使用扫荡功能快速获取资源。此外，竞技场（PvP）和远征系统也复用战斗能力。

**玩家体验目标**：战斗流程顺畅、布阵策略感强、动画表现清晰、结算数据准确、扫荡便捷高效。

**核心组件**：
- `CampaignTab` — 关卡地图入口
- `BattleFormationModal` — 战前布阵弹窗
- `BattleScene` — 全屏战斗场景
- `BattleAnimation` — 战斗动画Hook
- `BattleSpeedControl` — 战斗速度控制
- `BattleResultModal` — 战斗结算弹窗
- `SweepModal` / `SweepPanel` — 扫荡弹窗/面板
- `ArenaTab` — 竞技场面板
- `ExpeditionTab` — 远征面板
- `CombatIcons` — 战斗技能/装备图标

**引擎支撑**：`BattleEngine`、`DamageCalculator`、`BattleStatistics`、`SweepSystem`、`RewardDistributor`、`CampaignProgressSystem`

## 战斗流程完整路径

### 路径A：正常战斗
```
出征Tab → 选择章节 → 点击可挑战关卡 → BattleFormationModal（查看敌方/战力对比/一键布阵）
→ 点击「⚔️出征」→ BattleScene（全屏战斗动画）
→ 战斗结束 → BattleResultModal（星级/奖励/统计）→ 点击「确认」→ 返回关卡地图
```

### 路径B：扫荡
```
出征Tab → 点击三星关卡的「⚡扫荡」→ SweepModal（次数/消耗/预计奖励）
→ 点击「确认扫荡」→ 显示扫荡结果 → 点击「取消」→ 返回关卡地图
```

### 路径C：竞技场
```
更多▼ → 🏟️竞技 → ArenaTab → 点击对手「挑战」→ 弹出战斗结果（胜/负+积分变化）→ 确定
```

### 路径D：远征
```
更多▼ → 🧭远征 → ExpeditionTab → 选择路线 → 点击「出征🚀」→ 弹出远征结果 → 确认
```
