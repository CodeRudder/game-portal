# ACC-10 商店系统 — 用户验收标准

## 模块概述

商店系统是玩家获取道具、资源和稀有物品的核心消费渠道。玩家通过底部导航「更多」菜单或「更多Tab」进入商店面板，在4个特色商店（杂货铺、竞技商店、远征商店、联盟商店）中浏览商品、查看价格与折扣、完成购买操作。系统提供完整的货币余额展示、限购追踪、库存管理、购买确认和反馈提示，确保玩家的每一次消费都清晰透明、操作流畅。

**核心玩家体验目标：**
- 商品信息一目了然（名称、描述、价格、折扣、限购进度）
- 购买流程简洁安全（确认弹窗 → 反馈提示）
- 货币余额实时可见，不足时给出明确提示
- 各商店特色分明（不同货币、不同商品池、不同限购规则）

**涉及组件：** `ShopPanel`（商店面板）、`TradePanel`（商贸面板）
**引擎支撑：** `ShopSystem`（商店核心）、`CurrencySystem`（货币系统）、`ArenaShopSystem`（竞技商店）、`AllianceShopSystem`（联盟商店）、`PrestigeShopSystem`（声望商店）、`TokenShopSystem`（代币商店）

## 附录：关键测试路径

### 路径A：标准购买流程
```
主界面 → 更多▼ → 🛒商店 → 选择商店Tab → 浏览商品 → 点击购买 → 确认弹窗 → 确认 → 观察成功提示 → 检查余额变化
```

### 路径B：货币不足场景
```
商店面板 → 选择高价商品 → 点击购买 → 确认弹窗 → 确认 → 观察「XXX不足」提示 → 余额不变
```

### 路径C：限购耗尽场景
```
商店面板 → 反复购买限购商品 → 达到上限 → 再次尝试 → 观察购买失败提示 → 每日重置 → 可再次购买
```

### 路径D：多商店切换
```
商店面板 → 杂货铺购买 → 切换竞技商店 → 浏览商品 → 切回杂货铺 → 验证之前购买记录保留
```

### 路径E：商贸面板（TradePanel）
```
更多▼ → 💱交易 → 商贸面板 → 查看商路列表 → 开通商路 → 派遣商队 → 查看行情 → 刷新价格
```
