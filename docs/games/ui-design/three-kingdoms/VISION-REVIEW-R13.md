## P0 改进优先级

### P0 — 必须修复
1. **主界面 UI 布局** (6.0)：弹窗遮挡核心信息，需优化信息层级
2. **武将展示** (6.0)：列表式展示信息密度低，需增加缩略预览

### P1 — 强烈建议
3. **主界面三国主题感** (6.0)：图标风格偏卡通，缺乏三国特征辨识度
4. **武将三国主题感** (6.0)：头像/界面缺乏三国元素

### P2 — 优化提升
5. **关卡微动画**：场景预览增加呼吸感动画
6. **科技树分支**：增加前置条件与分支路径
7. **资源联动机制**：增加资源间相互影响

## 结论

R13 全局均分 **7.5**，较 R12 (7.0) 提升 **+0.5**，创下历史新高。

**本轮亮点：**
- 关卡系统视觉呈现从 6.0 提升至 8.0，场景预览缩略图 + 类型图标 + 进度可视化效果显著
- 地图系统保持 8.0 高分，地形多样性和三国主题感表现优秀
- 科技树和关卡系统的三国主题感均达到 9.0

**待改进：**
- 主界面仍是短板 (6.5)，UI 布局和主题感需重点优化
- 武将系统展示方式需升级，从列表式转向卡片式展示
- 建议下轮 (R14) 聚焦主界面布局重构 + 武将卡片化展示
