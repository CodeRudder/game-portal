# 建筑系统 UI 设计（v3.0 大幅重写）

> **v3.0 重写说明**：Round 1 评分 5.8/10（C+），为全模块最低。本次全面重写补全：8座建筑完整等级数据表、建筑依赖关系图、手机端完整适配方案、建筑队列管理、升级动画反馈、建筑与科技/武将联动、离线建造进度、产出收取机制。

## 一、设计总览

### 设计理念
建筑系统是放置游戏的**核心产出引擎**。玩家每次回归，第一眼应看到"我的城池在生长"——建筑自动产出资源、离线持续建造、升级带来可见变化。视觉上延续"水墨江山·铜纹霸业"风格，建筑外观随等级递进：茅屋→瓦房→楼阁→宫殿。

### 8座建筑总览

| 建筑 | 分区 | 核心产出 | 解锁条件 | 等级上限 | 特殊联动 |
|------|------|---------|---------|:-------:|---------|
| 🏛️ **主城** | 核心中央 | 全资源加成 | 初始 | 30 | 解锁其他建筑等级上限 |
| 🌾 **农田** | 左侧民生 | 粮草/秒 | 初始 | 25 | 科技「屯田」加成 |
| 💰 **市集** | 左侧民生 | 铜钱/秒 | 主城Lv2 | 25 | 科技「通商」加成 |
| ⚔️ **兵营** | 中央军事 | 兵力/秒 | 主城Lv2 | 25 | 武将统帅加成 |
| 🔨 **铁匠铺** | 中央军事 | 装备强化材料 | 主城Lv3 | 20 | 科技「锻造」解锁高级材料 |
| 📚 **书院** | 右侧文教 | 科技点/秒 | 主城Lv3 | 20 | 科技研究速度加成 |
| 🏥 **医馆** | 右侧文教 | 伤兵恢复速率 | 主城Lv4 | 20 | 武将续航加成 |
| 🏯 **城墙** | 上方防御 | 城防值/防御加成 | 主城Lv5 | 20 | 科技「固城」加成 |

### 建筑依赖关系图

```
                    ┌─────────┐
                    │  主城Lv1 │ ← 初始解锁
                    └────┬────┘
              ┌──────────┼──────────┐
              ▼          ▼          ▼
         ┌────────┐ ┌────────┐ ┌────────┐
         │农田Lv1 │ │市集Lv2 │ │兵营Lv2 │ ← 主城Lv2解锁
         └────────┘ └────────┘ └────┬───┘
                                    │
              ┌──────────┬──────────┤
              ▼          ▼          ▼
         ┌────────┐ ┌────────┐ ┌────────┐
         │书院Lv3 │ │医馆Lv4 │ │铁匠Lv3 │ ← 主城Lv3/4解锁
         └────────┘ └────────┘ └────────┘
                    │
              ┌─────┴─────┐
              ▼            ▼
         ┌────────┐  ┌────────┐
         │城墙Lv5 │  │招贤馆  │ ← 主城Lv5解锁（招贤馆见武将系统）
         └────────┘  └────────┘
```

**关键规则**：
- 每座建筑等级 ≤ 主城等级（主城Lv10时，其他建筑最高升到Lv10）
- 升级前置：部分高等级需要前置建筑达到指定等级（见等级数据表）
- 主城升级消耗最大，但提升全资源产出加成（每级+2%）

### 建筑卡片设计

每座建筑在城池场景中显示为**交互卡片**：

```
┌──────────────────┐
│   建筑外观图       │  ← 建筑立绘，随等级变化
│   (水墨风格)       │     Lv1-5: 茅屋/木栅
│                   │     Lv6-12: 瓦房/石墙
│                   │     Lv13-20: 楼阁/砖墙
│                   │     Lv21-30: 宫殿/铜门
├──────────────────┤
│ 🌾农田 Lv.12     │  ← 图标+名称+等级（金色徽章）
│ 产出: +8.2 粮草/s │  ← 当前产出（绿色小字）
│ ████████░░ 82%   │  ← 升级进度（仅升级中显示）
│ [▲升级 2,400粮]   │  ← 升级按钮+费用摘要
└──────────────────┘
```

- **卡片尺寸**：160×180px（核心建筑主城200×220px）
- **背景**：rgba(26,35,50,0.85) 墨玉青半透明 + 古铜金描边
- **等级徽章**：右上角圆形，直径28px，古铜金底色 + 深色等级数字
- **产出气泡**：建筑上方悬浮，资源图标+数值+"/s"
- **升级中状态**：卡片底部显示进度条 + 剩余时间倒计时
- **可升级标记**：资源充足时升级按钮金色发光脉冲；资源不足灰色
- **红点提示**：可升级且资源充足时，卡片右上角红点+数字

### 筛选栏设计

```
[全部●] [🌾民生] [⚔️军事] [📚文教] [🏯防御] [🏛️核心]  |  [▲可升级] [⏳升级中]
```

- **分类筛选**：点击高亮对应类别建筑，其余降低亮度至50%
- **状态筛选**：「可升级」过滤出资源充足的建筑；「升级中」过滤正在升级的建筑
- **红点角标**：「可升级」按钮显示当前可升级建筑数量

### 手机端建筑条目设计

每个建筑条目采用**双行卡片**布局：

```
┌─ 355×88px ──────────────────────────────────┐
│ ┌──┐                                        │
│ │建│ 🌾 农田 Lv.12              [可升级 ●]  │
│ │筑│ 产出: +8.2 粮草/s                       │
│ │图│ ████████░░ 82% (升级中 剩余3m)          │
│ │标│ [▲升级 2.4K粮/800钱]  [详情→]           │
│ └──┘                                        │
└─────────────────────────────────────────────┘
```

- **建筑图标**：48×48px，圆角8px，水墨风格缩略图
- **名称+等级**：16px 加粗，等级数字金色
- **产出信息**：12px 绿色小字
- **进度条**：高度6px（满足2mm最小可辨识要求），渐变色填充
- **升级按钮**：48×32px，虎符造型，资源充足金色/不足灰色
- **状态标签**：右上角，可升级(绿点)、升级中(蓝进度)、排队中(黄钟)
- **触控区域**：整条卡片可点击进入详情，升级按钮独立触控区≥44×44px
- **左滑操作**：左滑建筑条目→快捷菜单「详情 | 升级 | 加速」

## 五、建筑队列管理系统

### 设计理念

放置游戏玩家不应被"逐个点击升级"束缚。队列系统允许玩家**批量安排升级计划**，离线时自动推进。

### 队列槽位规则

| 主城等级 | 可用队列槽位 | 解锁说明 |
|:-------:|:----------:|---------|
| Lv1~5 | 1个 | 同时只能升级1座建筑 |
| Lv6~10 | 2个 | 第2槽位解锁 |
| Lv11~20 | 3个 | 第3槽位解锁 |
| Lv21~30 | 4个 | 第4槽位解锁 |

### 队列面板设计（PC端）

右上角悬浮面板，显示当前升级进度：

```
┌─ 升级队列 (2/3) ──────────────┐
│                                │
│ 🔨 兵营 Lv.11→12              │
│ ████████░░ 78%  剩余3m22s     │
│ [加速💎×5] [取消]             │
│                                │
│ ⏳ 市集 Lv.10→11  排队中      │
│ 预计: 3m22s后开始              │
│ [移除]                         │
│                                │
│ ┌──────────────────────────┐  │
│ │ + 添加升级 (可升级: 3)    │  │
│ └──────────────────────────┘  │
└────────────────────────────────┘
```

- **面板尺寸**：220×auto，最小高度120px
- **活跃升级**：显示进度条+剩余时间+加速/取消按钮
- **排队等待**：显示等待状态+预计开始时间+移除按钮
- **添加升级**：点击弹出可升级建筑列表，选择后加入队列
- **自动升级开关**：面板底部开关，开启后系统自动将可升级建筑加入队列

### 队列面板设计（手机端）

底部Tab栏上方展开的紧凑条：

```
┌─ 升级队列 (2/3) ──────────────────────┐
│ 🔨兵营 78% 3:22  |  ⏳市集 排队中      │
│ [展开详情▼]                            │
└────────────────────────────────────────┘
```

- **收起状态**：单行高度32px，显示摘要信息
- **展开状态**：向上展开至120px，显示完整队列
- **点击条目**：跳转到对应建筑的详情面板

### 队列管理交互

| 操作 | PC端 | 手机端 |
|------|------|--------|
| 添加升级 | 点击建筑升级按钮自动入队 | 建筑条目升级按钮 / 详情面板升级 |
| 取消升级 | 队列面板「取消」按钮，返还80%资源 | 长按队列条目→确认取消 |
| 调整顺序 | 拖拽队列条目重新排序 | 长按拖拽排序 |
| 加速升级 | 💎加速按钮，每💎减少1分钟 | 同PC端 |
| 自动升级 | 队列面板底部开关 | 队列展开面板底部开关 |

## 六、产出收取机制

### 自动累积 + 满仓提示

**核心规则**：建筑产出**自动累积**到资源池，玩家无需手动收取。

| 资源类型 | 存储上限 | 满仓行为 |
|---------|---------|---------|
| 粮草 🌾 | 粮仓容量（农田等级决定） | 满仓后溢出浪费，产出气泡变红 |
| 铜钱 💰 | 无上限 | 持续累积 |
| 兵力 ⚔️ | 兵营容量（兵营等级决定） | 满仓后停止产出，兵营卡片脉冲警告 |
| 科技点 📚 | 无上限 | 持续累积 |

### 产出气泡设计

每个建筑上方常驻显示产出气泡：

```
PC端：                    手机端：
┌──────────┐             ┌────────┐
│ 🌾+8.2/s │             │ 🌾+8.2 │
│ ▲累积1.2K│             │ 1.2K▲  │
└──────────┘             └────────┘
```

- **PC端气泡**：双行——产出速率 + 累积待领取值（仅满仓风险时显示）
- **手机端气泡**：单行精简——产出速率
- **满仓警告**：气泡背景从墨色变为暗红色 + 脉冲动画 + 文字"即将满仓!"
- **溢出状态**：气泡变红 + "溢出中!" + 军师建议自动推送"粮草将溢出"

### 一键收取（手机端）

- 右下角悬浮「一键收取」按钮（56×56px），仅在有可收取产出时显示
- 点击后所有产出气泡执行弹跳动画 → 资源数字飞向顶部资源栏
- 无产出时按钮隐藏，避免视觉干扰

## 七、建筑升级动画与反馈

### 升级开始动画

1. **确认特效**（0.3s）：升级按钮按下 → 虎符印章盖章特效（朱砂红圆形印章从天而降）
2. **资源扣除**（0.3s）：所需资源从顶部资源栏飞向建筑（金色粒子流）
3. **建造开始**（0.5s）：建筑周围升起脚手架光效（水墨线条构成的支架）
4. **进度条出现**：建筑卡片底部展开进度条

### 升级进行中动画

- **脚手架环绕**：水墨线条支架环绕建筑旋转（12fps循环）
- **进度条填充**：古铜金渐变从左到右填充
- **工人小动画**：建筑旁出现小人剪影忙碌（可选，高端设备）
- **时间倒计时**：卡片上显示"剩余 Xm Xs"

### 升级完成动画

1. **脚手架消散**（0.5s）：水墨支架化为墨点飞散
2. **建筑升级**（0.3s）：旧建筑淡出 → 新建筑淡入（scale 0.9→1.05→1.0 弹跳）
3. **等级数字弹跳**：等级徽章数字执行弹跳放大（1.0→1.3→1.0）
4. **产出飞字**：新产出数值从建筑飞出（如"+8.2→+10.5 粮草/s ↑"）
5. **全屏提示**：顶部短暂横幅"农田升级至Lv.13！产出+28%"
6. **音效**：古筝和弦 + 建筑落成音效

### 手机端动画简化

| 动画 | PC端 | 手机端 |
|------|------|--------|
| 确认特效 | 完整印章特效 | 简化为按钮缩放+金色闪光 |
| 资源飞向 | 粒子流 | 数字淡出 |
| 脚手架 | 水墨线条环绕 | 简化为进度条发光 |
| 完成弹跳 | 完整序列 | 等级弹跳+横幅提示 |
| 产出飞字 | 飞行动画 | Toast提示 |

## 九、离线建造进度设计

### 离线建造规则

| 离线行为 | 规则说明 |
|---------|---------|
| 正在升级 | 离线时间正常计入升级进度，回归时可能已完成 |
| 队列等待 | 排队中的升级在离线期间依次自动执行 |
| 自动升级 | 开启自动升级后，离线期间系统自动安排升级 |
| 产出累积 | 离线期间建筑产出持续累积（受离线效率系数影响） |

### 离线效率系数

```
离线效率 = min(1.0, 基础效率 + 科技加成 + VIP加成)

基础效率:
- 离线0~2小时:  100%
- 离线2~8小时:  80%
- 离线8~24小时: 60%
- 离线24~72小时: 40%
- 离线>72小时:  20%（封底）

科技加成:
- 科技「离线收益I」: +10%
- 科技「离线收益II」: +15%
- 科技「离线收益III」: +20%

VIP加成:
- VIP1: +5%, VIP3: +10%, VIP5: +20%
```

### 回归建造可视化

玩家回归后，建筑系统展示离线建造成果：

1. **已完成升级**：建筑卡片显示「NEW」标签 + 升级完成动画回放（加速版）
2. **正在进行升级**：进度条更新到当前位置
3. **产出累积气泡**：各建筑上方显示离线累积产出总量（金色大字）
4. **建筑列表顶部横幅**："离线期间，农田升至Lv.13，市集升至Lv.11！"
5. **一键收取离线产出**：点击后所有累积产出数字飞向资源栏

建筑详情面板增加「建造日志」入口：
