## 六、仍需改进项（按优先级排序）

### 🔴 P0 — 必须立即改进

#### 1. 视觉风格全面升级

**问题**: 整体评级仍为 C 级，视觉风格单调。水墨纹理、青铜纹饰、竹简材质等设计文档规范仍未落地。

**改进方案**:
- 统一引入水墨纹理背景（替代纯色/渐变背景）
- 云纹/青铜纹边框装饰（替代现代简约边框）
- 隶书/楷书字体（替代无衬线字体）
- 传统色彩体系全面应用：墨玉青 `#1A2332`、古铜金 `#C9A84C`、赤焰红 `#B8423A`

**影响范围**: 全部界面
**预期提升**: 综合分 +1.0 ~ +1.5

---

#### 2. 地图面板全面重构

**问题**: 地图面板综合仅 5.0（最低分），视觉质量 3 分、丰富度 3 分，是全局最大短板。

**改进方案**:
- 添加地形纹理（山脉、河流、平原、森林）
- 增加城池标记、势力边界线、古地名标注
- 加入军队、资源点、道路等战略元素
- 采用水墨地图风格，强化三国地理文化

**影响范围**: 地图面板
**预期提升**: 地图综合分 +2.0 ~ +3.0

---

### 🟡 P1 — 应尽快改进

#### 3. 信息呈现优化

**问题**: 部分面板信息层次不够清晰，缺乏统一的展示规范。

**改进方案**:
- 制定统一的 UI 组件规范：图标 + 文字 + 数值的组合方式
- 建立颜色编码体系（生产类绿色、军事类红色、文化类蓝色）
- 统一资源展示格式（图标 + 当前值/最大值 + 进度条）

**影响范围**: 全部面板
**预期提升**: 综合分 +0.3 ~ +0.5

---

#### 4. 三国主题视觉强化

**问题**: 主题命名已达 8 分水平，但视觉符号（立绘、纹饰、字体）仍未体现。

**改进方案**:
- 武将面板加入立绘和古风边框
- 建筑图标融入三国符号装饰
- 全局使用古风字体和传统色彩
- 阵营色标识系统（蜀绿、魏蓝、吴橙）

**影响范围**: 全部面板
**预期提升**: 综合分 +0.5 ~ +0.8

---

### 🟢 P2 — 建议改进

#### 5. 关卡面板内容丰富化

**问题**: 关卡丰富度仅 4 分，结构线性，缺乏分支和变化。

**改进方案**:
- 增加关卡分支路线（主线/支线）
- 添加难度梯度标识和奖励预览
- 融入三国剧情节点（如"赤壁之战"、"官渡之战"）

**影响范围**: 关卡面板
**预期提升**: 关卡综合分 +1.0 ~ +1.5

---

#### 6. 交互反馈增强

**问题**: 交互设计均分 5.3，缺乏微交互和动态反馈。

**改进方案**:
- 按钮 hover 高亮与按压动效
- 建筑升级动画（闪光、粒子特效）
- 科技研发进度动画
- 状态变化过渡动画

**影响范围**: 全部面板
**预期提升**: 综合分 +0.3 ~ +0.5
