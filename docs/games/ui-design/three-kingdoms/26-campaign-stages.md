# 关卡系统 UI 设计（v1.0）

> **风格基调**：水墨江山·铜纹霸业
> **分辨率**：PC 端 1280×800 + 手机端 375×667
> **定位**：攻城略地故事挑战关卡，区别于科技树，是游戏核心PvE推进系统

---

## 一、入口设计

### 主入口
- 主界面导航 Tab「出征」→ 默认进入关卡地图
- 城池地图「点将台」建筑 → 「出征」选项
- 军师建议中「推荐推进关卡」点击跳转

### 解锁条件
- 完成新手引导第4步（首次出征）
- 无等级限制（关卡本身有推荐战力）

## 二、关卡地图（核心界面）

### 2.1 设计理念
- 以三国历史为背景的故事关卡路线图

- 每章8-12个关卡节点，用墨线串联

### 4.2 三星评价系统
- 每关3个星级条件（满足1个=1星，全满足=3星）
- 章节总星数影响章节奖励
- 三星通关后解锁扫荡功能

| 星级条件示例 | 说明 |
|-------------|------|
| 无武将阵亡 | 全员存活 |
| N回合内通关 | 限时挑战 |
| 总伤害>N | 输出挑战 |
| 仅用N名武将 | 人少挑战 |
| 不使用大招 | 限制挑战 |

## 五、关卡战斗过程

### 5.1 战斗场景
- 复用战斗系统（03-combat-system.md）
- 关卡专属背景（黄巾战场/虎牢关/赤壁江面等）
- 天气/地形效果（影响战斗属性）

### 5.2 关卡特殊机制
| 机制 | 效果 | 示例关卡 |
|------|------|---------|
| 雷暴 | 全体速度-10% | 广宗之战 |
| 火攻 | 每回合灼烧伤害 | 火烧赤壁 |
| 水淹 | 低地单位攻击-20% | 水淹七军 |
| 暴风雪 | 每回合冻伤 | 北伐 |
| 地形优势 | 高地攻击+15% | 定军山 |
| 夜战 | 命中率-15% | 偷袭 |

### 5.3 BOSS 战特殊设计
- BOSS 拥有2-3个技能+1个大招
- 血量分阶段（100%/50%/20%触发不同行为）
- 弱点系统（特定属性攻击造成额外伤害）
- BOSS 战不可扫荡，必须手动/半自动战斗

## 七、自动推图

### 7.1 功能说明
- 开启后自动挑战当前关卡
- 自动选择最优阵容
- 自动使用半自动战斗模式
- 失败时自动停止并通知

### 7.2 自动推图设置
- 最大挑战次数限制（防资源浪费）
- 战力差距阈值（低于推荐80%时暂停）
- 失败重试次数（0-3次）
- 获得稀有物品时暂停

## 八、关卡与放置循环

### 8.1 离线推图
- 离线期间自动推图（需开启自动推图）
- 离线推图效率：引用全局离线效率系数
- 离线推图仅推进已三星通关的下一关
- 遇到BOSS关卡自动暂停

### 8.2 挂机收益
- 当前关卡所在章节提供挂机收益
- 每小时产出 = 关卡基础产出 × (1 + 科技加成)
- 推进到更高章节，挂机收益提升
- 三星通关关卡收益 ×1.5

### 8.3 关卡进度与游戏进程
| 章节完成度 | 解锁内容 |
|:---------:|---------|
| 第一章 | 基础武将、建筑Lv1-5 |
| 第二章 | 精英武将、建筑Lv6-10 |
| 第三章 | 科技系统、建筑Lv11-15 |
| 第四章 | 远征系统、建筑Lv16-19 |
| 第五章 | 转生系统、全建筑解锁 |
| 第六章 | 终局内容、全武将解锁 |

## 十一、手机端专项适配

### 11.1 关卡路线图
- 竖向路线图（从下到上）
- 双指缩放查看全图
- 节点大小 ≥ 48×48px
- 滑动浏览长路线

### 11.2 战斗准备
- Bottom Sheet 弹出敌方情报
- 武将选择改为横向滑动卡片
- 「出征」按钮固定在底部

### 11.3 扫荡管理
- 全屏面板
- 章节选择为横向滑动
- 扫荡进度实时更新

### 11.4 触控规范
- 关卡节点 ≥ 48×48px
- 出征按钮 ≥ 48×48px
- 左右滑动切换章节
- 下拉刷新关卡状态
