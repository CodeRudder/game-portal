## 二、一键强化功能 ★核心新增★

### 2.1 功能说明

**一键强化**：系统自动选择材料，将武将强化到当前资源允许的最高等级。

**流程**：
```
1. 玩家点击"一键强化"按钮
2. 系统弹出预览面板：
   - 当前等级 → 强化后等级
   - 消耗资源明细（经验书×N、银两×N）
   - 属性变化预览（武力+5、智力+3...）
   - 战力变化（+120 ↑）
3. 玩家确认 → 执行强化
4. 强化动画：武将卡片金光闪烁 + 等级数字滚动递增
```

## 三、批量升级功能 ★核心新增★

### 3.1 功能说明

**批量升级**：选择多个武将，一键消耗资源将他们全部升级。

### 3.2 批量升级交互流程

**进入方式**：
- PC端：武将列表底部"批量升级"按钮
- 手机端：底部操作栏"批量升级"按钮，或长按卡片进入多选模式

**操作步骤**：
```
步骤1：选择武将（多选模式）
  - 武将卡片变为可勾选状态（左上角出现复选框）
  - 顶部显示"已选择 N 位武将"
  - 提供"全选"/"全选出战"/"全选未出战"快捷按钮
  - 选中的卡片边框变为金色高亮

步骤2：确认升级
  - 弹出批量升级预览面板
  - 显示每位武将的升级方案（当前等级→目标等级）
  - 显示总资源消耗
  - 资源不足时提示"资源仅够升级前N位武将"

步骤3：执行升级
  - 确认后依次执行（每个间隔150ms）
  - 升级成功的卡片播放金光闪烁动画
  - 完成后显示汇总结果
```

## 四、智能编队推荐 ★核心新增★

### 4.1 功能入口

- 武将列表底部"编队"按钮 → 编队界面 → "智能推荐"按钮
- 战斗准备面板中的"智能推荐"按钮
- 主界面编队快捷入口

### 4.2 推荐算法

```
推荐维度权重：
- 战力匹配（40%）：总战力尽可能接近关卡推荐战力
- 阵营羁绊（25%）：优先激活阵营加成
  - 3同阵营：攻击+15%
  - 6同阵营：攻击+25% + 防御+15%
  - 混搭3+3：攻击+10% + 特效加成
- 职业均衡（20%）：坦克+输出+治疗+控制合理搭配
- 关卡克制（15%）：根据敌人特性推荐克制武将
  - 多灼烧敌人 → 推荐高智/治疗型
  - 多物理敌人 → 推荐高防/坦克型
  - BOSS关 → 推荐高爆发+控制型

输出：1-3套推荐方案，每套包含：
- 6个武将及站位
- 总战力
- 匹配度评分（百分比）
- 推荐理由（如"3蜀羁绊激活+灼烧克制"）
```

## 七、武将碎片收集进度可视化 ★核心新增★

### 7.1 碎片收集规则

```
碎片获取途径：
1. 关卡掉落：特定关卡掉落特定武将碎片（每次1-3片）
2. 招募重复：抽到已有武将自动转化为碎片（按稀有度）
   - Uncommon: 5片  Rare: 10片  Epic: 20片
   - Legendary: 40片  Mythic: 80片
3. 商店兑换：每日刷新碎片商品
4. 活动奖励：限时活动碎片
5. 远征掉落：远征关卡随机碎片

升星消耗：
- 1→2星: 20碎片
- 2→3星: 40碎片
- 3→4星: 80碎片
- 4→5星: 150碎片
- 5→6星: 300碎片

升星效果：
- 每星全属性+10%
- 特定星级解锁新技能/强化现有技能
```

### 7.2 碎片进度UI

**武将卡片上的碎片指示**：
- 卡片底部显示微型进度条（宽度100%，高度3px）
- 进度条填充色：蓝色渐变
- 进度≥80%时进度条变为金色脉冲（提示即将升星）

**详情面板碎片区域**：
