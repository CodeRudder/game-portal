## 任务计划

### Task 1 (P1): I4 攻城中断处理

**目标**: 实现攻城中断处理机制 (退出→暂停→重连→继续)

**实现要点**:
- 攻城任务暂停: 暂停行军动画 + 暂停战斗计时
- 状态保存: 当前攻城进度持久化
- 重连恢复: 从存档恢复攻城状态并继续
- 中途退出: 取消攻城, 编队沿原路返回

**影响范围**: SiegeTaskManager + MarchingSystem + WorldMapTab
**新增测试**: >= 3个
**验证**: 中断→恢复→继续攻城全链路

### Task 2 (P1): I5 城防衰减显示

**目标**: 城防值实时衰减显示 + 恢复机制

**实现要点**:
- 攻城期间每秒城防递减动画
- 城防血条实时更新 (SiegeBattleAnimationSystem已有defenseRatio)
- 城防自然恢复 (非攻城期间缓慢恢复)

**影响范围**: SiegeBattleAnimationSystem + PixelWorldMap
**新增测试**: >= 3个

### Task 3 (P1): G5 攻城确认弹窗集成编队选择
