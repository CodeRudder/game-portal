## 对抗性评测重点

1. **失败路径完整性**: 攻城失败→伤亡→结算→回城是否完整闭环
2. **撤退路径覆盖**: sieging/paused/settling三种状态撤退是否都正确处理
3. **资源一致性**: 失败/撤退后资源总量是否守恒
4. **事件链路**: 失败/撤退路径的事件发射和监听是否完整
5. **边界条件**: 行军中撤退、集结中撤退、战斗最后时刻撤退
