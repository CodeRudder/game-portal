## 验收标准

1. 编队必须有将领+士兵，否则无法出征
2. 攻城后士兵有伤亡，武将可能受伤
3. 编队中的将领不能重复使用
4. 将领受伤后无法出征，需要恢复时间
5. 快捷键完整可用(V/方向键/+/-/Escape/空格)
6. 响应式布局(PC+手机)
7. 性能达标(60fps)
8. E2E全链路通过
9. 文档完整
10. 所有测试通过(0 failures)
