| 指标 | 数值 |

| Token 引用 (`var(--tk-z-*)`) | **30 处** |
| 硬编码数值 | **25 处** |
| Token 使用率 | **54.5%** |
| 高风险硬编码 (≥100) | **0 处** ✅ |

**分析**: 
- ✅ 已建立完整的 z-index token 体系（15 层级：base→scene→tab-bar→...→offline-modal）
- ✅ 无高风险硬编码（所有硬编码值均为 1-30 的组件内部层叠）
- ⚠️ 25 处硬编码均为**组件内部微层叠**（如 `z-index: 1` 用于 overlay、`z-index: 10` 用于按钮），不与全局 token 冲突
