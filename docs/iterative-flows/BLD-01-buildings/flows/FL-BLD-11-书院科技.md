# FL-BLD-11 书院科技

> **版本**: v2 | **最后更新**: 2026-05-05
> **归属功能**: 建筑系统 — 辅助建筑交互 (Phase E)
> **触发条件**: 玩家打开书院详情面板（简单建筑，无Tab，概览+升级合并）
> **结束条件**: 查看完毕或发起升级操作
> **前置状态**: 书院已建造且等级 >= 1
> **后置状态**: 查看完毕（升级操作走 FL-BLD-03）
> **来源文档**: [PRD BLD-6 书院互动设计](../../games/three-kingdoms/ui-design/prd/BLD-buildings-prd.md#bld-6) / [UI BLD-2 子Tab配置](../../games/three-kingdoms/ui-design/ui-layout/BLD-buildings.md#bld-2)

## 流程导航与结构

```
FL-BLD-11 书院科技
├── FL-BLD-11-01 查看科技点产出 📱
├── FL-BLD-11-02 查看研究队列 📱
├── FL-BLD-11-03 查看科技树预览
├── FL-BLD-11-04 加速研究 ⏱
└── FL-BLD-11-05 查看全建筑加成效果
```

## 关键要求

- 书院为简单建筑（无Tab），概览+升级合并显示。面板内5个功能区使用折叠/展开机制，默认折叠显示标题+核心数值
- 科技点每秒产出，受主城/科技/武将智力加成（科技点不适用声望加成，因声望已在主城加成因子中间接体现）
- 可排队多个科技研究，自动推进
- 科技完成 → 全建筑加成回流（农耕/商贸/冶炼/航海/求贤等）
- 研究加速消耗铜钱/元宝

### R2 新增约束

- **科技点产出公式**: `产出/秒 = base × (1 + castle%) × (1 + tech%) × (1 + hero_int%)`
  注: 不含声望加成因子，因声望已通过主城等级间接影响castle%因子
- **面板折叠/展开**: 5个功能区（产出/研究队列/科技树/加速/加成效果）默认折叠，点击标题栏展开。提供锚点导航快捷跳转
- **双击防护**: 加速/研究操作按钮点击后立即禁用

## 验收标准

- 科技点产出和加成明细正确显示
- 研究队列状态准确
- 科技树预览可浏览
- 加速操作正确执行

## 步骤详情

### FL-BLD-11-01 查看科技点产出

> **来源**: [PRD BLD-6 书院互动设计](../../games/three-kingdoms/ui-design/prd/BLD-buildings-prd.md#bld-6)

**用户感知**: 在书院详情面板看到 → 当前科技点产出速率（/秒）+ 加成来源展开列表 + 研究速度加成。面板内功能区默认折叠
**系统行为**: 计算科技点产出（`base × (1+castle%) × (1+tech%) × (1+hero_int%)`，不含声望因子） → 渲染产出区块
**关键要求**: 加成来源可点击展开查看明细。功能区折叠/展开，锚点导航快捷跳转
**验收标准**: 产出数据准确

**📱 空间要点**: 详情面板产出区块

---

### FL-BLD-11-02 查看研究队列

> **来源**: [PRD BLD-8 书院研究加速](../../games/three-kingdoms/ui-design/prd/BLD-buildings-prd.md#bld-8)

**用户感知**: 看到当前研究队列 → 当前研究项目（名称+进度条+剩余时间） + 排队中的项目列表
**系统行为**: 从科技子系统获取研究队列数据 → 渲染队列
**关键要求**: 队列自动推进，当前研究完成后下一项自动开始
**验收标准**: 队列状态准确，进度实时更新

**⏱ 时间要点**: 研究时间从数分钟到数小时不等

---

### FL-BLD-11-03 查看科技树预览

> **来源**: [PRD BLD-8 研究预览](../../games/three-kingdoms/ui-design/prd/BLD-buildings-prd.md#bld-8)

**用户感知**: 点击[科技树]链接/按钮 → 打开科技树面板（全屏或弹窗） → 浏览可研究科技+前置条件+效果预览
**系统行为**: 加载科技树数据 → 渲染科技节点和连线
**关键要求**: 已完成科技标记为绿色，可研究标记为黄色，未解锁标记为灰色
**验收标准**: 科技树正确渲染

---

### FL-BLD-11-04 加速研究

> **来源**: [PRD BLD-8 加速研究](../../games/three-kingdoms/ui-design/prd/BLD-buildings-prd.md#bld-8)

**用户感知**: 点击[加速]按钮 → 选择加速方式（铜钱/元宝） → 扣费 → 研究时间缩短
**系统行为**: 校验加速资源 → 扣除费用 → 减少剩余研究时间
**关键要求**: 研究加速与建筑升级加速机制独立
**验收标准**: 加速正确生效

**⏱ 时间要点**: 加速操作 <= 500ms

**异常分支**:

| 条件 | 处理方式 | 转至步骤 |
|------|---------|:--------:|
| 铜钱不足 | Toast"铜钱不足" | 11-02 |
| 无活跃研究 | [加速]按钮隐藏 | 11-01 |

---

### FL-BLD-11-05 查看全建筑加成效果

> **来源**: [PRD BLD-4 联动加成](../../games/three-kingdoms/ui-design/prd/BLD-buildings-prd.md#bld-4)

**用户感知**: 在书院面板看到 → 已研究科技对各建筑的加成效果列表（如"农耕改良→农田+10%"）
**系统行为**: 汇总所有已研究科技的加成效果 → 按建筑分组显示
**关键要求**: 加成效果回流到对应建筑
**验收标准**: 加成效果准确显示
