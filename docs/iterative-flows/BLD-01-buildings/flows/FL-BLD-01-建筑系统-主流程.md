# FL-BLD-01 建筑系统 主流程

> **版本**: v2 | **最后更新**: 2026-05-05
> **流程类型**: L1 主流程（全局路线图）
> **归属功能**: 建筑系统
> **触发条件**: 玩家点击建筑Tab进入建筑系统
> **结束条件**: 玩家离开建筑系统Tab
> **前置状态**: 玩家已进入游戏主界面
> **后置状态**: 建筑系统操作结果已持久化

## 关键要求

- 11座建筑各有关联子系统，建筑数据必须来自子系统，不能凭空设计
- 建筑等级上限受主城等级限制（所有建筑等级 <= 主城等级）
- 建筑升级不需确认弹窗，点击即执行，Toast反馈
- 4种核心资源（粮草/铜钱/矿石/木材）各有明确消费端，不存在僵尸建筑
- 升级队列槽位随主城等级增长（Lv1-5: 1个, Lv6-10: 2个, Lv11-20: 3个, Lv21-30: 4个）

### R2 新增约束

- **命名统一（PRD v3.0）**: 全部使用"酒馆"（非"招贤馆"）、"工坊"（非"铁匠铺"），UI文档需同步升级至v3.0
- **网格规格**: 6列 × 6行 = 36格（11座建筑 + 25格空地），PC端卡片180x120px
- **一键满级确认**: 一键满级操作有独立确认弹窗（总消耗明细 + 确认/取消），区别于单次升级的无确认设计
- **资源快照模式**: 一键满级/批量操作的资源计算基于"开始时刻的资源快照"，不追踪中间产出变化
- **双击防护**: 所有操作按钮在动画/请求期间进入禁用态，服务端响应完成后恢复
- **网络失败回滚**: 所有涉及资源扣除的操作，服务端使用事务机制——失败时自动回滚已扣资源
- **建造任务与升级队列**: 建造新建筑的任务共享升级队列槽位，建造前需检查队列是否有空槽
- **子系统数据合约**: 英雄/贸易/装备子系统提供最小数据契约（见附录）

## 验收标准

- 玩家可浏览所有11座建筑，锁定建筑显示解锁条件
- 玩家可对满足条件的建筑执行升级/建造操作
- 所有资源产出、升级进度、队列状态实时更新
- 手机端与PC端功能一致，布局适配

## 阶段流转

### Phase A: 浏览与导航

| 项目 | 说明 |
|------|------|
| **进入条件** | 玩家点击建筑Tab进入建筑系统 |
| **目标** | 玩家看到建筑网格（PC: 6x6网格 / 手机: 纵向列表），了解各建筑状态 |
| **子流程** | → FL-BLD-02 建筑网格浏览 |
| **退出条件** | 玩家点击某建筑卡片/空地，进入详情面板或建造选择 |
| **异常** | 网络断开时显示缓存数据+断线提示 |

### Phase B: 建筑操作

| 项目 | 说明 |
|------|------|
| **进入条件** | 玩家点击已建造建筑卡片进入详情面板（D区），或点击空地进入建造弹窗（E区） |
| **目标** | 执行建筑升级、建造新建筑、管理升级队列 |
| **子流程** | → FL-BLD-03 建筑升级 / → FL-BLD-04 建造新建筑 / → FL-BLD-06 升级队列管理 |
| **退出条件** | 操作完成（升级开始/建造完成/队列调整完毕），Toast反馈后回到浏览状态 |
| **异常** | 资源不足、队列已满、主城等级不足 |

### Phase C: 资源管理

| 项目 | 说明 |
|------|------|
| **进入条件** | 场景区右下角一键收取按钮始终可见 |
| **目标** | 一键收取所有建筑累积产出资源 |
| **子流程** | → FL-BLD-05 一键收取 |
| **退出条件** | 收取动画完成，资源入账 |
| **异常** | 无产出可收时按钮置灰 |

### Phase D: 复杂建筑交互

| 项目 | 说明 |
|------|------|
| **进入条件** | 玩家进入复杂建筑详情面板，切换到功能Tab |
| **目标** | 执行酒馆招募、工坊锻造、市舶司贸易等特殊功能 |
| **子流程** | → FL-BLD-07 酒馆招募 / → FL-BLD-08 工坊锻造 / → FL-BLD-09 市舶司贸易 |
| **退出条件** | 功能操作完成（招募结果揭晓/装备锻造完成/商队派遣完成），回到详情面板 |
| **异常** | 各子流程独立异常（资源不足、冷却中、建筑等级不足等） |

### Phase E: 辅助建筑交互

| 项目 | 说明 |
|------|------|
| **进入条件** | 玩家进入对应建筑详情面板 |
| **目标** | 管理兵营编队、查看书院科技产出、执行医馆治疗、管理城墙防御 |
| **子流程** | → FL-BLD-10 兵营编队 / → FL-BLD-11 书院科技 / → FL-BLD-12 医馆治疗 / → FL-BLD-13 城墙防御 |
| **退出条件** | 操作完成，回到详情面板 |
| **异常** | 各子流程独立异常 |

## 主流程异常汇总

| 异常ID | 条件 | 影响范围 | 处理方式 |
|--------|------|---------|---------|
| E-01 | 网络断开 | 全流程 | 显示缓存数据 + Toast"网络断开，数据可能不是最新" |
| E-02 | 服务器维护 | 全流程 | 建筑Tab显示维护公告，禁止操作 |
| E-03 | 资源不足 | Phase B/D/E | Toast提示缺少的具体资源及数量 |
| E-04 | 队列已满 | Phase B | Toast"升级队列已满"，引导查看队列管理 |
| E-05 | 主城等级不足 | Phase B/D/E | 红色标记前置条件未满足，按钮置灰 |
| E-06 | 建筑等级已达上限 | Phase B | [升级]按钮隐藏，显示"已达上限" |
