### [数据一致性] IF-03 | 编队生命周期终点矛盾——销毁 vs 复用

**检查项**: 编队状态流转跨流程一致性
**关联流程**: FL-MAP-09 Stage P10 步骤6 + FL-MAP-16 编队状态映射表

### P0 问题（按流程分组）

#### P0-INT-01 | AI敌方系统未定义，多个流程无法闭环
- **来源**: DP P1-14 + INT IF-11（合并）
- **涉及流程**: FL-MAP-04, FL-MAP-06, FL-MAP-15, FL-MAP-12
- **问题描述**: 18个流程中多处提及AI敌方行为（AI攻城、AI行军、AI占领领土），但从未定义AI行为规则。行军拦截(FL-MAP-04)无法触发、领土丧失(FL-MAP-06 E06-04)无法发生、离线变化(FL-MAP-15)无法计算AI领土变化。
- **修复建议**: 在INDEX.md头部声明AI系统范围。若当前实施范围外，标注AI相关分支为"预留接口"。若存在，补充基本AI行为声明。

#### P0-INT-02 | 攻城期间浏览地图缺少"返回攻城"入口
- **来源**: UP P0-03 + INT IF-05（合并）
- **涉及流程**: FL-MAP-09 Stage P7
- **问题描述**: Stage P7允许"缩放地图、拖拽查看"但无返回攻城画面的机制。用户拖拽离开后不知如何回到攻城，攻城进度是否持续可见未定义，攻城结束时如何唤回用户未说明。
- **修复建议**: 定义"返回攻城"浮动按钮（右上角，攻城期间常驻），攻城进度条缩小为mini条常驻顶部，攻城结束时全屏通知。

#### P0-INT-03 | 本地战斗引擎公式集不完整
- **来源**: DP P0-01（独立）
- **涉及流程**: FL-MAP-09 Stage P8, FL-MAP-09-11
- **问题描述**: 每回合城防衰减公式已给出，但完整战斗引擎缺失：攻方兵力是否每回合递减？守方是否反击？战斗时长10~60s与20回合的映射关系？攻方总战力完整公式仍引用PRD未内联。
- **修复建议**: 补充战斗引擎完整公式清单，内联攻方总战力计算公式。

#### P0-INT-04 | 编队总战力（攻方总战力）公式未内联
- **来源**: DP P0-02（独立）
- **涉及流程**: FL-MAP-09 Stage P3, AC-09-03
- **问题描述**: 城防衰减公式核心输入"攻方总战力"仅标注"详见PRD"。FL-MAP-07-02定义了武将战力系数公式，但"编队总战力"的聚合方式（兵力x系数？多将领求和？加权平均？）未定义。
- **修复建议**: 在FL-MAP-09 P3内联完整公式：编队总战力 = Σ(各编队兵力 x 进攻武将战力系数)。

#### P0-INT-05 | 城防初始值公式缺失
- **来源**: DP P0-03（独立）
- **涉及流程**: FL-MAP-09 Stage P8
- **问题描述**: "读取城防初始值（具体计算公式详见PRD）"——城防初始值是战斗引擎的核心输入，缺失此值无法开始战斗计算。城防系数已给出(1.0+城墙等级x0.05)，但基础城防值未定义。
- **修复建议**: 补充城防初始值公式（建议：基础城防值 x 城防系数，基础城防值按城市类型/等级查表）。

#### P0-INT-06 | 地形移动成本(moveCost)具体数值缺失
- **来源**: DP P0-04（独立）
- **涉及流程**: FL-MAP-04-01, FL-MAP-03-01
- **问题描述**: A*寻路权重计算依赖各地形moveCost值，仅水域(2.14)已知，其他4种地形（平原/山地/森林/沙漠）的moveCost均未给出。
- **修复建议**: 补充5种地形完整moveCost数值表。

#### P0-INT-07 | 粮草消耗公式缺失
- **来源**: DP P0-05 + UP P1-04（合并）
- **涉及流程**: FL-MAP-09 Stage P2
- **问题描述**: 粮草消耗是攻城条件校验的一部分，但"基于兵力数、行军距离、策略修正计算（具体公式详见PRD）"。缺此公式无法实现攻城条件校验。用户在征服失败后"重整旗鼓"的消耗也不透明。
- **修复建议**: 内联粮草消耗完整公式及各项系数。

#### P0-INT-08 | 5档伤亡判定优先级树缺失
- **来源**: DP P0-06（独立）
- **涉及流程**: FL-MAP-09-12, FL-MAP-17 S1
- **问题描述**: 5档定义有时间维度和城防维度两个判定轴，但未明确判定优先级。开发者可能理解为5个平行条件，导致T<20s但城防未归零时行为不确定。

#### P0-INT-09 | 数据持久化架构未定义
- **来源**: DP P0-07 + INT IF-12（合并）
- **涉及流程**: FL-MAP-01, FL-MAP-08, FL-MAP-15, FL-MAP-16 等
