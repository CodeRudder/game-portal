## 玩家流程

### P1: 装备面板 → 浏览兵种装备列表

```
1. 玩家打开"装备"Tab（EquipmentTab.tsx）
2. 背包网格展示已拥有装备（EquipmentBagManager 管理）
3. 玩家切换分类Tab: 全部/武器/防具/饰品/坐骑
4. 选择"品质降序"排序: 金色紫金刀(+12) → 紫色青龙甲(+8) → ...
5. 筛选"未穿戴": 显示背包中可用装备
6. 点击紫金刀查看详情:
   - 主属性: 攻击 +850 (基础200 × 紫色2.0倍 × (1+12×0.15))
   - 副属性: 暴击+120, 速度+80
   - 特殊词条: 破甲(紫色20%概率)
   - 套装: 战神套(1/4)
```

**验证点**: `calculateMainStatValue(eq) === 850`

### P2: 编组操作 → 武将穿戴装备 + 套装激活

```
1. 玩家打开"武将"面板 → 选择"关羽"
2. 装备栏4槽位: 武器(空)/防具(铁甲)/饰品(玉佩)/坐骑(空)
3. 从背包拖拽"紫金刀"到武器槽 → equipItem('guanyu', 'uid-123')
4. 系统自动卸下旧装备(铁甲返回背包) → replacedUid='uid-456'
5. 穿戴后属性变化:
   - 攻击: 320 → 1170 (+850)
   - 暴击: 5% → 17% (+12%)
6. 继续穿戴战神套防具"霸王铠" → 战神套计数=2
7. 套装效果激活: 战神2件套 → 攻击+15%
8. 武将总战力: 4,580 → 6,820
```

**验证点**: `getActiveSetBonuses('guanyu') 包含 {setId:'warrior', activeTiers:[2]}`

### P3: 兵种强化 → 强化系统 + 保护符

```
1. 玩家选择"紫金刀" → 打开强化面板(EnhancePanelComponent)
2. 当前等级: +12, 品质上限: 紫色+12(已达上限)
3. 提示"已达品质强化上限，请先提升品质"
4. 玩家回到背包，选择金色"方天画戟"+8
5. 强化到+9: 成功率85%, 消耗铜钱12,000+强化石8
6. 强化成功 → 属性提升
7. 继续强化+10: 成功率70%, 使用银保护符
8. 强化成功 → +10, 保护符未消耗(成功不消耗)
9. 强化+11: 成功率55%, 使用金保护符
10. 强化失败 → 等级不变(保护符生效), 消耗金保护符×1
```

**验证点**: `enhance('uid-789', true).outcome === 'fail' && currentLevel === 10`

### P4: 兵种升级 → 炼制系统 + 保底

```
1. 玩家打开铁匠铺 → 炼制面板(ForgePanelComponent)
2. 背包有3件蓝色装备 → 选择基础炼制
3. 消耗: 铜钱5,000 + 强化石3
4. 炼制结果: 蓝色→紫色(概率78%) → 成功！获得紫色"凤凰羽扇"
5. 连续炼制10次未出金色 → 保底计数器: targetedGoldPity=10
6. 第11次炼制 → 保底触发 → 必出金色装备
7. 获得金色"倚天剑" → 图鉴更新: 倚天剑 bestRarity=gold
```

**验证点**: `forgePityState.targetedGoldPity >= 10 → pityTriggered === true`

### P5: 编组战力总览 → 一键推荐 + 交叉验证

```
1. 玩家打开"编队"面板(FormationPanel.tsx)
2. 当前编队: 关羽/张飞/赵云/诸葛亮/马超
3. 编队总战力: 32,450
4. 点击"一键推荐装备"(EquipmentRecommendSystem)
5. 系统分析每位武将属性短板:
   - 关羽: 缺攻击 → 推荐战神套武器
   - 张飞: 缺防御 → 推荐守护套防具
   - 诸葛亮: 缺智力 → 推荐谋士套饰品
6. 展示对比弹窗: 更换前 vs 更换后属性
7. 确认后批量穿戴 → 编队战力: 32,450 → 41,280
8. 交叉验证:
   - 每件装备 power 之和 = 编队总战力增量
   - 套装效果加成已计入
```

**验证点**: `recommend('guanyu').totalScore > 0 && 套装建议.length >= 0`
